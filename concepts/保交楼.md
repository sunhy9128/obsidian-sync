# 保交楼

> 来源：巫师财经《吃透中国房价的一切，但是主神视角》

## 定义

**保交楼** = 在房企暴雷、项目停工的背景下，政府推动确保购房人能拿到房子（住宅交付）的政策行动。

> 「保交楼的背后其实是救房企，你得让他们给你干活」

## 核心命题

### 一、为什么保交楼？

#### 购房者角度

买了房却住不进去：

```
首付100-300万
       ↓
等待3年交房（期房制度）
       ↓
房企暴雷 → 项目停工
       ↓
房子烂尾 → 钱房两空
       ↓
成为「社会不安定因素」
```

#### 社会稳定角度

大量烂尾楼 → 大量无房可住的购房者 → 社会不安定因素：

| 群体 | 规模 | 风险等级 |
|------|------|---------|
| 纯刚需 | 数十万家庭 | ★★★☆☆ |
| 掏空六个钱包 | 数百万家庭 | ★★★★☆ |
| 涉及三代人 | 数千万人口 | ★★★★★ |

> 「这背后是一个个家庭几十年的积蓄，一旦暴雷，民怨沸腾」

### 二、保交楼的时间线

| 时间 | 事件 | 里程碑 |
|------|------|-------|
| 2021年 | 多家民营房企宣布债务违约 | 问题暴露 |
| 2022年7月 | 中央首次提出「保交楼、稳民生」 | 政策定调 |
| 2023年 | 各大银行出台房地产贷款展期政策 | 金融支持 |
| 2024年 | 「白名单」政策落地 | 机制完善 |
| 2025年 | 保交楼交付率大幅提升 | 成效显现 |
| 2026年后 | 进入「深水区」（硬骨头） | 新阶段 |

### 三、保交楼的成果数据

#### 全国数据

| 指标 | 数据 | 来源 |
|------|------|------|
| 白名单贷款 | 5.6万亿元 | 金融监管总局 |
| 支持住房交付 | 1400万套 | 金融监管总局 |
| 各省交付率 | 大多>90% | 省政府工作报告 |
| 河南 | 95% | 省住建厅 |
| 黑龙江/宁夏/青海/湖南 | 100% | 省住建厅 |

#### 评价

> 「根据各省政府工作报告和各省住建厅官网数据，大多数省份在过去的一年保交楼任务完成良好，交付进度基本达90%以上」

### 四、保交楼的运作机制

#### 三层结构

| 层级 | 主体 | 职责 |
|------|------|------|
| **政府** | 中央+地方 | 统筹协调、政策支持 |
| **金融** | 银行+白名单 | 提供贷款、资金监管 |
| **企业** | 房企 | 工程建设、按时交付 |

#### 白名单机制

```
1. 地方政府上报项目名单
         ↓
2. 住建部审核项目资质
         ↓
3. 金融监管总局推送至银行
         ↓
4. 银行独立评审+贷款
         ↓
5. 资金封闭管理（专款专用）
         ↓
6. 工程进度监控
         ↓
7. 按时交付
```

#### 资金封闭管理

> 「白名单项目的贷款金额达5.6万亿」

关键机制：**专户管理、专款专用、封闭运行**

```
销售收入 → 监管账户 → 只用于本项目施工 → 交付
                    → 优先偿还白名单贷款
```

### 五、保交楼与「分摊痛苦」

#### 在民生维度的「分摊痛苦」

| 维度 | 分摊痛苦（原意） | 保交楼 |
|------|----------------|--------|
| 主体 | 房价下跌的损失 | 烂尾楼的损失 |
| 方式 | A→B→C→D阶梯接盘 | 国家+银行+房企兜底 |
| 目的 | 避免一个人承担全部损失 | 避免购房者钱房两空 |
| 效果 | 维持社会稳定 | 维持社会稳定 |

#### 保交楼的意义

1. **政治意义**：房子是中国人的「根」，保交楼 = 保民生
2. **社会稳定**：减少「社会不安定因素」
3. **经济意义**：维持房地产链条正常运转
4. **金融意义**：防止银行坏账进一步恶化

### 六、保交楼的挑战与局限

#### 2026年后的「深水区」

> 「你也知道，2026年之后，保交楼将进入深水区，你需要做更多筹划」

| 挑战 | 说明 |
|------|------|
| 硬骨头项目 | 资金缺口大、法律关系复杂 |
| 地方财力 | 地方财政压力增大 |
| 银行意愿 | 银行风险偏好下降 |
| 房价下行 | 项目可抵押价值缩水 |

#### 保交楼的道德风险

| 问题 | 后果 |
|------|------|
| 房企知道国家兜底 | 过度冒险，反正有人救 |
| 购房者不关心房企风险 | 闭眼买期房 |
| 银行放松风控 | 不怕贷错，反正国家背书 |

### 七、保交楼 vs 稳三保

| 概念 | 范围 | 优先级 |
|------|------|-------|
| **保交楼** | 买了房的要拿到房 | 经济发展 |
| **稳三保** | 保工资、保民生、保运转 | 社会稳定 |

```
保交楼（经济维度）
       ↓
买了房的人有房住
       ↓
不成为社会不安定因素
       ↓
稳三保（社会维度）
       ↓
工资按时发
       ↓
民生基本保障
       ↓
政府正常运转
       ↓
社会秩序稳定
```

### 八、典型案例：恒大

#### 恒大的保交楼情况

| 时间 | 事件 | 影响 |
|------|------|------|
| 2021年 | 恒大债务危机 | 全国最大烂尾风险 |
| 2022年 | 境外债违约 | 外资恐慌 |
| 2023年 | 重组方案讨论 | 进展缓慢 |
| 2024年1月 | 香港高院清盘令 | 重组失败 |

#### 恒大的教训

1. **规模太大不能倒**：恒大 = 百万套房子 + 数十万购房者
2. **重组分歧**：境外vs境内债权人利益不一致
3. **实控人风险**：被调查 → 没人敢签字
4. **清盘结局**：债权人回收率0.5%-3%

**恒大的保交楼意义**：必须与债务重组同步推进，否则无人敢接手
