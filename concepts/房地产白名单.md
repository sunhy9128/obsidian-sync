# 房地产白名单

> 来源：巫师财经《吃透中国房价的一切，但是主神视角》

## 定义

**房地产白名单** = 2024年开始实施的房地产融资协调机制，由金融监管总局推动，对符合条件的房地产项目提供贷款支持，确保房子能建好交付。

> 「你得让他们给你干活，你站在高台上，看着底下这堆精明的商人，你一拍桌子，恩威并施，你说到，有些人可以救，但有些人不值得救，大家心领神会」

## 核心命题

### 一、为什么需要白名单？

#### 问题背景

| 时间 | 市场状况 | 风险 |
|------|---------|------|
| 2021年前 | 房企高杠杆+高周转 | 积累风险 |
| 2021年 | 调控收紧+销售下滑 | 风险暴露 |
| 2022年 | 多家房企暴雷 | 烂尾潮 |
| 2023年 | 信心危机 | 销售冰封 |

**核心矛盾**：
- 银行想收紧房地产贷款（风险大）
- 房企急需贷款续命（不然就烂尾）
- 购房者需要房子交付（民生问题）

#### 白名单的解决方案

> 「有些人可以救，但有些人不值得救」

**分类施策**：
- 好项目 + 好房企 → 正常贷款
- 好项目 + 坏房企 → 收并购或白名单
- 坏项目 + 坏房企 → 市场出清

### 二、白名单的核心数据

| 指标 | 数据 | 说明 |
|------|------|------|
| 白名单贷款金额 | **5.6万亿元** | 金融监管总局披露 |
| 支持住房建设交付 | **1400万套** | 覆盖全国主要项目 |
| 覆盖省份 | 31个 | 全国覆盖 |
| 平均单套贷款 | ~40万元 | 1400万套/5.6万亿 |

### 三、白名单的运作机制

#### 申报流程

```
1. 地方政府梳理项目
         ↓
2. 筛选符合条件的项目
   （土地/手续/工程进度等）
         ↓
3. 上报至住建部
         ↓
4. 推送至金融监管总局
         ↓
5. 分配至商业银行
         ↓
6. 银行独立评审+贷款
         ↓
7. 资金封闭管理
```

#### 准入条件（项目层面）

| 条件 | 说明 |
|------|------|
| 四证齐全 | 土地证/规划证/施工证/预售证 |
| 项目公司独立 | 隔离风险 |
| 资金封闭 | 销售收入优先还贷 |
| 股东支持 | 母公司不能完全躺平 |
| 地方政府认可 | 纳入保交楼计划 |

#### 准入条件（房企层面）

| 条件 | 说明 |
|------|------|
| 非失信被执行人 | 信用记录 |
| 非重大违法违规 | 合规经营 |
| 积极配合 | 不能躺平耍赖 |
| 有在途项目 | 真实在建工程 |

### 四、白名单的「分摊痛苦」逻辑

#### 风险分摊机制

| 主体 | 承担什么 | 动力 |
|------|---------|------|
| **国家** | 政策协调+信用背书 | 维稳 |
| **地方政府** | 协调+监督 | 民生 |
| **银行** | 贷款风险 | 利息收入+政策任务 |
| **房企** | 继续建设+按时交付 | 活命 |
| **购房者** | 等待+部分损失 | 拿到房 |

#### 各方利益平衡

```
国家：稳房价 → 稳人心
         ↓
地方政府：保交楼 → 稳民生
         ↓
银行：放贷款 → 收利息（但承担风险）
         ↓
房企：拿到钱 → 继续干活
         ↓
购房者：等待 → 拿到房
```

### 五、白名单 vs 传统房地产开发贷款

| 维度 | 传统开发贷 | 白名单贷款 |
|------|-----------|-----------|
| **对象** | 优质房企+优质项目 | 问题房企+问题项目 |
| **风险** | 银行自担 | 银行+政策共担 |
| **利率** | 市场利率 | 略低于市场（政策让利） |
| **审批** | 银行独立 | 政府和银行双重审核 |
| **用途** | 开发建设 | 专款用于交付 |
| **担保** | 土地抵押 | 封闭管理+项目抵押 |

### 六、白名单的局限性

#### 1. 规模限制

> 「你也知道，2026年之后，保交楼将进入深水区，你需要做更多筹划」

| 问题 | 说明 |
|------|------|
| 好项目用完了 | 剩下的都是硬骨头 |
| 资金缺口大 | 5.6万亿可能不够 |
| 地方财力受限 | 配套资金跟不上 |

#### 2. 道德风险

| 主体 | 道德风险表现 |
|------|-------------|
| 房企 | 反正有人兜底 → 继续激进 |
| 银行 | 有政策背书 → 放松风控 |
| 地方政府 | 反正有国家 → 过度申报 |

#### 3. 效率问题

| 问题 | 影响 |
|------|------|
| 审批流程长 | 项目等米下锅 |
| 银行动力不足 | 风险大+收益低 |
| 地方协调难 | 多部门联动效率低 |

### 七、白名单与12万亿化债方案

#### 两者的关系

| 项目 | 侧重点 | 解决什么问题 |
|------|-------|------------|
| **白名单** | 保交楼 | 烂尾楼 → 交付 |
| **12万亿化债** | 地方债 | 地方融资平台 → 展期降息 |

**共同目标**：防范系统性风险，实现软着陆

#### 协同效应

```
白名单（房地产）← 化债（地方政府）
     ↓                    ↓
保交楼              地方有财力支持
     ↓                    ↓
购房者拿到房      房地产正常运转
     ↓                    ↓
社会稳定          经济发展
```

### 八、未来展望

#### 短期（2026-2027）

| 任务 | 重点 |
|------|------|
| 收尾 | 已申报项目完成交付 |
| 分类 | 好项目继续支持，坏项目出清 |
| 风险 | 防止二次暴雷 |

#### 中期（2028-2030）

| 任务 | 重点 |
|------|------|
| 转型 | 从"救"到"稳"，建立长效机制 |
| 制度 | 完善预售制度，减少烂尾风险 |
| 模式 | 探索现房销售 |

#### 长期（2030+）

| 目标 | 说明 |
|------|------|
| 行业出清 | 房企数量大幅减少 |
| 模式转型 | 从高杠杆→精细化运营 |
| 制度完善 | 金融风险与民生保障平衡 |
