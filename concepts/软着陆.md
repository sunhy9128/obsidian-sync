# 软着陆

> 来源：巫师财经《吃透中国房价的一切，但是主神视角》

## 定义

**软着陆** = 房价在下行大势中，通过拉长时间轴，让价格下行的「夹角」变小，实现**缓慢、温和、阶梯式的价格回归**。

> 「房价的下行不是一年两年，不是五年十年，而是要**持续一代人，甚至是两代人**」

## 核心命题

### 一、主神的战略逻辑

作为「主神」（决策层），核心战略逻辑是：

```
稳房价 → 稳市场 → 稳银行 → 稳金融 → 稳秩序 → 稳社会 → 稳人心
```

| 传导链 | 说明 | 失控后果 |
|-------|------|---------|
| 稳房价 | 防止房价快速崩盘 | 市场恐慌 |
| 稳市场 | 防止抛售踩踏 | 流动性枯竭 |
| 稳银行 | 防止坏账爆发 | 系统性风险 |
| 稳金融 | 防止信用崩塌 | 金融危机 |
| 稳秩序 | 防止社会动荡 | 政治风险 |
| 稳社会 | 防止共识崩塌 | 亡天下 |
| **稳人心** | **最终目的** | - |

**稳人心是最终目的，而稳房价是实现手段。**

### 二、软着陆 vs 硬着陆

| 类型 | 形态 | 夹角 | 时间 | 后果 |
|------|------|------|------|------|
| **硬着陆**（跳楼） | 一路崩盘 | 逼近90度 | 短 | 银行系统崩溃、经济通缩、社会动荡 |
| **软着陆** | 拉长时间轴 | 夹角变小 | 长 | 分摊痛苦、阶梯传导、社会稳定 |

#### 硬着陆的传导链（小A的故事）

```
小A 1000万买房（首付300万+贷款700万）
       ↓
房价暴跌至300万
       ↓
小A算了算：首付300万能买一套新房，心态崩了
       ↓
收入下降（经济通缩周期）→ 裁员 → 断贷
       ↓
银行收回房子，拍卖得300万
       ↓
还欠银行400万 → 小A后半生继续打工还债
       ↓
小A成为「社会不安定因素」
       ↓
千千万万个「小A」形成共识 → 死亡冲锋
       ↓
社会信用体系崩塌
```

#### 为什么不能硬着陆？

> 「一旦大量的个体不贡献自己的【锚】了，不但人类顶层的那一小撮最聪明的个体无法继续享受，社会本身也会停滞不前，甚至倒退」

### 三、软着陆的时间跨度与目标

**终极目标**：

> 「就算房价会降很多，但你最终希望房价依旧会降到一个相对合理的位置，甚至还要有点小贵，然后一直震荡横盘」

可以理解为：买一个房子从工作25年变成工作15年（**25和15都是虚值，意会即可**）

**核心约束**：

> 「房价不能跌到普通人能轻易买到的地步，因为那样会失去'锚'的激励效应」

### 四、软着陆的实现机制

巫师的「四招」：

#### 1. 减少供给
- 房地产开发投资完成额：逐年-30%
- 房屋新开工面积：2025Q3较2023同期接近腰斩
- 目的：减缓价格下行速度

#### 2. 价格保护
- 窗口指导：限制挂牌价下限
- 业主自发保价联盟（但往往失败）
- 目的：防止恐慌性抛售

#### 3. 暗抬购买力
- 第四代住宅加量不加价
- 目的：让买方觉得"划算"，愿意继续玩

#### 4. 分摊痛苦（核心机制）
- A→B→C→D阶梯式接盘
- 详见：[[分摊痛苦]]

### 五、关键：防止「共识」

| 共识类型 | 结果 | 危险程度 |
|---------|------|---------|
| 一致看多 | 自杀式爆炒 | ★★★★☆ |
| **一致看空** | **暴跌连带着全国崩** | **★★★★★** |

**两种共识都危险，必须让市场对房价产生分歧。**

为什么分歧比共识更安全？
- 分歧 = 有人买、有人卖 = 成交量存在 = 市场活着
- 共识 = 无人接盘 = 流动性枯竭 = 市场死亡

### 六、量化指标体系

| 指标 | 现状 | 趋势 | 意义 |
|------|------|------|------|
| 房价收入比（北京/上海） | ~37倍 | ↓（从50倍下降30%） | 购买力改善 |
| 租金回报率 | ~2% | ↑（从1.3%回升） | 接近无风险利率 |
| 房地产开发投资完成额 | - | ↓↓（-30%/年） | 供给收缩 |
| 房屋新开工面积 | - | ↓↓（接近腰斩） | 供给收缩 |
| 房地产对GDP拉动率 | 负数 | ↓（拖累GDP） | 房地产从引擎变负担 |

### 七、国际比较：各国软着陆尝试

#### 日本1990年代：硬着陆的典型

| 指标 | 结果 |
|------|------|
| 房价跌幅 | 累计下跌约60-70% |
| 时间 | 持续10年以上 |
| GDP | 进入"失去的二十年" |
| 社会影响 | 自杀率上升、"消失的一代" |

**教训**：日本选择了快速出清（硬着陆），导致了长期的通缩螺旋。

#### 美国2008年：软着陆 vs 硬着陆的混合

| 措施 | 内容 |
|------|------|
| 量化宽松（QE） | 美联储购买MBS，向市场注入流动性 |
| 零利率政策 | 降低借贷成本 |
| TARP | 救助系统重要性金融机构 |
| 消费者保护 | 延缓止赎 |

**结果**：避免了金融机构连环倒闭，但房地产市场复苏缓慢（约5-7年才触底）。

#### 中国2021年至今：进行时

| 政策工具 | 具体措施 |
|---------|---------|
| 保交楼 | 白名单贷款5.6万亿，1400万套 |
| 债务重组 | 短债拉长+利息打折+债转股 |
| 12万亿化债 | 高息短债→低息长债 |
| 财政扩张 | 2025赤字率4%（改革开放以来新高） |

### 八、软着陆成功的充要条件

**必要条件**：
1. 供给侧收缩（减少新开工）
2. 需求侧管理（防止恐慌）
3. 金融体系稳定（银行不倒闭）
4. 社会预期分化（不形成一致看空）

**充分条件**：
1. 经济保持正增长（居民收入不崩溃）
2. 新增就业持续创造（社会秩序稳定）
3. 政策定力（不因短期压力而大幅刺激）
4. 时间换空间（代际交替稀释泡沫）
