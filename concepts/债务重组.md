# 债务重组

> 来源：巫师财经《吃透中国房价的一切，但是主神视角》

## 定义

**债务重组** = 债权人（如银行）与债务人（如房企）协商，修改原债务条款，包括但不限于：延长还款期限、降低利率、削减本金、债转股等，将债务人的短期还款压力分摊到更长时间维度。

> 「债务不会消失，只会转移。化的谁的债，把债化给了谁？」

## 核心命题

### 一、债务重组的操作方式

#### 四种主要手段

| 手段 | 机制 | 效果 |
|------|------|------|
| **短债拉长** | 今年还 → 3-5年后还 | 时间换空间 |
| **利息打折** | 5% → 3% 或 本金减免 | 降低还款压力 |
| **宽限期延长** | 6个月 → 18个月 | 延后偿债高峰 |
| **债转股/债转项目** | 债务 → 股份/项目收益权 | 出表+利益绑定 |

#### 数学示例

```
原始债务：本金1亿 + 利息2000万 = 1.2亿

重组方案：
1. 短债拉长：5年还清 → 10年还清
2. 利息减免：免除全部利息
3. 本金打折：1亿 → 5000万
4. 宽限期：前2年只还息不还本

最终债权人回收：5000万（约41%）
```

### 二、债权人为什么答应？

> 「债权人一看，我靠，你欠我本金加利息一共1个亿，现在东砍砍西砍砍，最后变成一堆新的债券和股票，只剩3000万了，真黑啊」

**但债权人还是答应了**，原因有三：

#### 原因1：没得选

| 选项 | 回收率 | 概率 |
|------|-------|------|
| 拒绝重组 | 0.5%-3%（清盘） | 视情况 |
| 接受重组 | 30%-50% | 更高 |

**数学决定**：接受重组的期望回收率 > 拒绝重组

#### 原因2：签字没压力

> 「账目不需要具体关键签字人承担个人责任，所以签字没压力」

重组是**集体决策**，个人无需承担"贱卖资产"的罪名。

#### 原因3：有能力吃下

> 「银行、信托等大型国家金融机构有能力吃下这个坏账」

- 国家背书 = 信用背书
- 规模够大 = 可以分散损失
- 政治考量 = 维稳优先

### 三、二次重组：最坏情况

> 「如果到时候房子卖不出去，股价也涨不上去，项目收益也不理想，那又得再砍一刀，出现**二次重组**，3000万再变900万」

#### 二次重组的触发条件

| 条件 | 说明 |
|------|------|
| 房价继续跌 | 抵押物价值不足 |
| 销售不畅 | 现金流无法改善 |
| 再融资失败 | 借新还旧不行 |
| 实控人风险 | 被调查/跑路 |

#### 恒大的教训

| 时间 | 事件 | 结果 |
|------|------|------|
| 2021年 | 恒大债务危机曝光 | 市场恐慌 |
| 2023年4月 | 离岸债重组方案 | Class A 77%同意，Class C 30%否决 |
| 2024年1月 | 香港高院清盘令 | 不同意重组 → 清盘 |
| 清盘回收 | 境外债债权人 | 回收率3%甚至0.5% |

**教训**：拖延和分歧只会让情况更糟，早重组 > 晚重组 > 清盘

### 四、债务重组的传导机制

#### 债务不会消失，只会转移

债务重组的本质是**锚的转移**：

```
原始债权人（银行/信托）
       ↓ 债务重组
债务人（房企）获得喘息
       ↓
部分损失由以下主体承担：
- 银行股东（股价下跌）
- 债券持有人（本金损失）
- 地方政府（土地财政减少）
- 纳税人（最终兜底）
- 消费者（通胀稀释）
```

#### 债务转移的三条路径

| 路径 | 传导机制 | 承担者 | 时间 |
|------|---------|-------|------|
| 银行承担 | 坏账拨备 | 银行股东/储户 | 当期 |
| 国家承担 | 财政兜底 | 纳税人 | 中期 |
| 社会承担 | 通胀稀释 | 全体货币持有者 | 长期 |

### 五、债务重组与软着陆

#### 在企业层面实现「分摊痛苦」

| 维度 | 分摊痛苦（宏观） | 债务重组（微观） |
|------|----------------|----------------|
| 主体 | 全社会 | 房企+债权人 |
| 方式 | A→B→C→D接盘 | 债转股+削债+展期 |
| 时间 | 代际分散 | 跨期分散 |
| 目的 | 避免社会动荡 | 避免企业倒闭 |

#### 债务重组的局限

1. **道德风险**：房企知道可以重组 → 过度冒险
2. **银行损伤**：坏账累积 → 信贷紧缩
3. **挤出效应**：资源流向低效房企 → 阻碍经济转型
4. **政府负担**：隐性债务膨胀 → 财政风险

### 六、典型案例：某大型房企债务重组

#### 重组前

```
总债务：2万亿
- 银行贷款：8000亿
- 信托贷款：4000亿
- 债券：5000亿
- 应付账款：3000亿

资产：1.5万亿（主要为土地+项目）
净资产：-5000亿（技术性破产）
```

#### 重组方案

| 债务类型 | 处理方式 | 债权人回收率 |
|---------|---------|------------|
| 银行贷款 | 展期10年+降息 | ~70% |
| 信托贷款 | 展期5年+部分债转股 | ~50% |
| 国内债券 | 折价兑付 | ~60% |
| 离岸债券 | 债转股+折价 | ~30% |
| 应付账款 | 展期+商转股 | ~40% |

#### 重组后

```
总债务：2万亿 → 8000亿（削债60%）
新增股权：原债权人获得30%股权
管理团队：留任（戴罪立功）
项目：继续建设→保交楼
```

### 七、债务重组的操作流程

```
1. 债务人申请 → 债权人受理
       ↓
2. 尽职调查 → 资产评估
       ↓
3. 重组方案制定 → 与主要债权人沟通
       ↓
4. 债权人投票 → 需达到法定多数
       ↓
5. 法院批准（如涉及离岸债）
       ↓
6. 执行重组 → 债务人获得喘息
       ↓
7. 后续监控 → 防止二次违约
```

**关键节点**：债权人投票——往往是最困难的环节（恒大Class C 30% vs 77%门槛）
