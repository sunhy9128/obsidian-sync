# 绝对亏与相对亏

> 来源：巫师财经《吃透中国房价的一切，但是主神视角》

## 定义

**绝对亏**：你持有的房产从1000万跌到800万，这200万的下跌。

**相对亏/赚**：如果你旁边的房子从1000万跌到850万，而你跌到了800万，你比别人多亏50万（或少亏），这50万就是「相对亏」或「相对赚」。

## 核心命题

### 一、绝对亏 + 相对赚

如果你持有的资产是区域内比较好的小区（密度小、位置好、物业好、质量好），那你资产相对抗跌：

> 「你**绝对亏** + 你**相对赚**」

#### 数学推导

```
你的房子：买入1000万 → 现在800万
         绝对亏 = 1000 - 800 = 200万（-20%）

周边同类：买入1000万 → 现在600万
         绝对亏 = 1000 - 600 = 400万（-40%）

你的相对赚 = 400 - 200 = 200万
```

**核心结论**：

> 「你跌的越多，那么你就赚的越多。房价越跌你越赚。」

这不是玩笑——而是一个**数学恒等式**：

```
相对赚 = 别人绝对亏 - 自己绝对亏
       = (买入价 - 别人现价) - (买入价 - 自己现价)
       = 自己现价 - 别人现价

当自己现价 > 别人现价时 → 相对赚
当自己现价 >> 别人现价时 → 大相对赚
```

### 二、为什么优质资产更抗跌？

| 护城河类型 | 例子 | 抗跌机制 |
|-----------|------|---------|
| 不可替代价值 | 学区、核心区位 | 需求刚性强 |
| 低密度 | 容积率低、绿化好 | 改善型需求支撑 |
| 优质物业 | 品牌物业、维护好 | 延缓老化速度 |
| 高素质业主 | 高收入、高教育程度 | 维修基金充裕 |

**机制**：当所有人都在卖的时候，优质资产的需求韧性更强，因为：
- 刚需（学区）不受价格影响
- 改善型买家更看重品质
- 高收入群体更能承受价格波动

### 三、相对亏的反面：相对赚

> 「自己的失败固然可恨，但兄弟的成功更让人揪心」

#### 心理机制

| 观察角度 | 心理感受 |
|---------|---------|
| 只看自己 | 痛苦（亏了200万） |
| 环顾四周 | 慰藉（别人亏更多） |
| 看别人亏400万 | 道心稳定 |
| 看别人亏更多 | 爽（空中飞人） |

**社交媒体放大效应**：
- 看到别人亏更多 → 心理慰藉
- 看到别人"抄底成功" → 焦虑

#### 相对赚的用途

1. **心理慰藉**：道心稳定，继续持有
2. **信心支撑**：「我的选择是对的」
3. **不卖的理由**：「反正比邻居强」

### 四、这个理论的现实意义

#### 1. 心理层面

在房产大幅下行时，持有优质资产可以维持「道心稳定」：

> 「我在大A亏了这么多，评论区不也都开香槟庆祝吗，所以啊，我们还是需要时不时往下比一比，看看那些更惨的来调整心态」

**方法**：
- 看看同小区更低成交价
- 看看周边小区的跌幅
- 看看那些"腰斩"的故事

#### 2. 资产配置层面

尽量选择区域内相对抗跌的小区：

| 选择策略 | 逻辑 | 风险 |
|---------|------|------|
| 选最抗跌的 | 相对亏概率低 | 流动性可能差 |
| 选性价比高的 | 跌过头+抗跌 | 判断难度大 |
| 分散配置 | 不把鸡蛋放一个篮子 | 管理复杂度高 |

#### 3. 操作层面

> 「早出手就是赚」——如果不卖，任何「相对赚」都是纸面富贵

**悖论**：
- 相对赚 = 没卖才能体现
- 卖了 = 拿到现金，但失去"相对抗跌"的护城河
- 不卖 = 纸面富贵，但可能继续跌

**巫师的答案**：「早出手就是赚」——**时间维度优先于相对表现**

### 五、两种亏的决策矩阵

| 情况 | 策略 | 理由 |
|------|------|------|
| 绝对亏 + 相对赚 | 持有 | 比别人强，心理慰藉 |
| 绝对亏 + 相对亏 | 考虑止损 | 双倍痛苦 |
| 绝对赚 + 相对赚 | 持有/部分兑现 | 双重胜利 |
| 绝对赚 + 相对亏 | 检视持仓 | 跑输大市 |

### 六、绝对亏与相对亏的数学边界

#### 极限情况

**极端相对赚**：
```
你的房：1000万 → 900万（-10%）
同类：1000万 → 100万（-90%）
相对赚 = 800万
→ 你几乎白送了一套房（相对角度）
```

**极端相对亏**：
```
你的房：1000万 → 100万（-90%）
同类：1000万 → 900万（-10%）
相对亏 = 800万
→ 你的"优质资产"变成了最差选择
```

**教训**：优质资产也可能变成贫民窟（[[贫民窟]]），关键是**质量下滑速度**。

### 七、与分摊痛苦的关系

[[分摊痛苦]] 描述的是**宏观层面的痛苦分摊**（ABCD阶梯式接盘）。

绝对亏与相对亏描述的是**微观层面的损失比较**（你和邻居比谁亏更少）。

| 维度 | 分摊痛苦 | 绝对亏/相对亏 |
|------|---------|--------------|
| 层面 | 宏观（整个社会） | 微观（个人） |
| 比较对象 | 时间轴上的多个人 | 空间上的邻居 |
| 核心问题 | 谁能承担损失 | 谁亏得更少 |
