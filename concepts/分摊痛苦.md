# 分摊痛苦

> 来源：巫师财经《吃透中国房价的一切，但是主神视角》

## 定义

**分摊痛苦** = 在房价下行趋势中，通过设计阶梯式传导链，让多个买家（A→B→C→D）分段承接价格下跌的痛苦，而不是让一个人（最初的买家）承担全部损失。

> 「以前是小A一个人承担了一切」

## 核心命题

### 一、为什么要分摊痛苦？

#### 硬着陆下小A的悲惨结局

```
初始状态：小A首付300万 + 贷款700万 = 1000万买房

房价暴跌路径：
1000万 → 700万 → 500万 → 300万

小A的损失：
1. 首付300万：没了
2. 房子没了（被拍卖得300万）
3. 还欠银行400万（700万贷款 - 300万拍卖款）
4. 后半生打工还债

小A的心态：
「我首付300万就能直接买一套新房了，心态已经崩了」
→ 断贷
→ 成为「社会不安定因素」
→ 希望世界毁灭，一切推倒重来
```

#### 千千万万个「小A」的后果

当大量小A同时崩溃：

| 层级 | 后果 |
|------|------|
| 个人 | 焦虑、抑郁、反社会倾向 |
| 家庭 | 婚姻破裂、家庭矛盾 |
| 社会 | 治安恶化、犯罪率上升 |
| 经济 | 消费萎缩、企业裁员、进一步通缩 |
| 金融 | 银行坏账飙升、系统性风险 |
| 政治 | 社会动荡、政权稳定性受挑战 |

> 「这种让大家贡献自己的【锚】方式，就算玩不转了，一旦这个模式停止了，不但没有新增的【锚】，还会让大家破坏以前的锚建立起的成果」

### 二、分摊痛苦的操作方式

#### 阶梯式传导：ABCD模型

```
小A（1000万买）→ 小B（800万接）→ 小C（600万接）→ 小D（400万接）
   亏200万          亏200万          亏200万           亏200万
   继续工作         认赌服输         教育投资         无奈持有
```

| 角色 | 处境 | 心态 | 行为 |
|------|------|------|------|
| 小A | 亏损200万，看走眼 | 「认了，继续努力赚钱」 | 努力工作还债 |
| 小B | 接盘后继续跌 | 「认赌服输，挂了」 | 降价出售 |
| 小C | 为孩子买学区 | 「亏200万就当教育投资了」 | 继续持有 |
| 小D | 跌到心理底线 | 「罢了，继续熬」 | 持有等待 |

**关键**：
- 每个人都只承担了**200万的损失**（而不是1000万或700万）
- 每个人都**心理上接受了**，没有崩溃
- 每个人都**继续努力工作**（贡献锚）

#### 分摊痛苦的三重效果

1. **消除社会不安定因素**：没人崩溃到要推翻一切
2. **维持「锚」的供给**：大家继续贡献劳动时间
3. **让经济系统得以继续运行**：游戏继续

### 三、分摊痛苦的其他形式

| 类型 | 传导方式 | 时间维度 | 空间维度 |
|------|---------|---------|---------|
| **房价分摊** | A→B→C→D阶梯接盘 | 代际分散 | 跨买家分散 |
| **保交楼** | 国家兜底交付 | 当期 | 跨纳税人 |
| **债务重组** | 短债拉长+债转股 | 跨期 | 跨债权人 |
| **化债方案** | 高息→低息置换 | 展期 | 跨地方中央 |

#### 保交楼：分摊痛苦在民生维度的体现

> 「保交楼的背后其实是救房企，你得让他们给你干活」

机制：
- 买了房的人至少能住进去 → 不会成为不安定因素
- 房企能继续运营 → 维持就业
- 银行坏账可控 → 金融稳定

#### 债务重组：分摊痛苦在企业维度的体现

> 「债权人一看，我靠，你欠我本金加利息一共1个亿，现在东砍砍西砍砍，最后变成一堆新的债券和股票，只剩3000万了」

机制：
- 债务人：获得喘息机会，继续经营
- 债权人：接受损失（但比清盘好）
- 时间：把损失分摊到未来

### 四、债务不会消失，只会转移

> 「化的谁的债，把债化给了谁？」

#### 债务转移的三条路径

| 路径 | 传导机制 | 承担者 |
|------|---------|-------|
| 银行承担 | 贷款展期+利息减免 | 银行股东/储户 |
| 国家承担 | 财政兜底+专项债 | 纳税人 |
| 社会承担 | 通胀稀释+货币化 | 全体货币持有人 |

#### 债务重组的数学

```
原始债务：1个亿
第一次重组：1亿 → 3000万（削债70%）
二次重组可能：3000万 → 900万（再削70%）

债权人选择：
A. 不同意 → 债务人清盘 → 回收率0.5%-3%（恒大案例）
B. 同意 → 回收率30%（假设）
```

**理性选择 = 接受重组**

### 五、分摊痛苦的成本与代价

#### 隐性成本

1. **道德风险**：房企知道有人兜底 → 过度冒险
2. **效率损失**：资源继续配置到低效的房地产
3. **代际不公**：未来纳税人承担过去的问题
4. **改革延迟**：用时间换空间 → 核心问题延后

#### 为什么要接受这些成本？

> 「成年人世界的选择，往往是两害相权取其轻」

**不这么做**的后果（硬着陆）：
- 银行系统崩溃
- 经济长期通缩
- 社会动荡
- 可能触发政治危机

**这么做**的后果（软着陆）：
- 纳税人承担部分损失
- 道德风险积累
- 改革窗口期后移
- 但：社会基本稳定，游戏继续

**两害相权取其轻** = **稳字当头**

### 六、分摊痛苦的心理机制

#### 为什么分摊就能接受？

| 单人承担（硬着陆） | 多人分摊（软着陆） |
|------------------|------------------|
| 损失绝对值巨大 | 损失绝对值较小 |
| 被迫承认失败 | "大家都一样" |
| 无力感 | 可接受感 |
| 崩溃 | 内化 |
| 反社会 | 继续工作 |

> 「四个人都心理上接受了，不但没有产生反社会倾向，而且依旧努力工作」

**心理关键**：不是损失大小的问题，而是**损失归因**的问题。
- 自己一个人扛 = 无力感 → 崩溃
- 大家一起来 = "不是我一个人的问题" → 内化 → 继续工作
