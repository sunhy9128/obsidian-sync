# 联系汇率制度

> 香港自 1983 年起实行的货币发行局制度 —— 港币与美元硬挂钩，1 美元 = 7.75-7.85 港元

## 一、定义

**联系汇率制度**（Linked Exchange Rate System, LERS）是香港自 **1983 年 10 月 17 日**起实行的货币制度，其核心是**港币与美元维持固定汇率**（最初 1 美元 = 7.80 港元，2005 年优化为 7.75-7.85 区间）。

它是**货币发行局制度**（Currency Board Arrangement, CBA）的一种特殊形式，也是全球少数仍在运行的高度僵化的汇率制度之一。

---

## 二、历史背景

### 2.1 港币汇率制度的演变

```
1863-1935：银本位制
     ↓
1935-1972：英镑挂钩（1 英镑 = 16 港元）
     ↓
1972-1974：港币与美元挂钩（1 美元 = 5.65 港元）
     ↓
1974-1983：自由浮动（港币贬值）
     ↓
1983 至今：联系汇率制度（1 美元 = 7.80 → 7.75-7.85）
```

### 2.2 1983 年转向联系汇率的原因

| 事件 | 影响 |
|------|------|
| 1982 年中英谈判 | 香港前途未明，港人信心动摇 |
| 1983 年 9 月 | 港币兑美元跌至 9.60 历史低点 |
| 1983 年 10 月 | 港府宣布联系汇率制度，信心恢复 |

> **关键时刻**：1983 年港币危机促使港府选择"放弃货币政策自主"换取"汇率稳定"，这是香港版的"三元悖论"选择。

---

## 三、运作机制

### 3.1 货币发行局制度的核心规则

| 规则 | 含义 |
|------|------|
| **100% 美元储备** | 港币基础货币必须有 100% 美元储备支持 |
| **固定汇率** | 港币兑美元在 7.75-7.85 区间内 |
| **套利机制** | 银行通过"发钞行"机制维持汇率 |
| **金管局干预** | 必要时买卖美元干预市场 |

### 3.2 港币发行机制（详细）

```
商业银行需要港币现钞
       ↓
向发钞行（汇丰、渣打、中银香港）申请
       ↓
发钞行必须以 1:7.80 汇率向金管局缴存美元
       ↓
金管局发给等值港币（负债证明书）
       ↓
商业银行获得港币现钞
       ↓
公众使用港币
```

### 3.3 套利机制：自动稳定器

```
情景 1：港币兑美元升至 7.75（强方）
  ↓
  套利者用港币换美元（向发钞行缴回港币、提取美元）
  ↓
  港币供应 ↓
  ↓
  港币利率 ↑
  ↓
  资金流入 ↓
  ↓
  港币重新走弱至 7.75-7.85

情景 2：港币兑美元贬至 7.85（弱方）
  ↓
  套利者用美元换港币（向发钞行缴存美元、提取港币）
  ↓
  港币供应 ↑
  ↓
  港币利率 ↓
  ↓
  资金流入 ↑
  ↓
  港币重新走强至 7.75-7.85
```

## 四、2005 年优化：强方/弱方兑换保证

### 4.1 2005 年 5 月的"三项优化"

| 优化项 | 目的 |
|--------|------|
| **强方兑换保证** | 1 美元 = 7.75 港元（金管局买美元）|
| **弱方兑换保证** | 1 美元 = 7.85 港元（金管局卖美元）|
| **弱方区间** | 7.75-7.85 之间由市场决定 |

### 4.2 优化后的运作

```
港币兑美元汇率
  7.75 (强方) ──────────────── 金管局在此卖出港币、买入美元
       ↓
     自由浮动区间
       ↓
  7.85 (弱方) ──────────────── 金管局在此卖出美元、买入港币
```

## 五、联系汇率的"三元悖论"意义

### 5.1 三元悖论回顾

```
        独立的货币政策
              △
             / \
            /   \
           /     \
          /   X   \
         /    ↑    \
        /  只能选  \
       /    两项    \
      /              \
  固定汇率 ─── 资本自由流动
```

### 5.2 香港的选择

| 维度 | 香港的选择 |
|------|----------|
| 汇率稳定 | ✅ 选择 |
| 货币政策自主 | ❌ 放弃（跟随美联储）|
| 资本自由流动 | ✅ 选择 |

> **关键洞察**：香港选择了"汇率稳定 + 资本开放"，放弃了"货币政策自主"。这意味着香港利率基本跟随美联储，无法通过利率调节经济。

### 5.3 三元悖论下的香港应对

| 政策维度 | 香港的实际操作 |
|---------|---------------|
| **货币政策** | 跟随美联储，无自主权 |
| **汇率政策** | 联系汇率制度（硬挂钩）|
| **资本管制** | 无（资本自由流动）|
| **宏观审慎** | 通过逆周期资本缓冲、压力测试等"非利率工具"调节 |

## 六、联系汇率制度下的金管局工具箱

### 6.1 货币政策工具

| 工具 | 说明 |
|------|------|
| **贴现窗利率** | 金管局向银行提供流动性的利率 |
| **外汇基金票据/债券** | 调节银行体系港元流动性 |
| **流动性调节窗口** | 2005 年推出，让银行可以更灵活调节流动性 |
| **金管局流动性工具** | 紧急流动性支持 |

### 6.2 汇率干预工具

| 工具 | 说明 |
|------|------|
| **强方/弱方兑换保证** | 2005 年引入 |
| **直接外汇干预** | 在 7.75-7.85 区间外操作 |
| **公开市场操作** | 调节港币流动性 |

### 6.3 宏观审慎工具

| 工具 | 说明 |
|------|------|
| **逆周期资本缓冲** | 调节银行资本要求 |
| **贷款与估值比率（LTV）** | 限制房贷 |
| **压力测试** | 确保银行稳健 |

## 七、1998 金融保卫战与联系汇率

### 7.1 国际空头的"双狙"为何可怕？

国际对冲基金（索罗斯）看到了香港联系汇率制度的脆弱性：

```
做空港股：
  ↓
恒指下跌 → 投资者信心下降
  ↓
  ↓
做空港币（抛空港币换美元）：
  ↓
  港币贬值压力 → 联系汇率受到威胁
  ↓
  1997 年：泰国、韩国、印尼都是因为汇率崩盘引发危机
  ↓
  如果港币失守：联系汇率制度崩溃（不可逆）
```

### 7.2 救市为什么必须用外汇基金？

> **关键约束**：在联系汇率制度下，金管局**不能凭空印港币**。所有救市资金必须来自**已积累的外汇储备**。

```
1998 救市资金流：
  卖美元资产 100 亿（外汇储备）  ← 已积累的公共储蓄
       ↓
  获得港元 100 亿
       ↓
  买入港股 100 亿
       ↓
  ✅ 基础货币不变
  ✅ 资产从"美元"换成"港股"
  ❌ 没有凭空创造货币
```

### 7.3 救市的成功原因

| 因素 | 说明 |
|------|------|
| 外汇储备充足 | 香港外汇基金规模大，足够救市 |
| 行动果断 | 1998.8.14 第一时间入市 |
| 多线作战 | 现货 + 期货同时操作 |
| 时间点精准 | 8 月 28 日期货结算日决战 |
| 国际支持 | 中国中央政府表态支持 |

## 八、联系汇率制度 vs 其他汇率制度

| 制度 | 汇率波动 | 货币政策自主 | 香港是否采用 |
|------|---------|------------|------------|
| **联系汇率** | 几乎为零 | 无 | ✅ 当前 |
| **固定汇率** | 极小 | 受限 | ❌ |
| **管理浮动** | 中等 | 部分 | ❌ |
| **自由浮动** | 大 | 充分 | ❌ |
| **货币局制度** | 几乎为零 | 无 | ✅（香港是其中一种）|

## 九、联系汇率制度的优缺点

### 9.1 优点

| 优点 | 说明 |
|------|------|
| 汇率稳定 | 消除汇率风险，便于贸易和投资 |
| 通胀控制 | 通过联系美元，间接锚定美联储的通胀纪律 |
| 信心锚定 | 1983 年成功锚定港人信心 |
| 制度简单 | 套利机制自动维持汇率 |
| 国际形象 | 提升香港国际金融中心地位 |

### 9.2 缺点

| 缺点 | 说明 |
|------|------|
| 货币政策无自主 | 必须跟随美联储 |
| 易受投机攻击 | 国际空头专门狙击联系汇率 |
| 经济周期不同步 | 香港与美国的商业周期可能错位 |
| 房价波动 | 美国低利率时期，香港房价飙涨 |
| 通缩输入 | 美国宽松时香港流动性被动宽松 |

### 9.3 争议与挑战

| 争议 | 焦点 |
|------|------|
| **是否应脱钩** | 长期看是否应改为浮动汇率 |
| **货币发行局** | 是否引入人民币资产作为储备 |
| **数字港元** | CBDC 对联系汇率的影响 |
| **国际化挑战** | 人民币国际化对港币的冲击 |

## 十一、未来展望

### 11.1 短期（2025-2030）

- 数字港元（e-HKD）试点可能推出
- 联系汇率制度将保持稳定
- 与人民币的"沪港通""深港通""债券通"继续扩展

### 11.2 中长期（2030+）

- **人民币国际化深化**对港币的潜在冲击
- **数字货币**可能重新定义货币发行局制度
- **中美关系演变**可能影响联系汇率的稳定性
- **香港国际金融中心地位**的巩固与挑战

---

## 十二、一句话总结

> **联系汇率制度是香港版的"三元悖论"选择**：放弃货币政策自主（跟随美联储），换取汇率稳定（7.75-7.85）+ 资本自由流动。这一制度使 1998 救市**只能在 L1 层级**操作（用外汇基金，不能印港币），与日本/美联储/瑞士央行的"印钞买股"形成鲜明对比。
>
> **联系汇率制度 = 香港的"独特制度基因"**，决定了香港金融政策的本质特征。
