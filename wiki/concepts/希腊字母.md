# 希腊字母（Option Greeks）

> 期权风险管理的"五维语言"——Delta/Gamma/Vega/Theta/Rho 共同构成期权世界的"度量衡"

## 一、定义

**希腊字母**（Greeks）是用于**衡量期权价格对各种风险因素敏感度**的**一组数学量**。它们以**希腊字母命名**（Delta、Gamma、Vega、Theta、Rho），是**期权定价、风险管理、动态对冲**的核心工具。

核心希腊字母：

| 字母 | 含义 | 风险因素 |
|------|------|---------|
| **Delta (Δ)** | 价格敏感度 | 标的价格 |
| **Gamma (Γ)** | Delta 变化率 | 标的价格的二阶 |
| **Vega (ν)** | 波动率敏感度 | 隐含波动率 |
| **Theta (Θ)** | 时间衰减 | 时间 |
| **Rho (ρ)** | 利率敏感度 | 无风险利率 |

> **核心比喻**：希腊字母是"**期权风险的"五维语言**"——每一个希腊字母描述期权对**一个风险维度**的敏感度。专业期权交易员用希腊字母"说话"。

## 二、Delta（Δ）—— 期权的"方向感"

### 2.1 定义

```
Delta = ∂V/∂S

含义：期权价格 V 对标的价格 S 的一阶偏导
即：标的价格变动 1 元 → 期权价格变动多少元
```

### 2.2 性质

```
Call 期权 Delta：
  - 范围：0 到 +1
  - 深度虚值：Delta ≈ 0
  - 平值：Delta ≈ 0.5
  - 深度实值：Delta ≈ +1

Put 期权 Delta：
  - 范围：-1 到 0
  - 深度虚值：Delta ≈ 0
  - 平值：Delta ≈ -0.5
  - 深度实值：Delta ≈ -1
```

### 2.3 实际应用

```
例：沪深 300 股指期权
  - 现货 4000 点
  - 行权价 4000 点（平值）
  - 距到期 30 天
  
  Call Delta = 0.5
  → 沪深 300 涨 10 点
  → Call 涨 5 点
       ↓
应用：
  1. 对冲比率：需要 0.5 手期货对冲 1 手期权
  2. 方向风险：Delta 衡量"方向敞口"
  3. 中性对冲：调整头寸使净 Delta = 0
```

详见 [[wiki/concepts/动态对冲|动态对冲]]。

## 三、Gamma（Γ）—— Delta 的"变化速度"

### 3.1 定义

```
Gamma = ∂Δ/∂S = ∂²V/∂S²

含义：Delta 对标的价格的二阶偏导
即：标的价格变动 1 元 → Delta 变动多少
```

### 3.2 性质

```
Gamma 特征：
  - 平值期权 Gamma 最大
  - 深度虚值/实值期权 Gamma 较小
  - 临到期平值期权 Gamma → 无穷大
  - Call 和 Put 的 Gamma 相同（同标的同行权价）
       ↓
Gamma 风险：
  - Delta 变化太快
  - 对冲成本增加
  - 期权做市商最大风险
```

### 3.3 实际意义

```
例：平值期权临到期
  Gamma = 0.05（很大）
  → 标的价格变动 1 元
  → Delta 变动 0.05
  → 需要多卖出 0.05 手期货
  → 频繁调整 → 高交易成本
       ↓
结论：
  Gamma 越大 → 越难对冲
  临到期平值期权 = "对冲噩梦"
```

### 3.4 Gamma Scalping

```
做市商常用策略：
  持有期权头寸（赚 Theta）
  持续调整 Delta（动态对冲）
  Gamma 变动带来额外收益
       ↓
当 Gamma 为正时：
  标的价格波动 = 做市商盈利
  因为：高抛低吸
       ↓
Gamma Scalping：
  - 赚取波动率的钱
  - 用 Theta 收入对冲调整成本
```

## 四、Vega（ν）—— 波动率敏感度

### 4.1 定义

```
Vega = ∂V/∂σ

含义：期权价格 V 对波动率 σ 的一阶偏导
即：波动率变动 1% → 期权价格变动多少
```

### 4.2 性质

```
Vega 特征：
  - 所有期权 Vega 都是正（无论是 Call 还是 Put）
  - 平值期权 Vega 最大
  - 临到期期权 Vega 趋近 0
  - 长期期权 Vega 较大
       ↓
Vega 风险：
  - 隐含波动率变化
  - 即使标的价格不变，期权价值也变化
  - 跨期权品种套利的核心
```

### 4.3 实际应用

```
例：沪深 300 股指期权
  - 隐含波动率 = 20%
  - Vega = 5 元
  → 隐含波动率上升 1%
  → 期权价格上涨 5 元
       ↓
应用：
  1. 波动率交易：单独交易 Vega
  2. 跨品种套利：Vega 不同的期权组合
  3. 风险管理：Vega 中性对冲
```

### 4.4 VIX 与 Vega 交易

```
VIX（芝加哥期权交易所波动率指数）：
  - 反映标普 500 未来 30 天的预期波动率
  - 被称为"恐慌指数"
  - VIX 高 = 市场恐慌
  - VIX 低 = 市场平静
       ↓
Vega 交易者：
  - 关注 VIX 变化
  - 通过期权组合交易波动率
  - "做多波动率"或"做空波动率"
```

## 五、Theta（Θ）—— 时间衰减

### 5.1 定义

```
Theta = ∂V/∂T

含义：期权价格 V 对时间 T 的一阶偏导
即：时间流逝 1 天 → 期权价格变动多少
```

### 5.2 性质

```
Theta 特征：
  - 几乎所有期权（买方）Theta 都是负
  - 临到期 Theta 衰减加速
  - 平值期权 Theta 衰减最快
  - 卖方 Theta 是正（赚时间价值）
       ↓
Theta 风险：
  - 时间是买方的"敌人"
  - 时间是卖方的"朋友"
  - 长期持有期权的"磨损"
```

### 5.3 实际应用

```
例：期权离到期还有 30 天
  Theta = -0.5 元
  → 每天期权价格自动下降 0.5 元
  → 即使标的价格不动
  → 期权也每天贬值
       ↓
应用：
  1. 期权买方：尽量短期持有
  2. 期权卖方：长期持有赚时间
  3. 临到期：平值期权 Theta 加速衰减
```

### 5.4 时间价值的"二八效应"

```
期权的时间价值：
  - 后 30 天：30% 时间价值
  - 最后 30 天：70% 时间价值衰减
  - 最后 7 天：50% 时间价值衰减
       ↓
这就是"期权的二八效应"：
  时间集中在最后阶段衰减
  临到期 = "时间集中爆发"
```

## 六、Rho（ρ）—— 利率敏感度

### 6.1 定义

```
Rho = ∂V/∂r

含义：期权价格 V 对无风险利率 r 的一阶偏导
即：利率变动 1% → 期权价格变动多少
```

### 6.2 性质

```
Rho 特征：
  - Call Rho > 0（利率上升 → Call 升值）
  - Put Rho < 0（利率上升 → Put 贬值）
  - 长期期权 Rho 较大
  - 短期期权 Rho 较小
       ↓
Rho 风险：
  - 利率大幅变动时
  - 长期期权对冲时
  - 一般被视为"次要"风险
```

### 6.3 实际应用

```
例：长期 Call 期权
  Rho = 0.1
  → 利率上升 1%
  → 期权价格上涨 0.1
       ↓
应用：
  1. 长期期权对冲
  2. 利率大幅变动场景
  3. 通常配合其他 Greeks 管理
```

## 七、希腊字母的"协同关系"

### 7.1 五个希腊字母的"分工"

```
5 个风险维度 + 5 个希腊字母：

       时间 → Theta
        ↑
        |
波动率 → Vega → 期权价格
        |       ↑
        ↓       |
   标的价格 → Delta → 期权价格
        |
        ↓
      Gamma（Delta 的"加速度"）
        |
        ↓
      利率 → Rho
```

### 7.2 综合风险管理

```
完整的期权风险管理：
  Delta 中性 → 消除方向风险
  Gamma 中性 → 消除 Delta 变化风险
  Vega 中性 → 消除波动率风险
  Theta 管理 → 接受时间衰减
  Rho 中性 → 消除利率风险（如果需要）
       ↓
"全 Greeks 中性"组合：
  - 几乎所有风险都被对冲
  - 只承担"时间价值"风险
  - 这是做市商的"圣杯"
```

## 八、希腊字母在 1998 香港案例中的影子

### 8.1 金管局操作的"Greeks 视角"

```
金管局的"立体对冲"：
  现货多头 + 期货多头 = 双向多头
  
Delta 视角：
  - 现货 Delta = +1
  - 期货 Delta = +1
  - 总 Delta = +2（不是中性）
  - 金管局主动承担方向风险
       ↓
Gamma 视角：
  - 期货 Gamma 较小
  - 但金管局不需要 Gamma 中性
  - 因为它追求"方向攻击"
       ↓
Vega 视角：
  - 没有期权头寸
  - 隐含波动率变化影响较小
  - 主要是标的方向变化
       ↓
Theta 视角：
  - 期货 Theta 接近 0
  - 没有时间衰减问题
       ↓
结论：
  金管局的"对冲"不是 Delta 中性对冲
  而是"主动方向攻击"
  但仍利用了期货的"双向交易"特性
```

## 九、希腊字母的"金融学意义"

### 9.1 期权风险的"模块化"

```
传统风险管理：
  - 风险是一个整体
  - 无法拆分
  - 难以精确对冲

期权时代：
  - 风险被拆分为多个维度
  - 每个维度有对应的 Greek
  - 可以独立管理
       ↓
意义：
  - 精细化风险管理
  - 风险管理产业化
  - 风险成为可交易的"商品"
```

### 9.2 希腊字母的"信息价值"

```
希腊字母告诉市场：
  - 当前市场情绪（Delta 分布）
  - 未来波动率预期（Vega 暴露）
  - 时间衰减速度（Theta 分布）
  - 利率预期（Rho 暴露）
       ↓
希腊字母是"市场的多维信息"
专业交易员通过希腊字母
"听懂"市场在说什么
```

## 十、希腊字母在"动态对冲"中的角色

### 10.1 动态对冲的"五步调整"

```
对冲者的工作流程：

步骤 1：每日计算 Greeks
  - 当前头寸的 Delta/Gamma/Vega/Theta
  - 与目标 Greeks 比较

步骤 2：识别偏差
  - Delta 偏离目标？→ 调整期货
  - Gamma 偏离目标？→ 调整期权
  - Vega 偏离目标？→ 调整期权行权价

步骤 3：执行调整
  - 买卖期货
  - 买卖期权
  - 注意交易成本

步骤 4：重新计算
  - 调整后重新计算 Greeks
  - 确认达到目标

步骤 5：监控
  - 持续监控 Greeks
  - 必要时再次调整
```

### 10.2 动态对冲的"成本"

```
动态对冲的成本来源：

1. Gamma 调整成本
   - 频繁调整的交易成本
   - 被称为"Gamma 税"

2. Vega 调整成本
   - 跨期权品种的价差
   - 流动性差异

3. Theta 损失
   - 时间价值衰减
   - 持续的成本

4. Bid-Ask 价差
   - 买入价差 + 卖出价差
   - 频繁交易的累计成本
       ↓
动态对冲的"年化成本"：
  - 简单：1-3%
  - 复杂：5-15%
```

详见 [[wiki/concepts/动态对冲|动态对冲]]。

## 十一、希腊字母与"经典案例"

### 11.1 LTCM 的"希腊字母灾难"

```
LTCM 的风险：
  - 大量做"收敛交易"
  - 动态对冲管理 Greeks
  - 模型假设：利差会收敛
       ↓
1998 年俄罗斯违约：
  - 隐含波动率飙升
  - 价差不收敛反而扩大
  - 模型失效
  - 动态对冲失效
  - 巨亏
       ↓
教训：
  希腊字母在极端市场可能"失真"
  模型风险 = 希腊字母计算的"隐藏风险"
```

详见 [[wiki/concepts/长期资本管理公司]]。

### 11.2 1987 黑色星期一的"Gamma 灾难"

```
1987 年组合保险策略：
  - 股价下跌 → 卖出股票
  - 股价上涨 → 买入股票
  - 持续调整
       ↓
问题：
  - 大规模组合保险同时触发
  - 大量卖出 → 股价大跌
  - 更多组合保险触发
  - Gamma 调整失效
  - "死亡螺旋"
       ↓
教训：
  动态对冲在"群体行为"下失效
  Gamma 调整可能加剧市场波动
```

详见 [[wiki/concepts/1987黑色星期一]]。

## 十二、希腊字母的"实际应用"

### 12.1 期权买方关注

```
买方应关注：
  1. Delta：方向敞口
  2. Gamma：Delta 变化
  3. Theta：时间成本
  4. Vega：波动率风险

买方的对冲：
  - 短期：主要关注 Delta
  - 中期：Delta + Theta
  - 长期：Delta + Vega
```

### 12.2 期权卖方关注

```
卖方应关注：
  1. Delta：方向风险（最大）
  2. Gamma：Delta 变化（巨大）
  3. Theta：时间收益
  4. Vega：波动率风险

卖方的对冲：
  - 必须 Delta 中性
  - 必须管理 Gamma
  - 赚取 Theta
  - 对冲 Vega
```

### 12.3 投资组合 Greeks

```
组合 Greeks：
  总 Delta = Σ 单个头寸 Delta × 数量
  总 Gamma = Σ 单个头寸 Gamma × 数量
  总 Vega = Σ 单个头寸 Vega × 数量
  总 Theta = Σ 单个头寸 Theta × 数量
       ↓
对冲目标：
  - 总 Delta = 0（消除方向风险）
  - 总 Gamma = 0（消除 Delta 变化）
  - 总 Vega = 0（消除波动率风险）
  - 接受总 Theta（赚时间价值）
```

## 十三、希腊字母的"哲学高度"

### 13.1 希腊字母与"风险管理哲学"

```
希腊字母代表的风险管理思想：
  1. **风险的"模块化"**
     - 把整体风险拆分为多个维度
     - 每个维度独立管理
     
  2. **风险的"可交易性"**
     - 风险不仅是"风险"
     - 风险是"可交易"的资产类别
     - Vega 本身可以单独交易
     
  3. **风险的"动态性"**
     - 风险持续变化
     - 对冲必须持续调整
     - 没有"一劳永逸"
```

### 13.2 希腊字母的"信息论"

```
希腊字母是"期权世界的度量衡"：
  - 期权 = 风险维度 × 敏感度
  - 5 个维度 = 5 个希腊字母
  - 完整描述期权的"风险画像"
       ↓
哲学意义：
  - 复杂的事物可以被"维度化"
  - 每个维度可以被"测量"
  - 通过测量 → 理解 → 管理
```

## 十四、希腊字母的"实务技巧"

### 14.1 风险监控仪表盘

```
专业期权交易员的"仪表盘"：

  ┌──────────────────────────────────┐
  │  Delta: +125.5   [▲ +5.2]       │
  │  Gamma: -15.2    [▼ -2.1]       │
  │  Vega:  +230.0   [▲ +12.5]      │
  │  Theta: -85.0    [▼ -3.0]       │
  │  Rho:   +12.5    [▲ +0.5]       │
  │                                  │
  │  净头寸: 多头 30% 净敞口        │
  │  现金:  ¥2,500,000             │
  │  风险等级: ⚠️ 中等             │
  └──────────────────────────────────┘
```

### 14.2 风险预警线

```
常见的风险预警线：

Delta 预警：
  - 净 Delta > 100 → 调整
  - 净 Delta > 200 → 立即调整
  - 净 Delta > 500 → 强制减仓

Gamma 预警：
  - 净 Gamma > 50 → 警惕
  - 净 Gamma > 100 → 立即对冲

Vega 预警：
  - 净 Vega > 1000 → 警惕
  - 净 Vega > 2000 → 立即对冲
```

## 十五、一句话总结

> **希腊字母 = 期权风险的五维语言（Delta/Gamma/Vega/Theta/Rho）**。
>
> 它把期权的**复杂风险**拆分为**5 个独立维度**，让风险管理变得**精细化、模块化、可交易**。每个希腊字母描述期权对**一个风险因素**的敏感度：**Delta 描述方向，Gamma 描述 Delta 的变化，Vega 描述波动率，Theta 描述时间，Rho 描述利率**。
>
> **LTCM 崩溃**和**1987 黑色星期一**告诉我们：**希腊字母依赖模型和市场连续性，在极端市场可能失真**。**好的风险管理不是"零风险"，而是"用五维语言把风险讲清楚，然后逐个管理"**。
