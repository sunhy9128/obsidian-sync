> 消费者剩余（CS）是消费者愿意支付与实际支付的差额；生产者剩余（PS）是实际收入与最低意愿收入的差额；两者之和构成总剩余（社会福利），是衡量市场效率的核心指标。

---

## 一、基本定义

### 1.1 消费者剩余（Consumer Surplus, CS）

```
CS = 消费者愿意支付的总金额 - 实际支付的总金额
   = ∫[0, Q*] (Pd(Q) - P*) dQ
   = 需求曲线以下、均衡价格以上的面积
```

| 概念 | 含义 |
|------|------|
| 意愿价格 | 消费者对每一单位商品愿意支付的最高价格 |
| 市场价格 | 实际交易中形成的均衡价格 |
| CS 来源 | 消费者对每一单位商品估值都高于实际支付价 |

### 1.2 生产者剩余（Producer Surplus, PS）

```
PS = 实际收入 - 生产者愿意接受的最低总收入
   = ∫[0, Q*] (P* - Ps(Q)) dQ
   = 供给曲线以上、均衡价格以下的面积
```

| 概念 | 含义 |
|------|------|
| 最低意愿价 | 生产者对每一单位商品愿意接受的最低价格 |
| 市场价格 | 实际交易中形成的均衡价格 |
| PS 来源 | 实际售价高于每一单位成本的部分 |

### 1.3 总剩余（Total Surplus, TS）

```
TS = CS + PS
   = 需求曲线以下、供给曲线以上的总面积
   = 衡量市场配置资源的总福利
```

## 二、图形说明

### 2.1 几何表示

```
价格 P
  │
Pd\│
  │\  ★  CS
  │ \  ███  (消费者剩余)
  │  \  ███
P*│───\──███─ Ps
  │   █\ ███
  │   █ \  ███
  │ PS\    ███
  │   █ █\  ███
  │   █   █  ███
  │   █   █    ███
  └─────────────── 数量 Q
              Q*
```

| 区域 | 含义 |
|------|------|
| 需求曲线左上 + 均衡价上 | 消费者剩余 CS |
| 供给曲线右上 + 均衡价下 | 生产者剩余 PS |
| 两线之间全部 | 总剩余 TS |

### 2.2 关键性质

- CS 总在需求曲线下方
- PS 总在供给曲线下方
- 均衡点上 TS 最大化（帕累托最优）
- 任何偏离均衡的交易都减少 TS

## 三、数值计算示例

### 3.1 离散情形

```
市场上 5 个消费者，对某商品最高愿付价格（元）：

消费者  A  B  C  D  E
愿付价 100 80 60 40 20

市场上 5 个生产者，最低接受价格（元）：

生产者  V  W  X  Y  Z
成本价  20 40 60 80 100

均衡时 P* = 60，Q* = 3
```

| 项目 | 计算 | 金额 |
|------|------|------|
| CS | (100-60) + (80-60) + (60-60) | 60 元 |
| PS | (60-20) + (60-40) + (60-60) | 60 元 |
| TS | 60 + 60 | 120 元 |

### 3.2 连续函数情形

```
需求函数：Q^d = 100 - 2P，即 P = 50 - 0.5Q
供给函数：Q^s = 20 + 3P，即 P = (Q-20)/3

均衡：
  50 - 0.5Q = (Q-20)/3
  150 - 1.5Q = Q - 20
  170 = 2.5Q
  Q* = 68
  P* = 16
```

| 指标 | 计算 | 数值 |
|------|------|------|
| CS | 0.5 × (50 - 16) × 68 | 1156 元 |
| PS | 0.5 × (16 - (-20/3)) × 68 ≈ 0.5 × 22.67 × 68 | 770.67 元 |
| TS | CS + PS | 1926.67 元 |

## 四、福利变化分析

### 4.1 价格管制的影响

#### 情形 1：价格上限（P_cap < P*）

```
结果：Q 实际交易量 < Q*（因为 P↓，Q^s↓）
  - 短缺产生
  - 出现黑市或排队
  - 死三角损失（Deadweight Loss, DWL）出现
```

| 福利方 | 变化 | 原因 |
|-------|------|------|
| 消费者福利 | 减少 | 部分消费者买不到 |
| 生产者福利 | 减少 | 售价低 + 产量少 |
| 总福利 | 减少 | DWL 出现 |

#### 情形 2：价格下限（P_floor > P*）

```
结果：Q 实际交易量 < Q*（因为 P↑，Q^d↓）
  - 过剩产生
  - 政府收购或减产
  - 死三角损失 DWL
```

### 4.2 税收的影响

```
对商品征收从量税 t 元/件
  消费者支付：Pc = P* + αt（承担 α 部分）
  生产者收到：Pp = P* - (1-α)t（承担 1-α 部分）
  实际交易量：Qt < Q*
```

```
价格 P
  │   S
  │    \  
Pc│─────\●  ← 含税价
  │      \\\
  │       \\\  税额 t
  │        \\\
Pp│─────────●  ← 实际收到价
  │         D
  └────────────── 数量 Q
       Qt  Q*
```

| 指标 | 变化 | 几何 |
|------|------|------|
| CS 变化 | 减少 | 梯形 Pc-P*-Q*-Qt |
| PS 变化 | 减少 | 梯形 P*-Pp-Q*-Qt |
| 政府税收 | 增加 | 矩形 t × Qt |
| DWL | 增加 | 三角形（缺口） |

### 4.3 补贴的影响

```
对商品补贴 s 元/件
  消费者支付：Pc = P* - αs
  生产者收到：Pp = P* + (1-α)s
  实际交易量：Qt > Q*
```

| 福利方 | 变化 |
|-------|------|
| 消费者 | 获益（CS 增） |
| 生产者 | 获益（PS 增） |
| 政府 | 损失（补贴支出） |
| 总福利 | 通常减少（DWL） |

## 五、福利分析的扩展应用

### 5.1 垄断 vs 完全竞争

| 指标 | 完全竞争 | 垄断 | 差异 |
|------|---------|------|------|
| 产量 | Q*（最大） | Qm < Q* | 垄断减产 |
| 价格 | P*（最低） | Pm > P* | 垄断抬价 |
| CS | 最大 | 较小 | 消费者受损 |
| PS | 较小 | 中等 | 生产者不一定增 |
| TS | 最大 | 较小 | DWL 出现 |
| DWL | 0 | > 0 | 资源错配 |

### 5.2 进口关税的福利分析

```
对进口商品征关税 t
  - 国内价格上涨 t
  - 国内产量增加
  - 进口减少
  - 政府获得关税收入
  - 净福利 = 收入 - 损失 → 通常为负
```

| 集团 | 影响 |
|------|------|
| 消费者 | 受损（CS 大幅减少） |
| 国内生产者 | 获益（PS 增加） |
| 政府 | 获益（关税收入） |
| 净福利 | DWL 增加 |

### 5.3 最低工资的福利分析

```
劳动市场管制：W_min > W*（均衡工资）
  - 就业量减少
  - 失业产生（Q^s > Q^d）
  - 在职者获益
  - 失业者受损
  - 总福利 DWL
```

## 六、关键概念总结

### 6.1 帕累托效率

| 概念 | 含义 |
|------|------|
| 帕累托最优 | 无法在不损一方情况下改善另一方 |
| 帕累托改进 | 一方改善同时不损他方 |
| 均衡关系 | 竞争市场均衡 = 帕累托最优（无外部性时） |

### 6.2 死三角损失（DWL）

```
DWL 出现的常见原因：
  - 价格管制（限价、限高）
  - 税收
  - 补贴
  - 垄断
  - 配额
  - 关税
  - 信息不对称
```

### 6.3 补偿原则

| 原则 | 标准 | 适用场景 |
|------|------|---------|
| 卡尔多-希克斯 | 赢家可补偿输家 | 政策评估 |
| 帕累托 | 不损任何方 | 严格评估 |
| 西托夫斯基 | 双方都认为更优 | 更严格 |
