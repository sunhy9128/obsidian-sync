# 阻断法令

> Blocking Statute — 各国反制美国长臂管辖的法律工具箱

**阻断法令**（Blocking Statute）是各国为对抗美国长臂管辖而制定的法律工具，旨在**阻断外国法律与措施的不当域外适用**对本国公民、法人和其他组织产生的影响。它通过法律手段明确"不承认"外国的域外管辖权，并为本国企业遵守外国不当制裁提供法律救济。阻断法令是国际法层面反制美元武器化的"法律战"工具，是各国对美国金融制裁的"制度性防御"。

---

## 一、核心定义与本质

### 1.1 定义

阻断法令是指**一国通过立法手段，禁止本国实体遵守外国不当的域外法律与措施**，并为遵守本国阻断法令的实体提供法律保护与救济的制度安排。

```
【阻断法令的核心要素】
├── 1. 法律基础：国内立法（不是行政命令）
├── 2. 阻断对象：外国法律与措施的不当域外适用
├── 3. 适用范围：本国公民、法人、其他组织
├── 4. 救济机制：禁令制度 + 损失补偿
├── 5. 反制手段：禁止遵守外国不当制裁 + 法律追责
└── 6. 豁免机制：特殊情况可申请豁免
```

### 1.2 阻断法令的本质

```
【阻断法令的本质】

表面：国内法律层面的"反制裁"工具
深层：对美国长臂管辖的"国际法挑战"

【阻断法令与金融武器化的对抗】
├── 美国：通过长臂管辖、次级制裁扩展美元霸权
├── 各国：通过阻断法令对冲长臂管辖
├── 本质：单边主义 vs 多边主义的法律博弈
└── 结果：国际金融治理的多极化趋势
```

### 1.3 阻断法令的类型谱系

```
【类型 ①：欧盟阻断法令】
├── 全称：Council Regulation (EC) No 2271/96
├── 制定：1996 年（针对美国对古巴制裁）
├── 升级：2018 年（针对美国退出伊核协议）
└── 特点：法律层级最高、执行机制最完整

【类型 ②：中国阻断办法】
├── 全称：《阻断外国法律与措施不当域外适用办法》
├── 制定：2021 年 1 月（商务部令 2021 年第 1 号）
├── 特点：与《反外国制裁法》《反外国不当域外管辖条例》协同
└── 法律层级：部门规章 + 法律协同

【类型 ③：俄罗斯反制裁法】
├── 全称：《关于影响俄罗斯利益的外国不法行为反制措施法》
├── 制定：2018 年（应对美国 CAATSA）
├── 特点：授权政府广泛的反制权
└── 2024 年起持续升级

【类型 ④：其他国家的类似立法】
├── 墨西哥：外交抗议 + 法律救济
├── 加拿大：部分立法限制
├── 日本/韩国：依赖美方豁免
└── 多数国家：观望
```

## 二、欧盟阻断法令（EU Blocking Statute）

### 2.1 发展历程

```
【欧盟阻断法令演进】

1996 年：
├── 美国通过赫尔姆斯-伯顿法、达马托法
├── 对古巴、利比亚、伊朗实施域外制裁
├── 欧盟强烈反对
├── 通过 Regulation 2271/96（欧盟阻断法令）
└── 目的：保护欧盟企业免受美国次级制裁

1996-2017：
├── 阻断法令实际执行有限
├── 多数欧盟企业主动遵守美国制裁
├── 阻断法令沦为"政治象征"
└── 美国通过执法压力压制欧盟企业

2018 年：
├── 特朗普退出伊核协议
├── 美国重启对伊朗全面制裁
├── 欧盟升级阻断法令
├── 增加"补偿机制"和"执行细则"
└── 但实际效果有限

2018-2024：
├── 美国对法国巴黎银行罚款 $89 亿
├── 欧盟阻断法令执行陷入尴尬
├── 多数欧盟企业选择遵守美国制裁
└── 阻断法令仍是"政治象征"
```

### 2.2 核心内容

```
【欧盟阻断法令的核心条款】

Article 1：禁止遵守
├── 禁止欧盟主体遵守美国对古巴的赫尔姆斯-伯顿法
├── 禁止欧盟主体遵守美国对利比亚、伊朗的达马托法
└── 违者：欧盟可处罚

Article 2：法院判决不可执行
├── 美国法院基于赫尔姆斯-伯顿法 / 达马托法的判决
├── 在欧盟境内不可执行
└── 欧盟法院可拒绝承认

Article 4-7：补偿机制
├── 因遵守美国制裁而损失的非美国实体
├── 可在欧盟法院起诉
├── 索赔损失 + 法律费用
└── 实际案例：极少（多数企业不愿对抗美国）

Annex：被禁止遵守的特定条款清单
├── 详细列出美国法律中不可遵守的条款
└── 实际执行困难
```

### 2.3 欧盟阻断法令的实际效果

```
【欧盟阻断法令的现实困境】

【困境 ①：法律 vs 商业的选择】
├── 多数欧盟企业选择遵守美国制裁
├── 因为美国市场太大、美元清算太重要
├── 阻断法令沦为"政治表态"
└── 实际索赔案例极少

【困境 ②：欧盟内部不统一】
├── 英国 2019 年退出 INSTEX
├── 法德"硬刚"，但承受力有限
├── 北欧国家倾向美国
├── 中东欧国家依赖美国安全保护
└── 欧盟"战略自主"难以落地

【困境 ③：执行机制薄弱】
├── 欧盟内部对阻断法令执行力度不一
├── 缺乏对违反阻断法令的惩罚
├── 实际威慑力有限
└── 多数企业仍主动遵守美国制裁
```

## 三、中国阻断办法

### 3.1 发展历程

```
【中国阻断办法演进】

2020 年：
├── 美国对中国企业实施大规模次级制裁
├── 华为、字节跳动、中兴等被列入实体清单
├── 中国意识到法律工具的必要性
└── 启动相关立法

2021 年 1 月 9 日：
├── 商务部发布《阻断外国法律与措施不当域外适用办法》
├── 商务部令 2021 年第 1 号
├── 与《反外国制裁法》《反外国不当域外管辖条例》协同
└── 构成中国反外国不当法律措施的"三大支柱"

2021 年 6 月 10 日：
├── 全国人大常委会通过《反外国制裁法》
├── 中国反制外国制裁的核心法律
├── 提供"以制裁反制裁"的法律基础
└── 与阻断办法形成层级互补

2023 年 12 月：
├── 商务部发布《反外国不当域外管辖条例》
├── 进一步细化阻断办法的执行机制
└── 形成"行政 + 立法"的双轨反制
```

### 3.2 核心内容

> 详见 [[阻断外国法律与措施不当域外适用办法]]：

```
【中国阻断办法的核心条款】

1. 触发机制：
   ├── 外国法律与措施的不当域外适用
   ├── 禁止或限制中国主体与第三国开展正常经贸
   └── 当事人可在 30 日内向商务部报告

2. 禁令制度：
   ├── 商务部可发布禁令
   ├── 要求中国主体不得承认、执行或遵守相关外国法律与措施
   └── 违者：依法追究法律责任

3. 损失补偿：
   ├── 因执行禁令而遭受损失的当事人
   ├── 可向造成损失的主体追偿
   └── 提供损失救济

4. 豁免机制：
   ├── 当事人因特殊情况无法遵守禁令
   └── 可向商务部申请豁免

5. 法律协同：
   ├── 与《反外国制裁法》协同
   ├── 与《反外国不当域外管辖条例》协同
   └── 形成系统性的反外国不当法律措施体系
```

### 3.3 中国反外国制裁法（AFSL）

```
【中国《反外国制裁法》核心内容】

通过时间：2021 年 6 月 10 日
法律层级：全国人大常委会法律
核心功能：

1. 反制清单制度
   ├── 国务院有关部门可制定反制清单
   ├── 列入清单的外国个人 / 实体
   ├── 禁止中国主体与其进行交易
   └── 冻结其在华资产

2. 反制措施
   ├── 拒绝签证、禁止入境
   ├── 冻结资产
   ├── 禁止交易
   └── 其他必要措施

3. 执行机制
   ├── 国务院负责具体执行
   ├── 各部门协同实施
   └── 法律层级高，震慑力强

4. 协同机制
   ├── 与阻断办法协同
   ├── 与反外国不当域外管辖条例协同
   └── 构成中国反外国制裁的"三把利剑"
```

### 3.4 中国反外国不当域外管辖条例

```
【中国《反外国不当域外管辖条例》】

发布：2023 年 12 月
发布方：国务院
法律层级：行政法规

核心内容：
├── 1. 明确"不当域外管辖"的法律标准
├── 2. 细化阻断办法的执行机制
├── 3. 加强部门协同
├── 4. 提供具体行政救济
└── 与阻断办法、反外国制裁法形成三层法律体系
```

### 3.5 中国反制法律体系的三层架构

```
【中国反外国法律措施的"三把利剑"】

第一层：法律（全国人大常委会）
└── 《反外国制裁法》（2021）

第二层：行政法规（国务院）
└── 《反外国不当域外管辖条例》（2023）

第三层：部门规章（商务部等）
└── 《阻断外国法律与措施不当域外适用办法》（2021）

【三层架构的特点】
├── 法律层级从高到低
├── 适用范围从宽到窄
├── 执行机制从原则到具体
└── 形成系统性反制体系
```

## 四、俄罗斯反制裁法

### 4.1 发展历程

```
【俄罗斯反制裁法演进】

2014：克里米亚事件后
├── 美国对俄实施多轮制裁
├── 俄罗斯开始制定反制措施
└── 早期以行政命令为主

2018 年 6 月：
├── 美国通过 CAATSA 法案
├── 对俄罗斯实施更严厉次级制裁
├── 俄罗斯通过《关于影响俄罗斯利益的外国不法行为反制措施法》
├── 授权政府广泛的反制权
└── 标志俄罗斯反制裁法律体系成熟

2022 年至今：
├── 俄乌冲突后
├── 美欧对俄全面制裁
├── 俄罗斯反制裁法持续升级
├── 涵盖范围扩大到"不友好国家"
└── 形成"全面反制裁"体系
```

### 4.2 核心内容

```
【俄罗斯反制裁法核心条款】

1. 反制对象
   ├── 对俄罗斯公民 / 法人的歧视性决定
   ├── 违反国际法的外国不法行为
   └── "不友好国家"清单（持续扩大）

2. 反制措施
   ├── 终止国际合作
   ├── 禁止商品进出口
   ├── 限制金融交易
   ├── 资产冻结
   ├── 人员入境限制
   └── 其他必要措施

3. 实施主体
   ├── 联邦政府（内阁）
   ├── 外交部、财政部、央行等协同
   └── 持续扩大"反制清单"

4. "不友好国家"清单
   ├── 涵盖美欧日澳加等 50+ 国家
   ├── 公民 / 实体在俄活动受限
   ├── 强制卢布结算（部分领域）
   └── 持续扩大中
```

### 4.3 俄罗斯反制裁法的"激进"特点

```
【俄罗斯反制裁法的特点】

1. **范围广泛**：涵盖所有对俄不利国家
2. **反制措施严厉**：包括金融资产冻结、贸易限制
3. **持续升级**：俄乌冲突后大幅升级
4. **与 SWIFT 切断反制相结合**：SPFS + 反制裁法
5. **强制卢布结算**：能源出口等关键领域
6. **代价极高**：但确保俄罗斯金融主权
```

## 五、阻断法令的国际比较

### 5.1 各国阻断法令对比

| 维度 | 欧盟 | 中国 | 俄罗斯 |
|------|------|------|--------|
| **法律层级** | 欧盟法规（直接适用）| 三层法律体系 | 联邦法律 |

| **核心触发** | 美国对古巴 / 伊朗 / 利比亚制裁 | 美国对中国企业的次级制裁 | 美欧对俄制裁 |

### 5.2 为什么各国的实际效果不同？

```
【阻断法令有效性的决定因素】

【俄罗斯】效果最强：
├── 政治决心强（生存危机）
├── 强制执行（卢布结算令）
├── 有替代基础设施（SPFS）
├── 与 CIPS 协同
└── 代价：经济孤立 + 制裁反噬

【中国】效果中等：
├── 政治决心强（但有选择）
├── 三层法律体系（已构建）
├── 有替代基础设施（CIPS / mBridge）
├── 一带一路提供应用场景
└── 代价：可控（多元布局）

【欧盟】效果较弱：
├── 政治决心不足（依赖美国安全）
├── 法律完整但执行薄弱
├── 缺乏独立基础设施（INSTEX 失败）
├── 内部对美国的依赖不对称
└── 代价：与美方摩擦 + 企业合规风险
```

## 六、阻断法令与美国长臂管辖的博弈

### 6.1 美国的长臂管辖困境

```
【美国长臂管辖的两难】

美国希望长臂管辖维持美元霸权
  ↓
各国通过阻断法令反制
  ↓
美国面临选择：
├── 严格执行长臂管辖 → 与各国关系恶化
└── 放松长臂管辖 → 美元霸权削弱

【美国的"走钢丝"】
├── 1. 继续使用长臂管辖，但范围扩大
├── 2. 对盟友的阻断法令选择性执法
├── 3. 对中俄的阻断法令严格执行
├── 4. 推动数字美元等新工具
└── 5. 但每次使用都削弱美元中立性
```

### 6.2 阻断法令的"反作用力"

```
【阻断法令的反作用力曲线】

阻断法令效用
100% │ ● 1996 欧盟阻断法令（100% 政治意义）
 80% │
 60% │     ● 2021 中国阻断办法（持续完善）
 40% │
 20% │        ● 2018 俄罗斯反制裁法（实际效用最强）
  0% └─────────────────────────────→ 时间
       1996  2021  2018  2024  未来
       
【反作用力的传导】
├── 阻断法令 → 长臂管辖执行困难
├── 阻断法令 → 替代清算系统建设加速
├── 阻断法令 → 各国去美元化加速
├── 阻断法令 → 美国长臂管辖边际效用下降
└── 总体：阻断法令是"金融武器化"的有效反制
```

### 6.3 阻断法令 vs 长臂管辖的"非对称性"

```
【两者的"非对称性"】

长臂管辖（攻击端）：
├── 优势：美国金融基础设施全球主导
├── 弱点：每次使用都削弱美元中立性
└── 趋势：边际效用下降

阻断法令（防御端）：
├── 优势：法律层面明确反制
├── 弱点：依赖替代基础设施（多数国家缺乏）
└── 趋势：随着替代基础设施完善，效用上升

【非对称博弈的演进】
├── 短中期：长臂管辖仍有优势（基础设施不对称）
├── 中长期：阻断法令效用上升（基础设施对称化）
└── 长期：可能形成"金融恐怖平衡"
```

## 七、阻断法令的实战案例

### 7.1 中国不可靠实体清单

```
【中国不可靠实体清单】

发布：2020 年 9 月（商务部）
性质：中国版"实体清单"

核心机制：
├── 1. 列入清单的外国实体
│   ├── 危害中国主权、安全、发展利益
│   └── 违反正常市场交易原则，中断与中国实体正常交易
├── 2. 反制措施
│   ├── 限制其在华贸易
│   ├── 限制其在华投资
│   └── 限制相关人员入境
└── 3. 实施案例
    ├── 已列入多家美国企业
    └── 包括军工、科技、监控等领域

【战略意义】
├── 中国版"长臂管辖"反制工具
├── 与阻断办法、反外国制裁法协同
└── 体现"以制裁反制裁"的法律基础
```

### 7.2 俄罗斯卢布结算令

```
【俄罗斯卢布结算令（2022 年）】

背景：
├── 俄乌冲突后，美欧对俄能源出口制裁
├── 俄罗斯决定反制
├── 普京签署总统令
└── 要求"不友好国家"用卢布结算天然气

执行机制：
├── 1. 买方在俄罗斯天然气工业银行（Gazprombank）开卢布账户
├── 2. 通过 SPFS 完成卢布支付
├── 3. 实际效果：
│   ├── 短期：欧盟内部分裂（匈牙利、奥地利支持）
│   ├── 中期：欧盟被迫替代为人民币结算
│   └── 长期：俄罗斯被逼全面拥抱人民币
└── 战略意义：俄罗斯反制裁法的"实战检验"

【关键洞察】
├── 强制执行 + 替代基础设施 = 反制裁有效
└── 没有 SPFS 和 CIPS，俄罗斯无法执行卢布结算令
```

## 九、关键洞察

> [!key-insight] 阻断法令 = 国际法层面的"反美元武器化"
>
> 阻断法令是对美国金融武器化的"法律反制"工具。它通过法律手段明确"不承认"外国的域外管辖权，是国际法层面挑战美元霸权的"软实力"工具。

> [!key-insight] 阻断法令 + 替代基础设施 = "完整反制"
>
> 单有阻断法令不够，还需要替代金融基础设施。俄罗斯（SPFS + 卢布结算令 + 阻断法令）的成功反制，源于**完整组合**；欧盟（仅有 INSTEX + 阻断法令）的失败，源于**基础设施缺失**。

> [!contradiction] 阻断法令的"政治—经济"两难
>
> 阻断法令在法律上明确反制长臂管辖，但在经济上需要企业承担"对抗美国"的成本。多数国家（包括欧盟）难以承担这一成本，导致阻断法令沦为"政治表态"。

> [!key-insight] 中国"三把利剑"的系统性
>
> 中国构建了"法律 + 行政法规 + 部门规章"的三层反外国法律措施体系（反外国制裁法 + 反外国不当域外管辖条例 + 阻断办法），这是国际反外国制裁领域最完整的法律体系之一。

> [!key-insight] 阻断法令的"长臂管辖悖论"
>
> 美国越是使用长臂管辖，越多国家制定阻断法令；阻断法令越完善，长臂管辖越难执行。这是一个"长臂管辖自我削弱"的悖论。
