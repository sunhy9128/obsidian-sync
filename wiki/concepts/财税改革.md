# 财税改革

> 二十届三中全会核心议题之一：开源节流，事权上收中央

## 一、核心定义

财税改革是指中国中央与地方政府在财政收入（税收、非税收入）、财政支出（事权与支出责任）、预算管理等方面的体制改革。

## 二、二十届三中全会主要内容

### 2.1 地方财政压力

| 压力源 | 说明 |
|--------|------|
| 土地财政收入 | 占比约50%，如今承压 |
| 化债问题 | 隐性债务风险 |
| 基层三保 | 保基本民生、保工资、保运转 |

### 2.2 开源措施

1. **消费税征收环节后移并下划地方** — 高档汽车、化妆品、名表等百亿/千亿级别税源
2. **合并三税** — 城市维护建设税、教育费附加、地方教育附加 → 地方附加税
3. **非税收入管理权限下放地方** — 国有资源有偿使用收入等
4. **研究数字经济新业态税收制度** — 直播带货等
5. **优化共享税分享比例** — 增值税、企业所得税、个人所得税
6. **扩大专项债券支持范围**
7. **清理专项转移支付，增加一般性转移支付**

### 2.3 节流措施

1. 依托行政权力取得的收入**全部纳入政府预算管理**
2. 国有资本经营预算中**国企上缴利润比例仅7.3%-11%**，有提升空间
3. **适当加强中央事权、提高中央财政支出比例**
4. **不得违规要求地方安排配套资金**

## 三、核心逻辑

> 钱归拢统一管理 → 减少地方隐形支出 → 强化国家重大战略任务和基本民生保障

**央地关系变化**：
- 十八届三中全会："保持现有央地财力格局总体稳定"
- 二十届三中全会：多方面增加地方收入 + 事权上收中央
