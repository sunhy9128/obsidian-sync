# 融资融券

> A 股杠杆交易与做空机制，俗称"两融"

**融资融券**包括两项业务：**融资**（向券商借钱买股票，做多杠杆）和**融券**（向券商借股票卖出，做空）。2010 年 3 月 31 日正式启动试点，是 A 股从"单边市"走向"双边市"的标志性制度变革。

---

## 一、基本概念

### 融资（Margin Trading）

```
投资者向券商融入资金 → 买入股票 → 股价上涨 → 卖出获利 + 归还本息
                          股价下跌 → 追加保证金 or 被强制平仓
```

- 本质：**杠杆做多**
- 杠杆倍数：通常 1-2 倍（监管上限：保证金比例 ≥ 100% 即 ≤ 1 倍杠杆，实际因标的折算率不同）
- 利息：年化 6-8%（随 LPR 调整，可协商）

### 融券（Securities Lending）

```
投资者向券商融入股票 → 卖出 → 股价下跌 → 低价买回归还股票
                           股价上涨 → 亏损 + 追加保证金
```

- 本质：**做空**
- 券源：券商自营持仓 + 转融通（中证金融） + 机构出借
- 融券费率：年化 2-10%（因券源稀缺度差异极大）

---

## 二、准入门槛与规则

### 投资者准入

| 条件 | 要求 |
|------|------|
| 资产门槛 | 开通前 20 日日均资产 ≥ **50 万元** |
| 交易经验 | ≥ 6 个月交易经验 |
| 风险评估 | 风险测评 C4（积极型）及以上 |
| 知识测试 | 通过融资融券知识测试 |

### 标的证券要求

不是所有 A 股都能融资融券。标的股票由交易所定期调整，通常满足：

- 上市时间 > 3 个月
- 流通市值 > 一定门槛（目前约 50 亿）
- 日均成交额 > 一定标准
- 股东人数 > 4000 户
- 非 ST/*ST 股票

**目前标的范围**：约 2200 只（覆盖 A 股约 40%），沪深 300 成分股基本全部纳入。

### 保证金与风控指标

| 指标 | 定义 | 监管下限 | 券商通常要求 |
|------|------|----------|-------------|
| **保证金比例** | 保证金 / 融资金额（或融券市值） | ≥ 100% | 100%~150%（因标的折算率不同） |
| **维持担保比例** | 总资产 / 总负债 | ≥ 130% | 警戒线 140%，平仓线 130% |
| **补仓线** | 低于时需追加保证金或减仓 | — | 140% |
| **平仓线** | 低于时券商强制平仓 | — | 130%（T+1 内未补仓） |

**维持担保比例的计算**：

```
维持担保比例 = (现金 + 证券市值) / (融资负债 + 融券负债) × 100%

例：100 万本金 + 融资 100 万买入 200 万股票
    若股价跌 35% → 证券市值 = 130 万
    维持担保比例 = 130 / 100 = 130% → 触及平仓线
```

## 三、两融余额：A 股最重要的情绪指标

**两融余额** = 融资余额 + 融券余额。每日收盘后公布，是判断市场热度最常用的量化指标之一。

### 融资余额（做多情绪）

| 状态 | 含义 | 信号 |
|------|------|------|
| 融资余额持续上升 | 杠杆资金看多，风险偏好提高 | 牛市/行情活跃期特征 |
| 融资余额快速下降 | 杠杆资金出逃，恐慌去杠杆 | 市场见顶/调整开始 |
| 融资余额见顶（天量） | 杠杆用尽，后续买盘枯竭 | 顶部预警信号 |
| 融资余额见底（地量） | 去杠杆接近尾声，卖压枯竭 | 底部区间的必要条件 |

### 融券余额（做空情绪）

A 股的融券余额在全球范围内**极低**——通常不到融资余额的 5%，这与港股（融券占 ~20%）和美股（做空活跃）形成鲜明对比。

| 市场 | 融券占两融比例 | 说明 |
|------|---------------|------|
| A 股 | ~2-5% | 做空极不发达 |
| 港股 | ~15-25% | 做空较活跃 |
| 美股 | ~30%+（含衍生品空头） | 做空生态发达 |

### 历史关键节点

| 时间 | 融资余额 | 市场状态 |
|------|---------|----------|
| 2015-06 高点 | ~**2.27 万亿** | 杠杆牛市顶点，随后去杠杆崩盘 |
| 2016-01 熔断 | ~1.1 万亿 | 熔断恐慌，融资快速下降 |
| 2018-12 低点 | ~**7500 亿** | 贸易战+去杠杆双重底部 |
| 2021-12 结构性高点 | ~1.7 万亿 | 核心资产行情尾声 |
| 2024-02 低点 | ~1.3 万亿 | 市场流动性危机后修复 |

> **核心规律**：融资余额与市场走势高度正相关。在 A 股，融资余额既是**行情的燃料**，也是**风险的放大器**——上行时追涨强化趋势，下行时强制平仓加剧踩踏。

## 四、转融通机制

**转融通**是中证金融（中国证券金融公司）向券商提供资金和证券的批发业务，是融资融券的上游环节。

```
                   转融通                        融资融券
中证金融 ────────→ 券商 ──────────→ 投资者
（批发）          （零售）          （终端）

转融资（资金批发）  →  券商融入资金  →  投资者融资买入
转融券（证券批发）  →  券商融入证券  →  投资者融券卖出
```

### 转融券的券源

| 出借人 | 占比 | 特点 |
|--------|------|------|
| 公募基金 | ~60% | 最大券源，出借 ETF/股票持仓 |
| 保险公司 | ~15% | 长期持股，出借意愿较高 |
| 社保基金 | ~10% | 稳定券源 |
| 券商自营 | ~10% | 自有持仓出借 |
| 其他（上市公司股东） | ~5% | 战略配售/限售股（科创板可出借） |

> 2023 年 8 月，监管层收紧战略投资者出借限售股用于融券，以平息"量化砸盘"争议。这是 A 股融券制度的一次重要调整。

---

## 五、融资融券的 A 股特色

### 5.1 融资独大，融券萎缩

A 股两融的核心特征是**严重不对称**：融资 ≈ 95%，融券 ≈ 5%。这与监管态度直接相关：

- **融资**被视为"正常的流动性工具"，政策上鼓励发展
- **融券**长期被贴上"做空"的负面标签，券源受限，制度上处处掣肘

### 5.2 与港美股做空的根本差异

| 维度 | A 股融券 | 港股融券 | 美股做空 |
|------|---------|---------|---------|
| 券源充足性 | 极有限 | 中等 | 充裕 |
| 做空成本 | 高（年化 8-20% + 仅少数券可融） | 中等（年化 2-8%） | 低（年化 0.3-3%） |
| 裸卖空 | 禁止 | 禁止 | 部分允许（做市商豁免） |
| 做空报告生态 | 几乎不存在 | 有（浑水/GMT 等） | 发达（机构 + 独立研究） |
| 做空对定价的影响 | 极小 | 中等（可抑制泡沫） | 显著（加速价格发现） |

### 5.3 融资与游资的共生关系

A 股许多题材股的短期爆发高度依赖融资资金——游资打板 → 融资盘跟进 → 股价加速。这解释了为什么 A 股题材股经常出现"涨过头"再"跌回原形"的现象。

## 六、风险与争议

### 对市场的影响

| 正面 | 负面 |
|------|------|
| 增加市场流动性 | 助涨助跌、加大波动 |
| 提供价格发现（融券方向） | 强制平仓引发连锁踩踏 |
| 丰富投资策略（对冲/套利） | 杠杆过度 → 系统性风险（2015 年教训） |

### 2015 年杠杆牛市的教训

2014-2015 年，场外配资 + 两融膨胀推动 A 股从 2000 点飙升至 5178 点。融资余额从 ~4000 亿暴增至 ~2.27 万亿。当监管开始清理场外配资并收紧两融时，强制平仓引发连锁抛售 → 千股跌停 → 流动性枯竭 → 国家队救市。这直接推动后续多项制度完善（两融宏观审慎管理、程序化交易监管等）。

### 当前监管框架

- 保证金比例不低于 100%（控制杠杆倍数）
- 单一股票融资/融券集中度限制（防范操纵）
- 两融标的定期调整机制
- 券商自主控制保证金比例（可高于监管下限）
