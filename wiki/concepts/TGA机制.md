# TGA机制

> 美国财政部在美联储的支票账户,联邦政府所有财政收支的清算中心

**TGA**(Treasury General Account,财政部总账户)是**美国财政部在美联储开设的支票账户**,用于联邦政府全部财政收支的清算。它是美联储负债端的核心组成部分,与流通中现金、银行准备金同级,但**不向公众开放、不支付利息、不被美联储放贷**。

## 关键事实

| 维度 | 内容 |
|---|---|
| 开户人 | 美国财政部 |
| 开户行 | **美联储**(联邦储备银行) |
| 账户性质 | **支票账户**(checking account) |
| 典型余额规模 | $3000 亿 - $1 万亿(波动) |
| 法律基础 | **31 U.S.C. § 3302** — 联邦政府资金必须存在美联储 |
| 是否付息 | 否(名义上 0%) |
| 是否被放贷 | **否** |
| 是否接受公众存款 | 否 |
| 谁能在美联储开户 | 仅限**银行 / 财政部 / 部分外国央行** |

## 美联储与商业银行的核心区别

| 维度 | 商业银行 | 美联储 |
|---|---|---|
| 接受个人/公司存款 | ✅ | ❌(只对机构) |
| 储蓄账户 | ✅ | ❌(无储蓄业务) |
| 活期账户 | ✅ | ⚠️(只对银行 + 财政部 + 部分外国央行) |
| 存款付息 | ✅ | ✅(IOR - Interest on Reserves,2008 后) |
| 用存款放贷 | ✅(部分准备金) | ❌(准备金不放贷) |
| 创造货币 | ❌(派生货币) | ✅(直接创造基础货币) |
| 盈利动机 | ✅ | ❌(非营利,有利润上缴财政部) |

## 美联储的三大职能

1. **货币政策** — 制定利率、调控货币供给
2. **金融稳定** — 监管银行、做最后贷款人
3. **支付结算 + 政府的银行** ← **TGA 在此**

第 3 项**不是**存款业务。美联储不接受个人存款、不办储蓄账户。

## 谁能在美联储开户?

只有三类机构:

| 账户持有人 | 账户名称 | 用途 |
|---|---|---|
| 美国财政部 | **TGA** | 联邦政府收支 |
| 存款类金融机构 | Reserve Account | 满足法定准备金 + 清算 |
| 外国央行 / 主权基金 | 一些账户 | 储备资产托管 |

**关键区别**:普通公司、个人**不能**在美联储开户。

## TGA 的实际运转机制

### 场景 1:财政部发行国债 → 钱进入 TGA

```
财政部宣布发 $100 亿 10 年期国债
    ↓
纽约联储代为拍卖 (auction)
    ↓
投标人提交报价(competitive / non-competitive)
    ↓
中标者名单确定
    ↓
结算日(T+1)
    ↓
中标者的开户行 reserve account -$100 亿
    ↓
TGA +$100 亿
    ↓
[美联储账上:负债端内部调整,A -$100 亿,B +$100 亿]
```

### 场景 2:财政部花钱 → TGA 减少

```
财政部通过 IRS 发 $1200/人 纾困金
    ↓
财政部在 Fedwire 发起支付指令
    ↓
收款人的开户行 reserve account 增加
    ↓
TGA 减少对应金额
    ↓
银行通过 ACH 把钱打进居民账户
```

### 场景 3:税收进入 TGA

```
纳税人 A 在 Chase 银行缴税 $5000
    ↓
Chase 通过 Fedwire 上缴
    ↓
Chase 的 reserve account -$5000
    ↓
TGA +$5000
```

## 美联储资产负债表的负债端结构

```
美联储负债端:

[现金]            ≈ $2.3 万亿     公众持有
[银行准备金]      ≈ $3 万亿       银行持有
[TGA]             ≈ $7000 亿      财政部持有
[其他央行存款]    ≈ 数百亿         外国央行持有
[逆回购 RRP]      ≈ 数千亿         一级交易商
```

**关键观察**:TGA 是美联储**自己的负债**,和现金、银行准备金同级。它们之间**划转**只是**美联储账本上的数字调整**,不创造也不销毁货币。

## Fedwire — 美联储的支付基础设施

- **Fedwire Funds Service** 每天处理约 70 万笔交易
- 约 $4 万亿美元的划转
- **实时结算**(real-time gross settlement, RTGS)
- TGA 的划转通过 Fedwire 完成

## TGA 水位的宏观意义

TGA 的余额变化是重要的宏观观察指标:

| TGA 状态 | 含义 |
|---|---|
| TGA 高 | 财政部在攒钱 / 还没花出去 |
| TGA 低 | 财政部在大量支出 / 需要发债补充 |
| TGA 急剧下降 | 通常是财政刺激期间(2020 疫情、2021 基建法案) |
| TGA 急剧上升 | 财政紧缩 / 国债净融资 |

### 2020 年实际水位变化

```
[2020-03-27] CARES Act 签署,$2.2 万亿授权支出
[2020-04]    TGA 从约 $4000 亿急剧下降
[2020-04 末] TGA 跌破 $3000 亿
[2020-05]    财政部紧急发新债补充 TGA
[持续循环]   整个 2020-2021,TGA 在低水位与发债补充之间循环
```

## 法律基础:31 U.S.C. § 3302

> "It shall be the duty of every Government officer... to deposit public money... in a Federal Reserve bank..."

**含义**:
- 联邦政府的所有钱**必须**存在美联储
- 财政部**不能**把钱存在商业银行
- 这是法律强制规定(防止政府资金被商业银行挪用)

## 直觉类比

| 维度 | 你的银行卡 | 财政部的 TGA |
|---|---|---|
| 开户行 | 商业银行 | 美联储 |
| 余额 | 你能花的 | 政府能花的 |
| 怎么变多 | 工资入账 / 转账收到 | 发债收入 / 税收 |
| 怎么变少 | 你去消费 | 政府支出 |
| 谁监管 | 商业银行 + 央行 | 美联储自己 |

## TGA 与 QE 的关系

QE 通过美联储从银行买旧国债影响 TGA:
- 美联储买旧国债 → 银行准备金增加 → **TGA 不直接受影响**
- 财政部发新债 → 银行准备金减少 → **TGA 增加**
- 两者**形式上独立**

## TGA 与"直升机撒钱"的区别

| 维度 | TGA + 国债融资(实际) | 直升机撒钱(假想) |
|---|---|---|
| 钱进入谁账户 | TGA(财政部) | 直接进入居民账户 |
| 是否创造货币 | 否(债是借的) | 是(印的) |
| 是否有债权人 | 公众(持债人) | 无 |
| 是否有债务记录 | 是 | 否 |
| 财政纪律 | 有约束 | 无约束 |
