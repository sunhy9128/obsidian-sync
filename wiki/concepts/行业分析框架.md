# 行业分析框架

> 系统评估行业吸引力、竞争格局和生命周期的方法论

行业分析处于宏观与个股之间——在确定了宏观方向（[[宏观经济]]）后，行业选择决定了超额收益的来源。行业分析框架帮助投资者回答"什么行业值得投"。

---

## 一、波特五力模型

> Michael Porter, 1980 — 最经典的行业竞争格局分析工具

**五力结构**：

```
                    潜在进入者
                        ↓ 威胁
            ┌───────────────────────┐
   供应商 ← →     行业内竞争     ← → 买方
   议价能力         (核心)         议价能力
            └───────────────────────┘
                        ↑ 威胁
                     替代品
```

### 各维度详解

| 维度 | 强威胁信号 | 弱威胁信号 | 对利润影响 |
|------|-----------|-----------|-----------|
| **行业内竞争** | 竞争者众多、产品同质化、退出壁垒高 | 寡头格局、差异化明显、高转换成本 | 直接影响定价权 |
| **潜在进入者威胁** | 技术门槛低、规模效应弱、无监管壁垒 | 高资本要求、专利/牌照保护、品牌壁垒 | 制约长期利润率 |
| **替代品威胁** | 替代品性价比优、用户切换成本低 | 无近替代品、替代品价格/性能劣势 | 限制涨价空间 |
| **供应商议价力** | 供应商集中度高、替代供应少、上游是必需品 | 供应充足、买方是重要客户、可后向整合 | 影响成本端 |
| **买方议价力** | 买方集中、产品标准化、信息透明 | 买方分散、差异化强、转换成本高 | 压制价格 |

### A 股实操案例

**白酒行业（五力极优）**：
- 行业内竞争：高端白酒寡头（茅五泸），品牌壁垒极深
- 进入者威胁：品牌壁垒 + 历史窖池不可复制
- 替代品威胁：社交属性下无近替代品
- 供应商议价力：粮食供应充足，议价力弱
- 买方议价力：经销商分散，高端白酒供不应求

→ 结论：五力量化得分高，是长期优质赛道（也是 A 股 2016-2021 核心资产行情的核心逻辑）

## 二、SWOT 分析

> 企业战略分析工具，同时适用于行业和个股

```
              正面（Helpful）           负面（Harmful）
     ┌─────────────────────┬─────────────────────┐
 内   │   S（优势）         │   W（劣势）          │
 部   │   内部可控的积极因素  │   内部可控的消极因素  │
 因   │   - 品牌/技术壁垒    │   - 成本高/效率低    │
 素   │   - 规模/成本优势    │   - 产品单一/依赖    │
     ├─────────────────────┼─────────────────────┤
 外   │   O（机会）         │   T（威胁）          │
 部   │   外部环境的积极因素  │   外部环境的消极因素  │
 因   │   - 政策利好/国产替代 │   - 监管加严/关税    │
 素   │   - 技术变革/新市场   │   - 技术颠覆/新对手  │
     └─────────────────────┴─────────────────────┘
```

**SWOT 的关键不是列出四象限，而是交叉分析**：

| 策略类型 | 逻辑 | 示例 |
|----------|------|------|
| **SO 策略** | 利用优势抓住机会 | 光伏龙头利用技术优势抓住海外需求爆发 |
| **WO 策略** | 改善劣势以把握机会 | 芯片公司引入技术补齐短板以承接国产替代 |
| **ST 策略** | 利用优势抵御威胁 | 龙头互联网公司用资本优势应对新进入者 |
| **WT 策略** | 减少劣势规避威胁（防御） | 传统零售收缩门店应对电商冲击 |

## 三、行业生命周期

每个行业经历从萌芽到衰退的演化过程，不同阶段的投资逻辑完全不同：

```
              市场/利润
                 ↑
                 │          成熟期
                 │        ┌──────┐
                 │  成长期／        ＼衰退期
                 │  ┌─┐               ┌─┐
                 │  │ │               │ │
                 │  │ │ 导入期        │ │
                 │  │ │  ┌─┐          │ │
                 │  │ │  │ │          │ │
                 └──┴─┴──┴─┴──────────┴─┴──→ 时间
```

| 阶段 | 特征 | 投资逻辑 | 风险 |
|------|------|----------|------|
| **导入期** | 技术/模式验证中、收入极低、亏损普遍 | 看技术突破和渗透率（"从0到1"） | 技术路线失败、现金流断裂 |
| **成长期** | 增速 20%+、竞争加剧、格局未定 | 看渗透率提升 + 市占率扩张（"从1到10"） | 竞争烧钱、估值透支 |
| **成熟期** | 增速 <10%、格局稳定、现金流充沛 | 看 ROE + 分红 + 护城河深度 | 需求萎缩、颠覆性创新 |
| **衰退期** | 负增长、产能过剩、企业退出 | 看残值 + 转型可能 + 分红 | 不可逆下滑、价值陷阱 |

**A 股行业生命周期分布**：银行/地产（成熟后期）、消费/医药（成熟期）、新能源/半导体（成长中后期）、AI/量子计算（导入期）

## 四、行业分类体系

### 标准分类

| 体系 | 层级 | 用途 |
|------|------|------|
| **申万行业分类**（A 股主流） | 一级 31 个→二级 134 个→三级 346 个 | A 股行业研究最常用，定期调整 |
| GICS（全球通用） | 11 个行业 → 24 个行业组 → 69 个行业 | 国际比较、MSCI 编制 |
| 证监会行业分类 | 门类 19 个 → 大类 90+ | 监管统计用途 |

### 按投资属性分类（实战分类）

| 类别 | 特征 | 代表行业 | 估值方法 |
|------|------|----------|----------|
| **周期型** | 利润随宏观周期大幅波动 | 钢铁、有色、化工、航运 | PB（低位布局）、PE 反向指标 |
| **成长型** | 收入/利润高速增长 | 新能源、半导体、创新药 | PEG、PS、远期 PE |
| **价值型** | 利润稳定、估值低、分红高 | 银行、公用事业、公路 | PE、PB、股息率 |
| **防御型** | 需求刚性、不随周期波动 | 消费必需品、医药、电信 | PE（稳定区间） |
| **主题型** | 靠政策/事件/概念驱动 | 军工、信创、元宇宙 | 情绪指标为主、EPS 为辅 |

> **行业轮动**就是在这五类之间切换——在[[宏观经济]]的不同阶段，各类行业的相对表现差异明显。详见[[行业轮动]]。

## 五、行业集中度（CR / HHI）

衡量行业竞争格局的核心量化指标：

**CRn（前 n 家集中度）**：
- CR5 > 60% = 寡头格局，龙头定价力强
- CR5 < 20% = 完全竞争，利润微薄

**HHI（赫芬达尔指数）**：
- HHI > 2500 = 高度集中
- HHI < 1000 = 分散竞争

**A 股重点行业集中度参考**：

| 行业 | CR5 估算 | 格局 | 含义 |
|------|----------|------|------|
| 白酒（高端） | ~95% | 极集中 | 龙头利润丰厚 |
| 空调 | ~80% | 高集中 | 格美争霸，利润稳定 |
| 光伏组件 | ~40% | 中低集中 | 激烈竞争，利润薄 |
| 餐饮 | <5% | 极分散 | 无全国龙头，标准化困难 |
| 房地产（2020 前） | ~20% | 中低 | 高杠杆扩张期 |
| 房地产（2024） | ~40% | 提升中 | 出清后集中度向头部集中 |

> 集中度提升 = 龙头最受益。A 股历史上各行业 CR5 提升期往往对应龙头股价超额收益期。

## 六、A 股特色行业驱动因子

| 行业 | 核心驱动力 | 观察指标 |
|------|-----------|----------|
| **白酒** | 批价 + 库存周期 + 商务活动指数 | 茅台批价、经销商库存、PMI |
| **光伏** | 装机量 + 组件价格 + 技术进步 | 全球新增装机、组件价格走势、电池效率 |
| **半导体** | 全球半导体周期 + 国产替代率 | 费城半导体指数、设备进口额、资本开支 |
| **银行** | 净息差 + 不良率 + 政策利率 | LPR、NIM、不良率、M2 增速 |
| **新能源车** | 渗透率 + 价格战 + 出海 | 月度渗透率、行业毛利率、出口数据 |
| **医药** | 集采 + 创新药管线 + FDA 批准 | 集采降价幅度、IND 数量、BD 交易额 |
