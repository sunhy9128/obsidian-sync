# 卖空机制

> 借入股票卖出，待股价下跌后买回归还 —— 卖空是金融市场上最被误解的核心机制之一

## 一、定义

**卖空**（Short Selling）是指投资者**借入**某只股票（或证券）**卖出**，然后在**未来某个时点**以**更低的价格买回**股票**归还**给出借方，从而**赚取价差收益**的投资行为。

核心本质：
- **借入 → 卖出 → 等待 → 买入 → 归还**的五步操作
- 卖空收益 = 卖出价 − 买入价 − 借股票费 − 交易费用
- 卖空是**先卖后买**，与正常买入"先买后卖"完全相反

## 二、卖空的标准流程

### 2.1 五步操作详解

```
Step 1: 借入股票
  空头 → 券商 → 借出方（机构/大股东/ETF）
  约定借股票费（年化）+ 归还时间
  
Step 2: 在市场上卖出
  按当前市场价卖出
  现金进入空头账户
  
Step 3: 等待股价下跌
  持仓期间持续支付借股票费
  持仓有"时间成本"
  
Step 4: 低价买入
  当股价下跌到目标位
  按新价买入相同数量股票
  
Step 5: 归还股票
  把买入的股票归还给出借方
  结算现金差额（盈利或亏损）
```

### 2.2 一个具体例子

```
假设：空头预期 X 股票将从 100 元跌到 80 元

Step 1: 借入 1000 股 X 股票（借股票费 1% 年化）
Step 2: 按 100 元/股卖出，收到 100,000 元
Step 3: 等待 3 个月
Step 4: 按 80 元/股买入 1000 股，支付 80,000 元
Step 5: 归还 1000 股给借出方

收益计算：
  卖出收入: 100,000 元
  买入成本: 80,000 元
  价差收益: 20,000 元
  借股票费: 100,000 × 1% × 3/12 = 250 元
  净利润: 19,750 元

如果股价不跌反涨到 120 元：
  买入成本: 120,000 元
  价差亏损: -20,000 元
  借股票费: 250 元
  净利润: -20,250 元（爆仓）
```

## 三、卖空的两个关键依赖

### 3.1 关键依赖一：必须能"借到"股票

```
卖空 ≠ 凭空卖出
卖空 = 借入 → 卖出 → 归还

借股票池决定：
  - 你能卖空多少
  - 你的卖空成本多高
  - 你能卖空多久
```

详见 [[wiki/concepts/借股票池|借股票池]]。

### 3.2 关键依赖二：必须能"买回"股票

```
无论股价如何变化，空头**必须**在某一天买回股票归还

如果找不到股票买回：
  → 强行平仓（无论价格）
  → 推高股价
  → 触发 [[wiki/concepts/轧空|轧空]] 机制
```

## 四、卖空的四种主要类型

### 4.1 普通卖空

| 项目 | 说明 |
|------|------|
| 借股票来源 | 券商/机构借出池 |
| 借股票费 | 通常 0.1%-2% 年化 |
| 期限 | 灵活，可随时平仓 |
| 风险 | 强制平仓风险 |
| 监管 | 合法，规范操作 |

### 4.2 裸卖空（Naked Short Selling）

| 项目 | 说明 |
|------|------|
| 借股票来源 | **没有实际借到** |
| 借股票费 | 理论上为 0 |
| 期限 | 卖出后必须在 T+N 日内借到 |
| 风险 | 极高，可能被监管处罚 |
| 监管 | **多数市场禁止** |

> **裸卖空**：在没借到股票的情况下就卖出"不存在的股票"。2008 年金融危机后，全球主要市场都加强了对此的限制。

### 4.3 集中卖空（Concentrated Short）

| 项目 | 说明 |
|------|------|
| 借股票来源 | 大量集中借入 |
| 特点 | 卖空头寸占流通股市值 > 5% |
| 风险 | 易触发轧空 |
| 披露 | 美国 SEC 规定需披露 |

### 4.4 监管卖空

| 制度 | 说明 |
|------|------|
| **涨跌幅限制** | 防止极端波动 |
| **禁止裸卖空** | 必须先借到股票 |
| **提价规则** | 卖空价格必须 ≥ 当前最优买价 |
| **熔断机制** | 暂停交易防止轧空 |

## 五、卖空的经济学意义

### 5.1 卖空的"三大功能"

| 功能 | 机制 | 益处 |
|------|------|------|
| **价格发现** | 双向交易 → 价格更准确 | 防止泡沫 |
| **风险管理** | 持有者可以对冲风险 | 锁定收益 |
| **流动性提供** | 提供双向流动性 | 交易更活跃 |

### 5.2 卖空的"三大风险"

| 风险 | 机制 | 害处 |
|------|------|------|
| **轧空风险** | 多方锁仓 → 空头无法平仓 | 爆仓 |
| **操纵市场** | 协调卖空 → 制造恐慌 | 假信号 |
| **系统性风险** | 大量卖空 → 流动性危机 | 金融不稳定 |

## 六、做空与"看空"的区别

| 维度 | 做空（Short Selling）| 看空（Bearish View）|
|------|-------------------|-------------------|
| 性质 | 投资行为 | 投资观点 |
| 风险 | **无限**（股价可无限上涨）| 无（非投资）|
| 成本 | 借股票费 + 保证金 | 无 |
| 利润 | 股价下跌获利 | 无直接获利 |

> **关键洞察**：看空不必做空，但做空必然看空。做空是**用钱表达观点**。

## 七、卖空与"市场操纵"的边界

### 7.1 合法卖空 vs 操纵市场

| 合法卖空 | 操纵市场 |
|---------|---------|
| 真实借入股票 | 裸卖空或虚假申报 |
| 单向操作 | 协调多方联合 |
| 基于研究 | 基于谣言/虚假信息 |
| 个人决策 | 合谋协调 |
| 风险自担 | 嫁祸他人 |

### 7.2 监管红线

- ❌ **裸卖空**：未实际借入就卖出
- ❌ **虚假卖空**：制造交易假象
- ❌ **协同卖空**：多方合谋压低股价
- ✅ **真实卖空**：必须实际借到股票

## 八、卖空在全球市场的差异

| 市场 | 卖空规则 | 关键限制 |
|------|---------|---------|
| **美国** | 较自由 | 提价规则 + 禁止裸卖空 |
| **香港** | 自由 | 提价规则 + 披露要求 |
| **日本** | 较严 | 历史上多次禁止卖空 |
| **中国 A 股** | **完全禁止** | 不允许融券做空（仅 ETF 期权等有限工具）|
| **欧洲** | 自由 | UCITS 基金有部分限制 |

> **中国 A 股特殊性**：中国普通 A 股**不允许直接融券做空**（仅融券标的证券、ETF 期权等可做空），这是与美国/香港市场最大的差异之一。

## 九、卖空的关键术语

| 术语 | 英文 | 含义 |
|------|------|------|
| **卖空比例** | Short Interest | 卖空头寸占流通股比例 |
| **Days to Cover** | Days to Cover | 按平均成交量平仓所需天数 |
| **借股票费** | Stock Loan Fee / Borrow Rate | 借入股票的成本 |
| **借股票池** | Securities Lending Pool | 可借出股票的总池 |
| **召回** | Recall | 借出方要求归还股票 |
| **轧空** | Short Squeeze | 卖空者被迫买入推高价格 |
| **裸卖空** | Naked Short Selling | 未实际借入就卖出 |

## 十、卖空与"轧空"的关系

> **卖空是"埋伏"，轧空是"收网"。**

```
正常做空：
  空头布局 → 借股票 → 卖出 → 等待下跌 → 买入 → 归还
  （时间的朋友）

轧空反扑：
  多方锁仓 → 借股票池枯竭 → 空头无法平仓 → 被迫高价买入
  （时间的朋友变成时间的敌人）
```

详见 [[wiki/concepts/轧空|轧空]]。

## 十一、对投资策略的启示

### 11.1 作为多方

- ✅ 识别"卖空比例异常高"的股票
- ✅ 警惕"借股票费飙升"信号
- ✅ 在轧空迹象出现时坚定持有
- ⚠️ 警惕监管介入（熔断/禁止卖空）

### 11.2 作为空头

- ❌ 警惕"卖空比例 > 20%"的股票
- ❌ 警惕"借股票费飙升"信号
- ❌ 警惕"股权集中度高"的股票
- ✅ 严格控制仓位和止损

### 11.3 作为观察者

- 📊 关注 SEC 每月公布的卖空数据
- 📊 关注借股票费率变化
- 📊 关注 Days to Cover 指标
- 📊 警惕"轧空临界点"

## 十二、一句话总结

> **卖空 = 借入卖出，等待下跌，买回归还**。它是一种"**先卖后买**"的逆向操作，本质是"**用时间换价差**"的多空博弈。卖空不是凭空卖出（必须借股票），卖空也不是无限盈利（股价可无限上涨），卖空是金融市场的"镜子"——它让价格更准确，但也让市场更脆弱。
