# 沃尔克规则（Volcker Rule）

> "我想要的是一部简单到能被一个酒鬼理解的法律"——保罗·沃尔克，2010 年金融危机调查委员会

## 一、核心定义

**沃尔克规则**（Volcker Rule）是 2010 年《多德弗兰克华尔街改革与消费者保护法案》（Dodd-Frank Act）的核心条款之一，以**前美联储主席保罗·沃尔克**（Paul Volcker）命名。规则的核心是**禁止美国存款机构从事某些类型的自营交易（proprietary trading），并限制其对对冲基金和私募股权基金的投资**。

**一句话总结**：**沃尔克规则 = 禁止银行用存款人的钱"赌博"**。

### 1.1 政策目标

```
沃尔克规则的核心目标：

1. **保护存款人**
   - 存款受 FDIC 保险
   - 存款不应被用于高风险投资
   - 银行亏损不应由纳税人承担

2. **防止"太大而不能倒"**
   - 限制银行的高风险业务
   - 减少系统性风险
   - 让银行回归"传统"信贷业务

3. **避免利益冲突**
   - 银行不应与客户"对赌"
   - 银行不应利用客户信息为自己谋利
   - 银行应做"中介"而非"投机者"

4. **维护金融稳定**
   - 银行是金融体系核心
   - 银行的冒险行为威胁整体稳定
   - 必须限制
```

## 二、保罗·沃尔克其人

### 2.1 沃尔克的传奇经历

```
保罗·沃尔克（Paul Volcker，1927- ）：

1979-1987：美联储主席
  - "沃尔克冲击"
  - 将联邦基金利率从 10% 升至 20%
  - 战胜通胀（13.5% → 3.2%）
  - 但导致 1980-1982 衰退
  - 奠定"中央银行独立性"原则
  - 被誉为"最伟大的央行行长"
  
2009-2011：奥巴马经济顾问委员会主席
  - 推动金融监管改革
  - 主张限制银行规模
  - 提出"沃尔克规则"
  
2010：金融危机调查委员会（FCIC）主席
  - 主导 2008 危机的官方调查
  - 揭露金融危机的真正原因
  - 推动沃尔克规则立法
```

### 2.2 沃尔克的核心观点

```
沃尔克对金融危机的反思：

1. "华尔街已经变质"
   - 从"金融服务"变成"自我服务"
   - 大银行不再为实体经济服务
   - 而是为自己赚钱

2. "银行太大"
   - 2008 年救助本质是"贿赂"
   - "大而不倒"是政治权力
   - 必须拆分或限制

3. "自营交易是赌博"
   - 用存款人的钱赌博
   - 赢了银行高管赚奖金
   - 输了纳税人买单
   - 必须禁止

4. "金融体系需要规则"
   - 自由市场不是放任
   - 需要明确的边界
   - 沃尔克规则就是这个边界
```

## 三、沃尔克规则的立法历程

### 3.1 立法时间线

```
2009.1 奥巴马就职
  - 承诺金融监管改革
  - 任命沃尔克为顾问
       ↓
2009.12 沃尔克提出原则
  - 禁止银行自营交易
  - 限制银行对冲基金投资
       ↓
2010.1 奥巴马国情咨文
  - "绝不让华尔街把赌场搬到大街上"
  - 正式推动沃尔克规则
       ↓
2010.7 多德弗兰克法案通过
  - 沃尔克规则成为法案 Section 619
  - 奥巴马签署
       ↓
2011-2013 规则细化
  - 美联储、OCC、SEC、CFTC、FDIC 五大监管机构
  - 长期争议和妥协
  - 2013.12 最终规则发布
       ↓
2014.4 规则生效
  - 银行开始调整业务
  - 关闭自营交易部门
  - 出售对冲基金/PE 投资
       ↓
2017-2018 特朗普政府放松
  - 2018.5 OCC 提案放松
  - 2019.8 五大监管机构联合发布修订规则
  - 简化"合规"要求
  - 实际放松监管
       ↓
2020-2021 COVID 危机
  - 沃尔克规则被部分豁免
  - 美联储为银行自营交易提供便利
  - 帮助银行交易国债支持市场
       ↓
2023-2024 进一步争议
  - 银行业反对严格的沃尔克规则
  - 国会讨论进一步修改
  - 监管的拉锯持续
```

### 3.2 立法中的激烈争议

```
支持方（沃尔克、消费者团体、学者）：
  - 银行自营交易是 2008 危机的重要原因
  - 必须限制
  - 保护存款人
  - 维护金融稳定
  
反对方（华尔街、银行游说）：
  - 自营交易提高流动性
  - 限制自营会降低市场效率
  - 影响银行盈利
  - 影响美国金融业全球竞争力
  
关键让步（最终版本）：
  - 大量"合规豁免"
  - 做市、对冲、避险被允许
  - 海外业务豁免
  - 实际上比沃尔克原意宽松
```

## 四、沃尔克规则的核心条款

### 4.1 三类限制

```
沃尔克规则限制的三类活动：

1. **自营交易（Proprietary Trading）**
   - 银行用自有账户做交易
   - 赌方向、赌利差、赌波动率
   - 实际禁止

2. **对冲基金 / 私募股权基金投资**
   - 银行投资对冲基金
   - 银行投资 PE 基金
   - 限制规模（基金资本的 3% 或银行一级资本的 3%）
   - 必须向投资者披露利益冲突

3. **某些短期交易**
   - 持有 60 天以内的资产（短期）
   - 如果是"投机性"也受限制
```

### 4.2 关键定义

```
"自营交易"的定义（争议巨大）：

禁止的（自营交易）：
  - 银行用自有资本
  - 持有金融头寸
  - 主要目的是短期转售获利
  - 没有"明确客户"或"对冲目的"
  
允许的（不属于自营交易）：
  - 做市（Market Making）：为客户提供流动性
  - 做市（Hedging）：对冲客户风险
  - 风险管理（Hedging）：对冲银行自身风险
  - 政府债券交易（部分允许）
  - 外国政府债券交易
  
关键问题：
  - 如何区分"自营"和"做市"？
  - 如何区分"投机"和"对冲"？
  - 大量争议和监管解释
```

### 4.3 "合规计划"要求

```
沃尔克规则要求银行制定详细的"合规计划"：

1. 书面合规政策和程序
2. 内部监控和报告系统
3. 高管层的"合规证书"
4. 定期内部和外部审计
5. 员工培训
6. 风险管理框架
       ↓
合规成本：
  - 大型银行每年花费 1-5 亿美元合规
  - 中型银行 5000 万 - 1 亿
  - 是沃尔克规则的主要批评点之一
```

## 五、沃尔克规则与做市业务

### 5.1 做市豁免的争议

```
关键争议：自营 vs 做市

定义：
  - 自营：用自有账户赌方向
  - 做市：为客户提供流动性，赚取买卖价差
       ↓
问题：
  - 银行常常用"做市"包装自营交易
  - 例如："我们持有 100 亿美元国债库存是为了做市"
  - 但实际上是在赌利率方向
       ↓
监管应对：
  - 2013 规则引入"持仓限额"（Inventory Limits）
  - 要求银行证明做市的合理规模
  - 但具体执行有难度
       ↓
实际结果：
  - 银行业大幅缩减做市业务
  - 流动性提供者减少
  - 债券市场流动性下降
  - 这是沃尔克规则的主要副作用
```

### 5.2 2018-2020 改革

```
沃尔克规则的修订（2019.8）：

简化合规：
  - 减少小银行的合规要求
  - 简化"对冲"定义
  - 允许更多"以客户为导向"的交易
  
放宽限制：
  - 银行对基金的投资限制适度放松
  - 保险类子公司不受同样限制
  - 外国银行海外子公司豁免扩大
  
影响：
  - 大银行欢呼（合规成本降低）
  - 消费者团体批评（监管放松）
  - 沃尔克本人公开反对（"背叛了我的初衷"）
```

## 六、沃尔克规则与 2008 危机

### 6.1 2008 危机前的银行冒险

```
2008 危机前，银行的冒险行为：

1. 自营交易
   - 摩根大通"伦敦鲸"事件（2012）：亏损 60 亿美元
   - 高盛、德银、巴克莱：自营交易规模巨大
   - 银行用存款人的钱赌博

2. 对冲基金 / PE 投资
   - 银行通过 SPV 投资对冲基金
   - 杠杆率 30+ 倍
   - 2008 年损失数千亿美元

3. 次贷 CDO 投资
   - 银行大量持有次贷相关 CDO
   - 用存款人的钱赌房地产
   - 雷曼破产时持有 600+ 亿次贷资产

4. 风险转移
   - 通过 SIV 把风险"藏"在表外
   - 通过 CDS 把风险"转嫁"
   - 但最终风险回到银行体系
```

### 6.2 沃尔克规则的设计逻辑

```
沃尔克规则的设计：

2008 危机的核心教训：
  - 银行冒险 → 用存款人的钱 → 纳税人买单
  - 必须切断"冒险 → 存款人 → 纳税人"的链条
  
沃尔克规则的解决方案：
  - 禁止银行自营交易（切断"冒险"）
  - 限制基金投资（减少关联风险）
  - 保护存款保险（切断"纳税人买单"）
       ↓
核心逻辑：
  - 存款保险 = 公共资源
  - 公共资源不应被用于投机
  - 必须限制
```

### 6.3 沃尔克规则的有效性争议

```
支持方的论点：
  1. 银行自营交易规模大幅下降（70%+）
  2. 银行对基金投资规模下降（80%+）
  3. 防止了 2008 类型的银行冒险
  4. 减少了"太大而不能倒"的根源
  
反对方的论点：
  1. 银行仍然冒险（只是换了形式）
  2. 影子银行填补了真空（更不透明）
  3. 市场流动性下降
  4. 合规成本高昂
  
实证证据：
  - 2009-2023：未发生大规模银行倒闭
  - 2023 SVB 危机：与沃尔克规则无关（利率风险）
  - 部分支持其有效性
  - 但不能直接证明因果关系
```

## 七、沃尔克规则的五大批评

### 7.1 批评一：自营交易没有消失，只是形式改变

```
银行冒险的演变：

2008 前：
  - 银行直接自营交易
  - 在交易账簿（Trading Book）持有大量资产
  - 雷曼、AIG 模式

沃尔克规则后：
  - 银行减少直接自营
  - 但通过子公司、附属机构交易
  - 通过结构化产品投资
  - 通过关联对冲基金
  
例子：
  - 摩根大通"伦敦鲸"（2012）：名义合规，但实际是赌博
  - 高盛交易部门持续盈利
  - 银行对私募信贷（Private Credit）的兴趣上升
       ↓
批评观点：
  - "换个马甲而已"
  - 监管套利
  - 没有根本解决问题
```

### 7.2 批评二：市场流动性下降

```
沃尔克规则与债券市场流动性：

做市商减少：
  - 银行减少交易账簿
  - 减少做市持仓
  - 大型银行退出部分做市业务
  
债券市场流动性指标：
  - 买卖价差扩大
  - 交易量下降
  - 波动率上升
  
2020 年 3 月：
  - 美国国债流动性"异常"
  - 平时最安全的资产
  - 沃尔克规则后第一次重大"流动性事件"
  - 美联储被迫紧急购买国债
  
批评观点：
  - 沃尔克规则减少了做市能力
  - 在危机时放大了流动性问题
```

### 7.3 批评三：合规成本高昂

```
合规成本估算（每年）：

摩根大通：~5 亿美元
高盛：~4 亿美元
美国银行：~4 亿美元
富国银行：~3 亿美元
中型银行：5000 万 - 1 亿
  
问题：
  - 大银行有规模优势
  - 小银行难以承担
  - 进一步集中银行业（"太大而不能倒"加剧）
  
批评观点：
  - 沃尔克规则想要"限制大银行"
  - 但实际让大银行更强（合规优势）
  - 适得其反
```

### 7.4 批评四：全球影响

```
对美国银行全球地位的影响：

欧洲银行：
  - 德意志银行、巴克莱、瑞银：自营交易规模相对较小
  - 部分原因是欧洲监管已经较严
  - 美国银行在自营交易上"略胜一筹"
  
沃尔克规则后：
  - 美国银行自营受限
  - 欧洲、亚洲银行仍有自营能力
  - 美国银行全球竞争力下降
  
批评观点：
  - 过度监管
  - 让美国金融业在全球竞争中处于劣势
  - 把业务推到海外（监管套利）
```

### 7.5 批评五：经济效率损失

```
沃尔克规则与经济效率：

理论损失：
  - 银行自营交易提供价格发现
  - 做市提供流动性
  - 套利让市场更有效
  - 限制这些活动 = 经济效率损失
  
实证数据：
  - 估计年损失 100-500 亿美元（学术界估算分歧大）
  - 流动性损失难以精确量化
  
批评观点：
  - "看不见的损失"
  - 但实际损失很大
  - 消费者最终承担成本
```

## 八、沃尔克规则与其他监管改革

### 8.1 多德弗兰克法案的全景

```
多德弗兰克法案（2010）的主要条款：

1. 沃尔克规则（Section 619）
   - 限制银行自营交易
   - 限制银行基金投资

2. 消费者金融保护局（CFPB）
   - 监管消费者金融产品
   - 防止掠夺性贷款

3. 金融稳定监督委员会（FSOC）
   - 识别系统性风险
   - 指定 SIFI（系统性重要金融机构）

4. 压力测试（Dodd-Frank Act Stress Tests, DFAST）
   - 美联储对大银行进行压力测试
   - 确保银行有充足资本

5. 中央清算（CCP）
   - 强制衍生品中央清算
   - 减少对手方风险

6. "有序清算"机制（OLA）
   - 大银行倒闭的清算程序
   - 避免"大而不倒"
```

### 8.2 沃尔克规则 vs 巴塞尔协议

| 维度 | 沃尔克规则 | 巴塞尔协议 |
|------|----------|-----------|
| **范围** | 美国 | 全球 |
| **重点** | 自营交易 + 基金投资 | 资本 + 流动性 + 风险 |
| **法律地位** | 国内法 | 国际协议 |
| **执行** | 美联储、OCC、SEC、CFTC、FDIC | 各国央行 |
| **批评** | 影响市场流动性 | 监管套利、过于复杂 |

### 8.3 沃尔克规则 vs 其他改革

```
监管改革全景（2008-2020）：

1. 巴塞尔 III（2010-2019）
   - 提高资本充足率
   - 引入流动性覆盖率（LCR）
   - 净稳定融资比率（NSFR）
   - 杠杆率

2. 多德弗兰克法案（2010）
   - 沃尔克规则
   - 消费者保护
   - 衍生品改革
   - 系统性风险监管

3. 影子银行监管（2010 后）
   - 货币市场基金改革
   - SIFIs 指定
   - 回购市场改革

4. 国际协调（FSB、BCBS）
   - 全球系统重要性银行（G-SIBs）
   - 全球系统重要性保险公司（G-SIIs）
   - 总损失吸收能力（TLAC）
```

## 九、特朗普时代的沃尔克规则修订

### 9.1 2017-2019 放松监管

```
特朗普政府行动：

1. 任命放松监管的官员
   - Jerome Powell（Fed 主席）
   - Randal Quarles（Fed 监管副主席）
   - Joseph Otting（OCC 署长）

2. 监管改革
   - 2018.5 OCC 提案简化沃尔克规则
   - 2019.8 五大监管机构联合修订
   - 简化合规要求
   - 扩大"合规豁免"范围

3. 主要修订内容
   - 简化做市豁免
   - 减少小银行的报告要求
   - 允许银行更多自营活动
   - 减少合规人员配置
```

### 9.2 沃尔克的强烈反对

```
沃尔克本人（2010s 后期）：

"他们削弱了我的规则"
"他们背叛了我的初衷"
"金融体系还是太危险"
"应该强化而不是削弱"
       ↓
沃尔克的最后呼吁：
  - 公开反对修订
  - 撰写文章警告
  - 与银行游说集团"斗争"
  - 但特朗普政府不为所动
       ↓
沃尔克于 2019.12.14 去世
  - 享年 92 岁
  - 留下未完成的事业
```

### 9.3 2019 修订的实际影响

```
2019 修订的实质性变化：

1. **简化合规框架**
   - 三个层次（之前是四个）
   - 中小银行豁免扩大
   - 大银行合规仍严格

2. **明确"对冲"定义**
   - 允许更多对冲活动
   - 减少对冲的合规负担

3. **优化"做市豁免"**
   - 持仓限额简化
   - 报告要求减少
   - 风险敏感度参数化

4. **子公司豁免**
   - 银行对基金投资部分豁免
   - 子公司监管放松
       ↓
批评观点：
  - "实质上是削弱了沃尔克规则"
  - 但没有完全"废除"
  - 银行业仍受一定限制
```

## 十、2020 COVID 与沃尔克规则

### 10.1 沃尔克规则的临时豁免

```
2020.3 COVID 危机：

问题：
  - 美元荒 + 国债流动性异常
  - 美联储需要银行做市支持
  
沃尔克规则豁免：
  - 美联储 2020.4 发布豁免
  - 允许银行大量交易国债
  - 豁免自营交易限制
  - "为市场提供流动性"
       ↓
争议：
  - 沃尔克规则的"原始目的"是限制冒险
  - 但危机时需要银行冒险
  - 矛盾：保护 vs 应急
```

### 10.2 永久性修改

```
2020-2021 讨论的永久性修改：

1. **大型银行的国债交易**
   - 完全豁免国债做市
   - 允许大型银行"做市支持"
   - 美联储在背后支撑

2. **回购市场的特殊地位**
   - 承认回购市场是关键基础设施
   - 银行回购活动豁免
   - 鼓励做市

3. **对冲基金例外**
   - 银行对冲基金投资限制适度放松
   - 条件更灵活

4. **量化指标的引入**
   - 引入"风险敏感度"指标
   - 而非简单的"交易限额"
   - 让银行根据风险调整
```

## 十一、沃尔克规则的国际影响

### 11.1 对其他国家的启示

```
各国监管反应：

   - 没有完全禁止

   - 但未完全禁止自营

   - 维持传统监管
   - 未大幅修改

4. **中国**
   - 影子银行监管（2018 资管新规）
   - 部分借鉴沃尔克规则思路
   - 但具体形式不同
```

### 11.2 与中国"金融监管"的对比

```
沃尔克规则 vs 中国金融监管：

共同点：
  - 限制银行冒险
  - 保护存款人
  - 维护金融稳定

差异：
  - 美国：明确禁止某些活动
  - 中国：通过窗口指导、监管协调
  - 美国：法律明确、可预期
  - 中国：灵活、调整空间大

中国"沃尔克规则式"措施：
  - 资管新规（2018）：禁止资金池、刚性兑付
  - 银行理财子公司：隔离风险
  - 宏观审慎监管：MPA 体系
  - 资本充足率：与巴塞尔对齐
```

## 十二、沃尔克规则的核心命题

### 12.1 沃尔克规则的五大成就

```
1. **限制了银行自营交易**
   - 自营交易规模下降 70%+
   - 银行对基金投资下降 80%+
   - 直接减少了银行冒险

2. **改变了银行文化**
   - 从"冒险赚钱"到"服务客户"
   - 高管薪酬与冒险脱钩
   - 风险意识提升

3. **保护了存款人**
   - 存款保险基金规模充足
   - 银行冒险明显减少
   - 纳税人风险下降

4. **维护了金融稳定**
   - 2009-2023 未发生大规模银行倒闭
   - 系统性风险下降
   - 与 2008 形成对比

5. **开启了后危机监管**
   - 多德弗兰克法案的核心
   - 影响全球监管
   - 改变了金融业
```

### 12.2 沃尔克规则的五大局限

```
1. **监管套利**
   - 银行通过子公司继续冒险
   - 影子银行填补真空
   - 形式合规，实质未变

2. **市场流动性下降**
   - 做市商减少
   - 债券市场流动性恶化
   - 2020 年 3 月国债流动性"异常"

3. **合规成本高昂**
   - 大银行每年 5 亿美元
   - 中型银行难以承担
   - 进一步集中银行业

4. **全球竞争劣势**
   - 美国银行竞争力下降
   - 业务外流
   - 监管套利到海外

5. **未能阻止 SVB 等危机**
   - 2023 SVB 危机与沃尔克规则无关
   - 但显示了银行风险的复杂性
   - 需要更全面的监管
```

### 12.3 沃尔克规则的"哲学反思"

```
沃尔克规则的核心问题：

1. **自由的边界**
   - 银行应该有"投机自由"吗？
   - 用存款人的钱赌博，应该允许吗？
   - "金融自由"的边界在哪？

2. **谁承担风险**
   - 银行冒险的收益归银行
   - 银行冒险的损失由纳税人承担
   - 这种不对称合理吗？

3. **大而不倒**
   - 沃尔克规则想要限制大银行
   - 但合规成本让大银行更强
   - 如何真正限制银行规模？

4. **监管 vs 创新**
   - 监管太严会抑制创新
   - 监管太松会引发危机
   - 平衡点在哪？

5. **公共利益 vs 私人利益**
   - 银行是公共机构（最后贷款人）
   - 但本质是私人企业
   - 这种矛盾的解决方式
```

## 十三、沃尔克规则与"最后贷款人"的关系

### 13.1 概念框架

```
最后贷款人（Lender of Last Resort）的逻辑：
  - 央行保护银行免于倒闭
  - 但必须避免道德风险
  - 沃尔克规则就是"减少道德风险"
  
逻辑链：
  1. 央行是最后贷款人
  2. 银行冒险 → 央行救助
  3. 这种预期让银行冒险
  4. 道德风险
  5. 沃尔克规则限制银行冒险
  6. 减少道德风险
  7. 但保留最后贷款人机制
```

### 13.2 2023 SVB 危机的反思

```
2023 SVB（硅谷银行）危机：

事件：
  - 2023.3 SVB 因利率风险倒闭
  - 48 小时内被 FDIC 接管
  - 引发区域性银行挤兑
  
与沃尔克规则的关系：
  - 与自营交易无关
  - 与对冲基金投资无关
  - 纯粹是利率风险
  
教训：
  - 沃尔克规则限制了"市场风险"
  - 但未限制"利率风险"
  - 银行风险的复杂性
  - 需要更全面的监管
```

## 十四、沃尔克规则的未来

### 14.1 未来 10 年的关键议题

```
1. **进一步放松还是强化**
   - 银行业持续施压放松
   - 消费者团体要求强化
   - 政治拉锯持续

2. **影子银行监管**
   - 沃尔克规则只覆盖存款银行
   - 影子银行成为新的风险源
   - 需要扩展监管范围

3. **气候变化风险**
   - 银行对化石燃料的投资
   - 是否应纳入"系统性风险"
   - 沃尔克规则的"绿色"版本？

4. **加密资产**
   - 银行是否应允许持有加密资产
   - 沃尔克规则是否适用
   - 新的监管挑战

5. **AI 与算法交易**
   - 高频交易是否属于"自营"
   - 算法驱动的风险管理
   - 新形式的银行冒险
```

### 14.2 国际协调的必要性

```
关键问题：
  - 美国沃尔克规则是"国内法"
  - 但金融市场是全球的
  - 银行可以转移到海外
  - 监管套利严重

国际协调的尝试：
  - 巴塞尔委员会（BCBS）
  - 金融稳定委员会（FSB）
  - 但具体规则差异大
  - 协调进展缓慢
```

## 十五、一句话总结

> **沃尔克规则 = 用最简单的法律解决最复杂的金融问题——禁止银行用存款人的钱"赌博"**。
>
> 它是**2008 金融危机的直接回应**，是**保罗·沃尔克职业生涯的"告别之作"**，也是**后危机金融监管的核心**。
>
> **沃尔克规则的有效性仍有争议**：
> - **支持方**：银行冒险减少，金融稳定提升
> - **反对方**：市场流动性下降，合规成本高昂，监管套利严重
>
> **核心矛盾**：**金融自由 vs 金融稳定**、**私人利益 vs 公共利益**、**国内监管 vs 全球套利**。
>
> **理解沃尔克规则 = 理解现代金融监管的本质**：**金融业是公共机构（最后贷款人），但本质是私人企业（盈利最大化）**。这一矛盾是金融监管的核心张力。**沃尔克规则用最简单的"禁令"试图解决，但金融创新总会找到新的形式**——**2020 年 3 月的国债流动性异常、2023 年的 SVB 危机**都显示了**银行风险的复杂性远超任何单一规则能覆盖**。
>
> **保罗·沃尔克在 2019 年去世前说："我想要的是一部简单到能被一个酒鬼理解的法律。"** —— **沃尔克规则实现了他的初衷，但金融体系永远比法律更复杂**。
