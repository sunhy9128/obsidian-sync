# 隐性刚兑

> 无正式契约但市场普遍预期的无条件兑付承诺 — 中国金融体系的特殊风险

**隐性刚兑**（Implicit Guarantee / Implicit Commitment）是指没有正式法律合同约束，但市场参与者普遍相信的"不会亏损"承诺。当产品发生损失时，发行方出于维护声誉、避免监管压力、维护客户关系等考虑，实际承担了损失赔付。

---

## 一、隐性刚兑的本质

### 1.1 显性 vs 隐性刚兑

| 类型 | 担保形式 | 法律依据 | 风险承担 |
|------|----------|----------|----------|
| **显性刚兑** | 银行保本理财（2018年前）、存款保险 | 合同条款、法律保护 | 发行方明确承担 |
| **隐性刚兑** | 银行理财"预期收益"、信托/私募"安慰函" | 无合同约束，市场预期 | 发行方声誉担保 |
| **真正的市场化** | 公募基金净值型 | 无任何保本承诺 | 投资者自担 |

### 1.2 隐性刚兑的形成逻辑

```
Step 1：某机构首次"兜底"
  → 产品亏损，机构为维护声誉主动赔付

Step 2：市场传播效应
  → "XX 机构的产品不会亏钱"
  → 投资者形成稳定预期

Step 3：竞争压力扩散
  → 其他机构面临客户流失压力
  → 被迫同样"兜底"以维持竞争力

Step 4：市场均衡
  → 整个行业形成隐性刚兑惯例
  → "买 XX 类产品不会亏"成为共识
  → 投资者不关注底层风险，只看收益率
```

## 二、中国金融体系中的隐性刚兑

### 2.1 银行理财

```
2018年前的银行理财产品：
- 产品合同：非保本浮动收益
- 市场实际：银行通常按"预期收益率"兑付
- 投资者认知：银行理财≈存款，有银行背书

为什么银行愿意刚兑？
- 维护客户关系（理财客户=潜在存款客户）
- 害怕声誉损失（"XX银行理财亏损"→存款搬家）
- 监管默许（监管机构也倾向于维稳）
- 资金池操作（用新资金偿付旧资金，本质是借新还旧）
```

### 2.2 信托产品

```
信托刚性兑付的历史：

2012-2017年：信托行业"零违约"
- 信托产品到期无法偿付时，信托公司自有资金垫付
- 或找到新投资者接盘（"发新还旧"）
- 或由关联方收购资产

为什么信托公司愿意刚兑？
- 监管要求（2014年《信托业保障基金》之前）
- 牌照价值：信托牌照稀缺，刚兑损失 < 失去牌照的风险
- 行业惯例：其他信托公司都在刚兑，不刚兑就失去客户

2018年后：打破刚兑开始
- 安信信托（2019）、四川信托（2020）等陆续违约
- 监管明确"卖者尽责，买者自负"原则
- 但部分存量产品仍存在隐性刚兑压力
```

### 2.3 私募基金

```
私募基金：
- 法律上：无任何保本承诺
- 实际上：部分管理人通过"安慰函"、"差额补足"等方式隐性刚兑

常见形式：
- 大股东承诺回购（明股实债）
- 管理人自有资金跟投（部分刚兑）
- "预期收益率"宣传（违规但普遍）
```

### 2.4 城投债与地方债务

```
城投债的隐性刚兑：

- 城投公司作为地方政府融资平台
- 市场认为：地方政府不会让城投债违约（维稳压力）
- 实际上：2018年前城投债"零实质违约"

为什么城投债享受隐性刚兑？
- 地方政府隐性担保（虽然没有法律依据）
- 银行持有大量城投债，不希望出现损失
- 违约可能引发区域性信用危机（地方官员的维稳压力）
- 再融资压力：违约后无法再发新债

→ "城投信仰"是典型的隐性刚兑
→ 2023年后城投非标违约频发，"信仰"开始动摇
```

## 三、隐性刚兑的风险

### 3.1 道德风险（核心危害）

```
投资者层面：
- 只看收益率，不看风险
- 风险定价扭曲（低风险高收益）
- "买者不负责"心态蔓延

发行机构层面：
- 为维持刚兑承担超预期风险
- 过度扩张规模（做大做强才能覆盖损失）
- 忽视底层资产质量

整体金融体系：
- 风险在体系内积累
- 最终由纳税人或央行承担
- 雷曼式"大而不能倒"问题
```

### 3.2 隐性刚兑 + 期限错配 = 危险组合

```
银行理财的"完美风暴"：

高收益（隐性刚兑）
        +
低期限（3-6个月理财）
        +
高风险（长期非标资产）
        ↓
投资者蜂拥购买
        ↓
资金持续流入 → 掩盖问题
        ↓
某产品亏损 + 市场传言
        ↓
投资者集中赎回
        ↓
理财资金池枯竭
        ↓
必须甩卖非标资产
        ↓
非标资产价格暴跌
        ↓
所有类似理财产品净值下跌
        ↓
更多赎回 → 更多甩卖 → 螺旋下跌
```

### 3.3 风险累积路径

```
第一阶段：扩张期
- 资金持续流入 → 规模快速扩张
- 亏损被新资金覆盖 → 问题不暴露
- 所有人都赚钱 → 皆大欢喜

第二阶段：压力期
- 底层资产质量恶化
- 或市场利率上升导致资产减值
- 流入资金减少

第三阶段：爆发期
- 无法覆盖到期兑付
- 机构选择：刚兑（损失自有资金）vs 违约（声誉损失+监管处罚）
- 选择刚兑 → 机构财务恶化
- 选择违约 → 市场恐慌蔓延

第四阶段：清盘期
- 机构财务无法支撑
- 监管介入或被动清盘
- 投资者最终承担损失（但往往已过多年）
```

## 四、隐性刚兑的量化规模

### 4.1 中国资管行业规模（2022）

| 产品类型 | 规模（万亿） | 隐性刚兑程度 |
|----------|-------------|-------------|
| 银行理财 | ~27 | 中高（逐步净值化） |
| 信托 | ~21 | 高（历史上刚兑普遍） |
| 券商资管 | ~7 | 中 |
| 私募基金 | ~20 | 中高（分化严重） |
| 公募基金 | ~26 | 低（净值化） |
| **合计** | **~100** | |

> 中国资产管理行业总规模约 100 万亿，其中相当部分存在不同程度的隐性刚兑预期。

### 4.2 隐性刚兑的代价

```
估算隐性刚兑累积的风险敞口：

假设银行理财规模的 5% 存在隐性风险：
27 万亿 × 5% = 1.35 万亿潜在损失

假设信托产品历史上约 2-3% 的项目存在隐性亏损：
21 万亿 × 2.5% = 0.53 万亿

加上其他资管产品、城投债务等
→ 隐性风险敞口可能达到 3-5 万亿规模

→ 这就是"灰犀牛"所在
```

## 五、打破隐性刚兑的改革

### 5.1 资管新规（2018）

| 改革 | 目的 | 效果 |
|------|------|------|
| **打破刚性兑付** | 明确禁止保本保收益承诺 | 逐步推进，部分完成 |
| **资金池转型** | 禁止滚动发行、期限错配 | 进行中 |
| **净值化** | 净值波动暴露真实风险 | 进行中 |
| **穿透式监管** | 底层资产透明化 | 部分实现 |

### 5.2 资管新规的成效与挑战

**成效**：
- 新发理财产品净值化比例大幅提升
- 资金池模式基本消除
- 刚兑预期有所弱化

**挑战**：
- 存量老产品规模仍较大
- 投资者教育滞后（很多投资者仍期望刚兑）
- 部分机构通过"摊余成本法"变相维持刚兑表象
- 净值下跌引发赎回压力（理财大规模赎回→债市波动）

### 5.3 2022-2023 年的"破刚兑"实践

| 事件 | 说明 |
|------|------|
| **2022年理财产品大面积浮亏** | 2022年11月债市调整，部分理财产品净值跌破1元，投资者大规模赎回 |
| **城投非标违约常态化** | 贵州、云南、甘肃等省出现城投技术性违约或展期 |
| **中融信托/民生信托逾期** | 房地产信托违约率高企，部分信托产品无法按期兑付 |
| **理财子公司净值化** | 成立理财子公司隔离风险，但客户期望仍需调整 |

## 六、隐性刚兑的全球比较

### 6.1 美国

| 领域 | 隐性刚兑情况 |
|------|-------------|
| **货币市场基金** | 2010年后不再"保1美元"，2020年SEC紧急恢复流动性工具 |
| **两房（房利美/房地美）** | 危机前市场认为有政府隐性担保 → 实际政府接管 → 隐性刚兑变显性 |
| **银行存款** | FDIC显性保险（25万美元），无隐性刚兑 |
| **投行/对冲基金** | 无隐性刚兑，破产风险自担 |

### 6.2 欧洲

| 领域 | 隐性刚兑情况 |
|------|-------------|
| **主权债务** | 2010年欧债危机：希腊国债被市场认为有欧盟隐性担保 |
| **银行债券** | "大而不能倒"救助 vs "内部纾困"（bail-in）改革 |
| **养老金** | 部分国家职业养老金有政府隐性担保 |

### 6.3 国际比较的核心结论

```
美国：
- 显性担保（存款保险）+ 明确的破产机制
- 隐性刚兑较少，但"大而不能倒"问题仍存在

中国：
- 显性担保有限 + 大量隐性刚兑
- 转型期阵痛：打破刚兑必然伴随风险暴露
- 城投债是最典型的隐性刚兑领域

→ 打破隐性刚兑是金融市场化改革的核心内容
→ 但改革必须渐进，避免系统性冲击
```

## 七、投资者的应对策略

### 7.1 识别隐性刚兑陷阱

| 警示信号 | 说明 |
|----------|------|
| "银行信誉担保" | 银行不应对非保本产品提供隐性担保 |
| "历史100%兑付" | 过去兑付不代表未来，刚兑正在被打破 |
| "高收益+低风险" | 高收益必然伴随高风险（无风险套利不可能持续） |
| "政府背景项目" | 政府隐性担保不等于无风险，城投债风险已在暴露 |
| "短期+高收益" | 期限越短收益越高，通常意味着底层资产风险越大 |

### 7.2 风险自担原则

```
在"卖者尽责、买者自负"原则下：

买者自负的前提是"卖者尽责"：
1. 销售机构是否充分揭示风险？
2. 风险测评是否真实反映投资者承受能力？
3. 产品风险等级是否与投资者风险偏好匹配？

买者自负的核心是"投资者自担"：
1. 了解所投资产品的底层资产
2. 评估自身风险承受能力
3. 分散投资，不押注单一高收益产品
4. 关注净值波动，而非只看历史收益
```
