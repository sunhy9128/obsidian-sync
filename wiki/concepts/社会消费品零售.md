# 社会消费品零售总额

> 国家统计局月度发布,反映国内消费需求的核心指标

**社会消费品零售总额**(简称"社零")是国家统计局每月发布的**消费需求总量指标**,涵盖**商品零售 + 餐饮收入**,反映国内最终消费需求的整体表现。

## 统计口径

| 维度 | 内容 |
|---|---|
| 范围 | 限额以上 + 限额以下企业、个体户的消费品零售 |
| 频率 | 月度(每月 15 日左右发布上月数据) |
| 同比 | 与去年同期比 |
| 环比 | 与上月比(季节性调整) |

## 关键细项

| 细项 | 关联 |
|---|---|
| **汽车** | 婚车传统 / 大宗消费 |
| 家电 | 新房配置 |
| 家具 | 新房配置 |
| 金银珠宝 | 婚庆三金 |
| 建筑装潢 | 新房装修 |
| 餐饮 | 日常消费 |
| 服装鞋帽 | 日常消费 |
| 通讯器材 | 日常消费 |

## 2026-05 数据(关键转折)

| 指标 | 数值 |
|---|---|
| 社零总额 | **41,090 亿元** |
| 同比 | **-0.6%**(3 年首次月下滑) |

| 细项 | 同比 |
|---|---|
| 汽车 | **-16%**(扎眼) |
| 家电 | -15% |
| 家具 | -8.7% |
| 金银珠宝 | -9% |
| 建筑装潢 | -13% |

> "之前有读者总结说这些都和结婚成家有关,还真是嗷,年轻人只要不结婚,就不想买房卖车买金银珠宝,更不需要家具家电装潢。"

## 解读

**核心驱动**:**不婚不育消费降级**

```
年轻人不结婚
    ↓
不买房(婚房)
    ↓
不买车(婚车)
    ↓
不买金银珠宝(婚庆三金)
    ↓
不需要家具家电装潢
    ↓
↓
汽车 -16% / 家电 -15% / 家具 -8.7% / 金银珠宝 -9% / 建筑装潢 -13%
↓
社零总额 -0.6%(3 年首次)
```

## 与 GDP 的关系

- 中国 GDP 约 55-60% 来自最终消费
- 社零是最终消费的核心代理变量
- 社零持续负增长 → 消费对 GDP 拉动减弱 → 经济整体承压
