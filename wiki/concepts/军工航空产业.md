# 军工航空产业

> 中国军工体系的核心组成——以"军工十子"为支柱、六条战线为框架的航空产业集群

## 一、核心框架：军工十子

"军工十子"是十大军工央企集团的简称，构成中国军工产业的骨架：

| 序号 | 央企集团 | 主要领域 | 核心资产 |
|------|----------|----------|----------|
| 1 | **中核集团** | 核技术 | 核武器、核动力 |
| 2 | **中国电科** | 电子信息、数据链 | 1.3万亿总资产，283亿利润（十子第一）|
| 3 | **中国电子** | 电子装备、软件 | |
| 4 | **中航工业** | 航空整机 | 资产证券化率92.5%（十子第一）|
| 5 | **航发集团** | 航空发动机 | 飞发分离（2016年）|
| 6 | **中国船舶** | 海上力量 | 南北船合并 |
| 7 | **兵器工业** | 地面武器 | |
| 8 | **兵器装备** | 轻武器、弹药 | |
| 9 | **航天科技** | 航天装备 | |
| 10 | **航天科工** | 导弹、防御 | |

> **数据来源**：Wind、上市公司财报、斯德哥尔摩国际和平研究所（SIPRI）

## 二、六条战线

### 2.1 核（第一条战线）
- **代表央企**：中核集团
- **核心价值**：军工包括航空产业**独立自主发展的基石**
- **战略意义**：避免"被拿袋洗衣粉冲进来把军工产业私有化"

### 2.2 电（第二条战线）
- **代表央企**：中国电科、中国电子
- **核心领域**：电子装备、系统软件、军用信息工程、数据链
- **战略价值**：**现代战争的关键**——空军的博弈本质是信息对抗
- **数据**：中国电科以1.3万亿总资产、283亿利润位列军工十子第一

### 2.3 航空（第三条战线）
- **代表央企**：中航工业、航发集团
- **核心业务**：航空整机设计、研发、总装
- **"飞发分离"（2016年）**：发动机与整机分开研制，打破"一厂一所一型号"旧模式

### 2.4 船舶（第四条战线）
- **代表央企**：中国船舶（南北船合并）
- **战略意图**：避免央企在国际竞标中"内斗"，拧成一股绳出海竞争

### 2.5 兵器（第五条战线）
- **代表央企**：兵器工业集团、兵器装备集团

### 2.6 航天（第六条战线）
- **代表央企**：航天科技、航天科工
- **战略价值**：空军作战中数据链的重要环节

## 三、军工产业四大基础理论

### 3.1 并购重组
- 军工十子由初期12大军工集团**拆分和并购重组**演变而来
- **近一年活跃案例**：中航成飞借壳中航电测（成飞=成都飞机工业）

### 3.2 资产证券化率
- 军工十子下属大量上市公司进入资本市场
- **航空工业资产证券化率92.5%**（军工十子第一）

```
资产证券化率 = 央企上市子公司持股资产 / 集团总资产
```

### 3.3 业绩增长
- 2019年起军工国企业绩显著增长
- **2019年定价议价规则改革**：
  - 旧："定价成本+5%利润"
  - 新："定价成本+5%目标价格+激励利润"
  - 实质：松绑，提供更乐观的议价能力和溢价空间

### 3.4 人事与订单周期
- 关键岗位人事落定→订单恢复→采购流程启动
- **观察指标**：全军装备采购信息网招标公告数量
- **规律**：2023年32周后公告数显著增加→推断人事体系落地

## 四、战机产业链

### 4.1 产业链结构

```
上游：原材料（纤维/高温合金/钛合金/隐身材料）、被动元器件、有源器件
中游：航空航发中游配套、中间工序
下游：整机制造（主机厂：成飞、沈飞、西飞）
```

### 4.2 业绩表现
- 2023年收入大幅增长，2023-2024增长放缓（人事调整影响）
- **毛利**：2022年价格调整后承压，但绝对值仍高

### 4.3 主机厂（链主）
- **定义**：负责飞机整机设计、研发、总装、集成的主导企业
- **代表**：中航沈飞、中航西飞
- **特点**：链主抗风险能力强，收入利润波动不大
- **股权激励**：中航沈飞、中航西飞2023年均发布股权激励

## 五、发动机产业链

### 5.1 国产化率
- 目前国产化率已很高，涵盖各类飞行器

### 5.2 产业链结构

```
上游：原材料、零部件制造
中游：整机制造
下游：运营、维修维护
```

### 5.3 需求侧分析

**增量侧（新增需求）**：
- 战机迭代
- 国产替代

**存量侧（维修换装）**：
- 发动机成本构成：研发10%、制造40%、**维护50%**
- F-35案例：全生命周期740亿美元，运维538亿（占2/3）

**市场规模**：约1335-1714亿元

### 5.4 维修三级体制
1. **基层级**：航空兵团机务大队
2. **中继级**：部队修理厂
3. **基地级**：航空修理工厂（大修、复杂改装）

## 六、导弹（制导炸弹）产业链

### 6.1 战略优势：效费比
- 成本优势：较巡航导弹价格便宜
- **战争本质**：不是打打杀杀，而是**消耗**
- 量大管饱才是财经视角的战争真谛

### 6.2 业绩表现
- 2020-2022增长迅猛
- 2023年起因人事调整导致需求递延
- **毛利**：平均40%+，逆势反转（定制化订单拉高利润）

## 七、军机军贸

### 7.1 出口规模（1949-2023）
- **总出口**：3751架，价值量2.15万TIV
- **TIV**（Trend Indicator Value）：武器交易价值衡量指标，消除货币波动、通胀影响

### 7.2 成飞：军贸主力
- **军贸出口价值量占总量51.19%**
- 机型：歼-7、歼-10、枭龙、歼教-5等
- 出口947架，但TIV高达1.1万（约占总价值一半）

### 7.3 军贸双子星

|------|----------|-------------|
| 成飞 | 947架 | 1.1万 |

| 西飞 | 54架 | - |
| 洪都 | 751架 | 218 |

### 7.4 主要出口机型TIV

| 机型 | 出口数量 | 总TIV | 单位TIV |

## 九、投资逻辑总结

| 赛道 | 毛利 | 特点 |
|------|------|------|
| 元器件 | 40%+ | 最赚钱 |
| 分系统 | 30%+ | |
| 部件/材料 | 30% | |
| 主机厂 | ~10% | 抗风险强，股权激励 |
| 导弹 | 40%+ | 效费比优势 |

### 核心结论
1. **主机厂**：受益于订单放量，业绩确定性最高
2. **航空发动机**：高端材料龙头受益于国产替代
3. **军贸出口**：有资质和性价比的供应商受益

## 十、81192的历史意义

### 事件回顾
- 2001年4月1日，王伟驾驶战机在海南岛东南70海里上空与美军侦察机碰撞
- 王伟牺牲，美军机迫降陵水机场
- 最终结果：美国表示遗憾，中国释放美军、交还飞机

### 产业视角
- **"如你所愿"**的真正含义：不是某件新装备的面世
- 而是**整个航空产业体系**——从原材料到整机、从维修到军贸的完整产业链
- 这个体系才是民族大规模协作的结晶

> "别人关注前半句'我已无法返航'，而我更疑惑后半句'请你们继续前进'——这里的'你们'是谁？体系中一个个设计师、工程师、上班族、操作员、保洁员，甚至是屏幕中的你并不在行业内，但合法纳税，也参与到了这场大规模协作中。"
