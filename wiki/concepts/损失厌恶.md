> 损失厌恶（Loss Aversion）指人们对**损失的感受强度**大于**等额收益的感受强度**，经验比值约为 **2:1**——失去 100 元的痛苦约等于得到 200 元的快乐。

---

## 一、核心概念

### 1.1 定义

| 维度 | 内容 |
|------|------|
| 学术定义 | 对损失的负效用强度大于对等额收益的正效用强度 |
| 提出者 | Kahneman & Tversky（1979，前景理论核心发现） |
| 经验比值 | 损失心理权重 λ ≈ 2.0~2.25 |
| 神经基础 | fMRI 证据显示损失激活更强的脑岛和纹状体 |
| 进化意义 | 对威胁的快速反应更具生存价值 |

### 1.2 数学表达

```
K-T 价值函数：
  v(x) = x^α ,        x ≥ 0   （收益域）
  v(x) = -λ·(-x)^β ,  x < 0   （损失域）

  其中 α = β ≈ 0.88，λ ≈ 2.25
  即 v(-x) ≈ -2.25 · v(x)
  
  示例：v(-100) ≈ -2.25 · v(100)
        即损失 100 元的负效用强度 ≈ 收益 225 元的正效用
```

## 二、禀赋效应（Endowment Effect）

### 2.1 概念

```
定义：一旦拥有某物，对其价值的评估会显著高于未拥有时
      愿意支付的金额（WTA > WTP）。

经典实验（Kahneman, Knetsch & Thaler, 1990）：
  - 随机给一半学生印有大学校徽的咖啡杯
  - 持杯者平均要价（WTA）：$5.25
  - 无杯者平均愿付（WTP）：$2.25~3.25
  - 比例约 2:1（与损失厌恶系数一致）
```

### 2.2 投资中的禀赋效应

| 表现 | 解释 |
|------|------|
| 长期持有"祖传股" | 主观估值偏高，不愿低价卖出 |
| 拒绝股权激励回购要约 | 公司回购价 < 投资者心理估价 |
| 装修后自评房价偏高 | 高估自住房产市场价值 |
| 收藏品主人要价虚高 | 情感因素叠加禀赋效应 |

### 2.3 减少禀赋效应

- 引入第三方客观估值
- 模拟"未拥有"情境
- 关注"机会成本"而非"沉没成本"
- 设定投资决策规则（如市盈率超过 N 倍强制卖出）

## 三、处置效应（Disposition Effect）

### 3.1 概念

```
定义：投资者倾向过早卖出盈利的"赢家"，而过久持有亏损的"输家"。

实证数据（Odean, 1998；Barber & Odean, 2001）：
  - 美国个人投资者 1991-1996 年
  - 盈利股平均持有 102 天即卖出
  - 亏损股平均持有 256 天才卖出
  - 处置效应造成年化收益损失约 1.5-3%
```

### 3.2 与损失厌恶的关联

| 决策情境 | 心理机制 | 行为后果 |
|---------|---------|---------|
| 浮盈 10% | 锁定"确定的收益" | 过早卖出，错失后续上涨 |
| 浮亏 10% | 避免"实现损失"的痛苦 | 长期持有，深套更深 |
| 综合 | 损失厌恶 + 反射效应 | "剪枝不止损" |

### 3.3 投资启示

```
应对策略：
  1. 预设计划：买入时即设定止盈止损位
  2. 重新框定：把"损失"重新框定为"机会成本"
  3. 关注税负：长期持有会失去"税损收获"（tax loss harvesting）
  4. 区分估值：盈利股贵了，亏损股便宜了，可能正是反向操作机会
```

## 四、沉没成本谬误（Sunk Cost Fallacy）

### 4.1 概念

```
定义：人们在做决策时，会考虑已经发生、不可收回的"沉没成本"，
      导致非理性地继续投入。

案例：
  - 看了一半很烂的电影，因"已经付了票钱"继续看
  - 某项目已亏损 100 万，因"前 100 万不能白花"继续追加
  - 一段糟糕的感情，因"已经付出多年"不愿放手
```

### 4.2 沉没成本 vs 损失厌恶

| 概念 | 范围 | 表现 |
|------|------|------|
| 损失厌恶 | 对未来得/失的偏好 | 损失未来 1 元 > 收益未来 1 元 |
| 沉没成本 | 对历史投入的执念 | 不可回收成本影响未来决策 |
| 关系 | 沉没成本加重损失厌恶 | 怕"白投"导致追加投入 |

### 4.3 理性决策原则

```
正确原则：沉没成本不影响未来决策
  → 决策应只考虑"未来现金流"和"机会成本"
  → 经济学 vs 心理学：
     经济学：理性人忽略沉没成本
     心理学：实际决策中难以忽略
```

## 五、损失厌恶的实验证据

### 5.1 经典实验

| 实验 | 设计 | 关键发现 |
|------|------|---------|
| 抛硬币赌局 | 输赢各 100 元 | 大多数人拒绝参与 |
| 礼品交换 | 随机分配杯 | 持杯者要价 > 无杯者愿付 |
| 框架反转 | 救 200 vs 救 600×1/3 | 选择随框架反转 |
| 标价 vs 议价 | 同一商品 | 议价卖出价 > 标价成交价 |

### 5.2 神经经济学证据

```
Tom et al. (2007) fMRI 研究：
  - 损失激活：脑岛、前扣带皮层（情绪反应）
  - 收益激活：纹状体（奖励反应）
  - 损失信号强度 ≈ 收益信号 2 倍
  
  神经层面的"不对称性"佐证心理层面的 2:1 比例
```

## 六、损失厌恶的边界

### 6.1 强度的影响因素

| 因素 | 损失厌恶程度 |
|------|-------------|
| 损失金额占财富比例 | 比例越大，损失厌恶越强 |
| 损失领域（健康/金钱） | 健康类损失厌恶更强 |
| 文化背景 | 西方 vs 东亚有差异 |
| 个体特质 | 风险偏好、经验水平 |
| 框架与情境 | 群体决策 vs 个人决策 |

### 6.2 损失厌恶弱化情形

- 长期专业训练的交易员（接近 EUT）
- 反复小额损失场景（健身卡、订阅制）
- 群体决策与责任分散
- "翻本心理"压倒损失厌恶

## 七、与其他概念的关系

### 7.1 在前景理论中的位置

```
前景理论四大支柱：
  1. 参考点依赖     → 评估起点
  2. 损失厌恶      → 价值函数斜率不对称
  3. 敏感性递减    → 价值函数曲率
  4. 概率权重      → 决策权重非线性

损失厌恶是 2 号支柱
```

### 7.2 与心理账户的关系

| 概念 | 焦点 |
|------|------|
| 损失厌恶 | 单笔得/失的不对称反应 |
| 心理账户 | 多个损益项目被分类记账 |

```
案例：  
  - 股票亏 1 万（账户 A）vs 基金赚 1 万（账户 B）
  - 投资者倾向"分账户思考" → 心理账户
  - 两个账户间的损失厌恶仍独立存在
```

## 八、应用与启示

### 8.1 营销与定价

- 免费试用的吸引力（避免"损失"框架）
- 退款保证（消除购后损失厌恶）
- 价格锚定（详见 [[锚定效应]]）

### 8.2 个人投资

| 策略 | 应对损失厌恶 |
|------|-------------|
| 定投策略 | 平滑波动，避免"亏损时恐慌卖出" |
| 资产配置 | 分散化降低单一损失冲击 |
| 长期持有 | 让短期波动"沉没" |
| 复盘记录 | 用规则代替情绪 |

### 8.3 制度设计

- 401(k) 默认参与（避免"选择"带来的损失厌恶）
- 保险产品（损失厌恶使保险需求被高估）
- 行为助推（Nudge）设计
