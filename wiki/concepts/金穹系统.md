# 金穹系统

> 美国拟建的全国性导弹防御系统（Golden Dome for America），重塑全球战略稳定格局。

## 概述

"金穹"（Golden Dome）是美国拟议中的下一代全国导弹防御系统，由特朗普政府推动，旨在实现对美国全境的"全方位导弹防御覆盖"——包括拦截洲际弹道导弹（ICBM）、高超音速导弹和巡航导弹。其名称呼应里根时代的"星球大战计划"（SDI）。

## 技术特征

| 维度 | 描述 |
|------|------|
| 防御范围 | 美国全境（区别于现有的区域防御） |
| 拦截对象 | ICBM、高超音速导弹、巡航导弹 |
| 核心技术 | 天基拦截器（Space-Based Interceptors）、天基传感器星座 |
| 现有基础 | 陆基中段防御（GMD）、萨德（THAAD）、爱国者（PAC-3） |
| 争议焦点 | 外空武器化、反导与进攻性武器界限模糊 |

## 战略争议

### 中俄的立场（2026年5月中俄联合声明）

- 谴责"金穹全球反导"系统
- 明确反对"外空武器化、先发制人打击/斩首"
- 反对"美国天基拦截、外空作战疆域"概念
- 重申"不首先在外空部署武器"

### 核心问题

1. **战略稳定破坏：** 金穹系统理论上可削弱他国的核威慑能力，打破"相互确保摧毁"（MAD）平衡
2. **外空军事化：** 天基拦截器意味着外空将成为作战疆域
3. **军备竞赛螺旋：** 刺激对手发展更先进的突防技术（高超音速、多弹头分导、诱饵）
4. **美俄核军控解体：** 新削减战略武器条约（New START）到期后缺乏替代框架

## 历史脉络

| 时期 | 系统 | 状态 |
|------|------|------|
| 1983年 | 星球大战计划（SDI） | 因技术不可行、成本太高终止 |
| 2002年 | 退出ABM条约 → 部署陆基中段防御 | 美国首次拥有有限NMD能力 |
| 2019年 | 退出INF条约 | 中导竞赛重启 |
| 2024年 | "金穹"概念提出 | 仍在概念/研究阶段 |

## 对全球格局的影响

1. **中美俄三角：** 金穹加剧了中俄的安全焦虑，成为中俄联合声明的核心关切之一
2. **核裁军前景：** 美国发展全国反导系统使核裁军更难推进
3. **盟友关系：** 欧洲和亚太盟友担心美国"防御单边主义"削弱延伸威慑的可信度
4. **技术竞赛：** 引发高超音速武器、反卫星武器、定向能武器等新一轮军备竞赛
