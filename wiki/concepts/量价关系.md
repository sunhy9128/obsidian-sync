# 量价关系

> 技术分析的核心：价格与成交量的相互验证

**量价关系**是技术分析中最基础也最重要的分析维度之一。成交量（量）是市场供需的直接体现，价格（价）是供需平衡的结果。量价结合分析能验证趋势可靠性、发现潜在反转、识别主力行为。

---

## 一、量价基本原理

### 1.1 核心理念

[[道氏理论]]明确提出：**成交量验证趋势**。这是量价分析的理论基石。

```
量先行于价：成交量变化往往领先于价格变化
量增价涨 = 趋势确认
量缩价涨 = 趋势可疑（顶背离预警）
```

### 1.2 量的本质含义

| 成交量 | 含义 |
|--------|------|
| 高成交 | 市场参与度高，方向性行情的概率大 |
| 低成交 | 市场参与度低，可能横盘或犹豫 |
| 天量 | 历史级别的参与度，往往对应顶部或底部 |
| 地量 | 历史级别的冷清，往往对应底部区间（抛压枯竭） |

## 二、经典量价形态

### 2.1 趋势中的量价关系

| 形态 | 量价组合 | 含义 | 操作 |
|------|----------|------|------|
| **量增价涨** | 成交量放大 + 价格上升 | 趋势健康，多头主导 | 持仓/顺势加仓 |
| **量增价跌** | 成交量放大 + 价格下跌 | 空头主导，下跌趋势 | 观望/做空 |
| **量缩价涨** | 成交量缩减 + 价格上升 | 上涨乏力，买盘不足 | 减仓/止盈 |
| **量缩价跌** | 成交量缩减 + 价格下跌 | 正常调整，抛压衰竭 | 关注底部信号 |
| **量平价稳** | 成交量平稳 + 价格横盘 | 观望等待，突破在即 | 等方向明确 |

### 2.2 底部量价形态

| 形态 | 特征 | 信号强度 | 说明 |
|------|------|----------|------|
| **地量见地价** | 成交量历史新低 + 价格低位 | ★★★★★ | 最经典的底部信号 |
| **底部堆量** | 低位成交量持续温和放大 | ★★★★ | 主力低位吸筹 |
| **缩量止跌** | 下跌中成交量快速萎缩 | ★★★ | 抛压枯竭 |
| **放量长下影** | 低位大成交量 + 长下影线 | ★★★★ | 有资金承接 |
| **二次探底量能萎缩** | 第二次回踩低点量能更小 | ★★★★★ | 双底确认信号 |

### 2.3 顶部量价形态

| 形态 | 特征 | 信号强度 | 说明 |
|------|------|----------|------|
| **天量见天价** | 成交量历史新高 + 价格高位 | ★★★★★ | 最经典的顶部信号 |
| **放量滞涨** | 成交量大 + 价格涨速放缓/不涨 | ★★★★ | 多头力量衰竭 |
| **高位量价背离** | 价格新高 + 成交量未跟上 | ★★★★ | 顶背离 |
| **拉高放量出货** | 快速拉抬后成交量暴增 | ★★★★★ | 主力派发尾声 |
| **高位长上影** | 高位大成交量 + 长上影线 | ★★★★ | 上方抛压重 |

### 2.4 突破量价形态

| 形态 | 特征 | 信号强度 | 说明 |
|------|------|----------|------|
| **放量突破** | 成交量放大 + 价格突破关键位 | ★★★★★ | 最可靠的突破信号 |
| **缩量突破** | 成交量不足 + 价格突破 | ★★ | 假突破风险大 |
| **突破后回踩缩量** | 突破后缩量回踩关键位 | ★★★★★ | 最强入场信号 |
| **缩量整理后放量** | 横盘缩量 + 突然放量 | ★★★★★ | 爆发前兆 |

## 三、A 股特色量价现象

### 3.1 涨停板量价

涨停板是 A 股独特的制度现象，配套的量价分析也很特殊：

| 形态 | 量价特征 | 含义 | 风险 |
|------|----------|------|------|
| **一字板** | 无成交量 + 无换手 | 极强情绪，一字到顶 | 打开即出货 |
| **T 字板** | 低成交量打开 + 快速回封 | 次强，强势延续 | 关注封单量变化 |
| **实体板（量增）** | 成交量放大 + 阳线实体 | 健康上涨 | 正常 |
| **实体板（量缩）** | 缩量涨停 + 实体阳线 | 筹码稳定，持股待涨 | 正常 |
| **烂板** | 涨停打开多次 + 成交量大 | 分歧极大，谨慎 | 高位烂板 = 出货 |
| **放量炸板** | 涨停打开 + 成交量暴增 | 主力出货 | 极度危险 |

### 3.2 换手率分析

| 换手率区间 | 市场含义 | 策略含义 |
|-----------|----------|----------|
| < 1% | 极度冷清 | 可能见底（地量见地价），也可能持续弱势 |
| 1-3% | 正常水平 | 无明显信号 |
| 3-5% | 活跃 | 个股交投活跃，有资金参与 |
| 5-10% | 过热 | 短线筹码交换剧烈，注意高位风险 |
| > 10% | 极度过热 | 筹码极度不稳定，随时可能反转 |

**A 股换手率与行情对照**：
- 2015 年 6 月：全市场换手率 > 15%，随后崩盘
- 2018 年 12 月：全市场换手率跌至 ~1%，随后 2019 年初大反弹
- 2024 年：市场换手率持续低迷（~2%），结构性行情为主

### 3.3 量比分析

量比 = 当日即时成交量 / 过去 5 日平均成交量

| 量比 | 含义 |
|------|------|
| < 0.5 | 极度缩量，观望 |
| 0.5-1.0 | 正常偏低 |
| 1.0-2.0 | 正常放量 |
| 2.0-5.0 | 明显放量 |
| > 5.0 | 异常放量，结合位置判断 |

**量比实战用法**：
- 量比 > 3 的个股需特别关注：可能是启动信号，也可能是出货
- 开盘 30 分钟量比 > 5 → 当日大概率有异动
- 跌停股量比 > 3 → 恐慌抛售，可能有抄底机会（但风险极大）

## 四、量价背离

量价背离是识别趋势衰竭的核心工具。

### 4.1 顶背离（量价背离）

```
形态：价格创出新高，但成交量未创出新高（相比前一波上涨）
含义：价格上涨但参与资金减少 = 趋势可能见顶
信号强度：★★★★☆
出现场景：常见于牛市末期、板块炒作尾声、个股高位派发
```

### 4.2 底背离（量价背离）

```
形态：价格创出新低，但成交量未创出新低（相比前一波下跌）
含义：价格下跌但抛压已减少 = 主力可能在吸筹
信号强度：★★★★☆
出现场景：常见于下跌末期、底部区域、主力建仓阶段
```

### 4.3 背离的级别与确认

| 背离级别 | 说明 | 可靠性 |
|----------|------|--------|
| 日线级背离 | 日 K 线图上的量价背离 | ★★★★，主要参考 |
| 周线级背离 | 周 K 线图上的量价背离 | ★★★★★，大级别趋势信号 |
| 分钟级背离 | 60 分钟/30 分钟图上的背离 | ★★，噪音多，辅助参考 |
| 二次背离 | 背离后出现再次背离 | ★★★，更确定但信号滞后 |
| 三次背离 | 极度罕见 | ★★★★，极强信号但可能已经错过大部分行情 |

## 五、量价与趋势周期

### 5.1 牛市的量价特征

| 阶段 | 价格 | 成交量 | 量价含义 |
|------|------|--------|----------|
| 吸筹期 | 底部震荡 | 温和放大 | 主力逐步建仓 |
| 启动期 | 突破关键位 | 明显放量 | 趋势确认 |
| 主升期 | 持续上涨 | 量价配合 | 健康上涨 |
| 赶顶期 | 高位加速 | 天量 + 价滞 | 主力出货尾声 |
| 顶部 | 滞涨 | 成交高位 | 趋势反转 |

### 5.2 熊市的量价特征

| 阶段 | 价格 | 成交量 | 量价含义 |
|------|------|--------|----------|
| 顶部派发 | 高位震荡 | 成交高位 | 主力出货 |
| 主跌期 | 持续下跌 | 量增价跌 | 恐慌抛售 |
| 赶底期 | 快速急跌 | 天量 + 价跌 | 最后恐慌（往往见底） |
| 底部 | 低位震荡 | 地量 | 抛压枯竭 |
| 吸筹期 | 底部抬升 | 温和放大 | 主力重新建仓 |

> **关键规律**：牛市末端的"赶顶量"和熊市末端的"赶底量"是天量/地量规律的最重要应用。

## 六、量价与主力行为

### 6.1 主力吸筹建仓的量价特征

| 阶段 | 成交量 | 价格 |
|------|--------|------|
| 试探建仓 | 间歇性放量 | 小幅上涨 |
| 压盘吸筹 | 缩量打压 | 小幅下跌 |
| 拉抬试盘 | 放量拉升 | 快速上涨后回落 |
| 正式建仓 | 放量拉升 | 持续推高 |
| 锁仓 | 缩量 | 价格稳定上行 |

### 6.2 主力出货的量价特征

| 阶段 | 成交量 | 价格 |
|------|--------|------|
| 出货初期 | 放量 | 高位震荡 |
| 拉高出货 | 暴量拉升 | 快速冲高 |
| 派发 | 高位放量 | 滞涨 |
| 砸盘出货 | 天量 | 快速下跌 |

## 七、量价分析与指标配合

| 指标 | 与量价配合 | 作用 |
|------|-----------|------|
| **MACD** | 量价背离 + MACD 背离 | 最强趋势衰竭信号 |
| **均线** | 放量突破均线 + 回踩缩量确认 | 趋势启动 / 确认 |
| **BOLL** | 放量突破 BOLL 上轨 | 突破确认 |
| **KDJ** | 高位放量 + KDJ 死叉 | 卖出信号 |
| **DDX** | 放量上涨 + DDX 飘红 | 主力确认买入 |
