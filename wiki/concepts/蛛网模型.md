# 蛛网模型

蛛网模型（Cobweb Model）是1930年代由**尼古拉斯·卡尔多（Nicholas Kaldor）** 提出的微观经济学模型，用于解释**生产具有时滞的商品**（尤其是农产品）的价格-产量周期性波动。

> **核心命题**：当供给响应滞后于价格信号时，市场不会自动出清，而是形成持续的"价格-产量"循环——这是农产品周期波动的底层数学结构。

## 一、模型假设

蛛网模型有三个关键假设：

### 1. 完全竞争市场
生产者数量众多，单个生产者无法影响市场价格，只能作为"价格接受者"。

### 2. 生产具有时滞
本期产量由**上一期价格**决定，而非本期价格。即：

```
Q_t^s = f(P_{t-1})
```

这是蛛网模型与传统供需均衡的核心区别——**生产者用上期价格信号做本期生产决策**。

### 3. 即期市场出清
**本期价格**由**本期需求**与**本期供给**（即上期决策的产量）决定：

```
Q_t^d = g(P_t)
P_t 由 Q_t^s = Q_t^d 决定
```

## 二、三种蛛网类型

蛛网模型的收敛性由**供给弹性**与**需求弹性**的相对大小决定：

### 1. 收敛型蛛网（Stable Cobweb）
- **条件**：供给弹性 < 需求弹性
- **结果**：波动幅度逐期缩小，最终收敛到均衡
- **现实**：极少数情况；工业品、标准化产品可能接近

```
P ↑     P₂
P₁ \    /
    \  /
     \/
     /\
    /  \
P_e —————————→ t

波动幅度逐期减小 → 收敛
```

### 2. 持续型蛛网（Perpetual Cobweb）
- **条件**：供给弹性 = 需求弹性
- **结果**：波动幅度保持不变，持续围绕均衡震荡
- **现实**：理论边界情况

### 3. 发散型蛛网（Explosive Cobweb）
- **条件**：供给弹性 > 需求弹性
- **结果**：波动幅度逐期扩大，最终远离均衡
- **现实**：**农业领域最常见**——生产具有时滞+生产者预期价格弹性高

```
P ↑      P₃
       /
P₁ \  /
    \/
    /\
   /  \
  /    P₂
P_e —————————→ t

波动幅度逐期增大 → 发散
```

## 三、为什么农产品普遍是发散型蛛网

农产品（生猪、粮食、蔬菜）满足蛛网模型所有发散条件：

| 条件 | 农产品 | 工业品 |
|------|--------|--------|
| 生产时滞 | 长（数月-数年） | 短（数天） |
| 供给弹性 | 大（产能扩张快） | 小（产能固定） |
| 需求弹性 | 小（刚需） | 大（可替代） |
| 蛛网类型 | **发散型** | 接近收敛型 |

**关键洞察**：农产品的"刚需"属性让需求弹性小（价格涨了也得买），而"生产时滞"属性让供给弹性大（一看到涨价就扩产）——这就是发散型蛛网的温床。

## 四、中国猪周期是发散型蛛网的极致案例

中国生猪养殖完美匹配蛛网模型的所有假设：

| 蛛网假设 | 中国猪养殖 |
|----------|------------|
| 生产时滞 | 母猪妊娠4个月+育肥6个月=10个月 |
| 完全竞争 | 散户众多，规模化程度有限 |
| 即期出清 | 现货市场即时交易 |
| 供给弹性 | 极高（母猪存栏可快速扩张） |
| 需求弹性 | 极低（猪肉刚需） |

这导致中国猪周期**振幅持续放大**——每一轮高价刺激的产能，下轮变成过剩；过剩淘汰的产能，下轮又变成短缺。详见 [[猪周期]]。

## 五、模型的局限性

蛛网模型虽然简洁，但有若干局限：

### 1. 理性预期假设过强
模型假设生产者只看上一期价格做决策。现实中，生产者会**形成预期**——预测下期价格而非机械跟随上期。理性预期学派认为这会让蛛网模型失效。

### 2. 完全竞争假设不现实
农产品市场存在大量规模化企业（如牧原、温氏），他们的决策更接近**古诺模型**而非完全竞争。

### 3. 政府干预被忽视
现实中政府通过储备、补贴、期货等工具干预农产品市场（见 [[猪周期]] 第六节），平滑蛛网波动。

### 4. 多产品联动被忽视
实际农产品之间存在**替代/互补关系**（猪肉与鸡肉、玉米与豆粕），单产品模型不够完整。

## 六、扩展与衍生模型

### 1. 理性预期蛛网模型
引入"前瞻性预期"，生产者用**对下期价格的预期**而非上期价格做决策。
- 结论：在某些参数下蛛网不出现，但现实中"信息不完全"和"学习成本"让它依然存在。

### 2. 多产品蛛网模型
考虑多个相互关联的产品（如猪肉-鸡肉-牛肉）。
- 结论：蛛网模型会"传染"——一个产品的波动会带动其他产品。

### 3. 库存蛛网模型
引入库存变量，生产者可在高价时储存、低价时抛售。
- 结论：库存可以**收敛蛛网**（平抑波动）——这是国家粮食储备的理论基础（见 [[战略储备]]）。

### 4. 期货蛛网模型
引入期货价格作为远期信号。
- 结论：期货价格发现功能可以**部分替代**上期价格信号，让蛛网模型从"机械跟随"转为"前瞻决策"。
- 这就是为什么2019年生猪期货上市后，被认为是平滑猪周期的工具之一。

## 七、政策启示

理解了蛛网模型，就能理解农产品政策的三层设计：

| 层级 | 工具 | 目标 | 案例 |
|------|------|------|------|
| **产能引导** | 能繁母猪预警、规模化补贴 | 平滑供给弹性 | 农业农村部月度预警 |
| **价格对冲** | 期货、套保 | 给生产者前瞻信号 | 生猪期货（2019） |
| **库存调节** | 战略储备、抛储 | 平滑蛛网振幅 | 猪肉储备调节机制 |

> **核心结论**：蛛网模型在理论上是发散的，但**通过产能引导、价格对冲、库存调节三层工具**，可以让实际波动收敛到合理范围。这是"逆周期调控"的微观经济学基础。
