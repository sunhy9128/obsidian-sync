# 信用利差

> 信用风险在债券定价中的具体体现

**信用利差**（Credit Spread）是投资者因承担发行人的信用风险而要求获得的额外收益。信用利差 = 信用债收益率 - 同期限国债收益率（无风险利率基准）。

---

## 一、基本原理

### 1.1 公式与含义

```
信用利差 = 信用债收益率 - 国债收益率（同期限）

例：5 年期 AA 企业债收益率 3.5%，5 年期国债收益率 2.8%
→ 信用利差 = 70 bps（0.7%）
```

**利差的经济学含义**：
- 对应发行人的**违约概率**（PD）和**违约损失率**（LGD）
- 信用利差 ≈ PD × LGD × 风险厌恶系数
- 信用利差是市场对信用风险的"定价"

### 1.2 信用利差 vs 无风险利率

债券市场定价可分解为：

```
信用债收益率 = 无风险收益率 + 信用利差
             = 国债收益率 + 信用利差
```

| 情况 | 含义 |
|------|------|
| 信用利差扩大 | 市场对信用风险担忧上升，或流动性收紧 |
| 信用利差收窄 | 信用环境改善，风险偏好上升 |
| 利差为负（异常） | 通常出现在极端流动性冲击中（2008 年、2022 年中） |

---

## 二、信用利差的构成

### 2.1 三大组成部分

| 成分 | 来源 | 说明 |
|------|------|------|
| **违约风险溢价** | 投资者要求的信用风险补偿 | 核心成分，随违约概率变化 |
| **流动性溢价** | 信用债流动性弱于国债 | 低评级、信用债市场深度不足时溢价更高 |
| **税收溢价** | 企业债利息需缴税（免税条款差异） | 政策债/国债免税，信用债不免税 |
| **凸性差异** | 不同债券对利率变化的敏感度差异 | 信用债与国债久期/凸性不同带来的相对价格调整 |

> 实际利差 ≈ 违约风险溢价 + 流动性溢价 + 税收调整 + 凸性差异

### 2.2 投资级与高收益级利差

按评级分层，信用利差可分为两大类型，其波动特征与经济周期相关性差异显著：

| 类型 | 评级范围 | 波动性 | 与经济周期相关性 | 主要反映 |
|------|----------|--------|----------------|----------|
| **投资级信用利差（IG Spread）** | BBB- 及以上 | 较小 | 中等相关性 | 主要反映流动性风险 |
| **高收益信用利差（HY Spread）** | BB+ 及以下 | 较大 | 高度相关 | 主要反映违约风险 |

### 2.3 历史利差统计

| 评级 | 投资级正常区间 | 高收益级正常区间 | 极端情况（危机） |
|------|--------------|----------------|-----------------|
| AAA | 30-80 bps | — | > 200 bps |
| AA | 50-120 bps | — | > 300 bps |
| A | 80-180 bps | — | > 400 bps |
| BBB | 120-250 bps | — | > 600 bps |
| BB | — | 300-600 bps | > 1000 bps |
| B | — | 500-1000 bps | > 2000 bps |

## 三、信用利差的驱动因素

### 3.1 宏观因素

| 因素 | 对利差的影响 | 逻辑 |
|------|-------------|------|
| **经济增长向上** | 利差收窄 | 企业盈利改善 → 违约概率下降 |
| **经济衰退** | 利差扩大 | 企业现金流恶化 → 违约概率上升 |
| **货币政策宽松** | 利差收窄 | 资金成本下降 → 再融资压力减轻 |
| **货币政策收紧** | 利差扩大 | 融资成本上升 → 低资质企业再融资困难 |
| **通胀预期上升** | 利差扩大 | 实际偿债能力下降 |
| **汇率贬值** | 利差扩大（外币债） | 汇率损失叠加信用风险 |

### 3.2 行业因素

| 行业特征 | 利差水平 | 说明 |
|----------|----------|------|
| **强周期行业**（钢铁、煤炭） | 高利差 | 盈利随宏观大幅波动，违约风险高 |
| **弱周期行业**（公用事业、医药） | 低利差 | 需求稳定，现金流可预期 |
| **高杠杆行业**（地产、金融） | 高利差 | 对利率和现金流变化极敏感 |
| **政策支持行业**（新能源、半导体） | 中低利差 | 政策背书降低隐性违约风险 |

### 3.3 机构行为因素

| 行为 | 对利差的影响 |
|------|-------------|
| **机构配置盘涌入** | 收窄（供不应求） |
| **基金大规模赎回** | 扩大（被动抛售） |
| **外资增配中国信用债** | 收窄 |
| **城投信仰强化** | 城投债利差系统性收窄 |
| **城投信仰弱化** | 城投债利差扩大 |

## 四、信用利差与经济周期

### 4.1 经典利差周期

```
经济衰退 → 违约概率上升 → 信用利差扩大（高收益债利差更敏感）
↓
政策宽松 → 融资成本下降 → 企业现金流改善
↓
经济复苏 → 违约概率下降 → 信用利差收窄
↓
经济过热 → 通胀担忧 → 央行收紧 → 利差再次扩大
```

**扩张期特征**：经济向好 → 企业盈利改善 → 违约率下降；信用利差收窄、风险偏好上升；信用债表现优于国债。

**衰退期特征**：经济下行 → 企业盈利恶化 → 违约率上升；信用利差扩大、风险偏好下降；信用债大幅跑输国债。

**利差与经济周期的时滞关系**：

| 领先指标 | 领先利差的时期 |
|----------|--------------|
| PMI 触底 | 利差通常滞后 3-6 个月见顶 |
| 社融增速触底 | 利差滞后约 3 个月见顶 |
| 信用债净融资触底 | 利差同步或略微领先 |

### 4.2 利差扩张的预警信号

| 预警信号 | 说明 |
|----------|------|
| **投资级利差率先扩大** | 经济放缓信号（IG 利差通常比 HY 更早反应宏观拐点） |
| **高收益利差急剧扩大** | 衰退风险上升 |
| **BBB 级利差扩大** | 投资级下限被侵蚀，市场最警惕的信号 |
| **行业利差分化加剧** | 不是系统性而是结构性问题——影响有限 |
| **城投利差突然扩大** | 城投信仰弱化（2023 年贵州等地区已出现） |
| **点心债利差扩大** | 离岸市场对人民币资产信心下降 |
| ** CDS（信用违约互换）利差扩大** | 机构对冲工具反映的市场预期 |
| **利差曲线倒挂（HY 利差低于 IG）** | 异常现象，垃圾债相对投资级更具吸引力的极端定价 |

## 五、中国信用债市场与城投利差

### 5.1 中国信用债评级体系

| 评级序列 | 说明 |
|----------|------|
| AAA | 最高信用，极低违约风险 |
| AA+ / AA / AA- | 较高信用 |
| A+ / A / A- | 中等信用 |
| BBB+ 及以下 | 投资级下限，违约风险显著上升 |

> **A 股特色**：中国评级普遍偏高——大量实际资质一般的发行人被给予 AA 甚至 AAA 评级，造成评级失真。实际操作中需结合财务数据和隐性担保判断。

### 5.2 城投债利差特殊性

城投债是中国的"准国债"——市场普遍认为地方政府会兜底隐性债务：

| 现象 | 市场解读 |
|------|----------|
| 城投债利差系统性低于同评级产业债 | 城投信仰，隐性担保 |
| 区县级城投 > 地市级 > 省级 | 行政层级越高，担保越强 |
| 网红地区（贵州/云南/天津）城投利差 > 全国均值 | 债务压力重，信仰动摇 |
| 城投债"化债"政策出台 | 利差收窄（政策背书强化） |

**城投利差的定价逻辑正在发生变化**：
- 2023 年起，中央明确"一揽子化债"，城投信仰得到政策背书
- 但"打破城投刚兑"的长期方向不变——城投利差不会永久压低
- 投资级城投债本质上是**政策利率债**而非真正信用债

### 5.3 信用利差在 A 股债券配置中的应用

| 策略 | 逻辑 |
|------|------|
| **利差收窄期配置信用债** | 信用环境好 + 流动性充裕 → 加久期 / 加杠杆 |
| **利差扩大初期减配低评级** | 高收益债利差率先扩大 → 信用风险暴露 |
| **利差极端扩大后反转机会** | 利差极高 = 违约风险已被定价 = 超调 → 优质信用债机会 |
| **城投债利差收窄配置** | 政策宽松 + 化债预期 = 城投债系统性机会 |

## 六、信用利差与信用违约

### 6.1 违约对利差的非线性影响

| 违约事件 | 利差反应 |
|----------|----------|
| **单体违约（小型企业）** | 影响范围小，相关行业利差扩大 |
| **行业性违约（煤炭/光伏）** | 行业内利差系统性扩大 |
| **系统性违约（经济危机）** | 全市场利差急剧扩张 |
| **隐性违约（技术性违约）** | 短期冲击后利差快速恢复 |

### 6.2 信用利差作为先行指标

| 信号 | 含义 | 历史上类似情况 |
|------|------|---------------|
| 利差从低点快速扩大 | 违约概率上升预警 | 2014-2016 年民企违约潮、2018 年去杠杆民企危机 |
| BBB 利差突破 200 bps | 投资级信心动摇 | 2020 年 3 月（疫情冲击） |
| 高收益债利差突破 1000 bps | 垃圾债市场失血 | 2008 年金融危机（超过 1500 bps） |
