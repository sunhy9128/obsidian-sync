# 逆向选择

> 在签约前，掌握私人信息的一方利用信息优势做出对自己有利而对对方不利的选择，导致市场参与者质量下降、配置效率受损。

由 AI 系统化整理（2026-06-03），待补充原始来源素材。

逆向选择（Adverse Selection）是信息不对称在**签约前**（ex-ante）造成的市场失灵——是金融、保险、劳动力、二手市场普遍存在的现象。Akerlof 1970 年的"柠檬市场"论文是理论起点。

---

## 一、核心概念

### 1.1 定义

```
签约前
├─ 一方拥有私人信息（类型、质量、风险）
├─ 另一方只能基于平均估计决策
├─ 拥有信息的一方策略性选择（高质量者退出，低质量者涌入）
└─ 结果：市场参与者结构恶化，效率损失
```

### 1.2 与道德风险的对比

| 维度 | 逆向选择 | 道德风险 |
|------|----------|----------|
| 时点 | 签约前 | 签约后 |
| 隐藏内容 | 类型/质量 | 行为/努力 |
| 原因 | 信息分布不均 | 信息不可观察 |
| 典型案例 | 二手车市场 | 保险理赔 |
| 解决机制 | 信号、筛选、披露 | 激励、监督、惩罚 |

### 1.3 关键特征

- **自我强化**：高质量者退出 → 平均质量下降 → 价格下降 → 更多高质量退出
- **市场萎缩**：极端情况下市场完全消失
- **配置无效率**：低质量产品挤占高质量产品的市场份额
- **机会主义**：拥有信息优势方利用信息获利

## 二、Akerlof 二手车市场模型

### 2.1 模型设定

```
市场参与者：
├─ 卖方：N 个，车的真实质量 q 私有
├─ 买方：M 个，无法观察 q，只能看到价格
└─ 质量分布：q ~ F(μ, σ²) 已知分布，但个体不可知
```

### 2.2 均衡分析

```
买方愿出价（基于预期质量）：
P(q̄) = E[q | q̄] · V

其中 q̄ = 市场平均质量，V = 单位质量估值

卖方决策：
├─ 卖，当且仅当 P(q̄) ≥ qᵢ（车真实质量）
├─ 高质量车（q 大）：P(q̄) < q → 退出市场
└─ 低质量车（q 小）：P(q̄) ≥ q → 进入市场
```

### 2.3 自我强化机制

```
初始：质量均值 q̄₁，价格 P₁
      ↓ 高质量车退出
新均衡：质量均值 q̄₂ < q̄₁
      ↓ 价格 P₂ = P(q̄₂) < P₁
      ↓ 进一步恶化
... 恶性循环 ...
可能结果：
├─ 部分市场存在（低 q̄ 均衡）
└─ 市场完全消失（P → 0 时）
```

### 2.4 形式化解

```
设 λ = 卖家中高质量占比
市场存在条件：λ* ≥ λ_min

λ* 由下式决定（市场出清）：
P(λ*) = λ*·qH + (1-λ*)·qL
且 P(λ*) ≥ qH（高质量车主愿卖）

→ λ_min = (qH - qL) / (qH - qL) · 临界值
```

## 三、健康保险市场

### 3.1 信息不对称

```
保险公司（信息少）：
├─ 平均估计投保人健康状况
├─ 平均估计投保人风险偏好
└─ 平均估计投保人防范行为

投保人（信息多）：
├─ 自己真实健康状况
├─ 家族病史
├─ 高风险运动爱好
└─ 已有疾病（慢性病）
```

### 3.2 逆向选择过程

```
保险公司提供统一保费 P
├─ 健康人（低风险）：P > 自己期望赔付 → 不买
├─ 高风险者（高风险）：P < 自己期望赔付 → 抢着买
└─ 投保人池风险上升 → 公司亏损 → 提高 P
   → 进一步驱逐健康人 → 死亡螺旋
```

### 3.3 死亡螺旋 (Death Spiral)

```
                健康人退出
                     ↑
统一保费 P 偏高 → 池中风险上升 → 赔付率上升
                     ↓
              保险公司提价 → 更多健康人退出
                     ↓
                池中风险进一步上升
                     ↓
                 ... 直至市场崩溃
```

### 3.4 解决方案

| 机制 | 原理 | 实例 |
|------|------|------|
| 风险分级 | 按风险差异化定价 | 车险按年龄/驾龄 |
| 强制投保 | 所有人必须买 | 交强险、社保 |
| 健康告知 | 强制信息披露 | 投保单如实告知 |
| 等待期 | 抑制带病投保 | 90 天等待期 |
| 体检 | 第三方验证 | 寿险体检 |
| 团体保险 | 集体投保摊薄 | 企业团体险 |

## 四、金融市场

### 4.1 信贷市场

```
银行（信息少）：
├─ 不了解借款人真实风险
├─ 不了解项目真实质量
└─ 借款人可能粉饰报表

借款申请者（信息多）：
├─ 知道自己真实还款能力
├─ 知道项目风险
└─ 高风险者最积极借款
```

**逆向选择结果**：
- 高风险项目挤走低风险项目
- 银行要么提利率应对，要么惜贷
- 信贷配给（Stiglitz-Weiss 模型）：即使项目 NPV > 0 也可能拒贷

### 4.2 资本市场

| 现象 | 逆向选择表现 |
|------|--------------|
| IPO 折价 | 内部人利用信息差，新股折价发行 |
| 垃圾债 | 高风险企业发债获得更高溢价 |
| 资产证券化 | 资产质量被低估，优质资产不愿证券化 |
| 杠杆收购 | 高杠杆公司可能藏着风险 |

### 4.3 解决方案

| 机制 | 工具 |
|------|------|
| 信息披露 | 财报、季报、重大事项公告 |
| 信用评级 | 标普、穆迪、惠誉 |
| 抵押担保 | 不动产、设备抵押 |
| 关系贷款 | 长期客户关系软信息 |
| 金融科技 | 大数据风控、替代数据 |
| 政府担保 | 中小企业贷款风险补偿 |

## 五、劳动力市场

### 5.1 高质量工人被低估

```
雇主无法区分：
├─ 高能力（高保留工资）
├─ 中能力
└─ 低能力（低保留工资）

若雇主只能给平均工资：
├─ 高能力者：平均工资 < 自己价值 → 退出
├─ 低能力者：平均工资 > 自己价值 → 涌入
└─ 平均能力下降 → 雇主进一步压低工资
```

### 5.2 信号机制（Spence 1973）

```
教育作为信号：
├─ 高能力者：教育成本 c_low → 值得获取学位
├─ 低能力者：教育成本 c_high → 不值得获取学位
└─ 雇主观察教育 → 推断能力 → 差异化工资

→ 即使教育不增人力资本，也能起信号作用
```

### 5.3 筛选机制

```
雇主设计合同：
├─ 不同工资水平
├─ 不同培训/试用安排
├─ 不同福利组合
└─ 让不同类型员工自选 → 揭示真实类型
```

## 六、其他典型场景

### 6.1 公共品供给

```
项目质量不透明：
├─ 政府招标（招标方信息少）
├─ 承包商知道真实能力
└─ 低能力承包商压低报价抢标 → 豆腐渣工程
```

### 6.2 医疗市场

- 患者比医生更了解症状（但医生有专业判断）
- 高需求患者积极就诊，医生收费上升
- 医患信息不对称是医疗市场特殊性的根源

### 6.3 婚恋市场

- 双方关于性格、品行、婚史的信息不对称
- "剩女剩男"现象可部分由逆向选择解释
- 婚介平台通过认证、背书、推荐缓解

### 6.4 选举投票

- 候选人比选民更了解自己的真实能力
- 选民通过承诺、履历、辩论筛选

## 七、应对机制对比

| 机制 | 适用场景 | 优点 | 局限 |
|------|----------|------|------|
| 信号传递 | 知情方主动揭示 | 成本低 | 可能伪造 |
| 筛选机制 | 不知情方设计菜单 | 可信 | 设计复杂 |
| 强制披露 | 监管要求 | 普遍适用 | 形式合规 |
| 中介/评级 | 第三方评估 | 专业化 | 中介失灵 |
| 标准化合同 | 标准化产品 | 简单 | 失去弹性 |
| 长期关系 | 重复交易 | 自然筛选 | 范围有限 |

## 八、理论意义

### 8.1 学术地位

| 贡献 | 学者 | 年份 |
|------|------|------|
| 柠檬市场 | Akerlof | 1970 |
| 信号理论 | Spence | 1973 |
| 信贷配给 | Stiglitz & Weiss | 1981 |
| Rothschild-Stiglitz 保险 | 两人 | 1976 |
| 共获诺奖 | Akerlof, Spence, Stiglitz | 2001 |

### 8.2 政策含义

```
市场失灵 ≠ 政府必然有效
├─ 政府也有信息劣势
├─ 监管俘获问题
├─ 干预的次优性
└─ 需权衡干预成本与市场失灵
```

> 逆向选择理论告诉我们：**信息是市场的血液，堵塞信息就是堵塞市场**。
