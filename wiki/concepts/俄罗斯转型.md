# 俄罗斯转型

> Russian Transition — 1991-1994 年俄罗斯从计划经济向市场经济的"休克疗法"实验，是转型经济学最重要的"反面教材"

**俄罗斯转型**是指 1991 年苏联解体后，叶利钦政府在美国经济学家**杰弗里·萨克斯**等西方顾问指导下，推行"休克疗法"式的市场经济改革。这是 20 世纪最具争议性的经济转型实验，导致 GDP 累计下降 40%、恶性通胀（1992 年通胀 2,500%）、寡头化、预期寿命下降（男性下降 7 岁）。它是[[休克疗法]]的"失败典型"，也是[[华盛顿共识]]"教条应用"的代表案例，深刻影响了后续转型国家的策略选择。

---

## 一、核心定义与背景

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 时间 | 1991-至今（仍在持续）|
| 主导国家 | 俄罗斯（苏联解体后）|
| 关键人物 | 叶利钦、盖达尔、丘拜斯 |
| 关键顾问 | 杰弗里·萨克斯、芝加哥学派经济学家 |
| 转型策略 | 休克疗法（华盛顿共识）|
| 主要特点 | 价格自由化 + 寡头化私有化 + 货币可兑换 |

### 1.2 转型背景

```
【俄罗斯转型的历史背景】

苏联时期（1922-1991）：
├── 70 年的计划经济
├── 重工业为主
├── 资源型经济
├── 政治控制严密
└── 经济效率持续下降

戈尔巴乔夫时期（1985-1991）：
├── 1985-1991 改革
├── 1986 加速战略（Usorenie）
├── 1987 政治改革（Glásnost 公开性）
├── 1988 经济改革（Perestroika 重建）
├── 1990 取消宪法第 6 条（放弃党领导）
├── 1991-08 政变失败
└── 1991-12 苏联解体

解体后的俄罗斯：
├── 政治：混乱
├── 经济：崩溃边缘
├── 制度：法治薄弱
├── 行政：能力不足
├── 国际：孤立
└── 民众：期待改革
```

## 二、转型过程的三阶段

### 2.1 第一阶段：盖达尔改革（1991-1992）— 休克疗法的核心阶段

```
【盖达尔政府的"500 天计划"】

1991-10：叶利钦任命盖达尔为政府总理
├── 36 岁的经济学家
├── 推崇休克疗法
├── 与萨克斯密切合作
└── 目标是"快速市场化"

1991-11：开始实施"激进改革"
├── 1. 取消国家对价格的管制
├── 2. 取消国家对工资的控制
├── 3. 取消国家对企业的指令
└── 4. 推行贸易自由化

1992-01-02：价格自由化
├── 一次性放开 90% 商品价格
├── 仅保留少数关键商品管制
│   ├── 面包（短期管制）
│   ├── 牛奶
│   ├── 盐
│   ├── 糖
│   ├── 食用油
│   ├── 药物
│   └── 酒类
└── 立即引发恶性通胀

【500 天计划的失败】
├── 1. 通胀失控
├── 2. 卢布暴跌
├── 3. 储蓄价值清零
├── 4. 民众生活水平急剧下降
└── 5. 政治反对加剧
```

### 2.2 第二阶段：私有化与寡头化（1992-1994）

```
【俄罗斯私有化的"抵押拍卖"】

1992-10：私有化启动
├── 丘拜斯（Chubais）任私有化部长
├── 目标：快速私有化
├── 方式："抵押拍卖"（Loans for Shares）
└── 实际操作：寡头化

私有化的三个阶段：

阶段 ①（1992-1994）：凭证私有化
├── 每个俄罗斯人获得 10,000 卢布的私有化凭证
├── 可用于购买私有化企业的股份
├── 实际结果：少数人低价收购大量凭证
└── 形成"内部人交易"

阶段 ②（1994-1996）：货币私有化
├── 通过现金出售
├── 大量资产被低价出售
└── 形成"寡头集团"

阶段 ③（1995-1996）：抵押拍卖
├── 政府以"抵押"名义将大型国企交给寡头
├── 抵押人提供贷款给政府
├── 政府违约 → 抵押人获得企业
└── 实际是"国家资产的私人瓜分"

【七大寡头的形成】
├── 波塔宁（Potanin）：银行 + 矿业
├── 别列佐夫斯基（Berezovsky）：汽车 + 媒体
├── 古辛斯基（Gusinsky）：银行 + 媒体
├── 霍多尔科夫斯基（Khodorkovsky）：石油（Yukos）
├── 阿布拉莫维奇（Abramovich）：石油（Sibneft）
├── 弗里德曼（Fridman）：石油（TNK）
└── 马尔金（Malkin）：银行

【寡头化的问题】
├── 1. 寡头获取大量国家资产
├── 2. 普通民众未获益
├── 3. 政治影响力上升
├── 4. 经济决策被寡头扭曲
└── 5. 形成"寡头资本主义"
```

### 2.3 第三阶段：货币可兑换与金融危机（1992-1998）

```
【货币可兑换的过早实施】

1992-10：货币可兑换
├── 在外汇储备不足的情况下
├── 开放资本账户
├── 引发资本外逃
└── 卢布大幅贬值

1994-1995：卢布危机
├── 1994-10-11 "黑色星期二"
├── 卢布暴跌（1 美元 = 3,989 卢布）
├── 央行紧急干预
└── 重新引入部分外汇管制

1998：俄罗斯金融危机
├── 1998-08-17 卢布贬值
├── 1998-08-17 国债违约
├── "俄罗斯违约"震惊全球
├── 资本大量外逃
├── 卢布贬值 75%
└── 经济进一步衰退

【危机的原因】
├── 1. 资本账户开放过早
├── 2. 财政赤字过大
├── 3. 寡头转移资金
├── 4. 油价下跌（1998 年）
└── 5. 亚洲金融危机的溢出
```

## 三、俄罗斯转型的核心数据

### 3.1 经济数据

```
【1992-2000 年的关键经济数据】

| 年份 | GDP 增长 | 通胀率 | 失业率 | 卢布汇率（1 USD）|
|------|---------|--------|--------|---------------|
| 1991 | -13% | 160% | 0%（隐性）| 1.99 卢布 |
| 1992 | -14.5% | 2,508% | 5% | 415 卢布 |
| 1993 | -8.7% | 840% | 6% | 1,247 卢布 |
| 1994 | -12.6% | 215% | 8% | 3,553 卢布 |
| 1995 | -4.1% | 131% | 9% | 4,640 卢布 |
| 1996 | -3.6% | 22% | 10% | 5,560 卢布 |
| 1997 | 1.4% | 11% | 11% | 5,960 卢布 |
| 1998 | -5.3% | 84% | 13% | 9.7 卢布（新）|
| 1999 | 6.4% | 36% | 13% | 27 卢布 |
| 2000 | 10.0% | 20% | 12% | 28 卢布 |

【核心结论】
├── 1. GDP 累计下降 40%（1992-1998）
├── 2. 通胀最严重时 2,500%（1992）
├── 3. 卢布贬值 99%（1992-1998）
├── 4. 1999 年起开始恢复
└── 5. 2000 年起油价上涨推动经济恢复
```

### 3.2 社会数据

```
【社会指标的恶化】

预期寿命：
├── 1991：男 64 / 女 74
├── 1994：男 57 / 女 71（最低点）
├── 2000：男 59 / 女 72
└── 男性预期寿命下降 7 岁

人口变化：
├── 1991：1.48 亿
├── 2000：1.46 亿
├── 净流出 + 出生率下降
└── "死亡率危机"

贫富差距：
├── 1991：基尼系数 ~0.26（社会主义平均主义）
├── 1992-1998：基尼系数 ~0.40-0.45
├── 2000 年后：基尼系数 ~0.40（高位徘徊）
└── "寡头 + 贫困人口"的两极分化

社会问题：
├── 1. 失业率上升
├── 2. 养老金拖欠
├── 3. 医疗体系崩溃
├── 4. 犯罪率上升
├── 5. 酗酒、毒品问题恶化
└── 6. 人口寿命下降
```

### 3.3 政治数据

```
【政治指标的混乱】

议会冲突：
├── 1993-10 议会危机（叶利钦 vs 最高苏维埃）
├── 1993-10 坦克炮击白宫（议会大厦）
├── 1993-12 新宪法公投
└── 议会权力大幅削弱

总统 vs 寡头：
├── 1996：叶利钦连任（寡头支持）
├── 1999：叶利钦辞职
├── 2000：普京上任
└── 普京逐步收回寡头控制权

寡头政治影响：
├── 1996 总统选举：寡头提供资金支持
├── 媒体控制：3 大寡头控制媒体
├── 政策影响：寡头对政策的影响
└── 政治腐败：广泛存在
```

## 四、俄罗斯转型的五大失败原因

### 4.1 失败原因 ①：政治条件不具备

```
【政治条件的缺失】

缺失 ①：稳定的政治领导
├── 叶利钦健康问题（酗酒、心脏病）
├── 政府频繁更迭（1992-1999 7 任总理）
└── 政策连续性不足

缺失 ②：议会支持
├── 最高苏维埃（议会）反对改革
├── 1993 议会危机
└── 立法支持薄弱

缺失 ③：行政能力
├── 苏联解体后行政体系混乱
├── 公务员能力不足
├── 腐败问题严重
└── 政策执行困难

缺失 ④：社会共识
├── 民众对改革认识不一
├── 寡头 vs 普通民众分歧
└── 缺乏改革共识

【结论】
俄罗斯在缺乏政治条件的情况下推行休克疗法，是失败的根本原因之一
```

### 4.2 失败原因 ②：制度补全不彻底

```
【制度补全的失败】

缺失 ①：法治建设
├── 法律体系快速建立但执行薄弱
├── 司法独立性不足
├── 法律执行受政治干预
└── 形成"寡头资本主义"

缺失 ②：产权保护
├── 形式上保护私有产权
├── 实际产权经常被侵犯
├── 国有资产流失
└── 普通民众产权意识薄弱

缺失 ③：金融监管
├── 商业银行监管缺位
├── 央行独立性不足
├── 1998 年金融危机部分源于监管缺位
└── 形成"金融寡头"

缺失 ④：政府能力
├── 监管能力不足
├── 反垄断不力
├── 反腐败不力
└── 形成"国家失败"

【结论】
制度补全的失败，使休克疗法成为"无制度的市场化"，最终导致市场失序
```

### 4.3 失败原因 ③：私有化方式不当

```
【私有化的五大问题】

问题 ①：方式不当（"抵押拍卖"）
├── 形成寡头化
├── 普通民众未获益
└── 政治经济扭曲

问题 ②：速度过快
├── 4 年内私有化 ~70% 大型企业
├── 没有足够时间设计合理方案
└── "快速脱手"心态

问题 ③：缺乏透明度
├── 私有化过程不透明
├── 内部人交易普遍
└── 国有资产流失

问题 ④：缺乏公平性
├── 普通民众获得凭证贬值（寡头收购）
├── 形成"国家财产瓜分"
└── 社会不平等急剧扩大

问题 ⑤：缺乏法律基础
├── 私有化法律不完善
├── 法律执行困难
└── 后期频繁的法律纠纷

【结论】
私有化是俄罗斯转型失败的关键环节，"抵押拍卖"的方式被广泛批评
```

### 4.4 失败原因 ④：货币可兑换过急

```
【货币可兑换的五大问题】

问题 ①：外汇储备不足
├── 1992 年外汇储备仅几十亿美元
├── 难以抵御资本外逃冲击
└── 卢布大幅贬值

问题 ②：资本账户开放过早
├── 在制度补全之前开放
├── 形成"洗钱通道"
└── 资本大量外逃

问题 ③：汇率制度不当
├── 1992-1998 多次汇率制度调整
├── 没有稳定的汇率锚
└── 卢布汇率剧烈波动

问题 ④：财政赤字货币化
├── 1992-1998 央行大量印钞
├── 通胀失控
└── 形成"恶性循环"

问题 ⑤：国际资本流入不当
├── 大量短期资本流入
├── 1998 年突然撤离
└── 引发金融危机

【结论】
货币可兑换的过早实施，是 1998 年金融危机的重要原因
```

### 4.5 失败原因 ⑤：次序错误

```
【次序错误的五大表现】

错误 ①：稳定化不充分
├── 财政紧缩不够严格
├── 通胀失控
└── 1992 年通胀 2,500%

错误 ②：私有化过早
├── 在稳定化之前就开始
├── 形成寡头化
└── 政治后果严重

错误 ③：资本账户开放过早
├── 在制度补全之前
├── 资本外逃
└── 汇率危机

错误 ④：贸易自由化过快
├── 国内产业被冲击
├── 进口剧增
└── 国内失业

错误 ⑤：制度补全滞后
├── 法律、监管建设缓慢
├── 与市场开放不匹配
└── 形成"无制度的市场"

【McKinnon 顺序的违反】
├── 正确的顺序：稳定化 → 价格 → 贸易 → 私有化 → 金融自由化
├── 俄罗斯的顺序：价格 → 私有化 → 资本开放 → 稳定化（滞后）
└── "次序错误"是失败的核心原因之一
```

## 五、萨克斯的反思与争论

### 5.1 萨克斯的反思

```
【杰弗里·萨克斯的反思】

1990s：坚定支持
├── 玻利维亚、波兰案例是"成功"
├── 俄罗斯失败是"次序问题"
└── 坚持休克疗法

2000s：开始反思
├── 2005 年公开承认俄罗斯案例"失败"
├── 批评"芝加哥男孩"过于理想化
├── 强调"制度建设"的重要性
├── 提出"次序问题"的研究
└── 认为俄罗斯"时机 + 顺序"都错了

2010s：转向可持续发展
├── 关注非洲脱贫
├── 强调"目标导向"的发展模式
└── 批评"华盛顿共识"的简单化

2020s：综合视角
├── 整合休克 + 渐进的优势
├── 重视制度建设的复杂性
└── 强调"具体问题具体分析"
```

### 5.2 萨克斯与反对派的争论

```
【与斯蒂格利茨（Stiglitz）的争论】

斯蒂格利茨（前世界银行首席经济学家）：
├── 1. 俄罗斯的失败是"华盛顿共识"的失败
├── 2. 教条应用是错误的
├── 3. 应更重视制度建设
└── 4. 萨克斯过于乐观

萨克斯：
├── 1. 俄罗斯失败是"政治问题"
├── 2. 不是"理论问题"
├── 3. 强调次序和条件
└── 4. 仍是休克疗法的支持者

【与诺顿（Naughton）的争论】

诺顿：
├── 1. 中国模式证明"渐进"可行
├── 2. 俄罗斯失败证明"休克"有问题
└── 3. 应重视"中国经验"

萨克斯：
├── 1. 中国模式"不可复制"
├── 2. 中国有"特殊条件"
└── 3. 俄罗斯不能"渐进"
```

## 六、俄罗斯转型的长期影响

### 6.1 经济影响

```
【长期经济格局】

GDP 长期增长率（2000-2020）：
├── 2000-2008：年均 7%（油价上涨）
├── 2009：-7.8%（金融危机）
├── 2010-2014：年均 3%
├── 2015-2020：年均 0.5%（制裁）
└── 综合：依赖资源型增长模式

经济结构：
├── 资源型经济（石油 + 天然气）
├── 工业体系部分保留
├── 高科技产业相对薄弱
├── 服务业比重上升
└── 农业比重下降

国际经济地位：
├── GDP 全球第 6-12 位
├── 石油出口大国
├── 受西方制裁影响
└── "金砖国家"成员

【结论】
俄罗斯经济在长期仍是"资源型 + 国家主导"的模式，未能实现真正的高质量转型
```

### 6.2 政治影响

```
【政治格局的形成】

普京时代的"国家主义"：
├── 1. 收回寡头控制权
│   ├── 2000 古辛斯基流亡

│   ├── 媒体控制
│   └── 政治稳定
├── 3. 对外强势

└── 4. "主权民主"理念

俄罗斯通过强人政治重新建立了国家权威，但代价是法治薄弱和民主倒退
```

## 八、关键洞察总结

> [!key-insight] 俄罗斯转型 = 休克疗法的"反面教材"
>
> 俄罗斯是休克疗法最完整的实验场，也是最典型的失败案例。它证明了休克疗法在缺乏政治条件、制度基础、本土适配时的灾难性后果。

> [!key-insight] 失败的核心是"次序错误 + 制度缺位"
>
> 俄罗斯失败的核心不是休克疗法本身，而是"次序错误 + 制度缺位"。在没有稳定化之前私有化、没有制度补全之前开放资本账户，必然导致灾难。

> [!contradiction] 速度 vs 制度的权衡
>
> 休克疗法追求"速度"，但制度建设需要"时间"。这种张力是俄罗斯失败的核心矛盾。中国选择"渐进"，牺牲了"速度"，保留了"制度建设的时间"。

> [!key-insight] 政治条件是决定性因素
>
> 没有稳定的政治领导、议会支持、行政能力、社会共识，休克疗法必然失败。这是俄罗斯案例的核心教训。

> [!key-insight] "私有化方式"决定长期格局
>
> 俄罗斯的"抵押拍卖"导致寡头化，这是失败的关键环节。私有化方式比私有化本身更重要。

> [!key-insight] "次序错误"的代价
>
> McKinnon 顺序的违反（私有化过早 + 资本账户开放过早）是俄罗斯失败的核心原因之一。次序比速度更重要。

> [!key-insight] 中国模式的"反向证明"
>
> 中国 40+ 年的渐进改革是对俄罗斯失败的反向证明。但中国模式"不可复制"，每个国家应根据自身条件选择路径。
