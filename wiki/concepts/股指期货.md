# 股指期货（Stock Index Futures）

> 以股票指数为标的的期货合约 —— 全球金融市场最重要的"系统性风险管理工具"

## 一、定义

**股指期货**（Stock Index Futures）是以**股票价格指数**为标的的**标准化期货合约**。买卖双方约定在**未来某个特定日期**，按照**事先确定的指数点位**进行**现金交割**。

核心特征：
- **标的物**：股票价格指数（不是具体股票）
- **合约标准化**：合约乘数、到期日等固定
- **现金交割**：不交割股票，只交割差价
- **双向交易**：可以做多（看涨）也可以做空（看跌）

> **核心比喻**：股指期货是"**大市的方向盘**"——它让你**对整个市场**做多或做空，而**不需要选具体的股票**。

## 二、股指期货的"三大核心要素"

### 2.1 要素一：标的指数

```
主要股指期货的标的指数：

全球主要市场：
  - 美 S&P 500 期货
  - 美 Nasdaq 100 期货
  - 美 Dow Jones 期货
  - 欧 Euro Stoxx 50 期货
  - 英 FTSE 100 期货
  - 德 DAX 期货
  - 日 Nikkei 225 期货
  - 港 恒生指数期货
  - 韩 KOSPI 200 期货

中国市场：
  - 沪深 300 股指期货（IF）
  - 上证 50 股指期货（IH）
  - 中证 500 股指期货（IC）
  - 中证 1000 股指期货（IM）
```

### 2.2 要素二：合约乘数

```
合约乘数 = 期货价格 × 乘数 = 合约价值

中国主要合约乘数：
  - 沪深 300（IF）：300 元/点
  - 上证 50（IH）：300 元/点
  - 中证 500（IC）：200 元/点
  - 中证 1000（IM）：200 元/点

海外合约乘数：
  - S&P 500：50 美元/点
  - 恒指：50 港元/点
  - 日经 225：500 日元/点

例：
  沪深 300 期货 = 4000 点
  合约价值 = 4000 × 300 = 120 万元
  
  1 个点 = 300 元
  涨 10 个点 = 盈利 3000 元
  跌 100 个点 = 亏损 3 万元
```

### 2.3 要素三：到期日

```
中国股指期货的合约月份：
  - 当月（current month）
  - 下月（next month）
  - 下季（next quarter）
  - 隔季（next next quarter）
  - 共 4 个合约同时交易

例（2024 年 6 月）：
  - IF2406（2024 年 6 月到期）
  - IF2407（2024 年 7 月到期）
  - IF2409（2024 年 9 月到期）
  - IF2412（2024 年 12 月到期）

到期日：每月第三个星期五
交割方式：现金交割
```

## 三、股指期货的"三大功能"

### 3.1 功能一：风险管理（对冲）

```
投资者持有股票组合
担心大盘下跌
       ↓
卖出股指期货
       ↓
大盘跌 → 现货损失，期货盈利
       ↓
对冲市场风险
```

详见 [[wiki/concepts/期货对冲|期货对冲]] 和 [[wiki/concepts/套期保值|套期保值]]。

### 3.2 功能二：价格发现

```
股指期货市场是"价格发现的领先指标"：

  现货市场：4000 点
  期货市场：4010 点（升水 10 点）
       ↓
  期货市场预期：未来现货 = 4010
       ↓
  期货价格反映市场对未来预期
  期货市场是"未来价格的预测"
```

### 3.3 功能三：投机与套利

```
投机：
  - 单纯看多/看空指数
  - 用杠杆放大收益
  - 风险极高

套利：
  - 跨期套利：不同到期月
  - 跨品种套利：不同指数
  - 期现套利：现货 vs 期货
  - 跨市场套利：A 股 vs 港股
```

## 四、股指期货 vs 股指 ETF

| 维度 | 股指期货 | 股指 ETF |
|------|---------|---------|
| **杠杆** | 高（5-20 倍）| 无（1 倍）|
| **到期日** | 有 | 无 |
| **做空** | ✅ 直接做空 | ⚠️ 需融券 |
| **资金占用** | 保证金（5-15%）| 全额 |
| **交易成本** | 较低 | 较高（含管理费）|
| **跟踪误差** | 较小 | 较小 |
| **流动性** | 极好 | 极好 |
| **适合人群** | 专业投资者 | 各类投资者 |

> **核心区别**：股指期货 = **杠杆 + 做空 + 到期**；股指 ETF = **简单 + 长期 + 跟踪**。

## 五、股指期货 vs 股票

| 维度 | 股指期货 | 股票 |
|------|---------|------|
| **标的** | 指数 | 个股 |
| **杠杆** | 5-20 倍 | 1 倍 |
| **做空** | ✅ 直接 | ⚠️ 需融券 |
| **到期** | 有 | 无 |
| **分红** | 无（指数分红已计入）| 有 |
| **持有期成本** | 资金成本 | 资金成本 + 机会成本 |
| **选择** | 被动（指数）| 主动（选股）|
| **流动性** | 极高 | 视个股 |

## 六、中国股指期货的发展历程

### 6.1 关键时间线

```
1990 年代：探索
  - 1993 年：海南橡胶期货事件
  - 1993-1994 年：股指期货试点（暂停）

2006-2010 年：准备
  - 2006 年：推出沪深 300 指数
  - 2007 年：推出模拟交易
  - 2010 年 4 月 16 日：正式推出沪深 300 股指期货

2010-2015 年：发展
  - 推出上证 50、中证 500 股指期货
  - 套保业务大幅增长
  - 日均成交额达万亿

2015 年：危机
  - 股灾期间，股指期货被指为"元凶"
  - 2015 年 9 月：限制措施出台
  - 提高保证金、提高手续费
  - 日内开仓限制

2017-2019 年：松绑
  - 2017 年：两次松绑
  - 2018 年：进一步松绑
  - 2019 年：基本恢复至正常水平

2020 年至今：成熟
  - 推出中证 1000 股指期货（IM）
  - 推出股指期权
  - 套保业务规范化
```

### 6.2 2015 股灾与股指期货

```
2015 年 6-8 月：
  - A 股暴跌
  - 股指期货被指为"做空元凶"
  - 监管层对股指期货实施严格限制
       ↓
监管措施：
  - 非套保保证金从 20% 提至 40%
  - 日内平仓手续费大幅提高
  - 单一合约日开仓限制
       ↓
市场反应：
  - 股指期货流动性大幅下降
  - 套保功能严重受损
  - 价格发现功能受限
       ↓
2017-2019 年松绑后：
  - 市场逐步恢复
  - 监管框架更加完善
  - 套保业务重新活跃
```

> **关键反思**：股指期货是"无辜的工具"，不应承担"市场下跌的责任"。**限制股指期货并不能阻止市场下跌**——这是 2015 股灾的重要教训。

## 七、股指期货的"对冲应用"

### 7.1 应用一：投资组合对冲

```
场景：
  投资组合市值 1 亿元
  β 系数 = 1.0
  沪深 300 指数 = 4000 点
  合约乘数 = 300 元/点
       ↓
对冲手数：
  = 1 亿 / (4000 × 300)
  = 1 亿 / 120 万
  = 约 83 手
       ↓
操作：
  卖出 83 手 IF 期货
  → 锁定市场 β 风险
  → 保留选股 α 能力
```

详见 [[wiki/concepts/期货对冲|期货对冲]]。

### 7.2 应用二：择时对冲

```
场景：
  基金经理认为短期大盘要跌
  但不想卖出股票（避免税费、避免失去分红）
       ↓
操作：
  卖出股指期货
  短期持有（几天-几周）
  大盘稳定后平仓
       ↓
效果：
  - 短期下跌风险被对冲
  - 长期不损失股票仓位
```

### 7.3 应用三：现金管理

```
场景：
  保险/养老基金收到新资金
  计划未来 1-3 个月逐步建仓
  担心建仓前市场上涨
       ↓
操作：
  在建仓前买入股指期货
  （相当于"提前建仓"）
  逐步建仓现货的同时平仓期货
       ↓
效果：
  - 锁定建仓成本
  - 避免"踏空"风险
```

### 7.4 应用四：定向减持

```
场景：
  大股东计划减持 1 亿股
  当前股价 50 元
  担心减持期间下跌
       ↓
操作：
  卖出对应数量的股指期货
  减持期间逐步平仓期货
       ↓
效果：
  - 锁定当前价格
  - 在"无市场风险"环境下完成减持
```

## 八、股指期货的"风险与监管"

### 8.1 股指期货的"五大风险"

```
1. 杠杆风险
   - 5-20 倍杠杆
   - 保证金不足 → 强平
   - 损失可能超过本金

2. 流动性风险
   - 极端行情下流动性枯竭
   - 无法及时平仓
   - 滑点大幅增加

3. 基差风险
   - 期货与现货价差变动
   - 对冲不完美
   - 详见 [[wiki/concepts/基差风险|基差风险]]

4. 交割风险
   - 到期日临近时价格波动
   - 移仓成本
   - 现金交割的金额计算

5. 操作风险
   - 下单错误
   - 系统故障
   - 人为失误
```

### 8.2 中国监管的"五个特殊规定"

```
中国对股指期货的监管（2024 年）：

1. 投资者适当性：
   - 50 万资金门槛
   - 知识测试
   - 交易经验

2. 持仓限制：
   - 单个合约日内开仓限制（如 IF 20 手）
   - 套保额度申请
   - 不同账户合并计算

3. 保证金制度：
   - 合约价值的 8-12%
   - 强平线 = 保证金 25%
   - 风险准备金

4. 手续费：
   - 交易所手续费
   - 公司佣金
   - 日内平仓手续费较高

5. 异常交易监控：
   - 频繁撤单
   - 自成交
   - 实际控制关系账户
```

## 十、股指期货与 1998 香港案例

### 10.1 恒指期货的关键作用

```
1998 年 8 月 14-28 日：
  恒指期货是"决战的主战场"
       ↓
关键数据：
  - 8 月期指：临近期货结算日
  - 国际空头大量卖空 8 月期指
  - 金管局在 8 月期指大量做多
       ↓
决战日（8 月 28 日）：
  - 8 月期指结算日
  - 国际空头必须平仓
  - 金管局的"做多"形成强制平仓压力
  - 恒指从 6500 → 7820
       ↓
关键：
  恒指期货的对冲设计：
  - 时间：8 月 28 日（结算日）
  - 方向：做多
  - 数量：与现货匹配
  - 结果：与现货形成"立体轧空"
```

详见 [[wiki/concepts/1998香港金融保卫战]]。

### 10.2 恒指期货 vs 沪深 300 期货

| **市场环境** | 国际资金主导 | 国内资金主导 |
| **主要用途** | 套保 + 套利 | 套保 + 投机 |
| **基差** | 接近平价 | 长期贴水 |

## 十二、股指期货的"实战技巧"

### 12.1 对冲的"四个关键"

```
1. 选择合适的合约月份
   - 与风险期匹配
   - 避免频繁展期

2. 计算正确的对冲手数
   - 考虑 β 系数
   - 考虑合约乘数

3. 监控基差变动
   - 每日跟踪
   - 必要时调整

4. 注意移仓成本
   - 提前 1-2 周移仓
   - 避免到期日"逼仓"
```

### 12.2 投机 vs 套保的"选择"

```
选择套保，如果你：
  - 持有大量股票
  - 不想短期卖出
  - 关心下行风险

选择投机，如果你：
  - 没有股票头寸
  - 单纯看多/看空指数
  - 追求高收益
  - 风险承受能力强
```

## 十三、股指期货的"投资策略"

### 13.1 策略一：期现套利

```
期现套利（Cash-Futures Arbitrage）：
  期货价格 ≠ 现货价格 + 持有成本
       ↓
  例：
    现货 4000，期货 4200
    持有成本 100（3 个月）
    理论期货价 = 4000 + 100 = 4100
    实际期货价 4200 → 套利空间
       ↓
  操作：
    买入现货（建模拟组合）+ 卖出期货
    到期时平仓
    套利收益 = 100 - 交易成本
```

### 13.2 策略二：跨期套利

```
跨期套利（Calendar Spread）：
  远月期货 - 近月期货 = 价差
       ↓
  当价差偏离历史均值时
  → 做多近月 + 做空远月
  → 或反向
```

### 13.3 策略三：跨品种套利

```
跨品种套利：
  - IF（沪深 300） vs IH（上证 50）
  - IF（沪深 300） vs IC（中证 500）
       ↓
  利用价差变化获利
```

## 十四、股指期货的"局限性"

### 14.1 不能消除的"三大风险"

```
1. 基差风险
   - 期货 ≠ 现货
   - 基差变动 = 剩余风险
   详见 [[wiki/concepts/基差风险|基差风险]]

2. β 漂移风险
   - 投资组合 β 持续变化
   - 静态对冲可能失效
   - 需要动态调整

3. 流动性风险
   - 极端行情流动性枯竭
   - 无法及时平仓
   - 滑点大幅增加
```

### 14.2 中国股指期货的"特殊问题"

```
1. 长期贴水问题
   - 2010-2024 年长期贴水
   - 多头长期付出成本
   - 影响套保效果

2. 持仓限制
   - 单一合约日内开仓 20 手
   - 影响大资金套保

3. 投资者门槛
   - 50 万资金门槛
   - 排除中小投资者
```

## 十五、一句话总结

> **股指期货 = 以股票指数为标的的标准化期货合约**。
>
> 它是**全球金融市场最重要的"系统性风险管理工具"**——让投资者可以**对整个市场**做多或做空，而**不需要选具体股票**。股指期货的**三大功能**是**风险管理（对冲）、价格发现、投机套利**。
>
> **1998 香港金管局**把股指期货运用到极致——在 8 月期指上做多，**配合现货买入**，形成"立体轧空"。这证明了股指期货不仅可以"防守"，还能"进攻"——**好的对冲是"主动的"**。
