# 2008 全球金融危机

> 次贷违约 → CDO 崩塌 → 雷曼破产 → 全球金融危机 —— 21 世纪最严重的金融海啸

## 一、事件概要

**2008 年全球金融危机**（Global Financial Crisis, GFC）是 21 世纪**最严重的金融危机**，由美国**次级抵押贷款**（Subprime Mortgage）违约引发，通过**结构化金融产品**（CDO、CDS）扩散到全球金融体系，最终导致**雷曼兄弟**破产、**贝尔斯登**被收购、**美国国际集团**（AIG）被政府救助、**全球股市暴跌**和**实体经济衰退**。

关键数据：
- 标普 500 从 1565 点跌至 677 点（-57%）
- 全球股市市值蒸发约 30 万亿美元
- 美国失业率从 4.7% 飙升至 10%（2009.10）
- 美联储资产负债表从 9000 亿扩张至 4.5 万亿
- 美国财政赤字从 1600 亿飙升至 1.4 万亿
- 全球贸易下降 12%
- 影响深度：堪比 1929 年大萧条

## 二、时间线

### 2.1 危机前夜（2004-2007）

```
2004-2006 年：
  - 美联储低利率政策
  - 房地产泡沫形成
  - 次级贷款爆发性增长
  - CDO/CDS 创造繁荣

2007 年初：
  - 房地产价格开始下跌
  - 次贷违约率上升
  - 房贷违约率从 5% 升至 15%

2007 年 2 月：
  - HSBC 警告次贷风险
  - 新世纪金融公司（New Century）预警

2007 年 6 月：
  - 贝尔斯登旗下两只对冲基金倒闭
  - 次贷危机开始暴露
```

### 2.2 危机爆发（2007-2008）

```
2007 年 8 月：
  - 法国巴黎银行（BNP）暂停三只基金赎回
  - 全球信贷市场冻结
  - 美联储紧急注资

2007 年 9 月：
  - 北岩银行（Northern Rock）挤兑
  - 英国银行业危机
  - 美联储降息 50bp

2007 年底-2008 年初：
  - 标普 500 创新高
  - 市场认为"已经稳定"
  - 实际只是次贷暴露的初期

2008 年 3 月：
  - 贝尔斯登濒临破产
  - 美联储紧急救助
  - 摩根大通 2 美元/股收购贝尔斯登
```

  - 国会通过修改后的 7000 亿救助计划（TARP）

## 三、危机的"五大原因"

### 3.1 原因一：房地产泡沫

```
房地产泡沫的形成（2001-2006）：

1. 低利率环境
   - 2001-2003 美联储降息至 1%
   - 房贷成本极低

2. 房地产价格上涨
   - 房价年涨幅 10-20%
   - 投机需求旺盛
   - "房价永远涨"的神话

3. 次级贷款激增
   - 无收入证明贷款（NINJA）
   - 零首付贷款
   - "骗子贷款"

4. 银行疯狂放贷
   - 来者不拒
   - 评级机构配合
   - 短期利益驱动
```

### 3.2 原因二：结构化金融产品的"复杂化"

```
结构化金融产品链条：

1. 抵押贷款（Mortgage）
   - 购房者向银行借款
   - 银行持有房贷

2. 资产支持证券（ABS/MBS）
   - 银行把房贷打包成证券
   - 出售给投资者
   - 风险"转嫁"

3. 担保债务凭证（CDO）
   - 把 ABS 重新打包
   - 分为优先级/中间级/股权级
   - "风险分散"

4. 信用违约互换（CDS）
   - 给 CDO 买保险
   - AIG 大量承保
   - 风险再次"转嫁"

5. 合成 CDO
   - 用 CDS 创造"虚拟"CDO
   - 风险被无限放大
       ↓
问题：
  - 链条复杂，难以理解
  - 评级机构（标普、穆迪）给 AAA 评级
  - 实际风险被严重低估
```

### 3.3 原因三：评级机构的失职

```
评级机构（标普、穆迪、惠誉）：
  - 给次贷相关产品 AAA 评级
  - 与发行方有利益关系（评级费）
  - 模型基于历史数据（房价从未大跌）
  - "评级通胀"
       ↓
问题：
  - 投资者信任 AAA 评级
  - 实际风险被严重低估
  - 危机时 AAA 暴跌至垃圾级
       ↓
后果：
  - 信任崩溃
  - 整个市场失灵
```

### 3.4 原因四：金融机构的过度杠杆

```
2007 年金融机构的杠杆：
  - 雷曼兄弟：30+ 倍
  - 贝尔斯登：30+ 倍
  - 摩根士丹利：30+ 倍
  - 高盛：25+ 倍
  - 五大投行平均：~28 倍
       ↓
对比：
  - 普通银行：10-15 倍
  - 巴菲特（伯克希尔）：不到 2 倍
       ↓
问题：
  - 资产价格小幅下跌 → 资本金被吞噬
  - 评级机构下调评级 → 追加保证金
  - 流动性枯竭 → 强制平仓
  - 死亡螺旋
```

### 3.5 原因五：监管的"缺位"

```
监管问题：

1. 多头监管
   - SEC 监管投行
   - 美联储监管银行
   - 但投行控股公司无人监管
   - 监管套利

2. 影子银行系统
   - 投资银行
   - 对冲基金
   - 货币市场基金
   - 资产支持商业票据
   - 几乎无监管

3. 资本充足率要求低
   - 巴塞尔协议 II 不够严
   - 大型投行被豁免
   - 杠杆率无明确限制

4. 系统性风险无人监管
   - 没有"系统性风险监管机构"
   - 美联储只管银行
   - 投行控股公司成为监管盲区
```

## 四、危机的"三大阶段"

### 4.1 阶段一：次贷危机（2007-2008.3）

```
特征：
  - 次贷违约率上升
  - 房贷相关证券（MBS、CDO）价格下跌
  - 贝尔斯登倒闭
  - 美林损失扩大

应对：
  - 美联储多次降息
  - 提供流动性
  - 但未触及根本
```

### 4.2 阶段二：金融危机（2008.9-2008.10）

```
特征：
  - 雷曼破产
  - AIG 危机
  - 全球信贷冻结
  - 股市暴跌
  - 货币市场基金"破净"（跌破 1 美元）

应对：
  - 7000 亿 TARP 救助
  - 美联储救助 AIG
  - 美联储多种紧急工具
  - 全球央行协同降息
```

### 4.3 阶段三：经济衰退（2008.10-2009.6）

```
特征：
  - 全球经济衰退
  - 失业率飙升
  - 贸易大幅下降
  - 消费萎缩

应对：
  - 全球央行宽松
  - 财政刺激（中国 4 万亿、美国 8000 亿）
  - 美联储 QE1（1.7 万亿）
```

## 八、危机对全球金融体系的"重塑"

### 8.1 监管改革

```
2008 年后的监管改革：

1. 多德-弗兰克法案（2010 年）
   - 美国历史上最全面的金融监管改革
   - 设立"金融稳定监督委员会"（FSOC）
   - 设立"消费者金融保护局"（CFPB）
   - "沃尔克规则"：限制银行自营交易
   - 系统重要性金融机构（SIFI）特别监管

2. 巴塞尔协议 III
   - 提高资本充足率
   - 引入杠杆率
   - 引入流动性覆盖率（LCR）
   - 净稳定融资比率（NSFR）

3. 宏观审慎政策
   - 系统性风险成为央行目标
   - 逆周期资本缓冲
   - 系统重要性银行附加资本

4. 国际监管协调
   - G20 成为主要平台
   - 金融稳定理事会（FSB）
   - 全球系统性风险监管
```

### 8.2 央行角色的变化

```
2008 年后央行的"双重角色"：

1. 货币政策
   - 利率工具
   - 量化宽松
   - 前瞻指引

2. 金融稳定
   - 系统性风险监管
   - 最后贷款人

3. 财政协调
   - 与财政部门配合
   - 协调救助

4. 资产价格管理者

```

## 十、2008 危机的"金融学意义"

### 10.1 风险管理的"新认知"

```
2008 年揭示的金融脆弱性：

1. **系统性风险是真实的**
   - 个体风险可控
   - 系统性风险可能摧毁整个体系
   - 央行必须承担"最后贷款人"角色

2. **复杂性的"危险"**
   - 结构化金融产品过于复杂
   - 难以理解风险
   - 难以管理风险
   - "无知"是最大的风险

3. **杠杆的"危险"**
   - 平时低风险
   - 极端市场死亡
   - 必须严格限制

4. **监管的"必要性"**
   - 市场自发调节不够
   - 监管必须全面
   - 跨市场、跨机构、跨境
```

### 10.2 现代金融的"哲学反思"

```
2008 年后的金融哲学反思：

1. **"市场是有效的"吗？**
   - 不是
   - 充满了错误和泡沫
   - 必须有监管

2. **"风险是可以分散的"吗？**
   - 不可以
   - 极端市场相关 → 1
   - 必须有"系统性风险"视角

3. **"金融机构是理性的"吗？**
   - 不理性
   - 短期利益驱动
   - 必须有"行为"监管

4. **"全球化是好的"吗？**
   - 收益与风险并存
   - 危机传染
   - 必须有"防火墙"
```

## 十一、2008 危机与其他危机的对比

### 11.1 与历史危机的对比

## 十二、2008 危机的"经验教训"

### 12.1 对监管者

```
2008 年危机对监管的教训：

1. **必须全面监管**
   - 覆盖所有金融机构
   - 覆盖所有金融产品
   - 覆盖跨境金融活动

2. **必须前瞻监管**
   - 识别系统性风险
   - 防范"影子银行"
   - 监管创新产品

3. **必须宏观审慎**
   - 关注系统性风险
   - 关注金融机构间关联
   - 关注金融周期

4. **必须国际协调**
   - 危机无国界
   - 监管需协调
   - 央行需合作
```

### 12.2 对投资者

```
2008 年危机对投资者的教训：

1. **警惕结构性产品**
   - 复杂的往往是危险的
   - 简单可能更安全
   - 充分理解再投资

2. **警惕高杠杆**
   - 平时低风险
   - 极端市场高风险
   - 限制杠杆

3. **警惕"这次不一样"**
   - 历史上每次都有"特殊"
   - 但市场规律不变
   - 保持敬畏

4. **保持充足现金**
   - 极端市场流动性枯竭
   - 现金是"安全资产"
   - 提前规划
```

### 12.3 对央行

```
2008 年危机对央行的教训：

1. **必须承担最后贷款人角色**
   - 不救个别机构
   - 救整个系统
   - 提供流动性

2. **必须超越传统工具**
   - 利率工具有限
   - 创新工具必要
   - 量化宽松

3. **必须关注金融稳定**
   - 不只是通胀和增长
   - 还要关注金融周期
   - 还要关注系统性风险

4. **必须财政协调**
   - 单靠央行不够
   - 必须财政配合
   - 协调救助
```

## 十三、2008 危机的"历史定位"

### 13.1 金融史上的地位

```
2008 危机是金融史的转折点：

1. "大萧条"以来最严重危机
   - 影响深度可媲美 1929
   - 但应对更有效
   - 反弹更迅速

2. 现代金融监管的起点
   - 多德-弗兰克法案
   - 巴塞尔协议 III
   - 宏观审慎政策

3. 量化宽松的起点
   - 美联储开启 QE 时代
   - 改变了货币政策范式
   - 引发了通胀讨论

4. 央行角色的扩张
   - 从"通胀目标制"到"双重使命"
   - 加上"金融稳定"
   - 加上"财政协调"

5. 全球化的反思
   - 危机传染
   - 全球化"退潮"
   - 民粹主义兴起
```

### 13.2 与化债的联系

```
2008 危机对中国化债的启示：

1. **直接债务工具的风险**
   - CDO 复杂化的教训
   - 中国化债用"标准化债券"
   - 避免复杂化

2. **结构化金融的警惕**
   - 中国没有复杂 CDO
   - 化债是"标准化债务置换"
   - 保持简单

3. **央行的"绕道支持"**
   - 中国央行不直接买债
   - 避免"中国版 QE"
   - 避免财政货币化

4. **银行体系稳健**
   - 中国银行体系未受冲击
   - 化债不影响银行核心
   - 系统性风险可控
```

详见 [[wiki/concepts/化债]] 和 [[wiki/concepts/扩表与缩表]]。

## 十四、2008 危机的"经验教训汇总"

### 14.1 五个核心教训

```
1. **警惕结构性金融**
   - 复杂产品隐藏风险
   - 简单可能更安全
   - 充分理解再投资

2. **警惕高杠杆**
   - 平时低风险
   - 极端市场死亡
   - 限制杠杆

3. **监管必须全面**
   - 覆盖所有金融机构
   - 覆盖所有产品
   - 关注系统性风险

4. **央行是最后守护者**
   - 必须承担最后贷款人
   - 必须超越传统工具
   - 必须与财政协调

5. **全球化有风险**
   - 危机无国界
   - 必须国际协调
   - 必须有"防火墙"
```

## 十五、一句话总结

> **2008 全球金融危机 = 次贷违约 + CDO 崩塌 + 雷曼破产 + 系统性危机 + 美联储"无限救助"**。
>
> 它是 21 世纪**最严重的金融危机**，与 1929 年大萧条相提并论。它揭示了**金融体系的三个根本脆弱性**：**结构化产品的复杂化**（CDO 链条）、**系统性风险的真实存在**（个体稳健不等于系统稳健）、**全球化金融的传染性**（危机无国界）。
>
> **2008 年的直接遗产**是**多德-弗兰克法案、巴塞尔协议 III、宏观审慎政策、量化宽松时代**。**2008 年之后的 10 多年**，金融体系相对稳定，但**2020 年的流动性危机**和**未来的未知挑战**仍在提醒我们：**金融市场的根本脆弱性没有消失**。**真正防范危机不是"消除风险"，而是"敬畏风险、全面监管、协同应对"**。
