# 金砖支付系统

> BRICS Payment System — 金砖国家去美元化与"非美元清算联盟"的核心基础设施

**金砖支付系统**（BRICS Pay / BRICS Payment System）是金砖国家（BRICS）正在推动建设的多边支付清算体系，旨在建立金砖国家之间的本币结算机制，减少对美元和 SWIFT 的依赖。它是金砖国家"去美元化"战略的核心基础设施，也是新兴市场国家挑战美元霸权的制度性尝试。其主要形态包括 BRICS Pay、本币结算框架、多边 CBDC 桥接以及金砖共同货币的讨论。

---

## 一、核心定义与本质

### 1.1 定义

金砖支付系统是指**金砖国家之间建立的多边支付清算体系**，旨在实现：

```
【金砖支付系统的四大目标】
├── 1. 金砖国家之间的本币结算
├── 2. 减少对美元 / SWIFT / CHIPS 的依赖
├── 3. 推动金砖国家贸易便利化
└── 4. 探索金砖共同货币的可能性
```

### 1.2 金砖国家的演进

```
【金砖国家演进历程】

2006：金砖四国外长会议（俄、印、中、巴）
2009：首届金砖峰会（俄罗斯叶卡捷琳堡）
2010：南非加入（金砖五国）
2011-2022：金砖五国机制运行
2023：金砖国家宣布扩容
2024-2026：金砖扩容落地
  ├── 新增正式成员：
  │   ├── 沙特阿拉伯
  │   ├── 阿联酋
  │   ├── 伊朗
  │   ├── 埃及
  │   ├── 埃塞俄比亚
  │   └── （其他申请中）
  └── 金砖+ 概念形成（覆盖更多发展中国家）
```

### 1.3 金砖支付系统的本质

```
【金砖支付系统的本质】

表面：金砖国家的"支付便利化"
深层：新兴市场国家集体挑战美元霸权的制度尝试

├── 地缘层面：金砖国家 = 全球南方的主要经济体
├── 经济层面：金砖+ 国家合计 GDP 已超过 G7
├── 政治层面：金砖国家希望在国际金融治理中获得更大话语权
└── 战略层面：去美元化 + 应对美国长臂管辖
```

## 二、金砖支付系统的核心机制

### 2.1 五大核心机制

```
【机制 ①：本币结算】
├── 金砖国家之间的贸易用本币结算
├── 减少美元中介
├── 已部分实现（中俄、中沙、中阿）
└── 持续扩大中

【机制 ②：双边本币互换】
├── 央行间签订本币互换协议
├── 提供短期流动性支持
├── 中国已与多个金砖国家签订
└── 持续扩大中

【机制 ③：BRICS Pay / BRICS Bridge】
├── 金砖国家共同的支付系统
├── 类似 SWIFT + CIPS 的多边版本
├── 已开始技术开发
└── 数字货币版（mBridge 桥接）探索中

【机制 ④：金砖共同货币（探索）】
├── 长期目标：建立金砖共同货币
├── 类似欧元的设计逻辑
├── 仍处于讨论阶段
└── 但 SDAR、BRIQ 等讨论活跃

【机制 ⑤：多边 CBDC 桥接】
├── 基于数字货币的清算
├── mBridge 是典型代表
├── 金砖国家央行间的 CBDC 互联
└── 长期演进方向
```

### 2.2 本币结算的具体路径

```
【本币结算的典型场景】

中俄贸易：
├── 能源贸易：人民币 / 卢布
├── 一般商品：人民币为主
├── 90%+ 已用人民币结算
└── 卢布结算占少数

中沙贸易：
├── 石油贸易：试点人民币结算
├── 一般商品：美元 / 人民币 / 沙特里亚尔
└── 趋势：人民币结算扩大

中印贸易：
├── 一般商品：美元 / 人民币 / 卢比
├── 央行本币互换协议签订
└── 趋势：本币结算扩大

金砖内部：
├── 多边本币结算框架
├── 央行本币互换协议网络
└── 持续扩大中
```

### 3.1 关键里程碑

| 时间 | 里程碑事件 |
|------|-----------|
| 2009 | 首届金砖峰会 |
| 2014 | 金砖国家签订《新开发银行协定》|

| 2015 | 金砖国家应急储备安排（CRA）启动 |

| 2022 | 俄乌冲突后，金砖支付系统讨论加速 |

| 2024 | 金砖扩容落地，新增 6 国 |
| 2024-2026 | BRICS Pay、金砖共同货币等讨论活跃 |

### 3.2 金砖国家新开发银行（NDB）

```
【金砖国家新开发银行（NDB）】

成立：2014 年签订协定，2015 年开业
总部：上海（中国）
性质：金砖国家共同所有的多边开发银行
初始资本：$1000 亿（5 个创始成员国均摊）
行长：轮流担任（首任为印度人 K.V. Kamath）

核心功能：
├── 1. 基础设施融资
├── 2. 可持续发展项目融资
├── 3. 金砖国家本币贷款（已实现）
├── 4. 应急储备安排（CRA）的支持
└── 5. 与世行 / 亚投行 / 丝路基金协同

战略意义：
├── 1. 挑战 IMF / 世行的金融治理垄断
├── 2. 提供"无政治条件"的发展融资
├── 3. 推动金砖国家本币贷款
└── 4. 是金砖支付系统的"机构载体"
```

### 3.3 金砖国家应急储备安排（CRA）

```
【金砖国家应急储备安排（CRA）】

成立：2014 年签订，2015 年启动
规模：$1000 亿
├── 中国：$410 亿
├── 巴西 / 印度 / 俄罗斯：各 $180 亿
└── 南非：$50 亿

核心功能：
├── 1. 应对短期国际收支危机
├── 2. 与 IMF 联动（但可独立使用）
├── 3. 提供流动性支持
└── 4. 类似 IMF 的"区域性自救机制"

战略意义：
├── 1. 减少对 IMF 的依赖
├── 2. 增强金砖国家金融安全
├── 3. 是金砖国家"金融防火墙"
└── 4. 为未来金砖共同货币做准备
```

## 四、金砖支付系统的战略意义

### 4.1 对美元霸权的挑战

```
【金砖支付系统的战略意义】

1. **挑战美元的"中介货币"地位**
   └── 金砖国家间本币结算，减少美元交易

2. **挑战 SWIFT 的报文垄断**
   └── BRICS Pay 或金砖桥可绕开 SWIFT

3. **挑战 IMF / 世行的金融治理垄断**
   └── NDB、CRA 是替代方案

4. **挑战美国长臂管辖**
   └── 金砖国家间清算不受美国管辖

5. **挑战美元储备地位**
   └── 金砖央行储备多元化
```

### 4.2 金砖国家的差异化诉求

```
【金砖国家的差异化诉求】

中国：
├── 推动人民币国际化
├── 扩大 CIPS 与金砖国家合作
├── 探索 mBridge 与金砖支付系统对接
└── 在金砖支付系统中起主导作用

俄罗斯：
├── 应对美国制裁
├── SPFS 与金砖支付系统对接
├── 卢布结算扩大
└── "被迫"加速去美元化

印度：
├── 推动卢比国际化
├── UPI / INFT 与金砖支付系统对接
├── 谨慎平衡（避免过度依赖中国）
└── "印度制造"的金融基础设施

巴西：
├── 推动雷亚尔国际化
├── 减少对美元依赖
└── 农业大国，关注大宗商品本币结算

南非 / 沙特 / 阿联酋 / 伊朗 / 埃及 / 埃塞俄比亚：
├── 加入金砖+，寻求新的金融合作机会
├── 减少对美元 / SWIFT 的依赖
└── 探索本币结算 / CBDC 互联
```

### 4.3 金砖支付系统 vs 美元体系

| 维度 | 美元体系 | 金砖支付系统 |
|------|---------|-------------|
| 主导方 | 美国 | 金砖国家集体 |
| 货币 | 美元 | 多种本币 + 未来共同货币 |
| 治理 | 美方主导（G7、IMF）| 多边协商 |
| 覆盖 | 全球 | 金砖+ 成员国 |
| 基础设施 | SWIFT + CHIPS + 美债 | BRICS Pay + CIPS + SPFS + NDB |
| 政治属性 | 美元霸权 | 反美元霸权 |
| 抗制裁能力 | 弱（依赖美国基础设施）| 强（独立基础设施）|

## 五、金砖支付系统的核心挑战

### 5.1 五大挑战

```
【挑战 ①：内部协调困难】
├── 金砖国家利益诉求差异
├── 中印边界问题
├── 俄印关系微妙
├── 巴西、南非相对中立
└── 伊朗、阿联酋等地缘复杂

【挑战 ②：技术基础设施不足】
├── BRICS Pay 仍在开发
├── 各方技术水平差异
├── 与现有系统对接复杂
└── 数字货币桥接标准未统一

【挑战 ③：经济实力分布不均】
├── 中国 GDP 远超其他金砖国家
├── 可能形成"中国主导"
├── 印度希望"对等"
├── 其他国家各有算盘
└── 类似欧元区的内部博弈

【挑战 ④：美元惯性极强】
├── 美元仍是全球主导货币（~58% 储备）
├── 美元流动性最好
├── 美债市场规模最大
└── 短期难以撼动

【挑战 ⑤：外部压力】
├── 美国对金砖国家施压
├── 长臂管辖威胁
├── 盟友体系压力（G7）
└── 部分国家"选边站"压力
```

### 5.2 金砖共同货币的可行性

```
【金砖共同货币的探索】

目标：
├── 类似欧元的金砖共同货币
├── 减少对美元的依赖
├── 提升金砖国家金融主权

挑战：
├── 1. 各成员国经济结构差异巨大
│   ├── 中国：制造业大国
│   ├── 俄罗斯：能源大国
│   ├── 巴西：农业 + 矿业
│   ├── 印度：服务业 + IT
│   └── 难以建立统一的中央银行
├── 2. 财政协调困难（类似欧元区难题）
├── 3. 政治意愿不足（各国不愿让渡主权）
└── 4. 美元惯性极强

可能路径：
├── 路径 ①：从"金砖结算单位"（类似 SDR）起步
├── 路径 ②：先建立金砖支付系统 + 本币结算
├── 路径 ③：未来逐步过渡到共同货币
└── 短期不可行，长期探索中
```

## 六、金砖支付系统与其他系统的对比

### 6.1 全球主要清算基础设施矩阵

| 系统 | 主导方 | 锚定货币 | 战略意图 |
|------|--------|---------|---------|
| **SWIFT + CHIPS** | 美欧 | 美元 | 美元霸权 |
| **CIPS** | 中国 | 人民币 | 人民币国际化 |
| **SPFS** | 俄罗斯 | 卢布 | 应对美国制裁 |
| **INSTEX** | 欧盟 | 欧元 | 绕过美国制裁 |
| **mBridge** | 多边央行 | 多币种 CBDC | 数字货币清算 |
| **金砖支付系统** | 金砖国家 | 多种本币 | 去美元化 + 反美元霸权 |
| **UPI / INFT** | 印度 | 卢比 | 印度版反制裁基础设施 |

### 6.2 金砖支付系统的独特性

```
【金砖支付系统的独特性】

1. **多边性**：不是单一国家主导（区别于 CIPS / SPFS）
2. **反霸权性**：明确针对美元霸权（区别于 INSTEX）
3. **政治性**：与金砖+ 地缘政治框架紧密联系
4. **渐进性**：从本币结算到共同货币的渐进路径
5. **开放性**：不断扩容（区别于 CIPS / SPFS 的封闭性）

【金砖支付系统的"非对称优势"】
├── 1. 经济规模大（金砖+ 合计 GDP 已超 G7）
├── 2. 资源禀赋齐全（能源、农业、制造业、科技）
├── 3. 政治意愿明确（去美元化共识）
├── 4. 多边治理（避免单一国家主导）
└── 5. 持续扩容（吸收更多发展中国家）
```

## 七、金砖支付系统的核心案例

### 7.1 中俄本币结算（2022-至今）

```
【中俄本币结算的成功案例】

背景：
├── 2022 俄乌冲突后，俄罗斯被切断 SWIFT
├── 中俄贸易需要新的支付通道
└── CIPS + 央行本币互换 + 商业银行合作

结果：
├── 90%+ 中俄贸易用人民币结算
├── 俄罗斯央行人民币储备占比 ~30%+
├── 莫斯科交易所人民币交易量跃居全球前三
└── 形成"人民币-卢布"双轨结算

对金砖支付系统的启示：
├── 1. 制裁压力可加速本币结算
├── 2. 央行间合作 + 商业银行配合是关键
├── 3. CIPS + 本币互换是有效模式
└── 4. 可推广到其他金砖国家
```

### 7.2 中沙石油人民币结算试点（2023-至今）

```
【中沙石油人民币结算】

背景：
├── 沙特是 OPEC 核心，美国传统盟友
├── 中国是沙特最大石油买家
├── 沙特考虑部分石油用人民币结算
└── 2023 年起逐步扩大试点

意义：
├── 1. 挑战石油美元体系
├── 2. 沙特"东方转向"的金融体现
├── 3. 推动金砖支付系统在中东落地
└── 4. 沙特加入金砖+ 后加速
```

### 7.3 金砖国家 NDB 本币贷款

```
【NDB 本币贷款案例】

背景：
├── NDB 初期以美元贷款为主
├── 2018 年起推动本币贷款
└── 2024 年本币贷款占比已超 50%

意义：
├── 1. 多边开发银行本币贷款的突破
├── 2. 减少金砖国家对美元融资的依赖
├── 3. 推动本币国际化（人民币、卢比、卢布）
```

## 九、关键洞察

> [!key-insight] 金砖支付系统 = 新兴市场对美元霸权的"集体挑战"
>
> 与 CIPS / SPFS 等单一国家主导的清算系统不同，金砖支付系统是**多边集体行动**。它的真正力量不在于技术或规模，而在于**集体政治意愿**。

> [!key-insight] 金砖支付系统 = "美元武器化"的反作用力
>
> 美国越频繁使用金融制裁、越多长臂管辖，越多国家希望加入金砖+、越积极推动本币结算。金砖支付系统是"美元武器化悖论"的直接体现。

> [!contradiction] 金砖支付系统的"内部张力"
>
> 金砖国家希望挑战美元霸权，但又不愿意被中国主导（特别是印度）。这种"内部张力"决定了金砖支付系统必须采用多边治理，而非单一国家主导。

> [!key-insight] 金砖支付系统 = "去美元化"的制度载体
>
> 此前去美元化主要靠双边协议（CIPS、SPFS 等），金砖支付系统提供了**多边集体行动**的制度载体。这是从"双边去美元化"到"多边去美元化"的跃迁。

> [!key-insight] 金砖扩容 = 金砖支付系统的"网络效应"
>
> 金砖+ 持续扩容意味着更多国家加入支付系统 → 规模效应 → 美元依赖下降 → 更多国家愿意加入 → 这是正反馈循环。
