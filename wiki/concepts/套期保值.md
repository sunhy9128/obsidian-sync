# 套期保值（Hedging）

> 中文金融术语的标准表述 —— 与"期货对冲"含义相同，但更强调"保值的稳健性"

## 一、定义

**套期保值**（Hedging，中文又称"套保"）是指**通过期货（或衍生品）市场**建立**与现货市场方向相反**的头寸，**抵消现货市场价格波动风险**的交易行为。

> **"套期"** = "套住"未来某个时间点
> **"保值"** = 锁定价值，避免损失
> **"套期保值"** = 锁定未来的价值

"套期保值"是中文金融术语的标准表述，与"期货对冲"（Futures Hedge）含义完全相同，但更强调"**保值的稳健性**"。

详见 [[wiki/concepts/期货对冲|期货对冲]]。

## 二、"套期保值"与"期货对冲"的关系

### 2.1 词源对比

| 术语 | 来源 | 强调 |
|------|------|------|
| **Hedging** | 英文 | "对冲"动作本身 |
| **期货对冲** | 中英混合 | 强调使用期货工具 |
| **套期保值** | 中文 | 强调"保值的稳健性" |

### 2.2 含义对比

```
Hedging（英文）：
  强调"动作"——做一笔反向交易来对冲风险

期货对冲（中英混合）：
  强调"工具"——使用期货这种特定工具

套期保值（中文）：
  强调"目的"——锁定未来某期的价值，避免损失
```

### 2.3 使用场景

| 场景 | 常用术语 |
|------|---------|
| **国际金融市场** | Hedging / Futures Hedge |
| **中国学术/官方** | 套期保值 / 套保 |
| **企业实务** | 套期保值 / 套保 |
| **金融教材** | 套期保值 |
| **新闻报道** | 套期保值 / 对冲（混用）|

## 三、套期保值的"四大原则"

### 3.1 原则一：品种相同或相近

```
套期保值的标的
  必须与现货品种**相同或高度相关**

  ✅ 正确：大豆现货 + 大豆期货
  ❌ 错误：大豆现货 + 豆粕期货（相关性 < 1）

  例外：
  - 跨品种套保（相关性 0.7+）
  - 跨市场套保（同品种不同市场）
  - 跨期套保（同品种不同到期月）
```

### 3.2 原则二：数量相等或相当

```
套期保值的数量
  应与现货数量**匹配**（考虑 β 系数）

  ✅ 正确：1000 吨大豆现货 + 1000 吨大豆期货
  ❌ 错误：1000 吨大豆现货 + 500 吨大豆期货

  例外：
  - β 系数调整
  - 部分套保（部分头寸对冲）
```

### 3.3 原则三：月份相同或相近

```
套期保值的月份
  应与现货交易时间**匹配**

  ✅ 正确：3 个月后卖出大豆 + 3 个月后到期期货
  ❌ 错误：3 个月后卖出大豆 + 1 个月后到期期货
  
  操作：
  - 1 个月期货到期 → 平仓
  - 换成 3 个月期货（roll over）
  - 一直保持到现货交易完成
```

### 3.4 原则四：交易方向相反

```
套期保值的方向
  必须与现货**相反**

  ✅ 正确：
    持有大豆现货（多头）→ 卖出大豆期货（空头）
    计划买入大豆（空头）→ 买入大豆期货（多头）
  
  ❌ 错误：
    持有大豆现货（多头）→ 买入大豆期货（多头）
    （这是"投机"或"加仓"，不是套保）
```

## 四、套期保值的"三种类型"

### 4.1 类型一：买入套期保值（多头套保）

**适用场景**：担心未来**价格上涨**

```
典型用户：
  - 加工企业：担心原料涨价
  - 贸易商：未来需要进货
  - 投资者：计划买入股票

操作：
  现货：空头（计划未来买入）
  期货：多头

效果：
  价格上涨 → 现货成本增加，期货盈利抵消
  价格下跌 → 现货成本减少，期货亏损抵消
```

### 4.2 类型二：卖出套期保值（空头套保）

**适用场景**：担心未来**价格下跌**

```
典型用户：
  - 生产商：担心产品跌价
  - 农场主：担心收获季节价格低
  - 投资者：持有股票担心下跌

操作：
  现货：多头（已持有）
  期货：空头

效果：
  价格下跌 → 现货损失，期货盈利抵消
  价格上涨 → 现货盈利，期货损失抵消
```

### 4.3 类型三：综合套期保值

**适用场景**：复杂风险管理需求

```
组合形式：
  - 部分套保（部分头寸对冲）
  - 滚动套保（持续展期）
  - 跨期套保（远近月对冲）
  - 跨品种套保（相关性套保）
```

## 五、套期保值的"四个关键公式"

### 5.1 套期保值比率

```
套期保值比率 = β 系数
  = Cov(ΔS, ΔF) / Var(ΔF)
  
  其中：
    ΔS = 现货价格变动
    ΔF = 期货价格变动

最简单情形：β = 1（完美对冲）
实际中：β < 1（不完全对冲）
```

### 5.2 套期保值数量

```
对冲数量 = 现货头寸 × β / 合约乘数

例：
  现货头寸 = 1000 万股股票
  β = 1.2（该股票比指数波动大 20%）
  沪深 300 期货合约乘数 = 300 元/点
  
  沪深 300 指数 = 4000 点
  合约价值 = 4000 × 300 = 120 万
  
  对冲手数 = (1000 万 × 1.2) / 120 万 = 100 手
```

### 5.3 套期保值有效性

```
套保有效性 = Var(未套保) - Var(套保后) / Var(未套保)
           = 风险降低比例

例：
  未套保组合年化波动率 = 20%
  套保后组合年化波动率 = 2%
  
  套保有效性 = (20² - 2²) / 20² = (400 - 4) / 400 = 99%
```

### 5.4 套期保值损益

```
套期保值净损益 = 现货损益 + 期货损益
             ≈ 基差变动 × 套保数量

例：
  基差从 +10 变为 +5（基差缩小 5）
  套保数量 1000 吨
  套保净损益 = -5 × 1000 = -5000 元（损失）
```

## 六、套期保值的"实际案例"

### 6.1 案例一：航空公司燃油套保

```
场景：
  航空公司预计 6 个月后需要 100 万吨航空煤油
  当前价格 = 6000 元/吨
  担心未来 6 个月油价上涨
       ↓
套保操作：
  在原油期货市场做多
  数量 = 100 万吨
  合约月份 = 6 个月后到期
       ↓
效果：
  油价涨 10%：
    现货成本：增加 100 万 × 600 = 6 亿元
    期货盈利：6 亿元
    净影响：0（锁定成本）
       ↓
  油价跌 10%：
    现货成本：减少 6 亿元
    期货亏损：6 亿元
    净影响：0（仍然锁定）
```

### 6.2 案例二：农场主小麦套保

```
场景：
  农场主预计 3 个月后收获 5000 吨小麦
  当前价格 = 2500 元/吨
  担心收获时价格下跌
       ↓
套保操作：
  在小麦期货市场做空
  数量 = 5000 吨
  合约月份 = 3 个月后到期
       ↓
效果：
  小麦跌 10%：
    现货收入：减少 5000 × 250 = 125 万元
    期货盈利：125 万元
    净影响：0
       ↓
  小麦涨 10%：
    现货收入：增加 125 万元
    期货亏损：125 万元
    净影响：0
```

### 6.3 案例三：上市公司股东减持套保

```
场景：
  上市公司大股东计划 6 个月内减持 1000 万股
  当前股价 = 50 元
  担心减持期间股价下跌
       ↓
套保操作：
  在沪深 300 期货做空
  数量 = 1000 万股 × β / 合约乘数
  假设 β = 1.0，合约乘数 = 300 元/点，指数 4000 点
  对冲手数 = 1000 万 / 120 万 = 约 83 手
       ↓
效果：
  大盘跌 10%：
    现货损失 ~5000 万（1000 万股 × 50 × 10%）
    期货盈利 ~5000 万
    净影响：~0
       ↓
  关键意义：
    大股东在"无市场风险"环境下完成减持
    保留个股选择能力
```

### 6.4 案例四：1998 香港金管局

```
金管局的"套期保值"具有特殊性：

传统套保：
  现货多头 + 期货空头 = 中性

金管局的"套保"：
  现货多头 + 期货多头 = 双向多头

这是"立体套保"：
  - 现货：买入恒指成分股
  - 期货：恒指期货做多
  - 长期持有：永久锁仓

详见 [[wiki/concepts/1998香港金融保卫战]]。
```

## 七、套期保值 vs 投机 vs 套利

### 7.1 三者的本质区别

| 维度 | 套期保值 | 投机 | 套利 |
|------|---------|------|------|
| 目的 | **降低风险** | 追求收益 | 无风险套利 |
| 现货头寸 | **有** | 无 | 有 |
| 期货头寸 | **与现货相反** | 单向 | 双向 |
| 净头寸 | **接近 0** | 单向 | 接近 0 |
| 风险 | 降低 | 高 | 极低 |
| 收益 | 锁定 | 期望收益 | 微小但稳定 |

### 7.2 中国监管的区分

```
中国监管机构（证监会）对三者的区分：

1. 套期保值：
   - 必须有现货头寸
   - 必须服务于"风险管理"
   - 期货头寸方向与现货相反
   - 受"套保额度"限制（持仓限制较宽）

2. 投机：
   - 没有现货头寸
   - 单纯追求收益
   - 期货头寸方向任意
   - 受"投机额度"限制（持仓限制较严）

3. 套利：
   - 跨市场/跨品种/跨期
   - 利用价差获利
   - 风险极低
   - 受"套利额度"限制
```

## 八、套期保值的"会计处理"

### 8.1 套期会计（Hedge Accounting）

```
会计准则（中国 2017 年新准则）：

1. 套期保值有效部分：
   - 期货损益直接计入"其他综合收益"（OCI）
   - 不影响利润表
   - 现货交易完成时转入损益

2. 套期保值无效部分：
   - 期货损益计入"投资收益"
   - 影响利润表
   - 需要单独披露

3. 套期关系终止：
   - 期货头寸平仓
   - 损益转入当期
```

### 8.2 套保的"披露要求"

```
中国 A 股上市公司：
  - 套期保值业务需在年报中披露
  - 披露套保规模、损益、有效性
  - 重大套保业务需董事会审议

例：
  "本公司 2024 年开展原材料套期保值业务，
   涉及金额 X 亿元，
   期货损益 Y 万元，
   套期保值有效性 Z%。"
```

## 九、套期保值的"风险与局限"

### 9.1 套期保值的局限

#### 局限一：基差风险

```
基差变动 = 套期保值的"剩余风险"

详见 [[wiki/concepts/基差风险]]。
```

#### 局限二：交易成本

```
套期保值的成本：
  - 手续费
  - 保证金占用
  - 资金成本
  - 会计成本
  
  例：
  套保 1 年的总成本
  = 1000 万 × 0.5% × 1 年 = 5 万元
  = 平均 4167 元/月
```

#### 局限三：流动性限制

```
某些品种的期货流动性差：
  - 无法以理想价格成交
  - 滑点增加
  - 套保成本上升
```

#### 局限四：品种限制

```
中国市场的套保限制：
  - 部分品种没有期货（如多数中小盘股票）
  - 部分品种流动性差
  - 跨品种套保相关性低
```

### 9.2 套保失败的原因

```
1. 方向错误：
   - 现货多头 + 期货多头（投机而非套保）

2. 数量错误：
   - 套保数量不匹配
   - 过度套保或套保不足

3. 时间错误：
   - 期货合约月份与风险期不匹配

4. 品种错误：
   - 套保品种与现货相关性低

5. 极端行情：
   - 期货市场流动性枯竭
   - 套保头寸无法平仓
```

## 十、套期保值在 1998 香港案例中的应用

### 10.1 金管局的"特殊性"

```
金管局的"套保"是"攻击性套保"：

传统套保（防守）：
  持有现货担心下跌 → 卖期货（防守）
  净头寸中性，风险被对冲

金管局的"套保"（攻击）：
  持有现货 + 买期货 = 双向多头
  净头寸多头，主动追求收益
       ↓
这种"套保"不是传统意义的套保
而是"立体攻击"的"防御部分"
```

### 10.2 金管局"套保"的成功要素

```
1. 时机选择：
   - 选择 8 月期指（临近期货结算日）
   - 8 月 28 日决战
   → 最优时机

2. 方向选择：
   - 期货做多（不是做空）
   - 与现货多头形成"双向多头"
   → 主动攻击

3. 数量选择：
   - 现货：30 只成分股
   - 期货：8 月期指
   → 数量匹配

4. 长期持有：
   - 9 年持有期
   - 持续维持轧空压力
   → 长期视角
```

## 十二、套期保值的"现代发展"

### 12.1 场外套保（OTC Hedging）

```
场外衍生品（OTC）：
  - 互换（Swaps）
  - 远期（Forwards）
  - 期权（Options）
  - 信用违约互换（CDS）

场外套保特点：
  - 定制化
  - 灵活性高
  - 流动性低
  - 信用风险高
```

### 12.2 动态套保（Dynamic Hedging）

```
动态套保：
  - 持续调整对冲头寸
  - 适应市场变化
  - 减少基差风险

详见 [[wiki/concepts/动态对冲]]。
```

### 12.3 量化套保（Quantitative Hedging）

```
量化套保：
  - 算法决定对冲头寸
  - 实时调整
  - 机器学习优化

例：
  投资公司用量化模型
  - 实时计算最优对冲比率
  - 自动调整头寸
  - 减少人为错误
```

## 十三、套期保值的"五个常见误解"

### 13.1 误解一：套保 = 零风险

```
错！

套保降低风险，但：
  - 不消除基差风险
  - 仍有流动性风险
  - 仍有交易成本
  - 仍有操作风险
```

### 13.2 误解二：套保 = 零收益

```
错！

套保"锁定"的不是"零收益"
而是"未来某期的价值"
如果选择时机好，套保也能获得收益
```

### 13.3 误解三：套保 = 投机

```
错！

套保和投机的本质区别：
  - 套保有现货头寸
  - 投机无现货头寸
  - 套保追求稳定
  - 投机追求高收益
```

### 13.4 误解四：套保越多越好

```
错！

过度套保：
  - 增加成本
  - 错失上涨收益
  - 复杂化操作
  
最优套保 = 适度对冲 + 保留选择权
```

### 13.5 误解五：套保 = 会计游戏

```
错！

套保是真实的经济行为
  - 影响现金流
  - 影响风险敞口
  - 影响公司价值
  
但套保有"会计游戏"的一面：
  - 不同会计处理影响利润
  - 套保会计的运用
  - 报表披露的优化
```

## 十四、套期保值的"决策框架"

### 14.1 是否套保的决策

```
决策框架：

1. 是否有"风险敞口"？
   - 持有现货（多/空）
   - 计划未来交易
   - 受市场风险影响

2. 套保成本是否合理？
   - 套保成本 < 风险价值
   - 套保不影响核心业务

3. 套保工具是否可得？
   - 有相应的期货/期权
   - 流动性足够
   - 合规允许

4. 套保后管理能力？
   - 监控基差
   - 调整头寸
   - 会计处理

5. 套保是否影响税务？
   - 中国对套保有税务规定
   - 套保损益的税务处理
```

### 14.2 套保的"五步操作"

```
五步操作法：

步骤 1：识别风险
  - 现货头寸是什么
  - 风险在哪里
  - 风险有多大

步骤 2：选择工具
  - 期货 vs 期权 vs 互换
  - 哪个相关性最高
  - 哪个成本最低

步骤 3：确定头寸
  - 计算套保数量
  - 考虑 β 系数
  - 考虑合约乘数

步骤 4：执行套保
  - 建立头寸
  - 监控基差
  - 必要时调整

步骤 5：结束套保
  - 现货交易完成
  - 期货头寸平仓
  - 评估套保有效性
```

## 十五、一句话总结

> **套期保值 = 通过期货市场对冲现货风险，本质是"金融保险"。**
>
> 它通过"**品种相同 + 数量相等 + 月份相近 + 方向相反**"的四大原则，把价格波动的风险"**转移**"给愿意承担风险的人（投机者）。套期保值是中文金融术语的标准表述，与"期货对冲"含义相同，但更强调"**保值的稳健性**"。
>
> 套期保值的本质是"**锁定未来某期的价值**"——它不追求"零风险"，而是追求"**已知的小风险**"代替"**未知的大风险**"。**1998 香港金管局**把套期保值升华为"立体攻击"——它不是"防守"，而是"主动追求收益"。**好的套期保值不是消除风险，而是把风险转化为可以管理的形式**。
