# 轧空（Short Squeeze）

> 借股票池枯竭 → 卖空者被迫买入 → 价格上涨 → 更多空头爆仓 —— 金融市场最经典的多空博弈

## 一、定义

**轧空**（Short Squeeze）是指**多方持有股票不释放**（或买方力量极强），导致**借股票池枯竭**，**卖空者无法在合理价位买入股票平仓**，被迫在更高价格买入，**进一步推高股价**，形成**正反馈循环**，最终**空头大规模爆仓**的市场现象。

核心本质：
- **卖空者**：必须买回股票 → 时间敌人
- **多方**：可以无限期持有 → 时间朋友
- **借股票池**：多空博弈的"水库"

## 二、轧空的形成机制

### 2.1 标准五阶段模型

```
阶段 1：卖空头寸积累
  - 卖空者大量建仓
  - 卖空比例上升
  - 借股票费开始上升

阶段 2：多方锁仓
  - 机构/大股东/散户拒绝借出股票
  - 借股票池开始萎缩
  - 借股票费飙升

阶段 3：触发临界点
  - 借股票池接近枯竭
  - 卖空者无法借到新股票
  - 已有空头面临"必须平仓"压力

阶段 4：强制平仓正反馈
  - 已有空头被迫在市场上买入
  - 买入推高股价
  - 更多空头触及止损线
  - 借股票费继续飙升
  - 借出方开始"召回"借出的股票
  → 进一步减少借股票池

阶段 5：轧空高潮
  - 价格暴涨
  - 空头大规模爆仓
  - 借股票费可能达到 100%-1000%+ 年化
  - 借股票池几乎完全枯竭
  → 轧空完成
```

### 2.2 完整的反馈循环

```
多方锁仓
  ↓
借股票池萎缩
  ↓
借股票费飙升
  ↓
新空头无法进入
  ↓
已有空头成本急增
  ↓
已有空头必须平仓
  ↓
买入推高股价
  ↓
借出方召回股票
  ↓
借股票池进一步枯竭
  ↓
更多空头被强制平仓
  ↓
股价继续上涨
  ↓
（循环回到"借股票费飙升"）
  ...
  ↓
轧空高潮
```

## 三、轧空的四个关键指标

### 3.1 卖空比例（Short Interest）

| 比例 | 含义 |
|------|------|
| < 5% | 正常 |
| 5%-10% | 略高 |
| 10%-20% | 警戒 |
| 20%-50% | 危险，轧空风险高 |
| > 50% | 极高，轧空随时可能 |
| > 100% | **必然轧空**（部分卖空头寸是裸卖空）|

### 3.2 Days to Cover（平仓天数）

```
Days to Cover = 卖空头寸总量 / 平均日成交量

含义：按当前成交量平仓需要多少天

< 1 天：流动性极好，正常
1-3 天：流动性正常
3-5 天：流动性略紧
5-10 天：流动性紧张
> 10 天：平仓极难，轧空风险高
```

### 3.3 借股票费（Stock Loan Fee）

| 借股票费 | 含义 |
|---------|------|
| < 1% | 借股票充足 |
| 1-5% | 正常 |
| 5-20% | 偏紧 |
| 20-100% | 紧张 |
| > 100% | 极度紧张，轧空风险高 |

### 3.4 股权集中度

| 集中度 | 含义 |
|--------|------|
| 分散 | 大量散户，流动性好 |
| 略集中 | 部分大股东，流动性正常 |
| 高度集中 | 少数大股东持有，轧空风险高 |

## 四、轧空的四种类型

### 4.1 类型一：自然轧空

| 维度 | 说明 |
|------|------|
| 触发 | 卖空比例高 + 基本面改善 |
| 速度 | 慢（数周-数月）|
| 案例 | 多家 2020-2021 中概股 |

### 4.2 类型二：基本面轧空

| 维度 | 说明 |
|------|------|
| 触发 | 公司回购/收购/私有化 |
| 速度 | 中（数天-数周）|
| 案例 | VW 2008、保时捷收购案 |

### 4.3 类型三：散户轧空

| 维度 | 说明 |
|------|------|
| 触发 | 散户联合买入 + 拒绝借出 |
| 速度 | 快（数天-数周）|
| 案例 | GameStop 2021 |

### 4.4 类型四：央行轧空

| 维度 | 说明 |
|------|------|
| 触发 | 央行/政府入市托市 |
| 速度 | 极快（数小时-数天）|
| 案例 | 1998 香港金管局 |

## 五、轧空的金融学本质

### 5.1 时间维度的多空博弈

```
多方优势：
  - 持有股票是"无限期"的（无时间成本）
  - 可以选择不卖（不参与博弈）
  - 可以选择不借（强化轧空）

空头劣势：
  - 借股票是有期限的（时间成本）
  - 必须平仓（无论价格）
  - 借股票费是"固定支出"（每天累积）
  - 借股票可能随时被召回
```

> **关键洞察**：空头是"**时间的朋友的对立面**"——多头的耐心越强，空头的成本越高。

### 5.2 流动性约束的"刚性与柔性"

```
刚性约束：
  - 借股票池有限（不可无限扩张）
  - 借股票需付费用（无法回避）
  - 卖空头寸必须平仓（无法回避）

柔性约束：
  - 借股票费可变（市场调节）
  - 借股票期限可延（但有成本）
  - 平仓可分批（但有冲击成本）
```

### 5.3 多空博弈的不对称性

```
多方：
  - 最大损失 = 0（股票可不卖）
  - 潜在收益 = 无限（股价可涨到天）
  - 时间成本 = 0

空头：
  - 最大损失 = 无限（股价可涨到天）
  - 潜在收益 = 100%（股价最多跌到 0）
  - 时间成本 = 借股票费 × 持仓天数
```

> **结论**：空头天然处于**风险/收益不对称**的劣势地位。轧空机制就是这种不对称性的极端表现。

## 六、经典案例

### 6.1 案例一：1998 香港金融保卫战（央行轧空）

```
背景：
  国际空头大举做空港股 + 港币
  恒指从 8000 点跌至 6500 点

金管局救市：
  8 月 14 日开始：
    - 大量买入恒指成分股
    - 在恒指期货市场做多
  8 月 28 日（期货结算日）：
    - 单日成交 80 亿美元（创纪录）
    - 国际空头爆仓

轧空机制：
  - 现货买入 → 流通股减少
  - 长期持有 → 借股票池萎缩
  - 期货做多 → 锁定未来价格
  - 双向夹击 → 强制平仓
  - 借股票费飙升 → 加剧爆仓
```

详见 [[wiki/concepts/1998香港金融保卫战]]。

### 6.2 案例二：Volkswagen 2008（基本面轧空）

```
背景：
  大众汽车流通在外股份很少
  卖空头寸约 12%（看似不高）
  
  实际上：
  保时捷通过期权累积了 74.1% 股权
  加上州政府 20%，总计 94.1%
  市场上可流通的股票：~5.99%

2008 年 10 月 26 日（周日）：
  保时捷宣布已通过期权控制大众 74.1% 股权
  
2008 年 10 月 27-28 日：
  空头无法买到股票平仓
  股价从 200 欧元 → 1000 欧元（2 天涨 5 倍）
  数十亿欧元空头头寸爆仓

关键点：股权高度集中 + 卖空头寸大 = 必然轧空
```

详见 [[wiki/concepts/Volkswagen轧空事件]]。

### 6.3 案例三：GameStop 2021（散户轧空）

```
背景：
  2020 年：GME 卖空头寸一度达到 140% 流通市值
  多家对冲基金（Melvin Capital）做空 GME

2020 年 12 月：
  WallStreetBets 论坛发现异常
  散户开始大量买入 GME

2021 年 1 月：
  散户拒绝借出股票给做空者
  借股票费从 1% 飙升至 1000%+
  
  股价从 4 美元 → 483 美元（涨 12000%+）
  
  后果：
  - Melvin Capital 亏损 53%
  - 多家对冲基金破产
  - 引发监管争议

关键点：散户"锁仓" + 卖空头寸过大 = 经典轧空
```

详见 [[wiki/concepts/GameStop轧空事件]]。

## 七、轧空对市场参与者的影响

### 7.1 对卖空者（空头）

| 影响 | 说明 |
|------|------|
| **爆仓风险** | 巨亏甚至破产 |
| **信誉受损** | 投资人撤资 |
| **职业生涯** | 重大失败可能终结 |
| **监管风险** | 可能被指控操纵市场 |

### 7.2 对多方

| 影响 | 说明 |
|------|------|
| **获利机会** | 把握轧空时点可获暴利 |
| **监管风险** | 联合锁仓可能构成操纵 |
| **声誉风险** | 散户"逼空"机构有争议 |

### 7.3 对市场

| 影响 | 说明 |
|------|------|
| **价格扭曲** | 短期价格严重偏离基本面 |
| **流动性冲击** | 借股票市场剧烈波动 |
| **监管反思** | 引发卖空规则修改 |
| **市场教育** | 提升参与者对卖空机制的理解 |

## 八、轧空的监管应对

### 8.1 监管的两难困境

```
轧空的两面性：
  ✅ 揭示卖空过度 → 防止泡沫
  ❌ 引发市场操纵 → 损害公平

监管的两难：
  放任轧空 → 价格扭曲、流动性冲击
  干预轧空 → 保护空头（让投机者赚钱）
```

### 8.2 监管的常见工具

| 工具 | 说明 | 效果 |
|------|------|------|
| **熔断机制** | 暂停交易 | 暂时缓解 |
| **禁止卖空** | 临时性禁止 | 强力但争议 |
| **提高保证金** | 增加做空成本 | 抑制做空 |
| **披露要求** | 公开卖空数据 | 信息透明 |

### 8.3 1998 香港 vs 2021 GME 的监管反应对比

| 维度 | 1998 香港 | 2021 GME |
|------|----------|----------|
| 监管反应 | **支持**救市 | **限制**散户交易 |
| 卖空规则 | 未变化 | 多次修改 |
| 多空态度 | 偏向多方 | 偏向空头（券商）|
| 后续影响 | 强化联系汇率 | Robinhood 限制买入引争议 |

## 九、轧空与相关概念的对比

### 9.1 轧空 vs 逼空

| 维度 | 轧空 | 逼空 |
|------|------|------|
| 性质 | 客观机制 | 主观策略 |
| 触发 | 借股票池枯竭 | 主动锁仓 |
| 主体 | 自然发生 | 主动发起 |
| 关系 | 概念更广 | 概念较窄 |

### 9.2 轧空 vs 流动性陷阱

| 维度 | 轧空 | 流动性陷阱 |
|------|------|-----------|
| 发生位置 | 股票市场 | 货币市场 |
| 参与者 | 散户/机构/央行 | 央行/商业银行 |
| 核心机制 | 借股票池枯竭 | 利率失效 |
| 政策应对 | 监管介入 | 非常规货币政策 |

### 9.3 轧空 vs 央行入市干预

| 维度 | 轧空 | 央行入市干预 |
|------|------|------------|
| 主体 | 任何多方 | 央行/政府 |
| 资金 | 自有资金 | 外汇基金/印钞 |
| 目的 | 击退空头 | 维护金融稳定 |
| 案例 | GME 2021 | 1998 香港 |

## 十、对投资策略的启示

### 10.1 识别轧空风险

```
预警信号（同时出现需高度警惕）：
  ⚠️ 卖空比例 > 20%
  ⚠️ 借股票费 > 10% 年化
  ⚠️ Days to Cover > 5 天
  ⚠️ 股权集中度高
  ⚠️ 多头叙事强劲
```

### 10.2 避免做空"高危股"

```
做空高危股的特征：
  - 卖空比例 > 20%
  - 借股票费 > 10% 年化
  - 散户关注度高
  - 公司回购计划
  - 收购传闻
  - 财报关键节点
```

### 10.3 把握轧空机会

```
参与轧空的方式：
  ✅ 提前布局（识别预警信号）
  ✅ 坚定持有（不被震荡洗出）
  ✅ 分批止盈（避免贪婪）
  ⚠️ 警惕监管介入
  ⚠️ 警惕"反向轧空"（轧多）
```

## 十一、一句话总结

> **轧空 = 借股票池枯竭 + 卖空者被迫买入 + 价格暴涨的正反馈**。
>
> 它的本质是"**时间维度的多空博弈**"——多方可以无限期持有（时间是朋友），空头必须平仓（时间是敌人）。当空头的"时间劣势"达到临界点，轧空就会发生。
>
> 轧空是**金融市场最戏剧性**的事件之一，它让空头爆仓、让多方狂欢、让监管反思。**1998 香港、2008 VW、2021 GME**——三个时代的三个经典案例，共同书写了轧空的历史。
