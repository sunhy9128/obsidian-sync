> 心理账户（Mental Accounting）由 Richard Thaler（1980）提出，指人们会在心智中**将不同来源、不同用途的金钱分类记账**，违反"金钱可替代性"原则，影响消费、投资和储蓄决策。

---

## 一、核心概念

### 1.1 定义

| 维度 | 内容 |
|------|------|
| 学术定义 | 个人或家庭在心智中对经济交易进行编码、分类、评估的过程，结果形成一套"心理账户系统" |
| 提出者 | Richard Thaler（1980, "Toward a Positive Theory of Consumer Choice"） |
| 核心违反 | 违反新古典经济学的"金钱可替代性"（Fungibility of Money） |
| 学派地位 | 行为经济学三大支柱之一（与前景理论、过度自信并列） |
| 奖项 | Thaler 2017 年获诺贝尔经济学奖 |

### 1.2 金钱可替代性 vs 心理账户

| 维度 | 标准经济学 | 行为经济学（心理账户） |
|------|----------|---------------------|
| 金钱性质 | 完全可替代（1 元 = 1 元） | 视来源/用途而定 |
| 100 元中奖 | 与 100 元工资无差异 | 比 100 元工资"更敢花" |
| 决策原则 | 综合财富最大化 | 分账户独立决策 |
| 数学表达 | U(W) 单一效用函数 | U(W_a, W_b, ...) 分账户效用 |

## 二、三大功能系统

### 2.1 Thaler 的三部分框架

```
1. 感知（Perception）
   - 将活动分类到不同账户
   - 例：工资 → 必要开支；奖金 → 享乐消费

2. 编辑（Editing）
   - 简化决策（合并、抵消、标注）
   - 例：年度旅游预算 5000 元

3. 评估（Evaluation）
   - 与参考点比较
   - 产生得/失的效用
```

### 2.2 心理账户的核心操作

| 操作 | 含义 | 示例 |
|------|------|------|
| 分类（Categorization） | 划分资金用途 | 工资、奖金、红包分别入账 |
| 标记（Labeling） | 给账户命名 | "旅游基金"、"教育金" |
| 预算设定（Budgeting） | 各账户设上限 | 餐饮月预算 2000 元 |
| 抵消（Cancellation） | 收益与损失配对 | 用年终奖抵消家电支出 |
| 分摊（Allocation） | 跨期资金分配 | 月供、季度缴费 |

## 三、经典案例

### 3.1 红包钱 vs 工资（Thaler 经典）

```
场景：
  - A：年底发 3000 元工资
  - B：过年收 3000 元红包
  → 同样是 3000 元，但 A 多用于储蓄，B 多用于消费
  → 红包钱"心理贬值"，被视为"意外之财"（house money）

延伸：
  - 彩票中奖 → 容易大手大脚花掉
  - 退税 → 容易当作"小钱"消费
  - 投资收益 → 部分人会"勇敢"加仓（用浮盈的钱）
```

### 3.2 沉没成本与账户封闭

```
案例：100 元电影票
  A：自掏 100 元买票 → 看了一半很烂 → 大多数继续看完
     （因"账户已扣 100 元"）
  B：免费获得电影票 → 看了一半很烂 → 大多数离场
  → 同样是 0 增量成本，但决策不同
  → 心理账户将"自付"和"免费"区分对待
```

### 3.3 混合损失与支付分离

```
商家技巧：让消费者分次感受"收益"和"损失"
  案例：超市购物返券
  - 实际降价 50 元 → 消费者感受"少"（直接损失）
  - 满 100 减 50 → 消费者分两次感受
    * 第一次：花 100 元（损失 100）
    * 第二次：减 50 元（收益 50）
  → 分次感受的"痛苦"低于一次性

消费者技巧：合并感受
  - 把所有月度消费合算 → 减少"分次痛苦"
```

### 3.4 信用卡悖论

```
经典现象：信用卡消费 > 现金消费
  原因：
  1. 现金支出有"物理痛感"（接触纸币）
  2. 信用卡支付无即时痛感
  3. 心理账户将"信用卡账户"与"现金账户"分开
  4. 月底统一还款时，痛苦被摊薄
  
  启示：减少信用卡可降低冲动消费
```

## 四、心理账户的投资应用

### 4.1 多账户陷阱

| 现象 | 表现 | 问题 |
|------|------|------|
| 股票账户独立 | 赚钱开心、亏钱也只关心账户 | 忽视整体组合 |
| 货币基金 vs 股票 | 两者本质上无差异 | 但被分账户管理 |
| 短炒 vs 长线账户 | 短炒浮亏 ≠ 真实亏 | 心理账户加剧处置效应 |

### 4.2 应对策略

```
原则：追求"整合账户"思维
  1. 定期计算总资产/总收益
  2. 资金一视同仁（不分来源）
  3. 关注机会成本而非分项
  4. 用资产配置代替择时
  5. 自动化投资减少人为分账户
```

### 4.3 心理预算的有益面

| 益处 | 说明 |
|------|------|
| 强制储蓄 | "旅游基金"账户避免挪用 |
| 财务目标 | 购房/教育金有专门账户更易达成 |
| 现金流管理 | 防止"月光"，分账户有纪律 |
| 应急准备 | 心理账户中"应急账户"确保流动性 |

```
新观点：合理利用心理账户是"行为助推"
  - 默认参与养老账户（默认开启）
  - 标注意外收入用途（"这个红包存起来"）
  - 工资分账自动转账
```

## 五、与其他概念的关系

### 5.1 与前景理论

| 概念 | 焦点 | 关系 |
|------|------|------|
| 前景理论 | 单次决策的得/失 | 心理账户将多次得/失分组 |
| 心理账户 | 多次决策的整合/分离 | 共同违反"整合公理" |

```
案例：股票亏 1 万 + 基金赚 1 万
  - 心理账户：分账户记录（股票 -1 万，基金 +1 万），两个账户都"已结案"
  - 前景理论：合并评估为 0，但实际感受是两个独立事件
```

### 5.2 与损失厌恶

- 心理账户决定"损失"如何被感知
- 分账户使每笔损失都"小而清晰"
- 整合账户才能"用收益抵消损失"

### 5.3 与锚定

- 账户预算上限可作"锚"
- "剩余 200 元预算"是消费决策的常见锚

## 六、行为助推（Nudge）的应用

### 6.1 Save More Tomorrow

```
Thaler & Benartzi (2004) 实验：
  - 让员工预先承诺"未来加薪的一部分用于储蓄"
  - 因锚定+现状偏见 → 参与率从 4% → 78%
  - 3 年后储蓄率从 3.5% → 11.6%
```

### 6.2 默认选项设计

| 场景 | 行为助推设计 | 原理 |
|------|------------|------|
| 401(k) 退休计划 | 默认参加而非"主动选择" | 现状偏见 |
| 器官捐献 | 默认"同意" | 现状偏见 + 损失厌恶 |
| 绿色能源 | 默认"绿色" | 心理账户中"环保"账户 |
| 订阅服务 | 默认续费 | 沉没成本 + 心理账户 |

### 6.3 反馈设计

- 提供"账户整合视图"（总资产变化曲线）
- 用游戏化方式让储蓄有"成就感"
- 减少月度账单数量（合并账户）

## 七、批评与争议

### 7.1 主要批评

| 批评 | 核心观点 |
|------|---------|
| "理性"账户 | 心理账户可能是有效记忆/规划工具 |
| 文化差异 | 西方"个人账户" vs 东亚"家庭账户" |
| 启发 vs 偏差 | 适度分类可能优于完全整合 |
| 实证难度 | 难以大样本随机实验验证 |

### 7.2 学术争议

```
Chevalier & Ellison (1997) 等：
  - 部分"心理账户"行为可由理性模型预测
  - 投资者可能"主动"分账户追求特定效用函数
  
反方观点：
  - 即使"工具性账户"是理性的，多数人并非有意
  - 实验情境下大量证据指向"系统性偏差"
```
