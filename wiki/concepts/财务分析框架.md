# 财务分析框架

> 从财务报表中解读企业基本面：盈利能力、成长性、财务健康度、现金流质量

财务分析是基本面研究的核心环节——在选定行业后，通过财报数据筛选和验证具体标的。三张表（利润表、资产负债表、现金流量表）构成完整的企业财务画像。

---

## 一、三张表的核心逻辑

```
利润表（流量）   资产负债表（存量）    现金流量表（流量）
├─ 收入           ├─ 资产（资金运用）   ├─ 经营现金流
├─ 成本           │  ├─ 流动资产         │  → 造血能力
├─ 费用           │  └─ 非流动资产       ├─ 投资现金流
├─ 利润           ├─ 负债（资金来源）     │  → 扩张意愿
└─ → EPS          │  ├─ 流动负债         └─ 筹资现金流
                  │  └─ 非流动负债         → 融资行为
                  └─ 所有者权益
                     └─ 净资产
```

**核心钩稽关系**：
- 利润表的净利润 → 进入资产负债表的留存收益（所有者权益）
- 利润表的收入 ≠ 现金流入（应收/预收的存在）
- 现金流量表的经营现金流 ≈ 利润的"含金量"验证
- 资产负债表的资产 = 负债 + 所有者权益（永恒等式）

## 二、关键财务比率体系

### 2.1 盈利能力

| 指标 | 公式 | 判断标准 | 说明 |
|------|------|----------|------|
| **毛利率** | (收入 - 成本) / 收入 | >30% 较优，>50% 优秀 | 反映定价权和成本控制，看趋势比看绝对值重要 |
| **净利率** | 净利润 / 收入 | 行业差异大 | 反映综合盈利效率 |
| **ROE（净资产收益率）** | 净利润 / 净资产 | >15% 优秀，>20% 卓越 | **最重要的综合指标**——衡量股东资本的盈利效率 |
| **ROA（总资产收益率）** | 净利润 / 总资产 | >5% 尚可 | 衡量全部资产的盈利效率 |
| **EBITDA 利润率** | EBITDA / 收入 | 行业比较用 | 消除折旧/摊销/资本结构影响 |

**杜邦分析**：ROE 的三重拆解（识别 ROE 的来源和风险）

```
ROE = 净利率 × 资产周转率 × 权益乘数
     ↑         ↑           ↑
  盈利效率    运营效率     杠杆水平
```

| 驱动类型 | 特征 | 代表行业 | 风险 |
|----------|------|----------|------|
| **高净利率驱动** | 高毛利、品牌壁垒 | 白酒（茅台 ROE 30%+） | 低风险、可持续 |
| **高周转驱动** | 薄利多销、运营能力强 | 零售（沃尔玛） | 经营效率下降风险 |
| **高杠杆驱动** | 借钱生钱、规模驱动 | 银行（ROE 10-12%但有杠杆加成） | ❌ 高杠杆不可持续，利率/资产质量变化冲击大 |

> **判断原则**：高净利率 + 高周转的 ROE 质量最优。高杠杆驱动的 ROE 需警惕——银行 ROE 虽高但加了 10x+ 杠杆。

---

### 2.2 成长性

| 指标 | 公式 | 判断标准 |
|------|------|----------|
| **收入增长率** | (本期收入 / 上期收入) - 1 | >10% 良好，>20% 优秀 |
| **净利润增长率** | (本期净利润 / 上期净利润) - 1 | 需与收入增速匹配 |
| **EPS 增长率** | (本期 EPS / 上期 EPS) - 1 | 股价的长期驱动因子 |
| **扣非净利润** | 剔除一次性项目 | 比净利润更能反映真实经营趋势 |

**成长质量检查**：
- 利润增速 > 收入增速 → 利润率提升（好）
- 收入增速 > 利润增速 → 竞争加剧/费用扩张（需验证）
- 利润增长但经营现金流下降 → 利润质量存疑（警惕）

---

### 2.3 财务健康度

**偿债能力**：

| 指标 | 公式 | 安全线 | 说明 |
|------|------|--------|------|
| 流动比率 | 流动资产 / 流动负债 | >1.5 | 短期偿债能力 |
| 速动比率 | (流动资产 - 存货) / 流动负债 | >1.0 | 去除变现慢的存货 |
| 资产负债率 | 总负债 / 总资产 | 行业差异大 | <50% 稳健，>70% 偏高 |
| 利息保障倍数 | EBIT / 利息费用 | >3 | 利润覆盖利息的能力 |

**A 股关注重点**：
- **高存高贷**：账面巨额现金 + 巨额有息负债 = 可能资金被占用或造假信号
- **有息负债率**：短期借款 + 长期借款 + 应付债券 / 总资产，比资产负债率更反映真实债务压力
- **隐性负债**：对外担保、未决诉讼、表外负债（影子银行相关）

---

### 2.4 现金流质量

| 模式 | 含义 | 评价 |
|------|------|------|
| 经营 + / 投资 - / 筹资 +/- | 主业造血 + 投资扩张 | ✅ 成熟成长期企业 |
| 经营 + / 投资 + / 筹资 - | 主业造血 + 收缩 + 还债 | ⚠️ 收缩期，关注衰退原因 |
| 经营 - / 投资 - / 筹资 + | 亏损 + 靠融资扩张 | ❌ 烧钱模式，关注现金流断裂风险 |
| 经营 - / 投资 + / 筹资 - | 亏损 + 变卖资产 + 还债 | ❌ 极度危险信号 |

**关键验证公式**：

```
经营性现金流 / 净利润 > 1 → 利润质量高（赚的是真钱）
经营性现金流 / 净利润 < 0.5 → 利润质量低（应收/存货堆积）
自由现金流（FCF）= 经营现金流 - 资本开支 → 真金白银的回报能力
```

## 三、A 股财务分析特色

### 3.1 财报季节性与节奏

| 报告 | 截止 | 披露期 | 重要性 |
|------|------|--------|--------|
| 一季报（Q1） | 3/31 | 4/30 前 | 全年定调 |
| 中报（H1） | 6/30 | 8/31 前 | 验证上半年、预示下半年 |
| 三季报（Q3） | 9/30 | 10/31 前 | 修正全年预期 |
| 年报（Q4） | 12/31 | 次年 4/30 前 | **最重要**，审计报告、完整数据 |

### 3.2 常见财务陷阱

- **业绩洗澡**：大额一次性减值（商誉、存货），为下年低基数高增长铺路
- **应收账款虚增收入**：收入高增长但应收同步暴增 = 可能向渠道压货
- **现金流与利润背离**：连续多年净利润为正但经营现金流为负 = 高度警惕
- **关联交易调节利润**：通过关联方进行非公允交易平滑利润
- **商誉减值**：收购形成的商誉在业绩不达标时一次性减值计提

### 3.3 监管关注信号

交易所问询函中常见的财务质疑方向（这些也是投资者应关注的预警）：

- 毛利率明显高于同行且无合理解释
- 收入确认时点与行业惯例不一致
- 大额异常交易缺乏商业实质
- 关键财务数据与业务数据（水电费、员工数、产能）不匹配

## 四、财务筛选实战框架

### "四步筛选法"

```
第 1 步：排除劣质
├─ 扣非净利润连续 2 年亏损 → 淘汰
├─ 经营现金流连续 2 年为负 → 高度警惕
├─ 资产负债率 > 80%（非金融）→ 谨慎
├─ 审计意见非标准无保留 → 淘汰
└─ 大股东质押比例 > 80% → 警惕

第 2 步：质量排名
├─ ROE > 15%（核心指标）
├─ 毛利率 > 30% 或 行业前 1/3
├─ 经营现金流 / 净利润 > 0.8
└─ 收入增长率 > 10%

第 3 步：估值验证
├─ PE / PEG 在行业合理区间
├─ 对比自身历史估值分位
└─ 股息率 > 国债收益率 × 0.5（价值股底线）

第 4 步：催化因素
├─ 业绩趋势改善（拐点）
├─ 行业景气度上行
├─ 新业务/新产品兑现
└─ 政策/改革红利
```
