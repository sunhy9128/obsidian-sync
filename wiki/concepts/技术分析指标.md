# 技术分析指标

> 基于价格、成交量等市场数据的量化工具，辅助判断趋势与买卖时机

技术分析指标建立在[[道氏理论]]和[[艾略特波浪理论]]的趋势框架之上，将价格行为转化为可量化的信号。按功能分为四大类：**趋势类**、**震荡类**、**量能类**、**波动率类**。

---

## 一、趋势类指标

用于识别和确认趋势方向，在趋势市场中表现最佳。

### 移动平均线（MA）

最基础的趋势指标，有简单移动平均（SMA）和指数移动平均（EMA）两种。

| 类型 | 常用参数 | 用途 |
|------|----------|------|
| 短期均线 | MA5, MA10, MA20 | 短线趋势，A 股 5 日/10 日为经典组合 |
| 中期均线 | MA50, MA60 | 中期趋势，60 日线常被视为"生命线" |
| 长期均线 | MA120, MA200, MA250 | 牛熊分界线（250 日为年线） |

**核心用法**：
- **金叉/死叉**：短线上穿长线为金叉（买入信号），下穿为死叉（卖出信号）
- **多头/空头排列**：短期 > 中期 > 长期为多头排列（强势）；反之为空头排列（弱势）
- **葛兰碧八大法则**：价格与均线的八种关系对应买卖信号（最经典的均线交易系统）

### MACD（指数平滑异同移动平均线）

由 Gerald Appel 在 1970 年代提出，是均线系统的进化——用快慢两条 EMA 的差值来反映趋势动量的变化。

**组成**：
- **DIF 线**：快线（通常 EMA12）与慢线（通常 EMA26）之差
- **DEA 信号线**：DIF 的 9 日 EMA
- **柱状图（MACD Bar）**：DIF 与 DEA 的差值

**核心用法**：

| 信号 | 含义 | 说明 |
|------|------|------|
| DIF 上穿 DEA（金叉） | 买入信号 | 零轴上方金叉更强（多头确认），零轴下方金叉偏弱（反弹） |
| DIF 下穿 DEA（死叉） | 卖出信号 | 零轴下方死叉更凶 |
| **顶背离** | 趋势可能见顶 | 价格创新高但 DIF 未创新高 → 浪 5 末端典型信号 |
| **底背离** | 趋势可能见底 | 价格创新低但 DIF 未创新低 → 浪 3 或 C 浪末端常见 |
| 零轴穿越 | 多空转换 | DIF 上穿零轴 = 转多，下穿零轴 = 转空 |

> **背离是 MACD 最有价值的功能**，也是确认[[艾略特波浪理论]]中浪 5 末端的重要辅助工具。

#### MACD 参数调整与适用周期

默认参数 (12,26,9) 源自 Gerald Appel 针对美股的研究。不同参数组合适用不同周期：

| 参数 (快,慢,信号) | 灵敏度 | 适用场景 | 说明 |
|-------------------|--------|----------|------|
| (12,26,9) | 标准 | 日线、中线趋势 | 默认参数，最通用 |
| (6,13,5) | 高 | 短线/ T+1 博弈 | A 股短线常用，信号频繁但假信号多 |
| (5,35,5) | 低 | 周线/长线趋势 | 过滤噪音，适合大级别趋势判断 |
| (21,55,9) | 极低 | 月线级别（A 股看牛熊） | 趋势最稳，极少交易信号 |

**A 股实战推荐**：
- 日线用 (12,26,9)：中线趋势跟踪
- 60 分钟用 (6,13,5)：短线买卖点
- 周线用 (5,35,5)：判断牛熊大方向

> **核心原则**：先用长周期参数定方向（周线），再用短周期参数找入场点（60 分/日线）。

#### MACD 背离的四种形态

背离是 MACD 最有价值的信号，但传统"线背离"在强趋势中容易反复失效。完整的背离体系包括：

| 背离形态 | 识别方法 | 可靠性 | 说明 |
|----------|----------|--------|------|
| **线背离** | 价格新高/低，DIF 未新高/低 | ★★★☆ | 最常见，但在强势趋势中容易出现二次/三次背离 |
| **柱背离** | 价格新高/低，MACD 红/绿柱面积缩小 | ★★☆☆ | 比线背离出现早，但假信号更多 |
| **面积背离** | 价格新高/低，MACD 红/绿柱累计面积缩小 | ★★★★ | 量化的背离测量，可靠性较高 |
| **趋势线背离** | 价格突破趋势线，DIF 未同步突破 | ★★★★ | 结合趋势线过滤，假信号较少 |

**双周期确认法**（最实用的过滤方法）：

```
日线级别出现底背离信号
  └→ 等待 60 分钟线也出现底背离（或金叉确认）
      └→ 此时入场，胜率显著提高
```

#### MACD 柱状图分析

柱状图的方向变化比 DIF 线拐点更敏感，可提前发现趋势衰减：

| 柱状图状态 | 含义 |
|-----------|------|
| 红柱伸长 | 多头加速（持仓） |
| 红柱缩短 | 多头衰减（警惕） |
| 绿柱伸长 | 空头加速（观望） |
| 绿柱缩短 | 空头衰减（关注买入机会） |
| 绿柱翻红柱 | DIF 上穿 DEA（金叉确认） |
| 红柱翻绿柱 | DIF 下穿 DEA（死叉确认） |

> **A 股特色用法**：MACD 柱状图在强势涨停行情中会快速扩张——连续大阳线后红柱开始缩短但价格仍在涨 → 滞涨信号，次日容易出现高开低走。

#### 零轴的多空意义

| 位置 | 含义 | 操作含义 |
|------|------|----------|
| DIF 在零轴上方 | 多头市场（中期趋势向上） | 回调做多为主 |
| DIF 在零轴下方 | 空头市场（中期趋势向下） | 反弹做空/观望 |
| DIF 从下方上穿零轴 | 空翻多确认 | 中线建仓信号 |
| DIF 从上方下穿零轴 | 多翻空确认 | 中线离场信号 |

#### MACD 单指标交易系统（日线）

```
买入:
1. MACD 底背离出现
2. DIF 上穿 DEA（金叉）
3. 柱状图翻红
信号确认 → 开仓

止损:
- 底背离结构被破坏（价格创新低且 DIF 同步新低）
- 或 MACD 再次死叉

卖出:
1. MACD 顶背离出现
2. DIF 下穿 DEA（死叉）
3. 柱状图翻绿
信号确认 → 平仓
```

## 二、震荡类指标

用于识别超买/超卖状态和潜在反转点，在震荡市中表现最佳。

### RSI（相对强弱指数）

由 Welles Wilder 在 1978 年提出，测量一段时间内价格上涨幅度占总波动的比例。

**计算**：
```
RSI = 100 - [100 / (1 + 平均涨幅 / 平均跌幅)]
```

**参数与阈值**：

| 参数 | 超买 | 超卖 | 说明 |
|------|------|------|------|
| RSI(6) | >80 | <20 | 短线敏感，A 股常用 |
| RSI(14) | >70 | <30 | 默认参数，中线参考 |
| RSI(20) | >65 | <35 | 大级别趋势，周线级别 |

**核心用法**：
- **超买/超卖回归**：RSI 进入超买区 ≠ 立即卖出，强势市场中可长时间维持在超买区
- **背离**：价格新高但 RSI 未新高（顶背离，看跌）；价格新低但 RSI 未新低（底背离，看涨）
- **A 股特征**：RSI 在强势行情中容易钝化（持续超买），需结合趋势线过滤

#### RSI 钝化与处理

**钝化定义**：RSI 长时间停留在超买区（>80）或超卖区（<20），失去正常参考意义。

| 钝化类型 | 发生条件 | 应对方法 |
|----------|----------|----------|
| 牛市钝化（超买区横盘） | 强势上涨行情，RSI 持续 >80 | 忽略 RSI，转为用 MACD 跟踪趋势 |
| 熊市钝化（超卖区横盘） | 持续阴跌，RSI 连续 <20 | 不急于抄底，等待 RSI 回到 30 上方 |
| 急涨钝化 | 连续涨停/大阳线 | RSI 瞬间触顶，无意义 |

**A 股实战示例**：

```
RSI < 20: 进入超卖区，但！！！
  不一定立即反弹。A 股的特点是超卖可以持续 3-5 天或更久。
  ✅ 正确用法：RSI < 20 → 关注，等 RSI 再次回到 >20 时入场
  ❌ 错误用法：RSI < 20 就抄底

RSI > 80: 进入超买区
  在强势股中，RSI > 80 是常态（强者恒强）。
  ✅ 正确用法：RSI > 80 + MACD 红柱缩短 → 减仓
  ❌ 错误用法：RSI > 80 就卖出
```

#### RSI 形态与趋势线

RSI 的 K 线图形态在识别背离时比价格更清晰：

- **RSI 头肩顶 / 头肩底**：价格图震荡不易判断时，RSI 图若呈现清晰的头肩顶 → 确认顶部。RSI 的形态突破比价格形态突破通常快 1-3 天发生
- **RSI 趋势线突破**：连接 RSI 的高点/低点画趋势线，RSI 突破下降趋势线 → 价格随后突破。RSI 趋势线突破是识别**先行信号**的好工具——很多情况下 RSI 先于价格突破趋势线

#### RSI 分区交易策略

| RSI 区间 | 市场状态 | 策略 |
|----------|----------|------|
| 80-100 | 超买区间 | 不追多；若出现柱背离/顶背离 → 减仓 |
| 50-80 | 多头强势 | 顺势做多（在 50 附近止跌反弹是加仓点） |
| 30-50 | 空头弱势 | 观望或短线做空（反弹到 50 附近受压制是减仓点） |
| 0-30 | 超卖区间 | 不杀跌；若出现底背离/金叉 → 关注买入 |

### KDJ（随机指标）

由 George Lane 创立，通过比较收盘价与一定时期内价格区间的位置来判断超买超卖。

**组成**：K 值（快线）、D 值（慢线）、J 值（K-D 的 3 倍差值）

**参数**：常用 (9,3,3)，A 股短线常调整为 (5,3,3) 以提高灵敏度

**核心用法**：
- K 上穿 D = 买入信号（在低位 <20 更可靠）
- K 下穿 D = 卖出信号（在高位 >80 更可靠）
- **A 股特色**：KDJ 在散户中使用极广，存在自我应验效应——很多人同时买卖导致信号短暂有效后反转

#### KDJ 计算逻辑

```
1. RSV = (收盘价 - N 日内最低) / (N 日内最高 - N 日内最低) × 100
2. K 值 = 2/3 × 前日 K + 1/3 × RSV（平滑）
3. D 值 = 2/3 × 前日 D + 1/3 × K（再次平滑）
4. J 值 = 3 × K - 2 × D（反映 K/D 偏离程度）
```

默认参数 (9,3,3) 中：N=9 为 RSV 计算周期（9 天价格区间），第一个 3 为 K 的平滑系数，第二个 3 为 D 的平滑系数。

#### J 值的特殊用法

J 值是 KDJ 相比其他震荡指标最独特的功能——它可以偏离常规区间：

| J 值 | 含义 | 信号强度 |
|------|------|----------|
| J > 100 | 极度超买，严重偏离均值 | ★★★☆ 短线回调概率高 |
| J < 0 | 极度超卖，严重偏离均值 | ★★★☆ 短线反弹概率高 |
| J 从 >100 回落 | 短线顶部信号 | ★★☆☆ 需配合价格确认 |
| J 从 <0 回升 | 短线底部信号 | ★★☆☆ 需配合价格确认 |

> **A 股特色用法**：A 股超短线交易中常用 KDJ(5,3,3) 配合 J 值判断次日涨跌概率——J < 0 的品种次日反弹概率 >60%（非保证，仅为概率参考）。但这种用法的有效性正随着市场效率提升而下降。

#### KDJ 与 RSI 的本质区别

| 维度 | KDJ | RSI |
|------|-----|-----|
| 衡量对象 | 收盘价在价格区间的位置 | 上涨与下跌力量的相对强度 |
| 敏感度 | 更敏感（含 J 值可追踪极端） | 相对平滑 |
| 极致信号 | J 值可到 >100 或 <0，提供极端信号 | RSI 区间 0-100，极致钝化 |
| 适用场景 | 震荡市（高抛低吸） | 趋势+震荡均可用 |

#### KDJ 钝化与 A 股应对

KDJ 的钝化问题比 RSI 更严重——因为 KDJ 是"区间位置"指标，一旦价格持续上涨，K/D 值会长时间粘在 80-100 区间。

| 应对策略 | 方法 | 适用 |
|----------|------|------|
| 切换到长周期 | 日线钝化→看周线 KDJ | 中线持仓 |
| 改用 MACD | 当 KDJ 持续高位时，放弃 KDJ 信号 | 强势趋势股 |
| 只看 J 值极端 | K/D 钝化时仅关注 J 值是否 >100 或 <0 | 超短线 |
| KDJ + 均线配合 | KDJ 金叉发生在 MA20 上方 → 有效信号；MA20 下方 → 反弹 | 趋势过滤 |

#### KDJ 金叉/死叉分级

| 金叉类型 | 位置 | 可靠性 |
|----------|------|--------|
| 低位金叉 | K/D < 20, J < 0 后金叉 | ★★★★★ 最佳买入信号 |
| 中位金叉 | K/D 在 20-50 金叉 | ★★★☆☆ 适中，需确认趋势 |
| 高位金叉 | K/D > 80 金叉 | ★★☆☆☆ 容易钝化+假金叉 |
| 顶部背离金叉 | 价格新低 + KDJ 未新低 | ★★★★☆ 反转信号 |

| 死叉类型 | 位置 | 可靠性 |
|----------|------|--------|
| 高位死叉 | K/D > 80 后死叉 | ★★★★★ 最佳卖出信号 |
| 中位死叉 | K/D 在 50-80 死叉 | ★★★☆☆ 适中 |
| 低位死叉 | K/D < 20 死叉 | ★★☆☆☆ 低位杀跌容易踏空 |

## 三、量能类指标

### 成交量（Volume）

最简单的量能指标，但信息量最大。

**核心关系**（配合[[道氏理论]]成交量验证原则）：

| 现象 | 含义 |
|------|------|
| 价升量增 | 趋势健康，多头持续 |
| 价升量缩 | 上涨乏力，可能出现回调（顶背离预警） |
| 价跌量增 | 恐慌抛售，可能接近底部（底部放量） |
| 价跌量缩 | 正常调整，缩量整理（抛压衰竭） |
| 天量天价 | 成交量创历史新高时价格常在顶部区域 |

**A 股特色指标**：

| 指标 | 说明 | 参考值 |
|------|------|--------|
| **换手率** | 成交股数 / 流通股数 | A 股正常 1-3%，活跃 >5%，过热 >10% |
| **量比** | 当前成交量 / 过去 5 日均量 | <0.5 极度缩量，>2 显著放量 |
| **北向资金** | 沪股通 + 深股通净买入 | A 股特色——北向持续流入是重要看多信号 |
| **融资余额** | 融资买入的累计余额 | 融资余额上升 = 杠杆资金看多 |

## 四、波动率类指标

### BOLL（布林带）

由 John Bollinger 在 1980 年代创立，基于均线和标准差构建的价格通道。

**组成**：
- **中轨**：MA20（基础均线）
- **上轨**：中轨 + 2×标准差
- **下轨**：中轨 - 2×标准差

**核心用法**：

| 现象 | 含义 |
|------|------|
| 价格触及上轨 | 短期超买，但强势中价格可沿上轨运行 |
| 价格触及下轨 | 短期超卖，需结合趋势确认方向 |
| **带宽收缩（Squeeze）** | 波动率低点→突破在即（横有多长竖有多高） |
| 带宽扩张 | 趋势正在加速，顺势操作 |
| 价格沿上轨 / 下轨运行 | 趋势强劲，不宜逆势开仓 |

**A 股适用性**：BOLL 在 A 股识别突破信号效果较好——带宽极度收缩后常伴随方向性行情。但需注意带宽扩张也可能出现在行情末端（诱多/诱空）。

## 五、指标选择与组合策略

### 不同市场状态的指标选择

| 市场状态 | 首选指标 | 应避免 |
|----------|----------|--------|
| 趋势上涨 | MACD（趋势跟踪）、MA（多排确认） | RSI、KDJ（过早超买信号） |
| 趋势下跌 | MACD（底背离判断）、量比（恐慌抛售） | KDJ（钝化失去参考价值） |
| 横盘震荡 | RSI、KDJ（高抛低吸信号） | MACD（频繁金叉死叉，噪音极大） |
| 剧烈波动 | BOLL（带宽判断极端位置） | 均线系统（滞后严重） |
| 单边上涨 | MACD（趋势跟踪） | 均线（辅助）、KDJ/RSI（钝化无用） |
| 单边下跌 | MACD（底背离） | RSI（超卖判断辅助）、KDJ（钝化） |
| 底部反转 | MACD（底背离+金叉） | RSI（底背离辅助） |
| 顶部反转 | MACD（顶背离+死叉） | RSI（顶背离辅助） |
| 涨停/强势股 | MACD（跟踪趋势） | 成交量（辅助）、KDJ（严重钝化） |

### 经典组合

1. **道氏 + MACD + 成交量**：判断趋势方向 → 确认动量 → 验证量价关系
2. **波浪 + MACD 背离**：用 MACD 背离辅助数浪（浪 5 末端 + 顶背离 = 强烈反转信号）
3. **BOLL + 成交量 + RSI**：带宽收缩 + 放量突破 + RSI 中性区域抬头 → 突破确认

### 三指标信号权重打分

| 维度 | MACD | RSI | KDJ |
|------|------|-----|-----|

### 三指标组合信号矩阵

**做多信号（三个指标同时指向）**：

```
MACD: 零轴上金叉 / 底背离
RSI:   从超卖区回升 / 底背离
KDJ:   低位金叉 / J 值从 <0 回升
→ 信号强度最大化，强烈做多信号
```

**做空/减仓信号**：

```
MACD: 顶背离 / 死叉
RSI:   顶背离 / 从超买区回落
KDJ:   高位死叉 / J 值从 >100 回落
→ 卖出减仓
```

## 常见误区

- **指标越多越好？** 错。2-3 个相互验证的指标远好于 5-6 个互相矛盾的指标
- **技术指标可以预测未来？** 不能。指标只是对过去价格的概率性总结
- **金叉了就一定涨？** 不是。震荡市中金叉死叉频繁反复，胜率大幅下降
- **背离就是反转信号？** 背离仅提示趋势减缓，不一定立即反转——可能出现二次背离、三次背离
- **A 股技术面更有效？** 散户占比高确实使技术信号在短期更明显，但政策干预可瞬间打破所有技术形态
- **RSI 超卖就抄底？** A 股超卖可维持 3-5 天或更久，RSI < 20 仅是关注信号，不是入场信号
- **KDJ 金叉/死叉都同等可靠？** 不是。按位置分级——低位金叉/高位死叉最可靠，高位金叉/低位死叉常为陷阱
