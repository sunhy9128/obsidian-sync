> 现代投资组合理论（Modern Portfolio Theory, MPT）由 Harry Markowitz 于 1952 年提出，其核心思想是：在给定预期收益下，通过资产配置降低组合方差；或在给定方差下最大化预期收益，实现风险与收益的最优权衡。

---

## 一、理论框架

### 1.1 两大核心假设

| 假设 | 内容 |
|------|------|
| 预期收益用均值度量 | 投资者关注 E(R) |
| 风险用方差/标准差度量 | 投资者关注 σ²(R) 或 σ(R) |

### 1.2 投资组合的两个维度

```
            E(R) ↑
            │      ★ 有效前沿（Efficient Frontier）
            │     ╱
            │    ╱     ← 给定风险下收益最高
            │   ╱
            │  ╱
            │ ╱
            │╱__________→ σ(R) 风险
```

### 1.3 Markowitz 贡献链

```
Markowitz (1952) MPT  ──→  Sharpe (1964) CAPM
        │                       │
        └─→ 均值-方差            └─→ 资本市场线 (CML)
            优化求解                  引入无风险资产
```

## 二、组合预期收益与方差

### 2.1 组合预期收益

```
E(Rp) = Σ wi × E(Ri)     (i = 1, ..., n)

简单示例（两资产等权）：
E(Rp) = 0.5 × E(R_A) + 0.5 × E(R_B)
```

### 2.2 组合方差

```
σ²_p = Σᵢ Σⱼ wᵢ × wⱼ × σᵢⱼ
     = Σᵢ wᵢ² σᵢ² + 2 Σᵢ<ⱼ wᵢ wⱼ Cov(Rᵢ, Rⱼ)

矩阵形式：
σ²_p = wᵀ Σ w
其中 Σ 为协方差矩阵，w 为权重向量
```

### 2.3 两资产组合方差展开

```
σ²_p = w₁² σ₁² + w₂² σ₂² + 2 w₁ w₂ ρ σ₁ σ₂

参数：
  w₁, w₂   资产权重
  σ₁, σ₂   资产标准差
  ρ         相关系数（-1 ≤ ρ ≤ 1）
```

### 2.4 数值示例

假设 A、B 两资产：

| 资产 | E(R) | σ | w |
|------|------|---|---|
| A | 10% | 20% | 0.5 |
| B | 6% | 10% | 0.5 |

ρ = -0.5 时的组合：

```
E(Rp) = 0.5×10% + 0.5×6% = 8.0%
σ²_p = 0.25×0.04 + 0.25×0.01 + 2×0.5×0.5×(-0.5)×0.2×0.1
     = 0.01 + 0.0025 - 0.005
     = 0.0075
σ_p   = √0.0075 ≈ 8.66%
```

## 三、相关系数的影响

### 3.1 相关系数对组合风险的影响

| 资产 A σ | 资产 B σ | 权重 | ρ | 组合 σ |
|-----------|---------|------|---|--------|
| 20% | 10% | 0.5/0.5 | +1.0 | 15.00% |
| 20% | 10% | 0.5/0.5 | +0.5 | 13.23% |
| 20% | 10% | 0.5/0.5 | 0.0 | 11.18% |
| 20% | 10% | 0.5/0.5 | -0.5 | 8.66% |
| 20% | 10% | 0.5/0.5 | -1.0 | 5.00% |

> 当 ρ=1 时，组合 σ = 加权平均 σ（无分散化收益）
> 当 ρ=-1 时，组合 σ 可降至 0（完美对冲）

### 3.2 分散化的数学逻辑

```
ρ(i,j) = Cov(Ri, Rj) / (σᵢ × σⱼ)

当资产数量 n → ∞，等权组合：
σ²_p = (1/n) × σ̄² + (1 - 1/n) × σ̄²_corr
                          ↑
                  资产间协方差（系统风险）
```

### 3.3 系统性风险 vs 特质风险

| 类别 | 来源 | 可分散 |
|------|------|--------|
| 系统性风险 | 整个市场共有的风险（β 风险） | 不可分散 |
| 特质风险 | 个别公司、行业特有风险 | 可通过分散消除 |

```
组合方差分解（n 个等权资产）：
σ²_p = (1/n) × σ²_e + (n-1)/n × Cov_avg
        ↑                   ↑
     个体方差（可分散）    平均协方差（不可分散）
```

## 四、协方差矩阵

### 4.1 矩阵示例（3 资产）

| | A | B | C |
|---|---|---|---|
| A | 0.0400 | 0.0120 | -0.0080 |
| B | 0.0120 | 0.0225 | 0.0060 |
| C | -0.0080 | 0.0060 | 0.0900 |

> 对角线为各资产方差，非对角线为协方差。

### 4.2 实际估算的挑战

| 挑战 | 影响 |
|------|------|
| 样本期选择 | 不同窗口得到不同协方差 |
| 估计误差 | 协方差估计易受极端值影响 |
| 协方差稳定 | 实际协方差随时间漂移 |
| 数据需求 | n 资产需要 n(n+1)/2 个参数 |

> "维度灾难"：50 个资产需要估算 1225 个协方差，估计精度有限。

## 五、最小方差组合

### 5.1 定义

```
min  σ²_p = wᵀ Σ w
s.t. Σ wᵢ = 1
```

### 5.2 解析解

```
w_minvar = (Σ⁻¹ 𝟏) / (𝟏ᵀ Σ⁻¹ 𝟏)

其中 𝟏 为全 1 向量
```

### 5.3 数值示例（3 资产）

使用上面 3 资产的协方差矩阵，最小方差组合权重可能是：
- A: 60%, B: 35%, C: 5%
- 组合方差：约 0.0076
- 组合标准差：约 8.72%

> 最小方差组合的预期收益通常不是最高的，是风险厌恶投资者的保守选择。

## 六、有效前沿

### 6.1 定义

```
有效前沿 = 给定风险水平下，预期收益最大化的所有组合的集合
        = 给定预期收益下，风险（方差）最小化的所有组合的集合
```

### 6.2 完整可行域

```
E(R) ↑
  │        ╱  最小方差
  │       ╱     ↑
  │      ●       │ 无效集
  │     ╱│       │（次优）
  │    ╱ │       │
  │   ╱  │      ╱
  │  ╱   │     ╱
  │ ╱    │    ╱
  │╱     │   ╱
  └──────┼──┼──────────→ σ(R)
         ↑  ↑
        有效前沿
```

### 6.3 两资产有效前沿

| 权重 (A, B) | E(R) | σ | 是否有效 |
|-------------|------|---|----------|
| (0, 1) | 6% | 10% | 否 |
| (0.25, 0.75) | 7% | 8.84% | 否 |
| (0.5, 0.5) | 8% | 8.66% | 否 |
| (0.75, 0.25) | 9% | 9.74% | 是 |
| (1, 0) | 10% | 20% | 是 |

## 七、引入无风险资产

### 7.1 资本市场线（CML）

```
引入 Rf 后：
  投资者在 CML 上选择点 = 风险资产组合 M + Rf 借贷
  CML: E(Rp) = Rf + [(E(Rm) - Rf) / σm] × σp
  斜率 = 夏普比率
```

### 7.2 分离定理（Two-Fund Theorem）

```
任意最优组合 = 无风险资产 + 市场组合
├─ 风险资产部分始终是 M（最优风险组合）
└─ 杠杆/借出比例由投资者风险偏好决定
```

### 7.3 风险厌恶与配置

| 风险厌恶 | Rf 配置 | 风险资产 M | 杠杆 |
|----------|---------|------------|------|
| 极低（进取） | 0% | 100% | 0 |
| 中等 | 30% | 70% | 0 |
| 平衡 | 50% | 50% | 0 |
| 保守 | 80% | 20% | 0 |
| 进取+杠杆 | -50% | 150% | 1.5 倍 |

## 八、夏普比率

### 8.1 定义

```
夏普比率 (Sharpe Ratio) = (E(Rp) - Rf) / σp

含义：每承担 1 单位总风险所获得的超额收益
```

### 8.2 数值示例

| 组合 | E(R) | Rf | σ | 夏普 |
|------|------|------|---|------|
| A | 12% | 2% | 15% | 0.67 |
| B | 10% | 2% | 10% | 0.80 |
| C | 8% | 2% | 5% | 1.20 |
| D | 6% | 2% | 3% | 1.33 |

> C 和 D 风险低但夏普高，是 MPT 视角下的优秀组合。

### 8.3 夏普比率的局限

| 局限 | 描述 |
|------|------|
| 总风险 vs 系统风险 | 用 σ 而非 β，未区分可分散风险 |
| 分布假设 | 假设正态分布，忽略厚尾 |
| 时间窗口 | 不同窗口结果可能差异大 |
| Rf 选择 | 短期 vs 长期 Rf 显著影响 |

## 九、MPT 的扩展

### 9.1 主要扩展

| 模型 | 改进点 | 适用 |
|------|--------|------|
| Black-Litterman | 引入投资者主观观点 | 机构资产配置 |
| 风险平价 (Risk Parity) | 风险贡献相等 | 桥水 All Weather |
| 最小方差组合 | 不需要收益预测 | 保守配置 |
| 最大分散化 (Maximum Diversification) | 最大化分散度 | 因子投资 |
| 等权重 (1/N) | 简单朴素 | 实证发现难击败 |

### 9.2 等权重的反例

DeMiguel, Garlappi, Uppal (2009) 发现：

- 60/40 组合相比等权重优势并不明显
- 样本外 MPT 组合常跑输等权重
- 主要原因：协方差与均值估计误差

## 十、MPT 的局限

### 10.1 核心批评

| 局限 | 描述 |
|------|------|
| 输入敏感 | 对 E(R)、σ、Cov 估计极敏感 |
| 静态单期 | 实际投资是动态多期 |
| 分布假设 | 假设正态，不符合实际厚尾 |
| 稳定性差 | 协方差矩阵不稳定导致权重剧烈变化 |
| 卖空限制 | 实务中可能无法做空 |
| 交易成本 | 频繁调仓成本侵蚀收益 |
| 行为偏差 | 投资者不严格理性 |

### 10.2 实际应用建议

| 实务 | 建议 |
|------|------|
| 战略配置 | 长期 MPT 配置（如 60/40） |
| 战术调整 | 结合宏观观点（Black-Litterman） |
| 风险监控 | 用 MPT 监控组合风险贡献 |
| 回测验证 | 历史数据回测验证分散化效果 |
