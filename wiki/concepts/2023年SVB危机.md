# 2023 年 SVB 危机（March 2023 Banking Crisis）

> 2023 年 3 月硅谷银行倒闭——美国史上第二大银行倒闭事件，揭示利率风险的新挑战

## 一、事件概要

**2023 年 SVB 危机**（也称"硅谷银行危机"或"美国区域银行危机"）发生于 **2023 年 3 月**，由**硅谷银行**（Silicon Valley Bank，SVB）倒闭引发，随后蔓延至**签名银行**（Signature Bank）、**第一共和银行**（First Republic）等多家美国区域银行，最终被**美联储紧急推出 BTFP**化解。

**关键时间节点**：
- **2023.3.8**：SVB 宣布出售 210 亿美元证券、亏损 18 亿美元
- **2023.3.9**：SVB 股价暴跌 60%
- **2023.3.10**：SVB 被 FDIC 接管（48 小时挤兑）
- **2023.3.12**：签名银行被 FDIC 接管
- **2023.3.16**：瑞银收购瑞信（瑞士）
- **2023.3.19**：美联储推出 BTFP 工具
- **2023.4-5**：第一共和银行被接管
- **2023.5.1**：第一共和银行被 FDIC 拍卖给摩根大通

**关键数据**：
- SVB：2090 亿美元资产（美国史上第二大倒闭）
- 签名银行：1100 亿美元资产
- 第一共和银行：2290 亿美元资产
- 3 月累计银行倒闭规模：5500 亿美元+
- 美国银行业短期借款（BTFP）：1600 亿美元+

## 二、SVB 的"独特性"

### 2.1 SVB 是什么？

```
硅谷银行（Silicon Valley Bank）：

成立：1983 年
总部：加利福尼亚州圣克拉拉
母行：SVB Financial Group（SIVB，NASDAQ 上市）
业务：专注于科技、生命科学、PE/VC 客户
资产规模：2090 亿美元（2022 年底）
存款规模：1730 亿美元（2022 年底）

特色：
  - 科技行业"专属银行"
  - 与硅谷生态系统深度绑定
  - 客户包括大量初创公司
  - 存款集中度高（VC 集中度高）
  - 资产端：长期国债 + MBS
       ↓
重要客户：
  - 红杉资本、Andreessen Horowitz 等顶级 VC
  - 数千家初创公司
  - 部分上市公司
```

### 2.2 SVB 的资产结构

```
SVB 资产端（2022 年底）：

现金：130 亿美元（占总资产 6%）
证券投资组合：1200 亿美元（占 58%）
  - HTM（持有至到期）：910 亿美元
  - AFS（可供出售）：260 亿美元
  - 平均久期：5.7 年
  - 平均收益率：1.6%（2020 年）
贷款：740 亿美元（占 35%）

负债端（2022 年底）：
  - 存款：1730 亿美元
    - 无保险存款：1520 亿美元（占 88%）
    - 保险存款：210 亿美元（仅占 12%）
       ↓
SVB 的独特风险：
  - 资产端久期长（5.7 年）
  - 负债端久期短（活期存款）
  - 资产收益率低（1.6%）
  - 这是经典的"久期错配"
```

## 三、SVB 倒闭的"三阶段"

### 3.1 第一阶段：基础裂痕（2020-2022）

```
2020-2022 年（疫情时代）：

1. 科技行业繁荣
   - 远程办公爆发
   - VC/PE 投资创纪录
   - 科技公司存款激增

2. SVB 存款爆炸性增长
   - 存款从 600 亿（2019）→ 1730 亿（2022）
   - 增长 2.9 倍

3. SVB 的投资策略
   - 存款太多 → 找不到足够贷款机会
   - 大量购买"长期证券"
   - 2020-2021 年锁定 1.5-2% 利率
   - 平均久期 5.7 年

4. 利率环境
   - 2020-2021 联邦基金利率 0-0.25%
   - 长债收益率 1.5-2.5%
   - SVB 大量配置"看似安全"的长期证券
```

### 3.2 第二阶段：美联储加息（2022-2023）

```
2022.3 美联储开始激进加息：

加息节奏：
  - 2022.3：+25bp
  - 2022.5：+50bp
  - 2022.6-11：连续 +75bp（4 次）
  - 2022.12：+50bp
  - 2023.2：+25bp
  - 2023.3.22：+25bp

累计加息：475bp（2022.3 至 2023.3）
联邦基金利率：0.25% → 4.75%
       ↓
对 SVB 的双重打击：
  1. 资产端：长期证券贬值
     - HTM 组合 910 亿 × 平均久期 5.7 年
     - 利率上升 475bp
     - 久期损失估算 150 亿+
     - AFS 已计入 25 亿损失
  2. 负债端：存款流失
     - 科技行业融资困难
     - 存款开始流失
     - 2022 年底开始"烧钱"
```

### 3.3 第三阶段：挤兑（48 小时）

```
2023.3.8 周三晚：SVB 公告
  - 出售 210 亿美元证券（AFS）
  - 亏损 18 亿美元
  - 计划增发 22.5 亿美元普通股
       ↓
2023.3.9 周四：股价崩盘
  - SVB 股价暴跌 60%（市值蒸发 95 亿）
  - 客户恐慌
  - VC 集中建议"撤资"
  - Founders Fund 等 VC 建议撤离
       ↓
2023.3.10 周五：挤兑
  - 单日存款提取请求 420 亿美元
  - 占总存款的 1/4
  - SVB 现金耗尽
  - 当日 17:00 美东被 FDIC 接管
  - 48 小时倒闭（史上最快）
       ↓
关键问题：
  - 为什么如此快？
  - 数字时代 + VC 集中建议 = 病毒式挤兑
  - 没有时间找买家
```

## 四、SVB 倒闭的"五大原因"

### 4.1 原因一：久期错配（核心）

```
SVB 资产负债表的"致命错配"：

资产端：
  - 910 亿美元 HTM 证券（5.7 年久期）
  - 锁定 1.6% 收益率（2020-2021 年）
  - 久期长 + 利率敏感

负债端：
  - 1730 亿美元活期存款
  - 久期 = 0（随时可提取）
  - 利率敏感但灵活

利率上升 475bp 的影响：
  - HTM 组合市值损失估算 150 亿+
  - 远超股东权益（160 亿）
  - 技术上接近资不抵债
```

### 4.2 原因二：存款集中度高

```
SVB 存款结构的"危险"：

存款集中度：
  - 前 5 大客户：~30% 存款
  - 前 50 大客户：~50% 存款
  - 大量 VC 关联账户

无保险存款占比：88%
  - 美国 FDIC 保险上限 25 万美元
  - 超出部分完全暴露
  - 1520 亿美元无保险存款
       ↓
风险：
  - 一旦恐慌，企业会立即撤离
  - 数字时代放大挤兑速度
  - 单日 420 亿撤资并非不可能
```

### 4.3 原因三：科技行业融资周期

```
科技行业与 SVB 的命运交织：

2020-2021 科技繁荣：
  - 零利率 + 远程办公 + VC 爆发
  - 科技公司存款大量流入 SVB
  - SVB 资产规模翻倍

2022-2023 科技寒冬：
  - 美联储激进加息
  - VC 投资大幅下降
  - 科技公司烧钱
  - 存款流出 SVB
       ↓
关键：
  - SVB 是科技行业的"专属银行"
  - 客户的兴衰 = SVB 的兴衰
  - 2022 年底存款已开始流失
```

### 4.4 原因四：管理失误

```
SVB 管理的"五大失误"：

1. 资产配置过度激进
   - 现金占比仅 6%
   - HTM 占比 58%
   - 在利率上升周期这是致命的

2. 套期保值不足
   - 没有利率对冲
   - 没用 IRS、CDS 对冲利率风险
   - 2008 后这类对冲银行都做了

3. 沟通失误
   - 2023.3.8 公告"亏损 18 亿"
   - 选择筹资而非自行消化
   - 暴露了真实财务状况
   - 引发恐慌

4. 缺乏应急计划
   - 没有"流动性管理"应急方案
   - 没有"巴塞尔 III 精神"
   - 没有压力测试

5. 治理失误
   - 董事会未发现风险
   - 风险管理部门失职
   - 监事会未阻止
```

### 4.5 原因五：监管失职

```
监管的"五大失职"：

1. 监管机构
   - SVB 受美联储监管（资产 > 700 亿）
   - 但同时受加州监管
   - 多头监管造成"谁都不负责"

2. 巴塞尔 III 未实施
   - SVB 是 SREP（监管审查程序）对象
   - 但未实施 LCR、NSFR（美国对中小银行豁免）
   - 流动性监管严重不足

3. HTM 会计准则
   - HTM 不计市值损失（仅在出售时确认）
   - 这隐藏了真实风险
   - 投资者看不到"危险信号"

4. 压力测试
   - SVB 在 1000-2500 亿区间
   - 每年压力测试一次
   - 但压力情景较温和

5. 监管文化
   - "硅谷成功故事"光环
   - 监管机构信任管理层
   - 缺乏质疑精神
```

## 五、危机的传染与扩散

### 5.1 签名银行（Signature Bank）

```
Signature Bank（3.12 被接管）：

特点：
  - 总部纽约
  - 资产 1100 亿美元
  - 加密行业"专属银行"
  - 客户包括 FTX 等
  - 存款中加密货币占比较高

倒闭过程：
  - 3.10 SVB 倒闭
  - 3.11-12 加密行业恐慌
  - 3.12 周日被纽约州监管机构接管
  - 美国史上第三大倒闭

原因：
  - 与 SVB 类似（久期错配）
  - 但加上了加密行业的风险
  - 数字时代放大挤兑
```

### 5.2 第一共和银行（First Republic）

```
First Republic Bank（5.1 被摩根大通收购）：

特点：
  - 总部旧金山
  - 资产 2290 亿美元
  - 高净值客户为主
  - 资产端低风险贷款

倒闭过程：
  - 3 月初股价暴跌
  - 11 家大银行存入 300 亿美元救助
  - 4 月存款仍持续流失
  - 5.1 被 FDIC 拍卖给摩根大通
  - 美国史上第二大倒闭

与 SVB 的差异：
  - 资产端相对稳健
  - 主要问题是存款流失
  - 不是久期错配，而是流动性危机
```

### 5.3 瑞士信贷（Credit Suisse）

```
Credit Suisse（3.19 被瑞银收购）：

背景：
  - 瑞士第二大银行
  - 165 年历史
  - 多年的丑闻和损失
  - 2022 亏损 80 亿瑞士法郎
  - 2023 存款大幅流失

事件：
  - 3.14 沙特国家银行拒绝注资
  - 瑞士央行（SNB）紧急声明
  - 3.19 瑞银以 30 亿瑞士法郎收购瑞信
  - 但瑞士政府承担损失
  - AT1 债券 170 亿全额减记（市场震惊）

意义：
  - 欧洲银行业危机
  - AT1 债券市场恐慌
  - 全球银行业震荡
```

### 5.4 危机传染机制

```
危机从 SVB 传染至全球的机制：

1. 直接传染
   - 与 SVB 有业务关联的银行
   - 美国区域银行（类似 SVB）
   - 加密银行

2. 间接传染
   - 投资者对类似银行的担忧
   - 银行股普遍下跌
   - KBW 银行指数 3 月下跌 20%+

3. 信心传染
   - 存款人对银行体系的信心下降
   - 银行业整体存款流出
   - 货币市场基金转入（避险）

4. 政策传染
   - 美联储 BTFP 推出
   - 欧央行跟进
   - 各国央行协同应对

5. 跨市场传染
   - 银行股 → 货币市场 → 国债 → 信贷市场
   - 短暂但剧烈的金融市场波动
```

## 六、美联储的紧急应对

### 6.1 BTFP（Bank Term Funding Program）

```
2023.3.19 推出 BTFP：

设计：
  - 期限：最长 1 年
  - 利率：1 年期 OIS + 10bp
  - 抵押品：国债、机构 MBS
  - 关键创新：按面值计价

按面值计价的意义：
  - 银行无需按市值计提损失
  - 用 2023.3 之前购买的国债
  - 银行可借面值 100% 的贷款
  - 即使国债市场价跌至 70%
       ↓
为什么重要：
  - 直接解决 SVB 类型的"未实现损失"
  - 银行可以借到现金，无需出售亏损资产
  - 给银行"时间"消化损失
       ↓
实际效果：
  - 3 月末 BTFP 余额：540 亿美元
  - 4 月：740 亿
  - 5 月：930 亿
  - 6 月：1030 亿（峰值）
  - 之后逐步下降
  - 2024.3 退出
```

### 6.2 美联储的"最后贷款人"工具

```
美联储应对 SVB 危机的工具：

1. BTFP（2023.3.19）：解决久期错配
2. 贴现窗口：常规流动性支持
3. FIMA Repo（2020）：外国央行美元流动性
4. 协调大银行救助：摩根大通、巴菲特等

5. 重要创新：与财政部协调
   - 财政部 250 亿美元 ESF 担保
   - 美联储实际操作
   - 但纳税人风险有限
```

### 6.3 货币政策的影响

```
危机对美联储政策的影响：

加息节奏：
  - 2023.2：+25bp（预期中）
  - 2023.3.22：+25bp（危机中仍加息）
  - 2023.5：+25bp（最后一次加息）
  - 2023.6：暂停
  - 2023.7：+25bp（最终一次）
  - 2023.9-2024.1：维持 5.5%
  - 2024.9：开始降息
       ↓
反思：
  - 美联储在危机中仍坚持加息
  - 但步幅缩小
  - 5 月开始"暂停"
  - 9 月确认"加息周期结束"
  - 这是"边加息边灭火"的典型
```

## 七、SVB 危机的"五大教训"

### 7.1 教训一：利率风险被低估

```
巴塞尔 III 主要关注：
  - 信用风险
  - 市场风险
  - 操作风险
  - 流动性风险

但低估了：
  - 利率风险（IRR）
  - 久期错配风险
  - 银行账簿利率风险（IRRBB）
       ↓
SVB 教训：
  - 利率风险同样可以致命
  - HTM 会计准则隐藏风险
  - 必须有更严格的利率风险管理
```

### 7.2 教训二：存款集中度是新风险

```
传统银行模型：
  - 大量零售存款
  - 客户分散
  - 单个客户影响小

SVB 类银行：
  - 大量企业存款
  - 客户集中
  - 单个客户影响大
  - 数字时代放大挤兑
       ↓
教训：
  - 必须监测存款集中度
  - 必须有"应急流动性"
  - 数字时代的银行挤兑更快
```

### 7.3 教训三：监管需要与时俱进

```
巴塞尔 III 的局限：

1. 中小银行豁免
   - LCR、NSFR 对 1000 亿以下不要求
   - SVB 正好是 1000-2500 亿之间
   - "豁免"反而成为漏洞

2. HTM 会计准则
   - 隐藏未实现损失
   - 投资者看不到
   - 应该按市值披露

3. 监管文化
   - "成功故事"反而危险
   - 必须质疑式监管

4. 多头监管
   - 美联储 + 加州
   - 责任不清
```

### 7.4 教训四：数字时代放大危机

```
传统挤兑：
  - 储户到银行排队
  - 速度有限（每天数千人）
  - 银行有时间反应

数字时代挤兑：
  - 一封邮件/短信即可触发
  - 速度极快（每小时数十亿）
  - 银行来不及反应
  - 病毒式传播

SVB 案例：
  - 3.10 一天内 420 亿撤资
  - VC 集中建议
  - 银行瞬间死亡
```

### 7.5 教训五：BTFP 是新的"最后贷款人"

```
BTFP 的历史意义：

1. 解决未实现损失
   - 银行可以借面值
   - 避免"被迫出售"亏损资产

2. 央行直接介入银行账簿
   - 不只是市场账簿
   - 解决银行特有的利率风险

3. "传统工具的扩展"
   - 不是 QE
   - 不是财政货币化
   - 是最后贷款人的延伸

4. 对巴塞尔 III 的补充
   - 巴塞尔 III 主要应对信用风险
   - BTFP 应对利率风险
   - 央行工具库扩展

未来：
  - BTFP 可能成为"永久工具"
  - 类似 SRF（永久性回购工具）
  - 央行工具库持续扩展
```

## 九、SVB 危机与中国银行业

### 9.1 中国银行业相对稳健的原因

```
中国银行业未受 SVB 危机传染的主要原因：

1. 利率环境不同
   - 2022 中国 LPR 下降（不是上升）
   - 2023 继续宽松
   - 资产负债没有"反向"风险

2. 资产结构不同
   - 中国银行持有更多贷款（不是证券）
   - 久期较短
   - 利率敏感性不同

3. 监管更严格
   - 中国实施巴塞尔 III
   - LCR、NSFR 都要求
   - 流动性风险监管严格

4. 存款基础不同
   - 零售存款为主（储蓄文化）
   - 集中度较低
   - 数字挤兑可能性较小

5. 国有控股背景
   - 国有大行不会破产
   - 政府隐性担保
   - 储户信心较高
```

### 9.2 中国银行业的潜在风险

```
但中国银行业并非完全无风险：

1. 房地产相关贷款
   - 银行大量持有房地产贷款
   - 房地产下行风险大
   - 不良率上升

2. 地方政府债务
   - 银行通过影子银行大量持有
   - 化债的银行业影响
   - 隐性风险

3. 中小银行风险
   - 部分中小银行资产质量差
   - 包商银行、锦州银行案例
   - 区域银行风险

4. 净息差压力
   - LPR 持续下调
   - 银行净息差缩小
   - 盈利能力下降

5. 存款搬家
   - 资金从银行流向理财
   - 从中小行流向大行
   - 负债结构变化
```

## 十、SVB 危机的核心命题

### 10.1 五大成就（监管角度）

```
1. **揭示了利率风险的重要性**
   - 巴塞尔 III 主要关注信用风险
   - SVB 揭示利率风险同样致命
   - 推动巴塞尔 III 最终方案

2. **强化了大型银行监管**
   - 美联储审查 1000-2500 亿资产银行
   - 巴塞尔 III 实施加速

3. **创新了央行工具**
   - BTFP 解决久期错配
   - 央行工具库扩展
   - 永久性工具可能

4. **加强了会计准则**
   - AFS、HTM 披露要求
   - "未实现损失"必须公开
   - 投资者保护加强

5. **提升了数字时代监管意识**
   - 数字挤兑的认识
   - 监管机构的快速反应
   - 跨机构协调
```

### 10.2 五大局限

```
1. **教训未充分吸取**
   - 巴塞尔 III 实施有 5 年过渡
   - 部分银行可能仍未准备好
   - 下次危机可能再次发生

2. **BTFP 的退出**
   - 2024.3 BTFP 已退出
   - 类似风险可能再次出现
   - 没有永久工具

3. **监管套利仍存在**
   - 银行将业务移到非监管领域
   - 影子银行规模未减
   - 加密银行问题未解

4. **跨境协调不足**
   - 美国 BTFP 仅美国适用
   - 各国央行反应不同
   - 瑞士瑞信处理特殊

5. **政治化**
   - 监管改革受政治影响
   - 未来放松的可能性
   - 政策连续性问题
```

### 10.3 SVB 危机的历史定位

```
SVB 危机是后金融危机时代的重要事件：

地位：
  - 2020 年代最大银行倒闭
  - 揭示了巴塞尔 III 的局限
  - 推动了巴塞尔 III 最终方案
  - 创新了央行最后贷款人工具

影响：
  - 银行业监管的重新审视
  - 利率风险管理的重要性
  - 数字时代的新挑战
  - 央行工具库的扩展
       ↓
理解 SVB 危机 = 理解后危机时代银行监管的演进
```

## 十一、一句话总结

> **2023 年 SVB 危机 = 硅谷银行 48 小时倒闭引发的美国区域银行危机——揭示了利率风险的新挑战、巴塞尔 III 的局限、数字时代挤兑的速度**。
>
> **核心原因**：**久期错配**（5.7 年资产 vs 0 年负债）+ **存款集中度**（88% 无保险）+ **科技行业周期**（VC 寒冬）+ **管理失误**（未对冲利率风险）+ **监管失职**（HTM 隐藏损失）。
>
> **传染机制**：**直接传染**（类似银行）+ **信心传染**（市场恐慌）+ **跨市场传染**（银行股 → 货币市场 → 国债）+ **跨境传染**（瑞士瑞信）。
>
> **美联储应对**：**BTFP**（按面值计价、解决未实现损失）+ **贴现窗口** + **协调大银行救助** + **政策创新**。
>
> **五大教训**：**利率风险被低估**、**存款集中度是新风险**、**监管需要与时俱进**、**数字时代放大危机**、**BTFP 是新的最后贷款人**。
>
> **核心命题**：SVB 危机揭示了**巴塞尔 III 的局限**——主要关注**信用风险**，但**利率风险**同样可以致命；**数字时代**银行挤兑速度极快（48 小时）；**监管需要持续演进**以应对新挑战。**未来 10 年，加密资产、气候风险、AI 等新议题将持续推动监管的演进**。
