# 扩表与缩表

> 美联储通过买卖国债等资产调整其资产负债表规模的两种相反方向操作

**扩表**(Balance Sheet Expansion,主要工具是 QE)与**缩表**(Balance Sheet Contraction,主要工具是 QT)是美联储**资产负债表操作**的两个相反方向,本质都是通过资产端的买卖影响负债端的**银行准备金规模**,从而调控市场流动性。

## 一句话区分

| 操作 | 方向 | 主要工具 | 央行资产负债表 |
|---|---|---|---|
| **扩表 / QE** | 买入资产 | 大规模资产购买 | **变大** |
| **缩表 / QT** | 卖出/到期不续资产 | 到期不再投资 / 主动卖出 | **变小** |

## 美联储资产负债表结构

### 资产端

- **国债**(SOMA 账户持有):约 $4.2 万亿(2026)
- **MBS**(机构抵押贷款支持证券):约 $2.2 万亿
- 其他(回购、贷款等):小额
- **总计**:约 $6.7 万亿(2026)

### 负债端

- **流通中现金**:约 $2.3 万亿
- **银行准备金**:约 $3 万亿
- **TGA**(财政部账户):约 $7000 亿
- **逆回购 RRP**:约 $3000 亿
- **总计**:约 $6.7 万亿

## 扩表(QE)的运作机制

### 操作流程

```
美联储决定 QE
    ↓
纽约联储执行(通过 SOMA 账户)
    ↓
从一级交易商(primary dealers,如 Goldman, JPM)买入已存在的国债
    ↓
美联储 → 银行:创造银行准备金(reserve credits)
    ↓
银行 reserve account 余额增加
    ↓
银行系统流动性暴增 → 长期利率下行
```

### QE 的传导渠道

1. **信号渠道**:央行承诺买资产 → 投资者预期低利率
2. **期限溢价渠道**:央行买长债 → 压低长债收益率
3. **流动性渠道**:准备金增加 → 银行放贷能力增强
4. **财富效应渠道**:资产价格上涨 → 居民消费增加
5. **汇率渠道**:利率下行 → 美元走弱 → 出口竞争力提升

### QE 的历史里程碑

| 时期 | 名称 | 规模 | 触发事件 |
|---|---|---|---|
| 2008-11 → 2010-03 | QE1 | $1.7 万亿 | 金融危机 |
| 2010-11 → 2011-06 | QE2 | $6000 亿 | 欧债危机 |
| 2012-09 → 2014-10 | QE3 | $1.6 万亿 | 财政悬崖担忧 |
| 2019-10 → 2020-03 | 早期 QE | $5000 亿 | 流动性紧张 |
| **2020-03 → 2022-03** | **QE4(疫情 QE)** | **$4.6 万亿** | **COVID-19** |

**累计**:美联储资产负债表从 2008 年的 $8000 亿扩张到 2022 年峰值的 $9 万亿,**翻了 10 倍**。

## 缩表(QT)的运作机制

### 操作流程

QT 有两种方式:

**(一)被动缩表(到期不再投资)**

```
美联储持有 10 年期国债到期(2025)
    ↓
按惯例应买新债替换
    ↓
QT 模式下:不再投资
    ↓
现金回流财政部(TGA 增加)
    ↓
美联储资产负债表自动缩减
```

**(二)主动缩表(直接卖出)**

```
美联储决定主动卖出部分国债
    ↓
通过一级交易商卖出国债
    ↓
交易商付现金给美联储(准备金 -)
    ↓
美联储资产负债表缩减
```

**注**:美联储 QT 主要用**被动缩表**(到期不再投资),主动卖出较少使用。

### QT 的传导渠道(与 QE 相反)

1. **流动性回收**:准备金减少 → 银行放贷能力下降
2. **长债利率上行**:央行需求减少 → 期限溢价上升
3. **资产价格下跌**:流动性收紧 → 风险资产承压
4. **美元走强**:利率相对走高
5. **金融条件收紧**:整体融资环境趋紧

### QT 的历史里程碑

| 时期 | 名称 | 缩表规模 | 触发事件 |
|---|---|---|---|
| 2017-10 → 2019-07 | QT1 | $7000 亿 | 货币政策正常化 |
| 2019-08 → 2019-09 | QT 暂停 | 暂停 | 流动性紧张(9 月 repo crisis) |
| **2022-06 → 2024-09** | **QT2(疫情 QT)** | **约 $2 万亿** | **抗通胀** |

**累计**:美联储资产负债表从 2022 年峰值 $9 万亿缩减至 2024-09 的约 $7 万亿。

## QE 与 QT 的关键对比

| 维度 | QE(扩表) | QT(缩表) |
|---|---|---|
| 方向 | 买入资产 | 卖出 / 不续资产 |
| 资产负债表 | 变大 | 变小 |
| 流动性 | 注入 | 回收 |
| 利率 | 下行 | 上行 |
| 资产价格 | 上涨 | 承压 |
| 美元 | 走弱 | 走强 |
| 主要目的 | 应对危机 / 刺激经济 | 抗通胀 / 政策正常化 |
| 实施难度 | 低(直接买) | **高**(市场可能动荡) |
| 退出难度 | 难(缩减可能引发流动性危机) | 难(可能加剧市场波动) |

## 关键的技术细节

### 1. 美联储 SOMA 账户

- **System Open Market Account**(系统公开市场账户)
- 由纽约联储管理
- 持有美联储的大部分国债 + MBS
- 2026 年规模约 $6.7 万亿
- 是 QE / QT 操作的具体载体

### 2. 一级交易商(primary dealers)

- 与美联储直接交易的金融机构
- 主要包括:Goldman Sachs, JP Morgan, Citigroup, BNY Mellon 等 24 家
- 美联储买/卖国债必须通过它们
- 它们是 QE / QT 操作的"中间人"

### 3. 缩表上限(cap)

美联储 2017 年首次 QT 时设定了月度上限:
- 国债:最初 $60 亿/月,逐步升至 $300 亿/月
- MBS:最初 $40 亿/月,逐步升至 $200 亿/月
- 总上限:$500 亿/月

2022 年第二次 QT 时类似。

### 4. 缩表的"隐形下限"

- 美联储不希望准备金降到"短缺"水平
- "充裕准备金"(ample reserves)框架下,银行体系需要足够缓冲
- 实际缩表通常在准备金接近"舒适下限"时停止
- 2019 年 9 月回购市场危机就是缩表过度的教训

## QE / QT 与加降息的关系

### 传统认知

```
QE → 利率下行(扩张性货币政策)
QT → 利率上行(紧缩性货币政策)
加降息 → 通过联邦基金利率直接调控
```

### 现实中的复杂组合

QE / QT 与加降息**可以独立操作**,常见组合:

| 组合 | 含义 | 历史案例 |
|---|---|---|
| 加息 + 缩表 | 双重紧缩 | 2017-2019、2022-2024 |
| 降息 + 扩表 | 双重宽松 | 2020 疫情 |
| 加息 + 扩表 | 罕见(利率工具与资产负债表工具反向) | 几乎无 |
| 降息 + 缩表 | 罕见(沃什主张的政策组合) | 2019 末 / 2026 讨论中 |

**关键洞察(沃什的政策组合)**:

> "降息 + 缩表"意味着**双重紧缩**:
> - 表面降息(鸽派信号)
> - 实质缩表(回收流动性)
> - 仅看利率方向会严重误读美联储真实立场

参见 [[wiki/entities/量化紧缩|量化紧缩]] 中关于沃什政策组合的论述。

## QE / QT 对市场的影响

### 资产价格

| 资产类别 | QE 期间 | QT 期间 |
|---|---|---|
| 股票 | 上涨(流动性 + 财富效应) | 承压(流动性回收) |
| 债券 | 价格上涨(收益率下行) | 价格下跌(收益率上行) |
| 房地产 | 上涨(抵押贷款利率低) | 承压 |
| 大宗商品 | 上涨(美元走弱) | 承压 |
| 黄金 | 上涨(实际利率下行 + 美元走弱) | 承压 |
| 加密货币 | 上涨(流动性充裕) | 承压 |

### 对实体经济

| 维度 | QE 效果 | QT 效果 |
|---|---|---|
| 信贷 | 扩张 | 收紧 |
| 消费 | 提振 | 抑制 |
| 投资 | 提振 | 抑制 |
| 通胀 | 推升 | 抑制 |
| 就业 | 改善 | 承压 |
| 出口 | 提振(弱美元) | 抑制(强美元) |

## 历史关键节点(2008-2026)

| 时点 | 事件 | 资产负债表 |
|---|---|---|
| 2008-09 | 金融危机,启动 QE1 | $8000 亿 |
| 2014-10 | QE3 结束 | $4.5 万亿 |
| 2017-10 | 启动 QT1 | $4.5 万亿 |
| 2019-09 | **QT 暂停**(回购危机) | $3.8 万亿 |
| 2020-03 | 启动 QE4(疫情) | $4.2 万亿 |
| 2022-03 | 停止 QE4,准备加息 | $9 万亿(峰值) |
| 2022-06 | 启动 QT2 | $9 万亿 |
| 2024-09 | QT2 接近尾声 | $7 万亿 |
| 2026 | 维持高位,讨论下一步 | **$6.7 万亿** |

## 缩表为什么难?——"退出困境"

### 问题 1:准备金短缺

- 银行体系需要"充裕准备金"
- 缩表过猛 → 准备金不足 → 利率波动(如 2019 9月)
- 美联储必须有"隐形下限"

### 问题 2:市场流动性

- 美联储是国债最大持有者
- 缩表 → 国债市场流动性下降
- 可能引发国债抛售压力

### 问题 3:金融稳定

- 长债利率快速上行 → 资产价格下跌
- 可能引发连锁反应(养老金、保险、银行)
- 2023 年 3 月 SVB 银行危机就是 QT 过度的潜在风险

### 问题 4:政治压力

- 缩表 = 流动性回收 = 资产价格下跌
- 总统、国会会施压要求停止
- 历史上多次缩表被政治干预打断

## 中国 / 日本 / 欧央行的对应操作

### 中国央行(中国人民银行)

- 主要用**存款准备金率**调节(不是 QE)
- [[wiki/concepts/MLF|MLF]] / [[wiki/concepts/PSL|PSL]] 等结构性工具
- 不大规模买卖国债(目前)
- 与美联储 QE / QT 不同

### 日本央行(BOJ)

- 长期 QE(2013-2024)
- 2024 年开始**缓慢 QT**(缩减国债购买)
- 但仍在购入长债,**YCC** 已结束
- 与美联储 QT 节奏差异显著

### 欧洲央行(ECB)

- 2015-2022:APP 资产购买计划
- 2023-2024:停止再投资 + 主动卖出
- 与美联储 QE / QT 节奏相近

## 当前(2026)美联储资产负债表状态

- **规模**:约 $6.7 万亿(峰值 $9 万亿的 74%)
- **政策方向**:QT 已基本完成,讨论下一步
- **市场关注**:
  - 是否再次扩表应对经济放缓
  - 是否启动"无限 QE"配合财政部
  - 缩表 / 扩表的时机选择
