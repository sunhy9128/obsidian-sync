# 债市定价逻辑

> 债券如何定价，以及影响债券定价的核心因素

债券是"固定收益"——表面看收益是固定的，但债券价格会随着利率环境变化而大幅波动。**债市定价**研究的就是债券价格与利率、信用、流动性、通胀预期等各因素之间的关系。

---

## 一、债券定价基础

### 1.1 基本公式

债券价格 = 未来现金流折现到现值：

```
P = Σ [C / (1+y)^t] + F / (1+y)^n

其中：
P = 债券价格
C = 每期票息（票面利率 × 面值）
F = 面值（到期归还）
y = 到期收益率（YTM）
n = 剩余期限（年）
t = 每期（1,2,3,...,n）
```

**简化理解**：
- 折现率（y）上升 → 未来现金流的现值下降 → 债券价格下跌
- 折现率（y）下降 → 未来现金流的现值上升 → 债券价格上涨

> 这就是债券定价的**第一性原理**：债券价格与收益率成反向关系。

### 1.2 定价的三个核心变量

| 变量 | 含义 | 对价格的影响 |
|------|------|-------------|
| **收益率（YTM）** | 折现率，是最重要的定价因子 | 反向——收益率↑ 价格↓ |
| **票面利率** | 债券的名义利率 | 正向（票息越高，定价越高） |
| **剩余期限** | 距离到期的时间 | 期限越长，久期越大，对利率越敏感 |

> **票面利率 vs 收益率**：发行时两者相等；持有过程中若市场利率变化，债券价格就会偏离面值，票面利率不变但收益率随之调整。

## 二、收益率曲线与定价

### 2.1 收益率曲线的定价含义

债券不只有一个"收益率"——不同期限对应不同收益率。**收益率曲线**就是期限与收益率的关系图，是债券市场最重要的定价基准。

```
收益率曲线示意：
  ↑
  │                        ╭─────────────── 长端（10年+）
  │                  ╭─────╯
  │             ╭────╯
  │        ╭───╯
  │    ╭───╯
  │ ╭──╯
  └─────────────────────────────────────→
    3M  1Y  3Y  5Y  7Y  10Y  20Y  30Y
```

### 2.2 曲线形态与债市定价

| 曲线形态 | 市场含义 | 债券配置策略 |
|----------|----------|-------------|
| **陡峭化**（长端 >> 短端） | 经济增长预期强 / 通胀预期上升 | 拉长久期，锁定长期高收益 |
| **平坦化**（长端 ≈ 短端） | 增长预期减弱 / 央行维持低利率 | 缩短久期，减少利率风险 |
| **倒挂**（长端 < 短端） | 衰退预期强烈（经典衰退信号） | 短久期防御，等待宽松 |
| **陡峭化加剧** | 市场对经济过热担忧加深 | 减长久期 |
| **平坦化加剧** | 央行宽松预期强化 | 加长久期 |

### 2.3 债券久期（Duration）的定价含义

久期是债券价格对利率变动的**敏感度**：

```
价格变动 ≈ - 久期 × 收益率变动

例：久期 7 年的债券，收益率上升 1%：
价格下跌 ≈ 7 × 1% = 7%
```

| 债券类型 | 典型久期 | 对利率的敏感度 |
|----------|----------|---------------|
| **超短债** | 0-1 年 | 极低，接近货币基金 |
| **短债** | 1-3 年 | 低 |
| **中债** | 3-7 年 | 中 |
| **长债** | 7-10 年 | 高 |
| **超长债** | 20-30 年 | 极高（30 年国债对利率极度敏感） |

## 三、影响债市定价的五大因子

### 3.1 央行货币政策（最重要的因子）

| 政策行动 | 对收益率的影响 | 逻辑 |
|----------|--------------|------|
| **降息 / 降准** | 收益率下行 | 资金面宽松，无风险利率降低 |
| **加息 / 升准** | 收益率上行 | 资金面收紧，无风险利率上升 |
| **量化宽松（QE）** | 短端压低 + 长端传导 | 直接买入债券，推高价格、压低收益率 |
| **量化紧缩（QT）** | 收益率上行 | 抛售债券，流动性收紧 |
| **前瞻指引** | 市场提前定价 | 央行预期管理影响市场对未来利率的判断 |

> **债市最核心的交易逻辑**：跟着央行的货币政策的**方向**和**速度**做债。

### 3.2 通胀预期（长端定价的核心）

| 类型 | 对长端的影响 | 说明 |
|------|-------------|------|
| **实际通胀上升** | 长端收益率上行 | 债券持有期购买力受损，要求更高收益率补偿 |
| **通胀预期上升** | 长端收益率上行 | 反映在国债收益率 = 实际利率 + 通胀预期 |
| **通胀预期稳定** | 长端收益率稳定 | 长端锚定通胀目标 |
| **通缩预期** | 长端收益率急剧下行 | 日本、欧洲的教训 |

**费雪公式的债市含义**：

```
国债收益率 ≈ 实际利率 + 通胀预期

例：实际利率 2%，通胀预期 2.5%
→ 10 年国债收益率 ≈ 4.5%

若通胀预期升至 3.5%，其他不变
→ 10 年国债收益率 ≈ 5.5%（上行 1%）
→ 10 年国债价格下跌约 7%（久期 7 × 1%）
```

### 3.3 经济增长预期

| 经济数据 | 对债市的影响 |
|----------|-------------|
| PMI > 50 且趋势向上 | 增长预期强 → 长端收益率上行 |
| GDP 增速超预期 | 债券被抛售 → 收益率上行 |
| 工业增加值回暖 | 实体融资需求增加 → 债券吸引力下降 |
| 零售数据强劲 | 消费驱动增长 → 债券承压 |

> **核心逻辑**：经济好 → 企业借贷扩张 → 债券供给增加 + 资金流向风险资产 → 债券价格跌、收益率升

### 3.4 信用利差

对于**信用债**（非国债），定价还需要加上**信用利差**：

```
信用债收益率 = 国债收益率 + 信用利差

信用利差 = 投资者要求的信用风险补偿
```

| 信用等级 | 信用利差（参考） | 含义 |
|----------|----------------|------|
| AAA（最高信用） | +0-50 bps | 接近国债 |
| AA | +50-150 bps | 低信用风险 |
| A | +150-300 bps | 中等信用 |
| BBB | +300-500 bps | 投资级下限 |
| BB 及以下 | +500+ bps | 高收益（垃圾债） |

**信用利差扩大** = 市场对信用风险的担忧增加 = 信用债被抛售 = 信用债价格下跌

### 3.5 流动性溢价

| 情况 | 对债券定价的影响 |
|------|-----------------|
| 市场流动性充裕 | 投资者愿意接受更低流动性溢价 → 债券价格偏高 |
| 市场流动性紧张 | 流动性溢价上升 → 债券价格偏低（折价） |
| 机构大规模抛售 | 流动性快速恶化 → 踩踏式下跌（2022 年英债危机） |
| 央行大规模买入（QE） | 流动性充裕 → 债券价格偏高 |

## 四、A 股债市（中国债券市场）定价逻辑

### 4.1 中国债券市场结构

| 品种 | 发行人 | 特点 | 久期 |
|------|--------|------|------|
| **国债** | 中央政府 | 零信用风险，流动性最好 | 1-50 年 |
| **地方债** | 地方政府 | 隐性担保，流动性次之 | 3-30 年 |
| **政金债** | 政策性银行 | 国家信用背书 | 1-20 年 |
| **铁道债** | 铁路总公司 | "准主权"信用 | 5-20 年 |
| **信用债（企业债）** | 企业 | 存在信用风险 | 1-10 年 |
| **城投债** | 地方政府融资平台 | 隐性政府担保（争议大） | 1-10 年 |
| **可转债** | 上市公司 | 含转股期权，定价复杂 | 5-6 年（平均） |

### 4.2 中国特色定价因素

| 因素 | 影响 | 说明 |
|------|------|------|
| **银行配置盘主导** | 压低收益率 | 银行以负债成本为锚，追求稳定收益 |
| **央行政策利率（LPR）** | 影响短端定价 | LPR 降 → 短端收益率降 |
| **资金面（DR007）** | 隔夜利率影响 | 资金面紧 → 短端收益率上行 |
| **城投信仰** | 城投债定价偏低 | 市场默认政府兜底，信用利差被人为压低 |
| **外资持有中国债券** | 压低中美利差 | 2020-2023 年外资大量流入中国债券市场 |

### 4.3 中美债市定价逻辑差异

| 维度 | 美债 | 中债 |
|------|------|------|
| **定价锚** | 美联储政策利率（联邦基金利率） | MLF / LPR / 资金面 DR007 |
| **长端定价核心** | 通胀预期 + 经济增长预期 | 政策导向 + 城投风险 |
| **曲线的政策含义** | 倒挂 = 衰退信号 | 陡峭化 = 宽信用预期 |
| **外资影响力** | 较小（美元霸权） | 较大（外资增配中债） |

## 五、债券定价的利率风险

### 5.1 久期与凸性的综合影响

| 因子 | 公式 | 作用 |
|------|------|------|
| **久期** | ΔP/P ≈ -D × Δy | 近似线性修正 |
| **凸性** | 加入凸性后：ΔP/P ≈ -D×Δy + ½×C×(Δy)² | 上凸（Convex）为正 → 利率上升时少跌，下跌时多涨 |

**凸性的实战意义**：
- 凸性为正的债券：利率下降时涨幅 > 利率上升时跌幅 → 投资者偏好高凸性
- 含权债券有负凸性（callable bonds）：利率下降到某一点后可能被提前赎回
- 可转债有一定正凸性

### 5.2 久期管理策略

| 策略 | 操作 | 适用环境 |
|------|------|----------|
| **子弹策略** | 集中于某一期限（如 10 年） | 预期利率稳定 |
| **阶梯策略** | 均匀配置各期限 | 稳健型，不判断方向 |
| **哑铃策略** | 配置短端 + 长端，中端少配 | 利率方向不明确 |
| **免疫策略** | 久期匹配负债，消除利率风险 | 负债明确的机构（养老金等） |

## 六、债市定价的交易框架

### 6.1 宏观定价框架

```
债市定价 = 政策利率预期 + 通胀预期 + 增长预期 + 信用利差 + 流动性溢价

步骤：
1. 判断货币政策方向（降息？加息？维持？）
2. 估计通胀预期（实际通胀 + 通胀预期调查）
3. 评估经济增长（PMI/GDP → 债市多空）
4. 评估信用环境（信用利差扩大还是收窄）
5. 综合得出各期限的合理收益率水平
```

### 6.2 曲线交易策略

| 策略 | 操作 | 盈利逻辑 |
|------|------|----------|
| **陡峭化交易（Barbell → Bullet）** | 买长端 + 卖短端 | 预期经济复苏，长端上行 |
| **平坦化交易（Bullet → Barbell）** | 卖长端 + 买短端 | 预期衰退，长端下行 |
| **骑乘收益率策略** | 买入中短期债券持有一段时间 | 债券到期时间缩短，久期缩短，价格上涨 |
