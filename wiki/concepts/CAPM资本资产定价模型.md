> CAPM（Capital Asset Pricing Model）由 William Sharpe（1964）、John Lintner（1965）、Jan Mossin（1966）独立提出，描述在均衡市场中，单一证券的预期收益与其系统性风险（β）之间的线性关系。

---

## 一、模型公式

### 1.1 核心公式

```
E(Ri) = Rf + βi × [E(Rm) - Rf]

符号：
  E(Ri)   资产 i 的预期收益率
  Rf      无风险利率
  E(Rm)   市场组合的预期收益率
  E(Rm) - Rf  股权风险溢价 (ERP)
  βi     资产 i 对市场风险的敏感度
```

### 1.2 数值示例

| 资产 | β | Rf=2% | E(Rm)-Rf=6% | E(Ri) |
|------|---|-------|-------------|-------|
| 短期国债 | 0 | 2% | 6% | 2.00% |
| 防御型股票 | 0.6 | 2% | 6% | 5.60% |
| 市场组合 | 1.0 | 2% | 6% | 8.00% |
| 高弹性科技股 | 1.5 | 2% | 6% | 11.00% |
| 杠杆 ETF | 2.0 | 2% | 6% | 14.00% |

## 二、β 系数的经济含义

### 2.1 β 的定义

```
βi = Cov(Ri, Rm) / Var(Rm)

或等价地：
βi = ρ(i,m) × σi / σm
```

其中 ρ(i,m) 是资产 i 与市场的相关系数。

### 2.2 β 的直观解释

| β 值 | 含义 | 市场下跌 10% 时 |
|------|------|----------------|
| β = 0 | 与市场无关 | 资产不变 |
| 0 < β < 1 | 防御型 | 跌幅 < 10% |
| β = 1 | 与市场同步 | 跌 10% |
| β > 1 | 进攻型 | 跌幅 > 10% |
| β < 0 | 反向（罕见） | 上涨 |

### 2.3 β 的决定因素

| 因素 | 对 β 的影响 |
|------|-------------|
| 行业属性 | 周期股高 β，公用事业低 β |
| 经营杠杆 | 固定成本高 → 高 β |
| 财务杠杆 | 负债多 → 高 β |
| 收入波动性 | 现金流波动大 → 高 β |
| 与宏观经济相关性 | 与 GDP 相关性高 → 高 β |

## 三、CAPM 假设

### 3.1 核心假设清单

| 假设 | 内容 | 现实偏离 |
|------|------|----------|
| 理性投资者 | 期望效用最大化、风险厌恶 | 投资者有认知偏差 |
| 单期投资 | 持有期一致 | 多期、动态调整 |
| 交易市场 | 无交易成本、无税收 | 实际有显著成本 |
| 可卖空 | 可无成本卖空任意资产 | 卖空受限（尤其 A 股） |
| 资产可分 | 资产可无限细分 | 现实中按手交易 |
| 价格接受者 | 单个投资者不影响价格 | 大资金有冲击成本 |
| 同质预期 | 所有人对均值、方差、协方差看法一致 | 信息不对称 |
| 存在无风险资产 | 投资者可以 Rf 借入借出 | 实际有借/贷利差 |

### 3.2 假设的层级关系

```
                    ┌──────────────┐
                    │ CAPM 成立     │
                    └──────┬───────┘
                           │
        ┌──────────────────┼──────────────────┐
        │                  │                  │
        ▼                  ▼                  ▼
   理性投资者        无摩擦市场         同质预期
   (效用理论)        (无税/无成本)       (一致信念)
        │                  │                  │
        └──────────────────┼──────────────────┘
                           │
                           ▼
                    市场组合 M 是
                    最优风险组合
```

## 四、证券市场线（SML）

### 4.1 SML 公式

```
E(R) = Rf + β × ERP
斜率 = ERP (股权风险溢价)
截距 = Rf
```

### 4.2 SML 图像

```
E(R)
  │
14%├──                                          ● β=2
  │                                       ● β=1.5
11%│                                    
  │                            ● β=1
 8%├──                  ● (市场组合 M)
  │              ● β=0.5
 5%├──     ● β=0.25
  │  ● β=0
 2%├●─────┬─────┬─────┬─────┬─────┬──→ β
  │  0   0.5   1.0   1.5   2.0
  └───┬───┘
     Rf=2%
```

### 4.3 资产定价位置判断

| 资产位置 | 含义 | 投资动作 |
|----------|------|----------|
| 在 SML 上方 | 实际预期收益 > CAPM 理论值 | 价格被低估，买入 |
| 在 SML 上 | 公平定价 | 持有 |
| 在 SML 下方 | 实际预期收益 < CAPM 理论值 | 价格被高估，卖出 |

## 五、CAPM 的扩展

### 5.1 主要扩展模型

| 模型 | 提出者 | 形式 |
|------|--------|------|
| Fama-French 三因子 | Fama & French (1993) | 加入 SMB（小盘）、HML（价值） |
| Carhart 四因子 | Carhart (1997) | 加入 UMD（动量） |
| Fama-French 五因子 | Fama & French (2015) | 加入 RMW（盈利）、CMA（投资） |
| 流动性调整 CAPM | Amihud & Mendelson (1986) | 加入流动性溢价 |
| ICAPM | Merton (1973) | 引入多期、状态变量 |
| Consumption CAPM | Lucas & Breeden (1977) | 基于消费而非市场组合 |

### 5.2 多因子模型示意

```
Fama-French 三因子：
E(Ri) = Rf + β_MKT × ERP
            + β_SMB × E(SMB)
            + β_HML × E(HML)
```

## 六、CAPM 的局限

### 6.1 实证困难

| 局限 | 描述 |
|------|------|
| 市场组合不可观测 | 真实市场组合 M 应包含所有资产，实际用指数代替 |
| 滚动 β 不稳定 | β 随时间变化，回归窗口选择影响结果 |
| 低 β 之谜 | 低 β 组合长期收益 > CAPM 预测（Pzena, Frazzini 2014） |
| 崩盘风险未定价 | CAPM 不区分左尾风险 |
| 流动性未定价 | CAPM 不考虑买卖价差、冲击成本 |
| 异象不消失 | 价值、动量、小盘等异象长期显著 |

### 6.2 低 β 之谜（Frazzini, Israel, Moskowitz 2012）

| 组合 | β | 实际年化收益 | CAPM 预测 |
|------|---|-------------|-----------|
| 高 β 十分位 | 1.61 | 11.3% | 13.3% |
| 低 β 十分位 | 0.35 | 11.2% | 5.4% |

> 低 β 组合跑赢 CAPM 预期 580 bp/年，构成对 CAPM 实证有效性的严重挑战。

## 七、CAPM 的应用场景

### 7.1 主要应用

| 场景 | 用途 |
|------|------|
| 资本预算 | 估算项目折现率（WACC 中的股权成本） |
| 股票估值 | 折现现金流模型（DCF） |
| 业绩评估 | Jensen's Alpha = R_p - [Rf + β(R_m - Rf)] |
| 资源配置 | 风险调整后收益排序 |
| 监管定价 | 受监管行业的合理回报率 |

### 7.2 实际应用注意

| 注意 | 建议 |
|------|------|
| 经验 vs 隐含 ERP | 历史 ERP 与前瞻 ERP 可能不同 |
| 单期 vs 多期 | DCF 使用多期 ERP |
| 行业 β 调整 | 行业可比公司法估算 β |
| 去杠杆 | 杠杆 β → 资产 β → 重新加杠杆 |
