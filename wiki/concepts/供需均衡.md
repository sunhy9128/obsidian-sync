> 供需均衡（Supply and Demand Equilibrium）指在市场价格机制作用下，供给量与需求量相等的状态，此时市场价格为均衡价格、交易数量为均衡数量。

---

## 一、核心概念

### 1.1 需求（Demand）

| 要素 | 说明 |
|------|------|
| 定义 | 消费者在一定时期内、一定价格水平下、愿意且能够购买的商品数量 |
| 影响因素 | 商品自身价格、消费者收入、相关商品价格、偏好、预期 |
| 需求定理 | 其他条件不变时，价格上升 → 需求量减少（反方向变动） |
| 表示方式 | 需求表、需求函数 Q^d = f(P)、需求曲线 |

### 1.2 供给（Supply）

| 要素 | 说明 |
|------|------|
| 定义 | 生产者在一定时期内、一定价格水平下、愿意且能够提供出售的商品数量 |
| 影响因素 | 商品自身价格、生产成本、技术水平、相关商品价格、预期 |
| 供给定理 | 其他条件不变时，价格上升 → 供给量增加（同方向变动） |
| 表示方式 | 供给表、供给函数 Q^s = g(P)、供给曲线 |

### 1.3 市场均衡

```
价格 P
  │
  │  S     E (均衡点)
  │   \   /\
  │    \ /  \  P*  ← 均衡价格
  │     X    \
  │    / \    \
  │   /   D    \
  │  /          \
  └─────────────── 数量 Q
         Q*  ← 均衡数量
```

## 二、均衡价格的决定

### 2.1 数学表达

```
均衡条件：Q^d(P*) = Q^s(P*)

示例：
  需求函数：Q^d = 100 - 2P
  供给函数：Q^s = 20 + 3P

  令 Q^d = Q^s：
  100 - 2P = 20 + 3P
  80 = 5P
  P* = 16

  代入任一方程：
  Q* = 100 - 2×16 = 68
```

### 2.2 三种市场状态

| 状态 | 条件 | 价格变动方向 | 走向 |
|------|------|-------------|------|
| 供过于求 | Q^s > Q^d | P ↓ | 趋向均衡 |
| 供不应求 | Q^d > Q^s | P ↑ | 趋向均衡 |
| 供需均衡 | Q^d = Q^s | 不变 | 稳定状态 |

### 2.3 均衡的性质

- 帕累托效率：均衡状态下无法在不损他方情况下使一方获益
- 自动实现：只要价格机制自由作用，市场自动趋于均衡
- 信息效率：价格反映所有供需信息（信息有效市场假说的微观基础）

## 三、均衡的变动（四象限分析）

### 3.1 需求变动

| 情形 | 需求曲线移动 | 均衡价格 | 均衡数量 |
|------|-------------|---------|---------|
| 需求增加（收入↑、偏好↑） | D → D' 右移 | P* ↑ | Q* ↑ |
| 需求减少（收入↓、偏好↓） | D → D' 左移 | P* ↓ | Q* ↓ |

### 3.2 供给变动

| 情形 | 供给曲线移动 | 均衡价格 | 均衡数量 |
|------|-------------|---------|---------|
| 供给增加（成本↓、技术↑） | S → S' 右移 | P* ↓ | Q* ↑ |
| 供给减少（成本↑、灾害） | S → S' 左移 | P* ↑ | Q* ↓ |

### 3.3 双变情形（同时变动）

```
情形 A：需求↑ + 供给↑ → Q* 一定 ↑，P* 不确定
情形 B：需求↑ + 供给↓ → P* 一定 ↑，Q* 不确定
情形 C：需求↓ + 供给↑ → P* 一定 ↓，Q* 不确定
情形 D：需求↓ + 供给↓ → Q* 一定 ↓，P* 不确定
```

### 3.4 数值示例

```
原始：Q^d = 100 - 2P，Q^s = 20 + 3P → P* = 16，Q* = 68

需求增加（消费者收入提高）：Q^d = 130 - 2P
  130 - 2P = 20 + 3P → P* = 22，Q* = 86
  结论：P* ↑ 6 元，Q* ↑ 18 件

供给增加（技术进步）：Q^s = 50 + 3P
  100 - 2P = 50 + 3P → P* = 10，Q* = 80
  结论：P* ↓ 6 元，Q* ↑ 12 件
```

## 四、价格机制的功能

### 4.1 资源配置三功能

| 功能 | 说明 | 例子 |
|------|------|------|
| 信号传递 | 价格反映稀缺性 | 油价上涨 → 节省用油 |
| 激励引导 | 引导生产者和消费者行为 | 高利润行业吸引投资 |
| 收入分配 | 在生产要素间分配 | 工资 = 劳动服务的价格 |

### 4.2 亚当·斯密"看不见的手"

- 个人追求自身利益最大化
- 在完全竞争市场中 → 自发实现社会资源最优配置
- 前提：完全竞争、无外部性、信息对称

### 4.3 均衡的局限

- 实际市场常存在摩擦：信息不对称、外部性、垄断、公共物品
- 此时需政府干预或制度安排
- 详见 [[市场失灵与外部性]]

## 五、应用案例

### 5.1 农产品价格

| 情形 | 影响 | 原因 |
|------|------|------|
| 风调雨顺年 | 农产品价格↓，农民收入↓ | 供给曲线大幅右移 |
| 自然灾害年 | 农产品价格↑ | 供给曲线大幅左移 |
| 蛛网理论 | 价格周期性波动 | 农产品供给反应滞后 |

### 5.2 房地产调控

| 政策 | 作用机制 | 均衡影响 |
|------|---------|---------|
| 限购 | 抑制需求 | 需求曲线左移，P ↓、Q ↓ |
| 限价 | 直接干预 | 形成 P 低于均衡 → 短缺 |
| 增加土地供应 | 增加供给 | 供给曲线右移，P ↓、Q ↑ |
| 房贷利率 | 影响需求 | 提高利率 → 需求 ↓ |

### 5.3 股票市场

- 股票价格由买卖双方供需决定
- 利好消息 → 需求↑ → 股价↑
- 增发股票 → 供给↑ → 股价↓
- 与商品市场差异：股票是资产，受预期影响更大
