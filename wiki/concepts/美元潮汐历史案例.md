# 美元潮汐历史案例（Dollar Tide Historical Cases）

> 自 1980 年代以来，每一次美元加息周期几乎都伴随新兴市场危机。这不是偶然——是美元潮汐结构性的结果。本页汇编 1982 以来最重要的 EM 危机案例，展示美元收割的完整图景。

## 一、案例地图

```
1982 拉美债务危机 → "失去的十年"
        ↓
1994 龙舌兰危机 → 墨西哥比索崩盘
        ↓
1997 亚洲金融危机 → 索罗斯做空泰铢 + 多国连锁
        ↓
1998 俄罗斯卢布危机 → 主权违约 + LTCM
        ↓
2001 阿根廷债务违约 → 钉住美元 + 比索崩溃
        ↓
2013 Taper Tantrum → 脆弱五国货币普跌
        ↓
2014-2015 俄罗斯卢布危机 → 油价 + 制裁
        ↓
2018 土耳其阿根廷 → 里拉崩盘 + 阿根廷 IMF 救助
        ↓
2022 斯里兰卡违约 → 美联储加息的首次主权违约
        ↓
2022-2024 全球紧缩周期 → 脆弱国家普遍承压
        ↓
2026 新一轮加息周期？ → 待观察
```

## 二、1980s 拉美债务危机：沃尔克紧缩引爆

### 背景（1970s 播种期）

```
1970s 美联储低利率（应对滞胀）
     ↓
拉美"经济奇迹"：巴西、墨西哥、阿根廷高速增长
     ↓
拉美国家大量借入美元债（petrodollar recycling）：
  - 墨西哥：1970s 外债从 $50 亿增至 $800 亿
  - 巴西：外债 $200 亿 → $1000 亿
  - 阿根廷：军政府借款扩张
     ↓
美元债务利率低 + 拉美资源出口收入稳定 → "良性循环"
```

### 收割过程（1979-1982）

```
1979.08 沃尔克任美联储主席
     ↓
1980-1982 美联储激进加息：联邦基金利率最高 20%
     ↓
美元走强（DXY 突破 130）
     ↓
拉美货币贬值（钉住美元制度下实际高估）
     ↓
美元债务本币计价翻倍 + 利率上升 → 还债压力暴增
     ↓
1982.08.12 墨西哥财长宣布"无法按期还债"
     ↓
拉美国家集体违约 → 资本逃离 → 经济崩溃
     ↓
"失去的十年"（The Lost Decade）
  - 拉美 GDP 1980s 几乎零增长
  - 通胀失控：阿根廷 5000%+（1989）
  - 失业率高企
  - 民主化进程倒退
```

### 抄底过程

```
拉美违约后 → 国际银行巨额不良贷款
     ↓
IMF + 世行 + 美国财政部"贝克计划"（1985）"布雷迪计划"（1989）
     ↓
债务重组：旧债换折扣新债（如面值 35 美分换 1 美元新债）
     ↓
美国资本（手握美元）以极低价格收购：
  - 墨西哥石油公司股份
  - 巴西矿山
  - 阿根廷农场（潘帕斯草原）
  - 智利铜矿
     ↓
拉美几十年发展成果被合法收割
```

### 核心数据

| 国家 | 外债（1982） | 危机后 GDP 损失 |
|------|------------|--------------|
| **墨西哥** | $870 亿 | 1980s 累计损失 30%+ |
| **巴西** | $1000 亿 | 1980s 累计损失 20%+ |
| **阿根廷** | $450 亿 | 1989 通胀 5000%+ |

## 三、1994 龙舌兰危机：墨西哥比索崩盘

### 背景

```
1990-1993 美联储降息（应对 1990-91 衰退）
     ↓
资本大量流入墨西哥（NAFTA 预期 + 利率差）
     ↓
墨西哥比索钉住美元 + 大量短期美元债务（tequilas）
     ↓
经常账户逆差扩大 + 政治不稳定（萨利纳斯末年）
```

### 收割过程

```
1994.02 美联储加息（1994 年共加息 6 次）
     ↓
1994.12.19 墨西哥比索贬值 15%（突然宣布）
     ↓
1994.12.22 比索自由浮动 → 几天内跌 40%+
     ↓
外资恐慌性撤离 → 股市暴跌（-50%）
     ↓
1995.01 墨西哥外汇储备告急
     ↓
美国 + IMF 救助 $500 亿（克林顿政府提供 $200 亿互换）
     ↓
"龙舌兰危机"（Tequila Crisis）蔓延拉美
  - 阿根廷：银行危机
  - 巴西：雷亚尔贬值
  - 整个拉美资产价格暴跌
```

### 核心数据

- 比索：3.5:1 → 7.5:1（贬值 60%+）
- 股市：下跌 50%+
- GDP：1995 衰退 6%
- 失业率：从 3% 升至 7%

## 四、1997 亚洲金融危机：教科书级收割

详见 [[1997 亚洲金融危机]]。核心：

```
索罗斯做空泰铢（1997.07.02 泰国放弃固定汇率）
     ↓
泰铢贬值 50%+ → 蔓延东南亚
     ↓
印尼盾贬值 80%+ → 苏哈托下台
     ↓
韩元贬值 50%+ → IMF 救助 570 亿
     ↓
马来西亚拒绝 IMF → 实施资本管制（"反 IMF 路径"）
     ↓
香港金管局击败索罗斯（联系汇率保住了）
     ↓
1998.08 俄罗斯卢布崩盘 + 主权违约
     ↓
LTCM 倒闭 → 美联储紧急降息
     ↓
中国坚持人民币不贬值（"定海神针"）
```

## 五、1998 俄罗斯卢布危机：违约 + 制裁

### 背景

```
1990s 俄罗斯转型期经济
  - 1992-1998 休克疗法 + 私有化
  - 油价从 $20 跌至 $10（1998.08）
  - 财政赤字严重 + 大量短期国债（GKO）
  - 亚洲金融危机蔓延
```

### 收割过程

```
1998.08.17 俄罗斯宣布：
  - 卢布贬值（从 6.3:1 → 21:1）
  - 暂停偿付外债（主权违约）
  - 90 天延期偿付
     ↓
国际炒家狙击 GKO 国债 → 央行耗尽外储
     ↓
违约冲击全球：
  - 全球股市暴跌
  - 美俄利差扩大
  - 新兴市场资金外逃
     ↓
LTCM（长期资本管理公司）倒闭
  - 因俄罗斯违约 + 利差交易失败
  - 美联储紧急救助（15 家银行注资 $3.6 亿）
     ↓
美联储紧急降息 → 美元潮汐暂时反向
```

### 核心数据

- 卢布：6.3:1 → 21:1（贬值 70%+）
- 俄罗斯股市：下跌 90%+
- 主权违约：$400 亿
- 油价：$20 → $10（同期）

## 六、2001 阿根廷债务违约：钉住美元制度的代价

### 背景

```
1991 梅内姆政府推出"兑换计划"（Convertibilidad）
  - 比索钉住美元 1:1
  - 货币政策完全跟随美联储
  - 大量短期美元债务
```

### 收割过程

```
1990s 美联储降息 → 资本流入阿根廷 → 经济繁荣
     ↓
1999 巴西雷亚尔贬值（拉美传染）→ 阿根廷出口受冲击
     ↓
2000-2001 美国互联网泡沫 → 全球资本回流美元
     ↓
阿根廷经常账户恶化 + 财政赤字扩大
     ↓
2001.07 阿根廷开始出现挤兑
     ↓
2001.12.01 政府宣布限制取款（" Corralito"）
     ↓
社会动荡 → 5 位总统 2 周内更替
     ↓
2002.01 比索放弃钉住 → 暴跌至 4:1
     ↓
2002 阿根廷比索贬值 70%+ → GDP 衰退 10.9%
     ↓
2005 阿根廷违约 $820 亿（史上最大违约）
     ↓
2014 再次违约（"秃鹫基金"对决）
```

### 核心数据

- 比索：1:1 → 4:1（贬值 75%+）
- GDP：2002 衰退 10.9%
- 失业率：从 13% 升至 25%
- 贫困率：从 25% 升至 50%+

## 七、2013 Taper Tantrum：缩减恐慌

### 背景

```
2008-2013 美联储 QE（QE1+QE2+QE3+扭曲操作）
  - 美联储资产负债表从 $8000 亿扩至 $4.5 万亿
  - 全球流动性泛滥
     ↓
EM 享受资本流入 + 货币升值 + 资产泡沫
```

### 收割过程

```
2013.05.22 伯南克在国会作证：暗示 QE 可能"缩减"（Taper）
     ↓
市场解读为 QE 即将退出 → 美债收益率飙升
     ↓
脆弱五国货币普跌（详见 [[脆弱五国（Fragile Five）]]）：
  - 印度卢比：创历史新低
  - 印尼卢比：央行干预
  - 巴西雷亚尔：暴跌
  - 土耳其里拉：暴跌
  - 南非兰特：暴跌
     ↓
EM 央行被迫加息应对：
  - 印度：+150bp
  - 印尼：+175bp
  - 巴西：+250bp
  - 土耳其：+125bp
     ↓
2013.12-2014 美联储正式缩减 QE → 资本持续流出
```

## 八、2014-2015 俄罗斯卢布危机：油价 + 制裁

### 背景

```
2014.02 乌克兰危机 → 美欧制裁俄罗斯
     ↓
2014 中东 OPEC 拒绝减产 + 美国页岩油革命
     ↓
油价从 $110 暴跌至 $30（2014-2016）
```

### 收割过程

```
俄罗斯收入主要靠石油出口 → 油价暴跌 → 卢布贬值压力
     ↓
制裁 → 外资撤离 → 卢布加速贬值
     ↓
2014.12 卢布崩盘：1 天跌 20%+（俄罗斯央行紧急加息至 17%）
     ↓
2015 卢布全年贬值 50%+（从 33:1 跌至 80:1）
     ↓
俄罗斯经济衰退：
  - 2015 GDP 衰退 -2%
  - 通胀飙升（15%+）
  - 实际工资下降
```

### 核心数据

- 卢布：33:1 → 80:1（贬值 60%+）
- 油价：$110 → $30（同期）
- GDP：2015 衰退 -2%
- 通胀：15%+

## 九、2018 土耳其阿根廷危机：脆弱五国再评估

### 土耳其里拉崩盘

```
2018 美联储加息（4 次）
     ↓
特朗普对土耳其钢铝加征关税 + 制裁土耳其牧师
     ↓
土耳其经常账户逆差大 + 通胀 15%+ + 埃尔多安反对加息
     ↓
2018.08 里拉崩盘：1 天跌 20%
     ↓
土耳其央行被迫加息（24%）
     ↓
土耳其陷入货币危机 + 高通胀（80%+）
     ↓
延续到 2021-2023：埃尔多安坚持低利率政策 → 通胀失控 80%+
```

### 阿根廷比索危机

```
2018.04 美联储加息 + 美元走强
     ↓
阿根廷马克里政府 IMF 救助（$570 亿）
     ↓
2018 比索年内贬值 50%+（黑市价更高）
     ↓
2019.10 马克里败选 → 费尔南德斯当选
     ↓
2020 阿根廷再次违约（$650 亿）→ 史上第 9 次违约
```

## 十、2022 斯里兰卡违约：美联储激进加息的代价

### 背景

```
2020-2021 美联储无限 QE
     ↓
斯里兰卡大量借美元债（基础设施建设）
     ↓
2021.04 总统拉贾帕克萨禁止化肥进口（"有机农业"）→ 农业崩溃
     ↓
2022 美联储激进加息（全年 +425bp）
     ↓
美元走强 + 油价上涨 + 食品危机
```

### 收割过程

```
2022.01 卢比贬值 30%+（钉住美元制度崩溃）
     ↓
外汇储备从 $80 亿（2021） → $5000 万（2022）
     ↓
2022.04 斯里兰卡宣布"暂停偿付所有外债"
     ↓
2022.07 示威者占领总统府
     ↓
2022.07 拉贾帕克萨总统出逃 → 辞职
     ↓
斯里兰卡陷入政治经济双重危机
     ↓
IMF 救助（$29 亿，2023）
     ↓
违约规模：$125 亿
```

### 核心数据

- 卢比：200:1 → 360:1（贬值 80%+）
- 通胀：70%+（2022.09）
- GDP：2022 衰退 -7.8%
- 外储：$80 亿 → $5000 万

## 十三、案例的共同规律

### 规律 1：触发条件 = 美联储加息周期

```
每次 EM 危机都在美联储加息周期的早中期爆发：
  - 1980-1982 沃尔克：1982 拉美
  - 1994-1995 格林斯潘：1994 龙舌兰
  - 1997-1998 格林斯潘：1997 亚洲
  - 2004-2006 格林斯潘/伯南克：EM 局部
  - 2015-2018 耶伦/鲍威尔：2018 土耳其阿根廷
  - 2022-2024 鲍威尔：2022 斯里兰卡
```

### 规律 2：触发顺序

```
1. 美元走强（DXY ↑）
     ↓
2. EM 货币贬值（汇率渠道）
     ↓
3. EM 外债负担飙升（资产负债表渠道）
     ↓
4. 资本外逃加速（金融渠道）
     ↓
5. 央行干预耗尽外储
     ↓
6. 资产价格暴跌 + 违约
     ↓
7. IMF 救助 / 资本管制 / 货币贬值
     ↓
8. 美国资本抄底
```

### 规律 3：EM 危机 = "基本面的胜利"

> 每次危机后，幸存者都会反思：
> - 为什么相信"这次不一样"？
> - 为什么忽视美元周期？
> - 为什么被低利率诱惑？
> - 为什么没准备好充足的防御？

详见 [[新兴市场为避免被美国薅羊毛采取了哪些措施]]。

## 十四、一句话总结

> **美元潮汐历史案例 = 自 1980 年代以来，美联储每一次加息周期都伴随新兴市场危机**。
>
> 最重要的案例包括：**1982 拉美**（沃尔克紧缩引爆）、**1994 龙舌兰**（墨西哥比索崩盘）、**1997 亚洲**（索罗斯做空泰铢）、**1998 俄罗斯**（卢布崩盘+主权违约）、**2001 阿根廷**（钉住美元的代价）、**2013 Taper Tantrum**（脆弱五国）、**2018 土耳其阿根廷**、**2022 斯里兰卡违约**、**2022-2024 全球紧缩**。
>
> **核心规律**：触发条件 = 美联储加息周期 + EM 自身脆弱性（外债、经常账户逆差、汇率僵化）。**触发顺序**：美元走强 → EM 货币贬值 → 资本外逃 → 央行干预 → 资产暴跌 → 违约 → 美国资本抄底。
