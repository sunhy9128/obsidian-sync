## Basic Information
- **Type**: concept（经济学/宏观经济学 — 价格与货币理论）
- **Sources**: [[sources/我们已经处于新一轮加息周期中或前夜|我们已经处于新一轮加息周期中或前夜]]（新增）
- **Definition**: 通胀（Inflation）指物价水平持续、普遍上涨的现象，通常以消费者价格指数（CPI）同比变化率衡量。CPI 反映居民购买的商品和服务价格变动，PPI 反映企业生产端的价格变动，二者构成通胀分析的核心指标体系。

## Description

### 一、CPI（消费者价格指数）

CPI（Consumer Price Index）衡量**居民**购买的一篮子固定商品和服务价格随时间的变化。

CPI_t = (当期一篮子商品价格 / 基期一篮子商品价格) × 100
π_t = (CPI_t - CPI_{t-12}) / CPI_{t-12} × 100%

**中国 2024 参考篮子构成**：

| 大类 | 权重 | 子项示例 |
|------|------|----------|
| 食品烟酒 | 27% | 粮食、猪肉、鲜菜、烟酒 |
| 居住 | 24% | 房租、水电、建材 |
| 教育文化娱乐 | 14% | 培训、文娱、景点 |
| 交通通信 | 13% | 油价、车辆、通讯 |
| 医疗保健 | 10% | 药品、诊疗 |
| 衣着 | 6% | 服装、鞋类 |
| 生活用品及服务 | 4% | 家居、家政 |
| 其他 | 2% | 首饰、理发 |

> 不同国家篮子不同：美国 CPI 住房权重 ~33%；欧元区权重侧重能源。

**CPI 的两个版本**：
- **总体 CPI（Headline）**：反映全部商品价格，固定权重；
- **核心 CPI（Core）**：剔除食品与能源，剔除后更能反映**潜在通胀趋势**，是央行决策的关键参考。

### 二、PPI（生产者价格指数）

PPI（Producer Price Index）衡量**企业**出厂端的价格变化，位于价值链上游，通常领先 CPI 3-6 个月，构成"PPI 上涨 → 企业利润压缩 → 传导到 CPI → 央行加息"的典型传导链。但在中国"猪周期"叠加下，CPI 也可独立于 PPI 剧烈波动。

### 三、通胀的三大类型

| 类型 | 曲线移动 | 触发因素 | 政策反应 | 案例 |
|------|----------|----------|----------|------|
| **需求拉动型** | AD → | 货币宽松、财政刺激、出口暴增 | 加息（长期有效） | 1970s 美国；2020-2021 美联储 QE |
| **成本推动型** | AS ← | 油价、工资、原材料 | 难以应对（加息进一步抑制供给） | 1973 第一次石油危机；2022 俄乌冲突 |
| **结构性** | 部分 | 部门间不均衡 | 精准调控 | 中国"猪周期" |

### 四、通胀的三大成本

- **皮鞋成本（Shoe-Leather Cost）**：通胀 ↑ → 实际利率 ↓ → 持币机会成本 ↑ → 频繁跑银行取款。
- **菜单成本（Menu Cost）**：企业调整价格的成本（印新菜单、重新报价、谈判合同），导致微观价格"粘性"（Sticky Price）。
- **财富再分配**：债务人受益、债权人受损；固定收入者（退休人员、年金领取者）受损；持有实物资产（房产、股票）者受益。
- **其他成本**：税收扭曲（名义资本利得被通胀推高）、国际竞争力下降（实际汇率上升）、资源配置扭曲、预期失稳（通胀预期脱锚进入恶性循环）。

### 五、恶性通胀案例

| 案例 | 峰值数据 | 核心原因 |
|------|----------|----------|
| 魏玛德国（1923） | 1 美元 = 4.2 万亿马克 | 一战赔款 + 罢工 + 货币超发 |
| 津巴布韦（2008） | 年化通胀 8900 万亿% | 战争 + 土地改革 + 货币超发 |
| 委内瑞拉（2018） | 通胀 1,698,488%（IMF 估算） | 石油收入崩溃 + 财政赤字货币化 |
| 民国金圆券（1948-49） | 1 美元 = 12 亿金圆券 | 内战军费 + 货币超发 |

> **共性**：① 财政赤字货币化 ② 政治不稳定 ③ 货币信心崩塌 ④ 一旦预期脱锚，物价飞涨速度极快。

### 六、通胀的"对偶"：通缩

| 维度 | 通胀 | 通缩 |
|------|------|------|
| 物价 | 持续上涨 | 持续下跌 |
| 实际利率 | 名义利率 - π，可能为负 | 名义利率 - π，偏高 |
| 消费 | 提前购买 | 延后购买（等待降价） |
| 投资 | 资产升值 | 实际利率高，抑制投资 |
| 债务 | 实际负担下降 | 实际负担上升 |
| 案例 | 1970s 美国 | 1990s 日本；2020 中国疫情初期 |

### 七、新一轮通胀的结构性驱动因素（来自新来源）

本轮通胀并非单一能源成本上涨，而是**多因素叠加的复合结构**，包括：

1. **能源成本上升**（上游推力）— 参见 [[concepts/输入型通胀|输入型通胀]] 机制；
2. **霍尔木兹海峡危机**（地缘风险溢价）— 影响全球原油运输通道；
3. **能源中下游产品**价格传导（炼油、化工、运输）；
4. **输入型通胀** — 通过大宗商品、国际能源价格向国内传导；
5. **中欧贸易战 + 301 条款关税战**（参见 [[concepts/贸易战|贸易战]]）— 推高进口商品价格；
6. **日韩加薪**（劳动力成本端）— 推动区域制造成本上升；
7. **地缘博弈下的国防与产业财政支出**（参见 [[concepts/财政政策乘数|财政政策乘数]]）— 财政扩张释放需求端压力；
8. **新一轮加息周期**（参见 [[concepts/新一轮加息周期|新一轮加息周期]]、[[concepts/美元加息周期|美元加息周期]]）的滞后效应。

> 多元驱动力意味着央行仅靠加息难以全面抑制通胀——加息可以压制需求拉动型通胀，但对**成本推动型**与**结构性**通胀效果有限，凸显了**多源诊断**的重要性。
