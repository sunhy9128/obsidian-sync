# Treuhand 私有化

> Treuhandanstalt — 1990-1994 年东德国有资产的"快速私有化"实验

**Treuhand 私有化**是指 1990 年 6 月成立的 **Treuhandanstalt**（托管局/信托局）对东德国有资产的拆分、出售、清算、重组过程。该机构管理了约 8000 家国有企业、400 万员工、约 6000 亿美元资产，是 20 世纪最大规模的私有化实验。其"快速脱手"的私有化策略引发了"廉价抛售"争议，但也被认为是两德统一成功的关键机制之一。托管局于 1994 年 12 月 31 日完成使命后正式解散。

## 二、成立背景

### 2.1 紧急成立的法律基础

```
【1990 年东德国有资产的"紧急托管"】

1990-03-01：东德大选前的"人民财产"国有化
├── 东德人民议院（即将下台）
├── 通过《人民财产法》
├── 宣布原东德所有国营企业
│   └── 转为"人民财产"（Volkseigentum）
└── 由新设的 Treuhandanstalt 接管

【成立的法律逻辑】
├── 1. 防止统一前资产被瓜分
├── 2. 建立统一的管理机构
├── 3. 为后续私有化做准备
└── 4. 政治过渡的紧急安排
```

### 2.2 接手时的资产规模

```
【Treuhand 接手的"遗产"】

| 指标 | 数量 |
|------|------|
| 国营企业（VEB） | ~8000 家 |
| 员工总数 | ~400 万 |
| 总资产 | ~$6000 亿 |
| 总负债 | ~$3000 亿 |
| 净资产 | ~$3000 亿 |
| 土地 | 约占东德国土 1/3 |

【资产构成】
├── 大型联合企业（Kombinate）：200+ 家
├── 中小型企业：数千家
├── 土地、森林、房产
├── 银行、保险等金融机构
├── 农业合作社（LPG）
└── 文化、教育、医疗等公共机构
```

## 三、组织架构与运作

### 3.1 组织架构

```
【Treuhand 的组织结构】

主席（President）
├── 主席团（管理委员会）
├── 各部门
│   ├── 私有化部
│   ├── 重组部
│   ├── 基础设施部
│   ├── 金融部
│   ├── 法律部
│   └── 国际关系部
└── 15 个地区办公室

【人员规模】
├── 初期：~200 人
├── 1991：~1000 人
├── 1992：~3000 人
└── 1993：峰值 ~4000 人

【员工构成】
├── 西德借调人员
├── 东德原有国营企业管理者
├── 国际顾问
├── 投资银行家
└── 法律专家
```

### 3.2 私有化的四大策略

```
【Treuhand 私有化的四种方式】

方式一：整体出售（Mass Privatization）
├── 大型 VEB 联合企业
├── 整体出售给战略投资者
├── 优先考虑"保留就业"的方案
└── 例：Carl Zeiss Jena 出售给西德 Carl Zeiss

方式二：拆分出售（Spin-off）
├── 拆分大型联合企业为多个独立公司
├── 分别出售给不同投资者
├── 例：Jenoptik（光学）拆分自 Carl Zeiss
└── 例：Infineon Dresden（半导体）拆分自 Robotron

方式三：清算关闭（Liquidation）
├── 无法挽救的企业
├── 直接关闭
├── 员工失业 + 资产处理
└── 例：~3500 家企业被清算

方式四：管理层收购（MBO）
├── 现有管理层收购
├── 通常需要外部资金支持
├── 案例较少（多被批评为"内部人交易"）
└── 例：少数中型企业
```

### 3.3 私有化的决策原则

```
【Treuhand 的"四不原则"】

不 ①：不保护"国营企业"
├── 不试图挽救所有企业
└── 让市场决定生死

不 ②：不保护"本土资本"
├── 优先西德/外国投资者
└── 即使东德人买得起也优先外部投资者

不 ③：不保护"原有管理层"
├── 多数原东德管理者被解雇
└── 引入新管理团队

不 ④：不考虑"政治后果"
├── 短期失业大量产生
├── 但追求"快速私有化"
└── 被批评为"对社会不负责"

【实际决策标准（按重要性排序）】
├── 1. 投资者提供的就业承诺
├── 2. 投资规模
├── 3. 收购价格
└── 4. 战略合理性
```

## 四、私有化的实际过程（1990-1994）

### 4.1 第一阶段：接管与盘点（1990.06 - 1991.06）

```
【紧急期】

1990-06-17：Treuhand 成立
1990-07：货币兑换（与 Treuhand 并行）
1990-08：开始接手第一批国营企业
1990-10：两德统一
1990-12：完成大部分企业登记

【第一阶段的工作】
├── 1. 紧急接管所有国营企业
├── 2. 评估资产状况
├── 3. 建立管理团队
├── 4. 准备私有化方案
└── 5. 处理紧急财务危机（多数企业已濒临破产）

【挑战】
├── 缺乏完整的企业账目
├── 资产负债不清
├── 管理层能力不足
└── 政治压力巨大
```

### 4.2 第二阶段：大规模私有化（1991.06 - 1992.12）

```
【高峰期】

1991：开始大规模私有化
├── 平均每月私有化 500+ 家企业
├── 与西德投资者大规模谈判
└── 与国际投资者开始接触

【典型案例】

Carl Zeiss Jena（光学）：
├── 1991：出售给西德 Carl Zeiss
├── 保留 ~50% 就业
├── 后期成立 Jenoptik
└── 成为东德成功案例

Robotron（计算机）：
├── 1990-1991：拆分
├── 部分出售给 Siemens
├── 部分成立独立的 Elster（能源管理）
└── 部分清算

化工联合企业（Buna, Leuna）：
├── 1990-1994：多次尝试出售
├── 最终部分出售给西方投资者
├── 部分清算
└── 大量员工失业

【第二阶段的成果】
├── 私有化 ~8000 家企业
├── 重新私有化失败率：~80%（被重新售出）
├── 直接私有化失败：~20%
└── 总就业损失：~250 万人
```

### 4.3 第三阶段：清算与收尾（1993.01 - 1994.12）

```
【清理期】

1993：意识到需要更长时间
├── 仍有 ~3000 家企业未处置
├── 大量工人失业
└── 财政负担沉重

1993-1994：清理与收尾
├── 关闭剩余不可挽救的企业
├── 处理土地与基础设施
├── 完成所有剩余私有化
└── 准备解散

1994-12-31：Treuhand 解散
├── 累计私有化 ~14000 家企业
├── 清算 ~3500 家企业
├── 总员工损失：~400 万 → 实际剩余就业 ~150 万
└── 总财政赤字：~$2700 亿（由联邦政府承担）
```

## 五、私有化的财务成果

### 5.1 财务总账

```
【Treuhand 的最终财务结果】

接手资产：~$6000 亿（账面价值）
实际处置：
├── 私有化收入：~$700 亿
├── 清算资产价值：~$300 亿
└── 累计回收：~$1000 亿

承担的成本：
├── 失业救济：~$500 亿
├── 提前退休：~$400 亿
├── 企业重组补贴：~$300 亿
├── 环保清理：~$200 亿
├── 其他支出：~$300 亿
└── 总支出：~$1700 亿

最终赤字：~$2700 亿（由联邦预算承担）
```

### 5.2 关键财务指标

```
【财政赤字的资金来源】

├── 1. 联邦政府直接拨款
├── 2. 团结税（Solidaritätszuschlag）
├── 3. 出售资产的收入
├── 4. 借入债务（后转为联邦债务）
└── 5. 欧盟团结基金（少量）

【长期成本（1990-2020）】
├── Treuhand 直接成本：~$2700 亿
├── 后续失业救济：~$5000 亿+
├── 提前退休成本：~$3000 亿+
└── 总财政成本：~$2 万亿（30 年累计）
```

## 六、典型案例研究

### 6.1 成功案例

#### ✅ 案例一：Carl Zeiss Jena（光学）

```
【Carl Zeiss Jena 私有化】

背景：
├── 耶拿的光学巨头
├── 东德"科技明星"企业
├── 但受计划经济体制束缚

私有化过程：
├── 1991：与西德 Carl Zeiss 谈判
├── 关键条件：保留耶拿总部 + 50% 就业
├── 最终方案：西德 Carl Zeiss 接管
└── 1991-1995：大规模投资 + 现代化

结果：
├── 保留 ~5000 个就业岗位
├── 成为东德"现代化成功案例"
├── 衍生出 Jenoptik（独立上市公司）
├── 至今仍是耶拿的核心企业
└── 是 Treuhand 少数"保留本土品牌"的案例

【成功因素】
├── 1. 行业有竞争力（光学）
├── 2. 投资者保留品牌 + 就业
├── 3. 当地政府支持
└── 4. 工人愿意配合
```

#### ✅ 案例二：大众透明工厂（德累斯顿）

```
【大众德累斯顿透明工厂】

背景：
├── 1990 年代东德汽车业崩溃
├── 但大众看好东德汽车工业基础
└── 决定在德累斯顿建"透明工厂"

过程：
├── 2002：建成投产
├── 投资 €1.86 亿
├── 雇佣 ~600 工人
└── 象征意义大于经济意义（产能有限）

意义：
├── 1. 树立"东德可以造豪华车"的形象
├── 2. 重塑东德汽车工业基础
├── 3. 是 Treuhand 私有化后"再投资"的典范
└── 4. 但经济回报有限（主要做 Phaeton 豪华车）
```

### 6.2 失败案例

#### ❌ 案例一：国营煤矿（褐煤）

```
【东德国营褐煤矿区】

背景：
├── 东德是世界最大褐煤生产国之一
├── 是空气污染的主要来源
├── 提供数万就业岗位

Treuhand 决策：
├── 1990-1994：私有化多数煤矿
├── 但环保成本极高
├── 最终多数被关闭或重组
└── 数万矿工失业

后果：
├── 1. 短期：大规模失业
├── 2. 中期：环境治理（耗资数十亿）
├── 3. 长期：能源转型（逐步淘汰褐煤）
└── 4. 是东德"环保成本"的典型案例

【教训】
├── 不是所有私有化都成功
├── 行业重组需要更长时间
└── 环境成本常被低估
```

#### ❌ 案例二：东德造船业

```
【东德造船业】

背景：
├── 东德有 4 家大型造船厂
├── 集中于波罗的海沿岸
├── 是计划经济下"出口创汇"的重要行业

私有化过程：
├── 1990-1993：与西德、挪威、亚洲投资者谈判
├── 多数最终清算
├── 部分出售但很快关闭
└── 数万工人失业

结果：
├── 4 家船厂全部倒闭
├── 数万就业岗位消失
├── 至今这些沿海城市仍在转型中
└── 是东德"产业崩溃"的典型案例

【教训】
├── 计划经济的"出口导向"产业
├── 在西方市场竞争中难以生存
├── 私有化只能加速而非延缓崩溃
└── 留下大量"工业废墟"待处理
```

## 七、五大争议

### 7.1 争议一："廉价抛售"

```
【廉价抛售论】

指控：
├── Treuhand 以过低价格出售东德资产
├── 大量优质资产被西德/外国投资者"低价收购"
├── 损害了东德的长期利益
└── 类似"经济殖民"

证据：
├── 多数资产售价仅为账面价值的 10-30%
├── 西德企业收购后立即提价或合并
└── 优质品牌（如 Trabant）被废弃

反指控：
├── 多数国营企业资不抵债
├── 不低价出售无人接手
├── "快速脱手"是政治必须
└── 是"次优选择"而非"最优方案"

【学界评价】
├── 主流观点：确实是"廉价抛售"，但别无选择
├── 少数派：更系统化的方案可能效果更好
└── 历史结论：1990s 政治环境决定的结果
```

### 7.2 争议二：本土企业家的排斥

```
【本土资本被排斥】

指控：
├── Treuhand 优先西德/外国投资者
├── 东德本土企业家难以参与
├── 错过了培育本土资本的窗口期
└── 形成"东德经济被西德接管"

证据：
├── 多数东德本土买家被拒（缺乏资金 + 经验）
├── 西德企业用东德资产获取超额利润
└── 形成"利润汇回西德"的长期格局

反指控：
├── 东德本土资本极其有限
├── 缺乏现代企业经营管理能力
├── 优先外部投资者是合理选择
└── 但可以采取更多"本土培育"措施

【学界评价】
├── 主流观点：本土资本排斥是结构性失误
├── 影响：东德 30 年来仍缺少本土大企业
└── 后果：利润持续外流，本土创新不足
```

### 7.3 争议三：就业保护不足

```
【就业保护的失败】

实际数据：
├── 1990：400 万 Treuhand 员工
├── 1994：~150 万仍就业
├── 净失业：~250 万人
├── 加上未就业的"工作年龄段"人口：~400 万人
└── 占东德劳动力 ~50%

指控：
├── Treuhand 优先"价格"而非"就业"
├── 大量工人被抛向社会
├── 长期失业 + 提前退休 + 福利依赖
└── 形成"东德社会问题"

反指控：
├── 不私有化同样会大规模失业
├── 计划经济企业本身已经"隐性失业"
├── 私有化是"显性化"不可避免的过程
└── 长期看，私有化是必要的

【学界评价】
├── 主流观点：就业保护是 Treuhand 的最大失误
├── 后果：长期失业 + 福利依赖 + 人口外流
└── 30 年后仍未完全解决
```

### 7.4 争议四：政治化的"快速脱手"

```
【快速脱手的政治动机】

背景：
├── 科尔政府 1990-1994 大选压力
├── 1990-12 全德选举，科尔承诺"快速统一"
├── 1994 大选前需要"展示成绩"
└── 形成"快速私有化"的政治压力

证据：
├── 多数优质资产在 1991-1992 集中出售
├── 价格普遍偏低（"快速脱手"心态）
├── 部分本可以保留的企业被出售
└── 形成"为政治而私有化"的批评

【学界评价】
├── 主流观点：政治化是 Treuhand 的结构性问题
├── 影响：长期经济格局
└── 教训：私有化应避免政治压力干扰
```

### 7.5 争议五：民主合法性的质疑

```
【民主合法性的问题】

核心问题：
├── Treuhand 是过渡期紧急机构
├── 缺乏民主监督
├── 决策过程不透明
├── 影响数十万人的命运
└── 但没有真正的公共问责

指控：
├── "私下决定数百万人的命运"
├── 缺乏议会有效监督
├── 与公众沟通不足
└── 形成"私有化黑箱"

反指控：
├── 紧急情况下必须快速决策
├── 议会主要负责法律框架
├── 公开所有细节会影响私有化效果
└── 总体上是负责任的机构

【学界评价】
├── 主流观点：民主合法性确实不足
├── 教训：大规模私有化需要更多公众参与
└── 影响：至今仍有"东德被掠夺"的叙事
```

## 八、长期影响

### 8.1 经济影响

```
【Treuhand 的长期经济遗产】

积极影响：
├── 1. 快速建立市场经济结构
├── 2. 引入现代企业管理
├── 3. 融入德国及全球经济
├── 4. 引入大量外资和先进技术
```

## 十、关键洞察总结

> [!key-insight] Treuhand 是"政治豪赌"的执行机构
>
> Treuhand 不是独立机构，而是科尔政府"快速统一"政策的执行者。其"快速脱手"是政治压力的结果，而非经济理性的选择。

> [!key-insight] 私有化 ≠ 现代化
>
> Treuhand 实现了所有权转移，但未能建立"自主现代化"机制。东德本土产业的空心化至今仍是核心结构问题。

> [!contradiction] "廉价抛售"的两难
>
> Treuhand 被批评"廉价抛售"，但当时确实别无选择——不低价无人接手。这是 1990s 政治经济环境的"次优解"。

> [!key-insight] 民主合法性的缺失
>
> Treuhand 是过渡期紧急机构，缺乏民主监督。这种"私下决定数百万人的命运"的方式至今仍受争议。

> [!key-insight] "快速脱手"的长期成本
>
> 短期看，Treuhand 完成了私有化任务。长期看，"快速脱手"导致的本土产业空心化是东德经济持续困境的根源之一。
