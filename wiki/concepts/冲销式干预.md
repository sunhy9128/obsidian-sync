## Basic Information
- **类型（Type）**: concept
- **来源（Sources）**: [[冲销式干预]]
- **定义（Definition）**: 冲销（冲销式干预）是央行在外汇干预时为抵消对国内流动性的影响而进行的对冲操作。央行在外汇市场买入外币时，会向市场投放本币导致流动性宽松——冲销的作用就是把这部分多余的流动性再收回来，从而同时实现汇率稳定和国内货币政策独立。

## Description

冲销式干预是央行外汇干预与货币政策之间的"桥梁"工具——其核心逻辑在于：央行在外汇市场买入外币（干预汇率）时必然投放本币流动性，而冲销操作就是通过卖出本国债券或发行央票等方式把这部分多余的本币流动性再收回来，从而同时实现汇率稳定和国内货币政策独立这两个目标。作为连接外汇市场与货币政策的枢纽性工具，冲销式干预在理论上完美地规避了"蒙代尔三元悖论"的限制，但这一效果高度依赖**资本管制**这一前提条件。

冲销操作的工具谱系丰富，主要包括：公开市场操作（发行央行票据）、正回购、上调存款准备金率、SLF/MLF/PSL 等结构性工具，以及外汇掉期冲销。不同工具在灵活性、力度、期限上各有差异，央行会根据流动性管理需要进行组合使用。冲销式干预与[[非冲销式干预]]的关键区别在于对国内货币供应量的影响——冲销保持中性，非冲销则会导致货币扩张。

冲销式干预存在显著的局限性。其有效性受制于五大因素：资本管制强度、冲销规模、市场贬值预期、外汇储备充裕度以及长期使用累积的利息成本与汇率损失。中国是全球最广泛使用冲销式干预的经济体之一，从 2002-2011 年的央票+正回购+升准组合，到 2013 年后 MLF/PSL 等结构性工具的精细化调控，再到 2018 年后离岸央票的延伸使用，冲销工具的演进折射出中国外汇市场与货币政策的协同变迁。2015 年[[8-11汇改]]后，人民币贬值压力下外汇储备消耗速度快于冲销回笼速度，曾形成半冲销状态。

### 一、基本原理

央行目标：在不改变国内流动性的前提下阻止本币贬值

操作步骤：
Step 1：外汇市场买入美元，卖出人民币
        → 人民币汇率稳定
        → 但人民币被投放 → 国内流动性增加 → 通胀压力

Step 2：冲销操作——央行卖出本国债券（或发行央票）
        → 回笼多余的人民币流动性
        → 国内流动性恢复中性
        → 通胀压力被抵消

结果：汇率稳定 + 国内货币供应不变 = 货币政策独立

### 二、冲销 vs 非冲销对比

| 维度 | 冲销式干预 | 非冲销式干预 |
|------|-----------|-------------|
| **操作** | 买入外币 + 卖本币债券冲销 | 买入外币但不冲销 |
| **对国内流动性的影响** | 中性（抵消） | 扩张（货币投放） |
| **对汇率的影响** | 有（直接干预） | 有（且更强） |
| **对通胀的影响** | 中性 | 上升 |
| **货币政策独立性** | 维持 | 受损 |
| **典型使用者** | [[中国央行]] | 美联储（较少用） |
| **理论效果** | 短期有效，长期有限 | 短期+长期均有效，但副作用大 |

### 三、冲销工具

| 工具 | 机制 | 特点 |
|------|------|------|
| **公开市场操作（央行票据）** | 央行发行央票，收回商业银行流动性 | 最灵活，可精细调控 |
| **正回购** | 央行向商业银行卖出债券，约定未来买回 | 短期工具，到期后需续作 |
| **存款准备金率上调** | 提高商业银行法定准备金率，锁定超额准备金 | 力度大，影响深远，但不够灵活 |
| **SLF/MLF/PSL** | 央行向金融机构提供资金，调节流动性结构 | 结构性工具，定向调控 |
| **外汇掉期冲销** | 用外汇掉期将本币流动性置换为外币流动性 | 国际常用，兼顾外汇储备管理 |

### 四、冲销的局限性

> **蒙代尔三元悖论**告诉我们：汇率稳定、货币政策独立、资本自由流动三者不可兼得。冲销式干预的核心假设是：**资本管制存在**。如果没有资本管制，套利资金会迅速抵消冲销效果。

| 失效条件 | 冲销效果 |
|----------|----------|
| 资本管制较弱 | 套利资金绕过管制，抵消冲销 |
| 冲销规模不足 | 流动性继续扩张 |
| 市场形成持续贬值预期 | 干预无效，预期自我实现 |
| 外汇储备不足 | 无法持续干预 |
| 长期使用 | 成本递增（利息成本 + 汇率损失） |

### 五、关键应用场景

- **外汇占款扩张期（2002-2013）**：中国通过央票+正回购+升准组合有效对冲双顺差和热钱流入带来的流动性扩张
- **结构性调控期（2013 至今）**：MLF/PSL 替代传统工具，实现定向流动性管理
- **离岸市场延伸（2018 至今）**：离岸央票用于离岸 CNH 冲销，延伸了冲销地理边界
- **半冲销状态（2015-2016）**：8.11 汇改后外汇储备消耗快于冲销回笼速度
