# 团结税

> Solidaritätszuschlag — 1991-2020 年西德为支援东德建设而征收的所得税附加税

**团结税**（Solidaritätszuschlag，"Soli"）是德国于 1991 年开始征收的所得税附加税，初始目的是筹集资金支持东德重建（Aufbau Ost，"东部建设"）。税率最初为 7.5%，后调整为 5.5%，是德国历史上最大规模的财政转移支付工具之一。该税于 2021 年正式取消，但东德地区仍通过其他财政渠道获得转移支付，是两德统一"长期补贴"的核心机制。

## 一、基本信息

| 项目 | 内容 |
|------|------|
| 全称 | Solidaritätszuschlag |
| 中文译名 | 团结税 / 团结附加税 |
| 实施时间 | 1991-2020（完整征收）；2021 起部分取消 |
| 立法基础 | 《团结税法案》（Solidaritätszuschlaggesetz）|
| 税率 | 初期 7.5%，1998 起 5.5% |
| 适用范围 | 个人 + 企业所得税附加 |
| 初始目的 | 东德重建（Aufbau Ost）|
| 后续目的 | 统一相关开支 + 一般联邦支出 |
| 累计收入 | ~€1700 亿（1991-2019）|
| 年度峰值 | ~€200 亿（1990s 末）|

## 二、团结税的诞生

### 2.1 政治背景

```
【团结税诞生的政治逻辑】

1990-10-03：两德统一
├── 东德 5 个州加入联邦
├── 立即需要大量资金重建
├── 西德财政部估算：至少 DM 1000 亿/年
└── 必须找到财政来源

【政治抉择】
├── 选择 ①：增加一般税收（所得税、增值税等）
│   └── 缺点：影响所有纳税人 + 经济活力
├── 选择 ②：发行国债（赤字融资）
│   └── 缺点：违反稳定法 + 长期债务压力
├── 选择 ③：团结税（专项附加税）
│   └── 优点：专项使用 + 不影响主要税基 + 强调团结
└── 最终选择：团结税（强调"国家团结"的道德合法性）

【立法过程】
├── 1991-01-01：团结税法案通过
├── 1991-07：联邦宪法法院批准
├── 1991-08：开始征收
└── 实际生效：1991-08-01
```

### 2.2 初始税率的设计

```
【1991 年初始税率】

个人：
├── 所得税附加：7.5%
└── 资本利得税附加：7.5%

企业：
├── 公司所得税附加：7.5%
└── 法人税附加：7.5%

【税收设计原理】
├── 不超过边际所得税率的 50%（避免"反向激励"）
├── 全额抵扣所得税（避免双重征税）
└── 与所得税同步缴纳
```

## 三、团结税的收入与使用

### 3.1 累计收入（1991-2019）

```
【团结税的累计收入】

| 时期 | 年度收入 | 累计收入 | 占 GDP 比例 |
|------|---------|---------|-----------|
| 1991-1995 | DM 100-150 亿/年 | DM 500 亿 | ~0.3% |
| 1996-2000 | DM 130-180 亿/年 | DM 1300 亿 | ~0.5% |
| 2001-2005 | €100-130 亿/年 | €2000 亿 | ~0.5% |
| 2006-2010 | €120-150 亿/年 | €2700 亿 | ~0.5% |
| 2011-2015 | €130-160 亿/年 | €3500 亿 | ~0.5% |
| 2016-2019 | €180-200 亿/年 | €4300 亿 | ~0.5% |

总计：约 €1700 亿（按欧元计算）
DM 1000 亿（按 1991 年币值）

【收入规模】
├── 仅覆盖东德重建需求的 30-40%
├── 其余由联邦预算 + 借款承担
└── 是补充性而非唯一资金来源
```

### 3.2 收入的使用

```
【团结税的使用结构】

用途 ①：东德重建（Aufbau Ost）
├── 基础设施投资
├── 城市建设
├── 环境治理
├── 教育与科研
└── 占总收入 60-70%

用途 ②：失业救济 + 提前退休
├── 东德失业者救济金
├── 东德提前退休补贴
└── 占总收入 20-30%

用途 ③：联邦一般支出（后期）
├── 2010s 后逐步用于一般支出
├── 受宪法法院限制（专项税）
└── 占总收入 10-20%

用途 ④：团结相关其他项目
├── 2020 疫情后用于一般支出
└── 占总收入 5-10%
```

## 四、团结税的财政转移机制

### 4.1 联邦财政转移的"三大支柱"

```
【德国财政转移支付体系】

第一支柱：团结税（Soli）
├── 1991-2020 年
├── 累计 €1700 亿
└── 主要来自所得税附加

第二支柱：联邦补充拨款（Bundesergänzungszuweisungen）
├── 始于 1989 年（合并前）
├── 主要给财力较弱的州
├── 包括东德 5 个州
└── 年度规模 ~€100 亿

第三支柱：财政平衡机制（Länderfinanzausgleich）
├── 德国宪法第 107 条
├── 富裕州向贫穷州转移
├── 横向 + 纵向财政平衡
└── 年度规模 ~€200 亿

【东德获得的总转移】
├── 1991-2020 累计：~€2 万亿
├── 占 GDP 比例：每年 4-5%
├── 主要来源：团结税 + 联邦拨款 + 财政平衡
└── 体现"国家团结" + 财政平衡双重原则
```

### 4.2 转移支付的"层级结构"

```
【东德财政转移的流向】

中央政府（联邦）
├── 团结税收入（~€170 亿/年）
├── 一般预算拨款（~€300 亿/年）
├── 特别拨款（专项）
└── 累计每年 ~€500-700 亿

联邦州政府
├── 财政平衡（横向转移，~€100 亿/年）
├── 联邦补充拨款（~€100 亿/年）
└── 累计每年 ~€200-300 亿

地方政府（市政府）
├── 联邦州拨款
├── 自身税收（市政税）
└── 累计每年 ~€100-150 亿

【总转移规模】
├── 联邦 → 东德 5 州：~€700 亿/年
├── 州内转移：~€300 亿/年
├── 总计：~€1000 亿/年
└── 占东德 GDP ~30-40%
```

## 五、团结税的争议与调整

### 5.1 五大争议

#### ⚠️ 争议一：永久化的"专项税"

```
【专项税的合法性争议】

原始承诺：
├── "团结税是临时性的"
├── "用于东德重建"
├── "完成后取消"
└── 形成广泛政治共识

实际情况：
├── 1995 年首次"到期"评估 → 延期
├── 2000 年再评估 → 延期
├── 2010 年后 → 改为一般支出
├── 2019 年宪法法院裁决：专项税用于一般支出违宪
└── 2021 年部分取消，2025 年后完全取消

【争议核心】
├── "团结税"的名称 vs 实际用途
├── 仍征收 → 但已用于非东德重建
├── 形成"专项税一般化"的争议
└── 反映政治承诺 vs 实际执行的张力
```

#### ⚠️ 争议二：西德纳税人的负担

```
【西德纳税人的不满】

负担规模：
├── 30 年累计 €1700 亿
├── 年度 €150-200 亿（占 GDP 0.5%）
├── 几乎全部由西德纳税人承担
└── 持续 30 年的"统一成本"

争议焦点：
├── 1. 公平性：东德人享受高福利，西德人承担
├── 2. 长期性：30 年仍未"独立"
├── 3. 效率性：大量资金是否有效使用
├── 4. 替代方案：发国债 vs 加税
└── 5. 终止时点：何时结束

【政治表现】
├── 西德人对"东德持续依赖"的不满
├── "东德人是西德纳税人负担"的舆论
├── 政党选举中的争议议题
└── 推动团结税取消的政治动力
```

#### ⚠️ 争议三：东德的"福利依赖"

```
【东德"福利依赖"的形成】

数据：
├── 东德就业率长期低于西德 ~10%
├── 提前退休人口比例高于西德
├── 失业救济领取率高
├── 福利支出占东德 GDP 30%+
└── 形成"高福利-低就业"结构

争议焦点：
├── 1. 福利制度是否削弱了工作激励
├── 2. 提前退休是否过度慷慨
├── 3. 转移支付是否替代了内生增长
├── 4. 是否形成"东德依赖"
└── 5. 长期可持续性

【学界争议】
├── 主流观点：福利依赖确实存在
├── 少数观点：依赖程度被高估
└── 折中观点：合理保障 vs 过度福利的边界
```

#### ⚠️ 争议四：团结税的"道德合法性"

```
【"国家团结"的道德合法性】

支持方观点：
├── 1. 国家团结是宪法原则
├── 2. 东德人是德国公民
├── 3. 帮助他们是道德义务
├── 4. 30 年付出是合理代价
└── 5. 团结税是这种义务的体现

反对方观点：
├── 1. 30 年已"团结"足够
├── 2. 西德人不应承担无限责任
├── 3. 应转向"机会平等"而非"结果平等"
├── 4. 东德应"自立"
└── 5. 团结税应逐步取消

【政治演变】
├── 1990s：团结税是政治共识
├── 2000s：争议开始
├── 2010s：取消呼声高涨
├── 2020：正式部分取消
└── 2025+：完全取消
```

#### ⚠️ 争议五：使用效率的质疑

```
【资金使用效率的争议】

批评观点：
├── 1. 大量资金用于"形象工程"
├── 2. 部分投资回报率低
├── 3. 行政成本过高
├── 4. 缺乏长期规划
└── 5. 资源错配

支持观点：
├── 1. 30 年完成大规模基础设施重建
├── 2. 现代化水平接近西德
├── 3. 环境治理取得重大进展
├── 4. 教育与科研投入显著
└── 5. 整体效率合理

【学界评价】
├── 主流观点：基本成功，但有改进空间
├── 少数观点：30% 资金被浪费
└── 教训：大规模转移支付需要严格监管
```

## 六、团结税的取消过程（2019-2025）

### 6.1 取消的法律过程

```
【团结税取消的关键节点】

2019-11：联邦宪法法院裁决
├── 团结税自 1995 年起仍征收
├── 但已用于一般支出（非专项）
├── 违反宪法（专项税原则）
└── 要求重新设计或取消

2020：取消方案
├── 2021-01-01：90% 纳税人免征团结税
├── 高收入者仍需缴纳（5.5%）
└── 过渡安排

2025-01-01：完全取消
├── 所有纳税人免征团结税
├── 仍保留"团结"概念
└── 联邦预算通过其他方式承担东德重建
```

### 6.2 取消的财政影响

```
【团结税取消的财政影响】

2021 后收入减少：
├── 减少 ~€100 亿/年（90% 取消）
├── 2025 后减少 ~€200 亿/年（完全取消）
└── 占联邦预算 ~5%

弥补方式：
├── 1. 联邦借款增加（违反"债务刹车"）
├── 2. 其他税收调整
├── 3. 联邦支出削减
└── 4. 东德重建资金改由一般预算承担

【对东德的实际影响】
├── 名义上团结税取消
├── 但转移支付仍在继续（联邦预算）
├── 东德仍每年获得 ~€1000 亿转移
└── 仅资金来源从"团结税"转为"一般预算"
```

## 九、关键洞察总结

> [!key-insight] 团结税是"国家团结"的财政体现
>
> 团结税不仅是财政工具，更是德国"国家团结"的道德体现。它将统一成本分摊给全体德国人，巩固了统一共识。

> [!key-insight] "临时税"变"长期税"的教训
>
> 团结税从"临时性"承诺变成 30 年长期税，反映了"政治承诺 vs 实际执行"的张力。这给其他国家的临时税设计提供了重要教训。

> [!contradiction] 团结 vs 公平
>
> 团结税体现了"国家团结"原则，但也引发"代际公平"和"地区公平"的争议。这种张力是大规模转移支付的永恒难题。

> [!key-insight] "福利依赖"的形成
>
> 30 年的团结税 + 转移支付确实推动了东德现代化，但也形成了"福利依赖"。这是大规模转移支付的常见副作用。

> [!key-insight] 民主合法性的关键作用
>
