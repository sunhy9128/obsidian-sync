# 利率平价理论

> 在无套利条件下，两国利率之差应等于远期汇率与即期汇率之差（抛补版），或等于预期本币贬值率（无抛补版）。

## 一、核心定义

### 1.1 概念本质

**利率平价**（Interest Rate Parity, IRP）描述利率与汇率之间的均衡关系，是国际套利的**无套利条件**。

| 维度 | 抛补利率平价 (CIP) | 无抛补利率平价 (UIP) |
|------|--------------------|--------------------|
| 是否对冲汇率风险 | 是（用远期合约锁定） | 否（暴露于汇率风险） |
| 涉及合约 | 即期 + 远期 | 即期 + 预期 |
| 实证支持 | **很强**（几乎无偏差） | **较弱**（存在溢价/折价） |

利率平价理论（IRP）是国际金融学的基础理论，描述了两国利率差异与即期/远期汇率之间的关系。高利率货币的远期汇率倾向于贴水（贬值），低利率货币的远期汇率倾向于升水（升值）。

### 1.2 基本思想

> 资金应流向回报率最高的地方；但**汇率预期**会抵消利差，否则存在无风险套利。

## 二、抛补利率平价（CIP）

### 2.1 公式推导

**情境**：美国利率 1 年期 r_US = 5%，欧元区 r_EZ = 3%，即期汇率 S（欧元/美元）。

| 路径 | 操作 | 1 年后美元回报 |
|------|------|--------------|
| **路径 A（美国）** | 借入 1 美元 → 存美国 → 1 美元 × (1 + 5%) = 1.05 美元 | 1.05 |
| **路径 B（欧元区+对冲）** | 1 美元换欧元 → 存欧元区 → 换回 + 锁远期 | 1 × S × (1+3%) / F = 1.05 |

无套利要求：

```
(1 + r_US) = (S / F) × (1 + r_EZ)
```

**整理得 CIP 公式**：

```
F = S × (1 + r_US) / (1 + r_EZ)
```

或近似：

```
(F - S) / S ≈ r_US - r_EZ
```

> 利率更高的国家货币，**远期贴水**（forward discount）；利率低的国家货币**远期升水**（forward premium）。

### 2.2 计算示例

假设：
- S = 1.10 美元/欧元
- r_US = 5%，r_EZ = 3%

```
F = S × (1 + r_US) / (1 + r_EZ) = 1.10 × 1.05 / 1.03 ≈ 1.1214 美元/欧元
```

> 美元利率更高 → 远期美元/欧元汇率上升（美元远期**升水**）。

### 2.3 CIP 的实证地位

CIP 在金融市场上**几乎严格成立**，原因是：
- 套利**无风险**（远期锁定）
- 银行等机构可低成本套利
- 偏离稍大即被对冲基金捕捉

> 2008 金融危机后，CIP 出现**系统性偏离**（外汇掉期基差），详见第五节。

## 三、无抛补利率平价（UIP）

### 3.1 公式推导

**情境**：投资者**不锁定**远期汇率，仅凭预期持有外币资产。

| 路径 | 操作 | 1 年后美元回报（预期） |
|------|------|----------------------|
| 美国 | 1 美元存美国 | 1.05 美元 |
| 欧元区 | 1 美元换欧元 → 存欧元区 → 1 年后换回（按预期 S_e） | (1 / S) × 1.03 × S_e = 1.03 × (S_e / S) |

无套利要求（**预期**回报相等）：

```
1 + r_US = (1 + r_EZ) × (S_e / S)
```

**整理得 UIP 公式**：

```
(S_e - S) / S ≈ r_US - r_EZ
```

> 利率更高的国家货币，预期**贬值**（贬值率 ≈ 利差）。

### 3.2 与 CIP 的关系

| 维度 | CIP | UIP |
|------|-----|-----|
| 关键变量 | 远期汇率 F | 预期即期汇率 S_e |
| 关系 | F 由套利决定 | S_e 由预期决定 |
| 实证偏离 | 极小 | 显著 |
| 偏离含义 | 几乎不需要检验 | 存在**汇率风险溢价** |

> **外汇风险溢价**（Foreign Exchange Risk Premium）：
> F ≠ S_e 时，CIP 仍成立，但 UIP 偏离：
> ```
> S_e = F + 风险溢价
> 预期贬值率 = 利差 + 风险溢价
> ```

### 3.3 UIP 的实证"谜团"

**Forward Bias Puzzle**（远期偏差之谜）：

实证发现：
- 高息货币**倾向于升值**而非贬值（与 UIP 相反）
- 远期汇率是**有偏**的预期器
- 1980s 后系统化套利"溢价"

**可能解释**：
- 套息交易（Carry Trade）盈利
- 风险厌恶 + 期限溢价
- 央行干预

## 四、套利机制

### 4.1 抛补套利（Covered Arbitrage）

```
若 F 偏离 CIP 隐含值：
  - F 太高：卖远期（做空远期美元） + 借入美元 + 投资欧元区
  - F 太低：买远期（做多远期美元） + 借入欧元 + 投资美国
```

银行间市场套利会迅速推动 F 回归 CIP 水平。

### 4.2 无抛补套利（投机资本流动）

```
若 E[S_e] 偏离 UIP 隐含值：
  - 预期本币贬值 < 利差：资金流入高息国（套息）
  - 预期本币贬值 > 利差：资金流出
```

资金流动影响即期汇率 S，进而通过预期反馈。

### 4.3 套息交易（Carry Trade）

借**低息货币**、买**高息货币**，赚利差。实证长期盈利说明 UIP 持续偏离。

**经典案例**：
- 2000s：借日元（0.5%），买澳元/纽元（6-7%）
- 2022-2024：借日元（-0.1%→0.5%），买美元（5%+）
- 2022 年美元/欧元：美联储加息使美欧利差扩大至约 2 个百分点，按 IRP 美元应对欧元贬值，但套利交易与资金流向使汇率在短期内偏离理论值

```
总回报 = 利差 + 汇率变动
     = (r_high - r_low) + (S_end - S_start) / S_start
```

### 4.4 偏离原因（综合）

IRP 在现实中偏离的常见原因包括：
- **资本管制**——跨境资金流动受限
- **交易成本**——买卖价差、手续费
- **政治风险**——汇率管制、没收风险
- **流动性差异**——不同货币市场深度不同
- **银行融资摩擦与监管约束**（2008 后 CIP 基差的主导解释，详见第五节）

## 五、CIP 的现代偏离（外汇掉期基差）

### 5.1 现象

2008 金融危机后，CIP 出现**系统性偏离**：

```
CIP 基差 = (F_actual / F_implied_CIP) - 1
```

| 时期 | 基差状态 | 典型幅度 |
|------|---------|---------|
| 2008 危机前 | 接近 0 | < 5 bp |
| 2008-2010 | 显著偏离 | 50-150 bp |
| 2014-2015 欧央行负利率 | 欧元 vs CHF 偏离 | 30-80 bp |
| 2020-2022 美元荒 | 美元基差走高 | 50-100 bp |
| 2022-至今 紧缩 | 美元基差扩大 | 30-100 bp |

### 5.2 原因

| 原因 | 机制 |
|------|------|
| **银行融资摩擦** | 大行跨境融资的信用/监管成本 |
| **央行政策** | 负利率、QE 改变利率市场结构 |
| **资本/流动性监管** | 巴塞尔 III SLR/NSFR 限制套利 |
| **美元特殊地位** | 全球美元融资市场失衡 |

### 5.3 启示

CIP 偏离说明**无套利定理的现实局限**——摩擦、监管、资本结构使套利**有边界**。

## 六、利率平价、汇率与货币政策

### 6.1 央行政策传导

```
央行加息 → 利差扩大 → 即期本币升值
                       → 远期本币贴水（CIP）
                       → 吸引套息资本流入
                       → 强化升值
```

### 6.2 三难选择（不可能三角）

> 独立的货币政策 + 固定汇率 + 自由资本流动 → **三者不可兼得**

| 政策选择 | 资本流动 | 汇率 |
|---------|---------|------|
| 固定汇率 + 独立货币 | 必须管制 | 固定 |
| 浮动汇率 + 独立货币 | 自由 | 浮动 |
| 固定汇率 + 自由流动 | 自由 | 固定（放弃独立） |

### 6.3 中国情境

中国选**独立货币 + 固定汇率（爬行）** → 资本管制是必然。

## 八、关键易错点

| 误区 | 正确认识 |
|------|----------|
| 利率高 = 货币一定升值 | 利差会被远期贴水抵消（CIP）；可能反而贬值（UIP） |
| 抛补与无抛补可互推 | 实证上 UIP 偏离是常态 |
| 远期汇率 = 未来即期汇率 | 仅在**风险中性**下成立，现实中含风险溢价 |
| 套息无风险 | 汇率反向波动可吞噬多年利差（2022 年日元套息爆仓） |
| CIP 永远成立 | 2008 后存在系统性基差 |
