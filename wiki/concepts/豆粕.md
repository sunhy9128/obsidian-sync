# 豆粕

豆粕是大豆经压榨提油后的**副产品**，是全球最大的**植物蛋白饲料原料**。在中国生猪养殖业中，豆粕占猪饲料配方的15-20%，是连接"国际大豆价格"与"国内猪肉价格"的关键传导节点。

> **核心命题**：豆粕是大豆价格向CPI传导的"第二通道"——大豆不仅通过豆油直接影响CPI食用油项，更通过豆粕把肉类养殖成本嵌入CPI食品项。

## 一、生产工艺

### 1. 大豆压榨的两种工艺

| 工艺 | 副产物 | 占比 | 特点 |
|------|--------|------|------|
| **浸出法**（主流） | 豆粕（脱脂）+ 豆油 | 80%出粕率 | 效率高、出油率高 |
| 压榨法（物理） | 豆饼（含油）+ 豆油 | 约85%出饼率 | 传统工艺、油色好 |

中国大豆压榨几乎全部采用**浸出法**，副产品为豆粕。

### 2. 一吨大豆的产出

```
1吨大豆（16-18%含油率）
  ↓ 浸出法压榨
  ├─ 豆油：约170kg（出油率17%）
  └─ 豆粕：约790kg（出粕率79%）
       ├─ 蛋白含量：43-48%
       └─ 主要用途：饲料原料
```

**关键**：豆粕是大豆压榨的**主产品**（按重量），不是副产品。

## 二、营养与饲料地位

### 营养成分

| 成分 | 含量 |
|------|------|
| 粗蛋白 | 43-48%（高蛋白） |
| 赖氨酸 | 2.5-3.0%（猪的第一限制性氨基酸） |
| 蛋氨酸 | 0.6-0.7% |
| 粗纤维 | 5-7% |
| 代谢能 | 2.4-2.6 Mcal/kg（猪） |

### 在饲料中的地位

豆粕是**全球第一大蛋白饲料原料**，地位无可替代：

| 饲料蛋白源 | 全球产量占比 | 中国使用占比 |
|------------|--------------|--------------|
| **豆粕** | **70%+** | **约70%** |
| 菜粕（油菜籽） | 10% | 10% |
| 棉粕（棉籽） | 5% | 5% |
| 鱼粉 | 5% | 5% |
| DDGS（玉米酒精糟） | 5% | 5% |
| 其他 | 5% | 5% |

> **核心事实**：豆粕是养殖业的"蛋白之王"——主要因为其**氨基酸结构最接近动物需求**，无需额外补充即可满足猪、鸡、鱼的基本生长需求。

## 三、中国豆粕的"进口依赖悖论"

### 一个看似矛盾的事实

中国本土大豆产量约1600万吨（2023），但**豆粕消费量约7500万吨**——差距悬殊。

```
中国大豆供给结构（2023）
  本土种植：1600万吨（20%）  → 主要用于直接食用（豆腐、豆浆）
  进口大豆：约9000万吨（80%）  → 几乎全部用于压榨
```

**为什么本土大豆不用于压榨？**
- 中国本土大豆**含油率仅16-17%**（美国转基因大豆18-19%）
- 蛋白质含量高（更适合直接食用）
- 进口大豆（转基因）含油率高、出粕率高，更适合工业化压榨

**关键结论**：
- **中国豆粕几乎全部来自进口大豆压榨**
- 中国进口大豆的80%用于压榨，产出豆粕
- 进口大豆价格变动 → 豆粕价格变动 → 饲料成本变动 → 养殖成本变动

这就是为什么"中美大豆博弈"会直接传导到中国猪肉价格——见 [[大豆战争]]。

## 四、价格传导机制

### 完整链条

```
国际大豆价格（CBOT）
   ↓ 升贴水（CIF到岸价）
中国进口大豆到岸价
   ↓ 压榨利润
豆油出厂价 + 豆粕出厂价（"两粕一油"分摊成本）
   ↓
豆粕现货价（饲料厂采购）
   ↓
猪/鸡/鱼配合饲料价
   ↓
养殖成本（饲料占60-70%）
   ↓
生猪/家禽/水产价格
   ↓
CPI食品项（猪肉占2.5-3%，加上鸡鸭水产更高）
```

### 价格弹性

| 环节 | 价格弹性 | 传导速度 |
|------|----------|----------|
| CBOT大豆 → 豆粕 | 接近1：1（豆粕是大豆的主产品） | 数日 |
| 豆粕 → 饲料 | 0.7-0.8（豆粕占饲料15-20%） | 数周 |
| 饲料 → 养殖成本 | 0.6-0.7（饲料占养殖成本60-70%） | 1-2月 |
| 养殖成本 → 肉类价格 | >1（供应收缩时） | 3-6月（蛛网滞后） |

详见 [[大豆战争]]、[[猪周期]]、[[蛛网模型]]。

## 五、豆粕期货

### 大商所豆粕期货

| 维度 | 详情 |
|------|------|
| 上市时间 | 2000年7月 |
| 交易所 | 大连商品交易所（见 [[大商所]]） |
| 合约单位 | 10吨/手 |
| 报价单位 | 元/吨 |
| 主要用途 | 饲料企业套保、压榨企业套保 |

### 产业应用

豆粕期货是中国农产品期货中**成交最活跃的品种之一**：
- 饲料企业：锁定饲料成本（采购套保）
- 压榨企业：锁定加工利润（压榨套利）
- 养殖企业：锁定养殖成本
- 投机者：博弈中美关系、汇率、季节性供需

## 六、中国豆粕消费量与进口

### 消费量（2023年）
- 中国豆粕消费量：**约7500万吨**
- 占全球豆粕消费量的**30%+**
- 中国是全球第一大豆粕消费国

### 进口量
- 中国豆粕几乎不进口（2023年仅几万吨）
- **豆粕进口实质上是大豆进口的替代品**——中国选择进口大豆自己压榨，而非进口成品豆粕
- 这保留了国内的压榨产业链（虽然大部分已被外资控制，见 [[五神]]）

## 七、对CPI的传导（与[[猪周期]]的连接）

### 直接传导
- 豆粕 → 猪饲料 → 猪肉 → CPI食品项（权重2.5-3%）

### 间接传导
- 豆粕 → 鸡饲料 → 鸡肉 → CPI食品项
- 豆粕 → 水产饲料 → 鱼虾 → CPI食品项
- 豆粕 → 牛羊饲料 → 牛羊肉 → CPI食品项

> **累计效应**：豆粕价格变动通过整个肉类/水产板块，对CPI食品项的拉动可达 **5-10%**。

这就是 [[2025-02-04-中国粮食金融保卫战-巫师财经]] 中"大豆是奇点商品"的微观机制——大豆→豆粕→整个CPI肉类板块。

## 八、替代品与"去豆粕"问题

### 替代蛋白源

| 替代品 | 优点 | 缺点 | 替代上限 |
|--------|------|------|----------|
| 菜粕（菜籽粕） | 中国本土可产 | 含毒素（芥酸、硫苷） | 5-10% |
| 棉粕（棉籽粕） | 中国本土可产 | 含棉酚（毒） | 3-5% |
| DDGS（玉米酒精糟） | 中国玉米产量大 | 蛋白低（27%）、含霉菌毒素 | 5-10% |
| 玉米蛋白粉 | 中国可产 | 氨基酸结构不完整 | 3-5% |
| 昆虫蛋白（黑水虻） | 新型蛋白源 | 产量有限 | <1% |
| 合成氨基酸（赖氨酸/蛋氨酸） | 精准补充 | 无法完全替代 | 辅助 |

### 中国"低蛋白日粮"政策

中国农业农村部2018年发布《仔猪、生长育肥猪配合饲料》国家标准，推动**低蛋白日粮**：

```
传统配方：豆粕 18% → 蛋白 16%
低蛋白配方：豆粕 13% → 蛋白 14%（+ 合成氨基酸）
```

**政策效果**：
- 减少豆粕用量约5-8%
- 降低大豆进口依赖
- 减少氮排放（环保）
- 但**无法完全替代豆粕**——氨基酸平衡、适口性、动物健康都需要豆粕基础

## 九、核心命题

> 1. **豆粕是大豆向CPI传导的"第二通道"**——大豆不只通过豆油，更通过豆粕-饲料-肉类嵌入CPI
> 2. **中国豆粕几乎全部依赖进口大豆**——本土大豆用于直接食用而非压榨
> 3. **豆粕是"蛋白之王"**——氨基酸结构最接近动物需求，替代品都只能部分补充
> 4. **CBOT大豆价格 → 豆粕价格的传导接近1：1**——主产品价格高度同步
> 5. **饲料成本是猪周期的二次加速器**——豆粕涨价加速蛛网进入高价阶段
