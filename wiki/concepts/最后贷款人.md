# 最后贷款人（Lender of Last Resort）

> 央行最古老、最神秘、最强大的职能——在金融体系崩溃时充当最后的"守护者"

## 一、核心定义

**最后贷款人**（Lender of Last Resort，LOLR）是**中央银行的核心职能之一**，指央行在**金融市场流动性枯竭**、**银行体系面临系统性挤兑**、**常规融资渠道失灵**时，**紧急提供流动性支持**，防止金融机构倒闭引发系统性金融危机。

**一句话定义**：**最后贷款人 = 央行在危机时紧急提供流动性，防止系统性崩溃**。

**核心要素**：
- **最后**：只有当市场机制失灵时才介入
- **贷款人**：以贷款形式提供流动性（不是赠款）
- **央行角色**：中央银行是天然的"最后贷款人"

**历史奠基人**：
- **沃尔特·白芝浩**（Walter Bagehot，1826-1877）：1873 年《伦巴第街》提出经典原则
- **亨利·桑顿**（Henry Thornton，1760-1815）：早期理论贡献者
- **沃尔克**（Paul Volcker）、**伯南克**（Ben Bernanke）：现代实践者

## 二、白芝浩原则（Bagehot's Dictum）

### 2.1 经典原则

```
沃尔特·白芝浩 1873 年《伦巴第街》：

白芝浩原则（Bagehot's Dictum）的核心：

1. **自由放贷**
   - "自由地、大胆地放贷"
   - 央行必须准备无限制提供流动性

2. **惩罚性利率**
   - 以"高于市场的惩罚性利率"放贷
   - 防止道德风险
   - 让冒险的银行付出代价

3. **良好抵押品**
   - 只接受"良好抵押品"
   - 传统：国债
   - 现代：扩展到投资级公司债

4. **公开承诺**
   - 提前公开承诺"任何有偿付能力的银行"
   - 都能获得流动性支持
   - 防止恐慌蔓延

核心思想：
  - 央行的支持应该是确定的
  - 但不是廉价的
  - 危机时必须果断行动
```

### 2.2 白芝浩原则的现代演化

```
1873 年（白芝浩）：
  - 只救"有偿付能力的银行"（solvent but illiquid）
  - 抵押品限于国债
  - 惩罚性利率
       ↓
1907 年（Knickerbocker Trust 危机）：
  - J.P. Morgan 私人救助
  - 政府事后支持
  - 美联储 1913 年成立
       ↓
1970s-1980s（沃尔克时代）：
  - 严格遵循白芝浩原则
  - 救助大陆伊利诺伊银行（1984）
  - "太大而不能倒"开始争议
       ↓
1998 年（LTCM）：
  - 美联储协调 14 家私人银行救助
  - 私人和官方混合救助
  - 道德风险争议加剧
       ↓
2008 年（全球金融危机）：
  - 大规模突破白芝浩原则
  - 抵押品扩展到 ABS、MBS
  - 利率远低于惩罚性
  - 直接救助非银行（AIG、两房）
       ↓
2020 年（COVID）：
  - 无限 QE + 零利率
  - 全面突破经典原则
  - 央行实际"兜底"整个市场
```

## 三、最后贷款人的理论演进

### 3.1 三大理论流派

```
1. **白芝浩传统（经典）**
   - 救助有偿付能力但缺乏流动性的机构
   - 惩罚性利率
   - 良好抵押品
   - 防止道德风险

2. **弗里德曼学派（货币主义）**
   - 央行应保持货币供应稳定
   - 危机时大幅扩张货币供应
   - 1930s 大萧条：美联储"什么都没做"导致灾难

3. **新派（Bernanke、Mishkin）**
   - 央行应积极干预
   - 抵押品可以灵活
   - 利率不必惩罚性
   - 但必须有监管配合

现代主流：
  - 综合三大流派
  - 央行积极干预 + 监管强化 + 事后问责
```

### 3.2 最后贷款人的经济学基础

```
最后贷款人的理论逻辑：

1. **市场失灵论**
   - 金融恐慌是市场失灵
   - 信息不对称导致挤兑
   - 央行作为最后贷款人可纠正失灵

2. **公共物品论**
   - 金融稳定是公共物品
   - 单一机构无法提供
   - 必须由央行提供

3. **铸币税论**
   - 央行独有"印钞"权力
   - 可以提供无限流动性
   - 但印钞权不可滥用

4. **道德风险论（反对意见）**
   - 最后贷款人鼓励冒险
   - "大而不倒"激励银行冒险
   - 需要配套监管

核心矛盾：
  - 央行必须救助（防止系统性风险）
  - 但救助会鼓励冒险（道德风险）
  - 平衡是关键
```

## 四、最后贷款人的运作机制

### 4.1 经典运作流程

```
最后贷款人的运作流程：

Step 1：识别危机
  - 金融市场出现异常
  - 关键机构流动性枯竭
  - 系统性风险显现
  - 央行判断"必须介入"

Step 2：宣布支持
  - 公开承诺"市场有充足流动性"
  - 表明央行准备介入
  - 通常配合降息

Step 3：提供流动性
  - 通过常规工具：贴现窗口、再贷款
  - 通过紧急工具：紧急贷款、流动性拍卖
  - 直接干预：购买资产、QE

Step 4：维持稳定
  - 持续监测
  - 必要时追加措施
  - 直至市场恢复正常

Step 5：退出与问责
  - 逐步退出非常规工具
  - 事后问责与监管改革
```

### 4.2 央行的最后贷款人工具

```
央行的最后贷款人工具谱系：

| 工具 | 对象 | 时机 | 利率 |
|------|------|------|------|
| **贴现窗口** | 存款银行 | 持续 | 政策利率 + 50bp |
| **TAF** | 存款银行 | 紧急 | 拍卖利率 |
| **PDCF** | 一级交易商 | 紧急 | 贴现利率 |
| **紧急贷款** | 个别机构 | 个案 | 个案定价 |
| **QE** | 整个市场 | 危机 | 市场价格 |
| **回购市场** | 交易商 | 持续/紧急 | 拍卖利率 |
| **货币互换** | 外国央行 | 危机 | 政策利率 |
| **AMLF** | 货币市场基金 | 紧急 | 政策利率 |
| **CPFF** | 商业票据 | 紧急 | 政策利率 |
       ↓
工具演进：
  1873：贴现窗口
  1907：私人救助
  1930s：黄金/外汇干预
  1970s：货币互换
  1987：流动性紧急投放
  1998：私人协调救助
  2008：扩展到非银行
  2020：无限 QE
```

### 4.3 抵押品的演进

```
抵押品的演进：

1873 年：国债
  - 白芝浩原则
  - 只接受国债
  - 简单、清晰

1990s：扩展到高质量债券
  - 机构 MBS
  - 投资级公司债

2008 年：大幅扩展
  - ABS
  - CDO
  - 甚至垃圾债（部分）
  - 抵押品的"灵活化"

2020 年：几乎无限
  - 公司债 ETF
  - 一级和二级公司债
  - 包括部分"堕落天使"
       ↓
批评：
  - 抵押品灵活化 = 道德风险
  - 接受风险资产 = 央行承担信用风险
  - 可能造成"僵尸"机构
```

## 五、最后贷款人的历史案例

### 5.1 1873-1907：早期时代

```
1873 年（白芝浩出版《伦巴第街》）：
  - 标志最后贷款人理论的诞生
  - 但当时英格兰银行并未充分遵循

1890 年（巴林银行危机）：
  - 英国巴林银行因阿根廷投资失败
  - 英格兰银行协调私人救助
  - 第一次系统性救助

1907 年（Knickerbocker Trust 危机）：
  - 纽约 Knickerbocker Trust 银行挤兑
  - 私人银行家 J.P. Morgan 协调救助
  - 财政部紧急存款
  - 直接促成美联储 1913 年成立
       ↓
教训：
  - 最后贷款人是必要的
  - 但需要常设机构（央行）
  - 1907 危机催生美联储
```

### 5.2 1914-1929：美联储早期

```
1913 年美联储成立：
  - 部分原因就是为了 1907 危机
  - 设计为最后贷款人
  
1914 年（第一次世界大战）：
  - 美国金融市场冻结
  - 美联储紧急救援
  - 关闭股市 4 个月

1929-1933（大萧条）：
  - 美联储"几乎什么都没做"
  - 错误地坚持"自由市场"
  - 错误地相信"清算"是必要的
  - 弗里德曼：美联储是"罪魁祸首"
  - 9000+ 家银行倒闭
  - 货币供应下降 30%
       ↓
教训（Friedman-Schwartz）：
  - 央行必须积极干预
  - 央行"无为"是灾难性的
  - 这次教训深刻影响伯南克
```

### 5.3 1933-2007：黄金时代与改革

```
1933 年（罗斯福新政）：
  - 银行业改革
  - 联邦存款保险（FDIC）成立
  - 部分替代了最后贷款人职能
  - "存款保险 + 最后贷款人"的双重防护

1934-1980s（黄金时代）：
  - 布雷顿森林体系
  - 金融体系稳定
  - 最后贷款人较少动用
  - 但每次动用都成功

1984 年（大陆伊利诺伊银行 Continental Illinois）：
  - 美国第七大银行
  - 储户挤兑
  - 美联储 + FDIC 救助
  - "太大而不能倒"首次引发争议
  - 实际 90% 储户全额偿付
  - 股东和债权人承担损失
```

### 5.4 1990s：新兴市场危机与 LTCM

```
1992 年（ERM 危机）：
  - 索罗斯做空英镑
  - 英格兰银行干预失败
  - 不是传统最后贷款人案例
  - 但展示了央行"有限保护"

1994-1995 年（墨西哥龙舌兰危机）：
  - 比索贬值
  - 美国 + IMF 联合救助
  - 国际最后贷款人案例

1997-1998 年（亚洲金融危机）：
  - 泰国、韩国、印尼危机
  - IMF 救援（不是直接最后贷款人）
  - 美国财政部通过 ESF 提供支持

1998 年（LTCM）：
  - 长期资本管理公司濒临破产
  - 美联储协调 14 家私人银行救助
  - 关键转折：
    - 央行组织"私人救助"
    - 避免道德风险（纳税人不出钱）
    - 但为"央行干预非银行"开先例
```

### 5.5 2008 年：突破性的危机

```
2008 年（全球金融危机）：

美联储的"非常规"行动：

1. 传统工具（存款银行）
   - 贴现窗口（TAF，2007.12）
   - 期限拍卖

2. 扩展到投资银行（关键突破）
   - PDCF（2008.3）—— 一级交易商信贷
   - 这是 1913 年以来最大的央行职能扩展
   - 投资银行首次获得最后贷款人保护

3. 扩展到非银行金融机构
   - TSLF（2008.3）—— 证券借贷
   - AMLF（2008.9）—— 货币市场基金
   - CPFF（2008.10）—— 商业票据
   - 美联储首次直接介入非银行

4. 直接救助个别机构
   - 贝尔斯登（2008.3）—— 摩根大通收购
   - AIG（2008.9）—— 1823 亿美元
   - 两房（2008.9）—— 接管
   - 摩根大通（2008.9）—— 贝尔斯登交易支持

5. 大规模 QE
   - QE1（2008.11）—— 1.7 万亿
   - QE2（2010.11）—— 6000 亿
   - QE3（2012.9）—— 无限期 MBS 购买

关键突破：
  1. 最后贷款人从存款银行扩展到投资银行
  2. 从金融机构扩展到非银行金融市场
  3. 从"惩罚性利率"变为"市场利率"
  4. 抵押品从国债扩展到 ABS、CDO
  5. 美联储从"被动"变为"主动"
```

### 5.6 2010s：常规化与争议

```
2010-2014 后危机：
  - QE 持续
  - 但开始争论退出
  - 2013 "taper tantrum"
  - 投资者担心流动性退出

2014-2018 货币政策正常化：
  - 渐进加息
  - 2017.10 开始缩表
  - 流动性逐步退出

2019 年 9 月（回购市场危机）：
  - 缩表过快
  - 准备金稀缺
  - 回购利率飙升至 10%+
  - 美联储紧急投放
  - 2019.10 开始"有机扩表"
  - 教训：QT 必须谨慎

2020 年 3 月（COVID）：
  - 无限 QE
  - 工具扩展到公司债 ETF
  - 跨资产类别
  - 实际"兜底"整个市场
```

### 5.7 2023 年 SVB 危机：新挑战

```
2023 年 3 月（硅谷银行 SVB）：

事件：
  - SVB 因利率风险倒闭
  - 48 小时内被 FDIC 接管
  - 引发区域性银行挤兑
  - 标志：利率风险成为新挑战

美联储应对：
  - BTFP（Bank Term Funding Program，2023.3）
  - 允许银行用国债作抵押借款
  - 避免按市值计价损失
  - 实质上是"最后贷款人"的新工具

反思：
  - 巴塞尔 III 主要应对信用风险
  - 但利率风险被忽视
  - 最后贷款人也需要新工具
  - SVB 危机是巴塞尔 III 的压力测试
```

## 六、最后贷款人 vs 存款保险

### 6.1 双层防护机制

```
金融安全网的"双层防护"：

第一层：存款保险
  - 覆盖零售存款人
  - 防止挤兑
  - 美国 FDIC（25 万美元）
  - 中国存款保险（50 万元人民币）

第二层：最后贷款人
  - 覆盖金融机构
  - 提供紧急流动性
  - 美联储、欧洲央行
  - 中国人民银行

关系：
  - 存款保险处理"个别银行倒闭"
  - 最后贷款人处理"系统性风险"
  - 互补而非替代
  - 共同维护金融稳定
```

### 6.2 关键对比

| 维度 | 存款保险 | 最后贷款人 |
|------|---------|----------|
| **覆盖** | 存款人 | 金融机构 |
| **触发** | 银行倒闭 | 系统性危机 |
| **提供方** | 存款保险机构 | 中央银行 |
| **工具** | 偿付存款 | 紧急贷款、QE |
| **资金来源** | 保险基金 | 央行印钞权 |
| **道德风险** | 中等 | 高 |

## 七、最后贷款人 vs 道德风险

### 7.1 道德风险的核心问题

```
最后贷款人与道德风险的核心矛盾：

问题：
  央行救助 → 银行预期"被救" → 银行冒险 → 更多危机 → 更多救助
       ↓
这就是"道德风险循环"：
  1. 央行承诺保护
  2. 银行冒险增加
  3. 危机发生
  4. 央行救助
  5. 预期"下次还救"
  6. 冒险更多
       ↓
经典表述：
  "The expectation of being saved is the cause of the next crisis"
  "被救助的预期就是下次危机的原因"
```

### 7.2 白芝浩原则的反道德风险设计

```
白芝浩原则的反道德风险机制：

1. **惩罚性利率**
   - 借钱贵 = 不鼓励
   - 危机时也必须支付高息
   - 央行不是"免费午餐"

2. **良好抵押品**
   - 不接受烂资产
   - 银行必须"自证清白"
   - "有偿付能力的银行"才救助

3. **不救助股东**
   - 救助是贷款，不是补贴
   - 股东、债权人承担损失
   - 银行可以倒闭，只是有序倒闭

4. **事后问责**
   - 高管可能被追责
   - 监管可能改变
   - 防止"白救助"
```

### 7.3 现代道德风险的新维度

```
现代道德风险的复杂性：

1. **"太大而不能倒"（TBTF）**
   - G-SIBs（30+ 家）被认定
   - 实际"必须"救助
   - 银行越大，越安全
   - 反向激励

2. **影子银行**
   - 货币市场基金、对冲基金
   - 不受传统监管
   - 但 2008、2020 都获得支持
   - "隐性的最后贷款人"

3. **跨境溢出**
   - 银行跨国经营
   - 一国危机影响他国
   - 跨境协调困难
   - "谁应该是最后贷款人"

4. **央行越界**
   - 央行从最后贷款人变成"市场参与者"
   - 持有大量资产
   - 影响资产价格
   - "双重角色"的挑战

5. **道德风险的非对称性**
   - 危机时必须救助（系统性风险）
   - 但平时难以严格约束
   - 这种不对称是金融体系的核心难题
```

## 十、最后贷款人的五个核心原则（现代版）

```
基于历史经验，现代最后贷款人应遵循：

1. **确定性原则**
   - 央行应明确表态"在必要时介入"
   - 这种表态本身就是稳定器
   - 2008 前的不确定性是问题

2. **及时性原则**
   - 必须及时介入
   - 太晚介入会放大危机
   - 1929、2008 都是及时性问题

3. **足够规模原则**
   - 必须"无限"提供流动性
   - 规模不足 = 不可信
   - 2008 后美联储坚持"无限 QE"

4. **灵活性原则**
   - 抵押品可灵活
   - 利率不必惩罚性
   - 必须根据情况调整
   - 白芝浩原则可调整

5. **配套监管原则**
   - 救助必须有监管配合
   - 必须有事后问责
   - 必须有系统性风险防控
   - 沃尔克规则、TLAC 都是配套
```

最后贷款人 vs 财政货币化的关键区别：

  - 财政货币化 → 直接购债 → 货币化程度高
  - 中间地带：QE（购买国债，但央行自主决策）
       ↓
争议：

  - 2020 无限 QE 是救助市场还是救助财政？

## 十二、最后贷款人的核心命题

### 12.1 最后贷款人的五大成就

```
1. **防止了大萧条重演**
   - 1929-1933：美联储什么都没做
   - 2008 后：美联储积极干预
   - 2008 危机严重但远未崩溃

2. **建立了金融安全网**
   - 央行 + 存款保险 + 监管
   - 三层防护
   - 维护金融稳定

3. **扩展了央行职能**
   - 从"最后贷款人"到"市场参与者"
   - 从"被动"到"主动"
   - 央行影响力大幅扩大

4. **促进了金融创新**
   - 央行兜底 → 市场愿意承担风险
   - 衍生品、影子银行兴起
   - 金融体系效率提升

5. **应对了多次危机**
   - 1987 黑色星期一
   - 1998 LTCM
   - 2008 全球危机
   - 2020 COVID
   - 2023 SVB
```

### 12.2 最后贷款人的五大局限

```
1. **道德风险**
   - 央行救助 = 银行冒险增加
   - "太大而不能倒"
   - 反向激励

2. **越界风险**
   - 从"最后"变为"第一"
   - 央行深度参与市场
   - 政治化风险

3. **僵尸机构**
   - 应该倒闭的银行被救
   - 资源错配
   - 长期效率损失

4. **财政赤字货币化**
   - 最后贷款人可能滑向货币化
   - 央行独立性受损
   - 通胀风险

5. **资产价格泡沫**
   - QE 推高资产价格
   - 加剧不平等
   - 2008、2020 后均出现泡沫
```

### 12.3 最后贷款人的核心矛盾

```
最后贷款人面临的根本矛盾：

1. **稳定 vs 效率**
   - 稳定 → 央行介入 → 风险偏好下降 → 效率下降
   - 不介入 → 风险偏好高 → 创新 → 但可能危机

2. **道德风险 vs 系统性风险**
   - 严格限制救助 → 防止道德风险 → 但危机时"过晚"
   - 自由救助 → 防止系统性风险 → 但鼓励冒险

3. **央行独立性 vs 灵活性**
   - 独立性 → 可拒绝政府压力 → 但可能僵化
   - 灵活性 → 可快速响应 → 但可能被政治化

4. **国内 vs 国际**
   - 国内最后贷款人 → 清晰责任 → 但跨境溢出
   - 国际最后贷款人 → 协调 → 但责任不清（IMF 部分承担）

5. **传统 vs 创新**
   - 白芝浩原则 → 严格 → 适合传统银行
   - 现代央行 → 灵活 → 但"灵活"可能被滥用
```

## 十五、一句话总结

> **最后贷款人 = 央行在危机时紧急提供流动性的核心职能，是金融体系的"最后防线"**。
>
> 它有**白芝浩原则的经典框架**（自由放贷 + 惩罚性利率 + 良好抵押品），也有**现代的演化**（扩展到非银行、灵活抵押品、市场利率）。
>
> **最后贷款人的成就**：防止大萧条重演、建立金融安全网、应对多次危机（1987、1998、2008、2020、2023）。
>
> **最后贷款人的局限**：**道德风险**（鼓励冒险）、**越界风险**（从最后变第一）、**僵尸机构**（资源错配）、**资产价格泡沫**（QE 副作用）。
>
> **理解最后贷款人 = 理解央行职能的核心张力**：
> - **稳定 vs 效率**：央行的稳定支持是否过度？
> - **道德风险 vs 系统性风险**：救助 vs 不救助的两难
> - **独立性 vs 灵活性**：央行如何在政治压力下保持独立性？
> - **国内 vs 国际**：跨境银行需要国际最后贷款人吗？
>
> **2008 年后，最后贷款人已经突破白芝浩原则**——美联储实际保护了**投资银行、保险公司、货币市场基金、商业票据市场**，**QE 已经成为常规工具**。**2020 年无限 QE 是最激进的实践**——央行"兜底"整个市场。
>
> **未来 10 年，最后贷款人的边界将持续重塑**——**影子银行、加密资产、气候风险、CBDC**都将挑战传统框架。**理解最后贷款人，就理解了央行职能的演进逻辑——从"被动救助者"到"主动市场参与者"，这一转变仍在进行**。
