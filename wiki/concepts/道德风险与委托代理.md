# 道德风险与委托代理

> 签约后，掌握信息或行动优势的一方可能采取损人利己的行为——委托人如何设计激励契约使代理人利益与自身对齐，是现代契约理论的核心。

由 AI 系统化整理（2026-06-03），待补充原始来源素材。

道德风险（Moral Hazard）与委托代理问题（Principal-Agent Problem）是信息不对称在**签约后**（ex-post）的经典表现。区别于签约前的逆向选择，它关注的是行动/努力/责任在交易后变得不可观察。理解这一框架是设计公司治理、保险条款、金融监管、激励机制的基础。

---

## 一、核心概念

### 1.1 道德风险

```
签约后
├─ 一方（代理人）采取行动 a
├─ 行动影响双方利益 u(θ, a)
├─ 行动不可被对方（委托人）完全观察
└─ 代理人可能偏离委托人最优行动 → 道德风险
```

### 1.2 委托代理问题

```
委托人（Principal）：
├─ 风险中性、关注结果
├─ 不直接参与生产/管理
└─ 设计契约激励代理人

代理人（Agent）：
├─ 风险厌恶（相对于委托人）
├─ 直接行动、信息优势
└─ 自利，可能机会主义
```

### 1.3 与逆向选择的对比

| 维度 | 逆向选择 | 道德风险 |
|------|----------|----------|
| 时点 | 签约前 | 签约后 |
| 隐藏内容 | 私人类型 | 私人行动 |
| 工具 | 信号、筛选 | 激励、监督 |
| 案例 | 二手车 | 保险理赔 |
| 解决核心 | 让对方说实话 | 让对方做对的事 |

### 1.4 道德风险的根源

| 来源 | 说明 |
|------|------|
| 行动不可观察 | 委托人无法直接观察 |
| 行动不可验证 | 即便事后也无法证实 |
| 结果受随机因素影响 | 即便结果差也可能是运气 |
| 利益不对齐 | 代理人目标与委托人不一致 |

## 二、经典案例

### 2.1 保险市场

```
投保人（代理人）买火灾险后：
├─ 防范行为减少（懒得买灭火器）
├─ 高风险行为增加（乱接电线）
├─ 极端情况：故意纵火骗保
└─ 保险公司（委托人）承担损失
```

**形式化**：

```
投保人效用 U = U(W - P, e)
├─ W：财富
├─ P：保费
└─ e：防范努力水平（不可观察）

防范努力减少 → 损失概率上升 → 保险赔付增加
```

### 2.2 CEO 与股东

```
CEO（代理人）：
├─ 信息优势：真实经营状况
├─ 行动不可观察：实际努力程度
└─ 自利动机：薪酬、地位、权力

股东（委托人）：
├─ 关心公司价值最大化
├─ 信息劣势
└─ 通过董事会监督 CEO
```

**典型代理问题**：

| 行为 | 表现 | 后果 |
|------|------|------|
| 偷懒 (Perks) | 在岗消磨时光 | 经营效率低 |
| 帝国构建 | 盲目扩张规模 | 资源浪费 |
| 盈余管理 | 操纵财报 | 信息失真 |
| 在职消费 | 豪华办公室、专机 | 股东利益输送 |
| 风险转移 | 高风险投资 | 股东承担风险 |

### 2.3 银行与储户

```
储户（委托人）：
├─ 把钱存银行
├─ 关心资金安全
└─ 难以监督银行放贷

银行（代理人）：
├─ 决定资金运用
├─ 风险偏好可能过高
└─ "大而不倒"预期 → 道德风险
```

> 2008 年金融危机中，"大而不倒"（Too Big to Fail）激励银行从事高风险投资——储户/纳税人承担损失。

### 2.4 其他典型案例

| 场景 | 代理人 | 委托人 | 道德风险 |
|------|--------|--------|----------|
| 医生-患者 | 医生 | 患者 | 过度治疗 |
| 律师-客户 | 律师 | 客户 | 拖延、扩大诉讼 |
| 基金经理 | 基金经理 | 投资人 | 过度交易、风险投机 |
| 政府-纳税人 | 政府 | 纳税人 | 浪费、低效 |
| 中介-买卖双方 | 中介 | 双方 | 信息隐瞒 |

## 三、形式化模型

### 3.1 经典委托代理模型

```
委托人设计契约 w(q) 支付给代理人：
├─ q：可观察的结果（产出、利润等）
├─ w(q)：基于结果的支付
├─ a：代理人不可观察的努力
└─ θ：随机外生因素

代理人效用：U = u(w(q) - c(a))
├─ c(a)：努力成本，c' > 0, c'' > 0
└─ 约束：保留效用 Ū（机会成本）

委托人目标：max E[v(q - w(q))]
├─ 期望利润
└─ 风险中性
```

### 3.2 激励相容约束 (IC)

```
代理人选择努力 a*，最大化期望效用：
a* ∈ argmax E[u(w(q) - c(a))]

若 a* = a_P（委托人最优努力）：
w(q) 必须激励代理人选择 a_P
```

### 3.3 参与约束 (IR)

```
代理人接受契约的最低条件：
E[u(w(q) - c(a))] ≥ Ū
```

### 3.4 最优契约形式

| 条件 | 最优契约 |
|------|----------|
| 代理人为风险中性 | 固定工资（无激励） |
| 代理人为风险厌恶，委托人风险中性 | 全部剩余归代理人（让代理人成"剩余索取者"） |
| 双方都风险厌恶 | 风险与激励的权衡 |
| 努力可观察 | 强制选 a_P（无激励成本） |
| 努力不可观察 | 必须给予激励支付 |

> Mirrlees (1974) - Holmstrom (1979) 是委托代理理论的奠基工作。

## 四、代理成本

### 4.1 Jensen & Meckling (1976) 三类成本

```
总代理成本 = 委托人的监督成本
           + 代理人的担保成本
           + 剩余损失
```

| 成本 | 内容 |
|------|------|
| 监督成本 | 委托人监控代理人的支出（审计、董事会） |
| 担保成本 | 代理人承诺不损害委托人或提供抵押 |
| 剩余损失 | 代理人决策偏离委托人最优造成的福利损失 |

### 4.2 代理成本的影响因素

| 因素 | 代理成本 |
|------|----------|
| 信息不对称程度 | 越大，代理成本越高 |
| 结果不确定性 | 越大，越难评估努力，代理成本越高 |
| 代理人风险厌恶 | 越高，激励成本越高 |
| 监督技术 | 越先进，监督成本越低 |
| 代理人股权占比 | 越高，利益绑定越强 |

### 4.3 治理结构与代理成本

```
├─ 股权结构（分散 vs 集中）
├─ 董事会独立性
├─ 信息披露质量
├─ 经理人市场（声誉）
├─ 控制权市场（敌意收购）
└─ 法律对小股东保护
```

## 五、解决机制

### 5.1 激励契约

| 机制 | 原理 | 例子 |
|------|------|------|
| 绩效工资 | 工资挂钩业绩 | 销售提成 |
| 股权激励 | 让代理人成为股东 | 股票期权、限制性股票 |
| 奖金/年终奖 | 短期绩效奖励 | 业绩奖金 |
| 递延支付 | 长期绑定 | 递延奖金、退休金 |
| 锦标赛 | 相对排名激励 | 销售竞赛、晋升竞争 |
| 团队激励 | 集体奖励 | 团队奖金 |

### 5.2 监督机制

| 机制 | 工具 |
|------|------|
| 直接监督 | 现场检查、监控设备 |
| 财务监督 | 审计、财报、预算 |
| 第三方监督 | 评级机构、监管、媒体 |
| 行业自律 | 行业协会、职业道德 |
| 声誉机制 | 长期声誉积累 |

### 5.3 抵押/担保

```
├─ 抵押品：违约时委托人获得补偿
├─ 保证金：从事风险活动的押金
├─ 担保人：第三方担保
└─ 准备金：银行业资本充足率
```

### 5.4 信息透明

| 措施 | 效果 |
|------|------|
| 强制披露 | 减少信息差 |
| 独立审计 | 验证真实性 |
| 内部信息上链 | 区块链防篡改 |
| 公开竞价 | 竞争性筛选 |

### 5.5 法律与惩罚

- 合同法：明确违约责任
- 公司法：信义义务、派生诉讼
- 监管法：巴塞尔协议、Solvency II
- 刑法：欺诈入刑

## 六、金融市场应用

### 6.1 公司治理

```
股东 → 董事会 → 管理层 → 员工
       （委托人链）

每一层都存在代理问题：
├─ 股东-管理层：薪酬、决策
├─ 管理层-员工：努力、偷懒
├─ 大股东-小股东：隧道挖掘
└─ 实际控制人-公司：关联交易
```

**应对**：
- 独立董事制度
- 累积投票制
- 股东诉讼权
- 机构投资者参与
- 退市制度

### 6.2 资产管理

| 道德风险 | 表现 | 应对 |
|----------|------|------|
| 风险偏好 | 投资风格漂移 | 投资范围限制 |
| 信息披露 | 净值披露不透明 | 估值法规范 |
| 利益冲突 | 与自有资金博弈 | 防火墙、隔离账户 |
| 高频交易 | 损害客户利益 | 最佳执行义务 |
| 关联交易 | 利益输送 | 披露、禁止 |

### 6.3 银行与金融监管

```
巴塞尔协议 III 的代理问题应对：
├─ 资本充足率（强制自有资金）
├─ 杠杆率限制（约束扩张）
├─ 流动性覆盖（LCR）
├─ 系统重要性银行附加资本
└─ 压力测试
```

### 6.4 保险监管

- 偿付能力监管（Solvency II）
- 准备金提取
- 再保险要求
- 投资范围限制

## 七、行为扩展

### 7.1 有限理性视角

- 行为人不完全自利，也有互惠、公平、地位等动机
- 心理契约：隐含的责任感
- 实验经济学：信任游戏、公共品博弈

### 7.2 多任务代理

```
Holmstrom & Milgrom (1991)：
├─ 代理人有多个任务
├─ 强激励会扭曲努力配置
└─ 平衡多任务激励是关键
```

例：考核销售 → 忽视客户长期关系；考核利润 → 忽视员工培训。

### 7.3 团队代理

- 团队生产中个人贡献难以衡量 → 搭便车
- 解决方法：相互监督、利润分享、强制排名
