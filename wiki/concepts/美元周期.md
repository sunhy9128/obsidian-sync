# 美元周期（Dollar Cycle）

> 美联储的加息/降息如同"心脏跳动"——扩张期（降息+QE）输出美元，收缩期（加息+QT）回流美元。一个完整周期 5-10 年。理解美元周期，是理解全球资产价格波动、EM 危机、新兴市场命运的总开关。

## 一、一句话定义

**美元周期**（Dollar Cycle）指**美联储货币政策扩张与收缩交替的周期性波动**，核心指标是联邦基金利率 + 美联储资产负债表规模。每个完整周期包括**宽松期（降息+QE）**和**紧缩期（加息+QT）**两个阶段，周期切换通常以美国经济周期（衰退/复苏）和金融市场状况为驱动。

**与[[美元潮汐]]的区别**：美元周期强调**货币政策本身的涨落**（美联储动作），美元潮汐强调**全球资本的传导**（资本流向 + EM 反应）。

## 二、美元周期的两大支柱

### 支柱一：联邦基金利率（价格工具）

```
联邦基金利率（Federal Funds Rate）
  ├── 区间上限
  ├── 区间下限
  └── 美联储通过 IORB（准备金利率）+ ON RRP（隔夜逆回购）控制
```

**核心机制**：利率上升 = 美元资产收益率上升 = 吸引资本回流美国 = 美元升值。

### 支柱二：美联储资产负债表（数量工具）

```
美联储资产负债表（Fed Balance Sheet）
  ├── 扩表（QE）：购债释放基础货币 → 准备金增加 → 流动性扩张
  └── 缩表（QT）：停止再投资 + 主动卖债 → 准备金减少 → 流动性收缩
```

**核心机制**：扩表 = 美元流动性"放水" → 流向全球；缩表 = "抽水" → 流动性回流美国。

## 三、美元周期的四个阶段

```
       ┌─────────────────────────────────────────┐
       │   美元周期 4 阶段                        │
       │                                         │
       │   阶段 1：宽松（降息+QE）                │
       │   ├── 美联储降息                         │
       │   ├── 启动 QE                            │
       │   └── 美元贬值，资本流向 EM              │
       │       ↓                                 │
       │   阶段 2：平台（Hold）                   │
       │   ├── 利率稳定                           │
       │   ├── 缩表/扩表视情况                    │
       │   └── 全球流动性充裕，资产价格上涨       │
       │       ↓                                 │
       │   阶段 3：紧缩（加息+QT）                │
       │   ├── 美联储加息                         │
       │   ├── 启动 QT                            │
       │   └── 美元升值，资本回流美国             │
       │       ↓                                 │
       │   阶段 4：危机应对（紧急宽松）            │
       │   ├── 危机爆发 → 紧急降息 + 紧急 QE     │
       │   ├── 提供美元互换额度                   │
       │   └── 进入新一轮宽松周期                 │
       │       ↓                                 │
       └─────── (回到阶段 1) ─────────────────────┘
```

## 四、战后历次美元周期

| 周期 | 阶段 | 关键事件 | 对 EM 影响 |
|------|------|---------|----------|
| **1971-1980** | 美元危机 | 美元脱钩黄金、滞胀 | 拉美开始借美元债 |
| **1980-1982** | 沃尔克紧缩 | 加息至 20% | 1982 拉美债务危机 |
| **1985-1986** | 降息 | 广场协议 | 日元升值、日本泡沫 |
| **1990s** | 多次降息+加息 | 1992 ERM、1994 龙舌兰、1997 亚洲 | 多轮 EM 危机 |
| **2001-2003** | 降息 | 互联网泡沫 + 911 | 拉美恢复、亚洲复苏 |
| **2007-2008** | 紧急宽松 | 次贷危机 → QE1 | 全球金融危机 |
| **2008-2014** | QE | QE1/QE2/QE3 + 扭曲操作 | EM 信贷扩张、资本流入 |
| **2014-2015** | 紧缩预期 | 缩减恐慌（Taper Tantrum） | EM 货币贬值（卢布崩盘） |
| **2015-2018** | 加息 | 渐进加息 + 缩表 | 2018 土耳其阿根廷危机 |
| **2019-2020** | 紧急宽松 | COVID 冲击 → 0 利率 + 无限 QE | EM 短期资本流入 |
| **2022-2024** | 激进加息 | 加息至 5.5%，QT | 斯里兰卡违约、美元强势 |
| **2024-2025** | 降息 | 通胀回落 → 降息周期启动 | 新一轮资本流入 |
| **2026-?** | 新一轮加息？ | 任庄主判断"已处于新一轮加息周期中或前夜" | 待观察 |

## 五、美元周期的核心规律

### 规律 1：美联储政策只服务于国内目标

```
美国《联邦储备法》（Federal Reserve Act）：
  - 美联储的目标是充分就业 + 物价稳定
  - 不承担"全球最后贷款人"角色
  - 不为全球金融稳定负责
     ↓
后果：美联储政策经常"溢出"到全球
  - 加息：EM 货币贬值、债务危机
  - 降息：EM 资产泡沫、通胀输入
```

### 规律 2：美元周期与 EM 危机高度同步

> 自 1980 年以来，几乎每一次美元加息周期都伴随 EM 危机。

| 加息周期 | 时间 | EM 危机 |
|---------|------|---------|
| 1980-1982 | 沃尔克 | 1982 拉美债务危机 |
| 1994-1995 | 格林斯潘 | 1994 龙舌兰危机 |
| 1997-1998 | 格林斯潘 | 1997 亚洲金融危机 |
| 1999-2000 | 格林斯潘 | 2000 互联网泡沫 + EM 动荡 |
| 2004-2006 | 格林斯潘/伯南克 | 2006 EM 局部危机 |
| 2015-2018 | 耶伦/鲍威尔 | 2018 土耳其阿根廷危机 |
| 2022-2024 | 鲍威尔 | 2022 斯里兰卡违约、脆弱五国 |

### 规律 3：缩表（QT）的杀伤力 > 加息

```
缩表（QT）的传导：
  美联储停止再投资 → 银行体系准备金减少
       ↓
  美债买盘减少 → 美债收益率上升（长端）
       ↓
  美元流动性紧张 → 全球美元融资成本上升
       ↓
  EM 美元债务展期困难 → 违约风险飙升
     ↓
  加息杀伤力：利率传导（较慢）
  缩表杀伤力：流动性传导（更快）
```

### 规律 4：美元周期与油价负相关

```
美元走强 → 大宗商品价格下跌（以美元计价的商品更贵）
     ↓
资源出口国（俄罗斯、巴西、中东）收入下降
     ↓
EM 经常账户恶化 → 货币贬值压力
     ↓
反过来：美元走弱 → 大宗商品上涨 → 资源国受益
```

## 六、美元微笑理论（Dollar Smile Theory）

由 Stephen Jen（2001）提出。认为美元指数（DXY）在三种情景下都会走强，呈"微笑曲线"：

```
情景 1：美国经济强劲
  ├── 美联储紧缩 → 利差扩大 → 美元升值
  └── "微笑左半边"
  
情景 2：美国经济疲软（避险）
  ├── 全球风险偏好下降 → 资金回流美元避险
  └── "微笑底部"（即右半边起点）
  
情景 3：美国深度衰退（避险+紧缩）
  ├── 极端恐慌 → 美元和美债双涨（避险）
  └── "微笑右半边"
  
核心洞察：无论美国经济好坏，美元都可能走强
  - 经济好：利差驱动
  - 经济差：避险驱动
  → 美元是"双向风险对冲工具"
```

**对 EM 的含义**：美元微笑意味着 EM 在两种情景下都面临压力——美国好时被加息冲击，美国差时被避险冲击。

## 七、当前周期判断（2026 年）

根据 [[我们已经处于新一轮加息周期中或前夜]] 的分析：

**任庄主核心判断**（2026.06）：
- 市场已处于**新一轮美元加息周期中，或正处于其前夜**
- 通胀多因素推动（能源 + 输入型 + 贸易战 + 加薪 + 财政发力）
- 美联储降息预期被推迟
- 沃什"降息+缩表"组合被市场误读
- 鲍威尔等鹰派理事仍在任

**对 EM 的含义**：
- 美元新一轮走强周期可能开启
- 2025-2026 年降息预期带来的资本流入可能逆转
- EM 需要提前为新一轮潮汐做准备

详见 [[2026 新一轮美元紧缩周期与新兴市场]]。

## 八、一句话总结

> **美元周期 = 美联储货币政策扩张与收缩的周期性波动，是全球资产价格 + EM 命运的总开关**。
>
> 它的**两大支柱**是联邦基金利率（价格工具）和美联储资产负债表（数量工具）。**四大阶段**是宽松 → 平台 → 紧缩 → 危机应对，构成 5-10 年的完整周期。**核心规律**是：美联储只服务国内目标、美元周期与 EM 危机高度同步、QT 杀伤力 > 加息、美元周期与油价负相关、美元微笑（双向走强）。
>
> **理解美元周期 = 理解 EM 的命运**。每一轮美元周期，都对应一轮 EM 危机——这不是偶然，而是结构性的。
