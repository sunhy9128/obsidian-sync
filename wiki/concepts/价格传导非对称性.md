# 价格传导非对称性

> 上游价格变动传导至下游时，**涨价传导快，降价传导慢**——这就是价格传导的非对称性。这是理解CPI"猪性"、通胀粘性、以及为什么"成本推动"难以形成持续通缩的核心工具。

## 一、核心定义

价格传导非对称性（Asymmetric Price Transmission）是指**上游价格变动向中下游传导时，幅度、速度、方向存在不对称**的现象。

```
涨价方向：传导快、幅度大（企业提价）
降价方向：传导慢、幅度小（企业抵抗）
```

结果是：**价格易涨难跌**——这是许多商品和服务通胀具有"粘性"的微观原因。

## 二、为什么存在非对称性？

### 1. 菜单成本（Menu Costs）

企业调整价格需要成本（重新印菜单、更新系统、通知客户等）。小幅降价不值得付出这些成本，因此**小幅降价往往不传导**。

### 2. 竞争结构

- **寡头市场**：大企业不愿率先降价（避免价格战），等待竞争对手先降
- **完全竞争**：小企业被动接受价格，但降价后需求弹性有限

### 3. 合同约束

许多上下游合同是**长期锁价合同**，短期内无法随市场调整。只有合同到期续签时价格才能调整，导致传导滞后。

### 4. 损失厌恶

企业对"降价"的感知比"涨价"更强，因此**降价决策更谨慎**，降价幅度更小。

## 三、典型案例

### 案例1：原油→汽油

| 变动方向 | 传导速度 | 传导幅度 |
|----------|----------|----------|
| 原油↑10% → 汽油↑ | 约2-4周 | 约8-10%（部分传导） |
| 原油↓10% → 汽油↓ | 约4-8周 | 约5-7%（传导不充分） |

**原因**：加油站往往在原油下跌后仍维持高价一段时间（"价格粘性"），直到市场竞争压力迫使其下调。

### 案例2：猪周期中的"成本传导"

在[[猪周期]]中：

```
豆粕价格↑10%
  → 养殖成本↑
  → 猪肉价格↑（约6-9个月后传导）
  → 超市猪肉价格↑

但反过来：
豆粕价格↓10%
  → 养殖成本↓
  → 猪肉价格↓（传导更慢，幅度更小）
  → 消费者感知不明显
```

这解释了为什么猪肉价格**易涨难跌**——成本上升传导快，成本下降传导慢。

### 案例3：工资-通胀螺旋

| 传导方向 | 特征 |
|----------|------|
| 通胀↑ → 工资要求↑（生活成本上升） | 传导快，幅度大 |
| 通胀↓ → 工资↓（企业不愿降薪） | 传导慢，幅度小 |

结果：**通胀一旦起来，就很难快速回落**——工资-通胀螺旋具有粘性。

## 四、非对称性的度量

### 向下半段 vs 上半段

学者们通常用**ECM模型（误差修正模型）**来度量非对称性：

```
ΔP_t = α₀ + α₁⁺ ΔP_{t-1}⁺ + α₁⁻ ΔP_{t-1}⁻ + λE_{t-1} + ε_t
                          ↑
                    正向价格变动    负向价格变动
                    （涨价传导）    （降价传导）
```

- 若 α₁⁺ ≠ α₁⁻，则存在非对称性
- 若 |α₁⁺| > |α₁⁻|，则为**下行传导更慢**（更常见的类型）

## 五、非对称性对金融分析的影响

### 1. CPI的"粘性"

因为价格传导非对称，CPI中的**核心通胀一旦上升就很难快速下降**。这解释了为什么央行对抗通胀需要提前收紧——通胀起来后再控制，已经太晚。

### 2. 黄金的反直觉现象

在[[金融传导机制分析框架]]的黄金案例中，实际利率框架（涨价传导）和避险需求（降价传导）的非对称性，共同决定了金价对不同情景的非对称响应。

### 3. 企业利润率的周期

当上游原材料涨价时，拥有强定价权的龙头企业能**快速转嫁成本**（传导快），而没有定价权的企业**只能自己消化**（传导慢）。这导致行业内部出现"强者恒强、弱者淘汰"的分化。
