# 新质生产力

> 以高技术、高能效、高质量为特征的生产力，推动产业深度转型升级

## 一、核心定义

新质生产力是相对于传统生产力而言的，代表了新技术、新产业、新业态、新模式、新动能的发展方向，是实现高质量发展的关键抓手。

## 二、二十届三中全会表述

> "推动技术革命性突破、生产要素创新性配置、产业深度转型升级，推动劳动者、劳动资料、劳动对象优化组合和更新跃升，催生新产业、新模式、新动能，发展以高技术、高能效、高质量为特征的生产力"

## 三、主要产业方向

| 领域 | 说明 |
|------|------|
| 新能源车 | 电动汽车、氢能 |
| 新材料 | 先进材料、复合材料 |
| 生物医药 | 创新药、生物技术 |
| 航空航天 | 商业航天、低空经济 |
| 数字经济 | AI、大数据、云计算 |
| 量子技术 | 量子计算、量子通信 |
| 生命科学 | 基因技术、生物工程 |
| 人工智能 | 大模型、机器学习 |

### 低空经济
依托有人驾驶与无人驾驶飞行器，在低空空域进行载人、载物、城市管理等多场景飞行活动。

应用场景：旅游、农业、消防、巡检、物流等

## 四、关键配套政策

### 4.1 金融支持
- 鼓励和规范发展天使投资、风险投资、私募股权投资
- 更好发挥政府投资基金作用
- 发展**耐心资本**

### 4.2 人才支撑
- 建立高技术人才移民制度
- 加强基础学科、新兴学科、交叉学科建设
- 培养战略科学家、一流科技领军人才、创新团队

### 4.3 产业链安全
- 推动国有资本向**前瞻性战略性新兴产业**集中
- 健全强化集成电路、工业母机、医疗装备、仪器仪表、基础软件、工业软件、先进材料等重点产业链

## 五、制造业地位

| 数据 | 说明 |
|------|------|
| 2023年制造业占GDP比重 | **26%**（持续下降）|
| 2010年以来 | 制造业对GDP增长贡献率震荡下降 |

> "建立保持制造业合理比重投入机制，合理降低制造业综合成本和税费负担"
