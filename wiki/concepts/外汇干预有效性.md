# 外汇干预有效性

## 核心命题

> 外汇干预是强心针，不是根治药。

本概念记录干预有效性的判断框架和本案例的实证。

## 干预有效性的判断维度

### 短期有效 vs 长期有效

| 维度 | 短期（<1个月） | 中期（1-6个月） | 长期（>6个月） |
|------|--------------|----------------|--------------|
| **价格效应** | 显著 | 取决于基本面 | 回归基本面 |
| **信号效应** | 强 | 逐渐消退 | 需要重复验证 |
| **市场情绪拐点** | 明显 | 不确定 | 基本面主导 |

### 干预失效的典型模式

**本案例模式——"踩刹车但无法转弯"**：

1. 日元在160.70日元时有技术支撑（心理关口+历史支撑）
2. 日本当局用11万亿日元强行拉回155区间
3. 但在无基本面支撑的情况下，反弹在1个月内回吐80%+
4. 汇率再次逼近160关口

这是典型的**逆水行舟**失效模式：干预方向与宏观基本面逆流而行。

## 干预有效性的前置条件

1. **政策协同**：财政+货币同向发力
   - 本案例失败原因：日本政府积极财政 vs 央行结束负利率，方向相悖
2. **基本面不太坏**：经济周期支持汇率企稳
   - 本案例失败原因：日美利差持续扩大、贸易逆差扩大
3. **足够规模和持续性**：一次性大规模 vs 持续小规模
   - 本案例：规模足够大（11万亿），但缺乏持续性
4. **市场情绪临界点**：在极度单边预期时介入
   - 本案例部分满足：4月30日市场极度恐慌

## 干预有效性的代理指标

| 指标 | 含义 |
|------|------|
| 汇率绝对值变化 | 干预后汇价是否持续偏离原趋势 |
| 波动率变化 | 干预是否降低了日内波动 |
| 市场预期调查 | 如日经捷讯调查：是否认为干预必要 |
| 散户仓位变化 | FX散户逆势多头占比（本案例约60%） |
| 隐含波动率 | 期权市场对未来的预期是否改变 |

## 本案例实证

**输入**：11.7349万亿日元（约$780亿，2026年5月）

**产出**：
- 汇率从159-160区间短期拉回155区间（约升值4-5日元）
- 约1个月后回吐大部分升幅，再次逼近160

**投入产出比**：约1日元/11万亿日元 → 性价比极低

**但战略价值存在**：
- 防止了日内崩盘式下跌（可能直接跌穿160至162-165）
- 打断了单边做空预期，迫使投机者重新评估仓位
- 争取了政策协调的时间窗口
