# GDP（国内生产总值）

> 在一定时期内，一个国家或地区境内生产的所有最终产品和服务的市场价值总和。
> 由 AI 系统化整理（2026-06-03），待补充原始来源素材

---

## 一、核心定义

### 1.1 概念本质

GDP（Gross Domestic Product，国内生产总值）衡量的是**地域**范围内（而非国籍范围内）一定时期内（通常为季度或年度）所生产的最终产品与服务的**市场价值**。

> 注意三个关键限定：
> ① 是"生产的"而非"销售的"（生产即计入，含存货）
> ② 是"最终的"（Final Goods），不含中间品
> ③ 是"市场价值"（非市场活动如家务、自给农业不计入）

### 1.2 GDP 与 GNP 的区别

| 指标 | 衡量范围 | 计算方式 | 例子 |
|------|----------|----------|------|
| **GDP**（国内生产总值） | 地域概念：境内生产 | 不论国籍 | 中国境内工厂生产的产品 |
| **GNP**（国民生产总值） | 国籍概念：国民生产 | 不论地域 | 中国企业在海外子公司的产出 |

```
GDP = GNP - 本国居民海外生产 + 外国居民本国生产
   = GNP - 海外要素净支付 (NFP)
```

**GDP 净值（NNP）**：
```
NNP = GNP - 折旧
```

## 二、三种计算法（同一 GDP，三视角）

### 2.1 生产法（Production Approach）

按产业增加值逐层加总：

```
GDP = Σ 各行业增加值
    = Σ (各行业总产值 - 各行业中间消耗)
```

### 2.2 收入法（Income Approach）

按生产要素的收入加总：

```
GDP = 工资 + 租金 + 利息 + 利润
    = 劳动者报酬 + 营业盈余 + 折旧 + 间接税净额
```

### 2.3 支出法（Expenditure Approach）

按最终用途加总（最常用）：

```
GDP = C + I + G + NX
```

| 符号 | 含义 | 英文 | 中国 2024 占比（参考） |
|------|------|------|---------------------|
| C | 居民消费 | Consumption | ~38% |
| I | 投资 | Investment | ~42% |
| G | 政府支出 | Government Spending | ~16% |
| NX | 净出口 | Net Exports | ~+1% |

> 三种方法理论上结果相同，因"产出 = 收入 = 支出"（循环流动）。

### 2.4 计算示例

假设一国一年数据：
- 居民消费 200 亿
- 投资 100 亿
- 政府支出 80 亿
- 出口 50 亿，进口 40 亿

```
GDP = 200 + 100 + 80 + (50 - 40) = 390 亿
NX = 50 - 40 = 10 亿（贸易顺差）
```

## 三、名义 GDP 与实际 GDP

### 3.1 概念区分

| 指标 | 英文 | 价格基础 | 反映 |
|------|------|----------|------|
| **名义 GDP** | Nominal GDP | 当期价格 | 数量 × 当期价格 |
| **实际 GDP** | Real GDP | 基期价格 | 数量 × 基期价格（剔除价格变化） |
| **潜在 GDP** | Potential GDP | 充分就业下的产出 | 长期趋势产出 |

### 3.2 计算示例

| 年份 | 苹果产量 | 苹果价格 | 香蕉产量 | 香蕉价格 |
|------|---------|---------|---------|---------|
| 2024 | 100 | 2 元 | 50 | 4 元 |
| 2025 | 110 | 3 元 | 60 | 5 元 |
| **基期：2024** | — | — | — | — |

```
名义 GDP 2024 = 100×2 + 50×4 = 400
名义 GDP 2025 = 110×3 + 60×5 = 630

实际 GDP 2025 = 110×2 + 60×4 = 460（用 2024 价格）

实际增长率 = (460 - 400) / 400 = 15%
```

### 3.3 GDP 平减指数（GDP Deflator）

衡量 GDP 整体价格水平：

```
GDP Deflator = (名义 GDP / 实际 GDP) × 100
             = (630 / 460) × 100 ≈ 136.96
```

> 与 CPI 区别：CPI 篮子是固定的（拉斯佩尔斯指数），平减指数篮子随当期生产结构变化（帕氏指数）。

## 四、GDP 的局限

### 4.1 经济学批评

| 局限 | 说明 |
|------|------|
| **忽略非市场活动** | 家务、志愿服务、自给农业 |
| **忽略地下经济** | 现金交易、灰色产业 |
| **忽略闲暇** | 工作时间越长，GDP 越高，但福利可能下降 |
| **忽略环境成本** | 污染、碳排放、资源枯竭未扣除 |
| **忽略收入分配** | 总量增长但分配恶化也计入增长 |
| **忽略质量变化** | 苹果手机 2007 vs 2024 价格难比较（Hedonic 方法部分修正） |
| **难以跨国比较** | 汇率 PPP vs 汇率法 |

### 4.2 替代/补充指标

- **HDI**（人类发展指数）：寿命 + 教育 + 收入
- **GPI**（真实进步指标）：含环境与社会成本
- **GNH**（国民幸福总值）：不丹提出
- **绿色 GDP**：扣除环境损失

## 六、关键易错点

| 误区 | 正确认识 |
|------|----------|
| GDP 越高 = 居民越富 | 可能是分配恶化；GNI/人均 GDP 更能反映 |
| 二手车交易计入 GDP | **不计入**（已是最终品再转让） |
| 政府转移支付计入 GDP | **不计入**（无对应生产） |
| 股票买卖计入 GDP | **不计入**（金融资产转让，非生产） |
| 中间品计入 GDP | **不计入**（避免重复计算） |
