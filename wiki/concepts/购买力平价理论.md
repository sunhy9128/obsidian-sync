# 购买力平价理论

> 在无运输成本和贸易壁垒的情况下，同一种商品在所有国家的价格用同一货币衡量应相等——汇率将各国价格水平拉平。

## 一、核心定义

### 1.1 概念本质

**购买力平价**（Purchasing Power Parity, PPP）由瑞典经济学家卡塞尔（Gustav Cassel, 1922）系统化提出，核心命题：

> 两国货币的均衡汇率，应当使两国**可贸易的一篮子商品**以同一货币计价时价格相同。

PPP 是**长期汇率理论**——它告诉我们在没有摩擦的世界里汇率"应该"是什么，而不是短期汇率"会是"什么。

### 1.2 一价定律（Law of One Price, LOOP）

PPP 的微观基础：

```
P_domestic = S × P_foreign
```

| 符号 | 含义 |
|------|------|
| P_domestic | 商品在本国价格（人民币） |
| S | 即期汇率（直接标价：1 外币 = S 本币） |
| P_foreign | 商品在外国价格（外币） |

**示例**：一件商品在美国 10 美元，在中国 70 元，若一价定律成立则 S = 7 元/美元。

## 二、绝对 PPP 与相对 PPP

### 2.1 绝对 PPP（Absolute PPP）

某一**时点**的均衡汇率等于两国价格水平之比：

```
S = P / P*
```

| 符号 | 含义 |
|------|------|
| S | 即期汇率（本币/外币） |
| P | 本国一般物价水平 |
| P* | 外国一般物价水平 |

> 即期汇率的购买力平价水平 = 两国物价指数之比。

**示例（基期 2020）**：
- 美国 CPI = 100，中国 CPI = 100，汇率 = 7.0
- 2024 年美国 CPI = 120，中国 CPI = 108
- 绝对 PPP 隐含汇率 = 7.0 × (108/120) = 6.30 元/美元

### 2.2 相对 PPP（Relative PPP）

汇率的**变动率**等于两国通胀率之差——这是实务中更有用的版本：

```
ΔS / S ≈ π - π*
```

| 符号 | 含义 |
|------|------|
| ΔS / S | 汇率（直接标价）的变动率 |
| π | 本国通胀率 |
| π* | 外国通胀率 |

> 通胀更高的国家，货币应**贬值**。

**示例**：
- 美元通胀 5%，人民币通胀 2%
- 相对 PPP：美元应相对人民币贬值 3%
- 原汇率 7.0，1 年后 PPP 隐含 = 7.0 × (1.05/1.02) ≈ 7.21 元/美元

### 2.3 两种 PPP 的对比

| 维度 | 绝对 PPP | 相对 PPP |
|------|----------|----------|
| 形式 | 水平值 S = P/P* | 变化率 ΔS/S = π - π* |
| 时间维度 | 单一时间点 | 两个时间点之间 |
| 实用度 | 较弱（受基期选择影响大） | **更强**（通胀数据易得） |
| 与 LOOP 关系 | 是 LOOP 的一般化 | 是绝对 PPP 的一阶差分 |

## 三、汉堡指数（Big Mac Index）

### 3.1 概念与原理

《经济学人》自 1986 年发布的非正式指标，选取**麦当劳巨无霸**作为标准化商品（在不同国家原料构成相似），用当地价格与"应该"的 PPP 价格比较。

### 3.2 计算方法

```
本地 PPP = 本地巨无霸价格 / 美国巨无霸价格
隐含汇率 = 本地价格（折美元） / 美国价格
高估/低估 % = (实际汇率 - 隐含汇率) / 隐含汇率
```

### 3.3 2024 年示例数据（虚构）

| 国家 | 本币价格 | 美元折算 | PPP 隐含汇率 | 实际汇率 | 货币状态 |
|------|---------|---------|-------------|---------|---------|
| 美国 | 5.69 美元 | 5.69 | — | — | 基准 |
| 欧元区 | 5.30 欧元 | 5.78 | 0.93 欧/$ | 0.93 欧/$ | **平价** |
| 中国 | 24.4 元 | 3.38 | 4.29 元/$ | 7.25 元/$ | **低估 ~41%** |
| 瑞士 | 6.90 瑞郎 | 7.65 | 1.21 瑞/$ | 0.91 瑞/$ | **高估 ~25%** |

> 中国人民币长期被汉堡指数视为低估（2023 年数据亦显示约 41% 低估），常见解释为**非贸易品（劳动力、租金）价格远低于发达国家**。

### 3.4 汉堡指数的局限

- 忽略**非贸易品**（理发、房产）占比
- 忽略**品牌定价权**与本地化口味
- 原料构成（牛肉 vs 鸡肉）国别差异
- 麦当劳营销策略本地化

## 四、PPP 的三大问题

### 4.1 贸易品 vs 非贸易品问题

| 类型 | 是否可跨境套利 | PPP 适用性 |
|------|--------------|-----------|
| **贸易品** | 是（iPhone、大豆） | 适用 |
| **非贸易品** | 否（理发、本地餐饮） | **不适用** |

> 发展中经济体的**巴拉萨-萨缪尔森效应**（Balassa-Samuelson）：
> 贸易品部门生产率高 → 工资整体上涨 → 非贸易品价格被动抬升 → 整体物价高于发达国家 → 实际汇率被低估。

```
实际汇率（PPP / 市场汇率）≈ (P_tradable / P_nontradable)_country
```

### 4.2 统计差异问题

- **ICP**（国际比较项目）需把各国价格折算到统一商品篮子
- 不同国家**消费结构**与**市场分割**（消费偏好差异）
- 质量调整与品牌差异难以标准化
- 灰色市场与汇率管制扭曲价格信号

### 4.3 短期偏离问题

| 短期偏离原因 | 解释 |
|------------|------|
| **贸易成本** | 运费、关税、保险费使跨境套利无法消除价差 |
| **市场分割** | 各国消费偏好与品牌差异导致同一商品难以完全标准化 |
| 资本流动冲击 | 利率差异引发套息，资金流向高息国 |
| 黏性价格 | 商品价格调整慢于汇率 |
| 政府干预 | 央行入市干预、资本管制 |
| 预期因素 | 汇率自我实现的预期 |

> 实证：长期（5-10 年）PPP 接近成立，但短期偏差可达 30-50%，偏离呈现**半衰期 3-5 年**的均值回复。

## 五、PPP 与实际汇率

### 5.1 实际汇率定义

```
实际汇率 q = S × P* / P（直接标价）
```

- q 上升 → 本币**实际升值**（购买力增强）
- q 下降 → 本币**实际贬值**

### 5.2 名义汇率、实际汇率、PPP 关系

```
PPP 成立条件：S = P / P*  ⇔  q = 1
PPP 偏离：S ≠ P / P*        ⇔  q ≠ 1
```

### 5.3 实际有效汇率（REER）

对一篮子贸易伙伴加权的实际汇率：

```
REER = Π (S_i / P_i)^{w_i}  （几何加权）
```

| 用途 | 含义 |
|------|------|
| 反映**国际竞争力** | REER 上升 = 出口竞争力下降 |
| 央行**通胀监测** | 实际升值可能压低进口通胀 |
| 触发**外汇干预** | 偏离均衡过远时入场 |

## 六、PPP 的应用与局限

### 6.1 主要应用

| 场景 | 用途 |
|------|------|
| **跨国 GDP 比较** | 世界银行 ICP 项目（避免汇率波动干扰） |
| **国际贫困线** | 1.90/3.20 美元/天按 PPP 折算 |
| **长期均衡汇率估算** | 央行 EER 模型 |
| **学术比较** | 实际工资、生活水平 |

### 6.2 局限总结

```
1. 短期不准：偏离大、均值回复慢
2. 非贸易品失真：理发、住房不可贸易
3. 政策干扰：管制、关税、补贴扭曲价格
4. 篮子选择敏感：消费品 vs 资本品 PPP 差距大
5. 数据滞后：ICP 每 6-7 年一次
```

### 6.3 与其他汇率理论的关系

| 理论 | 关注对象 | 时间维度 |
|------|---------|---------|
| **购买力平价 PPP** | 贸易品价格（商品套利） | 长期 |
| **利率平价 CIP/UIP** | 资本收益（利率差与汇率预期） | 短期–中期 |
| **国际收支 BOP** | 经常账户与资本账户流量 | 中期 |

## 八、关键易错点

| 误区 | 正确认识 |
|------|----------|
| 汇率向 PPP 短期回归 | 短期偏离 30-50% 正常，半衰期 3-5 年 |
| 汉堡指数 = 真实 PPP | 单一商品近似，存在系统性偏差 |
| PPP 永远成立 | 仅长期近似成立，巴拉萨-萨缪尔森效应下结构差异显著 |
| PPP 准 = 货币没低估 | 可能仅是**非贸易品价格低**（如理发、房租） |
| 发展中国家货币 PPP 都低估 | 巴拿马、黎巴嫩等出现高估 |
