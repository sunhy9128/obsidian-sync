> 锚定效应（Anchoring Effect）指人们在判断与决策时，会**过度依赖最初接收到的信息（锚点）**，并基于该锚点做不充分调整，导致最终估计偏向锚点。

---

## 一、核心概念

### 1.1 定义

| 维度 | 内容 |
|------|------|
| 学术定义 | 个体在评估某一数值时，会以先前接收到的数字（锚点）为参照，偏离"锚"的调整幅度往往不足 |
| 提出者 | Tversky & Kahneman（1974） |
| 归类 | 启发式与偏差（Heuristics and Biases）核心发现之一 |
| 本质 | 启发式快速判断 → 节省认知资源 → 引入系统偏差 |

### 1.2 调整不足（Insufficient Adjustment）

```
决策过程（K-T 描述）：
  1. 接收锚点（任意数字、初始出价、第一印象）
  2. 评估目标值
  3. 沿锚点方向"调整"
  4. 因调整幅度不足，停留在"锚附近"而非真实值

调整公式（理论）：
  最终估计 = 锚 + α × (真实值 - 锚) ,  0 < α < 1
  α 越小 → 锚定效应越强
```

## 二、经典实验

### 2.1 幸运轮实验（Tversky & Kahneman, 1974）

```
设计：
  - 受试者转"幸运轮"得到 0~100 的随机数字
  - 然后被问"非洲国家在联合国中的比例是 >X 还是 <X"
  - X 是被试自己写的两位数

结果：
  轮盘高值组（≥65）：平均估计 45%
  轮盘低值组（≤10）：平均估计 25%
  差距：~20 个百分点
  
关键：锚点（轮盘数字）不应影响国家比例判断，但实际影响巨大
```

### 2.2 房产定价实验

| 实验 | 锚定 | 实际成交价 | 偏差 |
|------|------|----------|------|
| 房屋要价实验 | 卖方要价 30 万 | 实际成交 23 万 | 价格偏高 30% |
| 公寓求购实验 | 起始报价 5 万 | 实际成交 7 万 | 偏低 28% |
| 拍卖中锚 | 起拍价 100 元 | 最终均价 110 元 | 锚定显著 |

### 2.3 数字无关锚

```
关键发现：即使锚点与决策无关，也会产生锚定效应
  - 学生被问"红杉树最高多少米？"
  - 锚：电话号后四位作为随机数字
  - 高锚组：估计 282 米
  - 低锚组：估计 129 米
  → 显然电话号与树高无关，但仍产生影响
```

## 三、锚定效应的影响因素

### 3.1 增强锚定的因素

| 因素 | 作用 | 说明 |
|------|------|------|
| 锚点显著性 | 增强 | 显眼、权威、具体的锚点影响更大 |
| 认知负荷 | 增强 | 任务复杂时更依赖锚 |
| 知识缺乏 | 增强 | 领域生疏者更易被锚定 |
| 时间压力 | 增强 | 快速决策时调整不足 |
| 情绪唤醒 | 增强 | 高情绪状态下依赖锚点 |
| 锚与目标的"看似相关性" | 增强 | 表面相关但实际无关 |

### 3.2 减弱锚定的因素

- 锚点差异性大（不易形成单一锚）
- 受试者被提示"考虑反方向"
- 群体讨论稀释单一锚
- 专家知识（能识别锚的干扰）
- 高激励（如奖金对准确度的奖励）

## 四、应用领域与案例

### 4.1 谈判与商务

| 场景 | 锚的使用 | 效果 |
|------|---------|------|
| 薪资谈判 | 先开高薪者获更高最终价 | 首因效应 + 锚定 |
| 房产议价 | 卖家先要高价 | 买方调整后仍偏高 |
| B2B 报价 | 锚定 30% 毛利区间 | 客户谈判锚点 |
| 二手车交易 | 商家先报一口价 | 顾客心理锚定 |

### 4.2 零售与消费

```
定价策略：
  - 商品 A：99 元（无锚）
  - 商品 A+B：199 元
    A 被感知为"省钱" → 锚定 199 元 → 99 元是"占便宜"
  
原价划线：
  - 原价 1000 元 → 折后 500 元
  - 划线价作锚 → 折后价显得特别便宜
```

### 4.3 投资与金融

| 场景 | 锚的表现 | 后果 |
|------|---------|------|
| IPO 定价 | 发行价作锚 | 上市后波动受发行价影响 |
| 股票分析 | 52 周最高/最低价作锚 | 投资者偏好在锚附近交易 |
| 心理价位 | 整数关口（如 3000 点） | 锚定形成自我应验 |
| 买入价 | 个人成本价作锚 | 锚定导致处置效应 |
| 估值参考 | 同业 PE 中位数作锚 | 一致预期难以打破 |

### 4.4 法律与司法

```
经典实验（Englich, Mussweiler & Strack, 2006）：
  - 法官判决损害赔偿时
  - 实验组先看到原告"不相关"诉求（如要求 12 倍补偿）
  - 被告求偿越高 → 判决金额越高
  - 即使法官"知道"诉求不合理，仍受锚定影响
```

## 五、锚定与启示

### 5.1 个人决策的"反锚"策略

| 策略 | 操作 |
|------|------|
| 多源锚点 | 收集至少 3 个独立锚，避免单一锚支配 |
| 反向思考 | 主动问"如果相反呢？" |
| 锚点归零 | 重新框定问题，不接受初始数字 |
| 延迟决策 | 不要在接收锚后立即决策 |
| 提高专业 | 知识减少对锚的依赖 |

### 5.2 商业策略的"建锚"技巧

- 高开低走：先报高价，留谈判空间
- 价格套餐：高端套餐作锚，基础款显得便宜
- 暗示性锚：在大脑未识别前植入
- 权威锚：用专家/品牌建立信任锚
- 时间锚：让"时限紧迫"成为压力锚

## 六、与其他概念的关系

### 6.1 锚定 vs 损失厌恶

| 概念 | 焦点 | 关系 |
|------|------|------|
| 锚定 | 估计与判断偏向锚点 | 锚是"起点" |
| 损失厌恶 | 损失 > 收益 | 锚定后调整方向也受损失厌恶影响 |

```
案例：买入价 20 元的股票跌到 15 元
  - 锚定：15 元仍向 20 元调整 → 主观觉得"便宜"
  - 损失厌恶：不愿在 15 元卖出 → 怕"亏 5 元"
  - 两者叠加：持有不卖，期待回到 20 元
```

### 6.2 锚定 vs 心理账户

- 锚定：影响单一数值的判断
- 心理账户：影响多笔账目的分类与权衡
- 关联：账户"分类"本身就是一种锚定

### 6.3 锚定 vs 框架效应

- 框架：同一问题的不同表述影响选择
- 锚定：初始数字影响估计
- 重叠：锚定可视为框架的一种具体表现

## 七、神经基础

### 7.1 脑成像研究

```
fMRI 证据：
  - 锚定任务激活：内侧前额叶皮层（数值加工）
  - 调整过程：背外侧前额叶皮层（认知控制）
  - 锚点过度影响：前扣带皮层（冲突监控）
  - 与情绪关联：脑岛（情绪/价值评估）
```

### 7.2 跨文化差异

| 文化 | 锚定敏感度 |
|------|-----------|
| 西方（个人主义） | 较高 |
| 东亚（集体主义） | 部分研究显示略低 |
| 经验型 vs 分析型 | 经验型思维者更易锚定 |
