# 央行对冲工具（化债背景）

> 中国央行应对化债"流动性冻结"的工具组合 —— 2024 年净投放约 4-5 万亿

## 一、对冲的逻辑起点

### 1.1 为什么需要央行对冲？

```
化债机制：
  银行购买化债债
       ↓
  资金从银行账户流向政府账户
       ↓
  **银行体系流动性被"冻结"**
       ↓
  央行需要投放流动性 → "对冲"

量化目标：
  - 维持 M2 合理增长
  - 保持市场流动性充裕
  - 避免利率过度上行
  - 支持实体经济
```

### 1.2 对冲的总规模

| 指标 | 数据 |
|------|------|
| 2023 年净投放 | ~2 万亿 |
| 2024 年净投放 | ~4-5 万亿 |
| 2024 年降准幅度 | 累计约 1 个百分点 |
| MLF 余额变化 | 净增加 ~1 万亿 |

---

## 二、十大对冲工具详解

### 工具 1：MLF（中期借贷便利）⭐ 主力

#### 基本信息

| 维度 | 内容 |
|------|------|
| **本质** | 央行向商业银行提供中期抵押贷款 |
| **期限** | 3 个月、6 个月、**1 年期（主力）** |
| **抵押品** | 国债、央行票据、政策性金融债等 |
| **利率** | 化债期间约 2.5% → 2.0%（下调） |
| **发布时间** | 2014 年 9 月 |

#### 在化债中的作用

| 作用 | 说明 |
|------|------|
| **补充中期流动性** | 银行购买长期化债债 → 流动性下降 → MLF 补充 |
| **压低长端利率** | MLF 利率是 LPR 定价基础 |
| **置换到期 MLF** | 滚动续作，保持稳定 |

详见 [[wiki/concepts/MLF]]。

#### 化债期间典型操作

```
2023 年：MLF 净投放 ~3000 亿
2024 年：MLF 净投放 ~1 万亿
       ↓
央行持续通过 MLF 释放流动性
       ↓
与化债债的"流动性冻结"形成对冲
```

---

### 工具 2：PSL（抵押补充贷款）⭐⭐ 重要

#### 基本信息

| 维度 | 内容 |
|------|------|
| **本质** | 央行向政策性银行（国开行、农发行）提供长期低息资金 |
| **期限** | 3-5 年 |
| **利率** | 约 2.5%（低于 MLF） |
| **抵押品** | 政策性金融债等 |
| **发布时间** | 2014 年 4 月 |

#### 在化债中的作用

#### PSL 的"三大工程"支持

```
央行通过 PSL 给政策性银行（如国开行、农发行）
       ↓
政策性银行获得低成本资金
       ↓
用于支持：
  ① 保障性住房建设
  ② 城中村改造
  ③ "平急两用"公共基础设施
       ↓
相当于"绕道"支持地方基础设施
       ↓
但避免央行直接买债
```

#### 化债期间规模

| 时期 | PSL 余额 | 变化 |
|------|---------|------|
| 2022 年末 | ~2.8 万亿 | - |
| 2023 年末 | ~3.2 万亿 | +4000 亿 |
| 2024 年末 | ~3.8 万亿 | +6000 亿 |

> **PSL 是化债的重要"替代工具"**——通过政策性银行绕道支持基建，避免央行直接买债。

---

### 工具 3：保障性住房再贷款 ⭐⭐ 重要（2024 新设）

#### 基本信息

| 维度 | 内容 |
|------|------|
| **本质** | 央行向商业银行提供专项再贷款 |
| **规模** | **3000 亿元** |
| **利率** | **1.75%**（极低） |
| **期限** | 1 年期，可展期 |
| **发布时间** | 2024 年 5 月 |

#### 运作机制

```
商业银行收购存量商品房
       ↓
作为保障性住房使用
       ↓
商业银行向央行申请再贷款
       ↓
央行按收购金额的 60% 提供资金
       ↓
利率 1.75%（远低于市场利率）
       ↓
商业银行赚取利差（收购价低、贷款价高）
       ↓
同时化解房地产风险 + 支持化债
```

#### 战略意义

| 维度 | 内容 |
|------|------|
| **化解房地产风险** | 房企存量房被收购，避免烂尾 |
| **支持化债** | 房企回款，间接缓解地方土地财政压力 |
| **增加保障房供给** | 解决新市民、青年人住房问题 |
| **机制创新** | 央行结构性工具的新尝试 |

---

### 工具 4：买断式逆回购 ⭐ 2024 新增工具

#### 基本信息

| 维度 | 内容 |
|------|------|
| **本质** | 央行从二级市场买入债券（不同于传统逆回购） |
| **期限** | 3 个月 - 1 年 |
| **发布时间** | **2024 年 10 月** |
| **目的** | 释放中期流动性 |

#### 与传统逆回购的区别

| 维度 | 传统逆回购 | 买断式逆回购 |
|------|----------|------------|
| **债券所有权** | 短期质押 | **转移所有权** |
| **资金释放** | 短期 | 中长期 |
| **对冲效果** | 弱 | **强** |

#### 在化债中的作用

- 释放中期流动性
- 压低长端利率
- 支持银行体系扩表承接化债债

---

### 工具 5：再贷款再贴现 ⭐⭐ 精准滴灌

#### 基本信息

| 维度 | 内容 |
|------|------|
| **本质** | 央行向特定领域提供定向资金 |
| **类型** | 多种专项再贷款（科技、设备更新、保交楼等） |
| **利率** | 较低（1.75%-2.5%） |
| **规模** | 数千亿至上万亿 |

#### 主要专项再贷款

| 工具 | 规模 | 用途 |
|------|------|------|
| **科技再贷款** | 2000 亿 | 支持科技创新 |
| **设备更新再贷款** | 5000 亿 | 设备以旧换新 |
| **保交楼再贷款** | 数千亿 | 房地产保交楼 |
| **普惠养老再贷款** | 数百亿 | 养老服务 |
| **交通物流再贷款** | 数千亿 | 物流业 |

#### 在化债中的作用

- 精准支持实体经济
- 避免"大水漫灌"
- 减少银行扩表承接化债债的"挤压效应"

---

### 工具 6：降准（降低存款准备金率）⭐⭐⭐ 主力工具

#### 基本信息

| 维度 | 内容 |
|------|------|
| **本质** | 降低商业银行存款准备金率 |
| **化债期间降幅** | 2024 年累计约 1 个百分点 |
| **释放资金** | 约 2 万亿 |
| **效果** | 长期流动性 |

#### 化债期间降准节奏

| 时间 | 操作 | 累计释放 |
|------|------|---------|
| 2024.9 | 降准 0.5 个百分点 | ~1 万亿 |
| 2024.10 | 降准 0.5 个百分点 | ~2 万亿 |

#### 在化债中的作用

| 作用 | 说明 |
|------|------|
| **长期流动性** | 一次性释放，效果持续 |
| **压低资金成本** | 降低银行负债成本 |
| **支持信贷扩张** | 让银行有能力承接化债债 |

---

### 工具 7：下调政策利率

#### 化债期间利率下调

| 利率 | 2023 年初 | 2024 年末 | 降幅 |
|------|----------|---------|------|
| **MLF 利率** | 2.75% | 2.0% | -75 BP |
| **LPR 1Y** | 3.65% | 3.10% | -55 BP |
| **LPR 5Y** | 4.30% | 3.60% | -70 BP |
| **OMO 7 天逆回购** | 2.0% | 1.5% | -50 BP |

#### 化债中的作用

- 降低实体经济融资成本
- 压低长债收益率
- 减轻地方政府债务负担（即使是低息政府债）

---

### 工具 8：逆回购（公开市场操作）⭐ 日常工具

#### 基本信息

| 维度 | 内容 |
|------|------|
| **期限** | 7 天、14 天、28 天 |
| **频率** | 每个工作日 |
| **目的** | 调节短期流动性 |

#### 化债中的作用

- 平滑资金面波动
- 应对税期、节假日等季节性因素
- 与化债的短期流动性需求匹配

---

### 工具 9：SLF（常备借贷便利）

#### 基本信息

| 维度 | 内容 |
|------|------|
| **期限** | 1-3 个月 |
| **利率** | 较高（3.05%-3.35%） |
| **功能** | "利率走廊"上限 |
| **化债作用** | 应急工具，使用频率低 |

---

### 工具 10：国库现金管理

#### 基本信息

| 维度 | 内容 |
|------|------|
| **本质** | 财政部在央行账户的现金管理 |
| **化债期间作用** | 调节资金时序错配 |
| **具体操作** | 国库现金招标存入商业银行 |

## 三、工具组合——"双轮驱动"

### 3.1 总量工具 vs 结构性工具

```
化债对冲的工具组合：

【总量工具】（大水漫灌）
  - 降准（释放长期流动性）
  - MLF（中期流动性）
  - 逆回购（短期流动性）
  - 下调政策利率（压低利率）

【结构性工具】（精准滴灌）
  - PSL（政策性银行支持）
  - 保障性住房再贷款（房地产）
  - 各项专项再贷款（精准支持）
  - 买断式逆回购（中期创新）
```

### 3.2 化债对冲的"分工"

| 工具 | 释放规模 | 主要作用 |
|------|---------|---------|
| **降准** | ~2 万亿 | 长期流动性 |
| **MLF 净投放** | ~1 万亿 | 中期流动性 |
| **PSL** | ~6000 亿 | 政策性支持 |
| **保障性住房再贷款** | 3000 亿 | 房地产+化债 |
| **再贷款再贴现** | ~1 万亿 | 精准支持 |
| **买断式逆回购** | 新工具 | 中期创新 |
| **合计** | ~5 万亿 | 全面对冲 |

## 四、央行的对冲策略——三个层次

### 4.1 第一层：流动性对冲

| 工具 | 内容 |
|------|------|
| **降准** | 一次性释放 2 万亿 |
| **MLF** | 持续投放中期流动性 |
| **买断式逆回购** | 释放中期流动性 |
| **逆回购** | 调节短期波动 |

**目标**：保持银行体系流动性充裕

### 4.2 第二层：利率引导

| 工具 | 内容 |
|------|------|
| **MLF 利率下调** | 压低市场基准利率 |
| **LPR 下调** | 降低实体融资成本 |
| **OMO 利率下调** | 引导短端利率 |

**目标**：压低长端利率，支持化债债定价

### 4.3 第三层：结构性支持

| 工具 | 内容 |
|------|------|
| **PSL** | 支持"三大工程" |
| **保障性住房再贷款** | 化解房地产风险 |
| **专项再贷款** | 支持特定领域 |

**目标**：精准支持，化解化债的"次生风险"

## 五、对冲的有效性评估

### 5.1 成功之处

| 维度 | 评估 |
|------|------|
| **M2 维持增长** | 8%-9%（虽然低位但仍正增长） |
| **利率维持低位** | 化债债利率 2.5%-3% 较化债前 6%-8% 大幅下降 |
| **流动性基本充裕** | 银行间市场利率平稳 |
| **未出现系统性风险** | 银行净息差压力但未引发危机 |

### 5.2 局限之处

| 维度 | 评估 |
|------|------|
| **M2 增速低位** | 8%-9% 远低于历史 12%+ |
| **净息差持续收窄** | 银行盈利能力承压 |
| **CPI 通缩压力** | 0%-1% 通胀率反映需求不足 |
| **银行 ROE 下降** | 资本回报率持续下行 |

### 5.3 对冲的不完全性

> **2024 年央行净投放约 4-5 万亿，但化债冻结约 4-5 万亿**。
>
> 理论上完全对冲，但实际**部分对冲**：
> - 部分资金沉淀在政府账户
> - 部分资金被银行持有化债债
> - 流通速度下降导致"乘数效应"不足

## 六、与 QE 的对比

| 维度 | 央行对冲工具 | 美联储 QE |
|------|------------|----------|
| **扩表主体** | 央行 | 央行 |
| **资金来源** | 基础货币（印钞） | 基础货币（印钞） |
| **买债方式** | 主要通过 MLF（向银行） | 在二级市场（从投资者） |
| **是否买债** | ❌ 不直接买国债 | ✅ 大量买入国债 |
| **规模** | 5 万亿 | 4.5 万亿美元（2020） |
| **性质** | **结构性工具** | **总量工具** |

> **核心区别**：中国央行对冲是"绕道支持"（通过政策性银行、再贷款），美联储 QE 是"直接下场"（买债）。

详见 [[wiki/concepts/QE与化债对比]]。

## 七、央行的"红线"——绝不做的事

### 7.1 央行明确不做的

| 不做的事 | 原因 |
|---------|------|
| **❌ 直接买化债债** | 法律禁止、避免货币化 |
| **❌ 印钞给政府** | 避免恶性通胀 |
| **❌ 承担化债责任** | 避免道德风险 |
| **❌ 降低对实体经济的支持** | 保持货币政策的独立性 |

### 7.2 央行实际做的

| 做的事 | 性质 |
|--------|------|
| **✅ 通过结构性工具支持** | 间接支持 |
| **✅ 保持流动性充裕** | 总量调节 |
| **✅ 压低利率水平** | 引导市场 |
| **✅ 配合化债，但不兜底** | 政策协调 |

详见 [[wiki/concepts/财政货币化]] 与 [[wiki/concepts/化债]]。

## 八、对冲机制的金融学本质

### 8.1 本质公式

```
化债的"流动性冻结" + 央行"流动性投放" = 流动性新平衡

具体：
  化债冻结 4-5 万亿
  + 央行投放 4-5 万亿
  = 流动性基本平衡
  = M2 维持 8%-9% 增长
  = 银行体系流动性基本稳定
```

### 8.2 与"量化对冲"的类比

| 维度 | 量化对冲 | 央行对冲化债 |
|------|---------|------------|
| **主体** | 金融投资者 | 中央银行 |
| **目的** | 对冲市场风险 | 对冲流动性冻结 |
| **工具** | 衍生品 | 结构性货币工具 |
| **效果** | 锁定风险 | 保持流动性 |

## 九、未来展望——化债下半场的对冲工具

### 9.1 短期（2025-2026）

| 工具方向 | 内容 |
|---------|------|
| **继续降准** | 累计可能再降 0.5-1 个百分点 |
| **MLF 持续** | 保持中期流动性投放 |
| **结构性工具扩容** | 保障性住房再贷款可能扩大 |

### 9.2 中期（2027-2030）

| 工具方向 | 内容 |
|---------|------|
| **第三类债务处置** | 城投经营性债务可能需要新的结构性工具 |
| **特别国债** | 中央财政承担化债责任的潜在工具 |
| **政策性金融工具** | 进一步发挥政策性银行作用 |

### 9.3 长期（2030+）

| 工具方向 | 内容 |
|---------|------|
| **"日本化"风险** | 滚动续债常态化，结构性工具长期化 |
| **直接融资发展** | 减少对间接融资（银行）的依赖 |
| **财政体制改革** | 根本上解决地方债务问题 |

## 十、一句话总结

> **化债中央行对冲 = "5 万亿结构性投放"（降准 + MLF + PSL + 再贷款）**。
>
> **对冲的本质 = 通过政策性银行和结构性工具"绕道支持"**。
>
> **对冲的边界 = 不直接买债、不印钞给政府、不兜底化债**。
>
> **对冲的效果 = 部分有效**（M2 维持 8%-9% 增长，但低于历史水平）。
>
> **对冲的代价 = 银行净息差收窄 + 资本回报率下降**。
