# 国际收支

> 一国在一定时期内与外国发生的全部经济交易的系统记录，遵循复式记账原则：贷方 = 借方（会计恒等式恒成立）。

## 一、核心定义

### 1.1 概念本质

**国际收支**（Balance of Payments, BOP）是一国在特定时期内（季度/年度）与**境外居民**发生的：
- 实际资源流动（商品、服务、收入）
- 金融资产的买卖
- 单方面转移（援助、侨汇）

之**系统记录**。

> 注意：是**居民**对**非居民**的区分（按 IMF 居民定义），与国境不完全一致。

### 1.2 复式记账原则

每笔交易同时记**借方**（Debit）和**贷方**（Credit）：

| 记法 | 借方（Dr） | 贷方（Cr） |
|------|-----------|-----------|
| 商品 | **进口** | **出口** |
| 服务 | 服务进口（运输、旅游） | 服务出口 |
| 收入 | 收入支付（投资利润汇出） | 收入获得 |
| 金融 | 本国**负债增加**或本国**资产减少** | 本国**资产增加**或本国**负债减少** |

> **核心恒等式**：
> ```
> 经常账户 + 资本账户 + 金融账户 + 错误与遗漏 = 0
> ```
> 任何顺差或逆差必有对应项平衡（**会计恒等式**永远成立）。

### 1.3 顺差与逆差

| 概念 | 含义 | 典型后果 |
|------|------|---------|
| **顺差**（Surplus） | 资金净流入 / 资源净输出 | 外汇储备↑、本币升值压力 |
| **逆差**（Deficit） | 资金净流出 / 资源净输入 | 外汇储备↓、本币贬值压力 |

> 顺差≠绝对好事：日本 1990s 顺差但内需不振；美国长期逆差但维持全球金融中心地位。

## 二、BOP 三大账户

### 2.1 账户结构总览

```
国际收支平衡表
├── 一、经常账户 (Current Account)
│   ├── A. 商品（贸易）账户
│   ├── B. 服务账户
│   ├── C. 收入账户（要素收入）
│   └── D. 经常转移
├── 二、资本账户 (Capital Account)
│   └── 资本转移、非生产非金融资产交易
├── 三、金融账户 (Financial Account)
│   ├── A. 直接投资 (FDI)
│   ├── B. 证券投资
│   ├── C. 金融衍生品
│   ├── D. 其他投资（贷款、贸易信贷）
│   └── E. 储备资产
└── 四、错误与遗漏 (Errors & Omissions)
```

### 2.2 经常账户（Current Account）

最广为关注的账户，反映一国**国际收支的"贸易+收入"状况**。

```
经常账户 = 商品贸易 + 服务贸易 + 初次收入 + 经常转移
```

| 子项 | 内容 | 中国 2024 示例 |
|------|------|---------------|
| **商品** | 进出口货物 | +8500 亿美元（顺差） |
| **服务** | 运输、旅游、金融、知识产权 | -2300 亿美元（逆差） |
| **初次收入** | 投资利润、利息、工资 | -1500 亿美元（外企利润汇出） |
| **经常转移** | 侨汇、援助、捐赠 | -200 亿美元 |

> 经常账户顺差 = 国内储蓄 > 国内投资 = 净对外借出。

### 2.3 资本账户（Capital Account）

记录**非生产非金融资产**的转移（如专利、商标、特许权）和**资本转移**（债务豁免、移民转移）。

```
资本账户 = 资本转移 + 非生产非金融资产交易
```

> 现代 BOP 中资本账户规模**很小**（除转型经济体外）。

### 2.4 金融账户（Financial Account）

记录**金融资产与负债**的跨境交易，按功能分类：

| 子项 | 定义 | 典型例子 |
|------|------|---------|
| **直接投资 (FDI)** | ≥10% 股权获得控制权 | 中国企业收购海外公司 |
| **证券投资** | 股票、债券等证券交易 | QFII 买 A 股 |
| **金融衍生品** | 期货、期权、互换 | 套保头寸 |
| **其他投资** | 贷款、贸易信贷、货币/存款 | 银行跨境同业 |
| **储备资产** | 央行外汇储备、黄金、SDR | 央行干预市场 |

> 直接投资 vs 证券投资关键差异：**控制权**（10% 股权阈值）。

### 2.5 错误与遗漏（Errors & Omissions, E&O）

```
E&O = -(经常账户 + 资本账户 + 金融账户)
```

| 原因 | 解释 |
|------|------|
| **统计遗漏** | 地下钱庄、走私、现金携带 |
| **数据时差** | 进出口与收付款记录时间不同 |
| **离岸金融** | 避税地资金流难以追踪 |
| **汇率折算** | 多币种折算误差 |

> 经验法则：E&O 绝对值 > 经常账户绝对值的 5% 即偏大。

## 三、国际收支恒等式

### 3.1 基本恒等式

```
CA + KA + FA + E&O = 0

其中：CA 经常账户、KA 资本账户、FA 金融账户
```

### 3.2 储备资产变动

若忽略 E&O 和小 KA：

```
ΔReserves = -(CA + FA)

即：经常账户顺差 = 金融账户逆差 = 储备资产增加
```

**示例**：
- 经常账户顺差 +3000 亿美元
- 金融账户逆差 -2500 亿美元（净流出）
- E&O = 0
- → 储备资产增加 500 亿美元

### 3.3 会计恒等 vs 经济恒等

| 维度 | 会计恒等 | 经济意义 |
|------|---------|---------|
| CA + KA + FA + E&O = 0 | 永远成立 | 任何不平衡都有对应项 |
| 顺差/逆差 | 描述净流量 | 反映竞争力、储蓄-投资缺口 |
| 政策含义 | 无 | **分析意义**重大 |

## 四、顺差/逆差的宏观经济含义

### 4.1 经常账户的"宏观恒等"

```
CA = (S - I) + (T - G)
   = 私人储蓄 - 投资 + 政府储蓄
```

| 状态 | 含义 | 政策含义 |
|------|------|---------|
| **CA > 0（顺差）** | 国民储蓄 > 国民投资；净对外借出 | 内需不足或竞争力强 |
| **CA < 0（逆差）** | 国民储蓄 < 国民投资；净对外借入 | 投资过旺或消费过度 |

### 4.2 顺差的"双面性"

| 视角 | 解读 |
|------|------|
| **出口国** | 顺差 = 赚外汇、提振制造业、积累储备 |
| **顺差国** | 顺差 = 内需不足、资源输出（to 外国）、资产积累在外 |
| **美国逆差** | 全球金融中心地位，美元回流美债 |

> 日本教训：长期巨额顺差但内需疲软，被迫零利率。

### 4.3 不可持续的逆差

> **"双赤字"** = 财政赤字 + 经常账户逆差
> 警示信号：外债 / GDP 比率迅速上升

**国际警戒线参考**：
- 经常账户逆差 / GDP > 4% → 关注
- 短期外债 / 储备 > 1 → 危机风险高
- 储备覆盖进口月数 < 3 个月 → 危险

## 五、中国双顺差历史

### 5.1 "双顺差"定义

**经常账户顺差 + 资本与金融账户顺差** = 双顺差（同时记账层面"赚"和"借"）

### 5.2 中国 2000-2014 双顺差

| 时期 | 经常账户 | 资本与金融账户 | 外汇储备（年末） |
|------|---------|---------------|----------------|
| 2000 | +205 亿 | +20 亿 | 1656 亿 |
| 2005 | +1341 亿 | +912 亿 | 8189 亿 |
| 2008 | +4206 亿 | +463 亿 | 19460 亿 |
| 2014 | +2774 亿 | -504 亿 | 38430 亿（峰值） |
| 2018 | +241 亿 | -1730 亿 | 30727 亿 |
| 2023 | +2530 亿 | -1500 亿（估） | ~32000 亿 |

### 5.3 双顺差形成原因

| 原因 | 机制 |
|------|------|
| **人口红利** | 劳动密集型出口竞争力 |
| **储蓄率过高** | 居民储蓄率 35%+，投资率 45%+ |
| **汇率管制** | 强制结售汇（2008 前） |
| **FDI 大量流入** | 加工贸易 + "两头在外" |
| **国际产业转移** | 承接全球制造业 |

### 5.4 失衡代价

- **外汇占款**扩张 → 被动货币投放 → 通胀压力
- **人民币升值压力** → 出口压力
- **资产泡沫**（房地产、股市）
- **资本管制**强化 → 人民币国际化受阻

### 5.5 政策调整（"国际收支再平衡"）

- **2012** 起扩大人民币双向波动
- **2015.8.11** 汇改：参考一篮子货币定价
- **2017** 引入逆周期因子
- **鼓励资本输出**（"一带一路"、QDII 扩容）

## 六、外汇储备

### 6.1 构成

| 资产 | 占比（全球平均） |
|------|----------------|
| 外国国债 | ~65% |
| 外国机构债 | ~10% |
| 黄金 | ~10% |
| IMF 储备头寸 + SDR | ~5% |
| 其他 | ~10% |

### 6.2 功能

| 功能 | 说明 |
|------|------|
| **干预外汇市场** | 抛售美元 → 稳定本币 |
| **支付进口** | 3-6 个月进口覆盖 |
| **偿还外债** | 短期外债覆盖 |
| **市场信心** | 高储备 = 抗冲击能力 |
| **国际货币** | SDR 篮子、IMF 份额 |

### 6.3 适度规模

**格林斯潘-古德哈特规则**（经验）：
- 储备 / 进口 ≥ 3 个月
- 储备 / 短期外债 ≥ 1
- 储备 / M2 ≈ 10-20%

> 中国储备曾达 4 万亿美元（2014），过于庞大意味着**资源错配**（购入低收益美债）。

## 七、特里芬难题

### 7.1 概念

1960 年 Robert Triffin 提出**美元的两难**：

> 若美国**国际收支顺差**（输出美元）→ 美元稀缺 → 国际流动性不足
> 若美国**国际收支逆差**（输出美元）→ 美元泛滥 → 美元贬值、储备货币信心动摇

```
       美元作为全球储备货币
                │
     ┌──────────┴──────────┐
     │                     │
  顺差（稀缺）           逆差（贬值）
  → 流动性不足           → 信心动摇
```

### 7.2 历史验证

- **1971** 尼克松关闭黄金兑换窗口
- **1973** 布雷顿森林体系崩溃
- **1979-1985** 美元危机与广场协议
- **2010s** "特里芬难题"在人民币国际化讨论中复活

### 7.3 现代版本

> 任何主权货币作为国际储备货币，都面临类似的"两难"。

中国推动**人民币国际化**时必须面对：如果人民币成为储备货币，中国必须接受长期逆差——这与中国传统的"双顺差"政策矛盾。

## 九、关键易错点

| 误区 | 正确认识 |
|------|----------|
| 顺差 = 强国 | 日本、德国长期顺差但内需疲软 |
| 逆差 = 弱国 | 美国长期逆差但维持全球霸权 |
| 双顺差 = 强大 | 实际是结构失衡，资源错配 |
| BOP 永远平衡 | 仅**会计恒等**平衡；子账户有顺逆 |
| 储备越多越好 | 超过必要规模是**资源浪费**（机会成本） |
| 特里芬难题只针对美元 | 任何主权储备货币都面临 |
