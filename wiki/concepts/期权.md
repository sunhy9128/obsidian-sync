# 期权（Options）

> 在未来某时以约定价格买卖资产的权利 —— 金融衍生品的"皇冠明珠"

## 一、定义

**期权**（Options）是一种**金融衍生品**，赋予买方在**未来某个特定时间**（到期日），以**事先约定的价格**（行权价）**买入或卖出**某种**标的资产**的**权利**。

核心要素：
- **标的资产**：股票、指数、商品、外汇等
- **到期日**：合约到期的具体时间
- **行权价**：约定的买卖价格
- **权利金**：买方购买期权的成本
- **方向**：看涨（Call）或看跌（Put）

> **核心比喻**：期权是"**金融的保险**"——买方买的是"**权利**"（不是义务），付出的是"**保险费**"（权利金）。如果事情发生，可以"行权获益"；如果没发生，最多损失"保险费"。

## 二、期权的"四种基本类型"

### 2.1 看涨期权（Call Option）

```
定义：
  赋予买方在到期日（或之前）以行权价**买入**标的资产的权利
  
  买方付出权利金
  获得"以行权价买入"的权利
  
  适用场景：看涨标的资产

例：
  X 股票当前 100 元
  某 Call 期权：行权价 105 元，3 个月到期，权利金 3 元
  
  到期时股价 = 110 元：
    行权盈利 = 110 - 105 = 5 元
    净盈利 = 5 - 3 = 2 元（扣除权利金）
  
  到期时股价 = 100 元：
    不行权（无意义）
    净亏损 = -3 元（损失权利金）
  
  到期时股价 = 95 元：
    不行权
    净亏损 = -3 元
```

### 2.2 看跌期权（Put Option）

```
定义：
  赋予买方在到期日（或之前）以行权价**卖出**标的资产的权利
  
  买方付出权利金
  获得"以行权价卖出"的权利
  
  适用场景：看跌标的资产或对冲

例：
  X 股票当前 100 元
  某 Put 期权：行权价 95 元，3 个月到期，权利金 2 元
  
  到期时股价 = 90 元：
    行权盈利 = 95 - 90 = 5 元
    净盈利 = 5 - 2 = 3 元
  
  到期时股价 = 100 元：
    不行权
    净亏损 = -2 元
  
  到期时股价 = 110 元：
    不行权
    净亏损 = -2 元
```

### 2.3 欧式期权 vs 美式期权

| 维度 | 欧式期权 | 美式期权 |
|------|---------|---------|
| **行权时间** | 仅到期日 | 到期日**或之前** |
| **灵活性** | 较低 | 较高 |
| **权利金** | 较低 | 较高（多一份灵活性）|
| **主要市场** | 欧洲 | 美国 |
| **中国情况** | 中国主流（股指期权）| 较少 |

### 2.4 场内期权 vs 场外期权

| 维度 | 场内期权 | 场外期权 |
|------|---------|---------|
| **标准化** | 完全标准化 | 定制化 |
| **流动性** | 高 | 低 |
| **信用风险** | 交易所承担 | 双方承担 |
| **典型品种** | 50ETF 期权、沪深 300 期权 | 大宗商品期权、奇异期权 |

## 三、期权的"六大要素"

### 3.1 要素一：标的资产

```
期权可基于：
  - 个股（Apple、Tesla 等）
  - 指数（S&P 500、沪深 300）
  - 商品（原油、黄金）
  - 外汇（美元、欧元）
  - 利率（国债）
  - 加密货币（Bitcoin）
```

### 3.2 要素二：行权价（Strike Price）

```
行权价 vs 现货价的关系：

1. **平值（At The Money, ATM）**
   行权价 ≈ 现货价
   时间价值最大

2. **实值（In The Money, ITM）**
   Call：行权价 < 现货价
   Put：行权价 > 现货价
   内在价值为正

3. **虚值（Out of The Money, OTM）**
   Call：行权价 > 现货价
   Put：行权价 < 现货价
   内在价值为 0
```

### 3.3 要素三：到期日

```
期权的时间维度：
  - 周权（1 周内）
  - 月权（1-3 个月）
  - 季权（3-6 个月）
  - 年权（1 年以上）
  - LEAPS（长期期权，2-3 年）

关键：到期日越远 → 权利金越高（有时间价值）
```

### 3.4 要素四：权利金（Premium）

```
权利金 = 内在价值 + 时间价值

内在价值：
  Call: max(0, 现货价 - 行权价)
  Put: max(0, 行权价 - 现货价)

时间价值：
  期权价值超出内在价值的部分
  反映"未来不确定性"

例：
  Call 行权价 100 元，现货 105 元
  权利金 8 元
  内在价值 = 5 元（105-100）
  时间价值 = 3 元（8-5）
```

### 3.5 要素五：方向（Call/Put）

```
Call：看涨，买方期待价格上涨
Put：看跌，买方期待价格下跌

买方最大损失 = 权利金（有限）
买方最大收益 = 无限（Call）/ 大（Put）
卖方最大收益 = 权利金（有限）
卖方最大损失 = 无限（Call）/ 大（Put）
```

### 3.6 要素六：合约乘数

```
中国股指期权合约乘数：
  - 沪深 300 股指期权：100 元/点
  - 中证 1000 股指期权：100 元/点

例：
  沪深 300 股指期权权利金 = 50 点
  合约价值 = 50 × 100 = 5000 元
```

## 四、期权定价——Black-Scholes 模型

### 4.1 Black-Scholes 公式

```
C = S × N(d1) - K × e^(-rT) × N(d2)

其中：
  C = Call 期权理论价格
  S = 标的资产当前价格
  K = 行权价
  T = 到期时间
  r = 无风险利率
  N = 标准正态分布累计概率
  d1, d2 = 复杂计算项（涉及波动率）
```

### 4.2 五个关键变量

```
Black-Scholes 模型的五个关键变量：
  1. 标的资产价格 S
  2. 行权价 K
  3. 时间 T
  4. 利率 r
  5. 波动率 σ

其中，**波动率**是 Black-Scholes 模型无法预测的"隐藏变量"
也是动态对冲的"目标"
```

### 4.3 模型的局限性

```
Black-Scholes 模型的假设：
  - 标的价格服从对数正态分布
  - 波动率恒定
  - 无套利机会
  - 连续交易
  - 无交易成本
       ↓
局限性：
  - 极端市场波动率"尖峰厚尾"
  - "波动率微笑"现象
  - 跳空缺口无法预测
  - 流动性枯竭
```

## 五、期权的"四大功能"

### 5.1 功能一：保险（Hedging）

```
持有 1000 股 X 股票（@100 元）
担心下跌
       ↓
操作：
  买入 Put 期权（行权价 95 元，3 个月）
  支付权利金 3 元/股 × 1000 = 3000 元
       ↓
效果：
  股价跌到 90 元：
    现货损失：10 元 × 1000 = -10000 元
    Put 期权盈利：5 元 × 1000 = +5000 元
    净损失：-5000 元（优于 -10000 元）
  
  股价涨到 110 元：
    现货盈利：10 元 × 1000 = +10000 元
    Put 期权亏损：-3000 元（权利金）
    净盈利：+7000 元（仍能获利）
       ↓
本质上：
  - 保险费 = 3000 元
  - 保护范围 = 95 元以下
  - 保护期 = 3 个月
```

### 5.2 功能二：投机（Speculation）

```
看涨 X 股票从 100 元涨到 110 元
       ↓
方案 A：买入股票
  - 成本：100 × 1000 = 10 万元
  - 盈利：10 × 1000 = 1 万元
  - 收益率：10%

方案 B：买入 Call 期权
  - 成本：3 × 1000 = 3000 元
  - 盈利：5 × 1000 = 5000 元
  - 收益率：166%
       ↓
杠杆效应：
  期权的杠杆率 = 现货盈利 / 期权盈利
  = 1 万 / 5 千 = 2 倍
  
  但风险也不同：
  - 股票可能继续涨
  - 期权可能损失全部权利金
```

### 5.3 功能三：增强收益（Income）

```
持有 1000 股 X 股票（@100 元）
对短期上涨预期不强
       ↓
操作：
  卖出 Call 期权（行权价 105 元，3 个月）
  收取权利金 2 元/股 × 1000 = 2000 元
       ↓
效果：
  股价 100 元（不变）：
    期权费收入：+2000 元
    持仓收益：0
    总收益：+2000 元（额外收益）
  
  股价涨到 110 元：
    现货盈利：10 × 1000 = 1 万元
    期权被行权：损失 5 × 1000 = -5000 元
    期权费：+2000 元
    净盈利：7000 元（仍比单纯持有好，但少了上行空间）
  
  股价跌到 90 元：
    现货损失：-10 × 1000 = -1 万元
    期权不行权
    期权费：+2000 元
    净损失：-8000 元
       ↓
本质：
  卖出期权 = "卖保险"
  赚取"保险费"
  但承担"保险事故"的风险
```

### 5.4 功能四：组合策略（Spread）

```
常见组合：
  - 牛市价差（Bull Spread）：买低行权价 Call + 卖高行权价 Call
  - 熊市价差（Bear Spread）：买高行权价 Put + 卖低行权价 Put
  - 跨式（Straddle）：买同行权价 Call + Put
  - 宽跨式（Strangle）：买虚值 Call + 虚值 Put
  - 蝶式（Butterfly）：3 个行权价组合
  - 鹰式（Condor）：4 个行权价组合
       ↓
这些策略的核心：
  - 用期权组合表达"市场观点"
  - 控制最大损失和最大收益
  - 适应不同的"波动率预期"
```

## 六、期权的"风险与收益"

### 6.1 买方（Long）

```
最大损失：权利金（有限）
最大收益：
  - Call：无限（理论上）
  - Put：行权价 × 数量 - 权利金

特征：
  - "风险有限，收益可能无限"
  - 适合：保守型投机、对冲需求
  - 时间是敌人（Theta 损失）
```

### 6.2 卖方（Short）

```
最大收益：权利金（有限）
最大损失：
  - Call：无限（理论上）
  - Put：行权价 × 数量 + 权利金

特征：
  - "收益有限，风险可能无限"
  - 适合：增强收益、卖保险
  - 时间是朋友（Theta 收益）
  - 风险高，需要保证金
```

### 6.3 期权的"非线性收益"

```
Call 买方到期收益图：

  收益
   ↑          /
   |         /
 2 |        /   ← 最大收益（无限）
   |       /
 0 |─────/───── 行权价 100
   |    /
-3 |───/   ← 最大损失 = 权利金
   └──────────→ 股价
      95   100   105   110

非线性 = 期权的本质
也是期权定价和动态对冲的"难点"
```

## 七、期权的"Greeks"—— 风险管理核心

### 7.1 Delta（Δ）

```
Delta = 期权价格变动 / 标的价格变动

含义：期权对标的价格的敏感度
范围：
  - Call：0 到 +1
  - Put：-1 到 0
  - 深度实值：|Delta| 接近 1
  - 深度虚值：|Delta| 接近 0
  - 平值：|Delta| 约 0.5
```

### 7.2 Gamma（Γ）

```
Gamma = Delta 变动 / 标的价格变动

含义：Delta 变化的"速度"
特征：
  - 平值期权 Gamma 最大
  - 虚值/实值期权 Gamma 较小
  - 临到期平值期权 Gamma 接近无穷
       ↓
Gamma 风险：
  期权做市商最大风险
  动态对冲的成本来源
```

### 7.3 Vega（ν）

```
Vega = 期权价格变动 / 波动率变动

含义：期权对"波动率"的敏感度
特征：
  - 看涨和看跌期权 Vega 都是正
  - 平值期权 Vega 最大
  - 长期期权 Vega 较大
       ↓
Vega 风险：
  跨期权品种套利的核心风险
  隐含波动率变化的损失
```

### 7.4 Theta（Θ）

```
Theta = 期权价格变动 / 时间变动

含义：期权的"时间衰减"
特征：
  - 几乎所有期权 Theta 都是负（买方）
  - 临到期衰减加速
  - 平值期权 Theta 衰减最快
       ↓
Theta 风险：
  长期持有期权的"磨损"
  卖方"赚时间价值"
```

### 7.5 Rho（ρ）

```
Rho = 期权价格变动 / 利率变动

含义：期权对利率的敏感度
特征：
  - Call：正（利率上升 → Call 升值）
  - Put：负（利率上升 → Put 贬值）
  - 长期期权对利率更敏感
       ↓
Rho 风险：
  利率大幅变动时的对冲
  长期期权的核心风险之一
```

详见 [[wiki/concepts/希腊字母|希腊字母]]。

## 八、期权的"做市商业务"

### 8.1 做市商的角色

```
期权做市商：
  - 提供买卖报价
  - 赚取买卖价差
  - 承担方向性风险
       ↓
做市商的策略：
  - 持续报价
  - 动态对冲
  - 管理 Greeks
  - 赚取 Theta（时间价值）
```

### 8.2 做市商的风险

```
期权做市商面临的风险：

1. Gamma 风险（最大）
   - Delta 变化太快
   - 难以持续对冲

2. Vega 风险
   - 隐含波动率变化
   - 难以预测

3. Theta 风险
   - 短期波动影响
   - 时间价值损耗

4. 流动性风险
   - 极端市场无法平仓

5. 模型风险
   - 定价模型失效
   - 极端事件损失
```

## 十一、期权与"风险管理"

### 11.1 期权的"四个独特性"

```
1. 非线性收益
   - 风险曲线不对称
   - 提供"凸性"保护

2. 时间价值
   - 长期持有有时间成本
   - 卖方赚时间价值

3. 波动率维度
   - 期权可单独交易波动率
   - 波动率本身成为资产

4. 灵活组合
   - 多种行权价/到期月
   - 适应不同观点
```

### 11.2 期权 vs 期货 vs 现货

| 维度 | 现货 | 期货 | 期权 |
|------|------|------|------|
| **杠杆** | 1 倍 | 5-20 倍 | 5-100 倍 |
| **风险** | 无限 | 无限 | **有限**（买方）|
| **方向** | 单向 | 双向 | 双向 |
| **行权** | 即时 | 强制 | **权利** |
| **时间** | 无限 | 到期 | 到期 |
| **成本** | 全额 | 保证金 | 权利金 |
| **适合** | 长期投资 | 套保/投机 | 套保/投机/增强收益 |

## 十二、期权的"金融学意义"

### 12.1 期权的"价值发现"

```
期权市场是"波动率的价格发现"：

波动率：
  - 反映市场对未来不确定性的预期
  - VIX 指数（恐慌指数）
  - 隐含波动率 vs 实际波动率
  
  期权市场：
  - 隐含波动率 = 未来预期的"价格"
  - 反映市场情绪
  - 预测市场方向
```

### 12.2 期权的"风险管理革命"

```
期权改变了风险管理的方式：

无期权时代：
  - 风险完全暴露
  - 无法精细化管理
  - 只能"硬扛"或"卖出"

期权时代：
  - 风险可以"切分"
  - 单独交易方向风险
  - 单独交易波动率风险
  - 单独交易时间风险
  - 精细化对冲
```

## 十三、期权与"投机策略"

### 13.1 投机策略分类

```
按方向分类：
  - 看涨策略：买 Call、卖 Put
  - 看跌策略：买 Put、卖 Call
  - 中性策略：跨式、宽跨式

按波动率分类：
  - 预期波动率上升：买跨式、买宽跨式
  - 预期波动率下降：卖跨式、卖宽跨式

按时间分类：
  - 短期：周权、月权
  - 长期：年权、LEAPS
```

### 13.2 投机的"风险警示"

```
期权投机的"三大风险"：

1. 时间衰减（Theta）
   - 长期持有 = 持续损失
   - 临到期衰减加速

2. 波动率变化（Vega）
   - 隐含波动率变化
   - 即使方向对，也可能亏钱

3. 流动性风险
   - 虚值期权流动性差
   - 临到期可能无法平仓
       ↓
建议：
  期权不是"无脑赚钱"的工具
  需要深入理解
```

## 十四、期权与"对冲策略"

### 14.1 保护性 Put（Protective Put）

```
场景：持有股票担心下跌

操作：买入 Put 期权
       ↓
效果：
  - 股价下跌 → Put 盈利
  - 股价上涨 → 损失权利金，但股票盈利
  - 最大损失：行权价 - 现货价 + 权利金
  - 最大收益：无限（理论上）

本质："保险" = 买 Put
```

### 14.2 备兑 Call（Covered Call）

```
场景：持有股票，不强烈看涨

操作：卖出 Call 期权
       ↓
效果：
  - 股价不涨 → 赚权利金
  - 股价小涨 → 仍能赚（但损失上行空间）
  - 股价大涨 → 期权被行权，损失上行空间
  - 股价大跌 → 损失 + 权利金部分抵消

本质："卖保险" = 卖 Call
```

### 14.3 领口策略（Collar）

```
场景：持有股票，担心下跌但不愿付高额保险费

操作：
  - 买 Put（保护下行）
  - 卖 Call（补贴 Put 成本）
       ↓
效果：
  - 下行被锁定
  - 上行也被锁定
  - 净成本 ≈ 0
       ↓
本质：零成本保护 = 领口策略
```

详见 [[wiki/concepts/动态对冲|动态对冲]]。

## 十五、一句话总结

> **期权 = 在未来某时以约定价格买卖资产的权利，本质是"金融的保险"。**
>
> 它通过**非线性收益 + 时间价值 + 波动率维度**，让风险管理变得**精细化、模块化、可交易**。期权买方"风险有限、收益可能无限"，卖方"收益有限、风险可能无限"。
>
> **期权时代**改变了金融市场的面貌——它让**方向、波动率、时间**成为可独立交易的对象。**LTCM 的崩溃**和**1987 黑色星期一**告诉我们：**期权的力量是巨大的，但风险也是巨大的**。**好的期权策略不是赌博，而是精细化的风险管理**。
