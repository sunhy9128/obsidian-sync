# 万科宝能之争

## 概述

2015-2017年万科宝能股权之争是中国资本市场最典型的控制权争夺战，展现了[[中国特色企业治理]]与现代资本市场的深层矛盾。

## 三方博弈格局

| 阵营 | 代表 | 持股比例（停牌前） |
|------|------|-------------------|
| 防守方 | 王石管理层 + 安邦 | 约28% |
| 进攻方 | 宝能系（姚振华） | 24.26% |
| 原大股东 | 华润（傅育宁） | 15.29% |

## 战争时间线

| 时间 | 事件 |
|------|------|
| 2014年底 | 宝能系开始研究万科，筹备资金 |
| 2015年7月11日 | 宝能首次举牌，持股5%，耗资79亿 |
| 2015年8月27日 | 宝能持股15.04%，超过华润14.89% |
| 2015年8月31日 | 华润增持至15.29%，微弱反超 |
| 2015年11月27日 | 宝能再次超过华润 |
| 2015年12月7日 | 宝能持股达20.008%，华润装死 |
| 2015年12月8日 | 安邦举牌5%，加入防守方 |
| 2015年12月18日 | 万科紧急停牌，股价24.43元 |

## 核心矛盾

1. **王石的小算盘**：作为无股份的管理层，希望维持控制权
2. **打压股价暗中增持**：管理层可能一边打压股价一边增持
3. **宝能识破**：姚振华看穿万科的防守漏洞
4. **华润袖手**：关键时刻原大股东不作为

## 关键教训

1. **杠杆是双刃剑**：宝能用高杠杆撬动收购，但也受制于时间成本
2. **停牌是把利器**：可以拖延时间，让杠杆资金承压
3. **政治因素超越商业**：华润的袖手暗示了更深层的博弈
4. **控制权可以脱离股权**：中国特色治理的典型特征

## 事件影响

- 成为中国资本市场控制权争夺的经典案例
- 揭示了[[中国特色企业治理]]与现代资本市场的矛盾
- 推动了中国并购监管政策的完善
