# 借股票池

> 卖空机制的"咽喉" —— 多空博弈中真正决定胜负的资源

## 一、定义

**借股票池**（Securities Lending Pool）是指**金融机构、机构投资者、ETF、大股东**等将自己持有的股票**借出**给卖空者的**总供给池**。它是**卖空机制的关键约束**，决定了：

- 能卖空多少（数量约束）
- 卖空成本多高（价格约束）
- 能卖空多久（时间约束）

> **核心洞察**：卖空是"时间的朋友"的前提是"能借到股票"。"借不到股票"会让空头成为时间的朋友的反面——**时间的敌人**。

## 二、借股票池的构成

### 2.1 借出方构成

| 来源 | 占比 | 借出动机 |
|------|------|---------|
| **机构投资者** | ~60% | 赚取借股票费（被动收入）|
| **ETF 基金** | ~15% | 优化基金收益 |
| **养老基金/保险** | ~15% | 套利 + 流动性管理 |
| **大股东** | ~5% | 套利（保底收益）|
| **券商自营** | ~3% | 做市 + 库存管理 |
| **散户** | ~2% | 极少数（部分券商允许）|

### 2.2 借出方动机

#### 动机一：赚取"借股票费"

```
机构持有 1000 万股 X 股票
长期持有（无短期卖出计划）
↓
借给卖空者（年化 1% 借股票费）
↓
每年白赚 100 万 × 1% = 1 万
（对机构是不小的"零风险收益"）
```

#### 动机二：风险对冲

```
持有大量 X 股票
担心短期下跌但不想卖出（避免税务）
↓
借给卖空者
↓
如果 X 真的下跌，借股票费可以部分抵消
```

#### 动机三：流动性管理

```
机构需要在短期内周转资金
但又不想卖出持仓
↓
借出股票获得现金或抵押品
↓
解决短期流动性问题
```

### 2.3 借出方担忧

| 担忧 | 说明 |
|------|------|
| **股票被召回** | 借出后股价大涨，借出方也想买入 |
| **股票被恶意操纵** | 借出后被用于轧空 |
| **税务问题** | 借出可能涉及税务处理 |
| **信息披露** | 借出是否需公开披露 |

## 三、借股票费（Stock Loan Fee）—— 关键指标

### 3.1 借股票费的形成机制

```
借股票费 = 基础利率（通常 SOFR）+ 股票稀缺溢价

其中：
  - 基础利率：反映货币市场利率水平
  - 股票稀缺溢价：反映"借股票难易程度"
```

### 3.2 借股票费水平的含义

| 借股票费 | 含义 | 市场状态 |
|---------|------|---------|
| < 0.5% | 借股票容易 | 卖空者天堂 |
| 0.5%-2% | 正常 | 普通市场 |
| 2%-10% | 偏紧 | 卖空成本上升 |
| 10%-50% | 紧张 | 卖空者警惕 |
| 50%-100% | 极度紧张 | 轧空风险高 |
| > 100% | **轧空可能随时发生** | **GME 2021 时段** |

### 3.3 借股票费的极端案例

```
2021 年 1 月 GameStop 轧空期间：
  - 1 月 25 日：GME 借股票费 80% 年化
  - 1 月 27 日：GME 借股票费 600% 年化
  - 部分时段：1000%+ 年化

含义：
  - 借 100 万美元做空 GME，每年需付 600-1000 万美元借股票费
  - 这本身就让做空者"流血"（即使不爆仓）
```

## 四、借股票池的关键机制

### 4.1 借股票的三种状态

#### 状态一：可借

```
借股票池内有可用股票
借股票费正常
卖空者可正常借入
```

#### 状态二：稀缺

```
借股票池内可用股票少
借股票费飙升
新卖空者难以进入
已有空头面临"无股票可借"
```

#### 状态三：枯竭

```
借股票池内无可用股票
借股票费可能无意义（有钱也借不到）
新卖空者无法进入
已有空头面临"必须平仓"
```

### 4.2 借股票的"召回"（Recall）

**借出方可以召回借出的股票**：
- 通常需要提前通知（1-3 天）
- 借出方可以以"我看涨了"为由召回
- 召回后借出方可以：
  - 卖出自有股票获利
  - 重新出借给别人
  - 持有等待

> **召回的杀伤力**：召回后空头必须立即在市场上买入股票平仓。如果大量借出方同时召回，可能触发轧空。

### 4.3 借股票的期限与续借

| 期限类型 | 说明 |
|---------|------|
| **隔夜借** | 通常可续借 |
| **定期借** | 有固定期限，到期需归还 |
| **永久借** | 极少，需特殊安排 |
| **可召回** | 任何时候可召回 |

> **关键**：即使没有"到期"问题，借出方也可以随时**召回**（"可召回"是默认设置）。

## 五、借股票池与多空博弈

### 5.1 借股票池的"水资源"模型

```
把借股票池比作水库：
  - 水库容量 = 可借股票总量
  - 用水量 = 卖空头寸
  - 水位 = 借股票费
  
  水库满（水位低）→ 借股票费低 → 卖空者多
  水库空（水位高）→ 借股票费高 → 卖空者少
  水库枯竭 → 借股票费无意义 → 强制平仓
```

### 5.2 多方"锁仓"的威力

```
多方锁仓 = 减少水库供水
  ↓
水库水位急升（借股票费飙升）
  ↓
卖空者成本急增
  ↓
卖空者无法借到新股票
  ↓
已有卖空者被迫平仓
  ↓
平仓买入推高价格
  ↓
更多借出方召回股票
  ↓
水库进一步枯竭
  ↓
轧空正反馈
```

### 5.3 一个关键公式

```
可卖空量 = 借股票池容量 × 流通比例

借股票池容量 = 机构持仓 - 已借出 - 不愿借出
```

## 六、借股票市场的真实规模

### 6.1 全球借股票市场规模

| 指标 | 数据 |
|------|------|
| 全球借股票市场总规模 | ~$3-4 万亿 |
| 美国市场 | ~$1.5 万亿 |
| 欧洲市场 | ~$1 万亿 |
| 亚太市场 | ~$0.5-0.8 万亿 |

> **数据来源**：国际证券借贷协会（ISLA）估算

### 6.2 借股票费市场分层

| 资产类型 | 借股票费水平 | 流动性 |
|---------|------------|--------|
| **大盘蓝筹股** | 0.1%-1% | 极好 |
| **中盘股** | 1%-5% | 好 |
| **小盘股** | 5%-20% | 一般 |
| **波动大的小盘股** | 20%-100%+ | 紧张 |
| **特殊事件股票** | 100%+ | 几乎无法借到 |

## 七、借股票池与监管

### 7.1 监管的关注点

| 监管点 | 说明 |
|--------|------|
| **市场操纵** | 防止借股票池被用于操纵 |
| **系统性风险** | 防止借股票市场崩溃引发危机 |
| **公平性** | 防止内部人优先借股票 |
| **透明度** | 借股票数据和头寸的披露 |

### 7.2 关键监管规则

- **Regulation SHO**（美国 SEC）：规范卖空
- **提价规则**（uptick rule）：卖空价格必须 ≥ 上一个成交价
- **裸卖空禁令**：必须实际借到股票
- **披露要求**：卖空头寸需定期披露

## 八、借股票池与 1998 香港救市

### 8.1 金管局的"锁仓战术"

```
1998 年 8 月：
  金管局大量买入恒指成分股
        ↓
  关键操作：买入后**长期持有**（不卖）
        ↓
  流通股大幅减少
        ↓
  借股票池萎缩
        ↓
  国际空头**找不到股票借**来开新空头
        ↓
  已有空头面临**必须买回**的压力
        ↓
  8 月 28 日期货结算日 → 轧空
        ↓
  国际空头爆仓
```

> **本质**：金管局的"长期持有"就是一种**最大化的轧空姿态**——让国际空头永远找不到"释放的股票"。

详见 [[wiki/concepts/1998香港金融保卫战|1998 香港金融保卫战]]。

## 九、借股票池与 GameStop 案例

### 9.1 GME 借股票池枯竭过程

```
2020 年：
  GME 卖空头寸达到 ~140% 流通市值
  借股票池已经极度紧张
  借股票费开始飙升

2021 年 1 月：
  WallStreetBets 散户大量买入
  借股票池快速枯竭
  借股票费 100%-1000%+ 年化

2021 年 1 月 27 日：
  散户**拒绝借出**手中的 GME 股票
  借股票池几乎完全枯竭
  对冲基金**无法找到股票平仓**
  → 轧空完成
```

详见 [[wiki/concepts/GameStop轧空事件]]。

## 十、借股票池的"前向指标"作用

借股票池的数据是**判断轧空风险**的关键前向指标：

| 指标 | 阈值 | 信号 |
|------|------|------|
| 借股票费 | > 10% 年化 | 紧张 |
| 借股票费 | > 50% 年化 | 极度紧张，轧空可能 |
| 卖空比例 | > 20% | 卖空头寸过多 |
| Days to Cover | > 5 天 | 平仓困难 |
| 股权集中度 | 高 | 散户/坚定持有者多 |

## 十一、借股票池与流动性

### 11.1 借股票市场的"流动性悖论"

```
正常市场：
  借股票多 → 卖空多 → 流动性好
  （借股票市场活跃 = 流动性好的标志）

危机市场：
  借股票枯竭 → 卖空受限 → 流动性差
  （借股票市场枯竭 = 流动性危机的信号）
```

### 11.2 2008 金融危机的教训

```
2008 年：
  借股票市场冻结
  → 对冲基金无法做空（即使想做空）
  → 价格发现机制失效
  → 资产价格扭曲
  → 市场失灵
```

> **关键洞察**：借股票市场的"过度活跃"和"过度萎缩"都是危险的。健康的借股票市场应该是"适度活跃"。

## 十二、一句话总结

> **借股票池 = 多空博弈的"水库"**。
>
> 多方锁仓 = 减少供水（让水库干涸）→ 借股票费飙升 → 卖空者"渴死"。
> 空头布局 = 消耗供水（让水库见底）→ 借股票费上升 → 卖空者"提前撤退"。
>
> **借股票池是多空博弈的真正战场**。在卖空和轧空的舞台上，借股票费就是"战场温度计"。
