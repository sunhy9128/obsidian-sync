# 2020 年 3 月流动性危机

> COVID-19 冲击 + 风险平价踩踏 + 美元荒 + 美联储"无限 QE" = 21 世纪最戏剧性的流动性危机

## 一、事件概要

**2020 年 3 月流动性危机**（March 2020 Liquidity Crisis）发生于 **2020 年 3 月**，由 **COVID-19 全球大流行**触发，叠加**风险平价基金的群体性去杠杆**，导致全球金融市场陷入**流动性枯竭**和**资产价格暴跌**。这是 21 世纪以来最严重的流动性危机，最终被**美联储的"无限 QE"**化解。

关键数据：
- 美股 10 天内 4 次熔断（历史仅 1 次）
- 标普 500 从 3386 点跌至 2191 点（-35%）
- 美元指数从 95 飙升至 103（美元荒）
- 美联储资产负债表从 4.2 万亿扩张至 7.2 万亿
- 风险平价基金平均回撤 25%+

## 三、危机爆发的"五大原因"

### 3.1 原因一：COVID-19 外部冲击

```
COVID-19 的影响：

1. 供应链中断
   - 中国制造业停摆
   - 全球产业链冲击

2. 需求骤降
   - 消费者居家
   - 服务业崩溃

3. 不确定性
   - 不知道疫情持续多久
   - 不知道经济影响多大
   - 不知道政策应对

4. 财政不确定性
   - 各国财政政策不明
   - 救助计划未确定
```

### 3.2 原因二：风险平价基金的"群体行为"

```
风险平价策略（Risk Parity）：
  - 动态调整股票/债券比例
  - 维持风险贡献相等
  - 跨资产类别的动态对冲
       ↓
问题：
  - 大量基金使用类似策略
  - 极端市场 → 同时抛售
  - "群体踩踏"
       ↓
2020 年 3 月：
  - 风险平价基金平均回撤 25%+
  - Bridgewater All Weather 损失 20%+
  - 被动去杠杆
  - 加剧市场抛压
```

### 3.3 原因三：股票债券相关性"失灵"

```
传统认知：
  - 股票下跌 → 债券上涨
  - 股票债券负相关
  - 风险平价依赖此假设
       ↓
2020 年 3 月：
  - 股票下跌（COVID 恐慌）
  - 债券也下跌（流动性需求）
  - 风险平价策略失效
  - 被动去杠杆
  - 加剧抛售
       ↓
这就是"all assets correlation → 1"现象：
  - 极端市场
  - 所有风险资产同跌
  - 风险分散失效
```

### 3.4 原因四：美元荒（Dollar Shortage）

```
美元荒的形成：

1. 全球美元需求激增
   - 海外投资者抛售资产换美元
   - 美国企业海外资金回流
   - 美元避险需求

2. 美元供给中断
   - 银行不愿提供美元流动性
   - 美元货币市场基金停止赎回
   - 美元互换基差扩大

3. 美元指数飙升
   - 从 95 飙升至 103
   - 创近年新高

4. 新兴市场危机
   - 美元债务难以偿还
   - 资金大量流出
   - 部分国家接近违约
```

### 3.5 原因五：流动性枯竭

```
流动性枯竭的表现：

1. 买卖价差扩大
   - 平时 0.01% → 危机时 0.1%+
   - 做市商拒绝提供流动性

2. 信用利差扩大
   - 投资级公司债利差
   - 从 1% → 4%+
   - 高收益债利差
   - 从 3% → 10%+

3. 国债流动性异常
   - 平时最安全的资产
   - 2020 年 3 月出现"流动性问题"
   - 罕见
```

## 五、风险平价基金（Risk Parity）详解

### 5.1 风险平价的定义

```
风险平价（Risk Parity）：
  一种资产配置策略
  目标：让不同资产对组合的"风险贡献"相等
  而非"资金权重"相等
       ↓
经典组合：
  - 30% 股票
  - 60% 债券（加杠杆）
  - 10% 商品
  
  通过杠杆调整，使各类资产风险贡献相等
  - 股票风险贡献：33%
  - 债券风险贡献：33%
  - 商品风险贡献：33%
```

### 5.2 风险平价的核心机制

```
风险平价的"动态调整"：

风险平价基金：
  - 持续监控各类资产风险
  - 调整杠杆和配置
  - 维持目标风险贡献
       ↓
具体操作：
  - 股票波动率上升 → 减少股票
  - 债券波动率上升 → 减少债券
  - 调整杠杆
       ↓
与传统组合的区别：
  - 60/40 组合：固定比例
  - 风险平价：动态调整风险
```

### 5.3 2020 年 3 月的"踩踏"

```
2020 年 3 月的风险平价踩踏：

Step 1：股票下跌
  - COVID 冲击
  - 股票波动率飙升
  - 风险平价减少股票

Step 2：债券也下跌（美元荒）
  - 流动性需求
  - 美元避险
  - 债券也跌
  - 风险平价减少债券

Step 3：被动去杠杆
  - 风险预算用尽
  - 必须减仓
  - 卖出更多资产

Step 4：群体行为
  - 大量风险平价基金同时减仓
  - 加剧市场抛压
  - 死亡螺旋

Step 5：风险平价失效
  - 股票债券同时下跌
  - 风险分散失效
  - 损失放大
       ↓
结果：
  - 风险平价基金平均回撤 25%+
  - Bridgewater All Weather：-20%+
  - 历史上从未出现如此大回撤
```

## 十一、2020 年 3 月的"哲学高度"

```

1. **流动性是"易碎的玻璃"**
   - 平时完好无损
   - 极端市场突然碎裂
   - 必须提前规划

2. **相关性是"不稳定的"**
   - 平时表现良好
   - 极端市场完全失灵
   - 没有"永久的相关性"

3. **央行是"最终守护者"**
   - 投资者无法对抗央行
   - 央行的"手"决定一切
   - 理解央行 = 投资的核心

4. **"完美"对冲的虚妄**
   - 风险平价看似完美
   - 极端市场完全失效
   - 没有"圣杯"
```

### 11.2 现代金融的"反思"

```
2020 年 3 月后的金融反思：

1. **风险管理的"复杂性"**
   - 不是单一风险
   - 是多维风险
   - 包括外部冲击、流动性、相关性

2. **金融工程的"局限性"**
   - 风险平价是"顶级"策略
   - 极端市场失效
   - 金融工程不是万能

3. **央行的"双重角色"**
   - 流动性提供者
   - 资产价格管理者
   - 需要"独立性"和"灵活性"

4. **全球化金融的"脆弱"**
   - 美元荒的传导
   - 新兴市场危机
   - 全球化有"风险"
```

   - 群体踩踏
   - 难以预测

   - 极端市场失效

## 十五、一句话总结

> **2020 年 3 月流动性危机 = COVID-19 冲击 + 风险平价踩踏 + 美元荒 + 美联储"无限 QE"**。
>
> 它揭示了**风险管理的三个根本脆弱性**：**外部冲击的不可预测**（COVID 这种"未知未知"）、**风险分散的失效**（股票债券同跌）、**群体行为的危险**（风险平价同时减仓）。
>
> **2020 年不是个案**——它预示了 2022 年的债券市场危机、未来的未知危机。**真正的风险管理不是"完美对冲"，而是"敬畏风险、保持流动性、关注央行政策、理解群体行为"**。**美联储"无限 QE"是 2008 年的升级版，但"无限"也意味着"无限的责任"**。
