# 流动性风险深度

> LCR、NSFR、ALMM、内部流动性风险计量与 2008 危机教训

**流动性风险**（Liquidity Risk）是指银行无法在合理成本下及时获得充足资金，或无法以合理价格变现资产满足偿付需求的风险。2008 年金融危机的核心教训：**盈利性再强的银行，如果流动性枯竭，也会瞬间倒闭**。

---

## 一、流动性风险的两类来源

### 1.1 融资流动性风险（Funding Liquidity Risk）

```
无法以合理成本获得融资

触发因素：
- 存款大量流失（银行挤兑）
- 批发融资市场关闭（CP 市场、商业票据）
- 评级下调导致融资成本飙升
- 对手方收紧授信额度

典型案例：2008 年雷曼兄弟——
信用评级被下调 → 经纪商减少授信 → 融资渠道枯竭 → 流动性快速耗尽
```

### 1.2 市场流动性风险（Market Liquidity Risk）

```
无法以合理价格变现资产

触发因素：
- 市场深度急剧下降（买卖盘稀少）
- 价差急剧扩大（Bid-Ask 飙升）
- 资产价格非理性下跌
- 甩卖引发更大下跌（Fire Sale）

典型案例：2008 年次贷支持证券——
ABS 市场流动性消失 → 银行无法变现 MBS → 账面损失变现实损失
```

### 1.3 两类风险的关系

```
融资流动性风险 ←→ 市场流动性风险（双向强化）

融资收紧 → 银行被迫出售资产 → 市场供给增加 → 资产价格下跌 → 市场流动性恶化
                          → 资产抵押价值下降 → 融资能力进一步下降（循环）
```

## 二、Basel III 流动性监管框架

### 2.1 两大核心指标

| 指标 | 全称 | 目标 | 期限 |
|------|------|------|------|
| **LCR** | 流动性覆盖率 | 30 天短期抗压能力 | ≥ 100% |
| **NSFR** | 净稳定资金比率 | 长期结构性流动性 | ≥ 100% |

> 巴塞尔III（Basel III）**首次引入量化流动性监管要求**，以弥补 2008 年金融危机中暴露的银行流动性管理短板。

### 2.2 流动性覆盖率（LCR）

```
LCR = HQLA / 30天净现金流出 ≥ 100%

HQLA（高质量流动性资产）：
- 一级资产（0% Haircut）：现金、主权央行债券
- 二级资产（15% Haircut）：AA 以上评级公司债

30天净现金流出：
- 零售存款：稳定存款 3%/5%，不稳定存款 10%
- 批发存款：无担保 25%/100%，有担保 25%
- 契约性资金流出（贷款承诺、信用证等）
- 衍生品净流出
```

### 2.3 净稳定资金比率（NSFR）

```
NSFR = 可用稳定资金（ASF）/ 所需稳定资金（RSF）≥ 100%

可用稳定资金（ASF）：
- 1年期以上零售存款：90%
- 1年期以上无担保批发存款：50%
- 1年期以上有担保资金：100%

所需稳定资金（RSF）：
- 住房抵押贷款：50-70%（按 LTV）
- 公司贷款：30-100%（按期限和评级）
- 股票/黄金：50-100%
```

### 2.4 LCR vs NSFR 的互补关系

```
LCR（短期） + NSFR（长期） = 完整流动性覆盖

           LCR（30天）              NSFR（1年）
目标       应对短期流动性冲击       维持长期结构性平衡
关注       现金流入/流出           资金来源稳定性
调整       可在压力时期暂时降低     硬性要求，不可降低
```

## 三、内部流动性风险计量

### 3.1 现金流缺口分析

```
方法：将所有资产、负债按到期日分类，计算各时间段的净现金流缺口

时间段：1天/7天/30天/90天/1年/1年+

现金流缺口 = 现金流入 - 现金流出

分析维度：
- 契约性缺口：到期现金流
- 行为性缺口：客户可能提前支取、展期
- 压力性缺口：在流动性压力情景下的现金流
```

### 3.2 流动性风险限额体系

| 限额类型 | 说明 | 例子 |
|----------|------|------|
| **流动性比率** | 即时可用流动性 / 总资产 | > 10% |
| **期限错配限额** | 特定时间段最大缺口 | 30天缺口 < 资本 15% |
| **集中度限额** | 单个对手方最大融资比例 | < 资本 10% |
| **资产抵押率** | 可抵押资产 / 担保融资需求 | > 150% |
| **稳定资金缺口** | ASF - RSF | NSFR 满足前提 |

### 3.3 流动性风险计量指标

| 指标 | 定义 | 用途 |
|------|------|------|
| **流动性比率（Loan/Deposit）** | 贷款/存款比 | 传统监管指标 |
| **存贷比（LDR）** | 贷款/存款 | 中国银保监监控 |
| **LCR** | 30天流动性覆盖 | Basel III 标准 |
| **NSFR** | 1年稳定资金覆盖 | Basel III 标准 |
| **LCR 分级** | 按货币计 | 监测外币流动性 |
| **HQLA 缓冲** | 高质量流动性资产绝对量 | 绝对抗压能力 |
| **净流动性缺口** | 各期限现金流缺口 | 缺口结构分析 |

## 四、流动性风险压力测试

### 4.1 压力情景设定

| 情景 | 严重程度 | 持续时间 |
|------|----------|---------|
| **轻度压力** | 部分存款流失、融资成本上升 | 7 天 |
| **中度压力** | 大量零售存款流失、批发融资受限 | 30 天 |
| **重度压力** | 市场冻结、资产甩卖 | 90 天 |

### 4.2 压力下的行为假设

```
零售存款流失率：
- 稳定存款（保险覆盖）：3-5%（轻度）/ 5-10%（重度）
- 不稳定存款：10%（轻度）/ 25%（重度）

批发融资流出率：
- 无担保隔夜：25-100%
- 有担保：5-25%
- 信用证/担保：5-25%

资产流入折减：
- 正常到期收回：100%
- 展期回收：部分假设展期不续做
```

### 4.3 压力测试结果解读

```
LCR 压力测试 = HQLA / 压力下净流出

结果判定：
- LCR ≥ 100%：通过
- LCR < 100%：存在缺口 → 制定补缺计划

关键发现：
- HQLA 绝对量是否充足？
- 压力情景下的 HQLA 消耗速度？
- 批发融资到期集中度？
- 外币流动性缺口？
```

## 五、应急流动性计划（Contingency Funding Plan, CFP）

### 5.1 触发机制

```
预警指标（Level 1）：
- LCR < 110%
- 信用评级展望下调
- 负面舆情出现

升级指标（Level 2）：
- LCR < 100%
- 存款开始流失
- 融资成本显著上升

危机指标（Level 3）：
- LCR < 90%
- 批发融资渠道关闭
- 大规模存款外流
```

### 5.2 应对措施

| 措施 | 成本 | 时间 |
|------|------|------|
| **减少放贷** | 影响业务和声誉 | 即时 |
| **出售流动性较好的资产** | 潜在损失 | 1-3 天 |
| **央行借款（常备借贷便利）** | 利率成本 | 1-3 天 |
| **向央行申请紧急流动性支持** | 抵押品折价 | 数天 |
| **存款保险介入** | 系统性成本 | 数天 |

## 六、2008 年金融危机的流动性教训

### 6.1 危机的流动性传导

```
Step 1：次贷违约 → MBS 减值
Step 2：持有 MBS 的银行资产价值下降 → 资本减少
Step 3：评级下调风险 → 融资成本上升
Step 4：对手方减少授信 → 融资渠道收紧
Step 5：被迫甩卖资产 → 市场流动性恶化
Step 6：资产价格进一步下跌 → 资本进一步减少（循环）
Step 7：雷曼兄弟无力融资 → 破产
```

### 6.2 雷曼兄弟的教训

| 错误 | 后果 |
|------|------|
| 过度依赖隔夜回购融资 | 市场冻结时资金链瞬间断裂 |
| 持有大量低流动性资产 | 无法变现应对挤兑 |
| 资产负债表过度膨胀 | 去杠杆过程剧烈 |
| 对手方风险集中 | 贝尔斯登、AIG 等关联机构危机传染 |

### 6.3 危机后的改革

| 改革 | 内容 |
|------|------|
| **Basel III LCR** | 强制持有 30 天高质量流动性资产 |
| **Basel III NSFR** | 确保长期资金来源稳定性 |
| **央行流动性支持机制** | 美联储 BTFP、欧洲央行 LTRO |
| **存款保险提升** | 降低银行挤兑风险 |
| **OTC 衍生品清算** | 降低对手方风险 |

## 七、流动性风险与信用风险、市场风险的交叉

### 7.1 流动性螺旋

```
信用风险上升 → 银行收缩信贷 → 企业再融资困难
           → 企业被迫变卖资产 → 市场流动性恶化
           → 资产价格下跌 → 银行资产负债表恶化
           → 融资能力下降 → 进一步收缩信贷（循环）

→ 信用风险通过流动性渠道放大
```

### 7.2 流动性与市场风险

```
市场下跌 → 保证金追缴 → 强制平仓 → 市场进一步下跌

2008 年案例：
股市/债市下跌 → 量化基金去杠杆 → 强制抛售 → 市场进一步下跌
             → 各类型资产均承压 → 流动性全面枯竭
```

## 八、资产管理公司的流动性管理（与银行的差异）

| 维度 | 银行 | 资产管理公司 |
|------|------|-------------|
| **核心流动性风险** | 融资枯竭 | 赎回压力、强制抛售 |
| **应对工具** | 央行流动性支持 | 二级市场交易 |
| **期限管理** | 资产-负债匹配 | 开放式基金赎回管理 |
| **杠杆** | 高杠杆 + 期限错配 | 低杠杆或无杠杆 |
| **监管** | LCR/NSFR | 基金流动性管理规定 |

> 2020 年 3 月 COVID 冲击下，货币市场基金遭遇大规模赎回，触发 SEC 流动性管理紧急规则修订。
