# 回购市场（Repo Market）

> 华尔街的"血液循环系统"——影子银行体系加杠杆的核心机制，2008/2020 两次危机的"震中"

## 一、核心定义

**回购协议**（Repurchase Agreement，Repo）是一种**短期抵押融资工具**：卖方将证券卖给买方，同时约定在未来特定日期以约定价格买回。**回购利率**（Repo Rate）就是这种短期融资的利率。

**本质**：回购协议是"以证券作抵押的短期贷款"，是金融体系流动性的关键基础设施。

### 1.1 简单例子

```
银行 A 持有 100 万美元国债，需要短期现金
       ↓
与银行 B 签订回购协议：
  - 银行 A 以 100 万美元国债卖给银行 B
  - 银行 B 支付 99.9 万美元（10bp 折价）
  - 约定次日银行 A 以 100 万美元买回
       ↓
银行 A 获得：99.9 万美元现金（隔夜）
银行 B 获得：100bp 年化收益（10bp × 36.5）
```

### 1.2 关键术语

| 术语 | 含义 |
|------|------|
| **正回购（Repo）** | 借入现金、抵押证券 |
| **逆回购（Reverse Repo）** | 借出现金、获得抵押品 |
| **GCF Repo** | 一般抵押品回购 |
| **Tri-party Repo** | 三方回购（清算银行中介） |
| **SOFR** | 担保隔夜融资利率（2018 后替代 LIBOR） |
| **Haircut** | 抵押品折扣率 |
| **Initial Margin** | 初始保证金 |

## 二、回购市场为什么重要

### 2.1 金融体系的"血液循环"

```
回购市场的作用：

1. 银行日常流动性管理
   - 大银行通过回购借/贷数十亿美元
   - 满足监管准备金要求
   - 短期资金周转

2. 影子银行加杠杆的核心机制
   - 对冲基金通过回购放大杠杆（5-10 倍）
   - 投行通过回购融资购买证券
   - 加杠杆 = 借更多钱 = 风险放大

3. 货币政策的传导渠道
   - 美联储通过回购市场实施货币政策
   - 联邦基金利率 = 银行间隔夜无抵押利率
   - 回购利率 = 银行间隔夜抵押利率
   - 通常：联邦基金 < 回购 < Fed 贴现率

4. 债券市场的关键基础设施
   - 任何持有债券的机构都需要回购融资
   - 没有回购市场，债券流动性枯竭
```

### 2.2 回购市场的规模

```
美国回购市场规模（2024）：

国债回购（Tri-party + DVP）：
  - 每日交易量：~2 万亿美元
  - 未偿余额：~8 万亿美元
  
机构 MBS 回购：
  - 每日交易量：~3000 亿美元
  - 未偿余额：~2.5 万亿美元
  
股票回购（Prime Brokerage）：
  - 未偿余额：~1.5 万亿美元
  
总规模：~12 万亿美元（占美国 GDP 50%）

美国国债市场日均回购交易：
  - 2010：~5000 亿
  - 2020：~1.5 万亿
  - 2024：~2 万亿
```

## 三、回购市场的运作机制

### 3.1 参与者

```
资金提供方（Cash Lenders）：
  - 货币市场基金（MMF）
  - 大型商业银行
  - 央行（美联储）
  - 企业现金管理
  
资金需求方（Cash Borrowers）：
  - 对冲基金（高杠杆策略）
  - 投资银行（一级交易商）
  - 抵押贷款 REIT（杠杆运作）
  - 保险公司、养老金
  - 政府支持企业（GSE）
  
中介机构：
  - 一级交易商（Primary Dealers，24 家）
  - BNY Mellon（最大三方回购清算行）
  - Fixed Income Clearing Corporation（FICC）
```

### 3.2 抵押品与折价率（Haircut）

```
抵押品等级（决定折价率）：

| 抵押品类型 | 折价率 | 说明 |
|----------|--------|------|
| 美国国债 | 0-2% | 几乎无风险 |
| 机构 MBS | 2-5% | 房利美、房地美 |
| 公司债（投资级）| 5-10% | 信用风险 |
| 公司债（高收益）| 10-25% | 高风险 |
| 股票 | 15-30% | 波动大 |
| ABS / CDO | 25-50% | 复杂证券 |
| 私募股权 | 50%+ | 流动性极差 |
       ↓
折价率的含义：
  - 抵押品价值 100 万美元
  - 折价率 10%
  - 实际融资 90 万美元
  - 杠杆率 = 100/90 = 1.11 倍
       ↓
极端情况下：
  - 抵押品价值下降
  - 折价率上升
  - 实际融资减少
  - 借款人必须追加抵押品或还款
  → "Margin Call"
```

### 3.3 三方回购（Tri-party Repo）详解

```
三方回购的结构：

Step 1：交易达成
  - 借款方（投资银行）
  - 贷款方（货币市场基金）
  - 抵押品：美国国债
  
Step 2：清算行介入（BNY Mellon / JPM Chase）
  - 抵押品交给清算行
  - 现金交给借款方
  
Step 3：每日"滚动"
  - 大多数回购是隔夜
  - 每日重新达成
  - 抵押品"滚动"
  
Step 4：到期结算
  - 借款方还款 + 利息
  - 抵押品返还借款方
       ↓
关键特点：
  - 清算行作为中央对手方
  - 标准化、规模大
  - 但清算行只服务日间（不服务隔夜风险）
  - 隔夜风险由贷款方承担
```

## 四、回购市场与 2008 金融危机

### 4.1 危机前的"正常"回购市场

```
2006-2007 正常时期：

- 货币市场基金向投行提供回购融资
- 投行用回购资金购买 ABS / MBS / CDO
- 投行的杠杆率：25-40 倍
- 回购利率：~5%（接近联邦基金利率）
- 抵押品折价率：极低（次贷 CDO 仅 2-3%）
       ↓
问题的本质：
  - 投行的"基础"资本太少（股东权益）
  - 主要靠回购融资
  - 抵押品折价率过低（低估风险）
  - 一旦抵押品贬值，整个体系崩溃
```

### 4.2 危机中的回购市场崩溃（2008）

```
2007.8 危机信号：
  - BNP Paribas 冻结 3 只基金
  - 回购利率开始上升
  - 抵押品折价率扩大
       ↓
2008.3 贝尔斯登危机：
  - 贝尔斯登的对冲基金清盘
  - 抵押品（CDO）大幅贬值
  - 银行收紧对贝尔斯登的回购
  - 贝尔斯登被迫拍卖资产
  - 摩根大通紧急收购（美联储担保 300 亿）
       ↓
2008.9 雷曼时刻：
  - 9.13 雷曼与巴克莱、美国银行谈判失败
  - 9.14 雷曼无法获得隔夜回购融资
  - 9.15 雷曼申请破产
  - 9.16 回购利率飙升至 10%+（正常 5%）
  - 主要交易对手拒绝与雷曼交易
       ↓
关键事件：
  - 2008.9.16 Reserve Primary Fund "Breaking the buck"
  - 货币市场基金大规模赎回
  - 回购市场资金供给枯竭
  - 整个金融体系流动性冻结
```

### 4.3 美联储的紧急应对

```
危机应对工具：

1. PDCF（Primary Dealer Credit Facility，2008.3）
   - 直接向投行（一级交易商）贷款
   - 投行首次可获得"最后贷款人"支持
   - 美联储扩展其传统的"存款银行"范围

2. TSLF（Term Securities Lending Facility，2008.3）
   - 用国债换证券（流动性差的 MBS）
   - 缓解抵押品压力

3. AMLF（Asset-Backed Commercial Paper Money Market Mutual Fund Liquidity Facility，2008.9）
   - 美联储购买 ABCP
   - 支持货币市场基金

4. CPFF（Commercial Paper Funding Facility，2008.10）
   - 美联储直接购买 3 个月商业票据
   - 接管企业短期融资市场

5. Fed 实际接管回购市场
   - 2008 年底前
   - 美联储直接成为回购市场最大贷款方
   - 实质上"印钞"维持回购市场运转
```

### 4.4 危机后的"修复"

```
2008 危机后的关键改革：

1. 巴塞尔 III（2010）
   - 提高银行资本要求
   - 限制回购融资比例（净稳定资金比率 NSFR）
   - 流动性覆盖率（LCR）

2. 多德弗兰克法案（2010）
   - 加强影子银行监管
   - 限制银行对 SIV、CDO 的敞口

3. 中央清算（2010 后）
   - 强制回购中央清算
   - 通过 FICC、CME 等清算所
   - 减少对手方风险

4. 抵押品管理改革
   - 提高抵押品折价率
   - 加强估值
   - 限制高风险抵押品
```

## 五、回购市场与 2019 年 9 月"美元荒"

### 5.1 事件的本质

```
2019.9.17 事件：

背景：
  - 美联储从 2017.10 开始缩表（QT）
  - 银行体系准备金从 2.8 万亿降至 1.4 万亿
  - 准备金"稀缺"
       ↓
事件：
  - 公司税缴款日（9.15）
  - 国债拍卖结算（9.16）
  - 9.17 隔夜回购利率飙升至 10%+（正常 2%）
  - 美元荒
       ↓
美联储应对：
  - 9.17 紧急投放 530 亿美元流动性
  - 9.18 投放 750 亿
  - 之后连续多日投放
  - 10.11 宣布每月购买 600 亿短期国债（"有机扩表"）
       ↓
关键教训：
  - 即使没有危机，回购市场也可能冻结
  - QT 太快会引发流动性问题
  - 银行准备金"底线"被低估
```

### 5.2 SOFR 改革

```
LIBOR 的终结：

LIBOR（伦敦银行同业拆借利率）：
  - 基于银行"估算"而非实际交易
  - 2008 后被曝操纵
  - 2021 起逐步淘汰
  
SOFR（Secured Overnight Financing Rate）：
  - 基于实际回购交易
  - 由纽约联储发布
  - 2018.4 开始，2021 后成为主流
  
关键差别：
  - LIBOR：无抵押、估算
  - SOFR：有抵押（国债）、实际交易
  
应用范围：
  - 商业贷款
  - 抵押贷款
  - 衍生品（IRS、CDS）
  - 影响：~200 万亿美元金融资产
```

## 六、回购市场与 2020 美元荒

### 6.1 2020.3 美元荒的机制

```
2020.3 美元荒的形成：

Step 1：COVID 冲击
  - 全球资产抛售
  - 美元成为"避险资产"

Step 2：海外美元需求激增
  - 外国机构抛售美国资产换美元
  - 美国企业海外资金回流
  - 美元资金需求急剧上升

Step 3：国内美元短缺
  - 货币市场基金面临赎回
  - 银行收紧回购
  - 商业票据市场冻结

Step 4：回购市场冻结
  - 一级交易商无法融资
  - 对冲基金去杠杆
  - 美元指数从 95 飙升至 103
       ↓
本质：全球"美元荒" + 国内"回购冻结"
       ↓
美联储应对：
  - 无限 QE
  - 与多国央行美元互换
  - 直接支持回购市场
```

### 6.2 2020 后的新工具

```
美联储推出的新工具：

1. 央行美元互换（Central Bank Liquidity Swaps）
   - 美联储与外国央行互换美元
   - 9 家央行参与
   - 2020.3 后规模扩大

2. FIMA Repo Facility（2020.3）
   - 外国央行可用美国国债换美元
   - 缓解海外美元荒

3. 永久性回购工具（Standing Repo Facility, SRF，2021.7）
   - 日常支持回购市场
   - 利率 = 联邦基金 + 25bp
   - 防止 2019.9 类型事件重演
```

## 七、回购市场的"日常风险"

### 7.1 杠杆与追加保证金风险

```
对冲基金的回购风险：

策略：
  - 100 美元自有资金
  - 借 900 美元回购（10 倍杠杆）
  - 持有 1000 美元国债
       ↓
风险：
  - 国债价格下跌 5%（50 美元）
  - 抵押品价值降至 950 美元
  - 折价率 10% → 实际融资 855 美元
  - 必须还款 45 美元或追加抵押品
  → 追加保证金通知（Margin Call）
       ↓
危机放大：
  - 多个基金同时收到追加保证金通知
  - 集体抛售资产
  - 价格进一步下跌
  - 更多追加保证金通知
  - "死亡螺旋"
```

### 7.2 抵押品"飞向质量"（Flight to Quality）

```
危机时抵押品偏好变化：

正常时期：
  - 所有抵押品都接受
  - 折价率较低
  
危机时期：
  - 只接受国债、现金
  - 拒绝 ABS、MBS、公司债
  - 折价率大幅扩大
       ↓
问题：
  - 持有大量非国债抵押品的机构
  - 无法获得回购融资
  - 被迫抛售
  - 资产价格暴跌
```

### 7.3 交易对手风险

```
回购交易对手风险：

Step 1：交易达成
  - 借款人：投行
  - 贷款人：货币市场基金
  - 抵押品：投行的 ABS
  
Step 2：借款人破产
  - 抵押品属于贷款人还是借款人？
  - 法律上：贷款人是抵押品"所有人"
  - 但实际操作：可能需要法律程序
  
Step 3：损失
  - 即使有抵押品
  - 可能要打官司
  - 短期无法变现
       ↓
案例：
  - 雷曼破产时
  - 雷曼用 ABS 作为抵押品
  - 货币市场基金无法立即变现
  - 触发"打破美元"事件
```

## 八、回购市场与影子银行

### 8.1 影子银行的"血液循环"

```
影子银行依赖回购市场：

影子银行运作链：
  货币市场基金 → 回购融资 → 对冲基金/投行 → 购买 ABS/MBS
       ↑
       └────── 短期资金 ──────┘
       ↓
如果回购市场冻结：
  - 影子银行无法获得短期融资
  - 长期资产无法变现
  - 整个影子银行体系崩溃
  → 这就是 2008 危机
```

### 8.2 银行在回购市场的角色

```
银行的双重身份：

1. 传统业务：
   - 吸收存款（存款保险）
   - 监管资本要求
   - 美联储最后贷款人

2. 回购业务：
   - 通过回购融资（无存款保险）
   - 无监管资本要求（如果不在银行账簿）
   - 美联储"非最后贷款人"（2008 前）
       ↓
银行套利：
  - 存款利率 0%
  - 回购利率 5%
  - 通过回购融资"借短贷长"
  - 加杠杆 30+ 倍
       ↓
风险：
  - 如果回购市场冻结
  - 银行的"影子银行"业务瞬间死亡
  - 2008 危机正是如此
```

## 九、回购市场的全球对比

### 9.1 美国 vs 欧元区 vs 中国

| 维度 | 美国 | 欧元区 | 中国 |
|------|------|--------|------|
| **规模** | ~12 万亿美元 | ~3 万亿欧元 | ~2 万亿人民币（银行间） |
| **主要抵押品** | 美国国债、机构 MBS | 欧元区国债 | 中国国债、政策性金融债 |
| **利率** | SOFR（~5%） | ESTR（~4%） | DR007（~2%） |
| **监管** | SEC + Fed | ECB + ESMA | 中国人民银行 |
| **问题** | 货币市场基金脆弱 | 主权债信用风险 | 银行间杠杆 |

### 9.2 中国的回购市场

```
中国回购市场的特点：

1. 银行间回购占主导
   - 银行间债券回购占 80%+
   - 主要是质押式回购
   - 隔夜回购占比 70%+

2. 主要抵押品
   - 利率债（国债、政金债）
   - 信用债（公司债、企业债）
   - 央行票据

3. 主要参与者
   - 商业银行（最大）
   - 证券公司、基金、保险
   - 央行（公开市场操作）

4. 关键利率
   - DR001（银行间隔夜质押式回购利率）
   - DR007（7 天回购利率）
   - R007（非银机构 7 天回购）
   - 政策利率：OMO 7 天逆回购

5. 监管
   - 中国人民银行 + 银保监会 + 证监会
   - 2020 后规范"资金空转"
   - 限制银行同业负债比例
```

> 💡 **中国回购市场的"双层结构"**：
>
> - **银行间市场**（DR/R 系列）：机构主导、规模巨大、政策利率传导端
> - **交易所市场**（GC/R- 系列）：散户可参与的**国债逆回购**，详见 [[wiki/concepts/国债逆回购]] — 中国版"标准化平民级回购"，是 OMO 利率传导到零售端的最终环节，也是中国影子银行体系的"反向调节器"

## 十、回购市场的监管改革

### 10.1 主要监管框架

| 监管 | 重点 | 影响 |
|------|------|------|
| **巴塞尔 III** | NSFR（净稳定融资比率）、LCR | 限制银行回购期限错配 |
| **多德弗兰克** | 中央清算、抵押品管理 | 减少系统性风险 |
| **美联储规则** | 一级交易商监管、SRF | 提供最后贷款人支持 |
| **FSB** | 影子银行监测 | 全球协调 |

### 10.2 中央清算（Central Clearing）

```
中央清算的逻辑：

传统回购（双边）：
  - 每笔交易都是"双边"
  - 一方违约，另一方损失
  - 风险分散但不可控
  
中央清算后：
  - 所有交易通过中央对手方（CCP）
  - 违约由 CCP 承担
  - 损失分摊给所有成员
       ↓
优点：
  - 标准化
  - 透明度
  - 系统性风险可控
  
缺点：
  - CCP 本身成为系统性风险点
  - "大而不倒"的 CCP
```

### 10.3 "永久性回购工具"（SRF）

```
美联储 SRF（2021.7 推出）：

设计：
  - 任何合格机构可参与
  - 抵押品：美国国债
  - 利率：联邦基金 + 25bp
  - 期限：隔夜
  
目的：
  - 为回购市场设置"地板"
  - 防止 2019.9 类型事件
  - 提供永久流动性支持
  
影响：
  - 实际是"美联储最后贷款人"的日常化
  - 回购市场不再可能"冻结"
  - 但也意味着央行深度介入
```

## 十一、回购市场与货币政策

### 11.1 回购利率作为政策利率

```
政策利率的演进：

| 时期 | 政策利率 |
|------|---------|
| 1971-1981 | 联邦基金利率（FFR） |
| 1981-2008 | 联邦基金利率（成熟期） |
| 2008-2015 | FFR + 利率走廊（IORB） |
| 2015-2021 | FFR + 利率走廊 + IOER |
| 2021 后 | SOFR（替代 LIBOR） |
       ↓
当前框架：
  - 政策利率：联邦基金利率目标区间
  - 上限：SRF 利率（IORB + 25bp）
  - 下限：ON RRP 利率（IORB - 10bp）
  - SOFR 在走廊内波动
```

### 11.2 回购作为货币政策工具

```
美联储的回购操作：

1. 正回购（Repurchase Agreements）
   - 美联储卖出国债，收回现金
   - 收紧流动性
   
2. 逆回购（Reverse Repos, RRP）
   - 美联储买入国债，投放现金
   - 释放流动性
   
3. 公开市场操作（OMO）
   - 日常回购操作
   - 调节短期流动性

QE vs 回购：
  - QE：永久扩表（购买资产）
  - 回购：临时操作（卖回给市场）
  - 关键差别：QE 是长期、回购是短期
```

## 十二、回购市场的"五大风险"

```
1. **杠杆风险**
   - 30+ 倍杠杆是常见的
   - 抵押品小幅贬值 = 死亡螺旋
   - 2008 雷曼、AIG 的根源

2. **期限错配风险**
   - 隔夜融资 / 长期资产
   - 一旦短期融资冻结 = 立即危机
   - ABCP 危机的本质

3. **抵押品风险**
   - 抵押品估值错误
   - 危机时折价率急剧扩大
   - "飞向质量"效应

4. **交易对手风险**
   - 雷曼破产时
   - 抵押品"打官司"耗时数年
   - 影响 MMF

5. **流动性风险**
   - 看似"安全"的隔夜操作
   - 实际每天重新达成
   - 一旦没人愿意再借 = 立即崩盘
```

## 十三、回购市场的"五大教训"

```
1. **回购市场是金融体系的"血液循环"**
   - 没有它，体系无法运作
   - 美联储必须支持
   - 永久性 SRF 是必然选择

2. **杠杆是致命的**
   - 即使是"安全"机构（货币市场基金）
   - 也会被杠杆传染
   - 必须限制杠杆率

3. **抵押品是核心**
   - 估值、折价率、流动性都重要
   - 危机时只认国债、现金
   - 必须强制高质量抵押品

4. **监管必须覆盖"最后贷款人"**
   - 银行有 Fed 保护
   - 影子银行必须纳入监管
   - 否则"影子"将摧毁系统

5. **QE 的双刃剑**
   - 增加了回购市场的资金
   - 但也助长了杠杆
   - 退出时必须谨慎
```

## 十四、回购市场的"哲学高度"

### 14.1 "流动性"的本质

```
回购市场揭示的流动性真相：

1. 流动性不是"现金"
   - 流动性是"信任"
   - 信任导致借款
   - 借款带来流动性
   - 危机时信任崩塌 = 流动性消失

2. 流动性是"公共物品"
   - 单一机构无法提供
   - 必须由央行提供
   - 但央行提供 = 货币扩张

3. 流动性是"最便宜的"
   - 隔夜回购利率通常接近政策利率
   - 但危机时可飙升至 10%+
   - "最便宜的"也最脆弱
```

### 14.2 现代金融的"影子性"

```
回购市场揭示的"影子"性：

1. 银行在账簿上看似"安全"
   - 但通过回购融资
   - 实际是"影子银行"
   - 30+ 倍杠杆
   
2. 货币市场基金看似"无风险"
   - 但通过回购投向风险资产
   - "Breaking the buck" 随时可能
   
3. 对冲基金看似"灵活"
   - 但严重依赖回购融资
   - 一旦融资冻结 = 立即清盘

4. 美联储看似"强大"
   - 但必须随时准备接管回购市场
   - 永久性 SRF = 央行深度介入
```

## 十五、一句话总结

> **回购市场 = 现代金融体系的"血液循环系统"**。
>
> 它是**影子银行加杠杆的核心机制**，是**货币政策传导的关键渠道**，也是**2008、2019、2020 三次危机**的"震中"。
>
> **理解回购市场 = 理解现代金融的脆弱性**：
> - **杠杆**：30+ 倍是常见的
> - **期限错配**：隔夜融资 / 长期资产
> - **抵押品依赖**：流动性消失时只认国债、现金
> - **央行支持**：美联储 SRF 是必然选择
>
> **回购市场是 21 世纪金融危机的"震源"**——**2008** 从这里开始（贝尔斯登、雷曼）、**2019** 在这里预警（9.17 美元荒）、**2020** 在这里崩溃（COVID 美元荒）。**理解回购市场，就理解了为什么美联储必须充当"最后贷款人"，也理解了为什么"无限 QE"是必然选择**。
