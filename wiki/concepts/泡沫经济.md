# 泡沫经济

> 日本在1980年代后期至1990年代初形成的资产价格泡沫，表现为房地产和股市的极端升值后急剧崩盘。

## 核心数据

| 指标 | 1983年基数 | 1991年峰值 | 涨幅 |
|------|-----------|-----------|------|
| 主要城市工业+住宅用地价格指数 | 100 | ~300 | +200% |
| 主要城市商业用地价格指数 | 100 | ~600 | +500% |
| 东京圈土地价格指数 | 100 | ~400 | +300% |
| 日经225指数 | ~10,000 | ~39,000 | +290% |
| 日本股票总市值 | - | ~4万亿美元 | 全球占比~50% |

## 形成机制

```
出口型工业 + 技术升级
    ↓ 贸易顺差积累财富
企业和居民部门资金充裕
    ↓ 《广场协议》日元升值
国际热钱涌入 + 国内流动性过剩
    ↓ 金融自由化取消资金管制
大量资金涌入股市和楼市
    ↓ 财富效应吸引更多资金
自我强化的泡沫循环
```

## 典型特征

- **"东京可买下美国"** — 1980年代末的说法，日本国内资产总值约20万亿美元（占全球1/5）
- **全民投机** — 股票和房产每天增值，夜夜笙歌，奢侈品消费横扫全球
- **海外并购狂潮** — 索尼收购哥伦比亚影业、三菱收购洛克菲勒中心51%股权、日本人持有洛杉矶市区47%房产
- **学者宣称"新经济规律"** — 1989年底近四万点时，经济学家称传统理论已不适用

## 崩盘过程

> 详见 [[自杀式加息]] 和 [[wiki/sources/2020-04-25-日本经济崩盘始末-巫师财经]]

1. **1990年3月** — 《控制土地相关融资的规定》出台，停止向房地产输血
2. **1990年** — 央行急速加息（2.5%→6%），[[流动性消失术]] 启动
3. **1991年** — 房价开始断崖式下跌
4. **1992年** — 地价税（房产税）出台，楼市全面崩溃

## 极端案例

> 首付300万买1000万房子，贷款700万。房价跌到500万+失业 → 房子被银行收回抵500万债，还欠200万。"一夜回到奴隶制"
