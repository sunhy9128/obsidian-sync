# 期货对冲（Hedging with Futures）

> 用期货市场对冲现货市场风险 —— 投资者"分层管理风险"的核心工具

## 一、什么是对冲（Hedging）？

**对冲**（Hedging）是指**同时进行两笔方向相反的相关交易**，**抵消价格波动的风险**。期货对冲是其中**最经典、最标准**的形式。

对冲的本质：
- 用期货的盈亏抵消现货的盈亏
- 让总损益**接近中性**
- 牺牲一部分潜在收益，换取**确定性**

> **核心比喻**：对冲就是**"金融保险"**——你为飞机上的乘客买航空意外险，乘客出事时保险公司赔，没出事时保费归公司。期货对冲就是这个机制。

## 二、为什么需要期货对冲？

### 2.1 现货市场的"风险困境"

```
持有股票（或计划买入股票）面临的痛点：

1. 长期持有者：
   - 担心短期下跌吞噬利润
   - 不想卖出（避免税费、避免失去分红权）

2. 计划买入者：
   - 担心价格上涨导致成本失控
   - 需要锁定未来成本

3. 机构投资者：
   - 监管要求控制风险（如保险、养老金）
   - 业绩需要"稳定性"

4. 上市公司：
   - 大股东减持需要保护收益
   - 并购方需要锁定成本
```

### 2.2 期货市场的"对冲特性"

| 现货市场 | 期货市场 |
|---------|---------|
| 必须先买入才能卖出 | 可以直接做空 |
| 占用 100% 资金 | 只需 5-15% 保证金 |
| T+N 结算 | T+0 结算 |
| 持有期无限 | 合约有到期日 |
| 单向波动风险 | 双向波动风险 |
| 收益曲线左尾 | 收益曲线对称 |

## 三、两种基本对冲策略

### 3.1 空头对冲（Short Hedge）—— 保护多头头寸

**适用场景**：你已经**持有**股票或商品，担心短期下跌

```
操作流程：
  持有 1000 股 X 股票（@100 元，市值 10万元）
       ↓
  担心未来 3 个月股价下跌
       ↓
  操作：在期货市场**做空** X 股票期货合约
       ↓
  对冲数量 = 1000 股 = 1 手（假设合约乘数匹配）

情景 1：股价下跌 10%（@100 → @90）
  ┌─────────────────────────────────┐
  │ 现货市场：损失 1000 × 10 = -10,000 元
  │ 期货市场：盈利 1000 × 10 = +10,000 元
  │ 净损益 = 0 ✅（完美对冲）
  └─────────────────────────────────┘

情景 2：股价上涨 10%（@100 → @110）
  ┌─────────────────────────────────┐
  │ 现货市场：盈利 1000 × 10 = +10,000 元
  │ 期货市场：损失 1000 × 10 = -10,000 元
  │ 净损益 = 0 ✅（完美对冲）
  └─────────────────────────────────┘
```

> **核心结论**：无论股价涨跌，净损益 ≈ 0。这就是"对冲"。

### 3.2 多头对冲（Long Hedge）—— 锁定未来买入成本

**适用场景**：你**计划未来**买入股票或商品，担心价格上涨

```
操作流程：
  计划 3 个月后买入 1000 股 X 股票（@100 元）
       ↓
  担心 3 个月内股价上涨
       ↓
  操作：在期货市场**做多** X 股票期货合约
       ↓
  对冲数量 = 1000 股 = 1 手

情景 1：股价上涨 10%
  ┌─────────────────────────────────┐
  │ 现货成本：增加 1000 × 10 = +10,000 元
  │ 期货盈利：+10,000 元
  │ 净影响 = 0 ✅（锁定成本）
  └─────────────────────────────────┘

情景 2：股价下跌 10%
  ┌─────────────────────────────────┐
  │ 现货成本：减少 -10,000 元
  │ 期货亏损：-10,000 元
  │ 净影响 = 0 ✅（仍然锁定）
  └─────────────────────────────────┘
```

## 四、实际案例深度解析

### 4.1 案例一：养老基金的"持仓对冲"

```
场景：
  养老基金持有 1 亿元沪深 300 成分股
  计划持有 6 个月
  担心大盘下跌
       ↓
对冲操作：
  卖出沪深 300 股指期货
  计算对冲数量：
    1亿元 / (4000点 × 300元/点) = 约 83 手
  实际对冲 80-90 手（考虑 β 系数）
       ↓
效果：
  大盘跌 10%：
    现货损失 ~1000 万
    期货盈利 ~1000 万
    净损失 ≈ 0
       ↓
  关键意义：
    - 养老基金保留"选股超额收益"（α）
    - 消除"市场 β 风险"
    - 业绩更稳定（满足监管要求）
```

### 4.2 案例二：上市公司大股东减持

```
场景：
  大股东持有 1 亿股 X 股票（市值 10 亿）
  计划 3 个月内减持 2000 万股
  担心减持期间股价下跌
       ↓
对冲操作：
  在期货市场做空 2000 万股对应的期货
  锁定当前价格
       ↓
效果：
  情景 A：减持期间股价下跌
    现货：卖出价更低 → 损失
    期货：盈利 → 抵消
    大股东在"无市场风险"环境下完成减持
       ↓
  情景 B：减持期间股价上涨
    现货：卖出价更高 → 额外盈利
    期货：损失 → 抵消
    大股东锁定"当前价格"
```

### 4.3 案例三：1998 香港金管局的"立体对冲"

```
金管局的"完美对冲组合"：

1. 现货多头（买入恒指成分股）
   - 推高股价
   - 减少流通股
   - 形成轧空

2. 期货多头（恒指期货做多）
   - 锁定恒指 7800 点
   - 对冲现货持有的市场风险
   - 锁定期货收益

3. 立体效应：
   现货 + 期货 = 强制空头平仓
   无论市场怎么波动，金管局都有收益
   长期持有 = 持续轧空
       ↓
本质上：
  - 现货 = 攻击（推高股价）
  - 期货 = 防御（对冲市场风险）
  - 长期持有 = 维持轧空压力
```

> **1998 香港救市的"教科书级"**：金管局把对冲和轧空**完美结合**——既对冲了自己的市场风险，又通过现货买入把空头逼入绝境。详见 [[wiki/concepts/1998香港金融保卫战]]。

### 4.4 案例四：航空公司的"燃油对冲"

```
（虽然不是股票期货，但原理相同）

航空公司面临燃油价格波动
       ↓
操作：
  在原油期货市场做多
  锁定未来燃油采购成本
       ↓
效果：
  燃油涨价 → 现货成本增加，期货盈利抵消
  燃油跌价 → 现货成本减少，期货损失抵消
  航空公司锁定燃油成本
```

> **关键洞察**：对冲的原理**跨市场通用**——商品、外汇、利率、股票指数都能用对冲思维。

## 五、对冲的"三个关键要素"

### 5.1 要素一：方向相反

```
对冲的核心是"方向相反"
  持有多头现货 → 卖出期货
  持有空头现货 → 买入期货
  计划买入 → 买入期货（先锁定价格）
  计划卖出 → 卖出期货（先锁定价格）
```

### 5.2 要素二：数量匹配

```
对冲数量 = 现货头寸 × β 系数 / 合约乘数

其中：
  - β 系数：股票相对指数的波动性
  - 合约乘数：期货合约的标准乘数
    例：恒指期货 1 点 = 50 港元
    例：沪深 300 期货 1 点 = 300 元
```

### 5.3 要素三：时间匹配

```
期货合约的到期日
  应与现货头寸的"风险期"匹配
  
  例：
  现货头寸持有 3 个月
  → 选择 3 个月后到期的期货合约
  → 到期时现货卖出，期货平仓
```

## 六、对冲的"四大风险"

### 6.1 风险一：基差风险（Basis Risk）

```
基差 = 现货价格 - 期货价格

完美对冲要求：基差 = 0 且不变
但实际中基差会变化
       ↓
例：
  现货 100 元，期货 99 元（基差 = +1）
  3 个月后：
    现货 90 元（跌 10%）
    期货 88 元（跌 11.1%）
  对冲结果：
    现货损失 10 元
    期货盈利 11 元
    净盈利 1 元（基差变动带来）
```

详见 [[wiki/concepts/基差风险]]。

### 6.2 风险二：合约不匹配

```
期货合约有标准乘数和到期日
无法完全匹配现货头寸
       ↓
可能出现的"对冲缺口"：
  - 合约乘数 ≠ 现货数量
  - 到期日 ≠ 实际风险期
  - 合约标 ≠ 现货标的
```

### 6.3 风险三：流动性风险

```
某些期货品种流动性差
无法以理想价格成交
       ↓
结果：
  - 滑点增加
  - 对冲成本上升
  - 可能无法及时平仓
```

### 6.4 风险四：成本风险

```
期货交易的成本：
  - 手续费
  - 保证金占用
  - 资金成本
       ↓
频繁对冲会增加总成本
可能"对冲的损失"超过"对冲的收益"
```

## 七、对冲 vs 投机 vs 套利

| 维度 | 对冲（Hedging）| 投机（Speculation）| 套利（Arbitrage）|
|------|---------------|-------------------|------------------|
| 目的 | 降低风险 | 追求收益 | 无风险套利 |
| 现货头寸 | 有 | 无 | 有 |
| 期货头寸 | 与现货相反 | 单向 | 双向 |
| 净头寸 | **中性** | 单向 | 中性 |
| 风险 | 降低 | 高 | 接近 0 |
| 收益 | 锁定 | 期望收益 | 微小但稳定 |
| 适用者 | 机构、稳健投资者 | 投机者 | 量化、做市商 |

> **本质区别**：对冲 = **保险**；投机 = **赌博**；套利 = **搬砖**。

## 八、对冲的金融学本质

### 8.1 风险分层管理

```
投资者面临的风险：
  ├─ 系统性风险（大盘波动）
  │   → 对冲
  ├─ 行业风险
  │   → 部分对冲
  └─ 个股风险（选股能力）
      → 不对冲（这是你的 α）
       ↓
对冲的哲学：
  投资者**保留**自己有能力管理的风险
  投资者**转移**自己没有能力管理的风险
```

### 8.2 对冲的成本

```
对冲不是"免费的午餐"
  - 期货费用（手续费 + 保证金）
  - 基差风险（不完美对冲）
  - 机会成本（占用保证金）
       ↓
对冲的"等价交换"：
  用"成本"换取"风险降低"
  在"风险厌恶"和"风险承受"之间权衡
```

### 8.3 对冲的"信息价值"

```
对冲者**知道**自己在做什么：
  - 持有什么现货
  - 担心什么风险
  - 愿意支付多少对冲成本
       ↓
投机者**不知道**风险来源
  - 单纯赌方向
  - 不知道自己在承担什么风险
       ↓
结论：
  对冲 = 信息透明 + 风险可控
  投机 = 信息不透明 + 风险累积
```

## 九、对冲的"四类应用场景"

### 9.1 场景一：持仓保护

```
长期持有的机构投资者
  - 不想卖出（税务、纪律、策略）
  - 担心短期波动
  → 用期货对冲市场 β 风险
```

### 9.2 场景二：成本锁定

```
计划未来买入的企业/投资者
  - 并购、定增、到期调仓
  - 担心价格上涨
  → 用期货多头对冲
```

### 9.3 场景三：套期保值（企业层面）

```
企业面临原料/商品价格风险
  - 航空公司：原油价格
  - 制造业：原材料价格
  - 跨国企业：汇率风险
  → 用期货对冲
```

### 9.4 场景四：监管合规

```
金融机构受巴塞尔协议等监管
  - 必须控制市场风险敞口
  - 必须满足风险资本要求
  → 用期货对冲
```

## 十、对冲的"高级形式"

### 10.1 动态对冲（Dynamic Hedging）

```
期权做市商需要对冲 Delta 风险
       ↓
Delta 持续变化（Gamma 风险）
需要持续调整对冲头寸
       ↓
这就是"Delta Hedging" / "Greeks Hedging"
       ↓
经典案例：
  长期资本管理公司（LTCM）
  1998 年俄罗斯危机 + 动态对冲失效
```

详见 [[wiki/concepts/动态对冲]]。

### 10.2 跨市场对冲

```
持有美股 S&P 500 成分股
但没有 A 股对应的期货
       ↓
使用恒指期货或 H 股指数期货
（虽然不完全匹配，但能部分对冲系统性风险）
       ↓
适用于：跨市场配置、QDII 基金
```

### 10.3 跨品种对冲

```
持有沪深 300 成分股
但只有上证 50 期货
       ↓
使用上证 50 期货做对冲
（虽然相关性 < 1，但能部分对冲）
       ↓
关键：跨品种对冲的 β 系数
```

## 十一、对冲的"五大致命错误"

### 11.1 错误一：方向搞反

```
对冲的核心是"方向相反"
  - 持有多头应卖出期货
  - 持有空头应买入期货
  - 方向反了 = 加大风险而不是降低
```

### 11.2 错误二：数量不匹配

```
对冲数量需要精确计算
  - 合约乘数
  - β 系数
  - 净头寸
       ↓
数量错误 = 对冲不足或过度对冲
```

### 11.3 错误三：忽视基差风险

```
基差变动 = 对冲的"剩余风险"
  - 完美对冲在现实中不存在
  - 必须接受"不完美"
```

### 11.4 错误四：忽视流动性

```
对冲需要在"有事时能平仓"
  - 选择流动性好的合约
  - 警惕"无法平仓"风险
```

### 11.5 错误五：成本失控

```
对冲的成本
  - 手续费
  - 保证金
  - 资金占用
  - 频繁调整
       ↓
可能"对冲成本"超过"对冲收益"
```

## 十二、对冲的"哲学高度"

### 12.1 三种风险态度

```
对待风险的三种态度：
  
1. 风险厌恶（Risk Averse）
   → 完全对冲
   → 几乎消除风险
   → 收益最低
  
2. 风险中性（Risk Neutral）
   → 部分对冲
   → 保留部分风险
   → 收益中等
  
3. 风险偏好（Risk Seeking）
   → 不对冲
   → 完全暴露风险
   → 收益最高（或最低）
```

### 12.2 对冲的"价值判断"

```
对冲不是"对"或"错"的选择
而是"愿意为确定性支付多少"
       ↓
对冲的"决策框架"：
  - 我的核心能力是什么？
  - 哪些风险我可以管理？
  - 哪些风险我应该转移？
  - 我愿意为转移支付多少成本？
       ↓
好的投资者：
  保留自己有优势的风险
  转移自己没有优势的风险
  用对冲实现"风险分层"
```

## 十三、1998 香港金管局对冲的"教科书"细节

```
金管局的对冲设计堪称完美：

1. 时机选择：
   - 8 月 14 日（周五）开始：试探
   - 8 月 17-27 日：持续买入
   - 8 月 28 日（期货结算日）：决战
   → 时点选择完美

2. 方向选择：
   - 现货：买入（多头）
   - 期货：买入（多头）
   → 现货 + 期货 = 双重多头
   → 不是"对冲"，而是"攻击"
   → 主动出击，把空头逼入绝境

3. 数量设计：
   - 30 只恒指成分股
   - 约 1180 亿港元
   - 8 月期指（即将到期）
   → 现货 + 期货数量匹配

4. 退出策略：
   - 9 年持有期
   - 2007 年开始逐步卖出
   → 长期视角，不被短期波动影响
```

> **金管局的"对冲"实际上是"反攻"**——它不是"防御性的对冲"，而是"攻击性的轧空"。这是对冲和轧空的完美结合。

## 十四、一句话总结

> **期货对冲 = 用期货市场对冲现货市场风险，本质是"保险"。**
>
> 它通过"**方向相反 + 数量匹配 + 时间匹配**"的三要素，把价格波动的风险"转移"给愿意承担风险的人（投机者）。**对冲的代价是"成本"（手续费 + 基差风险），收益是"确定性"**。
>
> **1998 香港金管局**把对冲升华为"立体攻击"——现货 + 期货 + 长期持有，把空头逼入绝境。**1998 香港**因此成为**对冲的"教科书级应用"**——它证明了对冲不仅能"防守"，还能"进攻"。
