> 机会成本（Opportunity Cost）指为获取某物而放弃的次优选择的价值。在资源稀缺前提下，任何选择都有机会成本——即"做这件事的同时没做那件事，所放弃的那件事的最高价值"。

---

## 一、核心概念

### 1.1 定义

```
机会成本 = 做出的选择带来的收益 - 放弃的次优选择的最大收益

更一般地：
机会成本 = 资源用于此用途的收益 - 资源用于最佳替代用途的收益
```

### 1.2 机会成本的根本来源

| 前提 | 说明 |
|------|------|
| 稀缺性 | 资源有限，不能同时满足所有需求 |
| 选择性 | 必须做选择就必然有放弃 |
| 多用途 | 资源至少有 2 个潜在用途 |

### 1.3 与会计成本的区别

| 维度 | 会计成本 | 机会成本（经济成本） |
|------|---------|-------------------|
| 定义 | 实际支出的货币费用 | 放弃的次优选择的价值 |
| 范围 | 显性支出 | 显性 + 隐性 |
| 视角 | 财务记录 | 经济学决策 |
| 用途 | 财务报表、税务 | 投资决策、资源配置 |

## 二、成本分类

### 2.1 显性成本 vs 隐性成本

| 类型 | 显性成本（Explicit Cost） | 隐性成本（Implicit Cost） |
|------|----------------------|---------------------|
| 定义 | 实际支付给他人的费用 | 自己拥有的资源的机会成本 |
| 形态 | 现金、票据 | 自己的时间、资本、自有房产 |
| 会计记录 | 计入账簿 | 不直接记账 |
| 例子 | 工资、租金、原料 | 自己经营应得工资、自有资金利息 |

### 2.2 会计利润 vs 经济利润

```
会计利润 = 总收益 - 显性成本
经济利润 = 总收益 - 显性成本 - 隐性成本（机会成本）
经济利润 = 会计利润 - 隐性成本
```

| 利润类型 | 含义 | 决策含义 |
|---------|------|---------|
| 会计利润 > 0 | 收入覆盖显性支出 | 财务上盈利 |
| 经济利润 > 0 | 收入超过所有机会成本 | 资源真正增值 |
| 经济利润 = 0 | 正常利润 | 行业平均回报 |
| 经济利润 < 0 | 资源应用不当 | 应转投他业 |

### 2.3 沉没成本 vs 机会成本

| 类型 | 定义 | 决策相关性 |
|------|------|----------|
| 沉没成本 | 已发生且不可收回的成本 | 决策时不应考虑 |
| 机会成本 | 当前选择的次优选项价值 | 决策时必须考虑 |
| 共同误区 | 误把沉没成本当作机会成本 | 理性决策者忽略沉没 |

## 三、典型案例

### 3.1 案例 A：大学还是工作？

```
情况：高中毕业面临选择
  选择 1：读 4 年大学，总花费 20 万（学费 + 生活费）
  选择 2：直接工作，4 年工资共 40 万

如果读大学毕业后 4 年赚 80 万
```

| 项目 | 金额 | 性质 |
|------|------|------|
| 显性成本 | 20 万 | 学费生活费 |
| 隐性成本 | 40 万 | 工作 4 年工资 |
| 总机会成本 | 60 万 | 放弃的财富 |
| 大学收益 | 80 万 | 毕业后 4 年工资 |
| 净收益 | 80 - 60 = 20 万 | 读大学合算 |

### 3.2 案例 B：自雇 vs 打工

```
自己开咖啡店：
  年显性成本：30 万（房租、原料、人工）
  年收入：50 万
  会计利润：50 - 30 = 20 万

如果不自己开店，受雇于人：
  年薪：25 万
  自有资金 50 万存入银行：利息 2 万
  合计放弃：27 万
```

| 成本类型 | 金额 |
|---------|------|
| 显性成本 | 30 万 |
| 隐性成本 | 27 万（打工工资 + 利息） |
| 总经济成本 | 57 万 |
| 会计利润 | 20 万 |
| 经济利润 | -7 万（应放弃开店） |

### 3.3 案例 C：自有住房 vs 出租

```
自住自有住房：
  免租金（不付出）
  但放弃了出租收入

假设：同类房屋年租金 6 万
机会成本 = 6 万/年
```

### 3.4 案例 D：投资股票 vs 债券

```
资金 100 万
  股票预期回报：12%
  债券预期回报：4%

投资股票的机会成本 = 4%（放弃的债券收益）
```

### 3.5 案例 E：加班 vs 休息

```
加班 1 小时赚 100 元
  放弃：陪家人 / 休息 / 健康
  真实机会成本：放弃的 1 小时的最高替代价值

长期看：健康、家庭关系不能用金钱衡量
```

## 四、生产可能性边界（PPF）

### 4.1 PPF 定义

```
生产可能性边界（PPF）= 在资源和技术约束下，能生产的最大产出组合

典型形态：
  食品量
    │\
    │ \
    │  \   ← 边界外：不可达
    │   \
    │    \●  A 点（有效率）
    │     \
    │  ●   \ ← 内部点：有效率但未充分利用
    │       \
    └───────── 机器量
```

### 4.2 PPF 关键特征

| 特征 | 含义 |
|------|------|
| 边界外 | 资源不足、不可达 |
| 边界上 | 资源充分利用、有效率 |
| 边界内 | 资源闲置、低效率 |
| 凹向原点 | 机会成本递增 |

### 4.3 机会成本递增规律

```
多生产 1 单位食品
  → 必须放弃越来越多的机器
  → 机会成本递增

数学表达（边际替代率递减）：
MRS = dX/dY，|MRS| 随 X 增加而增加
```

### 4.4 PPF 移动的影响因素

| 移动 | 原因 |
|------|------|
| 外移 | 技术进步、资本积累、劳动力增长 |
| 内移 | 灾害、资源枯竭、战争 |
| 旋转外移 | 偏向某一产品的技术进步 |

## 五、机会成本的应用

### 5.1 个人决策

| 决策 | 显性成本 | 机会成本（隐性） |
|------|---------|----------------|
| 读研 2 年 | 学费 10 万 | 工作 2 年工资 30 万 |
| 跳槽 | 新工作通勤时间长 | 现公司的稳定与熟悉 |
| 健身 1 小时 | 健身卡 50 元 | 1 小时加班费 |
| 提前退休 | 退休金减少 | 继续工作的工资收入 |

### 5.2 企业决策

| 决策 | 显性成本 | 机会成本 |
|------|---------|---------|
| 扩建厂房 | 投资 1000 万 | 其他项目预期收益 |
| 自制零部件 | 人工 + 原料 | 外购成本 |
| 持有现金 | 利息损失 | 投资收益 |
| 库存过多 | 仓储费用 | 资金占用成本 |

### 5.3 政府决策

| 决策 | 显性成本 | 机会成本 |
|------|---------|---------|
| 军费开支 | 军费预算 | 民生、教育投资 |
| 基建项目 | 财政支出 | 减税的空间 |
| 救市政策 | 救助资金 | 长期结构调整 |
| 环保政策 | 治理投入 | 短期 GDP 损失 |

### 5.4 投资决策：折现现金流（DCF）

```
项目 NPV 计算中要扣除：
  - 占用资金的机会成本（折现率）
  - 占用土地的机会成本
  - 占用管理人员的机会成本
```

## 六、机会成本与其他成本概念

### 6.1 概念对比

| 概念 | 定义 | 用途 |
|------|------|------|
| 机会成本 | 放弃的次优选择价值 | 决策分析 |
| 沉没成本 | 不可收回的支出 | 决策时忽略 |
| 边际成本 | 多生产 1 单位的成本 | 产量决策 |
| 平均成本 | 总成本 / 总产量 | 单位经济效益 |
| 变动成本 | 随产量变化的成本 | 短期决策 |
| 固定成本 | 不随产量变化的成本 | 长期决策 |

### 6.2 理性决策原则

```
理性决策者应：
  ✓ 考虑所有成本（显性 + 隐性）
  ✓ 忽略沉没成本
  ✓ 比较所有可选方案
  ✓ 关注边际而非平均

误区：
  ✗ 只看会计成本
  ✗ 沉没成本谬误（"已经花了 1000 万了，不能放弃"）
  ✗ 忽略隐性成本
  ✗ 短视只看显性
```

## 七、深度思考题

### 7.1 思考题 1

```
你在 A 公司年薪 50 万。
朋友邀你创业，预计 1 年后月入 10 万，但可能失败。
你如何决策？
```

分析维度：
- 显性成本：放弃的 50 万年薪
- 隐性成本：创业投入的时间、精力
- 风险成本：失败概率 × 损失
- 心理成本：压力、不确定感

### 7.2 思考题 2

```
一台设备 5 年前花 100 万购买，
现在账面价值 60 万（折旧后），
卖出可收回 30 万。

继续使用：
  显性：维护费 5 万/年
  隐性：占用 30 万资金的利息 1.5 万/年

应否卖出？
```

决策原则：忽略沉没成本（已无法收回的 40 万），比较继续使用的机会成本与卖出的收益。
