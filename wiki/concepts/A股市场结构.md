# A 股市场结构

> 中国股票市场的制度框架、参与主体与运行机制

理解 A 股市场结构是投资落地的前提——不同板块、不同交易制度、不同参与者行为共同决定了 A 股独特的定价逻辑。

---

## 一、市场分层体系

```
A 股多层次市场
│
├─ 主板（上海 + 深圳）
│  ├─ 沪市主板（600/601/603）
│  └─ 深市主板（000/001）
│  → 定位：成熟期大型企业，"蓝筹股"集中地
│
├─ 科创板（688）
│  → 上海，2019 年设立
│  → 定位："硬科技"（新一代信息技术、高端装备、生物医药等）
│  → 制度突破：注册制、允许未盈利企业上市、涨跌幅 20%
│
├─ 创业板（300/301）
│  → 深圳，2009 年设立
│  → 定位：成长型创新创业企业
│  → 2020 年改革：注册制、涨跌幅 20%
│
├─ 北交所（8xx）
│  → 北京，2021 年设立
│  → 定位："专精特新"中小企业
│  → 来源：新三板精选层平移
│
└─ 新三板（基础层 / 创新层）
    → 非上市股份公司
    → 投资者门槛高（基础层 200 万，创新层 100 万）
    → 流动性极低
```

### 各板块定位对比

| 板块 | 成立 | 定位 | 上市标准 | 交易门槛 | 涨跌幅 |
|------|------|------|----------|----------|--------|
| 沪市主板 | 1990 | 大盘蓝筹、央企 | 盈利要求最严 | 无 | 10% |
| 深市主板 | 1991 | 中小规模成熟企业 | 盈利要求严 | 无 | 10% |
| 科创板 | 2019 | 硬科技（未盈利可上） | 5 套标准（市值+研发） | 2 年经验 + 50 万资产 | 20% |
| 创业板 | 2009 | 成长型创新企业 | 2 套标准（利润/市值） | 2 年经验 + 10 万资产 | 20% |
| 北交所 | 2021 | 专精特新 | 4 套标准（市值+利润/收入） | 2 年经验 + 50 万资产 | 30% |

## 二、交易制度

### 2.1 核心交易规则

| 规则 | A 股 | 港股 | 美股 |
|------|------|------|------|
| 交易时间 | 9:30-11:30, 13:00-15:00 | 9:30-16:00（午休） | 9:30-16:00（连续） |
| 结算周期 | T+1 | T+2 | T+2 |
| 涨跌幅 | 10%/20%/30%（板块各异） | 无 | 无 |
| 最小交易单位 | 100 股（1 手） | 因股价而异 | 1 股 |
| 做空机制 | 融券（极有限） | 丰富 | 丰富 |
| 停牌制度 | 较多（异常波动核查等） | 少 | 极少 |

### 2.2 A 股特色交易机制

- **集合竞价**（9:15-9:25）：开盘价确定机制，9:20 后不可撤单
- **连续竞价**（9:30-11:30, 13:00-15:00）：逐笔撮合，价格优先→时间优先
- **尾盘集合竞价**（14:57-15:00）：深市有，最后 3 分钟确定收盘价
- **大宗交易**：单笔 >30 万股或 >200 万元，可协议成交（不影响盘中价格）

## 三、投资者结构

A 股投资者结构是理解 A 股定价逻辑的核心——与成熟市场**截然不同**。

### 3.1 结构对比（自由流通市值口径）

| 投资者类型 | A 股 | 港股 | 美股 |
|-----------|------|------|------|
| **散户** | ~60% | ~20% | ~20% |
| **公募基金** | ~12% | ~10% | ~25% |
| **外资（北向）** | ~5% | ~40% | ~15% |
| **保险/社保** | ~8% | ~5% | ~10% |
| **私募/游资** | ~10% | ~5% | ~10% |
| **产业资本/其他** | ~5% | ~20% | ~20% |

### 3.2 A 股投资者的行为特征

| 类型 | 风格 | 持股周期 | 信息来源 | 影响 |
|------|------|----------|----------|------|
| **散户** | 趋势/博弈/题材 | 数天~数月 | 社交媒体、股吧 | 高换手率、追涨杀跌 |
| **公募基金** | 基本面/成长 | 季度~年度 | 券商研报、调研 | 重仓股效应（基金抱团） |
| **保险/社保** | 高股息/价值 | 年度+ | 内部研究 | 低换手、稳定器 |
| **外资（北向）** | DCF/基本面 | 季度~年度 | 全球视角 | A 股定价锚、偏好消费/金融 |
| **游资** | 打板/题材 | 数小时~数天 | 资金面、情绪 | 制造题材股异动 |

### 3.3 北向资金（陆股通）

| 项目 | 说明 |
|------|------|
| 开通时间 | 2014 年（沪股通）/ 2016 年（深股通） |
| 交易额度 | 无总额度限制（每日额度 520 亿元） |
| 持股偏好 | 消费（茅台、美的）、金融（招行、平安）、新能源（宁德） |
| 影响 | 被视为"聪明钱"，单日净流入/流出常影响市场情绪 |
| 长期意义 | 北向持仓结构不断改善 A 股定价效率 |

> **北向资金是 A 股最重要的增量资金来源之一**。2017-2021 年核心资产行情中，北向持续买入消费龙头，推动其估值中枢系统性上移。当北向持续大规模流出时（如 2022-2023），A 股估值承压。

## 四、指数体系

### 核心指数

| 指数 | 成分股 | 权重方式 | 代表意义 |
|------|--------|----------|----------|
| **上证综指**（000001） | 沪市全部 | 总市值加权 | 最传统、最广为人知；但受大金融/两桶油影响大 |
| **沪深 300**（000300） | 沪深两市前 300 | 自由流通市值加权 | **A 股最重要的基准指数**，代表大中盘蓝筹 |
| **中证 500** | 第 301-800 名 | 自由流通市值加权 | 中盘成长股的代表 |
| **中证 1000** | 第 801-1800 名 | 自由流通市值加权 | 小盘股的代表 |
| **创业板指** | 创业板市值前 100 | 自由流通市值加权 | 成长股/科技股风向标 |
| **科创 50** | 科创板市值前 50 | 自由流通市值加权 | 硬科技板块指标 |

### 风格指数

- **上证 50**：沪市超大盘（金融 + 消费）
- **中证红利**：高股息股票组合
- **科创 100**：科创板中小市值
- **北证 50**：北交所头部公司

## 五、监管体系

### 架构

```
国务院
  └─ 中国证监会（CSRC）—— 最高监管机构
       ├─ 上海证券交易所（SSE）
       ├─ 深圳证券交易所（SZSE）
       ├─ 北京证券交易所（BSE）
       ├─ 中国结算（CSDC）—— 证券登记结算
       ├─ 中证金融（CSF）—— 转融通/融资融券
       └─ 中国期货业协会等
```

### 注册制改革

2019 年科创板试点注册制→2020 年创业板推广→2023 年全面注册制落地。

**注册制 vs 核准制**：

| 维度 | 核准制（2019 前） | 注册制（2019 后） |
|------|------------------|------------------|
| 审核机构 | 证监会发审委 | 交易所审核、证监会注册 |
| 审核重点 | 企业持续盈利能力 | 信息披露的真实性/完整性 |
| 上市速度 | 漫长（1-3 年排队） | 大幅加快（6-12 个月） |
| 定价方式 | 23 倍 PE 隐形上限 | 市场化询价（允许破发） |
| 退市制度 | 退市极少 | 退市常态化（面值退/财务退） |

> **全面注册制的意义**：A 股从"审核把关"转向"信息披露"——上市公司供给大增，壳价值消失，对投资者的研究能力要求显著提高。

## 六、A 股特色现象

### 6.1 散户主导的定价特征

- **高换手率**：A 股年换手率 200-400%（全球最高之一），是美股（~100%）的 2-4 倍
- **题材炒作**：概念性炒作（雄安、元宇宙、AI 等）在短期可带来巨大波动，基本面短期内可被忽视
- **涨停板文化**：游资通过封板-次日高开模式收割跟风散户
- **近期偏好**：散户对近期利好/利空的反应远强于长期逻辑

### 6.2 政策敏感性

A 股对政策信号的敏感度远高于成熟市场：

| 政策类型 | 影响方式 | 历史实例 |
|----------|----------|----------|
| 货币政策 | 降准降息 → 流动性宽松 → 利好 | 2014-2015 降息周期催生牛市 |
| 财政政策 | 基建/减税 → 相关行业受益 | 2022 基建发力 → 中字头上涨 |
| 产业政策 | 补贴/准入 → 行业景气变化 | 2020-2022 新能源补贴 → 光伏/车大涨 |
| 监管政策 | 整顿/规范 → 行业冲击 | 2021 教育"双减" → 教培股归零 |
| 国家队干预 | 汇金/证金买入 → 托底 | 2015 救市、2024 汇金增持 ETF |

### 6.3 打新与次新股

历史上 A 股打新基本"稳赚不赔"（核准制下发行的折价效应），注册制后破发常态化。次新股因流通盘小、筹码干净，常成为炒作对象——这是 A 股独有的微观结构现象。

### 6.4 大小非解禁

限售股解禁是 A 股特有的供给侧冲击，解禁前后股价通常承压。2016-2020 年大量定增及 IPO 限售股解禁是压制结构性行情的重要因素。

## 七、估值与流动性特征

### 整体估值水平

- A 股估值整体在**全球居中**——低于美股（科技溢价），高于港股（折价）
- 沪深 300 PE 历史中枢约 12-15x（宽幅：8x（极低）~ 25x（极高））
- A 股存在明显的**大小盘估值差**——小盘股溢价长期存在

### 流动性分层

| 层次 | 特征 | 日均成交额 | 占据 A 股数量 |
|------|------|-----------|-------------|
| 核心资产 | 机构持仓、流动性好 | >10 亿 | ~100 只 |
| 一般活跃 | 有成交、可交易 | 1-10 亿 | ~1000 只 |
| 边缘品种 | 交易极少 | <1000 万 | ~2000 只 |
| 僵尸化 | 几乎无交易 | <100 万 | ~1000 只 |

> **A 股流动性分化严重**——头部 100 只股票占据约 30% 的总成交额，大量小盘股日均成交不足 500 万。流动性分层在注册制后加剧。
