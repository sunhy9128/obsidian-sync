# 银行扩表与流动性

> 银行扩表有两种本质截然不同的形式——贷款扩表创造货币，购债扩表只是换形态

## 一、核心区分——两种扩表方式

### 1.1 银行扩表的两种本质

| 类型 | 资产端 | 负债端 | 货币派生 |
|------|--------|--------|---------|
| **贷款扩表** | 贷款 ↑ | 存款 ↑ | ✅ **创造 M2** |
| **购债扩表** | 政府债 ↑ | 存款不变（或 ↓） | ❌ **不创造 M2** |

**核心区别**：
- **贷款扩表**：银行创造新的信用（货币派生）
- **购债扩表**：银行只是把存款换成债券，资产形式转换

### 1.2 一个核心判断

> **银行的"扩表"不等于"投放流动性"。**
>
> 只有**贷款扩表**才创造货币、投放流动性；
> **购债扩表**只是资产形态转换，**不会创造新的货币，反而可能减少流通中的货币**。

---

## 二、贷款扩表的机制

### 2.1 会计本质

```
银行发放 100 万元贷款给企业 A
       ↓
银行资产端：
   - 贷款 +100 万元
银行负债端：
   - 企业存款 +100 万元
       ↓
M2 增加 100 万元
       ↓
如果企业 A 把 80 万存为活期，20 万支付给企业 B
       ↓
企业 B 又存入银行 → 银行可再放贷
       ↓
通过货币乘数（假设 5x），M2 最多扩张 500 万元
```

### 2.2 货币派生机制

```
初始存款 → 银行贷款 → 派生存款 → 进一步贷款 → 更多派生存款
       ↓
        ↓
       货币乘数效应
       ↓
M2 大幅扩张
```

### 2.3 货币乘数公式

```
M2 = 基础货币 × 货币乘数
       ↓
货币乘数 = (1 + 现金漏损率) / (存款准备金率 + 现金漏损率)
       ↓
中国当前：
   - 存款准备金率 ~ 8%
   - 现金漏损率 ~ 5%
   - 货币乘数 ~ 7-8
```

### 2.4 贷款扩表的影响

| 维度 | 影响 |
|------|------|
| **M2** | 大幅扩张 |
| **流通速度** | 加快 |
| **通胀** | 推升 |
| **GDP** | 拉动 |
| **资产价格** | 上行 |

## 三、购债扩表的机制

### 3.1 会计本质

```
银行用 100 万元存款购买 100 万元政府债
       ↓
银行资产端：
   - 贷款不变（或减少）
   - 政府债 +100 万元
银行负债端：
   - 存款不变（资金仍来自存款）
       ↓
**M2 不变或下降**（如果资金是活期存款转为政府债）
       ↓
如果原本是贷款 → M2 下降（信贷货币 → 债券货币）
如果原本是现金 → 流通速度下降
```

### 3.2 关键差异

| 维度 | 贷款扩表 | 购债扩表 |
|------|---------|---------|
| **M2 影响** | 大幅扩张 | 不变或下降 |
| **货币创造** | 创造 | 不创造 |
| **流通速度** | 加快 | 减慢 |
| **通胀** | 推升 | 影响小 |
| **GDP** | 拉动 | 影响小 |
| **银行净息差** | 较高（贷款利率 4-5%） | 较低（政府债 2.5%-3%） |

### 3.3 购债扩表的传导路径

```
银行购入政府债
       ↓
资金从银行账户流向政府账户
       ↓
政府账户资金"冻结"（等待拨付）
       ↓
政府拨付给城投 → 城投还银行贷款（M2 下降）
                  → 城投支付项目款（部分回流）
       ↓
流通中的货币（M1）减少
       ↓
流通速度下降
```

## 四、"流动性冻结"的本质

### 4.1 流动性层级

| 资金形态 | 流动性等级 | 流通速度 |
|---------|----------|---------|
| 活期存款 | 高 | 最快 |
| 银行理财 | 中高 | 较快 |
| 银行持有的政府债 | 中（可质押/二级交易） | 准现金 |
| 政府专项账户 | 低 | 冻结 |
| 财政存款 | 低 | 冻结 |
| 居民储蓄 | 低 | 沉淀 |

### 4.2 化债"冻结"的量化

```
12 万亿化债债的购买方：
   ├── 银行自营 60-70% → ~7.5 万亿
   ├── 保险 10-15% → ~1.5 万亿
   ├── 基金/券商 5-10% → ~1 万亿
   └── 理财/其他 ~1 万亿

其中真正"冻结"在低流通状态的：
   ├── 政府专项账户（待拨付）~ 3 万亿
   ├── 财政沉淀资金 ~ 1 万亿
   └── 项目款支付周期 3-6 个月
```

> **保守估计**：12 万亿化债中，约 4-5 万亿在化债初期会处于"流动性冻结"状态。

### 4.3 与 QE 的对比

| 维度 | 美联储 QE | 中国化债 |
|------|----------|---------|
| **本质** | 央行扩表 | 银行扩表 |
| **资金去向** | 直接进入市场 | 进入政府账户 |
| **M2 影响** | 大幅扩张 | 短期收缩 |
| **流动性** | 注入 | 冻结 |

## 五、银行扩表 vs 央行扩表

### 5.1 关键区别

| 维度 | 央行扩表（QE） | 银行扩表 |
|------|--------------|---------|
| **扩表主体** | 中央银行 | 商业银行 |
| **买入资产** | 国债、政府债 | 各类资产 |
| **资金来源** | 印钞（基础货币） | 存款、准备金 |
| **M2 影响** | 间接（通过银行） | 直接 |
| **流动性** | 注入 | 因资产类型而异 |
| **副作用** | 通胀 | 银行 ROE 下降 |

### 5.2 央行扩表 → 银行扩表的传导

```
美联储 QE
       ↓
银行准备金增加
       ↓
银行有钱 → 选择扩表
       ↓
（如果贷款扩表）→ M2 扩张
（如果购债扩表）→ 资产结构变化
       ↓
最终效果取决于银行的资产选择
```

### 5.3 中国化债的特殊性

```
中国央行（不扩表）
       ↓
银行直接扩表（购债）
       ↓
央行通过 MLF、降准、再贷款等工具**对冲流动性**
       ↓
最终效果：银行扩表，但流动性被央行部分对冲
```

## 六、对银行体系的影响

### 6.1 银行的"三重损失"

| 损失维度 | 量化估算 | 说明 |
|---------|---------|------|
| **净息差收窄** | -10 至 -20 BP | 政府债收益率 2.5% 远低于贷款 4-5% |
| **ROA 下降** | -0.05% 至 -0.1% | 低收益资产占比上升 |
| **资本回报率** | 1-2% 下降 | 银行盈利能力整体承压 |
| **资产周转率** | 下降 | 政府债久期长，资产周转慢 |

### 6.2 银行的"三重缓冲"

#### 缓冲一：政府债的"准现金"属性

```
银行 A 持有 1 亿政府债
       ↓
可以质押给央行（MLF/SLF）获得流动性
       ↓
可以在二级市场卖出（流动性极佳）
       ↓
可以作为 HQLA（高质量流动性资产）
       ↓
**流动性并不"死"**，只是"形态转化"**
```

#### 缓冲二：央行结构性工具支持

- 央行可以**通过 MLF、再贷款等工具**释放流动性
- PSL 工具直接给政策性银行
- 保障性住房再贷款（3000 亿）

#### 缓冲三：化债后期流动性回流

```
银行购买化债债（资金流向政府）
       ↓
几个月后政府拨付 → 城投支付项目款
       ↓
资金回流到实体 → 部分回流到银行存款
       ↓
**流动性恢复**
```

## 七、宏观数据印证

### 7.1 中国当前数据

| 指标 | 当前值 | 历史水平 | 说明 |
|------|--------|---------|------|
| **M2 增速** | 8%-9% | 12%+ | 化债拖累 |
| **社融增速** | 8%-9% | 15%+ | 化债拖累 |
| **银行净息差** | 1.5% | 2%+ | 历史低位 |
| **CPI 通胀** | 0%-1% | 2%-3% | 通缩压力 |

### 7.2 解读

> **这些数据与"流动性冻结"的逻辑完全一致**：
> - M2 增速放缓 = 货币派生减弱
> - 净息差收窄 = 银行盈利压力
> - CPI 低位 = 流通速度下降

### 7.3 国际对比

| 国家 | 央行扩表后 M2 增速 | 通胀 |
|------|------------------|------|
| **美国** | 25%+（2020-2022） | 9%+（2022） |
| **欧元区** | 8%+ | 5-10% |
| **中国** | 8%-9% | 0%-1% |

> **中国 M2 增速远低于美国，通胀远低于美国——这就是"化债 vs QE"的真实差异**。

## 八、央行的对冲机制

### 8.1 数量型工具

| 工具 | 操作 | 效果 |
|------|------|------|
| **降准** | 降低存款准备金率 | 释放长期流动性 |
| **MLF 净投放** | 中期借贷便利 | 补充中期流动性 |
| **再贷款再贴现** | 结构性投放 | 精准支持 |
| **PSL** | 抵押补充贷款 | 支持三大工程 |
| **买断式逆回购** | 2024 年新增 | 释放流动性 |

### 8.2 价格型工具

- 降低 MLF 利率 → 引导 LPR 下行
- 降低存款利率 → 缓解银行负债端压力
- 降低实体融资成本 → 刺激信贷需求

### 8.3 对冲规模

> **2024 年央行净投放约 4-5 万亿**（含 MLF、再贷款、降准等），基本对冲了化债带来的"流动性冻结"。

### 8.4 对冲的不完全性

- 央行净投放 4-5 万亿 < 化债冻结 4-5 万亿
- M2 增速仍处于历史低位（8%-9%）
- **说明对冲是"部分"的，不是"完全"的**

## 九、对实体经济的传导

### 9.1 化债如何影响实体经济

```
银行购买化债债
       ↓
银行可用于贷款的资金减少
       ↓
实体经济获得贷款增速放缓
       ↓
企业投资意愿下降
       ↓
GDP 增速放缓
       ↓
通缩压力上升
```

### 9.2 数据印证

| 指标 | 化债前 | 化债后 | 变化 |
|------|--------|--------|------|
| **M2 增速** | 12% | 8%-9% | -3% |
| **社融增速** | 15% | 8%-9% | -6% |
| **CPI** | 2%-3% | 0%-1% | -2% |
| **GDP** | 5%+ | 4.5% | -0.5% |

### 9.3 一个关键判断

> **化债对实体经济是"短期负面、长期正面"**：
> - 短期：流动性冻结，信贷收紧，通缩压力
> - 长期：债务风险化解，金融体系健康，可持续增长

## 十、与扩表/缩表的关系

### 10.1 银行扩表的三种形式

| 形式 | 主体 | 效果 |
|------|------|------|
| **央行扩表** | 中央银行 | 印钱 → 流动性注入 |
| **贷款扩表** | 商业银行 | 创造货币 → 流动性扩张 |
| **购债扩表** | 商业银行 | 换形态 → 流动性冻结 |

### 10.2 中国化债的特殊性

```
中国当前：
   - 央行不扩表（无 QE）
   - 银行扩表（购债）
   - 央行通过结构性工具对冲
   ↓
效果：
   - 银行扩表，但 M2 收缩
   - 流动性冻结，但央行对冲
   - 通胀温和，但通缩压力
   - 银行 ROE 下降，但风险化解
```

### 10.3 一个核心总结

> **银行扩表不等于"投放流动性"，购债扩表反而是"流动性冻结"。**
>
> **化债让银行"被动扩表"，但这是"低效扩表"——资产规模扩张，但盈利能力下降。**
>
> **这种"扩表"抽取了金融体系的部分流动性，需要央行持续对冲。**

---

## 十一、一句话总结

> **银行扩表的两种本质**：
> 1. **贷款扩表 = 创造货币 → 投放流动性**
> 2. **购债扩表 = 换形态 → 冻结流动性**
>
> **化债是典型的"购债扩表"，看似扩表，实则流动性收紧**。
>
> **这是中国式"主动出清"的金融学本质——以银行盈利换金融稳定**。
