## 十二、量化规律总结

### 12.1 危机年 DXY 平均上涨幅度

```
危机 DXY 平均上涨：8-20%
危机 EM 货币平均贬值：30-75%
危机 EM 资本净流出：$1 万亿+
危机 EM 主权违约率：2-5 倍于常态
```

### 12.2 储备份额下降速度

```
1999-2009：71% → 62%（每年约 1pp）
2009-2019：62% → 59%（每年约 0.3pp）
2019-2026：59% → 57%（每年约 0.3pp）
```

**趋势**：下降速度放缓——**美元仍是最强储备货币**，但替代过程缓慢。

### 12.3 Fed 资产负债表 vs DXY 关系

```
Fed 扩表 → 美元贬值（DXY 下跌）
Fed 缩表 → 美元升值（DXY 上涨）

但有滞后：
- 2008-2014 扩表 $3.6T → DXY 从 89 跌至 73（-18%）
- 2020-2022 扩表 $4.7T → DXY 从 89 跌至 95（+7%，因加息预期）
- 2022-2025 缩表 $2.2T → DXY 在 95-108 区间震荡
```

## 十四、一句话总结

> **美元潮汐量化实证 = 1980s 以来每一次美联储紧缩周期都伴随 EM 危机——量化证据完整**。
