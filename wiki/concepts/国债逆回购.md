# 国债逆回购（中国交易所版）

> 中国资本市场最具"平民性"的标准化回购工具——连接散户资金与机构融资需求的"末梢血管"

---

## 一、定义与本质

### 1.1 一句话定义

**国债逆回购**，是指出资方（逆回购方，俗称"放款方"）通过**沪深交易所**的固定收益证券综合电子平台，将资金借给融资方（正回购方，俗称"借款方"），并以**国债**作为质押物；融资方在约定到期日还本付息、解除质押的标准化短期融资行为。

### 1.2 关键术语澄清

> ⚠️ **"逆回购"在中国语境下至少有三层含义**，必须严格区分：

| 语境 | 含义 | 参与主体 | 资金方向 |
|------|------|----------|----------|
| **本页面主题**：交易所国债逆回购（GC001等） | 散户/机构借出资金，国债作质押 | **任何合格投资者（含散户）** | 资金 → 融资方 |
| **央行 OMO 逆回购**（"7天逆回购"） | 央行投放基础货币，债券作质押 | 央行 → 一级交易商（银行） | 资金 → 银行体系 |
| **冲销式正回购** | 央行回笼流动性，债券作质押 | 央行 → 商业银行 | 资金 → 央行 |

**用户最常说的"国债逆回购"= 沪深交易所的 GC001/GC007 等品种**——这是本页面专注的对象。

### 1.3 与"正回购"的对应关系

```
正回购方（融资方，借入资金）：
  - 手里有国债
  - 缺现金
  - 通过回购把国债"质押"出去借钱
  - 到期还款 + 利息，拿回国债
       ↓
逆回购方（出资方，借出资金）：
  - 手里有闲钱
  - 想要短期无风险收益
  - 通过逆回购把钱"借出去"
  - 到期收回本金 + 利息，解除质押
       ↓
对同一笔交易：
  - 融资方说："我做正回购"
  - 出资方说："我做国债逆回购"
  - 双方在交易所的标准电子平台撮合
```

## 二、市场结构：中国回购市场的"双层架构"

### 2.1 双层结构概览

```
中国回购市场
      │
      ├── 银行间市场（机构主导）
      │     ├── 质押式回购（DR001、DR007、R007等）
      │     ├── 买断式回购
      │     └── 参与主体：商业银行、券商、基金、保险、信托
      │           ↓
      │         规模：~150 万亿 / 年成交
      │         监管：中国人民银行 + 外汇交易中心
      │
      └── 交易所市场（公众主导）⭐ 本页重点
            ├── 上交所固定收益平台 → GC001-GC182
            ├── 深交所综合协议平台 → R-001、R-007 等
            └── 参与主体：散户、机构、企业财务
                  ↓
                规模：~2 万亿 / 年成交
                监管：中国证监会 + 中证登 + 沪深交易所
```

### 2.2 关键差别

| 维度 | 银行间回购（DR系列） | 交易所国债逆回购（GC系列） |
|------|----------------------|----------------------------|
| **参与门槛** | 机构（5000 万起） | **散户 1000 元起** |
| **主要参与方** | 银行、券商、基金 | **散户、券商自营、企业财务** |
| **抵押品** | 国债、政金债、信用债 | **国债**（最优质抵押品） |
| **信用风险** | 交易对手信用 | **几乎为零**（中证登担保交收） |
| **利率发现** | 银行间真实成交 | **更受散户行为影响** |
| **政策地位** | 政策利率（OMO 7天逆回购）的传导端 | **不直接是政策利率** |
| **规模** | 巨大（占回购市场 80%+） | 较小（散户+企业财务） |

> 💡 **关键洞察**：交易所国债逆回购本质上是中国"标准化回购"对**公众资金**开放的安全接口——它把华尔街只有机构能玩的"回购"，做成了中国**散户**也能参与的"现金管理工具"。

## 四、交易机制详解

### 4.1 交易时间与下单方式

```
交易时间：
  - 交易日 9:30-11:30（上午）
  - 交易日 13:00-15:30（下午）
       ⚠️ 注意：15:00 后仍可下单至 15:30，但成交概率较低

下单方式：
  1. 在股票账户内输入代码（如 204001）
  2. 选择"卖出"（⚠️ 不是买入！）
     - 出资方做的操作本质是"卖出现金、买入债券质押权"
     - 交易所系统将"卖出"反向解释为"借出资金"
  3. 输入价格（年化利率 %）+ 数量（1000 元一手）
  4. 撮合成交 → 资金冻结、债券质押
```

### 4.2 资金与利息计算

```
成交日（T 日）：
  - 资金立即冻结（可用资金减少）
  - 债券质押给中证登

T+1 日（首次结算日）：
  - 资金可用（可买股票、可转账，但不可提现）

T+N 日（到期日）：
  - 本金 + 利息自动到账
  - 资金可用
  - 解除债券质押

T+N+1 日：
  - 资金可提现到银行卡
```

### 4.3 利息计算公式

```
利息 = 成交金额 × 年化利率 × 实际占款天数 / 365

⚠️ 关键："实际占款天数" ≠ "期限天数"

经典案例：周四买 1 天期（GC001）
  - T 日（周四）成交
  - T+1（周五）：首次结算
  - 实际到期日（次周四），因为周末和周五跨周末
  - 实际占款天数为：周五、周六、周日、下周一、下周二、下周三 = 6 天
  - 但"操作"是 1 天期 GC001
       ↓
  - 这是"周四效应"的来源：周四 GC001 利率会提前上涨
  - 因为周四 1 天 = 下周四才拿回钱 = 资金占用 6 天
```

### 4.4 重要日期效应

| 时点 | 利率表现 | 原因 |
|------|----------|------|
| **周四** | GC001 利率上涨 | 1 天期实际占用 6 天（含周末） |
| **长假前** | GC007/GC014 大幅飙升 | 跨节资金紧张 |
| **月末/季末/年末** | 各期限利率飙升 | 银行冲存款 + 报表压力 |
| **新股申购日** | GC001 异常飙升 | 大量资金锁定打新 |
| **央行 OMO 到期/续作日** | 利率反映政策意图 | 央行释放/收紧流动性 |

## 五、与影子银行的关系

### 5.1 关键定位：不是影子银行，但是影子银行的"末梢血管"

> ⚠️ **必须澄清**：**国债逆回购本身不是影子银行**。但它在中国金融生态中扮演了独特的角色——是连接"散户闲置资金"和"机构融资需求"的标准化桥梁。

```
中国影子银行资金链（简化）：

散户存款 → 银行（受存款利率管制）
散户闲置 → 国债逆回购 ⭐（散户唯一的"场内无风险"出借渠道）
              ↓
          国债质押融资
              ↓
          进入机构（券商自营、基金、企业财务）
              ↓
          资金进入实体经济 / 资本市场
```

### 5.2 为什么说它是中国影子银行的"安全外层"？

```
对比其他现金管理工具：

| 工具 | 风险 | 流动性 | 收益 | 是否影子银行属性 |
|------|------|--------|------|----------------|
| 活期存款 | 几乎零 | 极高 | 0.2% | 传统银行 |
| 货币基金（余额宝） | 极低 | 高 | 2% | 接近银行功能（准存款化） |
| **国债逆回购** | **几乎零** | **高** | **1.5%-10%** | **无**（标准化、场内） |
| 银行理财 | 中低 | 中 | 3-4% | 准影子银行（资管新规前） |
| 信托/私募 | 中高 | 低 | 5-10% | **典型影子银行** |

⭐ 国债逆回购的核心价值：在"银行存款"和"影子银行"之间提供了一个**零风险的标准化替代**——
  - 收益率高于活期（甚至常高于货币基金）
  - 流动性接近活期
  - 不存在任何影子银行的"通道、嵌套、刚兑"问题
```

### 5.3 中国 vs 美国：标准化回购的"平民化"

```
美国（华尔街）：
  - ON RRP（隔夜逆回购）只能由合格机构参与
  - 散户无法参与
  - 主要作用：吸收过剩流动性 + 利率走廊下限
       ↓
中国（沪深交易所）：
  - GC001 等散户可参与（1000 元起）
  - 主要作用：散户现金管理 + 短期利率信号
       ↓
⭐ 本质差别：
  - 美国 ON RRP 是"机构间的流动性吸收器"
  - 中国国债逆回购是"全民的现金管理工具"
```

### 5.4 国债逆回购在"中国式影子银行"中的具体角色

```
银行间"资金空转"的反向调节器：

传统影子银行链条：
  银行发行理财 → 信托通道 → 投向城投/地产
  （这里银行表内 → 表外的"出表"）

国债逆回购的角色：
  散户资金 → 国债逆回购 → 国债质押融资 → 给券商/基金/企业
  （这里资金直接"沉淀"在标准化市场）
       ↓
  国债逆回购的存在使得：
  - 大量散户闲置资金**不进入**高风险影子银行体系
  - 而是被"截留"在标准化、低风险、有担保的国债市场
  - 客观上**降低了影子银行体系的资金供给**
  - 是中国影子银行"出清"过程中的隐性正面力量
```

## 六、与央行 OMO 逆回购的"双轨体系"

### 6.1 双轨对比

| 维度 | 央行 OMO 7天逆回购 | 交易所国债逆回购 |
|------|----------------------|------------------|
| **操作主体** | 中国人民银行 | 投资者（任何合格主体） |
| **对手方** | 一级交易商（24 家银行） | 沪深交易所系统撮合 |
| **资金方向** | 央行投放基础货币 → 银行 | 资金在市场主体间流转 |
| **影响** | 直接影响 M0、超储率 | **不影响基础货币总量**（仅是资金再分配） |
| **政策含义** | **政策利率**（OMO 利率） | 不是政策利率，但高度同步 |
| **利率形成** | 央行决定（利率走廊下限） | 市场撮合 |
| **传导** | 央行 → 银行 → 实体 | 散户 → 机构 → 资本市场 |

### 6.2 双轨如何"咬合"

```
传导链条：

央行 OMO 7天逆回购（政策利率）
       ↓ 央行投放/回笼基础货币
DR007（银行间 7 天质押式回购）
       ↓ 银行间传导到非银
R007（非银 7 天质押式回购）
       ↓ 散户市场传导
GC007（上交所 7 天国债逆回购）
       ↓ 反映到
企业/个人的资金可得性与利率
```

> 💡 **核心洞察**：**GC 系列是 OMO 利率传导到"零售端"的最终环节**。当你在股票软件上看到 GC007 跳到 5%，意味着短期资金面已经"非常紧"——这通常是 DR007 > OMO 利率、央行拒绝续作 OMO 的市场信号。

## 七、影子银行视角下的"五个判断"

### 7.1 判断 1：国债逆回购不是影子银行，但定义了影子银行的"上限"

```
如果一项工具：
  ✅ 有标准化合约
  ✅ 有中央对手方（中证登）
  ✅ 有最优质抵押品（国债）
  ✅ 信息披露完整
  ✅ 监管框架明确
  → 它不是影子银行

国债逆回购满足以上所有条件，因此：
  - 它不是影子银行
  - 它反而构成了中国影子银行的"良性对照"
  - 投资者可以无风险地"留在场内"，不需要"出走"到影子银行
```

### 7.2 判断 2：国债逆回购的利率信号是"影子银行压力指数"

```
信号解读：
  - GC001 利率 < 2%（常态） → 流动性宽松，资金充裕
  - GC001 利率 3-5%（季末） → 流动性偏紧，银行表外回表压力
  - GC001 利率 > 10%（极端） → 流动性危机征兆（2013.6 钱荒、2016.9、2020 跨节）
       ↓
  当影子银行链条收缩时（资管新规、化债压降）：
  - 银行间资金紧张 → DR 系列飙升
  - 传导到零售端 → GC 系列飙升
  - 散户"用脚投票"回避 → 短期理财赎回
  - 形成"流动性紧张 → 影子银行收缩"的反向循环
```

### 7.3 判断 3：国债逆回购吸收了"准存款化资金"

```
2013-2018：
  - 余额宝等货币基金兴起
  - "银行存款 → 货币基金 → 银行间协议存款"的循环
  - 银行负债成本被推高
       ↓
2018 后：
  - 货币基金 T+0 赎回受限
  - 余额宝等开始分流到国债逆回购
  - 散户"现金管理工具箱"多元化
       ↓
⭐ 国债逆回购成为：
  - 货币基金的"竞争对手"
  - 准存款化资金的"分流器"
  - 客观上降低了银行对影子银行的负债依赖
```

### 7.4 判断 4：国债逆回购与"金融脱媒"的逆向操作

```
传统"金融脱媒"：
  存款人（个人/企业）→ 资本市场（直接融资）
       ↓
  这本应是好事，但中国路径变成了：
  存款人 → 银行理财 → 信托 → 城投/地产（影子银行）
       ↓
国债逆回购提供了一种"良性的脱媒"：
  存款人（散户）→ 国债逆回购 → 国债市场（标准化）
       ↓
  这是"脱媒到标准化市场"而非"脱媒到影子银行"
  ⭐ 政策含义：在金融脱媒的过程中，国债逆回购是中国版的"安全阀"
```

### 7.5 判断 5：未来发展方向——"散户级 ON RRP"是否会演变为新的政策工具？

```
观察点：
  - 央行是否可以通过"指定特殊平台"将 GC 系列纳入政策利率体系？
  - 证监会 + 央行是否可以联合创建"零售级回购便利工具"？
  - 数字人民币（e-CNY）是否可以接入 GC 平台？
       ↓
⭐ 趋势判断：
  - 国债逆回购作为"零售端利率"将越来越重要
  - 它是观察"政策利率传导效率"的最佳窗口
  - 但不会取代银行间市场作为主战场
  - 散户对 GC 利率的"体感"将成为央行政策的"最终考核指标"
```

## 八、风险与监管

### 8.1 风险分析

| 风险类型 | 是否存在 | 说明 |
|----------|----------|------|
| **信用风险** | **几乎为零** | 国债 + 中证登担保交收 + 标准化合约 |
| **市场风险** | **零** | 抵押品价值波动不影响已成交合约 |
| **流动性风险** | 极低 | T+1 资金可用，T+N+1 可提现 |
| **利率风险** | 低 | 利率波动影响新成交合约的收益，不影响已成交 |
| **操作风险** | 低 | 需要股票账户，操作失误可能买成"正回购"（融资方） |
| **政策风险** | 低 | 监管框架稳定，极端情况可能调整手续费或品种 |

### 8.2 监管框架

| 监管层 | 职责 |
|--------|------|
| **中国证监会** | 交易所市场总体监管 |
| **中证登** | 中央对手方、担保交收 |
| **沪深交易所** | 交易撮合、品种管理 |
| **中国人民银行** | 通过 OMO 利率间接影响 |

### 8.3 散户参与指南

```
✅ 适合人群：
  - 股市投资者（账户内有闲置资金）
  - 企业财务（短期闲置现金管理）
  - 任何想"现金管理升级"的人

❌ 不适合人群：
  - 想追求高收益（收益率 < 银行理财）
  - 没有股票账户
  - 完全不懂股票软件

📌 操作口诀：
  1. 节假日前 → 选 GC007/GC014（跨节利息累积）
  2. 季末/年末 → 任何 GC 都有机会（利率飙升）
  3. 周四 → GC001 实际占款 6 天（"周四效应"）
  4. 每天 14:30-15:00 → 观察尾盘利率（决定是否买入）
  5. 现金管理额度：建议保留部分 T+0 货币基金应对不时之需
```

### 10.1 国债逆回购 vs 货币基金

| 维度 | 国债逆回购 | 货币基金 |
|------|-----------|----------|
| 收益 | 1.5%-10%（波动大） | 1.5%-3%（稳定） |
| 流动性 | T+1 到期 | T+0/T+1 |
| 信用风险 | 几乎零 | 极低（但曾"打破净值"） |
| 操作复杂度 | 中（需股票账户） | 低（APP 直接买） |
| 适用场景 | 短期精准现金管理 | 长期闲置资金 |

### 10.2 国债逆回购 vs 同业存单

| 维度 | 国债逆回购 | 同业存单 |
|------|-----------|----------|
| 发行主体 | 融资方（券商/基金/企业） | 银行 |
| 抵押品 | 国债 | 无（银行信用） |
| 流动性 | 到期一次性 | 二级市场可转让 |
| 期限 | 1-182 天 | 1 个月-3 年 |
| 收益 | 与市场利率高度同步 | 略高于国债逆回购 |

### 10.3 三类"逆回购"概念厘清（再次强调）

```
1️⃣ 本页面主题：交易所国债逆回购（GC001/GC007等）
   - 沪深交易所
   - 散户可参与
   - 标准化场内合约
   - 中国本土特色

2️⃣ 央行 OMO 逆回购
   - 中国人民银行
   - 机构（银行）
   - 货币政策操作
   - 政策利率传导端

3️⃣ 美国 ON RRP（隔夜逆回购）

   - 合格机构（含 MMF）
   - 货币政策工具（利率走廊下限）
   - 美元体系
       ↓
⭐ 三者共同本质：
  都是"以国债/高质量债券为抵押的短期资金借贷"

  区别在于参与主体、政策目的、市场层级
```

## 十一、一句话总结

> **国债逆回购 = 中国版的"标准化平民级回购"**。
>
> 它不是影子银行（标准、场内、零信用风险），但它在中国影子银行体系中扮演了**关键的反向调节器**：
> - **分流散户闲置资金**：减少进入高风险影子银行的资金供给
> - **提供无风险替代**：让散户不需要"出走"到银行理财/信托
> - **传导政策利率**：是 OMO 利率传导到"零售端"的最终环节
> - **预警系统性风险**：GC001 > 10% 是金融压力的"温度计"
>
> **理解国债逆回购 = 理解中国金融体系"零售端"的现金管理逻辑 + 观察货币政策传导效率的最佳窗口**。
