# 囚徒困境

> 双方均按个体理性选择背叛，却得到比双方合作更差的结局——博弈论最经典的悖论，揭示个体理性与集体理性的冲突。

由 AI 系统化整理（2026-06-03），待补充原始来源素材。

囚徒困境（Prisoner's Dilemma, PD）由 Merrill Flood 和 Melvin Dresher 在 1950 年提出，Albert Tucker 将其形式化为"囚徒"故事。它是理解合作困境、机制设计、重复博弈、声誉机制的逻辑起点。

---

## 一、经典案例

### 1.1 故事原型

两名嫌犯被分别关押审讯，无法串供：

```
              嫌疑人 B
              坦白      抵赖
嫌 坦白   (-5, -5)  (0, -10)
疑
人 抵赖  (-10, 0)  (-1, -1)
A
收益单位：刑期年数（负数越大表示坐牢越久）
```

### 1.2 收益结构

| 情形 | A 收益 | B 收益 | 性质 |
|------|--------|--------|------|
| 都坦白 | -5 | -5 | 纳什均衡 |
| 都抵赖 | -1 | -1 | 帕累托最优 |
| A 坦白 B 抵赖 | 0 | -10 | A 占优 |
| A 抵赖 B 坦白 | -10 | 0 | B 占优 |

### 1.3 形式化条件

囚徒困境结构需满足（对两参与者）：

```
T > R > P > S
  ↑         ↑
  背叛诱惑  傻瓜收益
```

| 符号 | 含义 | 故事中 |
|------|------|--------|
| T (Temptation) | 对方合作我背叛的收益 | 0 |
| R (Reward) | 双方合作的收益 | -1 |
| P (Punishment) | 双方背叛的收益 | -5 |
| S (Sucker) | 对方背叛我合作的收益 | -10 |

> 关键：**双方合作优于双方背叛**（R > P），但**单方背叛又优于合作**（T > R 且 P > S）。这种结构使得"背叛"成为占优策略。

## 二、占优策略分析

### 2.1 迭代剔除

```
A 的视角：
  - 若 B 坦白：A 坦白 -5，A 抵赖 -10 → 选坦白
  - 若 B 抵赖：A 坦白 0，A 抵赖 -1 → 选坦白
∴ 坦白是 A 的严格占优策略

对称地，坦白是 B 的严格占优策略

→ 唯一纳什均衡：(坦白, 坦白) → (-5, -5)
```

### 2.2 困境的根源

| 视角 | 选择 | 理由 |
|------|------|------|
| 个体理性 | 背叛 | 不论对方如何，背叛更优 |
| 集体理性 | 合作 | 双方都合作总收益最高 |
| 实际结果 | 双方背叛 | 个体理性胜出 → 集体次优 |

> 囚徒困境的核心矛盾：**个体最优 ≠ 集体最优**。这是制度设计、契约理论、博弈论存在的根本原因。

## 三、N 人囚徒困境

### 3.1 公共品博弈

$n$ 个参与者，每人有 $c$ 元投入公共品池，池子按 $k$ 倍放大后均分（$1 < k < n$）：

```
参与者的边际收益 = k/n × c
参与者的边际成本 = c

私人理性条件：k/n < 1 → 个体不愿投入
集体最优条件：k > 1 → 大家投入可获更大池子
```

### 3.2 公地悲剧

```
n 个牧羊人共用一片草地
├─ 每人多放 1 头羊 → 个人收益 +1
├─ 但草地过度放牧 → 所有人收益减少
└─ 纳什均衡：所有人都过度放牧 → 公地荒废
```

哈丁 (1968) 提出：公地悲剧是 N 人囚徒困境的典型表现。

### 3.3 现实映射

| 囚徒困境结构 | 现实案例 |
|-------------|----------|
| 军备竞赛 | 双方裁军更好，但单方裁军会被对方超越 |
| 价格战 | 双方维持价格更好，但单方降价抢市场 |
| 学术不端 | 双方诚信更好，但单方造假抢先发表 |
| 员工偷懒 | 双方努力更好，但单方摸鱼占便宜 |
| 过度捕捞 | 见 3.2 公地悲剧 |
| 减排博弈 | 双方减排更好，但单方减排损失竞争力 |

## 四、走出困境的机制

### 4.1 重复博弈（最重要）

- 单次博弈中背叛是占优策略
- **无限次重复**博弈下，合作可成为均衡（详见 [[占优策略与重复博弈]]）
- 触发策略（Grim Trigger）维持合作：偏离一次则永远背叛

### 4.2 声誉机制

- 长期博弈中参与者会顾及未来收益
- 声誉如同抵押品，约束短期背叛冲动
- 现实：商业信用、Karma 系统、品牌价值

### 4.3 惩罚机制

| 机制 | 原理 | 现实 |
|------|------|------|
| 第三方惩罚 | 法律、监管、平台规则 | 反垄断法、证监会处罚 |
| 自发惩罚 | 同伴报复、社交排斥 | 行业协会、公开谴责 |
| 算法机制 | 以代码强制执行 | 智能合约、链上惩罚 |

### 4.4 制度与产权

- 科斯定理：明确产权 → 谈判解决外部性
- 赫维茨机制：激励相容的制度设计
- 现实中：捕捞配额、排放交易、专利保护

### 4.5 改变收益结构

- 把单次博弈改为多轮博弈（增加未来权重）
- 改变背叛的收益（如提高违规成本）
- 引入利他偏好/互惠偏好（行为改造）

## 五、现实金融案例

### 5.1 银行挤兑

```
所有储户：信任银行 → 银行稳健 → 大家都好
少数储户：挤兑取款 → 引发连锁 → 银行倒闭
→ 个体理性挤兑 → 集体次优（银行破产）
```

> Diamond-Dybvig (1983) 模型：银行挤兑是经典的囚徒困境，缺乏存款保险时尤甚。

### 5.2 行业价格战

```
两家航空公司：
├─ 双方都维持高价：各赚 100
├─ 一方降价抢市场：降价者赚 150，对方亏 50
└─ 双方都降价：各赚 30

→ 纳什均衡是双方都降价 → 价格战 → 全行业受损
```

### 5.3 公募基金"抱团"

- 集体重仓白马股（合作）→ 个股估值合理抬升
- 一方提前减仓（背叛）→ 先跑者得利
- 全体减仓（互相背叛）→ 估值坍塌
- 行为金融学中的"机构囚徒困境"

## 六、理论意义

### 6.1 学术地位

| 贡献 | 内容 |
|------|------|
| 理论基石 | 后续研究合作演化、机制设计、行为博弈的起点 |
| 实证工具 | 用于实验经济学（PD 实验）检验人类合作行为 |
| 跨学科 | 生物学（利他行为）、政治学（投票悖论）、社会学（集体行动） |
| 多次重复 | Robert Axelrod《合作的演化》(1984) |

### 6.2 经典变体

| 变体 | 特点 |
|------|------|
| 迭代囚徒困境 (IPD) | 重复博弈，研究触发策略的稳定性 |
| N 人囚徒困境 | 公共品博弈，搭便车问题 |
| 空间囚徒困境 | 引入网络结构，合作可演化 |
| 连续囚徒困境 | 投资水平/努力程度连续可调 |
| 噪声囚徒困境 | 引入执行误差，需宽容策略 |
