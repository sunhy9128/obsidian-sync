# 长臂管辖

> Long-arm Jurisdiction — 美国金融制裁全球威慑的核心机制

**长臂管辖**（Long-arm Jurisdiction）原本是普通法系中的法律概念，指一国法院对非本国公民或非本国境内发生的法律行为行使管辖权。在国际政治与金融领域，长臂管辖特指**美国通过国内立法，将原本适用于美国境内或美国公民的制裁规则，扩展适用于与美国有联系的境外实体（包括第三国企业、银行和个人）**的现象。它是 21 世纪以来美国实施"金融武器化"的核心机制，也是引发各国反弹的关键制度根源。

---

## 一、核心定义与法律基础

### 1.1 定义

长臂管辖是指**美国将其国内法的适用范围扩展到境外，对与美国有"最低联系"的第三国实体行使管辖权**的实践。在金融领域，它的核心表现是：

```
【长臂管辖的本质】
├── 美国国会通过国内立法（如《国际紧急经济权力法》IEEPA）
├── 美国财政部 OFAC 制定制裁名单
├── 美国法院和执法机构对全球实体执行
├── 任何使用美元或与美国有联系的实体都在管辖范围内
└── 违反 = 被美国金融体系排斥
```

### 1.2 法律基础

| 法律 | 全称 | 主要用途 |
|------|------|---------|
| **IEEPA** | International Emergency Economic Powers Act | 总统宣布国家紧急状态时实施经济制裁 |
| **NDAA** | National Defense Authorization Act | 国防授权法（年度立法）|
| **TWEA** | Trading with the Enemy Act | 对敌国贸易管制 |
| **OFAC 法规** | Office of Foreign Assets Control Regulations | 财政部外国资产控制办公室具体执行 |
| **爱国者法案** | USA PATRIOT Act | 反恐融资与制裁 |
| **2017 CAATSA** | Countering America's Adversaries Through Sanctions Act | 对俄罗斯、伊朗、朝鲜制裁法案 |
| **2024 NDS** | National Defense Strategy | 国家防务战略中的长臂管辖 |

### 1.3 与传统管辖权的区别

```
【传统管辖权】
├── 属地管辖：行为发生地
├── 属人管辖：行为人国籍
├── 保护性管辖：涉及本国重大利益
└── 普遍管辖：国际罪行

【长臂管辖】
├── "最低联系"标准
├── 美元交易、美国技术、美国公民参与
├── 全球任何实体都可能"被管辖"
└── "连接点"非常宽松
```

## 二、长臂管辖的运作机制

### 2.1 五大触发点

```
【触发点 ①：使用美元】
├── 任何交易涉及美元清算（CHIPS）
├── 即使在第三国境内
└── → 落入美国管辖

【触发点 ②：使用美国技术】
├── 任何产品含有美国零部件 / 软件
├── 即使最终用户在欧洲或亚洲
└── → 落入美国管辖

【触发点 ③：与美国实体有联系】
├── 任何美国公司、银行、个人的参与
├── 即使只是供应链中的一个小环节
└── → 落入美国管辖

【触发点 ④：通过美国金融机构】
├── 任何交易经过美国银行
├── 即使只是中转
└── → 落入美国管辖

【触发点 ⑤：美国公民 / 永久居民参与】
├── 美国公民无论身在何处
├── 永久居民、绿卡持有者
└── → 落入美国管辖
```

### 2.2 五大执行工具

```
【工具 ①：列入制裁名单（SDN List）】
├── 个人/企业被列入"特别指定国民"清单
├── 资产被冻结
├── 美国实体被禁止与其交易
└── 全球金融体系的"经济死亡"

【工具 ②：罚款】
├── 对违反制裁的外国企业处以巨额罚款
├── 典型案例：
│   ├── 汇丰银行：$19 亿（违反制裁）
│   ├── 法国巴黎银行：$89 亿（违反制裁）
│   ├── 阿尔斯通：$7.7 亿（伊朗交易）
│   └── 德意志银行：多次巨额罚款
└── 震慑效果极强

【工具 ③：禁止美元清算】
├── 切断外国银行的美元清算通道
├── 实际上等同于"金融死亡"
└── 配合 SWIFT 切断形成"双重绞杀"

【工具 ④：禁止美国市场准入】
├── 禁止外国企业进入美国市场
├── 禁止美国企业与其合作
└── "逐出美国市场"的威慑

【工具 ⑤：引渡高管】
├── 通过国际刑警组织发布红色通缉令
├── 极端情况下引渡外国企业高管
└── 阿尔斯通高管皮耶鲁齐案是典型
```

### 2.3 长臂管辖的传导链

```
【完整传导链】

美国 OFAC 制裁名单公布
    ↓
全球金融机构合规审查（KYC/AML）
    ↓
任何与被制裁方有联系的交易被拒绝
    ↓
被制裁方被国际金融体系孤立
    ↓
被制裁方的供应链 / 客户 / 合作方主动切断关系
    ↓
被制裁方经济活动大幅收缩
    ↓
被制裁方被迫改变政策或承受巨大损失

【传导链的关键特征】
├── 速度极快（电子化处理，秒级）
├── 全球同步（11000+ SWIFT 银行同步执行）
├── 自增强效应（金融机构主动合规，避免风险）
└── 实际效果远超"法律管辖权"范围
```

## 三、长臂管辖的历史演进

### 3.1 五个发展阶段

```
【Phase 1（1990s）：萌芽期】
├── 1996：赫尔姆斯-伯顿法（对古巴）
│   └── 允许美国法院审理外国公司与古巴的交易
├── 1996：达马托法（对利比亚、伊朗）
│   └── 对与伊朗能源领域合作的外国公司实施制裁
└── 标志：美国开始系统性地将国内法国际化

【Phase 2（2001-2010）：反恐扩张期】
├── 2001：9/11 后通过爱国者法案
├── 反恐融资制裁大量使用
├── OFAC 制裁名单迅速扩大
└── 长臂管辖体系化

【Phase 3（2010-2016）：大国博弈期】
├── 2012：切断伊朗 SWIFT
├── 2014：克里米亚事件后对俄制裁
├── 2015：开始对华为等中国企业实施制裁
└── 长臂管辖被"武器化"

【Phase 4（2017-2022）：全面武器化】
├── 2017：CAATSA 法案签署
├── 2018：退出伊核协议，重启对伊全面制裁
├── 2020：香港自治法
├── 2021：对中国涉疆、涉港官员制裁
├── 2022：对俄罗斯"金融核弹"
└── 长臂管辖成为美国对外政策的核心工具

【Phase 5（2023-至今）：反制与多元化】
├── 各国立法反制长臂管辖
│   ├── 欧盟《阻断法令》升级
│   ├── 中国《阻断外国法律与措施不当域外适用办法》
│   └── 俄罗斯《反制裁法》
├── 替代金融基础设施建设
├── 去美元化加速
└── 长臂管辖边际效用下降
```

### 3.2 标志性法案

| 年份 | 法案 | 主要内容 |
|------|------|---------|
| 1996 | 赫尔姆斯-伯顿法 | 对古巴，授权美国法院审理外国公司 |
| 1996 | 达马托法 | 对伊朗、利比亚能源领域 |
| 2017 | CAATSA | 对俄、伊、朝制裁法案 |
| 2020 | 香港自治法 | 对香港自治受损的制裁 |
| 2021 | 维吾尔强迫劳动预防法 | 对中国新疆相关供应链制裁 |
| 2024 | 反外国对手信息操作法案 | 信息战维度 |

## 四、长臂管辖的典型案例

### 4.1 阿尔斯通案（法国）

```
【阿尔斯通案（2013-2015）】

背景：
├── 阿尔斯通是法国能源巨头
├── 在伊朗有电力设备业务
├── 美国通用电气（GE）是其竞争对手

过程：
├── 2013：阿尔斯通高管皮耶鲁齐在纽约机场被捕
├── 美国指控：阿尔斯通在印尼贿赂官员（实际为换取伊朗业务）
├── 美国施压：要求阿尔斯通出售核心业务给 GE
└── 2015：阿尔斯通核心业务（电力）被 GE 收购

影响：
├── 法国"工业明珠"被美国公司收购
├── 暴露美国长臂管辖的"商业武器化"
├── 法国马克龙政府后来推动"战略自主"
└── 成为各国警惕长臂管辖的"标志性事件"
```

### 4.2 汇丰银行案（2012）

```
【汇丰银行洗钱案（2012）】

背景：
├── 汇丰银行在全球为毒贩、恐怖分子洗钱
├── 同时违反美国对伊朗、缅甸、古巴等的制裁
└── 被美国司法部调查

过程：
├── 2012：美国司法部起诉
├── 汇丰选择"延期起诉协议"（DPA）
├── 支付 $19 亿美元罚款
└── 任命独立监察员整改

影响：
├── 全球银行业的合规成本飙升
├── KYC / AML 成为银行业核心运营成本
└── "过度合规"成为金融机构的自我保护机制
```

### 4.3 法国巴黎银行案（2014）

```
【法巴银行违反制裁案（2014）】

背景：
├── 法国巴黎银行（BNP）为苏丹、伊朗、古巴处理美元交易
├── 违反美国制裁规定
└── 美国调查持续多年

结果：
├── 罚款 $89 亿美元（创纪录）
├── 13 名高管被禁止进入纽约金融业
├── 美元清算业务暂停 1 年
├── 法国央行行长被迫表态

影响：
├── 欧洲银行业对长臂管辖"深恶痛绝"
├── 推动欧盟"阻断法令"升级
└── 暴露美欧金融不对称依赖
```

### 4.4 中国华为案（2018-至今）

```
【华为长臂管辖案】

背景：
├── 2018：孟晚舟在加拿大被捕
├── 美国指控：华为违反美国对伊朗制裁
│   ├── 通过 Skycom 与伊朗交易
│   └── 涉嫌欺诈银行
├── 引渡要求 → 中美加三国博弈
└── 2021：孟晚舟获释回国

后续：
├── 美国对华为实施"实体清单"管制
├── 全球半导体企业对华为断供
├── 华为业务大幅收缩（手机业务）
└── 推动中国加速芯片自主化

战略影响：
├── 暴露美国长臂管辖的"科技武器化"
├── 推动中国加速科技自立
└── 强化中国对长臂管辖的反制决心
```

### 4.5 昆仑银行案

```
【昆仑银行案】

背景：
├── 昆仑银行（中国）为伊朗提供金融服务
├── 美国指控其违反制裁
├── 美国财政部警告：与昆仑银行交易的外国银行可能被切断美元清算

结果：
├── 昆仑银行主动收缩伊朗业务
├── 转向本币结算（人民币-伊朗里亚尔）
├── 成为中国规避长臂管辖的样本
└── 推动 CIPS 与伊朗合作

战略意义：
├── 中国金融机构"以退为进"
├── 主动规避长臂管辖，发展替代方案
└── 推动人民币在伊朗的使用
```

## 五、长臂管辖的反作用力

### 5.1 五大反制措施

```
【反制 ①：阻断法令】
├── 欧盟《阻断法令》（1996，2018 年升级）
├── 中国《阻断外国法律与措施不当域外适用办法》（2021）
├── 俄罗斯《反制裁法》
└── 法律上明确"不承认"美国长臂管辖

【反制 ②：替代金融基础设施】
├── CIPS（人民币跨境支付）
├── SPFS（卢布清算）
├── INFT（印度版 SWIFT）
├── mBridge（多边 CBDC 桥）
└── 减少对美元清算的依赖

【反制 ③：本币结算】
├── 中俄能源贸易 90%+ 人民币
├── 中沙石油部分人民币结算
├── 东盟 LCS（本地货币结算）扩大
└── 减少美元交易

【反制 ④：央行储备多元化】
├── 减持美债
├── 增持黄金、人民币、欧元
└── 减少美元资产的"可制裁性"

【反制 ⑤：技术自主】
├── 中国芯片自主化
├── 欧亚供应链"去美化"
└── 减少对美国技术的依赖
```

### 5.2 各国的反制策略对比

| 国家 | 主要反制手段 | 进度 |
|------|-------------|------|
| **欧盟** | 阻断法令 + INSTEX | 法律上完整，实际效用有限 |
| **中国** | 阻断办法 + CIPS + 一带一路 + 技术自主 | 系统性、渐进、持续 |
| **俄罗斯** | SPFS + 反制裁法 + 卢布结算令 | 全面但代价极高 |
| **伊朗** | SEPAM + 区域合作 + 加密货币 | 局部、非常规 |
| **印度** | UPI / INFT（探索）| 起步阶段 |
| **沙特** | 考虑部分人民币结算 | 谈判中 |

## 六、长臂管辖与金融武器化的耦合

### 6.1 长臂管辖是金融武器化的"使能器"

```
【没有长臂管辖的金融武器化】

金融制裁只能针对：
├── 美国境内的资产
├── 美国公民
├── 与美国有直接联系的实体
└── 范围有限

【有了长臂管辖的金融武器化】

金融制裁可以针对：
├── 全球任何使用美元的交易
├── 全球任何使用美国技术的实体
├── 全球任何与美国有联系的实体
└── 范围几乎无限
```

### 6.2 长臂管辖的"边际效用递减"

```
【使用越多，边际效用越低】

美国对 A 国制裁 → A 国反制
美国对 B 国制裁 → B 国反制
美国对 C 国制裁 → C 国反制
    ↓
全球范围内形成"长臂管辖疲劳"
    ↓
各国加速去美元化 + 替代系统建设
    ↓
美国长臂管辖的"震慑力"下降
    ↓
美国需要更频繁地使用长臂管辖
    ↓
进一步推动各国反制
    ↓
"长臂管辖悖论"形成
```

### 6.3 长臂管辖的"成本—收益"分析

```
【美国长臂管辖的"收益"】
├── 短期：迫使对手让步（如阿尔斯通案）
├── 中期：维护美元霸权
└── 长期：可能加速美元霸权终结

【美国长臂管辖的"成本"】
├── 短期：执法成本（司法部、OFAC 等）
├── 中期：与盟友摩擦（欧盟、中国、俄罗斯）
├── 长期：全球去美元化加速
└── 隐性：损害美国"法治国家"形象
```

## 七、关键洞察

> [!key-insight] 长臂管辖 = 美元霸权的"放大器"
>
> 美元霸权本身只让美国有"特权"，但长臂管辖让美国有"执法权"。两者结合形成"金融帝国"的完整权力结构。

> [!key-insight] 长臂管辖 = 美元武器化的"使能器"
>
> 没有长臂管辖，制裁只能影响美国境内；有长臂管辖，制裁可以影响全球。这是金融武器化能够"小成本、大效果"的关键。

> [!contradiction] 长臂管辖悖论
>
> 美国越是使用长臂管辖维护美元霸权，美元霸权的根基（"中立性"）就越被侵蚀。这是一个"使用越多、效用越低"的悖论。

> [!key-insight] 长臂管辖的"反向创新激励"
>
> 长臂管辖推动了全球"非美元清算基础设施"建设。从这个意义上说，长臂管辖是"被迫创新"的催化剂——如果没有长臂管辖，CIPS、SPFS、mBridge 等可能不会如此快速发展。

## 八、未来演进

### 8.1 短期（1-3 年）

```
├── 长臂管辖继续被使用
├── 各国反制力度加大
├── 替代金融基础设施持续扩容
└── "阻断法令"类法律被更多国家采用
```

### 8.2 中期（3-7 年）

```
├── 长臂管辖边际效用显著下降
├── 全球形成"区域分割"的清算格局
├── 数字货币跨境支付成为新战场
└── 长臂管辖难以延伸到数字货币领域
```
