# 存款保险（Deposit Insurance）

> 金融安全网的第一层——保护存款人，防止挤兑蔓延——1933 年大萧条的教训

## 一、核心定义

**存款保险**（Deposit Insurance）是**国家建立的、为存款人提供保障的制度安排**：当存款银行倒闭时，**存款保险机构**（Deposit Insurance Corporation）按照**法定限额**对存款人进行**赔付**，防止存款人因银行倒闭而遭受损失，从而**防止挤兑蔓延**、**维护金融稳定**。

**一句话定义**：**存款保险 = 政府对存款人的"隐性担保" + 显性化的赔付机制**。

**核心要素**：
1. **强制性**：法律强制所有存款银行参加
2. **有限性**：只赔付法定限额内的存款
3. **基金化**：银行事前缴费形成保险基金
4. **监管配合**：与央行最后贷款人、监管机构协同

**国际实践**：
- 美国：FDIC（1933 年成立）
- 欧元区：DGSD（存款担保计划指令，2014）
- 英国：FSCS（金融业赔偿计划）
- 中国：存款保险（2015 年正式实施）
- 日本：DICJ（存款保险机构，1971）
- 加拿大：CDIC（1967）
- 印度：DICGC（1961）

## 二、存款保险的历史起源

### 2.1 大萧条的教训（1929-1933）

```
1929-1933 大萧条：

背景：
  - 美国 1920s 经济繁荣
  - 银行大量贷款、投机房地产
  - 1929 股市崩盘

银行倒闭潮：
  - 1930-1933：9000+ 家银行倒闭
  - 占当时银行总数 40%+
  - 储户损失巨大
  - 信心崩塌

关键问题：
  - 没有存款保险
  - 储户损失 100%
  - 一有风吹草动就挤兑
  - 挤兑导致更多银行倒闭
  - 死亡螺旋
       ↓
教训：
  - 必须保护存款人
  - 必须打破"银行倒闭→挤兑→更多倒闭"的循环
  - 必须建立显性的存款保险
```

### 2.2 1933 年 FDIC 成立

```
罗斯福新政（1933）：

1933.6 Glass-Steagall Act 通过：
  - 银行业与投资银行业分离
  - 创建联邦存款保险公司（FDIC）
  - 1934.1 FDIC 正式运营

1933 Banking Act：
  - 联邦存款保险
  - 联邦监管
  - 利率管制（Q 条例）

罗斯福就职演讲（1933.3）：
  - "The only thing we have to fear is fear itself"
  - "我们唯一应该恐惧的是恐惧本身"
  - 重建信心是关键
       ↓
FDIC 的最初设计：
  - 保险额度：2500 美元（1934）
  - 现在 25 万美元
  - 银行事前缴费
  - 形成保险基金
  - 监管与保险合一
```

### 2.3 历史演进

```
1934-1980：
  - FDIC 保险额度 2500 → 4 万美元
  - 主要保护小额存款人
  - 美国银行业相对稳定

1980-1991（储贷危机）：
  - 1980-1988：1000+ 家储贷协会（S&L）倒闭
  - FSLIC（储贷保险）破产
  - 1989 FIRREA 法案：
    - 解散 FSLIC
    - 由 FDIC 接管
    - 重建保险基金
  - 教训：保险基金必须充足

1991-2008：
  - 保险额度 10 万美元
  - 银行业相对稳定
  - 但 2008 危机考验

2008 危机：
  - 150+ 家银行倒闭
  - FDIC 处置 150 家
  - 2008.10 紧急上调临时保险额度至 25 万美元
  - 2010 多德弗兰克法案：永久 25 万美元

2010 后：
  - 大银行倒闭减少
  - 2023 SVB 倒闭
  - 存款保险制度保持稳健
```

## 三、存款保险的运作机制

### 3.1 核心机制

```
存款保险的运作机制：

1. **强制参保**
   - 法律强制所有存款类金融机构参加
   - 银行定期缴费
   - 形成保险基金

2. **限额赔付**
   - 法定赔付限额
   - 美国 FDIC：25 万美元/人/银行
   - 中国：50 万元人民币/人/银行
   - 超出部分按清算资产分配

3. **事前缴费**
   - 银行按存款规模缴费
   - 风险调整（高风险银行缴更多）
   - 形成保险基金

4. **监管配合**
   - 存款保险机构 + 央行 + 银行监管
   - 三方协同

5. **银行处置**
   - 银行倒闭时
   - FDIC（美国）接管
   - 偿付存款人
   - 处置资产
```

### 3.2 赔付流程

```
银行倒闭时的赔付流程（以美国 FDIC 为例）：

Step 1：FDIC 接管
  - 监管机构认定"濒临倒闭"
  - FDIC 接管（通常是周五晚/周末）
  - 防止挤兑蔓延

Step 2：评估资产
  - 评估银行资产
  - 计算缺口（负债 > 资产）

Step 3：选择处置方案
  - 收购与承接（P&A）：卖给其他银行
  - 桥银行：临时运营
  - 直接偿付：按限额赔付存款人

Step 4：赔付存款人
  - 保险存款（25 万内）：立即赔付
  - 超出部分：清算资产后按比例偿付

Step 5：资产处置
  - 出售资产
  - 收回部分资金
  - 补充保险基金
```

### 3.3 保险额度的设计

```
保险额度的设计考量：

1. **覆盖范围**
   - 美国：25 万美元（覆盖 ~98% 存款）
   - 中国：50 万元（覆盖 ~99% 存款）
   - 欧元区：10 万欧元（覆盖 ~95% 存款）
   - 加拿大：10 万加元

2. **设计原则**
   - 覆盖大部分小额存款人
   - 大额存款人需要"自我监管"
   - 防止道德风险（过度冒险）

3. **动态调整**
   - 美国：1991 后未调整
   - 中国：2015 实施后未调整
   - 部分国家根据通胀调整

4. **临时上调**
   - 2008 美国：临时上调至 25 万
   - 危机时可临时上调
   - 2020 部分国家临时上调
```

### 3.4 保险基金的筹集

```
保险基金的筹集方式：

1. **事前缴费**
   - 银行按存款规模缴费
   - 美国 FDIC：年化 0.05-0.4%（根据风险）
   - 形成保险基金

2. **事后融资**
   - 基金不足时
   - 财政部借款
   - 但有上限

3. **基金规模目标**
   - 美国：占保险存款的 1.35%
   - 中国：占投保存款的 1.5%（目标）
   - 实际规模：
     - 美国 DIF：~1000 亿美元
     - 中国存款保险基金：~900 亿元人民币

4. **风险调整费率**
   - 高风险银行缴更多
   - 低风险银行缴更少
   - 体现"风险定价"
```

## 四、存款保险的五大功能

### 4.1 功能一：保护存款人

```
存款保险的核心功能：

1. **保护小额存款人**
   - 普通家庭储蓄
   - 小企业存款
   - 退休基金
   - 让存款人"无风险"

2. **防止存款人恐慌**
   - 存款人无需担心银行倒闭
   - 不需要"跑在前头"
   - 减少挤兑动机
```

### 4.2 功能二：防止挤兑蔓延

```
存款保险如何防止挤兑：

传统挤兑：
  - 银行 A 倒闭
  - 储户恐慌
  - 集体到银行 B 提款
  - 银行 B 流动性枯竭
  - 银行 B 倒闭
  - 蔓延至银行 C
  - 死亡螺旋

存款保险后：
  - 银行 A 倒闭
  - 存款人得到赔付
  - 银行 B 储户不恐慌（因为保险）
  - 不会蔓延
       ↓
存款保险 = "终止传染"的防火墙
```

### 4.3 功能三：降低银行融资成本

```
存款保险对银行融资成本的影响：

1. **存款利率更低**
   - 因为有保险
   - 银行融资成本降低
   - 可以提供更低利率贷款

2. **存款基础更稳定**
   - 不需要担心挤兑
   - 可以长期持有存款
   - 支持长期贷款

3. **金融危机成本降低**
   - 防止系统性风险
   - 减少央行救助需要
   - 减少纳税人负担
```

### 4.4 功能四：支持金融监管

```
存款保险与监管的关系：

1. **风险监测**
   - 存款保险机构是"风险监测者"
   - 必须评估银行风险
   - 推动银行审慎经营

2. **早期干预**
   - 发现问题及时干预
   - 避免银行倒闭
   - 减少保险基金支出

3. **监管协调**
   - 央行 + 监管 + 存款保险
   - 三方协同
   - "金融安全网"
```

### 4.5 功能五：处置问题银行

```
存款保险机构也是"问题银行处置者"：

1. **接管**
   - 银行濒临倒闭时接管
   - 防止挤兑

2. **处置**
   - 出售、合并、桥银行
   - 最小化处置成本

3. **追偿**
   - 追究高管责任
   - 处置资产
   - 补充保险基金
```

## 五、存款保险 vs 最后贷款人

### 5.1 关键对比

| 维度 | 存款保险 | 最后贷款人 |
|------|---------|----------|
| **覆盖** | 存款人 | 金融机构 |
| **触发** | 银行倒闭 | 系统性危机 |
| **提供方** | 存款保险机构 | 中央银行 |
| **工具** | 偿付存款 | 紧急贷款、QE |
| **资金来源** | 保险基金 | 央行印钞权 |
| **道德风险** | 中等 | 高 |
| **典型法律地位** | 显性（多数国家） | 隐性（多数国家） |

### 5.2 协同关系

```
金融安全网的双层防护：

第一层：存款保险
  - 处理"个别银行倒闭"
  - 保护存款人
  - 防止挤兑蔓延

第二层：最后贷款人
  - 处理"系统性风险"
  - 保护金融机构
  - 提供流动性

协同：
  - 存款保险 + 最后贷款人 + 监管 = 金融安全网
  - 三者缺一不可
  - 必须协同运作
```

### 6.2 美国 FDIC 详解

```
美国 FDIC：

成立：1933.6（通过 Glass-Steagall Act）
正式运营：1934.1

结构：
  - 董事会 5 人
  - 主任 1 名（总统任命）
  - 3 名由 OCC、Fed 主任兼任
  - 1 名公共代表

职能：
  1. 存款保险
     - 25 万美元/人/银行
  2. 银行监管
     - 州非会员银行
     - 互助储蓄银行
  3. 银行处置
     - 接管濒临倒闭银行
     - 处置资产
     - 偿付存款人

资金来源：
  - DIF（存款保险基金）
  - 2024 规模：~1000 亿美元
  - 目标：占保险存款的 1.35%

历史关键时刻：
  - 1933 创立
  - 1984 大陆伊利诺伊银行
  - 1980s 储贷危机
  - 2008 全球危机
  - 2023 SVB
```

### 6.3 中国存款保险详解

```
中国存款保险制度：

实施时间：
  - 2015.5.1《存款保险条例》正式实施
  - 由国务院发布

机构：
  - 存款保险基金管理公司
  - 国务院主管
  - 中国人民银行指导
  - 独立法人

保险额度：
  - 50 万元人民币/人/银行
  - 覆盖 99%+ 存款人

基金规模：
  - 2015 起步：~500 亿元
  - 2024 规模：~900 亿元
  - 目标：1.5% 投保存款

特点：
  - 较晚实施（2015）
  - 与央行最后贷款人配合
  - 与金融监管总局协调
  - 渐进式实施

挑战：
  - 部分中小银行风险
  - 存款集中度
  - 隐性担保文化
```

### 6.4 欧盟 DGSD

```
欧盟存款担保计划指令（DGSD）：

通过时间：2014
实施时间：各成员国分阶段

保险额度：
  - 10 万欧元/人/银行
  - 覆盖 ~95% 存款

特点：
  - 统一框架
  - 各成员国具体实施
  - 跨境存款保护
  - 与单一清算机制（SRM）配合

资金来源：
  - 各成员国存款保险机构
  - 银行缴费
  - 跨机构资金支持

关键案例：
  - 2012 西班牙 Bankia：存款人得到赔付
  - 2013 塞浦路斯危机：临时征收存款税（引发争议）
  - 2023 瑞信：瑞士特殊处理
```

## 七、存款保险的五大争议

### 7.1 争议一：道德风险

```
存款保险的道德风险问题：

问题：
  - 存款人"无风险"
  - 不关心银行风险
  - 银行冒险成本降低
  - 可能引发更大危机

体现：
  - 银行高息揽存（风险定价）
  - 银行进行高风险投资
  - 银行过度杠杆
  - "太大而不能倒"
       ↓
应对：
  - 限额赔付（保护小额）
  - 风险调整费率
  - 监管配合
  - "大额存款人"承担部分风险
```

### 7.2 争议二：保险额度设计

```
保险额度的争议：

观点一：高额度（覆盖 99%+ 存款）
  - 优点：社会稳定
  - 缺点：高道德风险

观点二：低额度（覆盖 80-90% 存款）
  - 优点：减少道德风险
  - 缺点：可能引发恐慌

国际差异：
  - 美国 25 万：覆盖 98%
  - 中国 50 万：覆盖 99%
  - 欧元区 10 万欧：覆盖 95%
  - 加拿大 10 万加元：覆盖 90%
       ↓
共识：
  - 覆盖 90-99% 存款人
  - 不覆盖"太大"的存款
  - 体现"小额必保、大额自负"
```

### 7.3 争议三：基金充足性

```
基金充足性问题：

问题：
  - 保险基金可能不足
  - 大型银行倒闭可能超过基金规模
  - SVB 资产 2090 亿，远超基金规模
       ↓
应对：
  - 基金规模目标（占保险存款 1-2%）
  - 财政部紧急借款
  - 央行最后贷款人支持
  - 风险调整费率
       ↓
问题：
  - 基金规模永远不够大
  - "big enough to fail"难设计
  - 实际上依赖央行最后贷款人
```

### 7.4 争议四：跨境存款保护

```
跨境存款保护的争议：

问题：
  - 银行跨境经营
  - 存款人在外国银行有存款
  - 哪个国家的存款保险？
  - 跨境合作困难

案例：
  - 2008 冰岛 Icesave：英国与荷兰储户
  - 2013 塞浦路斯：欧盟存款
  - 2023 瑞信：跨境存款

应对：
  - 欧盟 DGSD 协调
  - 双边协议
  - 但仍有"灰色地带"
```

### 7.5 争议五：银行处置方式

```
银行处置方式的争议：

传统方式：
  - 偿付存款人
  - 出售资产
  - 清算

现代方式：
  - 收购与承接（P&A）
  - 桥银行
  - "bail-in"（自救）
       ↓
问题：
  - 哪种方式最优？
  - 谁承担损失？
  - 股东？债权人？存款人？
  - 道德风险影响？
       ↓
现代趋势：
  - 优先 P&A（影响最小）
  - 桥银行（暂时）
  - 偿付（最后）
  - bail-in（股东、债权人）
```

## 八、存款保险与 2008 危机

### 8.1 2008 危机中的存款保险

```
2008 全球危机：

FDIC 的应对：
  - 2008 年处置 150+ 家银行
  - 绝大多数是小银行
  - 5 大投行未被 FDIC 处置
    - 贝尔斯登：被收购
    - 雷曼：破产
    - 美林：被收购
    - 高盛、摩根士丹利：转银行控股

关键事件：
  - 2008.10 紧急上调临时保险额度至 25 万
  - 2008.10.14 临时全额担保所有无保险存款
    - 这是 EESA 法案（TARP 7000 亿）的一部分
    - 防止系统性恐慌

教训：
  - 临时全额担保是必要的
  - 但有道德风险
  - 必须有退出策略
```

### 8.2 2008 后的多德弗兰克改革

```
多德弗兰克对存款保险的改革：

1. 保险额度永久上调
   - 10 万 → 25 万美元（2010）
   - 永久化

2. OLA（有序清算机制）
   - 大银行有序倒闭
   - FDIC 权力扩大
   - 但实际上不常动用

3. DFA Stress Tests
   - 大银行必须通过
   - 减少倒闭可能性

4. 巴塞尔 III 实施
   - 减少银行倒闭
   - 减少保险基金支出

详见 [[wiki/concepts/多德弗兰克法案]]
```

## 十、存款保险与道德风险

### 10.1 道德风险的根源

```
存款保险道德风险的根源：

1. **存款人不监督**
   - 不关心银行风险
   - 不"用脚投票"
   - 选择高息存款（不顾风险）

2. **银行冒险成本降低**
   - 即使倒闭，存款人得到赔付
   - 银行可以"高息揽存"
   - 冒险有"隐性担保"

3. **"太大而不能倒"**
   - 大银行倒闭引发系统性风险
   - 实际会被救助
   - 进一步鼓励冒险
```

### 10.2 应对道德风险的机制

```
应对道德风险的机制：

1. **限额赔付**
   - 25 万美元（中国 50 万）
   - 大额存款人需要"自我监管"
   - 选择稳健银行

2. **风险调整费率**
   - 高风险银行缴更多
   - 体现风险定价
   - 抑制冒险

3. **监管配合**
   - 银行监管严格
   - 央行最后贷款人"有限支持"
   - "bail-in"机制（自救）

4. **信息披露**
   - 银行风险评级公开
   - 存款人可查询
   - 推动"市场约束"

5. **市场退出**
   - 问题银行及时处置
   - 不轻易"救助"
   - 让市场承担风险
```

### 10.3 道德风险与金融稳定的平衡

```
核心矛盾：

过度保护 → 道德风险 → 银行冒险 → 危机
过度限制 → 道德风险小 → 但挤兑 → 系统性危机

平衡：
  - 限额赔付（覆盖小额）
  - 风险调整费率
  - 监管严格
  - "系统性风险例外"（极端情况）
  - 央行最后贷款人（最后防线）
       ↓
存款保险 = 在道德风险和系统性风险之间寻找平衡
```

## 十一、存款保险的核心命题

### 11.1 五大成就

```
1. **防止了大萧条重演**
   - 1933 后美国银行体系稳定
   - 没有大规模挤兑蔓延
   - 公众信心保持

2. **保护了小额存款人**
   - 99%+ 存款人受到保护
   - 普通家庭无损失
   - 社会稳定

3. **降低了银行融资成本**
   - 存款基础更稳定
   - 利率成本更低
   - 服务实体经济

4. **支持了金融监管**
   - FDIC 既是保险机构也是监管者
   - 早期干预机制
   - 处置机制完善

5. **建立了金融安全网**
   - 央行 + 监管 + 存款保险
   - 三层防护
   - 维护金融稳定
```

### 11.2 五大局限

```
1. **道德风险**
   - 存款人不监督
   - 银行冒险成本降低
   - "太大而不能倒"

2. **保险基金有限**
   - 大银行倒闭远超基金
   - 实际依赖央行最后贷款人
   - "big enough to fail"难设计

3. **跨境存款保护**
   - 跨境合作困难
   - 部分国家例外
   - 灰色地带

4. **"系统性风险例外"的模糊性**
   - 何时动用？
   - 谁决策？
   - 成本谁承担？

5. **道德风险 vs 金融稳定的两难**
   - 严格限制 = 风险低 = 但可能挤兑
   - 灵活保护 = 稳定 = 但道德风险高
```

### 11.3 存款保险的历史定位

```
存款保险是 20 世纪最重要的金融制度创新之一：

起源：
  - 1929-1933 大萧条的教训
  - 1933 美国 FDIC 创立
  - 后续全球效仿

影响：
  - 改变了金融业的运作方式
  - 维护了金融稳定
  - 保护了存款人
  - 但也带来了道德风险

演化：
  - 存款保险 1.0：1933-1980（基本制度）
  - 存款保险 2.0：1980-2008（应对储贷危机）
  - 存款保险 3.0：2008 后（与多德弗兰克、巴塞尔 III 整合）
       ↓
理解存款保险 = 理解现代金融安全网的核心
```
