# 马斯特里赫特条约（Maastricht Treaty / Treaty on European Union）

> 欧元诞生的法律基石——将"经济货币联盟"从梦想变为可操作的法律框架

## 一、核心定义

**《马斯特里赫特条约》**（Maastricht Treaty），全称**《欧洲联盟条约》**（Treaty on European Union，TEU），1992 年 2 月 7 日在荷兰马斯特里赫特签署，**1993 年 11 月 1 日正式生效**。该条约**奠定了欧盟（EU）的政治结构**和**欧元区（Eurozone）的法律基础**，是欧洲一体化进程中的**里程碑式条约**。

**关键事实**：

| 指标 | 数据 |
|------|------|
| 签署日期 | 1992 年 2 月 7 日 |
| 签署地点 | 荷兰马斯特里赫特 |
| 生效日期 | 1993 年 11 月 1 日 |
| 原签署国 | 12 个（欧盟前身欧共体 12 国） |
| 核心内容 | 三大支柱（CEE + CFSP + JHA）+ 经货联盟 |
| 货币联盟阶段 | 三阶段实施（1990-1999） |
| 修订 | 阿姆斯特丹条约（1997）、尼斯条约（2001）、里斯本条约（2009） |

## 二、签约背景：欧洲一体化的"加速时刻"

### 2.1 历史背景

```
1986 年《单一欧洲法案》（SEA）：
  - 1992 年建成单一市场
  - "四大自由"：商品、人员、服务、资本
  - 单一市场需要"货币"配合
       ↓
1989 年柏林墙倒塌：
  - 东西方对峙结束
  - 欧洲一体化加速
  - 德国统一谈判启动
       ↓
1990 年 7 月 ERM 危机：
  - 英国加入 ERM（1990.10）
  - 货币合作深化
       ↓
1989 年 4 月 Delors Report（德洛尔报告）：
  - 欧委会主席 Jacques Delors 主持
  - 12 国央行行长 + 1 欧委会成员组成委员会
  - 提出三阶段实施 EMU 路线图
  - 1989.4 提交报告
  - 1990.7 第一阶段启动（资本完全自由流动）
       ↓
1991.12 马斯特里赫特欧洲理事会：
  - 达成政治协议
  - 1992.2.7 正式签署
```

### 2.2 各方动机

```
德国（Bundesbank + 科尔政府）：
  - 维护低通胀传统
  - 主导"独立央行"原则
  - 担忧"政治化"货币
  - 把货币联盟"绑定"法国

法国（密特朗政府）：
  - 推动欧洲一体化
  - "用货币联盟换德国统一"
  - 担忧德国统一后"德国主导"
  - 接受"独立央行"换政治联盟

英国（梅杰政府）：
  - 反对单一货币
  - 坚持"国家主权"
  - 接受条约但拒绝加入 EMU
  - 1990.10 刚加入 ERM（"重投欧洲"）

意大利、西班牙、葡萄牙、希腊、爱尔兰：
  - 担心"被德国模式锁定"
  - 想要更多财政灵活性
  - 最终妥协
       ↓
荷兰、比利时、卢森堡：
  - 坚定支持一体化
  - 坚定支持单一货币
```

## 三、条约核心结构

### 3.1 三大支柱（Three-Pillar Structure）

```
马约建立的"三大支柱"结构：

第一支柱：欧洲共同体（European Communities）
  - 欧共体三支柱（ECSC、EEC、Euratom）
  - 经济、货币、社会政策
  - 决策：共同体方法（多数决）
       ↓
第二支柱：共同外交与安全政策（CFSP）
  - 外交政策协调
  - 政府间合作（一致决）
  - 最终目标：共同防务
       ↓
第三支柱：司法与内务合作（JHA）
  - 移民、避难、警察、司法
  - 政府间合作
  - 后转为"自由、安全与司法"区域
       ↓
后被里斯本条约（2009）简化：
  - 取消三大支柱
  - 统一为单一"欧盟法人"
  - CFSP 仍保留"政府间"特色
```

### 3.2 货币联盟条款

```
马约关于 EMU 的核心条款：

A. 经济货币联盟（Title II, 第 98-124 条）：
  - 单一货币（欧元）
  - 单一货币政策
  - ECB 与 ESCB
  - 经济政策协调
       ↓
B. 趋同标准（第 109j 条）：
  - 5 项硬指标
  - 强制执行
  - "成员资格"门槛
       ↓
C. 过渡安排（第 123-125 条）：
  - 1990-1993 第一阶段
  - 1994-1998 第二阶段
  - 1999 第三阶段（不可逆锁定）
       ↓
D. "无救助"条款（第 125 条）：
  - 成员国不对他国政府债务承担连带责任
  - 防止"道德风险"
  - 后被 ESM 实质违反
       ↓
E. 超额赤字程序（第 104 条）：
  - 3% 赤字、60% 债务限制
  - 处罚机制
  - 后被 2005、2011、2024 改革
```

### 3.3 三阶段实施路径

```
Delors Plan 三阶段：

第一阶段（1990.7.1 - 1993.12.31）：
  - 资本完全自由流动
  - 货币合作深化
  - 1992 ERM 危机（提前暴露问题）
  - 1992.9 "黑色星期三"
       ↓
第二阶段（1994.1.1 - 1998.12.31）：
  - 成立欧洲货币局（EMI，前身）
  - 收敛经济政策
  - 1995 马德里欧洲理事会命名"欧元"
  - 1996 都柏林欧洲理事会通过《稳定与增长公约》
  - 1998.5 布鲁塞尔欧洲理事会确认 11 个创始国
  - 1998.6.1 ECB 成立
  - 1998.10 "双主席"任命 Duisenberg + Trichet
       ↓
第三阶段（1999.1.1 起，不可逆）：
  - 锁定汇率
  - 单一货币政策
  - ECB 启动
  - 2002.1.1 纸币硬币流通
  - 各国货币退出
```

## 四、趋同标准详解

### 4.1 五大标准

```
《马约》第 109j 条规定的"成员资格"标准：

1. 价格稳定（通胀）
   - 不超过 3 个最低成员国平均通胀率的 1.5%
   - 含义：必须低通胀
   - 内涵：与德国模式一致
       ↓
2. 财政纪律（赤字）
   - 财政赤字 ≤ 3% GDP
   - 例外：严重衰退可豁免
   - 实际执行：多次违反
       ↓
3. 债务可持续
   - 政府债务 ≤ 60% GDP
   - 下降速度令人"满意"
   - 含义：低债务水平
   - 实际执行：意大利、希腊、比利时从未达标
       ↓
4. 汇率稳定
   - 加入 ERM II 至少 2 年
   - 未经历"严重"贬值
   - 内涵：与欧元挂钩的稳定
       ↓
5. 利率水平
   - 长期利率不超过 3 个最低通胀国平均利率 + 2%
   - 含义：金融市场认可
   - 内涵：市场预期通胀低
```

### 4.2 标准的"形式 vs 实质"

```
名义上：
  - 趋同标准 = 硬约束
  - 不达标 = 不能加入
  - 严格数学指标
       ↓
实际上：
  - 政治判断有空间
  - 3% 赤字的"严重衰退"豁免
  - 60% 债务的"令人满意"模糊
  - 实际执行不严
       ↓
典型"违规"：
  - 法国、德国 2003-2004 超过 3%
  - 欧盟理事会拒绝制裁
  - 2005 改革 SGP
  - 意大利、希腊始终超过 60% 债务
  - 仍加入欧元区
       ↓
2024 改革：
  - 转向"国别净支出路径"
  - 5 年调整期
  - 摆脱"机械"3% 规则
  - 但仍是约束
```

### 4.3 标准的"政治经济学"

```
为什么标准"形式严、实质松"？

1. 政治现实
   - 民主国家无法长期紧缩
   - 经济周期影响
   - 政治压力

2. 欧洲政治结构
   - 欧盟理事会 = 政府间
   - 制裁需要一致决
   - 大国可以"自我赦免"

3. 制度不对称
   - 货币联盟先于财政联盟
   - 危机时无强制约束
   - 退出成本大于违约

4. 经济周期
   - 3% 赤字对衰退国家困难
   - 顺周期 vs 逆周期争议
   - 凯恩斯 vs 货币主义之争
```

## 五、批准过程：民主 vs 精英

### 5.1 各国公投与议会

```
1992-1993 马约批准过程：

1992.6.2 丹麦公投：50.7% 否决
  - "例外条款"：丹麦获得 EMU 例外权
  - 1993.5.18 第二次公投通过（附加例外）
       ↓
1992.9.20 法国公投：51.05% 勉强通过
  - 仅领先 100 万票
  - 反映法国社会深度分裂
  - 密特朗政府"国家利益"宣传
       ↓
1992.11.12 德国议会批准
  - 1992.10 联邦宪法法院判决合法
  - 默克尔（当时议员）投票支持
  - 联邦参议院（Bundesrat）批准
       ↓
1993.5 英国议会批准
  - 梅杰政府执政党分裂
  - 部分保守党议员"叛变"
  - 23 名保守党议员反水
  - 政府依赖工党支持才通过
       ↓
其他 9 国：议会批准，无公投
       ↓
1993.11.1 条约生效
```

### 5.2 "民主赤字"问题

```
马约与"民主赤字"批评：

1. 谈判过程
   - 闭门谈判
   - 部长、专家参与
   - 议会"事后"批准
   - 公众讨论有限

2. 复杂文本
   - 300+ 页英文文本
   - 法律专业语言
   - 公众难以理解
   - 媒体简化报道

3. 后果
   - 精英 vs 民众
   - "布鲁塞尔阴谋"论
   - 民族主义反弹
   - 长期影响欧洲政治

4. 民主赤字的解决尝试
   - 1992 丹麦公投否决
   - 2005 法国、荷兰否决欧盟宪法
   - 2016 英国脱欧公投
   - 持续挑战
```

## 六、条约的核心条款

### 6.1 关键条款速查

```
马约核心条款：

第 A 条（共同条款）：
  - 公民身份
  - 自由、安全、司法区域
  - 经济与社会凝聚

第 B-G 条（共同目标）：
  - 经济货币联盟
  - 单一货币
  - 不可逆锁定

第 3 条（欧盟活动）：
  - 单一市场
  - 环境保护
  - 消费者保护
  - 跨境基础设施

第 8-8e 条（公民身份）：
  - 欧盟公民身份
  - 跨国选举权
  - 领事保护

第 98-124 条（经济货币联盟）：
  - 单一货币政策
  - 汇率政策
  - 经济政策协调
  - ECB 与 ESCB

第 105-111 条（ECB）：
  - 央行独立性
  - 价格稳定首要目标
  - 工具独立
  - 财务独立

第 125 条（"无救助"）：
  - 禁止为政府债务融资
  - 禁止承担连带责任
  - 后被 ESM、OMT、NGEU 实质违反

第 126 条（财政纪律）：
  - 3% 赤字、60% 债务
  - 多年度财政规划
  - 超额赤字程序

第 177-181a 条（CFSP）：
  - 共同外交
  - 系统性合作
  - 最终目标：共同防务

第 K 条（JHA）：
  - 移民、避难
  - 警察、司法
  - 民事、刑事合作
```

### 6.2 "无救助"条款的演变

```
第 125 条（无救助）的"违反史"：

1992 原始意图：
  - 防止道德风险
  - 模仿"州政府不能印钞"
  - 强迫财政纪律
       ↓
2010 首次违反：希腊救助
  - 1100 亿欧元
  - "贷款不是救助"
  - 政治上"重新解读"
       ↓
2012 ESM 永久机制：
  - 5000 亿欧元
  - 直接为政府融资
  - "系统性违反"
       ↓
2012-2015 OMT、QE：
  - ECB 购买主权债
  - 直接货币化
  - "事实上违反"
       ↓
2020 NGEU：
  - 8000 亿欧元共同债务
  - 拨款资助南欧
  - "永久违反"
       ↓
2026 现状：
  - 第 125 条仍未"正式"修改
  - 但"实质"被改写
  - "共同债务"是常态
  - "财政联盟"渐进建立
```

## 七、对欧洲一体化的影响

### 7.1 正面影响

```
1. 货币一体化
   - 欧元 1999 诞生
   - 20 国使用
   - 3.5 亿人口
   - 跨境贸易摩擦消失

2. 政治一体化
   - 欧盟公民身份
   - 共同外交
   - 司法合作
   - 移民合作

3. 法律框架
   - 欧盟法律优先
   - 欧洲法院权威
   - 公民权利保护

4. 制度演进
   - 后续条约不断修订
   - 里斯本条约（2009）整合
   - 制度灵活性增强
```

### 7.2 负面影响

```
1. 主权让渡
   - 各国失去货币主权
   - 部分财政主权
   - 部分法律主权
   - 民族反弹

2. 民主赤字
   - 欧盟机构"遥远"
   - 议会权力有限
   - 公众参与不足
   - 疑欧主义兴起

3. 制度复杂
   - 三大支柱
   - 决策程序复杂
   - 改革困难
   - 合法性危机

4. 内部分裂
   - 欧元区 vs 非欧元区
   - 申根区 vs 非申根区
   - 北欧 vs 南欧
   - 东西方差异
```

### 7.3 后续条约修订

```
马约的后续修订：

1997 阿姆斯特丹条约：
  - 强化 JHA 合作
  - 强化就业政策
  - 强化 CFSP
  - 简化程序

2001 尼斯条约：
  - 欧盟东扩准备
  - 加权投票制改革
  - 多数决扩展
  - 合作程序

2007 里斯本条约（原"欧盟宪法条约"）：
  - 整合三大支柱
  - 设立欧洲理事会常任主席
  - 设立 EU 外交高级代表
  - 强化议会权力
  - 2009.12 生效

2012 ESM 条约（欧洲稳定机制）：
  - 财政联盟雏形
  - 永久救助机制

2020 自主决定资源条例：
  - 欧盟"自有资源"
  - 摆脱成员国摊款
```

## 八、条约的"超国家性"与"政府间性"

### 8.1 超国家要素

```
马约的超国家部分：

1. 单一市场
   - 四大自由
   - 共同规则
   - 欧洲法院执行

2. 单一货币
   - ECB 独立
   - 共同货币政策
   - 不可逆锁定

3. 竞争政策
   - 共同规则
   - 欧委会执法
   - 兼并控制

4. 共同贸易政策
   - 欧盟统一对外
   - 共同关税
   - 共同谈判
```

### 8.2 政府间要素

```
马约的政府间部分：

1. 共同外交与安全政策（CFSP）
   - 协调为主
   - 一致决
   - 主权保留

2. 司法与内务合作（JHA）
   - 协调为主
   - 一致决
   - 主权保留

3. 财政政策
   - 协调为主
   - 共同规则
   - 但国家执行

4. 税收政策
   - 一致决
   - 主权保留
   - 几乎不统一
```

### 8.3 "双重性质"的张力

```
马约的双重性质带来的矛盾：

1. 决策机制
   - 共同体方法（多数决）
   - 政府间方法（一致决）
   - 难以统一

2. 合法性来源
   - 成员国授权
   - 欧盟机构独立
   - 双重合法性

3. 危机应对
   - 多数决决策慢
   - 一致决效率低
   - 危机时协调困难

4. 改革困难
   - 条约修订需各国批准
   - 多数决改革可能损害小国
   - 僵化风险
```

## 九、条约的历史意义

### 9.1 在欧洲一体化中的地位

```
马约的历史地位：

1. 货币联盟的法律基础
   - 1999 欧元诞生
   - 20 国使用
   - 不可逆锁定
   - 欧元区"既成事实"

2. 政治联盟的起点
   - 三大支柱
   - 欧盟公民身份
   - 共同外交
   - 政治一体化加速

3. 主权让渡的里程碑
   - 货币主权
   - 部分财政主权
   - 部分法律主权
   - 民族反弹的源头

4. 法律框架的延续
   - 后续条约不断修订
   - 但核心结构稳定
   - "宪法性"基础
```

### 9.2 与其他条约的关系

```
欧盟条约演进：

1951 巴黎条约（ECSC）：
  - 煤钢共同体
  - 6 国
  - 部门一体化
       ↓
1957 罗马条约（EEC、Euratom）：
  - 经济共同体
  - 原子能共同体
  - 6 国
  - 共同市场
       ↓
1986 单一欧洲法案：
  - 1992 单一市场
  - 12 国
```

## 十一、一句话总结

> **马斯特里赫特条约 = 1992.2.7 签署、1993.11.1 生效 = 欧洲一体化的里程碑 = 欧元诞生的法律基础 = 三大支柱结构的开端 = 欧元区"趋同标准"的法律来源**。
>
> 它将"经济货币联盟"从**梦想变为可操作的法律框架**，催生了**欧洲中央银行（ECB）**、**欧元区（Eurozone）**、**单一货币（欧元）**等一系列制度成果。它是 20 世纪**最重要的国际条约之一**——**挑战了"货币 = 主权"的传统信条**，**重塑了欧洲政治版图**，**为后续的里斯本条约、ESM、NGEU 奠定了基础**。
>
> 但马约也是**矛盾的**：
> - **货币联盟先于财政联盟**：导致 2010-2012 欧债危机
> - **无救助条款事实上被违反**：ESM、OMT、NGEU 都是"系统例外"
> - **民主赤字问题**：公投否决、疑欧主义、Brexit 都是后果
> - **三大支柱结构僵化**：里斯本条约才简化
>
> **理解马约 = 理解欧元区 = 理解欧元 = 理解欧洲一体化的成就与局限 = 理解 21 世纪多极货币体系的核心节点**。**未来 10 年，马约的"未完成拼图"——财政联盟、银行业联盟（EDIS）、政治联盟——将持续是欧洲一体化的核心议题。**
