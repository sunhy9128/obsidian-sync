# 量比

> 实时成交量对比指标，A 股短线择时的核心工具之一

**量比**（Volume Ratio, VR）是衡量当前成交量相对于过去一段时间平均水平的比率。公式简单，但实用价值极高——它能在盘中实时揭示哪些股票正在"异动"，是 A 股短线选股和择时的重要参考。

---

## 一、基本公式

```
量比 = 当前成交量 / 过去 N 个交易日的平均成交量
```

**通达信默认设置**：当前成交量 / 前 5 日平均成交量

| 设置 | 含义 |
|------|------|
| 量比 = 1 | 当前成交量 = 5 日均量 |
| 量比 > 1 | 当前成交量高于近期平均 |
| 量比 < 1 | 当前成交量低于近期平均 |

> 量比 > 1 时称为"放量"，量比 < 1 时称为"缩量"。但放量/缩量本身没有方向性——配合价格涨跌才有效。

---

## 二、量比分级与含义

| 量比区间 | 市场含义 | 策略含义 |
|---------|----------|----------|
| **< 0.5** | 极度缩量 | 观望；地量可能见底 |
| **0.5 - 0.8** | 明显缩量 | 无明显信号 |
| **0.8 - 1.2** | 正常水平 | 无异动 |
| **1.2 - 2.0** | 温和放量 | 关注，配合价格判断方向 |
| **2.0 - 5.0** | 明显放量 | 异动信号，结合位置判断 |
| **5.0 - 10.0** | 剧烈放量 | 重要异动，高位=警惕，地位=关注 |
| **> 10.0** | 历史天量级别 | 极度异动，需要立刻结合价格判断 |

## 三、量比的实战用法

### 3.1 开盘 30 分钟量比选股法

这是 A 股短线最经典的使用场景：

```
选股条件：
1. 开盘 30 分钟内量比 > 3
2. 涨幅在 3-8%（排除涨停和无量上涨）
3. 股价在 5 日均线上方
4. 行业板块内有跟风效应
```

**逻辑**：开盘 30 分钟是全天最活跃时段，量比急剧放大往往意味着有资金在当日集中建仓。配合涨幅区间过滤，可以排除无意义的高开后快速回落。

**注意事项**：
- 量比 > 3 的股票数量很多，需要结合板块联动、题材热度二次筛选
- 涨幅过大（>8%）追高风险大
- 排除新股上市首日（换手率天然极高）

### 3.2 盘中即时量比监控

| 场景 | 量比特征 | 含义 |
|------|----------|------|
| **盘中突然放量** | 量比从 1 快速升至 3+ | 有资金主动买入/卖出，关注 |
| **持续放量** | 量比维持 2+ 数小时 | 大资金参与，趋势延续概率高 |
| **脉冲式放量** | 量比瞬间飙升至 10+ 后回落 | 可能有消息刺激，追高需谨慎 |
| **尾盘放量** | 14:30 后量比急剧放大 | 主力异动信号，次日方向重要 |

### 3.3 量比与价格组合判断

量比必须结合价格方向才有意义：

| 量比 | 价格 | 组合含义 |
|------|------|----------|
| > 2 | 涨幅 > 3% | **量价齐升** — 最强信号，主力主动买入 |
| > 2 | 跌幅 > 3% | **放量砸盘** — 恐慌抛售或主力出货 |
| > 2 | 平盘震荡 | **分歧加大** — 方向待定，谨慎 |
| < 0.5 | 涨幅 > 3% | **缩量上涨** — 主力控盘或无人跟涨 |
| < 0.5 | 跌幅 > 3% | **缩量下跌** — 正常调整，抛压不大 |
| > 5 | 涨停 | **强势封板** — 次日溢价预期高 |
| > 5 | 炸板 | **高位分歧** — 极度警惕，次日大概率低开 |

### 3.4 量比在关键位置的用法

| 关键位置 | 量比要求 | 信号强度 |
|----------|----------|----------|
| 突破前高 | 量比 > 2 | ★★★★，有效突破概率大 |
| 突破均线 | 量比 > 1.5 | ★★★ |
| 底部反弹 | 量比 > 2 | ★★★★，有资金进场 |
| 顶部放量 | 量比 > 5 | ★★★★★，警惕主力出货 |
| 跌停打开 | 量比 > 3 | ★★★，可能是投机机会（风险极高） |

## 四、量比的进阶应用

### 4.1 移动量比（动态量比）

标准量比使用固定的前 5 日均量，在趋势性行情中可能失真。进阶用法：

| 方法 | 说明 |
|------|------|
| **相对量比** | 当前成交量 / 过去 20 日均量（过滤趋势性放量） |
| **量比均线** | 将量比做成均线走势，识别量比趋势 |
| **量比缺口** | 量比瞬间跳升至 5+ 且突破关键价位 → 重要信号 |

### 4.2 量比与板块联动

| 场景 | 量比组合 | 含义 |
|------|----------|------|
| 个股量比 > 5 + 板块内多只个股量比 > 3 | **板块联动** — 资金集体行动，可信度高 |
| 个股量比 > 5 + 板块内其他个股量比 < 1 | **个股独立异动** — 消息或主力行为，需进一步验证 |
| 整个板块量比同时 > 3 | **板块级别异动** — 政策/消息驱动概率大 |

### 4.3 量比与时间周期

| 时间段 | 量比参考意义 |
|--------|-------------|
| 开盘 30 分钟 | 量比最活跃，是短线选股黄金时段 |
| 10:00-11:30 | 量比回归正常，关注突破信号 |
| 午盘（13:00-14:00） | 量比意义降低，趋势延续信号 |
| 尾盘 30 分钟 | 量比突然放大 → 主力行为，次日重要 |
| 收盘前 5 分钟 | 量比参考价值最低 |

## 五、量比的局限性与注意事项

### 5.1 核心局限性

| 局限 | 说明 | 应对 |
|------|------|------|
| **无方向性** | 量比只反映量的变化，不反映方向 | 必须结合价格涨跌 |
| **N 日均量固定** | 趋势股 5 日均量随价格上涨而放大，量比被稀释 | 使用移动量比或调整对比周期 |
| **基本面盲区** | 量比放大可能源于消息/公告，而非基本面改善 | 结合个股基本面验证 |
| **量化干扰** | 高频量化交易会产生虚假量比信号 | 配合分时走势综合判断 |
| **小盘股失真** | 小盘股本身流动性低，正常量比波动大 | 提高门槛（量比 > 5 才关注） |

### 5.2 量比的使用误区

| 误区 | 正确理解 |
|------|----------|
| 量比高 = 上涨 | ❌ 量比高 + 放量砸盘 = 危险信号 |
| 量比低 = 下跌 | ❌ 量比低 + 缩量上涨 = 主力控盘 |
| 量比 > 3 就买入 | ❌ 还需要位置、题材、板块联动确认 |
| 量比可以用在所有市场 | ❌ 主要适用于 A 股（散户主导 + T+1 制度） |

## 六、量比与其他指标配合

| 配合指标 | 用法 | 效果 |
|----------|------|------|
| **5 日均线** | 量比放大 + 股价站稳 5 日均线 | 确认短期强势 |
| **成交量** | 量比放大 + 成交额绝对值同步放大 | 资金规模确认 |
| **DDX** | 量比放大 + DDX 飘红 | 量价 + 主力共振，信号最强 |
| **MACD** | 量比放大突破 + MACD 零轴上方 | 中线趋势确认 |
| **涨停股量比** | 涨停打开时的量比 > 3 | 判断是否主力出货 |
