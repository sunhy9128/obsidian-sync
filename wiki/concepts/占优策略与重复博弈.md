# 占优策略与重复博弈

> 当所有参与者都选择不会被对手任何策略压制的"占优策略"时得到的稳定均衡；以及当同一博弈无限重复时，合作如何通过触发策略实现自我维持。

由 AI 系统化整理（2026-06-03），待补充原始来源素材。

占优策略（Dominated Strategy）提供了求解博弈的"硬约束"，而重复博弈（Repeated Game）则给出了囚徒困境等合作问题的标准解——这是从一次性博弈走向长期合作的关键一步。

---

## 一、占优策略

### 1.1 严格占优

策略 $s_i$ **严格占优** $s_i'$，若对对手的任何策略组合 $s_{-i}$：

```
uᵢ(sᵢ, s₋ᵢ) > uᵢ(sᵢ', s₋ᵢ)   ∀ s₋ᵢ
```

含义：不论对手怎么选，$s_i$ 严格优于 $s_i'$——$s_i'$ **永远不会被理性参与者选择**。

### 1.2 弱占优

```
uᵢ(sᵢ, s₋ᵢ) ≥ uᵢ(sᵢ', s₋ᵢ)   ∀ s₋ᵢ
且至少存在某个 s₋ᵢ 使严格不等号成立
```

弱占优允许某些对手策略下收益相等，剔除更微妙——可能剔除掉所有均衡。

### 1.3 严格占优 vs 弱占优

| 维度 | 严格占优 | 弱占优 |
|------|----------|--------|
| 不等号 | > | ≥ |
| 剔除安全性 | 高 | 低（可能误剔除均衡） |
| 现实使用 | 经常 | 谨慎使用 |
| 典型例子 | 囚徒困境中的"坦白" | 占优混合策略中冗余纯策略 |

### 1.4 占优策略均衡

所有参与者都选择严格占优策略的策略组合：

```
例（囚徒困境）：
  A：坦白严格占优抵赖
  B：坦白严格占优抵赖
  → 占优策略均衡 = (坦白, 坦白)
```

## 二、迭代剔除

### 2.1 占优策略的迭代剔除法 (IESDS)

```
步骤 1：找到某参与者的严格占优策略，剔除被占优策略
步骤 2：在缩小的博弈中重复步骤 1
步骤 3：若最终剩下唯一策略组合，即为占优策略均衡
```

### 2.2 案例：性别战（修改版）

```
              女生
              足球    芭蕾
男 足球    (3, 2)   (0, 0)
   芭蕾    (0, 0)   (2, 3)
```

- 没有严格占优策略
- 但存在两个纯策略纳什均衡：(足球,足球)、(芭蕾,芭蕾)
- 占优策略剔除法不直接给出唯一解，需借助额外标准（如帕累托、风险）

### 2.3 案例：选美博弈

```
N 选手机器猜每轮淘汰"距均值最远者"
├─ 所有人选 0：均值 0，最优选 0
├─ 所有人选 0，一人选 0.5：被淘汰
├─ 理性：选 0 是占优策略
└─ 纳什均衡：所有人都选 0
```

Keynes "选美竞赛"是占优策略思维的真实写照——金融市场预期博弈的逻辑基础。

### 2.4 迭代剔除的局限性

| 局限 | 说明 |
|------|------|
| 不一定得到唯一结果 | 占优策略可能不存在 |
| 弱占优剔除可能破坏均衡 | 严格占优更安全 |
| 共同知识假设 | 需要所有参与者理性、知道彼此理性 |
| 多重均衡 | 协调博弈中无解 |

## 三、重复博弈基础

### 3.1 形式化

同一个阶段博弈 $G$ 重复 $T$ 次（$T$ 可能为 $\infty$）：

```
参与者收益（折扣因子 δ ∈ [0, 1)）：

Vᵢ = Σₜ₌₁ᵀ δᵗ⁻¹ · uᵢ(sᵗ)

δ 越接近 1 → 越重视未来 → 合作可能性越大
δ 越接近 0 → 越短视 → 单次博弈逻辑
```

### 3.2 关键概念

| 概念 | 含义 |
|------|------|
| 阶段博弈 | 每一期重复的子博弈 $G$ |
| 历史 $h^t$ | 到 $t$ 期为止所有参与者过去的策略 |
| 策略 $\sigma_i$ | 从历史到行动的映射 $\sigma_i : H \to S_i$ |
| 子博弈完美 | 重复博弈中的均衡必须构成子博弈完美均衡 |
| 触发策略 | 一种"以牙还牙"机制 |

### 3.3 触发策略 (Grim Trigger)

```
合作版本：
├─ 起始：选择合作
├─ 对方合作：继续合作
└─ 对方任何一期背叛：从下一期起永远背叛

→ 只要对方相信你会触发，就不敢偏离合作
```

> 触发策略是无限重复博弈中维持合作的最简单机制——其威胁是"未来永远不再合作"。

## 四、民间定理 (Folk Theorem)

### 4.1 定理内容

**Friedman 民间定理 (1971)**：

在无限重复博弈 $G^\infty$ 中，只要**未来足够重要**（$\delta$ 足够大），任何**可行且个体理性**的收益都可以成为子博弈完美均衡的平均收益。

```
条件：δ > δ*（取决于阶段博弈）
结论：合作收益可作为均衡结果
```

### 4.2 几何直觉

```
平均收益空间：
              帕累托边界
                ╱╲
               ╱  ╲   可行收益集
              ╱ 合作╲
             ╱   区   ╲
            ╱─────────╲
           ╱ 个体理性区 ╲
          ╱   (背背叛点)  ╲
         ╱_________________╲
```

- 阴影部分（合作区）= 个体理性 + 帕累托改进
- 民间定理：$\delta$ 足够大时，**整个合作区**都在均衡集内

### 4.3 经典案例：重复囚徒困境

```
阶段博弈（一次性）：
├─ 占优策略均衡：(背叛, 背叛) → P
├─ 合作解：(合作, 合作) → R > P
└─ 囚徒困境：个体理性背离集体最优

无限重复（δ 大）：
├─ 触发策略可支持合作 R 作为均衡
├─ 条件：δ ≥ (T - R) / (T - P)
└─ 民间定理：合作 R 是子博弈完美均衡结果
```

> 民间定理的意义：**未来足够重要时，合作可以自我执行**——无需第三方强制。

## 五、有限重复 vs 无限重复

### 5.1 关键差异

| 维度 | 有限重复 $T$ | 无限重复 |
|------|---------------|----------|
| 逆向归纳 | 最后一期背叛 → 倒数第二期...→ 全期背叛 | 不可逆推 |
| 合作可能性 | 仅 $T=1$ 时可能（特殊情况） | $\delta$ 大即可 |
| 触发策略 | 无法威胁"永远" | 可信威胁 |
| 实际意义 | 罕见的有限期博弈 | 长期关系模型 |

### 5.2 著名的逆向归纳悖论

```
有限重复囚徒困境（共同知识下）：
├─ T 期：理性参与者会选背叛（无未来）
├─ T-1 期：已知 T 期背叛，本期也背叛
├─ ...逐期逆推
└─ 所有期都选背叛 → 重复无意义
```

> 现实观察：人有"有限寿命"但仍会合作——这与共同知识假设不完全成立有关（行为博弈论的解释）。

## 六、现实应用

### 6.1 商业关系

- 长期供应商关系：触发策略维持稳定供货
- 客户忠诚度：未来收益约束当前行为
- 行业自律：怕被踢出圈子 → 不作弊

### 6.2 国际关系

```
军备控制协议：
├─ 双方遵守（合作）→ 双方节约军费
├─ 一方违约（背叛）→ 短期获益
└─ 无限重复 + 触发策略 → 合作可维持
```

> 核武器控制、贸易协定、气候协议本质都是重复博弈。

### 6.3 金融市场

| 现象 | 重复博弈视角 |
|------|--------------|
| 长期客户关系 | 银行珍惜信誉，不敢过度投机 |
| 行业自律 | 做空机构担心被列入"黑名单" |
| 央行信誉 | 通胀预期管理依赖重复承诺 |
| 基金经理 | 业绩压力下的"反转"行为 |

### 6.4 平台经济

- 平台用户评级：累积声誉 → 影响未来收益
- 司机/外卖员评分：触发策略维持服务质量
- 区块链/智能合约：算法保证的"以牙还牙"

## 七、相关扩展

### 7.1 演化博弈与重复博弈

- Axelrod 锦标赛：以牙还牙 (Tit-for-Tat) 表现最优
- 演化稳定策略 (ESS) 与重复博弈策略的关系
- 有限理性的合作演化

### 7.2 不完全信息重复博弈

- 参与者类型（合作型/机会型）未知
- 信号传递、声誉建立成为关键
- Kreps-Wilson "链式商店" 模型

### 7.3 短视/有限理性

- 实际人不会完美逆推
- 行为博弈：近似的重复合作（损失厌恶、不耐心折扣）
