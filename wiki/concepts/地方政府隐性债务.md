# 地方政府隐性债务

> 游离于预算管理之外、由地方政府融资平台承担、政府负有救助责任的债务 — 中国化债的核心战场

> **🔗 化债核心命题**：参见 [[wiki/concepts/化债核心命题]] — "政府通过化债债的形式将投资者的资金冻结，通过较低的成本为地方债争取时间"。本页面是该命题的**对象维度**：化债要置换的"高息短债"，就是这里定义的隐性债务。

## 一、定义

**地方政府隐性债务**（Hidden Local Government Debt）是指**地方政府在法定政府债务限额之外直接或者承诺以财政资金偿还以及违法提供担保等方式举借的债务**。

核心特征：
- 不纳入预算管理
- 由地方政府融资平台（LGFV/城投）承担
- 地方政府负有救助责任（隐性担保）
- 形成于 2014 年《预算法》修订后地方政府被赋予发债权限之前的"擦边球"模式

---

## 二、历史形成

### 2.1 形成时间轴

```
2008-2010：4万亿刺激计划，融资平台大爆发
2014：《预算法》修订，赋予地方政府发债权限
2014.10：43号文（《关于加强地方政府性债务管理的意见》）首次定义"或有债务"
2015-2018：第一轮置换债（约12万亿）
2018-2022：严控新增、化解存量
2023.7：政治局会议提出"一揽子化债方案"
2023.10：第二轮化债方案（12万亿）
```

### 2.2 形成机制

| 阶段 | 关键事件 | 隐性债务扩张 |
|------|---------|-------------|
| 2008-2010 | 4万亿刺激 | 融资平台从 3800 家增至 10000+ 家 |
| 2010-2013 | 银监会收紧平台贷款 | 转向信托、债券、影子银行 |
| 2014-2017 | 43号文、置换债 | 显性化部分，但新增未止 |
| 2018-2021 | 资管新规 | 通道业务收缩 |
| 2022-至今 | 房地产下行、土地财政坍塌 | 隐性债务风险显性化 |

## 三、规模与结构

### 3.1 规模演变

| 时间 | 规模（万亿元） | 备注 |
|------|--------------|------|
| 2013 | ~10 | 首次摸底 |
| 2018 | ~37 | 峰值 |
| 2023 | ~14.3 | 官方口径（已大幅压降） |
| 2024 | ~10（估计） | 化债方案落地后 |

### 3.2 主要构成

| 构成 | 占比 | 说明 |
|------|------|------|
| **银行贷款** | ~30% | 早期主渠道 |
| **城投债券** | ~30% | 公开发行的债券 |
| **信托贷款** | ~10% | 影子银行渠道 |
| **委托贷款** | ~5% | 银行通道业务 |
| **融资租赁** | ~5% | 类信贷工具 |
| **其他** | ~20% | 资管计划、应付账款等 |

## 四、四大核心风险

### 4.1 利率风险

- 隐性债务平均利率 **6%-8%**
- 政府债券利率约 **2.5%-3%**
- 利率差 3-5 个百分点，每年多支付利息数千亿

### 4.2 期限错配风险

| 维度 | 隐性债务 | 政府债券 |
|------|---------|---------|
| 期限 | 2-3 年 | 10-30 年 |
| 项目周期 | 基建项目 10-30 年 | - |
| 错配 | 严重 | 无 |

### 4.3 项目收益性风险

- 大量基建项目无现金流（公益类）
- 偿债依赖土地出让收入
- 房地产下行 → 偿债来源断裂

### 4.4 流动性风险

- 借新还旧比例高（部分省份 > 80%）
- 一旦再融资渠道收缩，立刻面临违约

## 五、与显性债务的对比

| 维度 | 显性债务 | 隐性债务 |
|------|---------|---------|
| **管理方式** | 纳入预算 | 不纳入预算 |
| **发行主体** | 地方政府 | 城投/国企 |
| **限额管理** | 有 | 无 |
| **信息披露** | 公开 | 不透明 |
| **利率** | 低（2.5%-3%） | 高（6%-8%） |
| **期限** | 长（10-30年） | 短（2-3年） |
| **政府责任** | 直接偿还 | 隐性担保 |

## 六、为什么会有隐性债务？

### 6.1 三大驱动

1. **地方政府的"刚性融资需求"**
   - 基础设施建设（公路、桥梁、保障房）
   - 政绩工程（开发区、新城）
   - 公共服务支出

2. **传统渠道的"三不"约束**
   - 不能：预算法不允许直接举债
   - 不足：政府债券额度有限
   - 不准：行业投向受限

3. **城投平台的"功能性"**
   - 替政府融资
   - 绕过监管限制
   - 承担政策性任务

### 6.2 城投模式的本质

```
地方政府的"手脚"：
- 政府不能直接干的事 → 由城投代劳
- 政府不能直接借的钱 → 由城投代借
- 政府不能直接担的责任 → 隐性由政府担
```

## 七、化债如何化解隐性债务？

### 7.1 核心机制

```
隐性债务（高息、短债、不透明）
        ↓  特殊再融资债券置换
显性债务（低息、长债、规范化）
```

### 7.2 三种主要方式

| 方式 | 操作 | 效果 |
|------|------|------|
| **置换** | 用政府低息债换城投高息债 | 降息拉长期限 |
| **偿还** | 用财政资金直接偿还部分 | 减总量 |
| **整合** | 城投整合、剥离、转型 | 减少主体数量 |

### 7.3 化债的"五不原则"

- 不救助：中央不出钱
- 不兜底：地方负主体责任
- 不新增：严控新增隐性债务
- 不打折：原则上不削债
- 不动摇：守住不发生系统性风险底线

## 八、隐性债务与影子银行

| 维度 | 联系 |
|------|------|
| **本质** | 隐性债务本质上是地方政府主导的"影子信贷" |
| **渠道** | 银行理财 → 信托 → 城投（典型影子银行链条） |
| **监管套利** | 绕过信贷规模、行业投向限制 |
| **风险传导** | 城投非标违约 → 信托 → 银行理财 → 居民 |

> **核心判断**：2023 年化债本质上是**对过去二十年"地方政府主导的影子银行"的总清算**，是 [[wiki/entities/影子银行|影子银行]] 体系的最终结局之一。

---

## 九、化债之后——地方债务的"新常态"

### 9.1 三个根本转变

| 旧模式 | 新模式 |
|--------|--------|
| 土地财政 → 偿债 | 股权财政 → 创收 |
| 城投举债 → 投资 | 政府债券 → 公开透明 |
| 隐性担保 → 兜底 | 市场化融资 → 自负盈亏 |

### 9.2 第三类债务——下一个战场

2024 年起，**城投经营性债务**（理论上政府不担责，但市场认为有隐性担保）成为下一阶段化债重点：
- 规模：可能超过 30 万亿
- 处理方式：转型、剥离、市场化
- 时间表：2026-2028 年
