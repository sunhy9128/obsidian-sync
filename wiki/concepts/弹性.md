> 弹性（Elasticity）衡量一个经济变量对另一个经济变量变化的反应敏感度，是微观经济学中描述"因果响应比例"的核心工具。

---

## 一、弹性的数学定义

### 1.1 一般公式

```
弹性系数 E = (ΔX/X) / (ΔY/Y) = (dX/dY) × (Y/X)

其中：
  X = 因变量（如需求量）
  Y = 自变量（如价格、收入）
  弹性 = X 变动百分比 / Y 变动百分比
```

### 1.2 弹性分类

| 维度 | 类型 | 自变量 |
|------|------|--------|
| 需求侧 | 需求价格弹性 | 商品自身价格 |
| 需求侧 | 需求收入弹性 | 消费者收入 |
| 需求侧 | 需求交叉弹性 | 其他商品价格 |
| 供给侧 | 供给价格弹性 | 商品自身价格 |
| 宏观 | 投资利率弹性 | 利率水平 |

## 二、需求价格弹性（Ed）

### 2.1 公式与判别

```
需求价格弹性 Ed = (%ΔQ) / (%ΔP)
             = (ΔQ/Q) / (ΔP/P)
             = (dQ/dP) × (P/Q)
             < 0（价格与需求量反向变动）
```

| Ed 绝对值 | 名称 | 含义 |
|----------|------|------|
| Ed > 1 | 富于弹性（Elastic） | 价格小幅变动引起需求量大幅变动 |
| Ed = 1 | 单位弹性（Unit Elastic） | 变动比例相等 |
| Ed < 1 | 缺乏弹性（Inelastic） | 价格大幅变动只引起需求量小幅变动 |
| Ed = 0 | 完全无弹性 | 需求量不随价格变动（如胰岛素） |
| Ed = ∞ | 完全弹性 | 价格不变下需求量无限 |

### 2.2 五类典型商品

| 商品类型 | 弹性 | 原因 | 例子 |
|---------|------|------|------|
| 奢侈品 | 高（Ed > 1） | 非必需、可替代多 | 名牌包、海外旅游 |
| 正常品 | 中等 | 有替代但必需 | 服装、家电 |
| 必需品 | 低（Ed < 1） | 无替代、必需 | 大米、食盐、自来水 |
| 急救品 | 极低 | 紧急、无替代 | 急救药、医疗 |
| 嗜好品 | 较高 | 受收入与偏好影响 | 烟、酒、咖啡 |

### 2.3 影响因素

| 因素 | 影响方向 | 解释 |
|------|---------|------|
| 替代品多寡 | 替代品多 → 弹性大 | 容易换买别家 |
| 必需品程度 | 必需 → 弹性小 | 不得不买 |
| 商品定义宽窄 | 定义宽 → 弹性小 | "食物"窄于"牛肉" |
| 时间长短 | 时间久 → 弹性大 | 长期更易调整 |
| 占收入比重 | 占比大 → 弹性大 | 大件商品敏感 |
| 习惯程度 | 强习惯 → 弹性小 | 难以戒断 |

### 2.4 弹性与总收益关系

```
总收益 TR = P × Q

Ed > 1：P ↑ → TR ↓（提价减收）
Ed = 1：TR 不变（单位弹性点收益最大）
Ed < 1：P ↑ → TR ↑（提价增收）
```

## 三、需求收入弹性（Ey）

### 3.1 公式与判别

```
需求收入弹性 Ey = (%ΔQ) / (%ΔY)
              = (dQ/dY) × (Y/Q)
```

| Ey 符号 | 名称 | 例子 |
|---------|------|------|
| Ey > 1 | 奢侈品 | 汽车、珠宝、海外旅游 |
| 0 < Ey < 1 | 必需品 | 食品、服装、日用品 |
| Ey < 0 | 低档品 | 低价方便面、二手商品 |
| Ey ≈ 0 | 收入无弹性 | 食盐、大米 |

### 3.2 应用：恩格尔系数

```
恩格尔系数 = 食品支出 / 家庭总支出

  Ey(食品) 小 → 富国（食品占比 < 30%）
  Ey(食品) 大 → 穷国（食品占比 > 50%）
```

## 四、需求交叉弹性（Exy）

### 4.1 公式

```
Exy = (%ΔQx) / (%ΔPy) = (dQx/dPy) × (Py/Qx)
```

### 4.2 商品关系判别

| Exy 符号 | 关系 | 含义 |
|---------|------|------|
| Exy > 0 | 替代品 | Py 升 → Qx 升（互为替代） |
| Exy < 0 | 互补品 | Py 升 → Qx 降（互相依赖） |
| Exy = 0 | 独立品 | 互不影响 |

### 4.3 应用案例

| 商品对 | 关系 | 弹性 | 商业含义 |
|-------|------|------|---------|
| 咖啡 vs 茶 | 替代 | Exy > 0 | 咖啡涨价→茶销量↑ |
| 汽车 vs 汽油 | 互补 | Exy < 0 | 油价涨→汽车↓ |
| 苹果 vs 梨 | 替代 | 中等 | 竞品定价依据 |
| iPhone vs 配件 | 互补 | 强 | 配件商依赖 iPhone |

## 五、供给价格弹性（Es）

### 5.1 公式

```
供给价格弹性 Es = (%ΔQs) / (%ΔP)
             = (dQs/dP) × (P/Qs)
             > 0（价格与供给量同向变动）
```

### 5.2 影响因素

| 因素 | 弹性大小 | 原因 |
|------|---------|------|
| 生产周期 | 周期长 → 弹性大 | 短期难以调整 |
| 产能利用率 | 余量大 → 弹性大 | 短期可扩产 |
| 储存性 | 易储存 → 弹性大 | 可库存调整 |
| 行业进入壁垒 | 壁垒低 → 弹性大 | 新企业可进入 |
| 资源约束 | 受限 → 弹性小 | 资源稀缺 |

### 5.3 短期 vs 长期

| 时段 | 供给弹性 | 原因 |
|------|---------|------|
| 短期 | 较小 | 厂房、设备无法改变 |
| 长期 | 较大 | 可调整规模、技术 |
| 即时 | 极小 | 库存已定 |

## 六、税收归宿中的弹性应用

### 6.1 税负分担原则

```
税负分担比例 = 供给弹性 / (供给弹性 + 需求弹性)
            取决于买卖双方弹性对比
```

### 6.2 极端情形

| 情形 | 税负归宿 | 实际案例 |
|------|---------|---------|
| 需求完全无弹性（Ed=0） | 100% 归消费者 | 食盐、胰岛素 |
| 需求完全弹性（Ed=∞） | 100% 归生产者 | 高度同质商品 |
| 双方弹性相等 | 各分担 50% | 一般工业品 |

### 6.3 数值示例

```
对商品征税 t 元/件

需求：Ed = 0.5（缺乏弹性）
供给：Es = 2.0（富于弹性）

消费者承担比例 = Es / (Es + Ed) = 2.0 / 2.5 = 80%
生产者承担比例 = Ed / (Es + Ed) = 0.5 / 2.5 = 20%

结论：消费者承担大部分税负
```

## 七、应用与计算示例

### 7.1 点弹性计算

```
需求函数：Q = 100 - 4P，P = 10

dQ/dP = -4
Ed = |dQ/dP × P/Q| = |−4 × 10 / 60| = 0.67

含义：缺乏弹性，提价会增总收益
```

### 7.2 弧弹性计算

```
价格从 10 涨到 12，需求量从 60 降到 50

弧弹性 = [(50-60)/((50+60)/2)] / [(12-10)/((12+10)/2)]
      = [(-10/55)] / [(2/11)]
      = -0.182 / 0.182
      = -1.0
```

### 7.3 现实应用

| 行业 | Ed | 定价策略 |
|------|-----|---------|
| 农产品 | 0.2 - 0.4 | 提价增收入、需政府补贴 |
| 汽油（短期） | 0.2 - 0.3 | 提价不影响销量 |
| 航空（远途） | 1.5 - 2.0 | 旺季敢提价 |
| 奢侈品 | 2.0 - 3.0 | 提价损失销量，需慎用 |
