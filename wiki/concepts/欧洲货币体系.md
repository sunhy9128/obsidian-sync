# 欧洲货币体系（European Monetary System, EMS）

> 欧元的"前身"——欧洲货币合作 20 年（1979-1999）的探索与失败

## 一、核心定义

**欧洲货币体系**（European Monetary System，**EMS**）是 **1979 年 3 月 13 日**启动、**1999 年 1 月 1 日**被**欧洲经济和货币联盟（EMU）**取代的**欧洲货币合作机制**。EMS 的核心是**欧洲汇率机制**（ERM）和**欧洲货币单位**（ECU），是欧元区（Eurozone）的**直接前身**。

**关键事实**：

| 指标 | 数据 |
|------|------|
| 启动日期 | 1979 年 3 月 13 日 |
| 终止日期 | 1999 年 1 月 1 日 |
| 持续时间 | 约 20 年 |
| 创始成员 | 8 国（德国、法国、意大利、荷兰、比利时、卢森堡、丹麦、爱尔兰） |
| 1989 扩容 | 西班牙、英国（1990.10） |
| 1992 危机后 | ERM 浮动区间扩大到 ±15% |
| 核心机制 | ERM（汇率机制） + ECU（货币篮子） |

## 二、历史背景：布雷顿森林崩溃后的"欧洲方案"

### 2.1 1970s 货币动荡

```
布雷顿森林体系崩溃（1971-1973）：

1944 布雷顿森林体系：
  - 美元与黄金挂钩（35 美元/盎司）
  - 其他货币与美元挂钩
  - ±1% 浮动
       ↓
1971.8.15 尼克松冲击：
  - 美元与黄金脱钩
  - 布雷顿森林体系崩溃
       ↓
1973 浮动汇率合法化：
  - 主要货币转向浮动
  - 欧洲货币剧烈波动
  - 美元危机频发
       ↓
欧洲的应对：
  - 1972 "蛇形浮动"（Snake in the Tunnel）
  - 6 国：德国、法国、意大利、荷兰、比利时、卢森堡
  - 共同浮动 ±2.25%
  - 1973 石油危机冲击
  - 法国、意大利先后退出
  - 1976 法国退出
  - 仅剩"小蛇"（德国、荷比卢、北欧）
       ↓
结论：
  - "蛇形浮动"不稳定
  - 法国（密特朗）+ 德国（施密特）推动新机制
  - 1979.3.13 EMS 启动
```

### 2.2 EMS 创立动机

```
EMS 的核心动机：

1. 减少汇率波动
   - 欧洲内部贸易占 60%+
   - 汇率波动损失巨大
   - 稳定汇率 = 促进贸易

2. 抑制通胀
   - 德国马克 = 低通胀"锚"
   - 法国、意大利历史高通胀
   - 通过汇率机制"锁定"低通胀

3. 推动一体化
   - 货币合作 → 政治合作
   - 准备货币联盟
   - "欧洲身份"建构

4. 抗衡美元
   - 美元霸权下欧洲受冲击
   - 集体行动稳定
   - 减少对美元依赖
```

### 2.3 EMS 的"政治交易"

```
EMS 创立 = 法国 + 德国"交易"：

法国（密特朗 + 德斯坦）：
  - 想要"经济货币联盟"
  - 担忧德国主导
  - 接受"锚定德国"换取货币稳定
       ↓
德国（施密特）：
  - 担忧欧洲被"通胀国"拖累
  - 担忧美元危机影响
  - 接受货币合作换取"政治联盟"承诺
       ↓
1978.4 哥本哈根欧洲理事会：
  - 达成 EMS 原则协议
  - 1978.7 不来梅欧洲理事会确认
  - 1979.3.13 正式启动
```

## 三、EMS 核心机制

### 3.1 ERM（欧洲汇率机制）

```
ERM（European Exchange Rate Mechanism）：

定义：
  - 成员国货币相互锁定
  - 共同浮动 ±2.25%
  - 围绕"中心汇率"
  - 央行必须干预维持
       ↓
核心机制：
  - 每个成员国设定"中心汇率"（parity）
  - 双边汇率浮动 ±2.25%
  - 1990.10 后加入国家可 ±6%
  - 1993.8 后扩大至 ±15%
       ↓
干预责任：
  - 汇率接近上限/下限时
  - 强币国央行买弱币
  - 弱币国央行卖强币
  - 双方必须干预
  - 无限制
       ↓
调整机制：
  - 理论上"中心汇率"可调整
  - 实际很少调整（"声誉"机制）
  - 1992-1993 危机后多次重新定值
```

### 3.2 ECU（欧洲货币单位）

```
ECU（European Currency Unit）：

定义：
  - 篮子货币（basket currency）
  - 各成员国货币加权平均
  - 既不是流通货币，也不是记账单位
  - 用于央行结算 + 私人交易
       ↓
权重公式：
  - 各成员国 GDP 占 50%
  - 各国贸易占 50%
  - 5 年调整一次
       ↓
权重（1989）：
  - 德国马克：30.1%
  - 法国法郎：19.0%
  - 英镑：13.0%
  - 意大利里拉：10.2%
  - 荷兰盾：9.4%
  - 比利时法郎：7.6%
  - 西班牙比塞塔：5.3%
  - 丹麦克朗：2.5%
  - 爱尔兰镑：1.1%
  - 希腊德拉克马：0.8%
       ↓
ECU 用途：
  1. 央行间结算（清算机制）
  2. 私人部门贷款/债券
  3. 干预市场的"锚定"
  4. 储备资产
  5. 1999.1.1 1 ECU = 1 欧元
```

### 3.3 信贷机制

```
EMS 信贷机制：

1. 极短期融资便利（VSTF）：
   - 期限：3-3.5 个月
   - 自动 + 无条件
   - 各国央行间无限制

2. 短期货币支持（STMS）：
   - 期限：3 个月（可延长至 9 个月）
   - 限额：与配额挂钩
   - 条件较松

3. 中期金融援助（MFA）：
   - 期限：2-5 年
   - 限额：与配额挂钩
   - 条件较严
   - 类似 IMF 贷款

       ↓
总资源：~300-400 亿欧元
本质：央行间相互授信
```

## 四、EMS 的"硬通货"理论

### 4.1 德国马克的"锚"角色

```
EMS 的"非对称"本质：

事实上：
  - 德国马克 = ERM 的"名义锚"
  - 德国 Bundesbank 主导 ERM 政策
  - 德国利率 = 欧洲利率
  - 德国通胀 = 欧洲通胀基准
       ↓
原因：
  - 德国战后低通胀
  - Bundesbank 独立性最强
  - 德国经济最强
  - 德国马克是"硬通货"
       ↓
后果：
  - 法国、意大利必须"跟随"德国
  - 牺牲了部分货币政策独立性
  - 历史上被称为"德国主导的欧洲"
       ↓
争议：
  - "Mark of the Bundesbank"
  - 法国"附庸化"
  - 民族反弹的根源
```

### 4.2 "信誉"机制

```
ERM 的"信誉"约束：

核心：
  - 央行必须维持汇率
  - 偏离成本极高
  - "声誉"是约束
       ↓
运作机制：
  - 央行公开承诺
  - 央行必须干预
  - 央行不得轻易贬值
       ↓
效果：
  - 法国、意大利"输入"了低通胀
  - 通胀从 1980s 两位数降到 1990s 2-3%
  - "credible"承诺
       ↓
副作用：
  - 1992 危机：固定汇率"不可信"
  - 投机者识别"差异"
  - 危机爆发
```

## 五、EMS 的历史演进

### 5.1 启动与初期（1979-1987）

```
1979.3.13 EMS 启动：

创始 8 国：
  - 德国、法国、意大利
  - 荷兰、比利时、卢森堡
  - 丹麦、爱尔兰
       ↓
初期问题（1979-1981）：
  - 第二次石油危机（1979）
  - 美元强周期
  - 多次重新定值
  - 1981.10 法国法郎贬值 3%
  - 1982.6 法国法郎再贬值 5.75%
  - 1983.6 法国法郎第三次贬值 2.5%
       ↓
转折点（1983）：
  - 法国"转向"严格ERM
  - 密特朗政府"反通胀优先"
  - 法国紧跟德国
  - "法德轴心"形成
       ↓
稳定期（1984-1987）：
  - ERM 相对稳定
  - 各成员通胀收敛
  - 准备"深化"
```

### 5.2 扩展与危机（1988-1993）

```
1989.6 西班牙加入 ERM（±6%）
1989.9 法国 200 周年，庆祝"欧洲货币稳定"
1990.10 英国加入 ERM（±6%）：
  - 梅杰政府"重投欧洲"
  - 1 英镑 = 2.95 马克
  - 引发市场预期
       ↓
1992.6 丹麦公投否决马约
1992.9.16 黑色星期三（详见 [[wiki/concepts/1992欧洲货币危机]]）：
  - 索罗斯等做空英镑
  - 270 亿英镑干预失败
  - 英国退出 ERM
  - 意大利里拉退出
       ↓
1993.1.30 ERM 内部 ±2.25% 浮动
1993.8.2 ERM 浮动扩大到 ±15%：
  - 实际变成"浮动汇率"
  - 信誉崩溃
  - ERM "名存实亡"
       ↓
1990s 后期：
  - ERM II（为东欧准备）
  - 各成员讨论货币联盟
  - 1999.1.1 欧元诞生
  - EMS 终止
```

### 5.3 EMS 的"成功与失败"

```
EMS 的"成功"：

1. 抑制通胀
   - 法国通胀从 1980s 10% 降到 1990s 2%
   - 意大利通胀从 15% 降到 4%
   - 通胀收敛
       ↓
2. 汇率稳定
   - 1984-1992 ERM 内部基本稳定
   - 1983 后几乎无调整
   - 汇率波动大幅下降
       ↓
3. 推动一体化
   - ERM 经验 → 货币联盟
   - 法国"输入"低通胀
   - "欧洲身份"建构
       ↓
4. 准备货币联盟
   - ERM 是"校准"
   - 各成员逐步收敛
   - 为马约创造条件
       ↓

EMS 的"失败"：

1. 1992 危机
   - 索罗斯等投机者击败 ERM
   - 暴露固定汇率脆弱
   - 1993 浮动区间扩大
       ↓
2. 南北分化
   - 德国 vs 南欧工资分化
   - 内部失衡
   - 不可调和矛盾
       ↓
3. 货币联盟准备不足
   - 缺乏财政联盟
   - 缺乏银行联盟
   - 2010-2012 危机再爆发
       ↓
4. 政治成本
   - 法国主权让渡
   - 民族反弹
   - 民主赤字
```

## 六、EMS 的"遗产"：通向欧元的桥梁

### 6.1 制度演进

```
EMS → EMU 的制度演进：

1979 EMS 启动：
  - 汇率合作
  - ECU 篮子
  - 央行信贷机制
       ↓
1986 单一欧洲法案：
  - 单一市场
  - 资本自由流动
  - 推动货币合作
       ↓
1989 德洛尔报告：
  - 三阶段路径
  - 1990 第一阶段
  - 1994 第二阶段
  - 1999 第三阶段
       ↓
1992 马约签署：
  - EMU 法律框架
  - 趋同标准
  - ECB + ESCB
       ↓
1994 EMI（欧洲货币局）：
  - 准备 ECB
  - 监督 ERM
  - 协调货币政策
       ↓
1998.6.1 ECB 成立：
  - EMI 升级
  - 央行权力集中
  - 准备欧元
       ↓
1999.1.1 EMU 启动：
  - 11 国加入
  - 欧元电子化
  - ERM 转换为 ERM II
       ↓
2002.1.1 欧元纸币流通：
  - EMS 时代结束
  - 欧元时代开始
```

### 6.2 关键人物

```
EMS 时代的关键人物：

Valéry Giscard d'Estaing（法国总统，1974-1981）：
  - EMS 概念的"始作俑者"
  - 1978 与施密特共同推动

Helmut Schmidt（德国总理，1974-1982）：
  - EMS 共同设计者

François Mitterrand（法国总统，1981-1995）：
  - 1983 转向"跟随德国"
  - 推动马约
  - 推动单一货币

Helmut Kohl（德国总理，1982-1998）：
  - 完成德国统一
  - 推动马约
  - 推动欧元

Jacques Delors（欧委会主席，1985-1995）：
  - 1989 德洛尔报告
  - 单一市场 + 货币联盟
  - "欧元之父"

Pierre Werner（卢森堡首相）：
  - 1970 Werner Plan
  - 货币联盟先驱
  - EMS 灵感来源
```

## 七、EMS 的金融学意义

### 7.1 货币合作的"中间形式"

```
货币合作的层次：

1. 浮动汇率（baseline）
   - 各国自主
   - 无合作
   - 易波动

2. 联合浮动（如 ERM）
   - 共同稳定
   - 内部合作
   - 维持国际浮动

3. 货币联盟（如欧元区）
   - 单一货币
   - 共同央行
   - 完全一体化
       ↓
EMS 是"中间形式"：
  - 介于完全浮动和货币联盟之间
  - 试验场
  - 过渡阶段
  - 经验积累
```

### 7.2 固定汇率的"承诺理论"

```
EMS 对固定汇率理论的贡献：

1. 信誉约束
   - 央行"承诺"维持汇率
   - 通货膨胀收敛
   - 政策约束效果
       ↓
2. 三角悖论验证
   - 固定汇率 + 资本开放
   - 必须放弃独立货币政策
   - 三选二的实践
       ↓
3. 投机攻击理论
   - 1992 危机验证 Krugman 模型
   - 投机者识别"不可持续"
   - 触发机制
       ↓
4. 政策可信度
   - "credible commitment"
   - 央行独立性重要性
   - 制度设计的关键
```

### 7.3 国际金融史定位

```
EMS 的历史地位：

1. 货币合作的"试验场"
   - 为欧元区积累经验
   - 暴露问题
   - 推动改革

2. 国际货币体系的多极化
   - 美元 - 马克 - 日元
   - ECU 的"实验"
   - 储备货币多元化

3. 政治经济学的案例
   - 主权让渡的成本
   - 民主赤字问题
   - 民族反弹的根源

4. 金融危机史的"前奏"
   - 1992 ERM 危机
   - 1997 亚洲危机
   - 2010-2012 欧债危机
   - 共同主题：固定汇率的脆弱
```

## 八、EMS 时代的危机与经验

### 8.1 1981-1983 法郎危机

```
背景：
  - 第二次石油危机（1979）
  - 美元强周期
  - 法国通胀压力
       ↓
危机（1981-1983）：
  - 1981.10 法郎贬值 3%
  - 1982.6 法郎再贬值 5.75%
  - 1983.3 法郎第三次贬值 2.5%
  - 密特朗政府"经济政策"困难
       ↓
转折（1983.6）：
  - 密特朗"转向"
  - "反通胀优先"
  - 严格紧缩
  - 跟随德国
       ↓
后果：
  - 法国失业率上升
  - 1986-1988 紧缩持续
  - 通胀从 1981 13% 降到 1988 2.7%
  - "法德轴心"形成
```

### 8.2 1992-1993 ERM 危机

```
背景（1990-1992）：
  - 1990 德国统一
  - 1990.10 英国加入 ERM
  - 1991 德国通胀压力上升
  - 1991-1992 德国央行加息
       ↓
危机（1992.9-1993.8）：
  - 1992.6 丹麦公投否决马约
  - 1992.9.16 黑色星期三（英镑、里拉退出）
  - 1992.11 瑞典克朗浮动
  - 1992.11 西班牙比塞塔贬值 6%
  - 1993.1 葡萄牙埃斯库多贬值
  - 1993.8.2 浮动区间扩大到 ±15%
       ↓
详见 [[wiki/concepts/1992欧洲货币危机]]
```

### 8.3 关键教训

```
EMS 的核心教训：

1. 固定汇率要求"经济基础一致"
   - 德国 vs 英国/意大利
   - 工资、通胀、生产率差异
   - "不可调和"矛盾
       ↓
2. 央行干预能力有限
   - 270 亿英镑/日 干预失败
   - 投机规模更大
   - 干预需要"政治决心"
       ↓
3. 货币联盟需要财政联盟
   - EMS 没有财政合作
   - 边缘国没有"最后贷款人"
   - 危机时无法协调
       ↓
4. 政治意愿至关重要
   - 1983 法郎危机：法国"转向"成功
   - 1992 ERM 危机：英国/意大利"放弃"
   - 1999 欧元成功：政治意愿
       ↓
5. "信誉"是双刃剑
   - 信誉可以稳定汇率
   - 信誉崩溃时"加倍"损失
   - 不可预测临界点
```

## 九、EMS 与其他货币体系的比较

### 9.1 与布雷顿森林体系比较

| 维度 | 布雷顿森林（1944-1971） | EMS（1979-1999） |
|------|----------------------|-----------------|
| **范围** | 全球 | 区域（欧洲） |
| **锚定** | 黄金/美元 | 德国马克/欧洲货币篮子 |
| **调整** | ±1% | ±2.25%（1990 后 ±6%，1993 后 ±15%） |
| **范围** | 政府间 | 政府间 + 超国家 |
| **主要工具** | IMF | ECU + 央行信贷 |
| **持续时间** | ~27 年 | ~20 年 |
| **终止原因** | 美元危机 + 美国放弃 | 货币联盟（被 EMU 替代） |

## 十、EMS 时代的精神遗产

### 10.1 对欧元的影响

```
EMS 的"软件"遗产：

1. 货币合作经验
   - 央行合作机制
   - 干预经验
   - 危机处理经验
       ↓
2. 通货膨胀收敛
   - 各成员通胀水平接近
   - "credible commitment"文化
   - 央行独立性共识
       ↓
3. 货币联盟的"试验场"
   - 暴露问题
   - 推动改革
   - 累积经验
       ↓
4. 政治意愿基础
   - "欧罗巴"理想
   - 法国-德国轴心
   - 货币主权的"集体让渡"
```

### 10.2 对全球金融治理的启示

```
EMS 对全球金融治理的启示：

1. 区域货币合作可行
   - 区域内汇率稳定
   - 区域内贸易促进
   - 共同货币的可能
       ↓
2. 固定汇率的"必要条件"
   - 经济基础一致
   - 政策协调
   - 危机处理机制
       ↓
3. 货币联盟的"前置条件"
   - 财政联盟
   - 银行联盟
   - 政治意愿
       ↓
4. 国际合作的双面性
   - 收益：稳定、效率
   - 成本：主权让渡、不平等
   - 需要"共同利益"基础
```

### 10.3 反思与争议

```
EMS 的历史争议：

1. "德国主导"批评
   - 法国"附庸化"
   - 南欧"听命"
   - "不平等合作"
       ↓
2. "民主赤字"批评
   - 央行独立"反民主"
   - 部长会议"远程"
   - 公众参与不足
       ↓
3. "过度一体化"批评
   - 民族特性丧失
   - 文化多样性牺牲
   - "欧洲合众国"警惕
       ↓
4. "未完成一体化"批评
   - 没有财政联盟
   - 没有政治联盟
   - 货币联盟"半成品"
```

## 十一、一句话总结

> **欧洲货币体系（EMS）= 1979-1999 欧洲货币合作机制 = ERM + ECU = 欧元的"前身"和"试验场"**。
>
> 它是欧洲一体化进程中的**关键中间阶段**——**从浮动汇率到单一货币的桥梁**。通过 **ERM（汇率机制）** 和 **ECU（货币篮子）**，EMS 在 20 年间成功**降低了欧洲内部的汇率波动**、**收敛了各成员国的通胀**、**培养了货币合作的"文化"**，为 **1999 年欧元的诞生**奠定了基础。
>
> 但 EMS 也是**矛盾的**：
> - **德国马克的"名义锚"角色**导致"非对称"的货币合作
> - **1992 年 ERM 危机**暴露了固定汇率的脆弱性
> - **缺乏财政合作**导致 2010-2012 欧债危机重演
> - **"民主赤字"**和**"德国主导"**批评始终存在
>
> **理解 EMS = 理解欧元的前世 = 理解货币合作的"中间形式" = 理解为什么欧元最终诞生 = 理解 2010-2012 欧债危机的根源**。**EMS 时代的精神遗产是"货币合作可以深化，但必须建立在经济基础一致、政治意愿充足、危机处理机制完善的基础上"**。**未来欧元区一体化的发展，仍在重复 EMS 时代提出的核心命题**。
