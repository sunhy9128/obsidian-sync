# 基差风险（Basis Risk）

> 完美对冲在现实中不存在 —— 基差是"对冲的剩余风险"

## 一、定义

**基差**（Basis）是指**现货价格**与**期货价格**的差额：

```
基差 = 现货价格 - 期货价格

例：
  沪深 300 指数现货 = 4000 点
  沪深 300 期货 = 3990 点
  基差 = 4000 - 3990 = +10 点
```

**基差风险**（Basis Risk）是指基差**变动**给对冲者带来的**剩余风险**。它是**完美对冲不存在**的根本原因。

> **核心洞察**：理论上，如果基差永远不变，对冲可以完美消除价格风险。但实际中基差会变动，因此对冲只能"近似完美"。

## 二、基差的形成机制

### 2.1 期货价格 ≠ 现货价格的根本原因

```
期货价格 ≠ 现货价格的原因：

1. 持有成本（Cost of Carry）
   - 持有现货需要资金（机会成本）
   - 持有现货有仓储成本（商品）
   - 持有现货可能有收益（股息、利息）

2. 持有收益（Carry Return）
   - 股票分红
   - 债券利息
   - 商品便利收益

3. 预期因素
   - 预期未来现货价格
   - 预期未来供求关系

4. 流动性溢价
   - 期货市场流动性溢价/折价
   - 不同到期月的价差
```

### 2.2 持有成本模型

```
期货理论价格 = 现货价格 × (1 + r × t) - 持有收益

其中：
  r = 无风险利率
  t = 时间（年化）
  持有收益 = 分红、利息等

例（沪深 300）：
  现货 = 4000 点
  无风险利率 = 3%
  3 个月到期
  分红率 = 2% / 年
  
  期货理论价格 = 4000 × (1 + 3% × 0.25) - 4000 × 2% × 0.25
              = 4000 × 1.0075 - 4000 × 0.005
              = 4030 - 20
              = 4010 点
  
  实际期货价格可能 = 3990 点（贴水）或 4030 点（升水）
  基差 = 4000 - 3990 = +10（期货贴水）
  或
  基差 = 4000 - 4030 = -30（期货升水）
```

## 三、基差的三种状态

### 3.1 升水（Contango）

```
升水 = 期货价格 > 现货价格
基差 = 现货 - 期货 < 0（负基差）

含义：期货价格高于现货
原因：
  - 持有成本 > 持有收益
  - 市场预期未来价格上涨
  - 期货市场流动性溢价

例：
  现货 100 元，期货 105 元
  基差 = 100 - 105 = -5（升水 5 元）
```

### 3.2 贴水（Backwardation）

```
贴水 = 期货价格 < 现货价格
基差 = 现货 - 期货 > 0（正基差）

含义：期货价格低于现货
原因：
  - 持有成本 < 持有收益
  - 市场预期未来价格下跌
  - 期货市场流动性折价
  - 现货市场需求紧张

例：
  现货 100 元，期货 95 元
  基差 = 100 - 95 = +5（贴水 5 元）
```

### 3.3 平价（Flat）

```
平价 = 期货价格 = 现货价格
基差 = 0

含义：期货价格等于现货
原因：
  - 到期日临近
  - 市场无明显预期
  - 持有成本与持有收益相等
```

## 四、基差变动对冲的"剩余风险"

### 4.1 完美对冲的假设

```
完美对冲要求：
  1. 基差 = 0
  2. 基差不变
  3. 期货价格变动 = 现货价格变动

如果满足：
  现货损失 = 期货盈利
  → 净损益 = 0（完美对冲）
```

### 4.2 基差变动的情况

```
情景 1：基差不变
  现货：100 → 90（跌 10 元）
  期货：99 → 89（跌 10 元）
  现货损失：10 元
  期货盈利：10 元
  净损益：0 ✅

情景 2：基差扩大（升水变深）
  现货：100 → 90（跌 10 元）
  期货：99 → 85（跌 14 元）
  现货损失：10 元
  期货盈利：14 元
  净盈利：+4 元（基差变动带来）

情景 3：基差缩小（升水变浅或转贴水）
  现货：100 → 90（跌 10 元）
  期货：99 → 95（跌 4 元）
  现货损失：10 元
  期货盈利：4 元
  净亏损：-6 元（基差变动带来）❌
```

### 4.3 基差变动的"剩余风险"

```
基差变动 = 对冲的"剩余风险"

可能的结果：
  ① 基差扩大 → 对冲者"超额盈利"
  ② 基差不变 → 完美对冲
  ③ 基差缩小 → 对冲者"亏损"

结论：
  - 完美对冲在现实中不存在
  - 对冲只能"近似完美"
  - 对冲的"剩余风险" = 基差变动风险
```

## 五、基差风险的具体类型

### 5.1 类型一：基差方向变动

```
基差方向变动：
  升水 → 贴水（或反向）
  
对冲影响：
  - 升水变深 → 期货跌得更多 → 期货盈利 > 现货损失
  - 贴水变深 → 期货涨得更多 → 期货亏损 > 现货损失
```

### 5.2 类型二：基差水平变动

```
基差水平变动：
  基差从 +5 变为 +10（贴水变深）
  
对冲影响：
  - 期货市场相对现货变得"更便宜"
  - 期货盈利 > 现货损失（如果做空期货）
  - 期货亏损 < 现货损失（如果做多期货）
```

### 5.3 类型三：基差波动率

```
基差波动率：
  基差在 ±5 之间频繁波动
  
对冲影响：
  - 频繁调整对冲头寸
  - 增加交易成本
  - 增加基差风险
```

## 六、基差风险的管理

### 6.1 减少基差风险的方法

#### 方法一：选择相关性高的合约

```
相关性：
  现货 vs 期货价格变动的相关系数
  
  完美对冲：相关系数 = 1
  实际对冲：相关系数 < 1
  
  措施：
  - 选择与现货最相关的期货合约
  - 同一品种的期货优先
  - 避免跨品种对冲（除非不得已）
```

#### 方法二：缩短对冲期限

```
基差风险随时间增加：
  - 1 个月对冲：基差风险小
  - 3 个月对冲：基差风险中
  - 6 个月对冲：基差风险大
  
措施：
  - 频繁展期（roll over）
  - 缩短对冲期
  - 增加调整频率
```

#### 方法三：动态调整对冲比率

```
对冲比率 = β 系数
  - 静态对冲：β 固定
  - 动态对冲：β 动态调整

动态对冲：
  - 每次市场变动都重新计算 β
  - 调整对冲头寸
  - 减少基差风险
```

#### 方法四：使用基差互换（Basis Swap）

```
基差互换：
  - 用一个基差（基差 A）交换另一个基差（基差 B）
  - 锁定基差风险
  
例：
  投资者有现货头寸，对冲时使用本地期货
  通过基差互换，将本地基差锁定为海外基差
  → 消除本地基差风险
```

### 6.2 接受基差风险

```
基差风险 = 对冲的"必要成本"
  - 完全消除基差风险 = 失去对冲的灵活性
  - 接受"合理范围"的基差风险 = 现实选择
  
合理范围：
  - 5% 内的基差变动可接受
  - 超过 10% 的基差变动需要重新评估
```

## 七、基差风险在 1998 香港案例中的作用

### 7.1 金管局的"基差选择"

```
1998 年 8 月：
  金管局同时操作：
    - 现货：买入 30 只恒指成分股
    - 期货：在 8 月期指做多
  
  对冲的"基差" = 恒指现货 - 恒指期货
       ↓
  8 月 14 日：
    恒指现货 = 6600 点
    8 月期指 = 6800 点（升水 200 点）
    基差 = 6600 - 6800 = -200（升水）
       ↓
  8 月 28 日：
    恒指现货 = 7820 点
    8 月期指 = 7820 点（即将到期，基差归零）
    基差 = 0
       ↓
  关键洞察：
    - 期货到期时基差必然归零
    - 期货价格上涨带来盈利
    - 现货价格上涨带来账面浮盈
    - 金管局的"对冲"实际上"双向盈利"
```

### 7.2 金管局对冲的"特殊性"

```
金管局的"对冲"不是典型对冲：

典型对冲：
  现货多头 + 期货空头 = 中性
  → 净损益 ≈ 0

金管局的"对冲"：
  现货多头 + 期货多头 = 双向多头
  → 不是"对冲"，是"攻击"
  → 净损益取决于市场走向
  → 但通过时机选择（8 月 28 日决战），最大化收益
```

> **金管局的"对冲"实际上是"立体攻击"**——它利用了期货市场**多头**的特性（做多恒指期货），而不是用空头对冲。这意味着金管局的"对冲"**承担了市场风险**，但通过**时机选择**和**市场判断**，**主动追求收益**。

## 八、基差与"市场情绪"

### 8.1 基差反映市场预期

```
基差 = 期货 - 现货 = 市场对未来的预期

升水（Contango）：
  - 期货 > 现货
  - 市场预期未来价格上涨
  - 持有成本 > 持有收益
  
贴水（Backwardation）：
  - 期货 < 现货
  - 市场预期未来价格下跌
  - 持有成本 < 持有收益
  - 现货市场紧张
```

### 8.2 基差的"恐慌指标"

```
基差急剧扩大：
  - 市场流动性紧张
  - 投资者大量购买期货避险
  - 期货价格飙升
  - 现货市场可能即将出问题
  
基差急剧缩小或反转：
  - 期货市场出现抛售
  - 现货市场可能即将反弹
  - 或者市场情绪转为乐观
```

## 九、基差与"做市商策略"

### 9.1 做市商的"基差套利"

```
做市商在期货市场提供流动性：
  - 买入报价 + 卖出报价 = 价差
  - 通过频繁交易获利
  - 承担基差风险
  
  关键：
    做市商通过"对冲"消除方向性风险
    保留"基差风险"作为利润来源
```

### 9.2 跨期基差套利

```
跨期套利（Calendar Spread）：
  买入近月期货 + 卖出远月期货
  价差 = 远月 - 近月
  
  如果价差扩大 → 盈利
  如果价差缩小 → 亏损
  
  这是基差套利的常见形式
```

## 十、基差风险在投资组合中的应用

### 10.1 投资组合对冲

```
投资组合对冲（Portfolio Hedging）：
  - 持有多个股票
  - 用股指期货对冲
  
  对冲比率：
    N_hedge = (β_portfolio × V_portfolio) / (P_future × Multiplier)
    
  其中：
    β_portfolio = 投资组合的 β 系数
    V_portfolio = 投资组合市值
    P_future = 期货价格
    Multiplier = 合约乘数
```

### 10.2 动态 β 调整

```
投资组合的 β 系数会变化：
  - 股票涨跌幅度不同
  - β 高的股票涨得快、跌得也快
  - 投资组合的 β 在持续变化
  
  动态调整：
    - 每天重新计算 β
    - 调整对冲头寸
    - 维持目标 β
```

## 十一、基差风险在新兴市场的特殊性

### 11.1 中国 A 股市场的基差

```
A 股市场特殊性：
  - 长期"股指期货贴水"（基差为正）
  - 沪深 300 期货长期贴水 10-30%
  - 2015 年贴水甚至达到 50%+
  
原因：
  - 投资者结构（散户多）
  - 套利机制不完善
  - 现货市场紧张
  - 市场情绪悲观
       ↓
对冲者的困境：
  - 用 A 股期货对冲 = 长期"贴水损耗"
  - 套利空间有限
  - 长期对冲成本高
```

### 11.2 1998 香港的基差

```
1998 年 8 月：
  - 8 月期指（即将到期）基差 = 0
  - 9 月期指（远月）基差较大
  - 11 月期指（更远月）基差最大
  
金管局选择 8 月期指的原因：
  - 基差小（即将归零）
  - 临近期货结算日
  - 国际空头也必须在 8 月期指平仓
  → 选择"基差最小"的合约 = 最优对冲
```

## 十二、基差与"风险溢价"

### 12.1 风险溢价的体现

```
基差中包含"风险溢价"：
  - 期货市场参与者要求的风险补偿
  - 流动性溢价的体现
  - 持有现货的"便利收益"
  
  升水时：风险溢价为正
  贴水时：风险溢价为负
```

### 12.2 基差与"市场恐慌"

```
市场恐慌时：
  - 投资者涌向期货市场避险
  - 期货价格上涨
  - 现货市场流动性枯竭
  - 基差急剧扩大（升水加深）
  
  例：2008 年金融危机
  - 股指期货一度升水 30%+
  - 反映极端市场恐慌
```

## 十三、基差风险 vs 其他风险

### 13.1 三类风险对比

| 风险类型 | 描述 | 是否可对冲 | 残留风险 |
|---------|------|----------|---------|
| **价格风险** | 现货/期货价格变动 | ✅ 可对冲 | 基差风险 |
| **基差风险** | 现货-期货价差变动 | ⚠️ 部分对冲 | 自身 |
| **流动性风险** | 无法及时平仓 | ⚠️ 难对冲 | 自身 |

### 13.2 对冲的"风险转换"

```
对冲的实质：
  - 把"价格风险"转换为"基差风险"
  - 价格风险：大、不可控
  - 基差风险：小、相对可控
  
  对冲的"价值"：
    把"未知的大风险"转换为"已知的小风险"
```

## 十四、对冲者的"基差管理"

### 14.1 基差管理的"五个原则"

```
1. 了解基差
   - 知道当前基差水平
   - 知道历史基差范围
   - 知道基差的季节性模式

2. 监控基差
   - 每天跟踪基差变动
   - 设置基差变动的预警线
   - 必要时调整对冲头寸

3. 接受基差风险
   - 完美对冲不存在
   - 基差风险是"对冲的代价"
   - 不要追求"零基差风险"

4. 灵活选择合约
   - 同一品种的不同到期月
   - 跨品种的相关性
   - 跨市场的联动性

5. 长期视角
   - 短期基差波动是"噪音"
   - 长期基差水平是"信号"
   - 不要被短期波动干扰
```

### 14.2 基差"信号"vs"噪音"

```
基差信号（值得关注）：
  - 基差持续扩大/缩小
  - 基差突破历史区间
  - 基差与基本面背离

基差噪音（短期波动）：
  - 每日的小幅波动
  - 季节性模式
  - 套利交易导致的波动
```

## 十五、一句话总结

> **基差风险 = 完美对冲不存在的根本原因**。
>
> 它来自**现货价格 ≠ 期货价格**的事实，让对冲只能"**近似完美**"。对冲的"价值"是把"**未知的大风险**"（价格风险）转换为"**已知的小风险**"（基差风险）。
>
> **接受基差风险 = 接受现实的不完美**。好的对冲者**不追求零基差风险**，而是**在合理范围内管理基差**。**1998 香港金管局**通过选择 8 月期指（基差最小）**最大化对冲效率**——这是对冲的"细节决定成败"。
