# DDX/DDY/DDZ 指标

## 定义

DDX、DDY、DDZ 指标是中国 A 股市场常用的短线技术分析指标组合，主要用于衡量大单（特大单、大单）的资金流向、涨跌贡献和买卖差分情况。DDX、DDY、DDZ 是通达信（Tongdaxin）软件的特色指标，追踪"大单"与"小单"的买卖差异，以此判断机构（主力）资金的进出方向。属于**资金流指标**这一大类，在 A 股的短线交易圈中广泛使用。

---

## 一、大单与小单的划分

DDX 系列的核心理念：A股的涨跌主要由"主力"（机构/大资金）驱动，散户（小单）只能跟随。大单买入 = 主力吸筹 = 未来上涨概率提高。

**通达信默认的单笔划分标准**：

| 类型 | 单笔成交股数 | 单笔成交金额（估算） |
|------|-------------|---------------------|
| **超级大单** | >100 万股 或 >500 万元 | 超大户/机构行为 |
| **大单** | 4-100 万股 或 20-500 万元 | 主力/中户 |
| **小单** | <4 万股 | 散户 |

> 注：通达信的大单标准可自行设置，4 万股/20 万元为默认阈值。不同券商软件设置可能略有差异。

## 关键特征

### DDX（大单动向）

```
DDX = 大单净买入量 / 成交量 × 100（%）
```

| DDX 值 | 含义 |
|--------|------|
| DDX > 0（红色） | 大单买入量 > 卖出量，主力资金净流入 |
| DDX < 0（绿色） | 大单卖出量 > 买入量，主力资金净流出 |
| DDX 持续为正 | 主力持续买入，强势信号 |
| DDX 由负转正 | 主力资金转向买入，关注 |

**多日平滑**：
- DDX(3) = 3 日累计 DDX
- DDX(5) = 5 日累计 DDX
- 短周期 DDX 波动大、噪音多；多日 DDX 更稳定

### DDY（大单动向/涨跌动因）

```
DDY = (大单卖出笔数 - 大单买入笔数) / 总成交笔数 × 100（%）
```

| DDY 值 | 含义 |
|--------|------|
| DDY > 0（红色） | 卖出的大单笔数多于买入的大单笔数（从笔数维度看空） |
| DDY < 0（绿色） | 买入的大单笔数多于卖出的大单笔数（从笔数维度看多） |

> DDX 衡量**金额维度**，DDY 衡量**笔数维度**。两者结合可以判断：是大金额集中买入（DDX 正，DDY 负）——主力少量大笔扫货？还是多笔大单分散出货（DDX 负，DDY 正）——主力分散出逃？

### DDZ（大单差分/大单质量）

```
DDZ = DDX × DDY 的综合评分
```

| DDZ 值 | 含义 |
|--------|------|
| DDZ 极高 | DDX 正且 DDY 负，大单高效率买入 = 最强信号 |
| DDZ 极低 | DDX 负且 DDY 正，大单高效率卖出 = 最弱信号 |
| DDZ 中性 | DDX/DDY 方向不一致，需谨慎 |

### 综合特征

- 三个指标均为衍生型成交量类指标，区别于传统均线、MACD 等趋势型指标，更侧重资金行为分析。
- 在通达信、同花顺、大智慧等国内主流行情软件中广泛内置。

## 应用

### 基础信号识别

| 信号 | DDX | DDY | DDZ | 含义 |
|------|-----|-----|-----|------|
| **主力吸筹** | 正 | 负 | 极高 | 大单集中买入，最强买入信号 |
| **主力派发** | 负 | 正 | 极低 | 大单集中卖出，最强卖出信号 |
| **主力对敲** | 正 | 正 | 中性 | 大单买卖均活跃，方向不明 |
| **散户拉升** | 负 | 负 | 中性 | 小单推涨，主力未参与（警惕） |
| **主力洗盘** | 负转正 | 正转负 | 逐步上升 | 快速下跌后主力重新吸筹 |

### 多周期共振

| 方法 | 说明 |
|------|------|
| **日线 DDX(5) + 分时 DDX** | 日线看方向，分时看具体时机 |
| **周线 DDX(3) + 日线 DDX(5)** | 大周期 DDX 向上 + 小周期 DDX 金叉 = 强信号 |
| **连续飘红** | DDX 连续 3-5 天飘红 > 主力持续买入，持股信号 |

### 配合 K 线形态

| K 线形态 | DDX 状态 | 信号意义 |
|----------|----------|----------|
| **涨停 + DDX 飘红** | 大单涨停买入 | 主力行为确认，强势延续概率高 |
| **涨停 + DDX 翻绿** | 大单在涨停板卖出 | "主力出货"型涨停，次日大概率低开 |
| **阳线 + DDX 持续为正** | 稳步上涨 + 主力持续买入 | 健康上涨，可持有 |
| **阴线 + DDX 持续为正** | 下跌中主力逆势吸筹 | 值得关注，可能是洗盘 |
| **大阳线 + DDX 飘绿** | 缩量上涨，主力出货 | 主力没参与，小心 |

### 短线择时与跟庄分析

- **短线择时**：通过 DDX 红柱持续放大捕捉主力建仓信号。
- **跟庄分析**：结合 DDY 排序功能筛选当日资金净流入排名靠前的个股。
- **拐点预警**：利用 DDZ 的急速跳变识别盘中大单异动，常用于打板或短线强势股筛选。
- **筛选强势股**（通达信常用思路）：

```
DDX(3) > 0        // 3日大单净流入
DDX(10) > DDX(3)  // 10日比3日更强 = 持续买入
DDZ > 80          // 大单质量指标极强
换手率 > 5%       // 流动性确认
```

- 与成交量、K 线形态、均线系统等传统技术分析工具结合使用。

## 局限性与缺陷

### 分拆单问题（最核心缺陷）

大资金可以通过**分拆大单**规避 DDX 追踪：

```
大资金意图：卖出 1 亿元某股票
规避手法：拆成 200 笔 50 万元的小单
结果：DDX 显示为"小单买入"，主力出逃被隐藏
```

> **这是 DDX 最致命的缺陷**。随着量化交易普及，主力机构普遍使用算法将大单拆成小单执行，DDX 有效性已大幅下降。

### 反向指标陷阱

在某些情况下 DDX 可能成为**反向指标**：
- 游资故意用大单买入拉升股价 → 吸引散户追涨 → 游资随后出货 → DDX 正但股价跌
- 主力试盘时的小单买入 → DDX 显示主力买入 → 实际为假信号

### 数据可靠性

- **Level-2 数据**（收费数据）才能准确区分大单/小单；Level-1（免费数据）精度有限
- 不同券商软件的数据处理逻辑可能不同，导致 DDX 数值存在差异
- 港股通/北向资金等通道的成交数据不纳入 DDX 统计

### 综合评价

| 维度 | 评分 | 说明 |
|------|------|------|
| A 股适用性 | ★★★ | 在散户中有一定参考价值 |
| 短线信号可靠性 | ★★ | 分拆单/量化普及后有效性下降 |
| 与其他指标配合 | ★★★ | 配合 K 线、成交量效果更佳 |
| 机构投资者使用 | ★ | 机构用算法拆单，不受 DDX 追踪 |

## 与其他量能指标的关系

| 指标 | 衡量对象 | 与 DDX 的关系 |
|------|---------|--------------|
| **成交量** | 总买卖量 | DDX 是成交量的结构分解 |
| **换手率** | 筹码活跃度 | 高换手 + DDX 飘红 = 更强信号 |
| **北向资金** | 外资方向 | 与 DDX 独立，可交叉验证 |
| **融资余额** | 杠杆资金 | 与 DDX 无直接关联 |
| **主力资金净流入（东方财富）** | 主动大单买入 | 与 DDX 类似，但统计口径不同 |

## 实战注意事项

1. **不要单独使用 DDX**：必须配合价格走势、K 线形态、成交量综合判断
2. **关注持续性**：单日 DDX 飘红意义有限，连续 3-5 天飘红才更可信
3. **警惕涨停板 DDX 翻绿**：主力在涨停出货是 A 股经典套路
4. **Level-2 数据更准**：免费数据的大单识别精度有限，有条件尽量用 Level-2
5. **小市值股票更有效**：大盘蓝筹因量化拆单，DDX 失真度高；小盘股相对更有效
