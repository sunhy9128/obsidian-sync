# 实际利率框架

> 实际利率 = 名义利率 - 通胀预期。这是黄金、零息资产、以及所有"避险资产"的定价核心。

## 一、核心命题

持有黄金不产生利息，因此**黄金的价值取决于持有它相比持有其他生息资产的机会成本**。这个机会成本就是实际利率。

```
黄金价格 ← 1 / 实际利率
           ↑
      持有黄金的机会成本

实际利率↑ → 持有黄金成本↑ → 金价↓
实际利率↓ → 持有黄金成本↓ → 金价↑
```

## 二、实际利率的构成

| 变量 | 定义 | 数据来源 |
|------|------|----------|
| 名义利率 | 10年期美债收益率（或联邦基金利率） | 市场实时价格 |
| 通胀预期 | 市场隐含的未来5-10年通胀率 | TIPS收益率（10年期） |
| 实际利率 | 名义利率 - 通胀预期 | 计算得出 |

**简化公式**：
```
实际利率 ≈ 10年美债收益率 - 10年TIPS收益率
          = 名义利率 - 通胀预期
```

## 三、实际利率框架的四大应用场景

### 场景1：黄金定价

| 实际利率方向 | 黄金方向 | 典型场景 |
|-------------|----------|----------|
| 实际利率↑ | 金价↓ | 美联储鹰派加息 + 通胀预期平稳 |
| 实际利率↓ | 金价↑ | 美联储降息 + 通胀预期下行 |
| 名义利率↑ + 通胀预期↑（等幅） | 金价平稳 | 滞胀初期 |

**关键案例**：2022年地缘冲突期间，原油↑→通胀预期↑，但美联储鹰派加息更激进→名义利率上行速度超过通胀预期→实际利率↑→金价不升反跌。

### 场景2：美联储政策分析

| 情景 | 名义利率 | 通胀预期 | 实际利率 | 黄金 |
|------|----------|----------|----------|------|
| 激进加息（控通胀） | ↑↑ | →或↓ | **↑** | 跌 |
| 降息（刺激经济） | ↓ | → | **↓** | 涨 |
| 防御性降息（经济恶化） | ↓ | ↓（更慢） | **↑或走平** | 承压 |
| 滞胀（加息控通胀但经济差） | ↑ | ↑ | **平稳或略升** | 区间震荡 |

### 场景3：新兴市场货币

实际利率是EM货币最核心的驱动因素之一：

```
美联储实际利率↑ → 美元资产吸引力↑ → 资本流入美国 → EM货币贬值
实际利率↑ → 黄金等零息资产吸引力↓ → 黄金跌
```

### 场景4：股票估值

高实际利率环境对高PE成长股杀伤力最大：

```
实际利率↑ → 折现率↑ → 未来现金流现值↓ → 成长股估值压缩
```

这就是为什么2022年科技股（高PE）在美联储加息时跌幅远超价值股。

## 四、"盛世古董乱世黄金"的失效条件

"乱世黄金"成立的前提是：**乱世→实际利率↓**。

但"乱世"有两条完全相反的路径：

| 路径 | 实际利率 | 黄金 | 典型案例 |
|------|----------|------|----------|
| 路径1：避险需求↑ → 黄金↑ | 实际利率↓（资金流入黄金） | **涨** | 2008年金融危机初期 |
| 路径2：地缘冲突→通胀↑→名义利率↑更快 | 实际利率↑ | **跌** | 2022年俄乌冲突 |

> **结论**："乱世黄金"只适用于路径1（避险需求主导且实际利率下行）。当冲突导致"通胀+鹰派加息"叠加时，实际利率框架主导，"乱世黄金"失效。

## 五、实际利率与黄金的量化关系

历史上，黄金与实际利率呈现显著负相关：

| 时期 | 10年美债实际利率 | 黄金价格 |
|------|-----------------|----------|
| 2020年8月（高点） | -1.0% | $2,067 |
| 2022年3月（俄乌） | -0.3% → +0.3% | $1,950→$1,900 |
| 2022年11月 | +1.7% | $1,727 |

**近似规律**：实际利率每上升1个百分点，黄金价格下跌约200-300美元（非精确，仅供直觉参考）。
