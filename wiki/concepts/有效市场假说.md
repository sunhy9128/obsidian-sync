> 有效市场假说（Efficient Market Hypothesis, EMH）由 Eugene Fama 于 1970 年系统化提出，认为在有效市场中，资产价格已充分反映所有可获得的信息，投资者无法持续获得超过市场平均水平的风险调整后收益。

---

## 一、核心命题

### 1.1 三大核心命题

| 命题 | 内容 | 推论 |
|------|------|------|
| 价格反映信息 | 市场价格已充分反映相应信息集 | 无法基于该信息集获得超额收益 |
| 信息随机到达 | 新信息以随机方式到达市场 | 价格变动不可预测（随机游走） |
| 套利被消除 | 理性套利者迅速消除价格偏差 | 价格围绕内在价值小幅波动 |

### 1.2 有效性的两个必要条件

```
                    ┌──────────────────────┐
                    │   信息反映条件         │
                    │   (Reflect Condition) │
                    └──────────┬───────────┘
                               │
                               ▼
   ┌────────────────────────────────────────────┐
   │  ① 没有交易成本（佣金、税费、滑点）           │
   │  ② 所有市场参与者都可无成本获得相关信息      │
   └────────────────────────────────────────────┘
```

## 二、三种市场形式

### 2.1 三种形式对比

| 形式 | 反映信息集 | 信息内容 | 检验方法 | 推翻难度 |
|------|------------|----------|----------|----------|
| 弱式（Weak） | 历史价格信息 | 股价、成交量、收益率历史 | 序列相关性、技术分析 | 容易 |
| 半强式（Semi-Strong） | 公开信息 | 财报、公告、宏观数据 | 事件研究法 | 中等 |
| 强式（Strong） | 全部信息（含内幕） | 公开+私人+内幕信息 | 内幕交易者收益检验 | 极难 |

### 2.2 三种形式的逻辑层级

```
强式有效  ⊃  半强式有效  ⊃  弱式有效
   │              │              │
   ▼              ▼              ▼
 全部信息       公开信息       历史信息
```

### 2.3 形式与投资策略有效性

| 形式 | 技术分析 | 基本面分析 | 内幕交易 |
|------|----------|------------|----------|
| 弱式失效 | 有效 | 有效 | 有效 |
| 弱式有效 | 无效 | 有效 | 有效 |
| 半强式有效 | 无效 | 无效 | 有效 |
| 强式有效 | 无效 | 无效 | 无效 |

## 三、检验方法

### 3.1 弱式检验

```python
# 弱式有效检验：收益率序列相关性
# H0: 收益率序列不相关（随机游走）
from scipy.stats import jarque_bera
returns = price.pct_change().dropna()
acf1 = returns.autocorr(lag=1)  # 一阶自相关系数
# 典型发现：acf1 接近 0，弱式有效
```

### 3.2 半强式检验

事件研究法（Event Study）核心步骤：

| 步骤 | 内容 |
|------|------|
| 1. 事件定义 | 公告日 t=0，窗口 [-T, +T] |
| 2. 估计正常收益 | 市场模型 / 市场调整收益 |
| 3. 计算异常收益 AR | AR = 实际收益 - 正常收益 |
| 4. 累计异常收益 CAR | CAR = Σ AR |
| 5. 显著性检验 | t 检验 CAR 是否显著≠0 |

### 3.3 强式检验

| 方法 | 比较对象 | 期望结果 |
|------|----------|----------|
| 共同基金业绩 | 基金收益 vs 基准 | 无显著超额收益 |
| 内部人交易收益 | 内部人收益 vs 市场 | 内部人不应显著跑赢 |

## 四、典型市场异象

### 4.1 主要异象汇总

| 异象 | 描述 | EMH 解释 | 行为金融解释 |
|------|------|----------|--------------|
| 日历效应 | 一月效应、周一效应 | 风险补偿 | 投资者情绪周期 |
| 动量效应 | 过去 3-12 月赢家继续赢 | 风险因子未识别 | 反应不足 |
| 反转效应 | 5 年期输家反弹 | 风险因子未识别 | 反应过度 |
| 小盘溢价 | 小盘股长期跑赢大盘 | 流动性差、风险高 | 投资者偏好大盘 |
| 价值溢价 | 低 P/B 股跑赢高 P/B 股 | 财务困境风险 | 过度悲观 |
| 处置效应 | 投资者偏好早卖盈利 | 税收、交易成本 | 心理账户、损失厌恶 |

### 4.2 异象对 EMH 的挑战

```
┌─────────────────────────────────────────────┐
│   异象存在 → 两种解释                       │
│                                             │
│   A. EMH 视角：异象是"风险补偿"               │
│      ├─ 因子模型（Fama-French）              │
│      └─ 风险未被传统模型识别                  │
│                                             │
│   B. 行为金融视角：异象是"认知偏差"           │
│      ├─ 套利有限（套利有风险，套利者有限）     │
│      └─ 偏差系统性存在                       │
└─────────────────────────────────────────────┘
```

## 五、行为金融的反驳

### 5.1 三大反驳支柱

| 学者 | 主要贡献 | 核心观点 |
|------|----------|----------|
| Kahneman & Tversky | 前景理论 | 损失厌恶、锚定、心理账户 |
| Shiller | 金融的非理性繁荣 | 资产价格偏离基本面由"动物精神"驱动 |
| Thaler | 行为经济 | 自我控制问题、社会偏好 |

### 5.2 套利限制

| 限制 | 描述 |
|------|------|
| 套利有风险 | 价格可能长期偏离，套利者可能爆仓 |
| 噪声交易者风险 | 套利者面临短期回撤与赎回压力 |
| 实施成本 | 卖空限制、保证金、监管约束 |

## 六、EMH 的实践含义

### 6.1 对投资者的启示

| 立场 | 投资策略 | 风险 |
|------|----------|------|
| 强信 EMH | 被动指数投资（ETF/指数基金） | 错过 alpha 机会 |
| 弱信 EMH | 因子投资（Smart Beta） | 因子拥挤、失效 |
| 不信 EMH | 主动选股 / 择时 | 多数跑输基准 |

### 6.2 A 股市场的有效性

| 维度 | 表现 | 解读 |
|------|------|------|
| 弱式有效 | 接近 | 序列相关性低 |
| 半强式 | 较弱 | 公告后漂移明显 |
| 强式 | 较弱 | 内幕交易、概念炒作 |

## 七、争论与现状

### 7.1 学术共识演变

| 时期 | 主流观点 |
|------|----------|
| 1970s-1980s | EMH 占主导 |
| 1990s | 行为金融崛起 |
| 2000s 至今 | 整合（Adaptive Market Hypothesis, Lo 2004） |

### 7.2 自适应市场假说（AMH）

Andrew Lo 提出的整合框架：

- 有效性是"程度"问题，不是"全有全无"
- 效率随时间、机构、技术环境变化
- 套利机会是"生态位"，会被竞争消除
- 解释了泡沫、崩盘的反复出现
