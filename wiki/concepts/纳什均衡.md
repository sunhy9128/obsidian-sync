# 纳什均衡

> 在给定其他参与者策略的情况下，没有任何一方能通过单方面改变自身策略而获得更大收益的策略组合——博弈论最核心的解概念。

由 AI 系统化整理（2026-06-03），待补充原始来源素材。

纳什均衡（Nash Equilibrium）由数学家 John Nash 在 1950 年证明存在性定理时提出，是非合作博弈论的核心均衡概念。理解纳什均衡是分析市场博弈、价格竞争、拍卖机制、契约设计的基石。

---

## 一、形式化定义

### 1.1 一般博弈框架

```
博弈 G = (N, S, u)
├─ N：参与者集合 {1, 2, ..., n}
├─ S：策略空间 S₁ × S₂ × ... × Sₙ
└─ u：收益函数 uᵢ(s₁, s₂, ..., sₙ) → ℝ
```

### 1.2 纯策略纳什均衡

策略组合 $s^* = (s_1^*, s_2^*, \ldots, s_n^*)$ 构成**纯策略纳什均衡**，若对每个参与者 $i$：

```
uᵢ(sᵢ*, s₋ᵢ*) ≥ uᵢ(sᵢ, s₋ᵢ*)   ∀ sᵢ ∈ Sᵢ
```

含义：给定对手策略 $s_{-i}^*$，参与者 $i$ 没有动机单方面偏离 $s_i^*$。

### 1.3 混合策略纳什均衡

参与者以概率分布 $\sigma_i$ 选择策略，均衡条件为：

```
uᵢ(σᵢ*, σ₋ᵢ*) ≥ uᵢ(σᵢ, σ₋ᵢ*)   ∀ σᵢ ∈ Δ(Sᵢ)
```

- 在剪刀-石头-布等无纯策略均衡的博弈中，必须使用混合策略
- 混合均衡中每个纯策略的期望收益**相等**（否则会偏向高收益的纯策略）

## 二、经典案例

### 2.1 情侣看电影（协调博弈）

```
              女生
              动作片    爱情片
男 动作片    (2, 1)   (0, 0)
   爱情片    (0, 0)   (1, 2)
```

| 均衡 | 收益 | 类型 |
|------|------|------|
| (动作片, 动作片) | (2, 1) | 帕累托最优的纳什均衡 |
| (爱情片, 爱情片) | (1, 2) | 帕累托最优的纳什均衡 |
| 双方随机化 | — | 混合策略纳什均衡 |

> 协调博弈存在**多个纳什均衡**，需借助额外机制（焦点、惯例、协商）选择。

### 2.2 猜硬币（Matching Pennies）

```
              乙方
              正      反
甲 正      (1,-1)  (-1, 1)
   反     (-1, 1)   (1,-1)
```

- 甲希望**相同**，乙方希望**不同**——零和博弈
- **无纯策略纳什均衡**
- 唯一均衡：双方各以 1/2 概率选正反

### 2.3 古诺竞争（Cournot Duopoly）

两寡头选择产量 $q_1, q_2$，市场价格 $P = a - b(q_1+q_2)$：

```
纳什均衡产量：
q₁* = q₂* = a / (3b)

纳什均衡价格：
P* = a / 3

总产量 = 2a/(3b) < 垄断产量 a/(2b)
```

| 市场结构 | 产量 | 价格 | 利润 |
|----------|------|------|------|
| 垄断 | a/(2b) | a/2 | a²/(4b) |
| 古诺均衡 | 2a/(3b) | a/3 | a²/(9b) |
| 伯特兰竞争 | a/b | 0 | 0 |

> 古诺均衡是介于垄断和完全竞争之间的**寡头均衡**——这是纳什均衡的工业组织典型应用。

## 三、纳什均衡的存在性

### 3.1 纳什定理（1950）

**定理**：任何**有限博弈**（参与者有限、策略有限）至少存在一个混合策略纳什均衡。

- 证明基于 Kakutani 不动点定理
- 策略数无限时，需更弱的连续性/凸性条件

### 3.2 唯一性与多重性

| 情况 | 性质 |
|------|------|
| 严格占优策略均衡 | 必唯一（且为纯策略） |
| 势能博弈（Potential Game） | 纯策略均衡存在 |
| 协调博弈 | 多重均衡（需焦点/帕累托标准） |
| 猜硬币类零和博弈 | 混合策略均衡唯一（值唯一） |

## 四、纳什均衡 vs 帕累托最优

### 4.1 关键对比

| 维度 | 纳什均衡 | 帕累托最优 |
|------|----------|------------|
| 关注点 | 个体最优反应 | 集体无改进空间 |
| 求解方法 | 迭代最优反应 | 社会福利最大化 |
| 存在性 | 任意有限博弈存在 | 取决于可行集与偏好 |
| 一致性 | 不一定帕累托最优 | 不一定纳什均衡 |
| 应用 | 预测实际行为 | 评价配置效率 |

### 4.2 囚徒困境的悖论

```
              嫌疑人 B
              坦白     抵赖
嫌 坦白    (-5,-5)  (0,-10)
疑 抵赖   (-10, 0) (-1,-1)
人
A
```

- **纳什均衡**：(坦白, 坦白) → (-5, -5)
- **帕累托最优**：(抵赖, 抵赖) → (-1, -1)
- 个体理性导致集体次优——**纳什均衡不一定帕累托最优**

> 纳什均衡是"稳定"配置，帕累托最优是"理想"配置；现实中常需借助制度/契约/重复机制使两者趋同。

## 五、求解方法

### 5.1 严格占优策略剔除

```
找到所有严格占优策略 → 剔除 → 在缩小博弈中重复
```

适用：占优策略博弈；多数简单博弈。

### 5.2 最佳反应函数

对参与者 $i$，最佳反应函数：

```
BRᵢ(s₋ᵢ) = argmax uᵢ(sᵢ, s₋ᵢ)
         sᵢ∈Sᵢ
```

纳什均衡 = 所有参与者最佳反应函数的**交点**：

```
纳什均衡 = ⋂ᵢ BRᵢ
```

### 5.3 混合策略条件

混合策略 $\sigma_i^*$ 是均衡，当且仅当所有被正概率使用的纯策略 $s_i$ 收益相同，且不小于未使用策略的收益：

```
uᵢ(sᵢ, σ₋ᵢ*) = vᵢ*   ∀ sᵢ ∈ supp(σᵢ*)
uᵢ(sᵢ, σ₋ᵢ*) ≤ vᵢ*   ∀ sᵢ ∉ supp(σᵢ*)
```
