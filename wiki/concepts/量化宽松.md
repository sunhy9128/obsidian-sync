# 量化宽松（Quantitative Easing / QE）

> 21 世纪央行最重要的"非传统货币政策"——当利率降至"零下限"时，央行通过"印钞买债"创造流动性

---

## 一、核心定义

### 1.1 一句话定义

**量化宽松**（Quantitative Easing，简称 **QE**），是指央行在**政策利率接近零下限**（Zero Lower Bound，ZLB）时，**大规模购买长期资产**（主要是政府债券、机构债、资产支持证券），通过增加基础货币投放、压低长期利率、刺激经济活动的**非传统货币政策工具**。

### 1.2 与传统货币政策的区别

```
传统货币政策（Open Market Operations, OMO）：
  - 工具：调整政策利率（联邦基金利率）
  - 规模：较小（数十亿至数百亿美元）
  - 期限：短期
  - 传导：利率 → 信贷 → 实体经济
       ↓
量化宽松（QE）：
  - 工具：购买长期资产
  - 规模：巨大（数千亿至数万亿美元）
  - 期限：长期（10-30 年）
  - 传导：资产价格 → 财富效应 → 投资消费
       ↓
⭐ 关键差别：
  - OMO 是"价格型工具"（调整利率）
  - QE 是"数量型工具"（扩大资产负债表）
  - OMO 是"流动性管理"（日常）
  - QE 是"货币创造"（危机应对）
```

## 二、QE 的理论基础

### 2.1 流动性陷阱与零下限

```
【流动性陷阱】（Kenes, 1936）
  - 当利率降至接近零时
  - 央行无法通过降息刺激经济
  - 因为"降无可降"
  - 货币需求变成"无限弹性"
       ↓
【零下限】（Zero Lower Bound, ZLB）
  - 名义利率不能低于 0%
  - 因为现金的替代效应
  - 如果名义利率 < 0%，所有人都囤现金
       ↓
【问题】
  - 当金融危机来袭，利率已经接近 0%
  - 央行无法再降息
  - 必须使用"非传统工具"
  - ⭐ QE 应运而生
```

### 2.2 QE 的传导机制

```
QE 的传导渠道：

【渠道 1：长期利率下降】
  - 央行大量购买长期债券
  - 长期债券价格上涨
  - 长期利率下降
  - 压低房贷利率、企业债利率
       ↓
【渠道 2：资产组合再平衡】
  - 央行购买债券 → 银行"被迫"释放资金
  - 银行资金流向股票、风险资产
  - 风险资产价格上涨
  - ⭐ "财富效应"
       ↓
【渠道 3：汇率贬值】
  - 货币供应增加 → 货币贬值
  - 本币贬值 → 出口增加
  - ⭐ 国际贸易通道
       ↓
【渠道 4：通胀预期】
  - 央行承诺"持续买债"
  - 市场预期通胀上升
  - 通胀预期上升 → 实际利率下降
  - ⭐ "前瞻指引 + QE" 组合
       ↓
【渠道 5：信号效应】
  - QE 表明央行"不惜一切代价"
  - 提振市场信心
  - ⭐ "中央银行信誉"
```

## 三、QE 的历史演进

### 3.1 QE 的"四个阶段"

```
【第一阶段：理论阶段（1990s-2000s）】

  日本银行（BOJ）1990s 实验：
  - 1990s 日本泡沫破裂
  - 1999 BOJ 实施"零利率政策"
  - 2001-2006 BOJ 实施"量化宽松政策"
    * 大量购买日本国债
    * 央行账户余额从 5 万亿增至 30 万亿日元
    * 但效果有限（日本陷入通缩）
  - ⭐ QE 的"先驱实验"
       ↓
【第二阶段：金融危机应对（2008-2014）】

  美联储 QE1/QE2/QE3：
  - 2008.11-2010.3 ：QE1（1.7 万亿美元）
  - 2010.11-2011.6 ：QE2（6000 亿美元）
  - 2012.9-2014.10：QE3（1.6 万亿美元）
  - ⭐ "现代 QE 范式"成型
       ↓
【第三阶段：欧日 QE（2014-2022）】

  ECB APP（2015-2018）：
  - 2015.1 ECB 启动 APP
  - 累计购买 2.6 万亿欧元
  - 配合 OMT、负利率

  BOJ QQE + YCC（2013-至今）：
  - 2013.4 黑田东彦启动 QQE（Quantitative and Qualitative Easing）
  - 2016.9 引入 YCC（Yield Curve Control）
  - ⭐ 日本版 QE+YCC
       ↓
【第四阶段：无限 QE + 新冠应对（2020-2022）】

  美联储"无限 QE"（2020.3）：
  - 2020.3.23 Fed 宣布"无限 QE"
  - 不设购买上限
  - 实际购买 ~3 万亿美元

  ECB PEPP（2020.3-2022.3）：
  - 规模 1.85 万亿欧元
  - 突破 Capital Key 约束
```

## 四、QE 的国际实践

### 4.1 美联储 QE（最详细）

```
美联储 QE 历史：

【QE1】（2008.11 - 2010.3）
  - 规模：1.7 万亿美元
  - 标的：MBS（1.25 万亿）+ 机构债（1750 亿）+ 长期国债（3000 亿）
  - 触发：金融危机最严重时刻
  - 效果：信贷市场解冻，股市从 7000 反弹至 11000

【QE2】（2010.11 - 2011.6）
  - 规模：6000 亿美元
  - 标的：长期国债
  - 触发：失业率持续 9%+，通缩风险
  - 效果：争议较大，效果有限

【QE3 / "QE Infinity"】（2012.9 - 2014.10）
  - 规模：每月 400 亿美元 MBS + 450 亿美元国债
  - 累计：~1.6 万亿美元
  - 触发：失业率高 + 通胀过低
  - 效果：失业率从 8.1% 降至 6.2%
  - ⭐ "开放式 QE" 的开端

【QE4 / "无限 QE"】（2020.3 - 2022.3）
  - 规模：~3 万亿美元（实际）
  - 标的：国债 + MBS + 公司债（创新）
  - 触发：COVID 危机
  - 效果：流动性危机快速化解
  - ⭐ 美联储最激进的 QE
```

### 4.2 ECB QE

```
ECB QE 历史：

【APP 1.0】（2015.1 - 2018.12）
  - 规模：累计 2.6 万亿欧元
  - 标的：成员国国债 + 机构债 + ABS
  - 触发：欧元区通胀持续过低
  - 特点：按 Capital Key 比例购买

【APP 2.0】（2019.11 - 2020.3）
  - 规模：每月 200 亿欧元
  - 触发：通胀过低

【PEPP】（2020.3 - 2022.3）
  - 规模：1.85 万亿欧元
  - 触发：COVID 危机
  - ⭐ 突破 Capital Key 约束
  - ⭐ 可以购买意大利、希腊等国债券

⭐ 累计：~5 万亿欧元（2015-2022）
   ECB 资产负债表从 2.2 万亿增至 8.8 万亿欧元
```

### 4.3 日本银行 QE

```
日本 QE 历史：

【QQE 1.0】（2013.4 - 2016.9）
  - 央行行长：黑田东彦
  - 规模：每年 60-80 万亿日元
  - 标的：长期国债 + ETF + J-REITs
  - ⭐ "质化量化宽松"（QQE）

【YCC 1.0】（2016.9 - 2023.4）
  - 政策：将 10 年期国债收益率控制在 0% 附近
  - ⭐ 全球唯一"收益率曲线控制"

【YCC 2.0】（2023.4 - 至今）
  - 放松 YCC（容忍 10 年期国债收益率 1%）
  - ⭐ YCC 政策退出

⭐ 累计：BOJ 持有日本国债 ~50%
   BOJ 持有日本股票 ETF ~7%
   ⭐ 全球最大 QE 实验
```

## 五、QE 的核心问题

### 5.1 QE 有效性的争议

```
QE 有效性的"四大争议"：

【争议 1：是否真正刺激经济？】
  - 支持：长期利率下降、股市上涨、财富效应
  - 反对：实体经济复苏有限、不平等加剧
  - 结论：QE 对金融市场有效，对实体经济作用有限

【争议 2：是否推升资产泡沫？】
  - 支持者认为：QE 创造"泡沫"
  - 反对者认为：QE 只是修复 2008 崩盘
  - 结论：长期 QE 可能加剧资产泡沫

【争议 3：是否导致贫富差距？】
  - 财富效应 → 持有资产者受益
  - 失业者、低收入者无资产受益
  - ⭐ QE 加剧不平等

【争议 4：是否引发通胀？】
  - 传统理论：印钞必然通胀
  - 实际情况：QE 后通胀温和
  - 但 2021-2023 全球通胀飙升
  - ⭐ "QE 的通胀效应滞后"
```

### 5.2 QE 的副作用

```
【副作用 1：资产价格泡沫】
  - 美股 2009-2024 涨 5 倍+
  - 房地产价格飙升
  - 加密货币兴起（投机）
  - ⭐ "QE 时代泡沫"

【副作用 2：财富不平等】
  - 资产持有者收益
  - 工薪阶层相对受损
  - ⭐ "QE 加剧 K 型复苏"

【副作用 3：金融脆弱性】
  - 利率长期过低 → 过度承担风险
  - 影子银行扩张
  - "Searching for yield"
  - ⭐ "QE 制造下一场危机"

【副作用 4：货币贬值】
  - 货币供应增加
  - 美元、欧元、日元相对贬值
  - ⭐ 国际货币竞争加剧

【副作用 5：央行独立性争议】
  - 央行与财政的边界模糊
  - "财政货币化"风险
  - ⭐ 政治化压力
```

## 六、QE 的退出（量化紧缩）

### 6.1 QT 的挑战

```
QE 退出 = 量化紧缩（Quantitative Tightening, QT）

【QT 的工具】
  - 停止再投资（自然到期）
  - 主动卖出债券（罕见）
  - 加息（配合）

【QT 的挑战】
  - 长期利率上升
  - 资产价格下跌
  - 市场流动性下降
  - ⭐ QT 比 QE 更难

【美联储 QT（2017-2019、2022-至今）】
  - 2017.10-2019.7：第一次 QT（缩减 7000 亿美元）
  - 2022.6 至今：第二次 QT（缩减 1.6 万亿美元）
  - ⭐ 2023.3 硅谷银行危机暴露 QT 风险
```

### 6.2 ECB QT

```
ECB QT（2023-至今）：
  - 2023.3 APP 停止再投资
  - 2024 起 PEPP 停止再投资
  - 配合加息
  - ⭐ 比 Fed QT 温和
```

## 七、QE 在中国

### 7.1 中国央行的"类 QE"工具

```
⭐ 中国没有传统意义上的 QE
   但有"类 QE"工具：

【工具 1：MLF（中期借贷便利）】
  - 2014.9 创设
  - 央行向商业银行提供 3-12 个月贷款
  - 抵押品：国债、政金债、信用债
  - 累计余额 ~5 万亿人民币（2026）

【工具 2：PSL（抵押补充贷款）】
  - 2014.10 创设
  - 服务政策性银行（国开行、农发行）
  - 利率优惠
  - 累计余额 ~3 万亿人民币

【工具 3：买断式逆回购】
  - 2024 新增
  - 央行从市场买断债券
  - 中期流动性投放

【工具 4：再贷款再贴现】
  - 结构性工具
  - 精准支持特定行业

⭐ 与传统 QE 的区别：
  - 中国工具是"中期再融资"而非"直接购买长期资产"
  - 中国工具是"结构性"而非"总量"
  - 中国没有"扩表"到极端规模
  - ⭐ "中国式 QE"是"温和的、定向的、结构性的"
```

### 7.2 中国央行的"结构性扩表"

```
⭐ 中国央行的"另类扩表"：

中国央行总资产 / GDP：
  - 2014：35%
  - 2020：40%
  - 2026：38%
  - 远低于美联储（~30% 但美元是全球货币）
  - 远低于 ECB（~65%）
  - 远低于 BOJ（~135%）

⭐ 评价：
  - 中国央行"扩表"更克制
  - 但通过 MLF、PSL 等工具达到类似效果
  - 化债背景下"中国式 QE"仍在探索
```

## 八、QE 的历史评价

### 8.1 QE 的五个判断

```
【判断 1：QE 是 21 世纪最重要的金融创新】
  - 没有 QE，2008 危机可能演变成大萧条
  - QE 改变了央行的"角色"（从利率管理者到资产管理者）

【判断 2：QE 加剧贫富差距】
  - 资产持有者受益
  - 工薪阶层受损
  - ⭐ 政治化压力

【判断 3：QE 制造下一场危机】
  - 长期低利率 → 过度承担风险
  - 影子银行扩张
  - "Searching for yield"
  - ⭐ "QE 的遗产"

【判断 4：QE 不能替代财政政策】
  - 单纯 QE 不能解决总需求不足
  - 需要财政政策配合
  - ⭐ "QE + 财政"才是有效组合

【判断 5：QE 的退出比 QE 更难】
  - QT 导致资产价格下跌
  - 市场流动性下降
  - 硅谷银行危机（2023.3）就是 QT 的副作用
  - ⭐ "QE 的退出"将定义未来 10 年
```

## 十、一句话总结

> **QE = 21 世纪央行最重要的"非传统货币政策"——当利率降至零下限，央行通过"印钞买债"创造流动性**。
>
> 它起源于日本（1990s 实验），成熟于美国（2008 QE1/QE2/QE3），扩展到 ECB（APP/PEPP），极致于日本（QQE/YCC）。**QE 改变了央行的本质**：从"利率管理者"变为"资产管理者"。
>
> **理解 QE = 理解五件事**：
> 1. **零下限的必然产物**：当利率不能再降，央行必须换工具
> 2. **传导渠道**：长期利率、财富效应、汇率、通胀预期、信号
> 3. **副作用**：资产泡沫、贫富差距、金融脆弱性
> 4. **退出困难**：QT 比 QE 更难（硅谷银行危机是教训）
> 5. **中国的"另类 QE"**：MLF、PSL、买断式逆回购——结构性、温和、定向
>
> 在中国化债、负利率、量化紧缩的今天，**QE 的真正遗产不是"印钞"，而是央行角色的根本性变化**——央行已经成为金融市场最大的"玩家"，而不是"裁判"。
