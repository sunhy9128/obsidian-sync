# 菲利普斯曲线

> 描述失业率与通胀率之间权衡关系的曲线：短期存在负相关，长期则在自然失业率处呈垂直。

---

## 一、原始菲利普斯曲线（1958）

### 1.1 历史背景

新西兰经济学家 A.W. Phillips 在 1958 年发表《1861-1957 年英国失业率与货币工资变化率的关系》，发现二者存在稳定的**负相关**关系。

### 1.2 原始形式

```
ẇ = f(u)        （ẇ = 名义工资变化率，u = 失业率）
```

> 工资变化率与失业率负相关：失业率高 → 工资上涨慢；失业率低 → 工资上涨快。

### 1.3 含义

- 失业率高时，劳动力市场宽松 → 工资上行慢
- 失业率低时，劳动力市场紧张 → 工资上行快
- 推断：政策制定者可在低失业与低通胀间"菜单选择"

---

## 二、改造后的菲利普斯曲线（1960s）

### 2.1 萨缪尔森-索洛版本

把工资变化替换为**物价变化（通胀率）**：

```
π = f(u)        （π = 通胀率，u = 失业率）
```

| 失业率 | 通胀率 |
|--------|--------|
| 高 | 低 |
| 低 | 高 |
| 中 | 中 |

### 2.2 政策含义：菜单选择

```
       π
       │       SPC（短期）
       │      ╱
       │    ╱
       │  ╱
       │╱
       └─────────── u
```

> 政策制定者沿 SPC 移动：低失业 → 接受高通胀；低通胀 → 接受高失业。

## 三、预期增强版（Friedman-Phelps）

### 3.1 1968 弗里德曼批评

弗里德曼（Milton Friedman）和费尔普斯（Edmund Phelps）独立提出：原始菲利普斯曲线**忽略了通胀预期**。

### 3.2 短期菲利普斯曲线（SPC）

```
π = π^e - α(u - u_n)
```

| 变量 | 含义 |
|------|------|
| π | 实际通胀率 |
| π^e | 预期通胀率 |
| u | 实际失业率 |
| u_n | 自然失业率 |
| α | 系数（> 0） |

> 关键：在预期**不变**时，降低失业率（u < u_n）可暂时推高通胀。

### 3.3 长期菲利普斯曲线（LPC）

```
π = π^e            （LPC 是一条 u = u_n 的垂线）
```

| 维度 | SPC | LPC |
|------|-----|-----|
| 形状 | 向右下方倾斜 | 垂直（在 u_n 处） |
| 移动 | π^e 改变 → SPC 上移/下移 | 固定 |
| 政策 | 短期可移动 | 长期无效 |

### 3.4 长期-短期关系图

```
        π
        │
   π^e↑ │     SPC_2
        │    ╱
        │   ╱  SPC_1
        │  ╱ ╱
        │ ╱╱
        │╱──── u_n  LPC（垂直）
        └────────── u
```

> 持续刺激：π^e 上移 → SPC 上移 → 失业率回到 u_n，但通胀永久上升。

## 四、滞胀与 1970s 油价冲击

### 4.1 滞胀定义

**滞胀**（Stagflation）= 停滞（高失业）+ 通胀（高 CPI）并存。

### 4.2 1970s 美国案例

| 年份 | 失业率 | CPI 同比 |
|------|--------|----------|
| 1973 | 4.9% | 6.2% |
| 1974 | 5.6% | **11.0%** |
| 1975 | 8.5% | 9.1% |
| 1979 | 5.8% | **13.3%** |
| 1980 | 7.1% | 13.5% |

> **石油危机**（1973 赎罪日战争，1979 伊朗革命）→ 油价 4 倍 → 成本推动型通胀 → 失业率上升 → 滞胀。

### 4.3 对菲利普斯曲线的挑战

1970s 滞胀证明：
- 失业与通胀**可以同时上升**（违反原始 SPC 形状）
- 供给冲击会让 AS 左移 → 同时推高通胀和失业
- 长期看，菲利普斯曲线整体可能右移

## 五、牺牲率（Sacrifice Ratio）

### 5.1 定义

为将通胀降低 1 个百分点，所需要承受的 GDP 损失（以失业率上升的形式，或累计 GDP 损失与通胀下降之比）。

```
SR = (累计失业率百分点) / (通胀下降百分点)
```

### 5.2 历史估计

| 时期 | 估计牺牲率 | 备注 |
|------|------------|------|
| 沃尔克时代（1979-1982） | ~5 | 通胀从 13% 降至 4%，失业率 10%+ |
| 软着陆时期 | 0.5-1 | 通过预期管理降低代价 |
| 现代估计 | 0.5-2 | 央行信誉越高，牺牲率越低 |

### 5.3 政策含义

- **央行信誉** → 降低 π^e → 降低牺牲率
- **前瞻指引** → 引导预期 → 软着陆
- **历史教训**：沃尔克式硬着陆代价巨大，但树立了美联储反通胀信誉

## 六、菲利普斯曲线与奥肯定律

### 6.1 奥肯定律

```
实际 GDP 增速 - 潜在 GDP 增速 = -2 × (u - u_n)
```

> 失业率高于自然失业率 1 个百分点 → GDP 低于潜在 GDP 2 个百分点。

### 6.2 联合关系

```
菲利普斯曲线：u ↔ π  （失业-通胀权衡）
奥肯定律：u ↔ y  （失业-产出缺口）
三者形成 u - y - π 三角
```

```
        通胀 π
          ▲
          │
          │   SPC
          │  ╱
          │ ╱
          │╱
          ●──────→ 失业率 u
          │ ╲
          │  ╲ OK（奥肯定律）
          ▼
        GDP 增速
```

> 货币政策（降息刺激）→ 总需求 ↑ → 实际 GDP 增速 ↑ → 失业率 ↓ → 通胀 ↑
> 路径：沿 SPC 上移，u 下降，π 上升

## 七、对菲利普斯曲线的现代挑战

### 7.1 2008 后的"消失"之谜

- 全球通胀与失业率关系弱化
- 解释：通胀预期被锚定（Anchored）、全球化压制物价、央行反应函数改变

### 7.2 平坦化（Flattening）

现代 SPC 比 1980s 更平 → 失业率变动对通胀影响小 → 政策"菜单"作用弱化

### 7.3 异质性（Heterogeneity）

- 不同行业、不同人群的菲利普斯曲线斜率差异大
- 鲍威尔时代开始关注"种族/收入"分群失业-通胀关系

## 八、关键概念对比

| 概念 | 短期 | 长期 |
|------|------|------|
| 失业-通胀关系 | 负相关（SPC） | 不相关（LPC 垂直） |
| 政策有效性 | 有效 | 无效 |
| 预期作用 | 锚定时弱 | 决定位置 |
| 牺牲率 | 有 | 长期视角下趋同 |
