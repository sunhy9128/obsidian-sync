# 汇率超调模型

> 货币供给扩张时，由于商品价格短期粘性而汇率即时充分调整，汇率会先**过度反应**到超过长期均衡值，再随商品价格调整逐步**回归**。

## 一、核心定义

### 1.1 提出者与背景

**鲁迪格·多恩布什**（Rüdiger Dornbusch, 1976）在论文 *"Expectations and Exchange Rate Dynamics"* 中提出：

> 在**理性预期 + 短期价格粘性 + 长期价格弹性**的假设下，货币冲击会使即期汇率**瞬时跳跃**到超过其长期均衡水平（**超调**），随后随商品价格调整**渐进回归**。

这是开放经济宏观经济学**最具影响力的模型之一**，与蒙代尔-弗莱明模型并列为基石。

### 1.2 一句话机制

```
货币扩张 → 货币市场即时出清（利率↓）→ 资金外流（汇率即时升）
         → 利率回升压力 → 但商品价格粘性 → 汇率超调
         → 长期：商品价格追上 → 实际汇率回归均衡
```

### 1.3 模型分类

| 模型 | 价格假设 | 预期 | 代表学者 |
|------|---------|------|---------|
| **弹性价格货币模型** | 商品价格灵活 | 静态预期 | Frenkel (1976) |
| **粘性价格货币模型** | 商品价格粘性 | 理性预期 | **Dornbusch (1976)** |
| **资产组合平衡模型** | 国内外资产不完全替代 | 理性预期 | Branson (1977) |
| **新开放经济宏观** | Calvo 定价 | 理性预期 + 微观基础 | Obstfeld-Rogoff (1995) |

## 二、模型核心

### 2.1 三大市场与三个方程

```
货币市场（出清）：
m - p = φy - λi         （实际货币供给 = 实际货币需求）
商品市场（IS）：
y = δ(s - p + y*) + γ(i - i*) - σ(i - E[Δs])   （总需求 = 总供给）
资产市场（UIP）：
i = i* + (E[Δs])        （无抛补利率平价）
```

| 符号 | 含义 |
|------|------|
| m, p | 名义货币供给、价格水平（对数） |
| y, y* | 国内外产出 |
| i, i* | 国内外利率 |
| s | 名义汇率（直接标价） |
| E[Δs] | 预期汇率变动 |
| φ, λ, δ, γ, σ | 系数（均为正） |

### 2.2 短期 vs 长期

| 维度 | 短期 | 长期 |
|------|------|------|
| 商品价格 p | **粘性（固定）** | 灵活调整 |
| 产出 y | 短期偏离潜在 | 回到潜在 ȳ |
| 利率 i | 由货币市场出清 | 回到 i* |
| 汇率 s | **瞬时超调** | 长期均衡 s̄ |

> 长期均衡：购买力平价成立，s̄ = p - p*（或 m - m* 决定的均衡）。

### 2.3 超调机制详解

**初始状态**：m, p, y, s, i 都处于均衡。

**冲击**：央行突然**扩张货币供给**（m↑）。

**瞬间（t=0）**：
1. 货币市场超额供给 → 利率 i↓
2. i < i* → 资金外流（UIP 要求 E[Δs] 维持）→ 即期汇率 s↑
3. 商品价格 p 粘性不变 → **实际汇率升值**
4. 升值幅度 > 长期均衡 → **超调**

**调整（t>0）**：
1. 实际升值 → 净出口下降 → 总需求 y↓
2. 总需求下降 → 商品价格 p 开始下降（粘性下逐步）
3. p↓ → 实际货币供给 ↑ → i 回升
4. i 回升 → 资金回流 → s 回落
5. 长期：p 完全调整到新 m 决定的水平，s 回到 s̄

## 三、数学形式化

### 3.1 超调幅度公式

简化解（线性化）：

```
长期均衡汇率：s̄ = -(1/λ) × m + const

短期汇率：s = s̄ + θ × (p̄ - p)      θ > 1

超调幅度：s₀ - s̄ = θ × (p̄ - p) > 0
```

> 超调系数 θ 取决于**货币需求利率弹性** λ 和**商品价格调整速度**。

### 3.2 价格调整动态

```
ḋ = π × (y - ȳ)        （价格变化率 = 通膨，与产出缺口相关）
```

- 短期：y < ȳ → 通缩压力 → p↓ 
- 长期：y → ȳ → 通缩 → 0 → p 稳定

### 3.3 时序图（文字版）

```
汇率 s
   │
   │ ●─────●           ← 长期均衡 s̄（新水平）
   │  \
   │   \
   │    \●             ← 短期超调 s₀
   │     \
   │      ●────────    ← 渐进回归
   │
   └──────────────── 时间 t
        t=0     t=∞
```

> 名义汇率 s 立即跳跃到 s₀，**超过**长期均衡 s̄，然后单调收敛。

## 四、政策冲击分析

### 4.1 货币供给冲击

| 维度 | 短期 | 长期 |
|------|------|------|
| 名义汇率 s | **大幅贬值**（超调） | 回到与 m 同比例的均衡 |
| 实际汇率 q | 升值 | 回到原水平 |
| 利率 i | 下降 | 回到 i* |
| 产出 y | 上升（贬值刺激） | 回到 ȳ |
| 价格 p | 不变（粘性） | 上升至新均衡 |

### 4.2 财政扩张冲击

| 维度 | 短期 | 长期 |
|------|------|------|
| 名义汇率 s | **升值** | 回到均衡 |
| 利率 i | 上升 | 回到 i* |
| 产出 y | 上升 | 回到 ȳ |

> 财政扩张通过"挤出"导致利率上升，吸引资金流入，本币升值——与封闭经济完全相反。

### 4.3 货币需求冲击

| 维度 | 短期 | 长期 |
|------|------|------|
| 货币需求 m↓ | s **贬值** | 回到均衡 |

### 4.4 冲击对比表

| 冲击 | 即期汇率 | 长期汇率 | 超调方向 |
|------|---------|---------|---------|
| 货币扩张 (m↑) | 大幅贬值 | 贬值（同比例） | 向上超调 |
| 货币紧缩 (m↓) | 大幅升值 | 升值（同比例） | 向下超调 |
| 财政扩张 (G↑) | 升值 | 不变 | 不超调 |
| 货币需求↓ | 贬值 | 贬值 | 向上超调 |
| 国外利率↑ | 贬值 | 贬值 | 向上超调 |

## 五、为什么新兴市场汇率波动大

### 5.1 模型预测

> Dornbusch 模型预测：**货币扩张越快、超调越大、通胀粘性越强的国家，汇率波动越大**。

### 5.2 新兴市场特征

| 特征 | 影响 |
|------|------|
| **通胀率高** | 价格粘性 → 超调幅度大 |
| **央行频繁干预** | 政策不确定性 → 汇率频繁跳跃 |
| **资本管制** | 资本流动不稳定 → 汇率压力 |
| **美元化**（部分国家） | 货币政策受外部约束 |
| **商品出口依赖** | 大宗商品价格冲击 → 汇率波动 |
| **薄外汇市场** | 流动性差 → 容易被冲击 |

### 5.3 新兴市场危机案例

| 危机 | 触发 | 汇率反应 |
|------|------|---------|
| 1994 龙舌兰（墨西哥） | 比索贬值预期 | 资本外流、比索暴跌 |
| 1997 亚洲金融危机 | 泰铢贬值、资本外逃 | 货币 30-50% 贬值 |
| 2001 阿根廷 | 比索与美元 1:1 不可持续 | 强制贬值 70%+ |
| 2014 卢布危机 | 油价暴跌 + 制裁 | 卢布腰斩 |
| 2018 土耳其里拉 | 埃尔多安反紧缩政策 | 里拉 30% 贬值 |
| 2022 日元贬值 | 美联储紧缩 | 1 美元 = 150 日元（25 年新低） |

> 新兴市场危机典型路径：货币扩张 → 经常账户逆差 → 资本流出 → 货币贬值 → 通胀 → 央行被动紧缩 → 经济衰退。

### 5.4 超调与金融危机

```
外部冲击 → 货币贬值预期
        → 资本外流（Dornbusch 超调）
        → 实际利率上升
        → 银行/企业外债危机
        → 自我强化的贬值
        → 储备耗尽 → 货币崩溃
```

## 六、模型的扩展与批评

### 6.1 模型成功之处

| 贡献 | 说明 |
|------|------|
| **解释短期波动** | 汇率波动 >> 通胀波动 → 实际汇率波动 |
| **解释超调现象** | 1980s 美元大幅升值-回归 |
| **政策启示** | 货币冲击有强烈短期汇率效应 |
| **理性预期应用** | Dornbusch 是先驱者之一 |

### 6.2 模型局限

| 批评 | 说明 |
|------|------|
| **商品价格粘性** | 大宗商品（油价）灵活，但工业品粘性 |
| **忽视资本结构** | 不区分 FDI / 证券投资 |
| **单一资产** | 假设国内外资产完全替代 |
| **微观基础不足** | 后来新开放经济宏观（NOEM）补充 |
| **美元特殊地位** | 模型对储备货币预测较弱 |

### 6.3 现代发展

| 扩展 | 代表 |
|------|------|
| **新开放经济宏观（NOEM）** | Obstfeld-Rogoff (1995) 引入微观基础 |
| **DSGE 开放模型** | Gali-Monacelli (2005) 小开放经济 |
| **金融摩擦** | Gertler-Karadi (2011) 纳入金融部门 |
| **异质预期** | 多版本预期、新闻冲击 |

## 八、关键易错点

| 误区 | 正确认识 |
|------|----------|
| Dornbusch = 货币模型 | 货币模型是**弹性价格**版；Dornbusch 加了**粘性** |
| 货币扩张必贬 | 是的（短期与长期），但幅度可能远超预期 |
| 汇率总是超调 | 需有**商品价格粘性**假设；灵活价格经济不超调 |
| 财政扩张贬本币 | **反**！Dornbusch 中财政扩张升本币 |
| 央行干预能阻止超调 | 短期可能，长期无效（套利+预期） |
| 模型只适合发达国家 | 同样适用于新兴市场（且更显著） |
