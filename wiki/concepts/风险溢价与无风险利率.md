> 风险溢价（Risk Premium）是投资者承担额外风险所要求的超额收益补偿；无风险利率（Risk-free Rate）则代表资本在"无风险"状态下应得的时间价值。两者共同构成 CAPM 等资产定价模型的基石。

---

## 一、概念总览

### 1.1 风险溢价的层级结构

```
                ┌───────────────────────────┐
                │      总收益 = Rf + Σ溢价   │
                └─────────────┬─────────────┘
                              │
       ┌──────────────────────┼──────────────────────┐
       │                      │                      │
       ▼                      ▼                      ▼
   无风险利率             系统性风险              个体/特质风险
   (Rf)                   (系统性溢价)            (可通过分散消除)
       │                      │
       ├─ 实际无风险利率        ├─ 股权风险溢价 (ERP)
       ├─ 通胀溢价             ├─ 期限溢价 (Term Premium)
       └─ 流动性溢价           ├─ 信用利差 (Credit Spread)
                              ├─ 通胀风险溢价
                              └─ 流动性风险溢价
```

### 1.2 风险溢价分类

| 类别 | 含义 | 典型数值 |
|------|------|----------|
| 股权风险溢价 ERP | 股票相对无风险利率的超额收益 | 4-7% |
| 期限溢价 | 长债相对短债的额外收益 | 0-2% |
| 信用利差 | 低信用等级债相对国债的额外收益 | 0.5-5%+ |
| 通胀溢价 | 通胀预期引致的额外收益 | 1-3% |
| 流动性溢价 | 流动性差资产的额外补偿 | 0-2% |

## 二、无风险利率（Risk-free Rate, Rf）

### 2.1 理论特征

| 特征 | 要求 |
|------|------|
| 无违约风险 | 信用等级为 0（理论不可能，但近似） |
| 无再投资风险 | 持有至到期时无现金流不确定 |
| 无通胀风险 | 实际利率稳定 |
| 无流动性风险 | 可随时变现且不损失 |

### 2.2 实务中的"无风险"代理

| 资产 | 国家 | 优点 | 缺点 |
|------|------|------|------|
| 10 年期国债 | 中国 | 期限长，反映长期 | 含期限溢价 |
| 1 年期国债 | 中国 | 短端，接近纯 Rf | 期限短 |
| 3 个月国库券 | 美国 | 国际通用 | 名义，需扣通胀 |
| 同业存单利率 AAA | 中国 | 信用好 | 仍含少量信用风险 |

### 2.3 中国市场常用 Rf

| 应用场景 | 常用 Rf 代理 |
|----------|--------------|
| 股票估值（DCF） | 10 年期国债到期收益率 |
| 短期资金成本 | 7 天逆回购利率 |
| 货币政策基准 | 1 年期 LPR / 7 天 OMO |
| 学术研究 | 1 年期定期存款利率（历史） |

## 三、股权风险溢价（Equity Risk Premium, ERP）

### 3.1 定义

```
ERP = E(Rm) - Rf
即：股权风险溢价 = 市场组合预期收益 - 无风险利率
```

### 3.2 ERP 估算方法对比

| 方法 | 数据 | 公式 | 优点 | 缺点 |
|------|------|------|------|------|
| 历史 ERP | 长期收益率数据 | Rm 历史 - Rf 历史 | 客观 | 样本期敏感 |
| 隐含 ERP | 当前股价、盈利 | gordon growth 反推 | 前瞻 | 假设多 |
| 调查 ERP | 学者/机构调研 | 主观期望 | 灵活 | 主观偏差 |
| 风险因子 ERP | 因子模型回归 | 多因子回归 | 多维 | 模型依赖 |

### 3.3 全球主要市场 ERP 历史均值

| 市场 | 长期 ERP | 估算区间 |
|------|----------|----------|
| 美国 | 5.5-7.0% | 1928-2025 |
| 欧洲 | 4.5-6.0% | 1900-2025 |
| 日本 | 3.0-5.0% | 1970-2025 |
| 印度 | 6.0-8.0% | 1990-2025 |
| 中国（A 股） | 4.0-7.0% | 1990-2025 |

> 学术研究（Dimson-Marsh-Staunton，DMS）表明，全球 ERP 长期均值约 5-6%。

### 3.4 中国 A 股 ERP 的特殊性

| 因素 | 影响 | 方向 |
|------|------|------|
| 散户主导 | 噪声交易，β 解释力弱 | 降低有效 ERP |
| 政策市特征 | "政策溢价"非市场化 | 提升 ERP |
| 卖空限制 | 套利不完全，定价偏差大 | 提升 ERP |
| 增长预期 | GDP 高增长 | 长期 ERP 偏低 |
| 投资者保护 | 法律不完善 | 提升 ERP |

## 四、期限溢价（Term Premium）

### 4.1 定义

```
期限溢价 = 长期国债收益率 - 短期国债收益率
反映：投资者持有长期债券所要求的额外补偿
```

### 4.2 期限溢价的来源

| 来源 | 机制 |
|------|------|
| 利率风险 | 长期债对利率更敏感（久期长） |
| 通胀不确定性 | 长期内通胀预测误差大 |
| 流动性偏好 | 偏好短期灵活性的投资者需要补偿 |

### 4.3 中国期限结构

| 时段 | 10Y-1Y 利差 | 解读 |
|------|-------------|------|
| 正常 | 50-100 bp | 健康收益率曲线 |
| 倒挂 | < 0 | 经济衰退预期 |
| 陡峭 | > 150 bp | 经济复苏预期 |

## 五、信用利差（Credit Spread）

### 5.1 定义

```
信用利差 = 信用债收益率 - 同期限国债收益率
反映：投资者承担信用风险所要求的额外补偿
```

### 5.2 信用利差的影响因素

| 因素 | 作用机制 |
|------|----------|
| 违约概率 PD | 越高，利差越大 |
| 违约损失率 LGD | 越大，利差越大 |
| 流动性 | 流动性差 → 利差扩大 |
| 税收 | 利息税差异影响相对利差 |
| 风险偏好 | 风险厌恶上升 → 利差扩大 |

### 5.3 中国信用利差区间

| 评级 | 利差范围（bp，相对国开债） |
|------|----------------------------|
| AAA | 30-80 |
| AA+ | 60-150 |
| AA | 100-300 |
| AA- | 200-500+ |
| A+ 及以下 | 500-2000+（视行业） |

## 六、通胀溢价

### 6.1 费雪方程（近似）

```
名义无风险利率 ≈ 实际无风险利率 + 预期通胀率
      Rf_nom      ≈     Rf_real    +    π^e
```

### 6.2 通胀溢价的分解

| 组成部分 | 含义 |
|----------|------|
| 预期通胀 | 市场对未来通胀的中性预期 |
| 通胀风险溢价 | 应对通胀不确定性的额外补偿 |

### 6.3 中国通胀溢价特征

| 时期 | 通胀水平 | Rf 调整 |
|------|----------|---------|
| 高通胀（07-08, 11, 22） | CPI > 3% | 需使用实际 Rf 折现 |
| 低通胀（15-20） | CPI < 2% | 名义 Rf 与实际 Rf 接近 |

## 七、流动性溢价

### 7.1 流动性维度

| 维度 | 含义 |
|------|------|
| 紧度（Tightness） | 买卖价差 |
| 深度（Depth） | 大单成交对价格影响 |
| 弹性（Resiliency） | 价格回归均衡的速度 |
| 即时性（Immediacy） | 成交速度 |

### 7.2 流动性溢价的来源

| 资产类型 | 流动性溢价 |
|----------|------------|
| 沪深 300 成分股 | 极低 |
| 中小盘股 | 中等 |
| 新三板 / 区域性股权 | 高 |
| 信用债（AA+） | 中等 |
| 私募债 / 非标 | 高 |

## 八、风险厌恶度与 ERP 估算

### 8.1 公式推导（简化）

```
ERP = E(Rm) - Rf
    = 风险厌恶度 × 风险方差
    = (1/2) × γ × σ²_m
（其中 γ 为相对风险厌恶系数，σ²_m 为市场方差）
```

### 8.2 风险厌恶度区间

| 投资者类型 | γ 值 | 隐含 ERP（σ=20%） |
|------------|------|------------------|
| 极度风险厌恶 | 6-10 | 12-20% |
| 典型股票投资者 | 3-5 | 6-10% |
| 学术标准值（Mehra-Prescott） | 2-3 | 4-6% |
| 准风险中性 | < 1 | < 2% |

### 8.3 Mehra-Prescott 股权溢价之谜（1985）

| 观察 | 历史 ERP ≈ 6% |
|------|---------------|
| 推算 | 需要 γ ≈ 30+ |
| 实际 | 实证 γ ≈ 1-3 |
| 矛盾 | 理性人应不持有股票 |

> 这是资产定价领域的核心未解之谜，至今仍是行为金融、习惯形成、灾难风险等理论的研究热点。

## 九、风险溢价的动态变化

### 9.1 影响因素

| 因素 | ERP 变化方向 |
|------|--------------|
| 经济不确定性上升 | ERP ↑ |
| 央行宽松（降息） | ERP ↑（Rf 降而 Rm 相对稳） |
| 风险偏好下降 | ERP ↑（避险情绪） |
| 经济强劲 | ERP ↓（投资者乐观） |
| 杠杆紧缩 | ERP ↑（融资成本上升） |

### 9.2 历史区间

| 时期 | 美国 ERP 范围 | 背景 |
|------|---------------|------|
| 1929 大萧条 | 极高 (>15%) | 极端不确定性 |
| 1950s 战后 | 5-7% | 战后繁荣 |
| 1980s 里根 | 4-6% | 高利率、低波动 |
| 2000 互联网泡沫 | 极低 (<3%) | 极度乐观 |
| 2008 金融危机 | 8-10% | 风险厌恶骤升 |
| 2020 疫情 | 8-12% | 极端避险 |
| 2024-2025 | 4-5% | 正常化 |
