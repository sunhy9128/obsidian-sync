# IS-LM模型

> 由 Hicks 和 Hansen 提出的凯恩斯主义总需求分析框架，描述产品市场（IS）与货币市场（LM）同时均衡时利率与收入的决定。
> 由 AI 系统化整理（2026-06-03），待补充原始来源素材

---

## 一、模型框架

### 1.1 历史背景

- **1936** 凯恩斯《就业、利息和货币通论》提出流动性偏好理论
- **1937** Hicks 在《凯恩斯先生与"古典派"》首次用 IS-LL 图示
- **1949** Hansen 改写为 IS-LM 模型
- **2000s 后** 被新凯恩斯 DSGE 模型逐渐取代，但仍是教科书核心

### 1.2 模型两大市场

| 市场 | 曲线 | 均衡条件 | 关键变量 |
|------|------|----------|----------|
| **产品市场** | IS 曲线 | 计划支出 = 总产出 | Y（收入）、C、I、G |
| **货币市场** | LM 曲线 | 货币需求 = 货币供给 | r（利率）、M、P |

### 1.3 模型假设

- 短期分析（价格粘性）
- 封闭经济（后扩展为 Mundell-Fleming 开放版本）
- 投资是利率的减函数
- 货币需求是收入的正函数、利率的负函数

## 二、IS 曲线（产品市场均衡）

### 2.1 推导起点：凯恩斯交叉

产品市场均衡条件：

```
Y = C + I + G           （封闭、无政府、无外贸）
Y = C(Y-T) + I(r) + G   （C 取决于可支配收入，I 取决于利率）
```

均衡条件：

```
Y = AD = C + I + G
```

### 2.2 储蓄-投资恒等式

```
S(Y) + T = I(r) + G
```

> 总储蓄 + 税 = 总投资 + 政府支出

### 2.3 IS 曲线方程

```
Y = C_0 + cY + I_0 - br + G
Y(1-c) = C_0 + I_0 + G - br
Y = (1/(1-c)) × (C_0 + I_0 + G) - (b/(1-c)) × r
```

设 α = 1/(1-c) 为**支出乘数**（政府支出乘数）：

```
Y = α(A) - α·b·r        其中 A = C_0 + I_0 + G
```

> IS 曲线：**r 越高 → I 越低 → Y 越低** → IS 向右下方倾斜

### 2.4 IS 曲线移动因素

| 移动方向 | 触发因素 |
|----------|----------|
| **右移** | G ↑、T ↓、投资信心 ↑、C 自主支出 ↑ |
| **左移** | G ↓、T ↑、投资信心 ↓、C 自主支出 ↓ |

## 三、LM 曲线（货币市场均衡）

### 3.1 货币需求函数

凯恩斯流动性偏好理论：

```
M^d = L(Y, r) = kY - hr     （k, h > 0）
```

| 变量 | 含义 | 与 M^d 关系 |
|------|------|-------------|
| Y | 收入 | 正相关（交易需求） |
| r | 利率 | 负相关（投机需求） |

### 3.2 货币市场均衡

```
M^s = M^d
M/P = kY - hr
```

### 3.3 LM 曲线方程

```
M/P = kY - hr
hr = kY - M/P
r = (k/h) × Y - (1/h) × (M/P)
```

> LM 曲线：**Y 越高 → 货币需求 ↑ → r 越高** → LM 向右上方倾斜

### 3.4 LM 曲线移动因素

| 移动方向 | 触发因素 |
|----------|----------|
| **右移** | 央行扩表 M ↑、P ↓（紧缩实际余额） |
| **左移** | 央行缩表 M ↓、P ↑（通胀吞噬实际余额） |

### 3.5 LM 曲线的三段

```
       r
       │    古典区（垂直）
       │    │   LM 曲线
       │    │  ╱
       │    │╱
       │   ╱│
       │  ╱ │   中间区
       │ ╱  │
       │╱───┤
       └────┴─── Y
       凯恩斯区（水平）
```

| 区域 | 名称 | 政策含义 |
|------|------|----------|
| **凯恩斯区**（水平段） | 流动性陷阱 | 货币政策无效，财政政策有效 |
| **中间区**（倾斜段） | 常态 | 货币+财政政策都有效 |
| **古典区**（垂直段） | 古典情形 | 货币政策有效，财政政策无效 |

## 四、均衡求解

### 4.1 联立方程

```
IS:  Y = α(A) - α·b·r
LM:  M/P = kY - hr
```

解出均衡 r* 和 Y*：

```
r* = (k·α·A - M/P) / (h + k·α·b)
Y* = (h·α·A + α·b·(M/P)) / (h + k·α·b)
```

### 4.2 均衡图示

```
         r
         │
   r*    │╲ LM
         │ ╲
         │  ╲
         │   ● 均衡点 (r*, Y*)
         │  ╱ IS
         │╱
         └────── Y
                Y*
```

## 五、政策效应分析

### 5.1 财政政策（IS 移动）

| 政策 | 移动 | 效果 |
|------|------|------|
| **扩张性财政**（G ↑, T ↓） | IS 右移 | Y ↑, r ↑（**挤出效应**） |
| **紧缩性财政**（G ↓, T ↑） | IS 左移 | Y ↓, r ↓ |

> **挤出效应**（Crowding Out）：r 上升 → 私人投资 I 减少 → 部分财政扩张被抵消。

### 5.2 货币政策（LM 移动）

| 政策 | 移动 | 效果 |
|------|------|------|
| **扩张性货币**（M ↑） | LM 右移 | Y ↑, r ↓ |
| **紧缩性货币**（M ↓） | LM 左移 | Y ↓, r ↑ |

### 5.3 政策效应对比

| 政策 | 利率 | 收入 | 适用 |
|------|------|------|------|
| 财政扩张 | ↑ | ↑↑ | 流动性陷阱时最有效 |
| 货币扩张 | ↓ | ↑↑ | 古典区时最有效 |

## 六、IS-LM 的局限

### 6.1 核心缺陷

| 局限 | 说明 |
|------|------|
| **静态模型** | 无预期，无动态调整 |
| **封闭经济** | 忽略国际资本流动与汇率 |
| **价格不变** | 短期粘性，但长期价格会调整 |
| **无微观基础** | 消费、投资函数缺乏最优化推导 |
| **货币政策外生** | 不考虑泰勒规则、央行反应函数 |
| **无金融部门** | 无信贷市场、风险溢价 |
| **财政-货币二分** | 现实中政府融资影响货币 |

### 6.2 后续扩展

| 扩展模型 | 解决的局限 |
|----------|------------|
| **Mundell-Fleming（开放 IS-LM）** | 加入汇率与资本流动 |
| **AD-AS** | 加入价格调整与长期 |
| **新凯恩斯（NK）** | 加入微观基础（粘性价格） |
| **DSGE** | 动态、随机、一般均衡 |
| **金融加速器（FA）** | 加入信贷市场摩擦 |

## 七、IS-LM 速查

### 7.1 关键公式

| 项目 | 公式 |
|------|------|
| 乘数 | α = 1/(1-c) |
| IS | Y = α(A) - α·b·r |
| LM | M/P = kY - hr |
| 均衡 r* | (k·α·A - M/P) / (h + k·α·b) |
| 均衡 Y* | (h·α·A + α·b·(M/P)) / (h + k·α·b) |

### 7.2 斜率判断

| 曲线 | 斜率 | 决定因素 |
|------|------|----------|
| IS | 右下 | b（投资对利率敏感度）、c（消费倾向） |
| LM | 右上 | k（交易需求）、h（投机需求） |

## 八、Mundell-Fleming 扩展（开放版）

### 8.1 三大市场均衡

```
IS*:  Y = C + I + G + NX(e)
LM*:  M/P = kY - hr
BP:   r = r*（资本完全流动）
```

> e = 汇率（直接标价，本币/外币）

### 8.2 政策有效性矩阵

| 资本流动 | 财政政策 | 货币政策 |
|----------|----------|----------|
| 完全流动 | 非常有效 | 无效（固定汇率）|
| 完全流动 | 无效（浮动汇率）| 非常有效 |
| 不流动 | 有效 | 有效 |

> 这是蒙代尔不可能三角的模型化体现。
