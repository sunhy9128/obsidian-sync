> 前景理论（Prospect Theory）由 Kahneman 与 Tversky（1979）提出，描述人们在不确定条件下做决策时，**以参考点为基准评估收益与损失**，而非追求期望效用最大化。

---

## 一、理论起源与发展

### 1.1 学术背景

| 维度 | 说明 |
|------|------|
| 提出者 | Daniel Kahneman & Amos Tversky |
| 发表年份 | 1979 年（《Econometrica》） |
| 核心奖项 | Kahneman 2002 年获诺贝尔经济学奖 |
| 理论地位 | 行为经济学的奠基性理论 |

### 1.2 与传统期望效用理论的对比

| 维度 | 期望效用理论（EUT） | 前景理论（PT） |
|------|------------------|--------------|
| 评估基准 | 财富绝对量 | 相对参考点的变化 |
| 效用函数 | 凹函数（风险厌恶） | S 型价值函数（损失域更陡峭） |
| 概率处理 | 客观概率 | 经过权重函数扭曲的主观概率 |
| 偏好公理 | 独立性公理、传递性 | 违反独立性公理 |
| 框架敏感性 | 框架不影响偏好 | 同一问题不同框架导致选择反转 |

## 二、四大核心发现

### 2.1 参考点依赖（Reference Dependence）

```
定义：人们评估结果时，以某个"参考点"（如现状、目标、预期）
      为基准，关心的是相对于参考点的"得与失"而非最终财富。

示例：
  - 参考点 0：当前工资 10000 元
  - 加薪 500 元 → 主观"收益 500"
  - 减薪 500 元 → 主观"损失 500"
  - 同样的 500 元在不同参考点下感觉不同
```

### 2.2 损失厌恶（Loss Aversion）

```
定义：同等金额的损失带来的负效用，大于收益带来的正效用。

经验比值：λ ≈ 2.0（损失的心理权重约为同等收益的 2 倍）
数学表达：v(-x) ≈ -2·v(x)，x > 0

示例：
  抛硬币赌局：
  - 正面：赢 100 元
  - 反面：输 100 元
  - 期望值 = 0，但大多数人拒绝参与（因损失厌恶）
```

### 2.3 敏感性递减（Diminishing Sensitivity）

```
定义：价值函数在远离参考点时边际递减（凹函数特征）。

含义：
  - 收益域：100 元 → 200 元的快乐 < 0 元 → 100 元的快乐
  - 损失域：-100 元 → -200 元的痛苦 < 0 元 → -100 元的痛苦
  - 财富越多，每增加 1 元的边际效用越小

类比：吃第一块蛋糕很香，第五块就腻了
```

### 2.4 概率权重（Probability Weighting）

```
定义：小概率被过度重视，大概率被低估。

典型权重函数（K-T 1992 累积形式）：
  π(0)   = 0
  π(1)   = 1
  π(0.01)= 0.05~0.06（5% 被放大为 5-6%）
  π(0.5) = 0.42
  π(0.9) = 0.80~0.85

表现：彩票被高估、保险被低估（除非有意外事件触发）
```

## 三、价值函数图像

```
v(x)
 ↑
 │            /
 │           /  收益域：凹函数（风险厌恶）
 │          /
 │─────────/──────── 0 (参考点)
 │        /
 │       /  损失域：凸函数（风险寻求）
 │      /
 │     /
 └────┴─────────────→ x（相对参考点的变化）
       -x
```

```
数学表达（K-T 1992 改进版）：
  v(x) = x^α ,  x ≥ 0
  v(x) = -λ·(-x)^β ,  x < 0

  其中 α = β ≈ 0.88（敏感性递减）
       λ ≈ 2.25（损失厌恶系数）
```

### 3.1 价值函数三特征

| 特征 | 位置 | 数学表现 |
|------|------|---------|
| 参考点 | 原点 | v(0) = 0 |
| 损失域更陡 | 左侧 | 斜率绝对值更大 |
| 敏感性递减 | 两侧 | 凹（收益）/凸（损失） |

## 四、四大偏好模式（Framing Effects）

### 4.1 确定性效应（Certainty Effect）

```
场景：疾病救治两种方案
  方案 A：确定救活 200 人
  方案 B：1/3 概率救 600 人，2/3 概率无人获救
  → 多数人选 A（确定性偏好，低估概率差异）
```

### 4.2 反射效应（Reflection Effect）

```
场景：疾病救治两种方案
  方案 A：确定死 400 人
  方案 B：1/3 概率无人死，2/3 概率 600 人全死
  → 多数人选 B（损失域内风险寻求）

关键：与 4.1 中将得/失表述反转时，选择也反转
     → 违反传统效用理论的"框架无关性"
```

### 4.3 分离效应（Isolation Effect）

```
场景：两阶段决策
  阶段 1：先赢 30 元（70% 概率）→ 大多数人接受
  阶段 2：再玩 抛硬币（赢 9 元或输 4 元，概率 50%）
  
  与一次性：
  - 70% 概率赢 30 元，接着 50/50 玩 → 等同于：
  - 35% 概率赢 39 元
  - 35% 概率赢 26 元
  - 30% 概率输 4 元

  人们分阶段评估，倾向于接受最终可能输的方案
  → 违反"复合程序公理"
```

### 4.4 隔离效应（Disjunction Effect）

```
定义：相同信息不同分隔方式会产生不同选择。

示例：
  1-球抽奖 vs 5-球抽奖：
  - 1-球抽奖：33% 概率赢 → 多数选参与
  - 5-球抽奖：平均 33% 概率但分布不同 → 参与率下降

  即"共同结果不同路径"的孤立处理导致偏好差异
```

## 五、数学形式化

### 5.1 原始前景理论（1979）

```
两阶段编辑 + 评估：

阶段 1（编辑）：将前景简化为更简单的表示
  - 编码（设参考点）
  - 组合（合并相同结果）
  - 分离（拆分风险/无风险部分）
  - 抵消（消除共同成分）
  - 简化（优势/劣势简化）

阶段 2（评估）：
  V(x, p; y, q) = π(p)·v(x) + π(q)·v(y)
  
  其中 π 为概率权重函数，v 为价值函数
```

### 5.2 累积前景理论（CPT, 1992）

```
突破：解决 1979 版对所有前景结果的概率权重需 ≤ 1 的问题
  
V(f) = Σ_i w_i^+ · v(x_i)  for gains
     + Σ_i w_i^- · v(x_i)  for losses

w_i 通过概率累积函数计算，权重函数可与概率无关
```

## 六、投资应用

### 6.1 处置效应（Disposition Effect）

| 现象 | 解释 |
|------|------|
| 投资者过早卖出盈利股 | 锁定"确定收益"（确定性效应） |
| 投资者过久持有亏损股 | 避免实现"确定的损失"（反射效应） |
| 行为后果 | 错失长期收益、形成"处置效应溢价" |

### 6.2 风险态度反转

| 情境 | 态度 | 表现 |
|------|------|------|
| 面对收益 | 风险厌恶 | 倾向落袋为安 |
| 面对损失 | 风险寻求 | 倾向赌一把回本 |
| 实战教训 | —— | 牛短熊长，熊市不止损 |

### 6.3 投资者启示

- 设定固定止损点，克服反射效应
- 避免因短期涨跌频繁查看账户
- 区分"账面浮亏"与"真实损失"，避免被框架误导
- 长期投资应更多关注基本面而非短期波动

## 七、批评与扩展

### 7.1 主要批评

| 批评者 | 主要观点 |
|--------|---------|
| Loomes & Sugden (1982) | 后悔理论可解释部分现象 |
| Starmer (2000) | 框架效应在小样本中不稳定 |
| 实证检验 | 不同文化、收入群体参数差异大 |

### 7.2 后续发展

| 理论 | 改进点 |
|------|-------|
| 累积前景理论（CPT, 1992） | 解决概率权重一致性问题 |
| 后悔理论（Regret Theory, 1982） | 引入情绪因素 |
| 排名依赖效用（RDU, 2002） | 用排名代替概率 |
| 习惯形成理论 | 引入参考点动态调整 |
