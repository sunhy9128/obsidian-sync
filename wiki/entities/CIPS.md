# CIPS（人民币跨境支付系统）

> Cross-border Interbank Payment System — 人民币国际化的"高速公路"

**CIPS**（人民币跨境支付系统，Cross-border Interbank Payment System）是中国为推进人民币国际化而建设的专门处理人民币跨境清算业务的金融基础设施，由中国人民银行于 2012 年决定建设，2015 年一期上线运行，2018 年二期全面投运。它是中国对标 SWIFT（报文）+ CHIPS（美元清算）的"二合一"跨境支付清算系统，承担人民币跨境支付的信息流与资金流双重职能。

| 项目 | 内容 |
|------|------|
| 全称 | Cross-border Interbank Payment System |

| 上线时间 | 2015 年（一期），2018 年（二期）|
| 运营主体 | 跨境银行间支付清算有限责任公司（CIPS Co., Ltd.）|

| 锚定货币 | 人民币 |
| 业务覆盖 | 跨境贸易、直接投资、跨境融资、个人汇款 |

## 二、技术架构

### 2.1 与 SWIFT + CHIPS 的本质区别

```
【SWIFT + CHIPS 模式（美元跨境支付）】
SWIFT（报文通讯）
  ├── 传递支付指令（信息流）
  └── + CHIPS（美元清算，资金流）
      └── 完整的美元跨境支付

【CIPS 模式（人民币跨境支付）】
CIPS（报文 + 清算二合一）
  ├── 既传报文（信息流）
  └── 又做清算（资金流）
      → 支付路径更短、效率更高
```

### 2.2 关键参数对比

| 维度 | CIPS | SWIFT | CHIPS |
|------|------|-------|-------|
| 成立时间 | 2015（一期）| 1973 | 1970 |
| 核心功能 | 报文 + 清算 | 报文通讯 | 美元清算 |
| 锚定货币 | 人民币 | 多币种 | 美元 |
| 日均处理量 | ~$800 亿（2024）| 4200 万+ 笔 | ~$1.5 万亿 |
| 参与机构 | ~1500 家 | 11000+ 家 | 主要美资银行 |
| 覆盖范围 | ~110 国家/地区 | 200+ 国家 | 全球美元交易 |
| 制裁风险 | 低 | 高（可被切断）| 高（可被冻结）|

### 2.3 参与者结构

```
CIPS 参与者体系
├── 直接参与者（~100 家）
│   └── 在 CIPS 开设账户，直接清算
├── 间接参与者（~1400 家）
│   └── 通过直接参与者进行清算
└── 境外参与者
    ├── "一带一路"沿线银行（占比 ~60%）
    ├── 东盟银行（占比 ~15%）
    ├── 欧洲银行（占比 ~10%）
    └── 其他地区银行（占比 ~15%）
```

### 2.4 处理流程

```
人民币跨境支付路径（CIPS 模式）
[汇款人] → 中国境内银行（或代理行）
    ↓
CIPS 系统
  ├── 报文传递（信息流）
  └── 人民币清算（资金流）
    ↓
[收款行] → [收款人]

耗时：几分钟到几小时
成本：远低于 SWIFT + CHIPS 路径
```

## 三、发展历程

### 3.1 关键里程碑

| 时间 | 里程碑事件 |
|------|-----------|
| 2012 | 央行决定建设 CIPS 系统 |
| 2015-10 | CIPS 一期上线（仅实时全额结算）|
| 2018-05 | CIPS 二期全面投运（混合结算模式 + 延长运营时间）|
| 2019 | CIPS 二期扩展业务范围，加入全额结算模式 |
| 2022 | 俄乌冲突后，CIPS 重要性凸显，扩容加速 |
| 2024 | 日均交易量达 ~$800 亿 |

### 3.2 三段式战略升级

```
Phase 1（2015-2020）：基础设施搭建期
├── 一期上线（2015）
├── 二期扩展（2018）
└── 定位："人民币跨境结算通道"（纯技术）

Phase 2（2020-2024）：备胎战略期
├── 2022 俄乌冲突后，CIPS 重要性凸显
├── 与 SWIFT 形成事实上的"双轨"
└── 定位："SWIFT 的 Plan B"（防御性）

Phase 3（2024-至今）：主动进攻期
├── 接入"一带一路"沿线国家
├── 推动人民币计价大宗商品（石油、天然气）
├── 与 mBridge、SPFS 形成非美元清算网络
└── 定位："非美元清算体系的核心节点"（进攻性）
```

## 四、战略意义

### 4.1 三大战略意图

**意图一：【备胎效应】降低制裁威慑的可信度**

```
即使美国切断 SWIFT，中国仍能通过 CIPS 完成跨境支付
  → 美国制裁工具的边际效用下降
  → 美元武器化的"成本收益比"恶化
```

**意图二：【通道效应】支撑"一带一路"贸易网络**

```
CIPS 作为人民币结算通道
  ├── 沿线贸易的人民币结算占比上升
  ├── 降低对美元/欧元的汇率风险
  └── 推动人民币在区域贸易中"先行先试"
```

**意图三：【武器化潜力】形成非对称反制能力**

```
美国制裁工具：
├── 切断 SWIFT 报文
├── 冻结美元储备
└── 长臂管辖

中国潜在反制工具：
├── 切断稀土、关键供应链
├── 人民币清算通道收紧
└── 大宗商品采购转向
    → 形成"相互确保金融摧毁"（MAD-Finance）
```

### 4.2 在人民币国际化体系中的位置

> 来自 [[人民币国际化#CIPS 的战略意义]]：

> CIPS 不是为了完全替代 SWIFT（短期不可能），而是：
> 1. **备胎**（Plan B）：如果 SWIFT 被切断，CIPS 能兜底
> 2. **接入"一带一路"**：沿线的贸易结算通道
> 3. **降低美元依赖**：中俄、中伊、中沙贸易直接人民币结算

```
人民币国际化的"三条腿"框架
├── 贸易结算（"结算货币"）
│   ├── 跨境贸易人民币结算
│   ├── 双边本币互换协议
│   └── CIPS ← 此处
├── 金融交易（"投资货币"）
│   ├── 离岸人民币市场（CNH）
│   └── 人民币债券市场（熊猫债+点心债）
└── 储备资产（"储备货币"）
    └── 各国央行持有人民币储备
```

## 五、CIPS 的结构性短板

| 短板 | 原因 | 战略影响 |
|------|------|---------|
| **美元清算仍依赖 CHIPS** | 多数国家仍需以美元为锚 | CIPS 真正的"独立性"有限 |
| **资本账户不开放** | 三元悖论约束 | 人民币难成主要储备货币 |
| **市场规模仍小** | 仅为 CHIPS 的 1/20 | 流动性差、网络效应弱 |
| **政治风险溢价** | 跨境政治敏感性 | 第三方国家的"中立选择"难题 |
| **海外央行接入困难** | 政治顾虑 + 美方施压 | 节点覆盖远不及 SWIFT |

## 六、CIPS 与替代系统的对比

### 6.1 全球主要跨境支付清算系统

| 系统 | 国家/地区 | 锚定货币 | 主要用途 | 战略意图 |
|------|----------|---------|---------|---------|
| **CIPS** | 中国 | 人民币 | 人民币跨境清算 | 人民币国际化 |
| **SPFS** | 俄罗斯 | 卢布 | 俄罗斯国内 + 周边清算 | 应对 SWIFT 制裁 |
| **INSTEX** | 欧盟 | 欧元 | 伊朗人道主义贸易 | 绕过美国次级制裁 |
| **SEPAM** | 伊朗 | 伊朗里亚尔 | 伊朗国内清算 | 国内替代 SWIFT |
| **UPI** | 印度 | 卢比 | 印度跨境支付试点 | 印度版 CIPS |

### 6.2 CIPS 与 mBridge 的协同

> 来自 [[央行数字货币#mBridge]]：

```
【mBridge（多边央行数字货币桥）】
├── 参与方：中国香港金管局、泰国央行、阿联酋央行、中国人民银行数研所
├── 状态：2024 年进入 MVP（最小可行产品）
├── 目标：近乎即时、低成本、点对点跨境支付
└── 与 CIPS 关系：
    ├── CIPS：传统银行账户的清算（账户型）
    └── mBridge：CBDC 的清算（数字型）
        → 两者构成"双轨"清算基础设施
```

CIPS 二期扩容

## 八、关键洞察

> [!key-insight] CIPS 的真正价值不在规模，而在"备胎可信度"
>
> 即使永远用不到，也让美国的制裁威慑打折扣。

> [!key-insight] CIPS 与 SWIFT 不是简单替代关系
>
> CIPS 是"清算系统"（处理人民币资金流），SWIFT 是"报文系统"（传递支付指令）。在多数情况下，CIPS 仍需通过 SWIFT 报文传递跨境支付指令，两者实际上是协同关系。

> [!contradiction] 制裁越多，替代动力越大
>
> 美国制裁工具的频繁使用 → 各国加快去美元化 → CIPS 等替代系统扩张 → 美元霸权被持续稀释。这是"金融核武器"无法频繁使用的根本原因。
