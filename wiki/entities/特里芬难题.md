# 特里芬难题

> 美元霸权的内在矛盾

## 核心定义

特里芬难题（Triffin Dilemma）由经济学家罗伯特·特里芬于1960年提出，解释了美元作为世界货币的内在悖论。

## 核心悖论

| 要求 | 矛盾点 |
|------|--------|
| **美元作为世界货币** | 需要全球央行持有美元储备 |
| **美元流出** | 需要美国保持贸易逆差 |
| **长期逆差** | 损害美元信誉 → 信心危机 |

### 简化理解

> "美元想要霸权，就必须流出；流出太多，霸权就会崩溃"

## 公式表达

$$美元信心 = \frac{黄金储备}{海外持有美元规模}$$

- 分子（黄金储备）稳定
- 分母（海外美元）持续增长
- 比值持续下降 → 信心危机

## 历史印证

| 年份 | 事件 |
|------|------|
| 1960 | 特里芬提出难题 |
| 1960s | 越战+福利膨胀 → 美元信心动摇 |
| 1971 | 尼克松宣布"关闭黄金窗口" — 违约 |
| 1974 | 石油美元接棒 — 新支撑 |
| 1971至今 | 纯信用货币霸权 |

## 为什么无解？

| 方案 | 问题 |
|------|------|
| 保持顺差 | 美元无法流出，全球无储备 |
| 保持逆差 | 信誉受损，信任危机 |
| 浮动汇率 | 汇率波动扰乱全球贸易 |

## 可能的替代？

| 货币 | 进展 |
|------|------|
| 欧元 | 曾有机会，但欧债危机削弱 |
| 人民币 | 国际化中 |
| SDR | 提款权，非流通货币 |
| 比特币 | 波动过大，非储备货币 |
