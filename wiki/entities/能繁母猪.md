# 能繁母猪

能繁母猪（Breeding Sow）是指**已经达到生殖年龄且具备正常繁殖能力的母猪**，通常指**8月龄以上、体重120kg以上、已配种或产仔的母猪**。能繁母猪存栏量是中国生猪产业的**核心产能指标**——它决定了未来10-12个月的生猪供应。

> **核心命题**：能繁母猪是猪周期的"产能心跳"——它的存栏数变化是预测未来猪价的最重要先行指标，比任何期货价格信号都更接近真相。

## 一、为什么是"能繁母猪"而不是"生猪存栏"？

| 指标 | 定义 | 用途 | 局限 |
|------|------|------|------|
| **能繁母猪存栏** | 已达繁殖年龄的母猪头数 | **预测10-12个月后的生猪供应** | 时滞较长 |
| 生猪存栏 | 所有活体生猪（含母猪、公猪、仔猪、育肥猪） | 反映当前存栏 | 波动大、易被屠宰调节 |
| 生猪出栏 | 当期屠宰上市的生猪 | 反映当期供应 | 已经形成事实 |
| 仔猪存栏 | 0-2月龄小猪 | 预测3-6个月后出栏 | 数据质量差 |

**核心公式**：

```
未来12个月生猪供应 ≈ 当前能繁母猪存栏 × PSY（每头母猪年提供断奶仔猪数）
```

因此，**能繁母猪存栏是生猪产能的"根"**——只要这个指标变化，整个10-12个月后的供应必然跟随。

## 二、关键参数

### PSY（Pigs Sold per Sow per Year）
每头母猪每年提供的断奶仔猪数，是衡量养殖效率的核心指标：

| 国家/地区 | PSY | 备注 |
|------------|-----|------|
| 丹麦 | 33+ | 全球领先 |
| 荷兰 | 30+ | 工业化养殖典范 |
| 美国 | 26-28 | 规模化养殖 |
| **中国** | **20-23** | 散户PSY低（15-18），规模化企业高（25-28） |
| 巴西 | 24-26 | 快速追赶 |

**关键洞察**：中国PSY整体水平低于发达国家，意味着同样的生猪供应需要**更多的母猪**。规模化转型是提升PSY的核心方向。

### MSY（Market Pigs per Sow per Year）
每头母猪每年提供的出栏生猪数（扣除死亡）：
- 中国平均MSY：约18-20头
- 规模企业MSY：24-26头
- 行业最佳（牧原）：27头+

### 其他参数
- 母猪利用年限：3-4年（6-8胎）
- 母猪妊娠期：114天
- 哺乳期：21-28天
- 断奶至配种：7-10天
- **总周期：约150天（5个月）**
- **从母猪决定受孕到生猪出栏：约10个月（4+6）**

## 三、能繁母猪存栏的"绿色区间"

农业农村部依据PSY=20-22的全国平均水平，划定了**约3900万头为合理存栏**：

| 区间 | 状态 | 含义 |
|------|------|------|
| **绿色区间（3700-4100万头）** | 正常 | 供需基本平衡 |
| **黄色区间（3400-3700万头）** | 偏紧 | 6-10个月后供应趋紧 |
| 黄色区间（4100-4400万头） | 偏松 | 6-10个月后供应过剩 |
| **红色区间（<3400万头或>4400万头）** | 异常 | 猪价将剧烈波动 |

### 历史关键节点

| 时间 | 能繁母猪存栏 | 状态 |
|------|--------------|------|
| 2012年 | 5068万头（历史峰值） | 严重过剩 |
| 2014年 | 4500万头 | 仍偏过剩 |
| 2018年7月 | 4500万头 | 非洲猪瘟前 |
| **2019年12月** | **2800万头** | **非洲猪瘟峰值（-38%）** |
| 2020年中 | 3600万头 | 恢复中 |
| 2021年6月 | 4564万头 | 恢复至疫情前 |
| 2022年 | 4390万头 | 略偏过剩 |
| 2023年 | 4042万头 | 黄色区间（偏紧） |
| 2024年 | 4043万头 | 接近绿色区间下限 |

## 四、能繁母猪存栏 → 猪价的传导

这是 [[蛛网模型]] 的核心公式：

```
能繁母猪存栏变化（ΔX）
   ↓ 影响未来生猪供应
   ↓ （约10-12个月传导）
   ↓ 生猪出栏变化（ΔY）
   ↓ 影响猪价（逆向）
   ↓ 猪价变化（ΔP）
   ↓ 反过来影响养殖户决策
   ↓ 能繁母猪存栏变化（下一轮）
```

### 弹性估算

根据农业农村部数据，能繁母猪存栏与10个月后猪价的**相关系数约-0.7**（强负相关）：

| 能繁母猪存栏变化 | 10个月后猪价预期变化 |
|------------------|---------------------|
| +5% | **-15% ~ -20%**（猪价弹性 > 1） |
| +10% | **-25% ~ -35%** |
| -5% | **+15% ~ +25%** |
| -10% | **+30% ~ +50%** |

> **关键发现**：能繁母猪存栏的微小变动（5-10%），能撬动猪价的剧烈波动（20-50%）。这就是蛛网模型"发散"的具体表现——产能变动被价格放大。

## 五、数据来源与监测体系

### 1. 农业农村部月度发布

**国家统计局 + 农业农村部** 每月15日左右发布上月末能繁母猪存栏数据。

**采集方式**：
- 规模养殖场（>500头）：直报系统，月度上报
- 散户：通过抽样调查推算
- 数据误差：约±2-3%

### 2. 行业协会

- **中国畜牧业协会** — 定期发布行业数据
- **博亚和讯** — 商业数据机构，发布高频数据
- **涌益咨询、Mysteel、芝华数据** — 商业数据源

### 3. 第三方监测

- 国家生猪市场（重庆）— 实时交易数据
- 大商所生猪期货 — 价格发现

### 4. 历史回溯

国家自2008年起系统统计能繁母猪数据，已积累15年+的连续数据序列，是研究猪周期的核心数据源。

## 六、能繁母猪作为政策工具

### 1. 预警机制

农业农村部定期发布"能繁母猪存栏预警"，引导养殖户：

- **绿色**：合理扩产
- **黄色（偏紧）**：适度补栏
- **黄色（偏松）**：控制补栏
- **红色**：政策强力干预

### 2. 母猪补贴

历史上多次实施"能繁母猪补贴"：
- 2007-2008年：每头母猪补贴100元（猪价高时鼓励补栏）
- 2010年：每头母猪补贴50元
- 2014-2015年：每头母猪补贴100元（猪价低时保护产能）
- 2019-2020年：非洲猪瘟后，补贴大幅提升至800元/头

### 3. 淘汰落后产能

2016-2018年环保禁养期间：
- 南方水网地区禁养区关停
- 落后小散户淘汰
- 推动PSY提升和规模化

## 七、能繁母猪数据的局限性

### 1. 数据精度
- 散户数据通过抽样推算，误差较大
- 部分养殖户**虚报**以获取补贴（虚高）
- 部分养殖户**瞒报**以避税（虚低）

### 2. PSY变化
- 同样数量的母猪，PSY不同时产出不同
- 规模化转型期（2018-2024），中国整体PSY从约18提升至22+
- 这意味着同样的能繁母猪存栏，实际产能可能增加20%+

### 3. 生产效率波动
- 非洲猪瘟、饲料成本、疾病都会影响 PSY
- PSY 不是一个常数——2019年ASF时期PSY降至15以下

### 4. 时滞长
- 能繁母猪存栏 → 猪价的传导需要10-12个月
- 在此期间任何冲击都可能改变预测

## 八、核心洞察

> 1. **能繁母猪存栏是猪周期的"先行指标"**——它的变化10个月后必然反映在猪价上
> 2. **3900万头是合理存栏**——低于3500万头进入红色区间，价格将飙升；高于4400万头进入过剩区间
> 3. **数据由农业农村部月度发布**——是政策制定和投资决策的核心依据
> 4. **弹性极强**——5-10%的存栏变动能撬动20-50%的猪价波动
> 5. **PSY是核心参数**——同等存栏下，效率差异决定实际产出
