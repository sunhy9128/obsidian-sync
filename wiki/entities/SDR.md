# SDR（特别提款权）

## 核心定义

特别提款权（SDR / Special Drawing Rights）是 IMF 在 1969 年创建的国际储备资产，旨在补充成员国官方储备。SDR 既不是货币也不是对 IMF 的债权，而是**对 IMF 成员国可自由使用货币的潜在求偿权**。持有 SDR 的国家可以将其兑换为篮子中的货币（美元、欧元、人民币、日元、英镑）以满足国际收支需求。

---

## 一、SDR 的独特属性

| 维度 | SDR | 美元 | 黄金 |
|------|-----|------|------|
| **发行主体** | IMF（成员国共同决策） | 美联储 | 自然界 |
| **发行方式** | 配额分配（按成员国份额） | 货币政策 | 开采 |
| **数量限制** | 有限（需要 IMF 董事会批准） | 无限（央行可以创造） | 有限 |
| **计息** | 有利息（SDRi 利率） | 有利息 | 无 |
| **使用范围** | 仅限官方部门（央行、国际组织） | 全球通用 | 全球通用 |
| **创造目的** | 补充储备资产 | 交易媒介 | 价值储藏 |

### SDR 不是货币，而是"储备资产凭证"

```
一个持有 100 SDR 的国家 A
  ↓
如果 A 需要美元
  ↓
A 可以将 100 SDR 卖给 IMF 指定的另一个国家 B
  ↓
B 给 A 对应价值的美元（按篮子汇率）
  ↓
A 的 SDR 持有减少，B 的 SDR 持有增加
  ↓
B 获得利息（因为提供了外汇），A 支付利息（因为使用了 SDR）
```

## 二、SDR 货币篮子

### 当前篮子构成（2022 年调整，2022.08.01 生效）

| 货币 | 权重（2022） | 权重（2016） | 变化 |
|------|------------|------------|------|
| 美元 | 43.38% | 41.73% | ↑ |
| 欧元 | 29.31% | 30.93% | ↓ |
| 人民币 | **12.28%** | 10.92% | ↑ |
| 日元 | 7.59% | 8.33% | ↓ |
| 英镑 | 7.44% | 8.09% | ↓ |

### 纳入标准

一种货币要加入 SDR 篮子必须同时满足两个条件：

1. **出口门槛**：该国（或货币联盟）的商品和服务出口在全球居前五位
2. **可自由使用**：该货币在国际支付中被广泛使用，在主要外汇市场被广泛交易

### 纳入历程

| 货币 | 纳入时间 | 备注 |
|------|----------|------|
| 美元 | 1969（初始） | 初始篮子中权重最高 |
| 欧元 | 1999 | 继承德国法郎/法国法郎权重 |
| 日元 | 1969（初始） | — |
| 英镑 | 1969（初始） | — |
| **人民币** | **2016.10.01** | **首个被纳入 SDR 的新兴市场货币** |

> [!key-insight] 人民币纳入 SDR 的意义（2016）
> 人民币纳入 SDR 是人民币国际化的重要里程碑。它证明国际社会接受人民币"可自由使用"——尽管中国仍然实行资本管制。IMF 的判断是：人民币在**支付和交易**层面已经足够国际化，即使资本账户不完全开放。

## 三、SDR 的历史与重大事件

### 创建背景（1969）

```
1960s：美元信心危机 + 全球流动性不足
  ├── 特里芬难题：美元供应全球储备的需要 vs 维持黄金可兑换
  ├── 全球贸易增长快于黄金产量 → 储备增长不够
  └── 解决方案：创造一种新的"人造储备资产"——SDR
```

### 第一次分配（1970-1972）

初始分配约 93 亿 SDR。但分配规模太小，未能阻止布雷顿森林体系的崩溃。

### 第二次分配——被遗忘的几十年（1979-1981）

分配 121 亿 SDR，之后 SDR 陷入长期停滞——浮动汇率时代来临，各国不再需要 SDR。

### 第三次：全球金融危机后的复苏（2009）

| 事件 | 规模 | 意义 |
|------|------|------|
| 2009 年 G20 伦敦峰会 | 2500 亿 SDR（约 $3500 亿） | 危机后的全球流动性注入 |
| 其中：特别分配 | 215 亿 SDR | 让 2009 年后加入 IMF 的国家获得公平分配 |

2009 年是 SDR 功能的重要转折点——从"补充储备资产"转变为"危机应对工具"。

### 第四次：COVID-19 大流行（2021）

| 事件 | 规模 | 分配 |
|------|------|------|
| 2021.08 历史最大分配 | **6500 亿 SDR（约 $9500 亿）** | 全部 190 个成员国按份额分配 |

**分配的流向问题**：
```
6500亿 SDR 分配
  ├── 发达经济体：~60%（约 $5700 亿）
  │   └── 这些国家并不需要 SDR
  ├── 新兴市场：~30%
  └── 低收入国家：~10%
      └── 这些国家最需要但拿得最少
```

争议：分配按 IMF 份额（经济总量）而非需求。最需要的国家只拿了 10%。

### 解决方案：转借（Channeling / Re-channeling）

```
低收入国家急需 SDR
  ↓
富裕国家自愿将 SDR 转借给低收入国家
  ↓
通过 IMF 的信托基金（PRGT / RST）运作
  ↓
2023 年承诺：约 1000 亿 SDR 转借
```

## 四、SDR 的利率与成本

```
SDRi 利率（SDR Interest Rate）
  ├── 计算：篮子货币国债利率的加权平均
  ├── 参考利率：
  │   ├── 美元 3 月期国债收益率
  │   ├── 欧元 3 月期 EONIA/€STR
  │   ├── 人民币 3 月期国债收益率
  │   ├── 日元 3 月期贴现国库券
  │   └── 英镑 3 月期国债收益率
  └── 每周调整

谁付利息、谁收利息？
  ├── 持有 SDR 超过分配额的国家 → 收利息（因为把外汇借出去了）
  └── 持有 SDR 少于分配额的国家 → 付利息（因为使用了 SDR 额度）
```

当前（2024-2025）SDRi 利率约为 4-4.5%，跟随美元利率走势。

## 五、SDR 与人民币国际化

SDR 对人民币国际化的作用有限但正面：

| 作用 | 说明 | 局限性 |
|------|------|--------|
| **象征意义** | IMF 背书 → 提升国际信心 | 象征意义 > 实际意义 |
| **储备需求** | 各国央行需持有人民币以满足 SDR 篮子权重 | 央行持有人民币的其他动机更重要 |
| **政策信号** | 证明资本管制≠不可自由使用 | IMF 的判断标准是实际使用，不是资本账户完全开放 |
| **储备多元化** | 推动部分央行增加人民币配置 | 进展缓慢，人民币仍仅占全球储备 ~2.5% |

> 核心判断：SDR 是国际货币体系改革的重要工具，但不会取代美元。它在全球流动性危机中发挥"最后一条救生索"的作用，但在日常国际金融中的角色有限。

## 六、改革提议：SDR 的未来

### 现有提议

| 提议 | 内容 | 可行性 |
|------|------|--------|
| **定期增发** | 每 5 年固定分配 SDR，不依赖危机驱动 | 美国国会反对（担心美元地位） |
| **需求导向分配** | 从按份额分配改为按需求分配 | 富国反对（利益受损） |
| **SDR 债券市场** | 建立 SDR 计价的私人债券市场 | 流动性不足，存在障碍 |
| **SDR 作为锚定货币** | 建立以 SDR 为锚的国际货币体系 | 市场深度不够 |
| **绿色转型融资** | 用 SDR 支持气候融资 | 政治可行，规模受限 |

### 结构性限制

| 限制 | 说明 |
|------|------|
| **分配机制** | 按份额分配，富国拿大头 |
| **仅限官方部门** | SDR 不能由私人部门持有或使用 |
| **无离岸市场** | 没有 SDR 计价的银行存款市场 |
| **美国否决权** | 增发需 IMF 85% 投票权，美国占 16.5%，有实际否决权 |
| **规模不足** | 全球储备约 $12 万亿，SDR 仅 ~$1 万亿（<10%） |
