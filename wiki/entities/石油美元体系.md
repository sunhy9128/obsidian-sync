# 石油美元体系

> 美元霸权的延续

## 核心定义

石油美元体系（Petrodollar System）是指1974年以来以石油美元计价和结算为核心的全球货币体系，是美元霸权的重要支柱。

## 历史背景

| 时间 | 事件 |
|------|------|
| 1971年 | 布林顿森林体系瓦解 |
| 1973年 | 第一次石油危机 |
| 1974年 | 沙特协议，石油美元诞生 |

## 核心机制

### 1. 沙特协议（1974年）

- **内容**：石油只能用美元计价和结算
- **交换**：美国提供军事保护 + 投资渠道
- **结果**：全球石油贸易以美元为核心

### 2. 美元循环

美元循环（Dollar Recycling / Petrodollar Recycling）是指美元从美国流出、经石油输出国收入结算、最终通过金融投资与贸易渠道回流至美国金融市场的完整闭环。

```
产油国 ← 石油销售 → 获得美元
 ↓
 美元投资美债/回流美国
 ↓
 美国获得融资 ← 全球供美元
```

**关键特征：**

- **以石油为锚定商品**：美元循环的核心载体是全球石油贸易的美元计价与结算机制
- **顺差国回流机制**：石油输出国积累的美元外汇储备通过购买美债、海外直接投资等方式回流美国
- **欧美银行中介**：回流资金主要进入西方商业银行，再以贷款形式投放至石油进口国
- **支撑美元霸权**：循环体系巩固了美元作为全球主要储备货币与结算货币的地位
- **规模庞大**：自20世纪70年代起累计循环规模达数万亿美元
- **高度依赖金融市场**：回流主要通过国债、机构债券及股票等资本市场渠道实现
- **支撑全球收支平衡**：缓解油价波动引发的国际收支失衡压力
- **与贸易失衡共生**：美国通过经常账户逆差输出美元，又通过资本账户顺差回收美元

### 3. 石油美元环流

| 环节 | 参与者 | 行为 |
|------|--------|------|
| 石油出口 | OPEC国家 | 获得美元 |
| 外汇储备 | 央行 | 投资美债 |
| 美元回流 | 美国 | 获得融资 |
| 军事保护 | 美国 | 提供安全 |

## 应用场景

美元循环概念常用于分析以下议题：

- 美国贸易逆差的可持续性
- 全球外汇储备的币种结构演变
- 石油美元体系对国际金融格局的长期影响
- 主要经济体（如中国8·11汇改）的汇率政策调整背景
- 主要储备持有国（如中国央行）的资产配置战略选择

**历史案例**：在20世纪70至80年代，OPEC国家因油价上涨积累巨额盈余，西方银行将这部分美元存款贷给拉美等发展中国家，间接支持了这些国家的工业化进程。该机制同时也是美国国债市场重要的境外资金来源，对全球美元基础货币供给具有结构性影响。

## 战略意义

| 对美国 | 对产油国 |
|--------|----------|
| 维持美元需求 | 美元储备安全 |
| 低成本融资 | 投资渠道 |
| 制裁武器 | 政治保护 |

## 与美元霸权

- **支柱地位**：石油是全球最大宗商品
- **定价权**：原油以美元计价
- **需求刚性**：无法回避美元
