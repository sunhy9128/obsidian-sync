# 乔治·索罗斯（George Soros）

> 击垮英格兰银行、做空泰铢、挑战香港金管局 —— 20 世纪最著名的对冲基金经理

## 一、人物概要

**乔治·索罗斯**（George Soros，1930 年 8 月 12 日 - ）是匈牙利出生的美国**金融家和慈善家**，是**量子基金**（Quantum Fund）的创始人，被誉为"**击垮英格兰银行的人**"。他以**宏观对冲**和**货币投机**闻名于世，是 20 世纪**最有影响力的金融人物之一**。

关键信息：
- **出生**：1930 年 8 月 12 日，匈牙利布达佩斯
- **国籍**：美国
- **职业**：金融家、慈善家
- **代表基金**：量子基金（1973-2011）
- **代表案例**：
  - 1992 年：击垮英格兰银行
  - 1997 年：做空泰铢，触发亚洲金融危机
  - 1998 年：做空港币，被香港金管局击败
- **慈善**：开放社会基金会（Open Society Foundations）

## 二、生平简介

### 2.1 早期经历

```

  - 二战期间幸存（家人多遇难）

1947-1956 年：

```

## 三、核心投资策略

### 3.1 反射性理论（Reflexivity）

```
索罗斯的核心哲学：
  金融市场不是"有效市场"
  而是"反射性"的
  
  含义：
  - 参与者的认知影响市场
  - 市场变化影响参与者
  - 形成"自我强化"的循环
       ↓
两个方向：
  - 自我强化（self-reinforcing）
  - 自我消亡（self-defeating）
       ↓
索罗斯的洞察：
  找到"反射性"的转折点
  在自我消亡之前撤退
  这是他"做空"成功的秘诀
```

### 3.2 宏观对冲（Macro Hedge）

```
索罗斯的"宏观对冲"策略：

1. 选择"有缺陷"的市场
   - 固定汇率制度（联系汇率、货币局）
   - 高估的资产价格
   - 过度依赖外资
   - 政治经济不稳定

2. 大规模做空
   - 借入货币 → 抛售
   - 借入股票 → 抛售
   - 做空期货
   - 立体攻击

3. 触发市场恐慌
   - 大量抛售推低价格
   - 市场恐慌
   - 跟风抛售
   - 价格暴跌

4. 在底部平仓
   - 价格超跌后买入平仓
   - 获得巨额利润
```

### 3.3 "动量交易"与"逆向交易"

```
索罗斯的两种交易：

1. 动量交易（Momentum）
   - 顺势而为
   - 推动趋势
   - 在趋势反转前撤退
   - 用于"自我强化"阶段

2. 逆向交易（Contrarian）
   - 寻找"反身性"的转折点
   - 在自我消亡之前做反向
   - 用于"自我消亡"阶段
       ↓
索罗斯的智慧：
  - 不僵化于"动量"或"逆向"
  - 取决于市场处于"反射性"的哪个阶段
  - 这是他"反射性理论"的实践
```

## 四、索罗斯的"三大经典战例"

### 4.1 战例一：1992 年击垮英格兰银行

```
背景：
  - 1990 年英国加入欧洲汇率机制（ERM）
  - 英镑汇率被锁定
  - 德国统一后马克强势
  - 英镑被高估
       ↓
1992 年 9 月 16 日（"黑色星期三"）：
  - 索罗斯大量做空英镑
  - 借入英镑 → 抛售
  - 规模：约 100 亿美元
       ↓
结果：
  - 英格兰银行干预失败
  - 英国宣布退出 ERM
  - 英镑贬值 15%+
  - 索罗斯获利 10 亿美元+
  - 被称为"击垮英格兰银行的人"
       ↓
意义：
  - 单一货币投机可击败央行
  - 固定汇率制度的脆弱
  - 索罗斯的"封神之战"
```

详见 [[wiki/concepts/1992欧洲货币危机]]（如存在）。

### 4.2 战例二：1997 年做空泰铢

```
背景：
  - 泰国长期实行固定汇率
  - 经济过热
  - 房地产泡沫
  - 大量短期外债
  - 国际收支恶化
       ↓
1997 年：
  - 索罗斯大举做空泰铢
  - 借入泰铢 → 抛售
  - 做空泰国股市
       ↓
1997 年 7 月 2 日：
  - 泰国宣布放弃固定汇率
  - 泰铢大幅贬值
  - 引发亚洲金融危机
       ↓
结果：
  - 泰国损失惨重
  - 危机蔓延至东南亚
  - 韩国、日本也受影响
  - 索罗斯获利巨大
       ↓
意义：
  - 固定汇率的"脆弱性"
  - 亚洲经济结构的"问题"
  - 国际资本流动的"危险"
```

### 4.3 战例三：1998 年做空港币

```
背景：
  - 1997 年亚洲金融危机蔓延
  - 国际空头集中攻击港币
  - 联系汇率制度受到威胁
       ↓
1998 年：
  - 索罗斯参与做空港币
  - 大量抛空港币
  - 同时做空港股、恒指期货
       ↓
香港金管局的应对：
  - 8 月 14 日：入市买入
  - 动用 1180 亿港元外汇基金
  - 8 月 28 日：决战日
       ↓
结果：
  - 国际空头失败
  - 索罗斯认亏离场
  - 港币联系汇率制度得以维持
  - 索罗斯"罕见"的失败
       ↓
意义：
  - 央行入市可击败投机
  - 联系汇率制度可以维持
  - 索罗斯的"挫败"
```

详见 [[wiki/concepts/1998香港金融保卫战]]。

## 五、索罗斯的"投资哲学"

### 5.1 三大思想

```
1. 反射性（Reflexivity）
   - 市场不是有效的
   - 认知和现实相互影响
   - 寻找转折点

2. 错误不可避免（Mistakes are Inevitable）
   - 市场参与者必然犯错
   - 错误是机会
   - 关键是发现错误

3. 金融市场是"实验"
   - 不预测，而是试错
   - 根据反馈调整
   - 动态决策
```

### 5.2 "金融大鳄"的争议

```
支持者观点：
  - 索罗斯揭示了"市场失灵"
  - 推动了金融市场的改革
  - 帮助发现"金融泡沫"
  - "做空"是合理的风险管理

反对者观点：
  - 索罗斯加剧了金融危机
  - 损害了发展中国家的经济
  - 投机行为破坏社会稳定
  - "金融大鳄"不顾民生
       ↓
索罗斯的回应：
  - "我做空是因为市场有问题"
  - "我只是揭示了真相"
  - "我对穷人更慷慨"
  - 通过慈善回馈社会
```

## 十五、一句话总结

> **乔治·索罗斯 = 量子基金创始人 + 击垮英格兰银行 + 1997 亚洲金融危机 + 1998 香港挫败 + 320 亿美元慈善 + 开放社会哲学**。
>
> 他是 20 世纪**最有影响力的金融人物之一**——既是**金融大鳄**（投机争议），也是**慈善家**（开放社会）。他的**反射性理论**揭示了金融市场的**非有效性**，他的**宏观对冲策略**展示了**单一货币投机击败央行**的可能。
>
> **索罗斯的传奇**是金融史的重要组成部分。**1992 年**他"击垮英格兰银行"，**1997 年**他"做空泰铢"引发亚洲危机，**1998 年**他"被香港金管局击败"——这三次经历揭示了**金融市场的复杂性**和**央行入市的力量**。**索罗斯的争议与成就并存**——这正是金融市场的真实写照。
