# 第一次石油危机

## Basic Information
- Type: event
- Source: [[石油美元体系]]

## Description
第一次石油危机发生于1973年，是[[石油美元体系]]建立的关键历史背景。[[OPEC]]成员国在赎罪日战争（Yom Kippur War）期间对支持以色列的国家实施石油禁运，导致全球[[原油价格]]在短时间内暴涨约四倍。这一事件引发了西方国家的严重经济衰退，直接暴露了西方国家对石油供应的脆弱性。在此背景下，美国寻求通过将美元与石油绑定来维护其[[美元霸权]]地位和国际影响力，为1974年[[沙特协议]]的签订创造了紧迫的政治条件。同时，这一危机也加速了[[布雷顿森林体系瓦解]]后美元锚定物的转换进程。
