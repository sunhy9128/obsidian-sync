# EU 理事会（Council of the European Union）

> **欧盟的核心决策机构**——成员国政府的代表，欧盟立法的"主战场"

---

## 一、机构概况

### 1.1 基本信息

| 维度 | 内容 |
|------|------|
| **中文全称** | 欧盟理事会（部长理事会） |
| **英文全称** | Council of the European Union |
| **简称** | EU Council / Council |
| **总部** | 比利时布鲁塞尔（Brussels） |
| **法律基础** | 《里斯本条约》（2009） |
| **性质** | 欧盟主要决策机构之一 |
| **主席** | 轮值主席国（每 6 个月轮换） |
| **常任主席** | ✗（无，与欧洲理事会不同） |
| **员工** | 约 3500 人（含秘书处） |

### 1.2 与"欧洲理事会"（European Council）的区别

```
⭐ 两个容易被混淆的机构：

【1】EU 理事会（Council of the European Union）
  - 也称"部长理事会"
  - 由各国部长组成
  - 主席：轮值
  - 立法主要决策者

【2】欧洲理事会（European Council）
  - 由各国首脑组成（国家元首 + 政府首脑）
  - 主席：常任（Charles Michel）
  - 设定欧盟整体政治方向
       ↓
⭐ 简言之：
  - EU 理事会 = 部长级决策
  - 欧洲理事会 = 首脑级决策
```

## 二、EU 理事会的构成

### 2.1 10 种配置形式

```
EU 理事会有 10 种不同的配置（按议题分组）：

1️⃣ **农业与渔业理事会（AGRIFISH）**
2️⃣ **竞争力理事会（COMPET）**
3️⃣ **经济与金融理事会（ECOFIN）**⭐ 与 ECB 关系最密切
4️⃣ **教育、青年、文化与体育理事会（EYCS）**
5️⃣ **就业、社会政策、卫生与消费者事务理事会（EPSCO）**
6️⃣ **环境理事会（ENV）**
7️⃣ **外交事务理事会（FAC）**
8️⃣ **司法与内政理事会（JHA）**
9️⃣ **总务理事会（GAC）**
🔟 **交通、电信与能源理事会（TTE）**
       ↓
⭐ 每月召开约 4 次理事会会议
   每次由相关部长参加
```

### 2.2 ECOFIN 与欧元区集团（Eurogroup）

```
⭐ ECOFIN 理事会是最重要的配置：

【ECOFIN（Economic and Financial Affairs Council）】
  - 由 27 国财长组成
  - 每月一次
  - 议题：经济政策、税收、金融监管、欧盟预算
  - ⭐ 与 ECB 关系最密切

【Eurogroup（欧元区集团）】
  - 欧元区 20 国财长
  - 每月在 ECOFIN 前召开
  - 议题：欧元区经济政策、ESM、ECB 政策
  - 主席：欧元集团主席（当前 Paschal Donohoe 爱尔兰财长）
  - ⭐ "欧元区财长俱乐部"
```

## 三、EU 理事会的核心职能

### 3.1 五大职能

```
【1】立法权**
  - 与欧洲议会共同立法
  - 普通立法程序（Ordinary Legislative Procedure）
  - ⭐ EU 立法的主要决策者

【2】预算权**
  - 与欧洲议会共同决定欧盟年度预算
  - 多年度财政框架（MFF）
  - ⭐ 欧盟预算控制

【3】经济政策协调**
  - European Semester（欧洲学期）
  - 国别建议（Country-Specific Recommendations）
  - ⭐ 欧元区经济治理

【4】国际协定**
  - 批准国际条约
  - 与第三国的协议
  - ⭐ 欧盟外交决策

【5】机构人事任命**
  - 任命 ECB 执行委员会成员
  - 任命欧洲法院法官
  - ⭐ 欧盟治理关键
```

### 3.2 与 ECB 的关系

```
EU 理事会 vs ECB：

【法律关系】
  - ECB 行长、副行长、执行委员由 EU 理事会任命
  - EU 理事会经欧洲议会同意
  - ⭐ EU 理事会决定 ECB 高层人事

【协调关系】
  - EU 理事会主席出席 ECB 理事会会议（无投票权）
  - ECB 行长出席 EU 理事会会议
  - ⭐ "相互合作但独立"

【欧元区治理】
  - Eurogroup 协调欧元区经济政策
  - 与 ECB 货币政策相互配合
  - ⭐ "欧元区双层治理"

【监督关系】
  - EU 理事会有权监督 ECB
  - 但不能干预 ECB 货币政策
  - ⭐ "独立但不孤立"
```

## 四、EU 理事会的决策机制

### 4.1 投票规则

```
⭐ EU 理事会的三种投票方式：

【1】合格多数（Qualified Majority Voting, QMV）】
  - 自 2024 起：55% 国家 + 65% 人口
  - 至少 15 个国家（占多数）
  - ⭐ 最常见的决策方式

【2】一致同意（Unanimity）】
  - 所有国家同意
  - 用于：外交、防务、税收、宪法
  - ⭐ 关键议题需一致同意

【3】简单多数（Simple Majority）】
  - 多数国家同意
  - 用于：程序性事项
  - ⭐ 较少使用
```

### 4.2 轮值主席

```
EU 理事会主席（轮值）：

【2024 上半年】比利时
【2024 下半年】匈牙利
【2025 上半年】波兰
【2025 下半年】丹麦
【2026 上半年】塞浦路斯
【2026 下半年】爱尔兰
       ↓
⭐ 轮值主席职能：
  - 主持理事会会议
  - 设定议程
  - 代表理事会发言
  - 推动妥协
  - ⭐ 主席国可推动"政治议程"
```

## 五、EU 理事会与 ECB 的合作

### 5.1 欧债危机中的协调

```
⭐ 2010-2015 欧债危机中的 EU 理事会：

【2010.5 希腊救助】
  - EU 理事会批准救助方案
  - 7500 亿欧元（EFSF + EFSM + IMF）
  - ⭐ "EU 政治意愿 + ECB 流动性"

【2012.6 经济增长方案】
  - EU 理事会 + Eurogroup 联合行动
  - ESM 与银行业联盟路线图
  - ⭐ "EU 政治 + ESM 财政 + ECB 货币"

【2012.7.26 Draghi "Whatever it takes"】
  - 与 EU 理事会协调
  - 默克尔与 Hollande 共同支持
  - ⭐ "ECB 行动 + EU 政治背书"

【2015 希腊救助】
  - EU 理事会 + Eurogroup 主导
  - 860 亿欧元第三轮救助
  - ⭐ "EU 政治意愿"
```

### 5.2 欧元区经济治理

```
⭐ EU 理事会在欧元区经济治理中的角色：

【机制 1：欧洲学期（European Semester）】
  - 每年 11 月启动
  - 各国提交稳定计划
  - EU 理事会 + 欧盟委员会评估
  - 4-7 月发布国别建议（CSRs）
  - ⭐ "欧元区经济体检"

【机制 2：欧洲稳定机制（ESM）】
  - ESM 由 Eurogroup 监督
  - EU 理事会通过 ESM 改革
  - ⭐ ESM 是 EU 理事会 + Eurogroup 的"工具"

【机制 3：银行业联盟（SSM/SRM）】
  - ECB 主导 SSM（监管）
  - EU 理事会参与治理
  - ⭐ "ECB 操作 + EU 政治监督"

【机制 4：复苏基金（NextGenerationEU）】
  - 2020 COVID 应对
  - 7500 亿欧元（EU 历史上首次共同债务）
  - 由 EU 理事会 + 欧洲议会通过
  - ⭐ "EU 财政联盟的突破"
```

## 八、EU 理事会的争议

### 8.1 主要争议

```
【争议 1：东西欧分歧】
  - 西欧（德国、法国）：财政纪律优先
  - 东欧（波兰、匈牙利）：转移支付优先
  - ⭐ 内部矛盾加剧

【争议 2：南北欧分歧】
  - 北欧（德国、荷兰）：紧缩
  - 南欧（意大利、希腊）：宽松
  - 欧债危机时期加剧
  - ⭐ 欧元区治理难题

【争议 3：一致同意 vs 效率】
  - 关键议题需一致同意
  - 决策效率低
  - ⭐ "民主"vs"效率"

【争议 4：民主赤字】
  - EU 理事会代表政府
  - 但欧盟公民没有直接选举
  - ⭐ "合法但不民主"
```

## 十一、一句话总结

> **EU 理事会 = 欧盟的"部长级决策机构"——成员国政府的代表，欧元区治理的核心**。
>
> 它由 27 国部长组成，每月召开不同配置的会议。其中**ECOFIN（经济与金融理事会）**和**Eurogroup（欧元区财长会议）**与 ECB 关系最密切。
>
> **理解 EU 理事会 = 理解四件事**：
> 1. **决策机制**：合格多数、一致同意、简单多数
> 2. **欧元区治理**：与 ECB、ESM 的协调
> 3. **欧债危机中的角色**：政治意愿 + ECB 行动
> 4. **内部矛盾**：东西欧、南北欧分歧
>
> EU 理事会是欧元区治理的"政治端"，**ECB 是"货币端"**。理解欧元区 = 理解这两个机构的协同。
