# 可转债

> 兼具债性 + 股性的混合金融工具，中国 A 股市场最重要的博弈工具之一

**可转债**（Convertible Bond）是上市公司发行的、在特定条件下可以转换为公司股票的债券。持有期内它表现为债券（保本付息），一旦触发转股条件，持有人可以选择将其转换为股票，分享股价上涨收益。

---

## 一、基本要素

### 1.1 核心条款

| 条款 | 说明 |
|------|------|
| **面值** | 均为 100 元（固定） |
| **票面利率** | 极低（0.2-2%/年），逐年递增设计 |
| **期限** | 通常 5-6 年 |
| **转股价** | 可转换为股票的每股价格 |
| **转股期** | 发行 6 个月后进入转股期 |
| **赎回条款** | 股价连续 X 天 ≥ 转股价 130% → 发行人有权按面值赎回 |
| **回售条款** | 股价连续 X 天 ≤ 转股价 70% → 持有人有权按面值回售给发行人 |
| **下修条款** | 股价持续低迷 → 发行人可主动下调转股价 |

### 1.2 转股价值的计算

```
转股价值 = (正股股价 / 转股价) × 100

例：正股 10 元，转股价 8 元
→ 转股价值 = (10 / 8) × 100 = 125 元
→ 每张可转债（面值 100）可换 12.5 股，理论价值 125 元
```

**可转债价格 vs 转股价值的关系**：

| 情况 | 可转债价格 vs 转股价值 | 含义 |
|------|----------------------|------|
| **股性强**（深度价内） | 可转债价格 ≈ 转股价值 + 溢价 | 主要反映正股上涨预期 |
| **债底支撑**（深度价外） | 可转债价格 ≈ 债底价值 + 期权价值 | 债底保护，下跌有限 |
| **平价附近** | 可转债价格 ≈ 转股价值 | 股债性平衡 |

### 1.3 债底价值（纯债价值）

当可转债被持有至到期且不转股，其价值等于未来票息 + 到期赎回的折现值：

```
债底价值 = Σ [票面利息 / (1+r)^t] + 100 / (1+r)^n

其中 r = 同评级企业债收益率
```

**债底保护**：可转债价格通常不会跌到债底以下（极端情况下存在短暂折价机会）。

## 二、可转债的四大条款：博弈的精髓

### 2.1 赎回条款（发行人的保护）

| 触发条件 | 效果 | 市场含义 |
|----------|------|----------|
| 正股连续 15/20 个交易日 ≥ 转股价 × 130% | 发行人有权按面值（100 元）强制赎回 | **核心博弈点**：持有人必须在强赎前转股或卖出，否则被低价赎回 |

> **理解**：130% 被称为"天花板"——股价涨到这个位置后，发行人有权强制赎回，持有人必须行动。这使得可转债在接近 130 元时成为"击鼓传花"博弈。

### 2.2 回售条款（持有人的保护）

| 触发条件 | 效果 | 市场含义 |
|----------|------|----------|
| 正股连续 15/20 个交易日 ≤ 转股价 × 70% | 持有人有权按面值（100 元）+ 应计利息回售给发行人 | 持有人的"安全网"——跌到债底附近时可用回售权保本 |

> **理解**：回售条款是持有人对付正股暴跌的最后防线，也是倒逼发行人"下修转股价"的机制。

### 2.3 下修条款（发行人的主动工具）

| 触发条件 | 效果 | 市场含义 |
|----------|------|----------|
| 正股在转股期持续低于转股价（无硬性标准，各公司自定） | 发行人可主动将转股价向下调整 | **最重要的博弈工具**：下修后转股价降低，转股价值提升，可转债价格上涨 |

**下修的博弈逻辑**：

```
股价暴跌 → 可转债价格跌至 90-100 元（债底附近）
         → 发行人若不下修，回售压力巨大（需掏出真金白银）
         → 发行人主动下修转股价 → 转股价值提升 → 可转债价格跳升
         → 持有人获利，发行人避免回售 → 皆大欢喜
```

### 2.4 转股价的确定与调整

| 类型 | 说明 |
|------|------|
| **初始转股价** | 不低于募集说明书公告前 20 个交易日均价和前 1 交易日均价 |
| **向下调整** | 因分红、送股、配股等导致的被动下修 |
| **主动下修** | 因正股持续低迷，发行人主动大幅下调转股价（博弈核心） |

## 三、可转债的定价框架

### 3.1 价值拆解

```
可转债价值 = 纯债价值 + 转股期权价值 + 赎回期权价值 - 回售期权价值

各成分详解：
纯债价值 = 票息现金流按同评级信用债收益率折现
转股期权 = 持有人分享正股上涨的期权（类似买入正股）
赎回期权 = 限制了正股上涨的潜在收益上限
回售期权 = 持有人拥有的保底权利
```

### 3.2 关键指标

| 指标 | 计算 | 参考含义 |
|------|------|----------|
| **转股溢价率** | (可转债价格 - 转股价值) / 转股价值 × 100% | 溢价低 = 股性强；溢价高 = 债性强 |
| **纯债溢价率** | (可转债价格 - 债底) / 债底 × 100% | 债性强弱的衡量 |
| **隐含波动率** | 期权定价模型反推的波动率 | 衡量市场对正股未来波动的预期 |
| **Delta** | 可转债价格变动 / 正股价格变动 | 衡量可转债的"股性"：Delta 高 = 股性强 |

### 3.3 各状态下可转债的定价逻辑

| 状态 | 转股溢价率 | 纯债溢价率 | 主要驱动因素 |
|------|-----------|-----------|------------|
| **深度价内** | < 5% | — | 正股价格是定价核心 |
| **平价附近** | 5-20% | — | 正股 + 溢价率博弈 |
| **轻度价内** | 20-50% | — | 正股 + 转股预期 |
| **价外（债底附近）** | 极高 | < 10% | 债底 + 下修预期 |
| **深度价外** | 极高 | 10-30% | 下修博弈（主要机会区间） |

## 四、A 股可转债市场特征

### 4.1 规模与结构

| 指标 | 数据（2024-2026 参考） |
|------|----------------------|
| 存续可转债数量 | ~500 只 |
| 存续规模 | ~8000 亿元 |
| 发行人类型 | 约 70% 民企、30% 国企/央企 |
| 平均转股溢价率 | 30-50%（整体偏溢价） |
| 平均债项评级 | AA 为主 |

### 4.2 投资者结构

| 投资者类型 | 占比 | 行为特征 |
|-----------|------|----------|
| **自然人（散户）** | ~50%（持有市值） | 主要持有低价可转债（< 105 元），博弈下修 |
| **公募基金** | ~30% | 持有偏平衡型可转债，配置为主 |
| **保险/私募** | ~10% | 持有低价/债性可转债，追求绝对回报 |
| **量化/游资** | ~10% | 持有低溢价可转债，T+0 日内交易 |

### 4.3 历史上的可转债行情

| 时间 | 行情特征 |
|------|----------|
| **2017-2018** | 信用环境收紧 + 大量可转债上市，低价可转债遍地（80-90 元），随后 2019 年大行情 |
| **2019-2020** | 正股大涨 + 转股潮，大量可转债涨至 130 元以上被强赎 |
| **2022** | 信用风险上升 + 正股下跌，可转债再次跌至债底附近 |
| **2024-2026** | 结构性行情：低溢价可转债跟随正股，高价可转债博弈强赎 |

### 4.4 下修博弈：最重要的策略

| 阶段 | 可转债价格 | 操作逻辑 |
|------|-----------|----------|
| **下修公告前** | 90-105 元（债底附近） | 买入博弈下修 |
| **下修公告当天** | 可能跳升 5-10 元 | 博弈正股联动 |
| **下修完成后** | 转股溢价率收窄，股性增强 | 可继续持有或转股 |

**下修博弈的风险**：
- 不是所有发行人都会下修（有些选择承担回售）
- 下修后正股继续下跌，可转债可能再次回落
- 信用风险：发行人在极端情况下可能违约

## 五、可转债策略

### 5.1 主要策略

| 策略 | 核心逻辑 | 适用场景 |
|------|----------|----------|
| **债底保护策略** | 价格跌至债底附近买入，等待反弹 | 市场整体下跌，优质可转债被错杀 |
| **下修博弈策略** | 买入深度价外可转债，等待下修 | 正股持续低迷但基本面未恶化 |
| **强赎博弈策略** | 买入低溢价 + 接近 130% 的可转债 | 强势市场中追求 130 元以上的收益 |
| **套利策略** | 转股溢价率为负时买入可转债并立即转股卖出 | 极罕见机会，通常被量化策略填平 |
| **双低策略** | 低价格（< 105）+ 低溢价（< 20%） | 兼顾安全性和弹性，量化筛选 |

### 5.2 可转债在资产配置中的意义

- **低风险投资替代**：对低风险偏好投资者，可转债是比纯股票更安全的选择
- **"下有保底、上有弹性"**：持有低价可转债 ≈ 持有打折的正股期权
- **在 A 股的特殊意义**：A 股散户占主导，可转债的 T+0 和涨跌停制度使日内交易活跃，博弈机会多
