# 净稳定资金比率（NSFR）

> 巴塞尔III流动性监管指标

## 核心定义

净稳定资金比率（Net Stable Funding Ratio，NSFR）是巴塞尔III框架下的长期流动性监管指标，要求银行保持足够的稳定资金以覆盖长期资产业务。

## 计算公式

```
NSFR = 可用稳定资金 / 所需稳定资金 × 100%
```

## 监管要求

NSFR必须≥100%，以确保银行有足够的稳定资金来源支持长期资产。
