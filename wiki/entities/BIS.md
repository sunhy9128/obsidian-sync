# BIS（Bank for International Settlements / 国际清算银行）

> **全球央行的"央行"——国际金融治理的"最古老机构"和"最隐秘机构"**

---

## 一、机构概况

### 1.1 基本信息

| 维度 | 内容 |
|------|------|
| **中文全称** | 国际清算银行 |
| **英文全称** | Bank for International Settlements |
| **简称** | BIS |
| **总部** | 瑞士巴塞尔（Basel） |
| **成立时间** | 1930 年 1 月 20 日 |
| **法律地位** | 国际组织（具有国际法人地位） |
| **员工数** | ~650 人（来自 60+ 国家） |
| **股东** | 60+ 中央银行 |
| **行长** | Agustín Carstens（墨西哥前央行行长，2024 至今） |
| **资产规模** | 约 4000 亿 SDR（约 5300 亿美元） |

### 1.2 BIS 的"双面性"

```
BIS 的"双面身份"：

【1】银行业监管的全球中心
  - 巴塞尔银行监管委员会（BCBS）所在地
  - ⭐ 巴塞尔协议 I/II/III 的诞生地
  - 金融稳定委员会（FSB）的秘书处
  - 全球系统重要性银行（G-SIB）评估

【2】央行合作的隐秘平台
  - 全球央行的"央行"（central bank of central banks）
  - 为央行提供金融服务
  - 召开"巴塞尔会议"（央行行长俱乐部）
  - ⭐ 全球最隐秘的国际金融机构
```

## 二、BIS 的历史

### 2.1 诞生背景（1930）

```
【诞生背景】
  - 1929 年大萧条
  - 德国无法支付一战赔款
  - 1930.1.20 海牙会议决定成立 BIS
  - 解决德国赔款问题
  - ⭐ "为处理一战赔款而创立"
       ↓
【早期功能】
  - 处理德国赔款的国际清算
  - ⭐ 但 1931 年德国暂停赔款
  - BIS 转向其他业务
```

### 2.2 二战时期的争议

```
【1940-1945 BIS 与纳粹】
  - 1940 BIS 总部被纳粹德国占据
  - BIS 仍维持运营
  - 协助德国获取外国资产
  - ⭐ 历史上最大的"道德污点"
       ↓
【战后的处理】
  - 1944 布雷顿森林会议讨论 BIS 去留
  - 美方主张解散
  - 欧洲央行行长集体反对
  - ⭐ BIS 得以保留
```

### 2.3 布雷顿森林体系下的 BIS

```
【1944-1971 布雷顿森林体系】
  - IMF 和世界银行成立
  - BIS 在布雷顿森林体系外运作
  - 成为欧洲央行的合作平台
  - 1961 年起 BIS 每月召开"巴塞尔会议"
  - ⭐ "央行行长俱乐部"成型
```

### 2.4 现代 BIS（1971-至今）

```
【1971-1988：协调期】
  - 布雷顿森林体系崩溃
  - 各国转向浮动汇率
  - BIS 成为央行协调平台

【1988-至今：监管标准化】
  - 1988 巴塞尔协议 I（资本充足率 8%）
  - 2004 巴塞尔协议 II
  - 2010 巴塞尔协议 III
  - ⭐ BIS 成为全球银行监管标准制定者

【2008 后的 BIS】
  - 2008 危机后，BIS 推动金融监管改革
  - 2010 BCBS 发布巴塞尔 III
  - 2013 FSB 在 BIS 设立
  - ⭐ BIS 在全球金融稳定中扮演更重要角色
```

## 三、BIS 的核心职能

### 3.1 五大职能

```
【1】银行业监管标准制定
  - BCBS 秘书处
  - 巴塞尔协议 I/II/III
  - 全球银行监管标准

【2】央行金融服务
  - 接受央行存款
  - 提供短期流动性
  - ⭐ 全球央行的"最后贷款人"

【3】研究与合作
  - BIS Quarterly Review（季度报告）
  - BIS Working Papers（工作论文）
  - BIS Annual Economic Report（年度报告）
  - ⭐ 全球最重要的中央银行研究机构

【4】金融稳定协调
  - FSB 秘书处
  - G20 金融监管协调
  - 系统重要性金融机构（SIFI）评估
  - ⭐ 全球金融稳定的"协调者"

【5】国际会议平台
  - "巴塞尔会议"（央行行长俱乐部）
  - 每月召开
  - 全球央行行长聚会
  - ⭐ "全球央行行长俱乐部"
```

### 3.2 BIS 的研究产品

```
BIS 的研究产品：

【1】BIS Annual Economic Report

【2】BIS Quarterly Review

【4】CGFS Papers

  - 银行监管标准
```

## 四、巴塞尔协议系列

### 4.1 巴塞尔协议 I（1988）

```
【背景】
  - 1980s 拉美债务危机
  - 国际银行业风险扩大
  - 主要银行资本充足率不足
  - 1988 BCBS 发布巴塞尔协议 I

【核心要求】
  - 资本充足率：8%（总资本/风险加权资产）
  - 资本定义：核心资本（一级）+ 附属资本（二级）
  - 风险加权：信用风险

⭐ 意义：
  - 第一个全球银行监管标准
  - 推动银行业稳健经营
  - 但只覆盖信用风险
```

### 4.2 巴塞尔协议 II（2004）

```
【背景】
  - 1990s 金融创新（衍生品、ABS）
  - 巴塞尔协议 I 不足
  - 2004 BCBS 发布巴塞尔协议 II

【三大支柱】
  - 支柱 1：最低资本要求
  - 支柱 2：监管审查
  - 支柱 3：市场约束

【新增】
  - 操作风险
  - 市场风险
  - 信用风险细化（内部评级法）
  - ⭐ 全面覆盖银行风险

⭐ 实施：2006-2008 全球实施
   但 2008 危机暴露其不足
```

### 4.3 巴塞尔协议 III（2010）

```
【背景】
  - 2008 全球金融危机
  - 巴塞尔协议 II 不足
  - 2010 BCBS 发布巴塞尔协议 III

【核心要求】

1️⃣ **资本充足率提高**
  - 一级资本（普通股）：4.5%
  - 一级资本：6%
  - 总资本：8%
  - 资本留存缓冲：2.5%

2️⃣ **杠杆率**
  - 一级资本 / 总资产 ≥ 3%
  - ⭐ 新增要求

3️⃣ **流动性要求**
  - LCR（流动性覆盖率）≥ 100%
  - NSFR（净稳定融资比率）≥ 100%
  - ⭐ 应对 2008 危机教训

4️⃣ **逆周期资本缓冲**
  - 0-2.5%
  - 视信贷周期而定

5️⃣ **系统重要性银行（SIFI）**
  - 1-3.5% 额外资本要求
  - ⭐ "大而不倒"应对
       ↓
⭐ 实施时间：2013-2023
   2023.1 完全生效
   2025.1 完成最终方案（Basel III Endgame）
```

### 4.4 巴塞尔协议 III 最终方案（2023-2025）

```
【Basel III Endgame】（2023.7）

【核心变化】
  - 风险加权资产（RWA）计算的标准化
  - 限制内部模型的使用
  - 提高资本要求的"标准化底线"
  - ⭐ 银行业最严格的监管标准

【实施时间】
  - 美国：2025.7 实施（推迟）
  - 欧盟：2025.1 实施
  - 英国：2025.1 实施（推迟）
  - 中国：2024.1 部分实施
```

## 五、BIS 与全球金融稳定

### 5.1 FSB 的诞生

```
【金融稳定委员会（FSB）】
  - 2009.4 G20 伦敦峰会决定成立
  - 替代原"金融稳定论坛"（FSF）
  - 总部设在瑞士巴塞尔（BIS 内）
  - ⭐ 协调全球金融监管
       ↓
【FSB 的核心任务】
  - 协调各国金融监管
  - 制定全球金融标准
  - 监测全球金融风险
  - 推动 G20 金融监管改革
       ↓
【FSB 主席】
  - 当前：Randal Quarles（前美联储副主席）
  - ⭐ 通常由央行行长或监管者担任
```

### 5.2 系统重要性银行（G-SIB）

```
【G-SIB 评估】

BCBS 每年评估 G-SIB：
  - 全球最系统重要性的 ~30 家银行
  - 需满足 1-3.5% 额外资本要求
  - 2024 G-SIB 列表：
    * 摩根大通（美国）
    * 美国银行
    * 花旗集团
    * 汇丰控股
    * 巴克莱（英国）
    * 德意志银行
    * 法国巴黎银行
    * 中国工商银行
    * 中国建设银行
    * 中国银行
    * ... 等 30 家
       ↓
⭐ 中国的 G-SIB：
  - 工行、建行、农行、中行
  - 2024 中国 G-SIB 共 4 家
  - 反映中国银行业的国际地位
```

## 六、BIS 的争议

### 6.1 主要争议

```
【争议 1：透明度不足】
  - BIS 被称为"最隐秘的国际组织"
  - 巴塞尔会议（央行行长俱乐部）讨论内容不公开
  - ⭐ 缺乏公众问责

【争议 2：与纳粹的关联】
  - 二战期间协助德国
  - 历史上最大的"道德污点"
  - ⭐ 历史负担

【争议 3：是否过于倾向西方】
  - 创始成员主要是西方央行
  - 中国、印度等新兴市场参与度有限
  - ⭐ 治理结构失衡

【争议 4：巴塞尔协议的过度监管】
  - 银行抱怨"合规成本过高"
  - 资本要求过严可能限制信贷
  - ⭐ "过度监管"争议
```

### 6.2 BIS 的"软实力"

```
⭐ BIS 的真正影响力：

【1】央行行长俱乐部】
  - 巴塞尔会议（G30）
  - 每月闭门讨论
  - ⭐ 全球货币政策的"非正式协调"

【2】标准制定权】
  - 巴塞尔协议 → 全球银行必须遵守
  - FSB 标准 → 全球金融监管协调
  - ⭐ "事实上的全球监管者"

【3】研究影响力】
  - BIS Quarterly Review 被全球央行引用
  - BIS 研究员常被各国央行挖角
  - ⭐ "央行研究的高地"
```

## 八、BIS 与中国

### 8.1 中国与 BIS 的关系

```
【中国加入 BIS】
  - 1996 中国人民银行成为 BIS 股东
  - 中国逐步参与 BIS 活动
  - 2018 中国人民银行副行长陈雨露担任 BIS 副总经理

【中国在 BIS 的角色】
  - 参与 BCBS 工作
  - 实施巴塞尔协议 III
  - 推动人民币国际化
  - ⭐ 从"参与者"到"积极贡献者"

【中国实施巴塞尔 III】
  - 2013 银监会发布《商业银行资本管理办法》（中国版 Basel III）
  - 资本要求：基本一致
  - LCR：2018 起实施
  - ⭐ 中国银行业稳健
```

## 十、一句话总结

> **BIS = 全球央行的"央行"——国际金融治理的"最古老、最隐秘、最具影响力"的机构**。
>
> 它成立于 1930 年（大萧条时期），总部位于瑞士巴塞尔。**BIS 既是银行业监管标准（巴塞尔协议）的制定者，也是央行行长俱乐部的"非正式聚会场所"**。它的影响力远超其 650 人的规模。
>
> **理解 BIS = 理解四件事**：
> 1. **巴塞尔协议**：全球银行业监管的核心标准（I/II/III）
> 2. **央行行长俱乐部**：巴塞尔会议是全球货币政策的"非正式协调"
> 3. **金融稳定委员会**：全球金融稳定的协调机构
> 4. **历史与争议**：纳粹关联、透明度、监管过度
>
> 在中国金融监管现代化、银行业国际化的今天，**BIS 是中国央行与全球央行合作的核心平台**——中国已经深度参与巴塞尔 III 实施，并在 BIS 担任重要职务。
