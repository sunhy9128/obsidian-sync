# 非洲猪瘟（African Swine Fever, ASF）

非洲猪瘟是由非洲猪瘟病毒（ASFV）引起的**猪的急性、烈性、高度接触性传染病**，致死率接近 100%。2018年8月首次在中国辽宁沈阳确诊，随后扩散至全国，导致中国生猪产业经历**百年未有的产能去化**。

> **核心命题**：非洲猪瘟是2018-2022年中国猪周期剧烈波动的最大单一外生冲击——它让蛛网模型的"自发波动"叠加"强制去产能"，创造了历史上最极端的猪周期。

## 一、病毒基础

| 维度 | 详情 |
|------|------|
| 病原 | 非洲猪瘟病毒（ASFV） |
| 科属 | Asfarviridae 科，Asfivirus 属 |
| 宿主 | 猪（包括家猪、野猪、疣猪） |
| 致死率 | **接近100%**（家猪） |
| 传播途径 | 直接接触、蜱虫叮咬、饲料污染、泔水、车辆/人员/工具 |
| 潜伏期 | 4-19天 |
| 疫苗 | **至今无安全商用疫苗**（病毒结构复杂、毒株多变） |
| 治疗 | 无有效治疗方法，一旦确诊全场扑杀 |

**与普通猪瘟的区别**：普通猪瘟（Classical Swine Fever, CSF）由黄病毒科病毒引起，有疫苗可用；非洲猪瘟（ASF）由非洲猪瘟病毒引起，**无疫苗**。两者常被混淆但完全不是同一种病。

## 二、全球历史

### 起源
- 1921年首次在肯尼亚确诊，命名"非洲猪瘟"
- 长期在撒哈拉以南非洲野猪间循环（无症状携带）

### 跨洲扩散
| 时间 | 事件 |
|------|------|
| 1957-1960 | 葡萄牙、西班牙（首次出非洲） |
| 1971 | 古巴、巴西、多米尼加 |
| 1978 | 马耳他、撒丁岛 |
| 2007 | 格鲁吉亚（通过猪肉制品从东非流入） |
| 2014 | 波兰、立陶宛、拉脱维亚、爱沙尼亚 |
| 2018年8月 | **中国辽宁沈阳**（首次确诊） |
| 2018-2019 | 蒙古、越南、柬埔寨、菲律宾、韩国 |
| 2019 | 东欧大规模爆发 |

## 三、中国疫情时间线（2018-2020）

### 2018年8月-12月：发现期
- 8月3日：辽宁沈阳沈北新区首例确诊（中国首次）
- 8月19日：确诊疫情扩散至河南郑州
- 12月：全国23个省份出现疫情，累计扑杀生猪超过60万头
- 但实际感染规模远高于扑杀数——很多养殖户为获补贴**瞒报**

### 2019年：爆发期
- 全国31个省级行政区全部出现疫情
- **能繁母猪存栏从2018年7月的4500万头跌至2019年12月的2800万头**（去化38%）
- 生猪存栏从4.3亿头跌至2.3亿头（去化47%）
- 中国生猪养殖业损失数千亿元
- 全球猪肉价格上涨

### 2020年：恢复期
- 疫情逐步得到控制（但仍有零星爆发）
- 国家出台"高致病性病原微生物实验室生物安全管理"等强化措施
- 2020年6月起生猪存栏开始环比回升
- 但产能恢复速度远低于预期

### 2021年至今：常态化防疫
- 常态化防控+分区防控（划分为5个大区）
- 2019年实施《非洲猪瘟防控强化措施指引》
- 2021年实施《非洲猪瘟等重大动物疫病分区防控工作方案》
- 至今没有商用疫苗

## 四、对中国猪周期的冲击

非洲猪瘟是**2018-2022年这一轮极端猪周期的核心外生冲击**：

```
2018年8月 ASF传入 → 能繁母猪存栏 -38% → 2019年10月猪价启动
                                                     ↓
2020年7-8月 猪价峰值35元/kg（历史最高）→ 散户疯狂复养
                                                     ↓
2021年 能繁母猪恢复至4500万头 → 产能过剩
                                                     ↓
2022年 猪价崩盘至14元/kg → 行业亏损 → 又一轮去产能
```

**关键数据**：

| 指标 | 2018年7月（疫情前） | 2019年12月（疫情峰值） | 变化 |
|------|---------------------|------------------------|------|
| 能繁母猪存栏 | 4500万头 | 2800万头 | **-38%** |
| 生猪存栏 | 4.3亿头 | 2.3亿头 | **-47%** |
| 猪肉价格 | 18元/kg | 52元/kg（2019年10月峰值） | **+189%** |
| 猪肉进口 | 119万吨（2017） | 211万吨（2019） | **+77%** |

## 五、对CPI的影响

详见 [[猪周期]] 与 [[CPI与通胀]]：

- 2019年：CPI同比从1.5%涨至5.4%（11月），猪肉贡献约2.4个百分点
- 2020年2月：CPI同比5.2%，猪肉贡献约2.5个百分点（**单一商品贡献近一半的通胀**）
- 央行未单独为猪价加息——因为这是结构性外生冲击，非货币现象

## 六、防控体系

### 1. 强制扑杀与补贴
- 确诊疫情的养殖场**全场扑杀**（无治疗方案）
- 国家补贴：800元/头母猪、200元/头肥猪（2020年标准）
- 但实际补贴远低于市场价，导致**瞒报动机强烈**

### 2. 运输与流通限制
- 跨省运输需检疫合格证
- 2021年起实施"运猪向运肉转变"政策
- 屠宰场分级管理

### 3. 监测与早期发现
- **能繁母猪存栏月度发布**（农业农村部主导，见 [[农业农村部]]）
- 疫情日报制度
- 屠宰场非洲猪瘟检测（PCR）

### 4. 强化生物安全
- 大规模养殖场建立"洗消中心"
- 人员、车辆、物资的多层消毒
- 饲料高温处理（防止病毒通过豆粕/玉米传入）

### 5. 疫苗研发
- 全球多个实验室攻关
- 中国农科院、军事医学科学院发布过候选疫苗
- 但**至今无安全商用疫苗**——病毒的复杂性让减毒活疫苗风险大、灭活疫苗效果差

## 七、对中国生猪产业结构的长期影响

非洲猪瘟**永久性地改变**了中国生猪养殖业的产业结构：

### 1. 散户加速退出
- 2018年：散户（年出栏<500头）占比约40%
- 2023年：散户占比降至约20%
- 散户退出原因：生物安全成本高、瞒报被处罚、复养成功率低

### 2. 规模化企业崛起
- 牧原、温氏、新希望等头部企业市占率从2018年的5%升至2023年的25%+
- 但**规模化也带来同步性风险**——头部企业决策高度同步，反而放大波动

### 3. 楼房养猪兴起
- 牧原等企业投资"楼房养猪"
- 多层建筑、自动化、生物安全可控
- 单体规模可达10万头以上

### 4. 进口依赖度上升
- 2018-2019年：猪肉进口从119万吨飙升至211万吨
- 主要进口来源：欧盟（西班牙、德国）、巴西、美国、加拿大
- 进口成为平抑国内价格的工具

## 八、与"蛛网模型"的关系

非洲猪瘟是**蛛网模型无法预测的外生冲击**——这是 [[蛛网模型]] 的局限性：

| 维度 | 蛛网模型 | 非洲猪瘟 |
|------|----------|----------|
| 性质 | 内生（价格-产量反馈） | **外生（疫病冲击）** |
| 可预测性 | 高（基于上期价格） | **极低**（黑天鹅） |
| 去产能速度 | 慢（蛛网衰减） | **极快（一次性 -38%）** |
| 政策反应 | 储备+期货+补贴 | **强制扑杀+生物安全** |

> **核心结论**：非洲猪瘟证明了——任何模型都无法预测黑天鹅。逆周期调控（见 [[猪周期]] 第六节）的真正价值，不在于消除波动，而在于**缩短极端波动的持续时间**。

## 九、对全球粮食安全的启示

非洲猪瘟的全球扩散给人类上了三课：

1. **高密度养殖的脆弱性**：现代养殖追求效率与规模，但疫病冲击下损失也成倍放大
2. **没有疫苗的疾病最危险**：ASF至今无疫苗，意味着防控只能靠扑杀+生物安全
3. **全球供应链的传染性**：通过猪肉贸易、旅客、野生动物，疫病可在数月内跨洲传播

这也是 [[2025-02-04-中国粮食金融保卫战-巫师财经]] 中"大豆是奇点商品"命题的延伸——**当一种农产品依赖国际贸易+规模化养殖+长产业链时，单一节点的黑天鹅就能撬动整个系统**。
