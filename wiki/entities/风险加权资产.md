# 风险加权资产（RWA）

> 银行按风险程度加权计算的资产总额

## 核心定义

风险加权资产（Risk-Weighted Assets，RWA）是根据资产风险程度赋予不同权重后计算的资产总额，用于计算银行的资本充足率。

## 计算公式

```
资本充足率 = 监管资本 / 风险加权资产 × 100%
```

## 风险权重

- 主权债权：0-20%
- 企业债权：20-100%
- 个人贷款：75-100%
- 衍生品：按市值计价

## 巴塞尔协议要求

- 一级资本充足率：≥6%
- 资本充足率：≥8%
- 加上资本缓冲后要求更高
