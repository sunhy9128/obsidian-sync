# 影子银行

> 游离于传统银行监管之外的信用中介体系 — 金融危机的温床

**影子银行**（Shadow Banking）是指从事类似于银行业务（期限转换、信用转换、杠杆操作）但**不受传统银行监管**的金融机构、金融产品和市场。与传统银行的区别在于：银行吸收存款受存款保险保护，影子银行依靠市场融资没有显性担保。

---

## 一、影子银行的运作逻辑

### 1.1 核心功能与传统银行相同

| 功能 | 传统银行 | 影子银行 |
|------|----------|----------|
| **期限转换** | 短期存款 → 长期贷款 | 短期融资 → 长期资产 |
| **信用转换** | 存款人 → 借款人 | 市场投资人 → 借款人 |
| **杠杆操作** | 吸收存款加杠杆 | 回购/衍生品加杠杆 |
| **信用创造** | 存款派生 | 资产购买派生 |

### 1.2 影子银行 vs 传统银行的风险差异

```
传统银行：
- 监管资本要求
- 存款保险保护
- 央行最后贷款人
- 流动性相对稳定

影子银行：
- 无资本要求（或极低）
- 无显性担保
- 无央行保护
- 市场恐慌时融资渠道瞬间枯竭（"挤兑"）
```

## 二、主要形态与市场

### 2.1 影子银行体系图谱

```
货币市场：
  货币市场基金（MMF）→ 购买商业票据（CP）、大额存单（NCD）
         ↓
证券化市场：
  ABS / MBS / CDO → 资产支持商业票据（ABCP）管道
         ↓
回购市场：
  回购协议（Repo）→ 隔夜融资买入更多资产（加杠杆）
         ↓
对冲基金：
  高杠杆策略 → 持有高风险资产
         ↓
特殊目的机构（SPV / SIV）：
  表外实体 → 持有资产证券化产品
         ↓
信托与资管：
  银行理财 → 资金池 → 投向非标资产（中国的影子银行）
```

### 2.2 各形态详解

#### 货币市场基金（MMF）

| 特征 | 说明 |
|------|------|
| **功能** | 聚集投资者资金购买短期债券（CP、NCD、国债） |
| **影子银行属性** | 将小额投资者资金汇聚，复制银行吸收存款的功能 |
| **风险** | 2020年3月 COVID 冲击时遭遇大规模赎回 → SEC 紧急修订规则 |
| **监管** | 美国 SEC 2010 年修订规则（MMF Reform），2016年进一步要求浮动净值 |

#### 资产支持商业票据（ABCP）

| 特征 | 说明 |
|------|------|
| **功能** | 企业通过 SPV 发行 ABCP 获得短期融资 |
| **影子银行属性** | 用资产（应收账款、租赁款）作抵押，复制银行短期贷款 |
| **风险** | 资产质量问题导致 2007 年 ABCP 市场冻结 |
| **规模** | 2007 年峰值约 $1.2 万亿 |

#### 回购市场（Repo）

| 特征 | 说明 |
|------|------|
| **功能** | 用证券作抵押借隔夜/短期资金（"抵押贷款"） |
| **影子银行属性** | 大量使用回购融资买入资产（加杠杆），本质是短期存款 |
| **风险** | 2008.9.16 回购市场利率飙升至 10%+（正常 < 5%） |
| **核心问题** | 一级交易商（大型投行）在回购市场大量融资，雷曼破产后市场冻结 |

#### 对冲基金

| 特征 | 说明 |
|------|------|
| **功能** | 私募资产管理，使用高杠杆 |
| **影子银行属性** | 杠杆操作 + 期限错配 + 信用中介 |
| **风险** | LTCM（1998）、Amaranth（2006）、次贷危机中大量对冲基金清盘 |

## 三、中国的影子银行

### 3.1 规模与结构

| 指标 | 数据 |
|------|------|
| 影子银行规模（2023） | 约 60-70 万亿人民币（占金融体系 ~20%） |
| 主要构成 | 银行理财、信托、券商资管、基金子公司、民间借贷 |
| 监管框架 | "资管新规"（2018）开始规范化整顿 |

### 3.2 中国影子银行的主要路径

```
银行发行理财产品（资金池）
        ↓
信托公司/券商资管计划（通道）
        ↓
投向：
  1. 地方政府融资平台（LGFV）贷款
  2. 房地产开发商贷款（规避信贷政策限制）
  3. 产能过剩行业贷款
  4. 股票/债券配资
```

### 3.3 中国影子银行的特殊风险

| 风险 | 说明 |
|------|------|
| **隐性担保** | 银行理财长期"刚性兑付"，投资者认为有银行背书 |
| **监管套利** | 绕过信贷政策，向受限制行业输送信贷 |
| **期限错配** | 理财产品多为 3-6 个月，底层资产 3-5 年 |
| **非标资产** | 缺乏流动性的非标准化债权资产（信托贷款、委托贷款） |

### 3.4 资管新规（2018）改革

| 改革要点 | 内容 |
|----------|------|
| **打破刚性兑付** | 禁止银行理财保本保收益 |
| **禁止资金池** | 每只产品单独管理、单独建账、单独核算 |
| **限制期限错配** | 理财产品期限不得短于底层资产期限 |
| **净值化转型** | 理财产品按净值计价，暴露真实风险 |
| **去通道** | 消除多层嵌套，禁止通道业务 |

## 四、影子银行与 2008 金融危机：完整的"解剖"

### 4.1 2008 危机的本质：影子银行体系的崩溃

```
传统认知（错误）：
  "2008 是次贷危机"
       ↓
更准确的认知：
  "2008 是影子银行体系的崩塌"
  次贷只是导火索，影子银行体系是燃料
       ↓
影子银行在 2008 的角色：
  1. 创造并放大次贷风险（CDO/ABS 链条）
  2. 隐藏真实杠杆（回购 + SIV）
  3. 跨机构传染（回购市场 → 商业银行）
  4. 触发系统性危机（雷曼 → AIG → 全球）
```

### 4.2 影子银行如何放大危机的"死亡螺旋"

```
Phase 1：经济下行/房价下跌
  - 房价从 2006 年峰值开始下跌
  - 次贷违约率从 5% 上升至 25%+
       ↓
Phase 2：MBS/CDO 减值
  - 持有 CDO 的影子银行浮亏
  - SIV（结构性投资工具）出现巨额亏损
  - 银行 SIV 的"隐性担保"开始暴露
       ↓
Phase 3：回购市场冻结
  - 2007.8 BNP Paribas 冻结 3 只基金 → 危机信号
  - 2007.9 Northern Rock 银行挤兑（英国）
  - 2008.3 贝尔斯登被摩根大通收购（紧急援助）
       ↓
Phase 4：ABCP 市场冻结
  - 资产支持商业票据（ABCP）市场 2007.8 冻结
  - 1.2 万亿美元 ABCP 无法滚动
  - SIV 失去短期融资来源
       ↓
Phase 5：货币市场基金"打破美元"
  - 2008.9.16 Reserve Primary Fund "Breaking the buck"
  - 净值跌破 1 美元
  - 引发 3 天内 3000 亿美元赎回
  - 整个 MMF 行业恐慌
       ↓
Phase 6：雷曼破产（2008.9.15）
  - 雷曼无法在回购市场获得融资
  - 缺乏政府担保（区别于贝尔斯登）
  - 申请破产（史上最大破产案，6390 亿美元资产）
       ↓
Phase 7：AIG 危机
  - AIG 卖出 5000 亿美元 CDS 担保
  - 雷曼、雷曼债券持有人触发 CDS 赔付
  - AIG 流动性枯竭
  - 美联储 850 亿美元救助（2008.9）
       ↓
Phase 8：系统性危机
  - 货币市场冻结（Ted 利差飙升至 4.5%）
  - 商业票据市场冻结
  - 信贷市场全面冻结
  - 全球金融危机爆发
       ↓
Phase 9：实体经济衰退
  - 信贷冻结 → 企业无法融资
  - 失业率从 5% 飙升至 10%
  - 全球 GDP 2009 年萎缩
```

### 4.3 2008 危机中的关键"影子银行"机构

#### **贝尔斯登的对冲基金（2007.6 倒闭）**

```
两只基金：
  - Bear Stearns High-Grade Structured Credit Fund
  - Bear Stearns High-Grade Structured Credit Enhanced Leveraged Fund
       ↓
规模：
  - 高等级基金：100 亿美元
  - 增强杠杆基金：6.4 亿美元（3 倍杠杆）
       ↓
崩溃过程：
  - 持有大量次贷 CDO
  - 2007.6 损失 90%+
  - 紧急追加保证金失败
  - 最终清盘
       ↓
历史意义：
  - 影子银行危机的第一张"多米诺骨牌"
  - 暴露次贷 CDO 的真实风险
  - 引发市场对 SIV、ABCP 的担忧
```

#### **ABCP 管道（2007.8 冻结）**

```
ABCP 的运作：
  企业（通用电气、卡特彼勒等）
       ↓
  出售应收账款给 SPV
       ↓
  SPV 发行 ABCP（短期商业票据）
       ↓
  货币市场基金/银行购买 ABCP
       ↓
  SPV 用 ABCP 资金支付企业
       ↓
ABCP 危机的传导：
  - 2007.8 ABCP 市场冻结（投资者拒绝购买）
  - SIV 无法滚动短期融资
  - SIV 只能出售长期资产（CDO）求生
  - CDO 价格暴跌 → 更多 SIV 倒闭
  - 这是"期限错配"危机的典型
       ↓
规模影响：
  - 2007 年 ABCP 峰值 1.2 万亿美元
  - 危机后降至 5000 亿美元（-58%）
```

#### **雷曼兄弟（2008.9.15 破产）**

```
雷曼的"影子银行"特征：
  - 不仅是投资银行
  - 也是 SIV 的主要发起人
  - 回购市场最大借款人
  - 持有大量次贷相关资产
       ↓
雷曼的杠杆：
  - 2008 年初：30+ 倍杠杆
  - 总资产 6390 亿美元，股东权益仅 250 亿
       ↓
雷曼倒闭过程：
  - 2008.3 贝尔斯登被救助（市场预期雷曼也会被救助）
  - 2008.9 财政部/美联储拒绝救助雷曼
  - 9.13 巴克莱银行、美国银行谈判失败
  - 9.15 申请破产保护
       ↓
雷曼倒闭的"全球冲击"：
  - 货币市场基金 Reserve Primary Fund 持有雷曼债券
  - 净值跌破 1 美元 → 3 天内赎回 3000 亿美元
  - AIG 卖出 5000 亿 CDS → 触发赔付 → AIG 危机
  - 全球信贷市场冻结
       ↓
雷曼倒闭的"政治争议"：
  - 为什么救助贝尔斯登但不救雷曼？
  - 财政部/美联储决策的政治性
  - 国会调查：保罗森"压力太大无法救援"
  - 这一决定"改变了历史"
```

#### **AIG（American International Group，2008.9 救助）**

```
AIG 的角色：
  - 全球最大保险公司
  - 通过 AIG FP（AIG Financial Products）卖出 CDS
  - 担保 5000+ 亿美元的 MBS/CDO 信用风险
       ↓
AIG 的"影子银行"特征：
  - CDS 本质是"信用保险"
  - 但 AIG 不需要监管资本（保险监管要求低）
  - 不透明（OTC 衍生品）
  - 集中风险（CDS 集中度高）
       ↓
AIG 的崩溃过程：
  - 2008.9 标普、穆迪下调 AIG 评级
  - CDS 交易对手要求追加抵押品（130 亿美元）
  - AIG 流动性枯竭
  - 2008.9.16 美联储 850 亿美元紧急贷款
       ↓
AIG 救助的最终规模：
  - 总额 1823 亿美元（历史最大单笔救助）
  - 91% 由美联储提供
  - 实际偿还 51 亿美元利息（纳税人净赚）
       ↓
AIG 救助的争议：
  - 道德风险（"太大而不能倒"）
  - 奖金争议（AIG 发放 1.65 亿美元奖金引发公愤）
  - 拯救对象是交易对手（高盛、德银、巴黎银行等）
```

#### **两房（房利美、房地美）**

```
两房的"特殊"地位：
  - 私人公司但有"政府隐性担保"
  - 持有/担保美国 50%+ 的住房抵押贷款
  - 监管：OFHEO（轻监管）
  - 既是"政府支持企业"（GSE）也是"影子银行"
       ↓
两房的危机：
  - 持有大量次贷/MBS
  - 2007 年开始亏损
  - 2008.7 股价暴跌 90%
  - 2008.9.7 政府接管
       ↓
政府接管方案：
  - 财政部 1000 亿美元优先股
  - 财政部 无限流动性支持
  - 实际最终救助 1875 亿美元
       ↓
意义：
  - 政府正式承担两房风险
  - 抵押贷款市场国有化
  - 为美联储 MBS 购买铺平道路
```

### 4.4 关键"金融基础设施"的崩溃

#### **回购市场（Repo Market）**

```
回购市场的角色：
  - "华尔街的命脉"
  - 每天数万亿美元交易
  - 大型投行通过回购融资购买证券（加杠杆）
       ↓
回购市场的运作：
  - 投行用证券作抵押借入隔夜资金
  - 利率通常接近联邦基金利率
  - 是"高杠杆"的来源
       ↓
回购市场的崩溃：
  - 2008.9 雷曼破产后
  - 9.16 回购利率飙升至 10%+（正常 5%）
  - 主要交易对手拒绝与雷曼交易
  - 货币市场基金"去杠杆"
       ↓
美联储应对：
  - 2008.9.16 推出新工具：PDCF（一级交易商信贷）
  - 2008.10 推出 TAF（定期拍卖工具）
  - 实际"接管"了回购市场
```

#### **商业票据市场**

```
商业票据（CP）的角色：
  - 企业短期融资的核心工具
  - 美国非金融企业 30% 短期融资来自 CP
  - 总额 ~1.7 万亿美元
       ↓
CP 市场的崩溃：
  - 2008.9 雷曼破产后
  - CP 利差从正常 50bp 飙升至 400bp+
  - 通用电气、AT&T 等大企业无法滚动 CP
  - 美国企业短期融资枯竭
       ↓
美联储应对：
  - 2008.10.7 创建 CPFF（商业票据融资工具）
  - 美联储直接购买 3 个月 CP
  - 实际接管 CP 市场
```

#### **货币市场基金（MMF）**

```
MMF 的角色：
  - 2 万亿美元规模
  - 企业短期现金管理的核心
  - 类似银行存款但无保险
       ↓
MMF 的"打破美元"事件：
  - 2008.9.16 Reserve Primary Fund
  - 净值从 1.00 美元跌至 0.97 美元
  - 行业首次"Breaking the buck"（自 1994）
       ↓
危机反应：
  - 3 天内 3000 亿美元赎回
  - 全行业恐慌
  - 国会紧急立法：临时担保所有 MMF（2008.9.19）
       ↓
美联储应对：
  - 推出 AMLF（资产支持商业票据货币市场基金流动性工具）
  - 直接为 MMF 提供流动性
```

### 4.5 危机中"影子银行"的资金流向

```
危机前的资金链：
  MMF → ABCP 管道 → SIV → 房贷/信用卡贷款
  ↑
  投资者（视为"安全"）
       ↓
危机中的崩塌：
  房贷违约 → MBS 减值
       ↓
  SIV 亏损 → ABCP 无法滚动
       ↓
  MMF 持有 ABCP 亏损 → 净值跌破 1 美元
       ↓
  投资者赎回 → MMF 抛售资产
       ↓
  资产价格暴跌 → 更多 SIV 亏损
       ↓
  "死亡螺旋"放大 10 倍
```

### 4.6 2008 危机后的"量化"总结

| 指标 | 危机前（2007） | 危机后（2010） | 变化 |
|------|---------------|---------------|------|
| **ABCP 规模** | $1.2 万亿 | $0.5 万亿 | -58% |
| **回购市场（美国国债）** | $2.5 万亿 | $4.0 万亿 | +60% |
| **MMF 规模** | $2.0 万亿 | $2.8 万亿 | +40%（避险） |
| **对冲基金总规模** | $1.8 万亿 | $1.5 万亿 | -17% |
| **影子银行（FSB）** | ~$55 万亿 | ~$60 万亿 | +9% |
| **Fed 资产负债表** | $0.9 万亿 | $2.4 万亿 | +167% |

### 4.7 影子银行危机的"五点教训"

```
1. **杠杆是致命的**
   - 雷曼 30+ 倍杠杆
   - 对冲基金 5-10 倍杠杆
   - 一旦资产减值 5%+，立即死亡

2. **期限错配是脆弱的**
   - ABCP 短期 / CDO 长期
   - 一旦短期融资冻结，长期资产无法变现

3. **透明度是核心**
   - SIV 在表外，无人知道真实风险
   - 衍生品 OTC，不透明
   - 危机时无法评估风险

4. **回购市场是关键基础设施**
   - 看似技术性，实则命脉
   - 一旦冻结，整个体系瘫痪
   - 美联储必须充当最后贷款人

5. **影子银行的"传染性"**
   - 通过回购、衍生品、SPV 高度关联
   - 一个机构倒闭，整个体系冻结
   - 这是 2008 比 1987 严重得多的原因
```

## 七、影子银行的"灰犀牛"特征

| 特征 | 说明 |
|------|------|
| **规模大** | 占金融体系的 25-30%，不可忽视 |
| **透明度低** | 表外业务、复杂结构，难以准确评估风险 |
| **相互关联** | 与传统银行通过回购、担保、信用支持高度关联 |
| **监管滞后** | 创新产品出现 → 监管跟进 → 套利转移，循环往复 |
| **顺周期放大器** | 经济好时加速扩张，经济差时加速收缩 |

> **灰犀牛 vs 黑天鹅**：影子银行风险是可预见的、渐进累积的系统性问题，而非突发性事件。这是"灰犀牛"（看得见、知道会发生、但准备不足的风险）。

## 八、化债与影子银行的"出清"

### 8.1 城投融资的影子银行链条

中国地方政府的"城投 + 影子银行"模式，是影子银行最具特色的本土形态：

```
居民购买银行理财（影子银行）
       ↓
银行理财 → 信托计划/资管计划
       ↓
投向城投债/委托贷款
       ↓
城投用于地方基础设施
       ↓
偿债来源 → 政府回购、土地出让
```

这一链条是过去 15 年中国地方政府"绕过监管搞基建"的核心机制，也是中国影子银行规模一度占金融体系 25%-30% 的主要原因。

### 8.2 化债是对"中国式影子银行"的总清算

**2023 年化债的本质** = 切断"地方政府主导的影子银行"链条：

| 维度 | 化债前 | 化债后 |
|------|--------|--------|
| 城投融资渠道 | 银行/信托/债券（影子银行+传统） | 政府债券（显性化） |
| 利率 | 6%-8%（市场化 + 隐性担保） | 2.5%-3%（政府债利率） |
| 期限 | 2-3 年 | 10-30 年 |
| 信息披露 | 不透明 | 法定显性 |
| 政府责任 | 隐性担保 | 法定偿还 |

**化债的"三连击"**：
1. **切断影子银行渠道**：资管新规（2018）→ 化债（2023）
2. **切断土地财政依赖**：房地产下行 → 城投偿债来源断裂
3. **切断隐性担保**：特殊再融资债券显性化 → 城投信用与政府信用逐步脱钩

### 8.3 化债的"副作用"——影子银行整体收缩

| 影响 | 说明 |
|------|------|
| **银行理财规模** | 净值化转型 + 城投标的下沉受限，规模增长放缓 |
| **信托业** | 政信合作业务大幅压降，信托公司转型艰难 |
| **融资租赁** | 城投售后回租业务收缩 |
| **信用债市场** | 城投债占信用债比重从 35% 降至 28% |

> **结论**：化债不是单一事件，而是中国影子银行体系的"终极出清"，将重塑未来 10 年的金融结构。
