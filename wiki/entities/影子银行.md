# 影子银行

> 游离于传统银行监管之外的信用中介体系 — 金融危机的温床

**影子银行**（Shadow Banking）是指从事类似于银行业务（期限转换、信用转换、杠杆操作）但**不受传统银行监管**的金融机构、金融产品和市场。与传统银行的区别在于：银行吸收存款受存款保险保护，影子银行依靠市场融资没有显性担保。

---

## 一、影子银行的运作逻辑

### 1.1 核心功能与传统银行相同

| 功能 | 传统银行 | 影子银行 |
|------|----------|----------|
| **期限转换** | 短期存款 → 长期贷款 | 短期融资 → 长期资产 |
| **信用转换** | 存款人 → 借款人 | 市场投资人 → 借款人 |
| **杠杆操作** | 吸收存款加杠杆 | 回购/衍生品加杠杆 |
| **信用创造** | 存款派生 | 资产购买派生 |

### 1.2 影子银行 vs 传统银行的风险差异

```
传统银行：
- 监管资本要求
- 存款保险保护
- 央行最后贷款人
- 流动性相对稳定

影子银行：
- 无资本要求（或极低）
- 无显性担保
- 无央行保护
- 市场恐慌时融资渠道瞬间枯竭（"挤兑"）
```

## 二、主要形态与市场

### 2.1 影子银行体系图谱

```
货币市场：
  货币市场基金（MMF）→ 购买商业票据（CP）、大额存单（NCD）
         ↓
证券化市场：
  ABS / MBS / CDO → 资产支持商业票据（ABCP）管道
         ↓
回购市场：
  回购协议（Repo）→ 隔夜融资买入更多资产（加杠杆）
         ↓
对冲基金：
  高杠杆策略 → 持有高风险资产
         ↓
特殊目的机构（SPV / SIV）：
  表外实体 → 持有资产证券化产品
         ↓
信托与资管：
  银行理财 → 资金池 → 投向非标资产（中国的影子银行）
```

### 2.2 各形态详解

#### 货币市场基金（MMF）

| 特征 | 说明 |
|------|------|
| **功能** | 聚集投资者资金购买短期债券（CP、NCD、国债） |
| **影子银行属性** | 将小额投资者资金汇聚，复制银行吸收存款的功能 |
| **风险** | 2020年3月 COVID 冲击时遭遇大规模赎回 → SEC 紧急修订规则 |
| **监管** | 美国 SEC 2010 年修订规则（MMF Reform），2016年进一步要求浮动净值 |

#### 资产支持商业票据（ABCP）

| 特征 | 说明 |
|------|------|
| **功能** | 企业通过 SPV 发行 ABCP 获得短期融资 |
| **影子银行属性** | 用资产（应收账款、租赁款）作抵押，复制银行短期贷款 |
| **风险** | 资产质量问题导致 2007 年 ABCP 市场冻结 |
| **规模** | 2007 年峰值约 $1.2 万亿 |

#### 回购市场（Repo）

| 特征 | 说明 |
|------|------|
| **功能** | 用证券作抵押借隔夜/短期资金（"抵押贷款"） |
| **影子银行属性** | 大量使用回购融资买入资产（加杠杆），本质是短期存款 |
| **风险** | 2008.9.16 回购市场利率飙升至 10%+（正常 < 5%） |
| **核心问题** | 一级交易商（大型投行）在回购市场大量融资，雷曼破产后市场冻结 |

#### 对冲基金

| 特征 | 说明 |
|------|------|
| **功能** | 私募资产管理，使用高杠杆 |
| **影子银行属性** | 杠杆操作 + 期限错配 + 信用中介 |
| **风险** | LTCM（1998）、Amaranth（2006）、次贷危机中大量对冲基金清盘 |

## 三、中国的影子银行

### 3.1 规模与结构

| 指标 | 数据 |
|------|------|
| 影子银行规模（2023） | 约 60-70 万亿人民币（占金融体系 ~20%） |
| 主要构成 | 银行理财、信托、券商资管、基金子公司、民间借贷 |
| 监管框架 | "资管新规"（2018）开始规范化整顿 |

### 3.2 中国影子银行的主要路径

```
银行发行理财产品（资金池）
        ↓
信托公司/券商资管计划（通道）
        ↓
投向：
  1. 地方政府融资平台（LGFV）贷款
  2. 房地产开发商贷款（规避信贷政策限制）
  3. 产能过剩行业贷款
  4. 股票/债券配资
```

### 3.3 中国影子银行的特殊风险

| 风险 | 说明 |
|------|------|
| **隐性担保** | 银行理财长期"刚性兑付"，投资者认为有银行背书 |
| **监管套利** | 绕过信贷政策，向受限制行业输送信贷 |
| **期限错配** | 理财产品多为 3-6 个月，底层资产 3-5 年 |
| **非标资产** | 缺乏流动性的非标准化债权资产（信托贷款、委托贷款） |

### 3.4 资管新规（2018）改革

| 改革要点 | 内容 |
|----------|------|
| **打破刚性兑付** | 禁止银行理财保本保收益 |
| **禁止资金池** | 每只产品单独管理、单独建账、单独核算 |
| **限制期限错配** | 理财产品期限不得短于底层资产期限 |
| **净值化转型** | 理财产品按净值计价，暴露真实风险 |
| **去通道** | 消除多层嵌套，禁止通道业务 |

## 四、影子银行与金融危机的关系

### 4.1 影子银行如何放大危机

```
经济下行/利率上升
  ↓
次级贷款违约率上升
  ↓
MBS/CDO 减值 → 持有这些资产的影子银行浮亏
  ↓
融资方收紧授信 → 影子银行融资困难
  ↓
影子银行被迫甩卖资产 → 市场流动性枯竭
  ↓
资产价格进一步下跌（甩卖→下跌→更多浮亏→更多甩卖）
  ↓
传染至传统银行 → 信贷冻结 → 实体经济衰退
```

### 4.2 2008 年危机中的影子银行

| 机构/市场 | 角色 | 危机中的表现 |
|-----------|------|-------------|
| **贝尔斯登对冲基金** | 持有大量 CDO | 2007.6 清盘 |
| **两房（房利美/房地美）** | 持有/担保 MBS | 2008.7 政府接管 |
| **雷曼兄弟** | 回购市场最大借款人 | 2008.9 破产 |
| **AIG（CDS 出售方）** | 信用风险担保 | 2008.9 救助 |
| **ABCP 管道** | 短期融资长期资产 | 2007.8 冻结 |
| **货币市场基金** | 持有 CP/ABCP | 2008.9 " Breaking the buck " |

## 六、监管应对

### 6.1 国际监管框架

| 监管举措 | 内容 |
|----------|------|
| **FSB 影子银行监测** | 每年发布全球影子银行监测报告 |
| **Basel III 改革** | 限制银行对影子银行的敞口（杠杆率、大额敞口限制） |
| **MMF 改革** | 货币市场基金引入浮动净值、赎回费用 |
| **OTC 衍生品清算** | 强制集中清算，减少对手方风险 |
| **SIFI 指定** | 系统重要性影子银行机构纳入监管 |

### 6.2 中国的监管演进

```
2012：银监会开始摸底影子银行
2013："8号文"限制银行理财投资非标资产
2017：开始统一监管（资管新规征求意见稿）
2018：资管新规正式落地
2021：理财净值化转型加速
2022-今：持续压降影子银行规模，推进整改
```

## 七、影子银行的"灰犀牛"特征

| 特征 | 说明 |
|------|------|
| **规模大** | 占金融体系的 25-30%，不可忽视 |
| **透明度低** | 表外业务、复杂结构，难以准确评估风险 |
| **相互关联** | 与传统银行通过回购、担保、信用支持高度关联 |
| **监管滞后** | 创新产品出现 → 监管跟进 → 套利转移，循环往复 |
| **顺周期放大器** | 经济好时加速扩张，经济差时加速收缩 |

> **灰犀牛 vs 黑天鹅**：影子银行风险是可预见的、渐进累积的系统性问题，而非突发性事件。这是"灰犀牛"（看得见、知道会发生、但准备不足的风险）。
