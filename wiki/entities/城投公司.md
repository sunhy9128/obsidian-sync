# 城投公司

> 中国地方政府融资的"第二财政" — 化债的核心主体

## 一、定义

**城投公司**（Local Government Financing Vehicle，LGFV）是**由地方政府及其部门/机构出资设立、主要承担政府投资项目融资功能的国有企业**。

本质上是**地方政府的"融资代理人"**：
- 政府不能直接干的事 → 由城投代劳
- 政府不能直接借的钱 → 由城投代借
- 政府不能直接担的责任 → 由城投承担（政府隐性担保）

---

## 二、历史发展

### 2.1 形成背景

| 阶段 | 时间 | 关键事件 |
|------|------|---------|
| **起源** | 上世纪 90 年代 | 上海等地率先成立城投 |
| **爆发** | 2008-2010 | 4万亿刺激计划，城投数量从 ~3800 增至 10000+ |
| **第一轮整治** | 2010-2014 | 银监会 19 号文、463 号文 |
| **规范化** | 2014-2018 | 43 号文、置换债、PPP 推广 |
| **严控** | 2018-2022 | 资管新规、隐性债务终身问责 |
| **化债** | 2023-至今 | 12 万亿化债方案、一揽子整合转型 |

### 2.2 数量演变

| 时间 | 城投数量（家） |
|------|--------------|
| 2008 | ~3,800 |
| 2014 | ~10,000（峰值） |
| 2020 | ~3,000（发债城投） |
| 2024 | ~2,000（持续整合中） |

---

## 三、运作模式

### 3.1 典型业务模式

```
地方政府
  ↓ 注入资产（土地、股权）
城投公司（地方政府 100% 控股）
  ↓ 发行债券、银行贷款
银行、信托、基金（市场融资）
  ↓ 募集资金
用于：
  - 基础设施建设（公路、桥梁）
  - 保障房、棚改
  - 园区开发
  - 公益类项目
  ↓ 偿债来源
  - 政府回购（BT 模式）
  - 土地出让收入返还
  - 政府补贴
  - 项目自身收益
```

### 3.2 城投的三重身份

| 身份 | 含义 |
|------|------|
| **政府代理人** | 替政府融资、实施基建项目 |
| **市场主体** | 自主经营、自负盈亏 |
| **债券发行人** | 在市场上发行债券，接受评级 |

> **核心张力**：城投既不是纯政府（不承担无限责任），又不是纯企业（享受隐性担保）。这种"半政府半市场"的属性，是城投风险的本质来源。

## 四、城投债

### 4.1 定义

**城投债**是城投公司发行的、**募集资金主要用于地方政府投资项目的企业债券**，包括：
- 企业债
- 公司债
- 中期票据
- 短期融资券
- 非公开定向债务融资工具（PPN）
- 信托计划（影子银行渠道）

### 4.2 规模演变

| 时间 | 城投债余额（万亿） | 占信用债比重 |
|------|------------------|-------------|
| 2014 | ~5 | 25% |
| 2018 | ~8（峰值） | 35% |
| 2023 | ~13 | 30% |
| 2024 | ~12（化债压降） | 28% |

### 4.3 城投债的特征

| 特征 | 说明 |
|------|------|
| **发行主体** | 地方政府融资平台（城投） |
| **偿债来源** | 项目收益 + 政府补贴 + 土地出让 |
| **隐性担保** | 市场普遍认为有政府背书 |
| **风险定价** | 历史上扭曲，未充分反映信用风险 |
| **定价机制** | 同省份城投"抱团"，差异化小 |

## 五、城投与四大传统融资模式

### 5.1 银行贷款（早期主渠道）

- 2008-2014 年主导
- 受 463 号文约束后逐步收缩

### 5.2 信托贷款（影子银行渠道）

- 2010-2017 年间高峰期
- 通过银信合作绕过监管
- 2018 年资管新规后大幅压降

### 5.3 城投债（公开市场）

- 2014 年起逐步替代非标
- 流动性好、信息披露相对规范
- 成为城投融资的"主战场"

### 5.4 融资租赁（类信贷）

- 通过售后回租实现融资
- 不计入贷款规模
- 2018 年起严控

## 六、城投与地方政府债务的关系

### 6.1 三层结构

```
显性债务（政府债）：
  - 地方政府发行
  - 纳入预算管理
  - ~40 万亿（2023）

隐性债务（城投）：
  - 城投发行/借款
  - 不纳入预算
  - 政府负救助责任
  - ~14.3 万亿（2023）

第三类债务（城投经营性）：
  - 城投经营性项目借款
  - 理论上政府不担责
  - 市场认为有隐性担保
  - 规模较大
```

### 6.2 城投 = 隐性债务的"主体"

**核心判断**：城投公司是中国地方政府隐性债务的**主要承载主体**。化债的本质，就是解决城投的高息短债问题。

## 七、化债与城投转型

### 7.1 化债对城投的影响

| 影响 | 说明 |
|------|------|
| **债务规模下降** | 高息债务被置换，整体规模压降 |
| **融资成本下降** | 从 6%-8% 降至 2.5%-3% |
| **期限拉长** | 从 2-3 年延至 10-30 年 |
| **主体整合** | 一县一平台，多平台合并 |
| **业务转型** | 减少政府类项目，转向市场化业务 |

### 7.2 城投的"三条转型路径"

#### 路径一：公益类城投 → 政府平台公司

- 继续承担政府投资项目
- 由政府购买服务偿还
- 实质是政府职能延伸

#### 路径二：经营性城投 → 市场化国企

- 转型为普通国有企业
- 自负盈亏、市场化融资
- 退出政府融资功能

#### 路径三：空壳城投 → 注销/合并

- 无业务的空壳平台
- 直接注销或被合并
- 2024 年起大规模清理

### 7.3 城投整合的"浙江模式"

| 措施 | 内容 |
|------|------|
| **数量压减** | 浙江省城投从 800+ 整合至 ~300 |
| **层级提升** | 县级平台整合为市级平台 |
| **业务聚焦** | 一平台一主业，避免同质竞争 |
| **市场化转型** | 鼓励混改、上市 |

## 八、城投的典型风险事件

### 8.1 2014：云南城投事件

- 国务院 43 号文出台
- 引发城投债首次系统性下跌

### 8.2 2016-2018：东北特钢、贵州多起违约

- 国企信仰破灭
- 投资者风险意识觉醒

### 8.3 2021-2023：地产、城投双杀

- 房企暴雷 → 土地财政坍塌
- 城投再融资能力大幅下降
- 云南、贵州多地非标违约

### 8.4 2023：化债前夜的至暗时刻

- 化债政策出台前
- 12 个重点省份城投非标违约频发
- 信托产品延期兑付

## 九、城投与影子银行

### 9.1 历史共生关系

| 阶段 | 关系 |
|------|------|
| 2010-2014 | 城投是影子银行的主要资金投向 |
| 2014-2018 | 影子银行是城投的重要融资渠道 |
| 2018-2022 | 资管新规切断部分渠道 |
| 2023-至今 | 化债切断影子银行链条 |

### 9.2 典型链条

```
居民购买银行理财（影子银行）
       ↓
银行理财 → 信托计划/资管计划
       ↓
投向城投债/委托贷款
       ↓
城投用于地方基础设施
       ↓
偿债来源 → 政府回购、土地出让
```

> **核心判断**：城投 + 影子银行 = 过去 15 年中国地方政府"绕过监管搞基建"的核心机制。2023 年化债是对这一机制的**总清算**。

---

## 十、未来展望

### 10.1 短期（2024-2026）

- 完成 12 万亿化债
- 城投数量大幅压减
- 公益性城投归位

### 10.2 中期（2027-2030）

- 第三类债务处置方案落地
- 城投全面转型为市场化主体
- 部分城投上市

### 10.3 长期（2030 后）

- 城投作为历史概念逐步退出
- 地方政府通过正规渠道融资
- 财政体制改革深化
