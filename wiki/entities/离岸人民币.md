# 离岸人民币

> CNH — 在中国大陆以外流通和交易的离岸人民币

**离岸人民币**（CNH，H 代表 Hong Kong）是指在中国大陆境外流通、交易和存放的人民币。CNH 与在岸人民币（CNY）共用同一个货币代码，但因流通区域和监管环境不同，形成了独立的汇率和利率体系。

---

## 一、离岸人民币市场的形成

### 1.1 发展历程

| 时间 | 里程碑事件 |
|------|-----------|
| 2003 年 | 香港银行开始提供人民币存款业务 |
| 2004 年 | 个人人民币业务（渣打、汇丰）正式开始 |
| 2010 年 | 离岸人民币外汇市场（CNH）正式建立 |
| 2012 年 | 伦敦成为第二个主要离岸人民币中心 |
| 2015 年 | 离岸央票发行机制建立（回笼 CNH 流动性） |
| 2023 年 | 跨境人民币使用范围扩大，CNH 影响力持续提升 |

### 1.2 主要离岸人民币中心

| 中心 | 地位 | 特点 |
|------|------|------|
| **香港** | 最大、最初 | 清算量最大，CNH 产品最丰富（存款、债券、衍生品） |
| **伦敦** | 欧洲核心 | 主要外汇交易中心，时区优势 |
| **新加坡** | 东南亚枢纽 | 贸易结算为主 |
| **法兰克福** | 欧洲人民币存款 | 欧洲人民币业务 |
| **首尔/台北** | 区域性 | 贸易结算使用 |

## 二、CNH vs CNY：核心差异

### 2.1 基本对比

| 维度 | CNY（在岸） | CNH（离岸） |
|------|------------|-------------|
| **流通区域** | 中国大陆 | 港澳台及海外 |
| **汇率形成** | 央行中间价 + 2% 波幅限制 | 完全市场化，由供需决定 |
| **参与主体** | 中国境内居民和企业 | 境外银行、企业、投资者 |
| **监管机构** | 中国人民银行（PBOC） | 无单一监管，各离岸中心分别监管 |
| **利率基准** | LPR / 存款基准利率 | HONG KONG INTERBANK OFFER RATE（HK Hibor）挂钩 |
| **汇率波动** | 受管制（2% 波幅限制） | 完全市场化，可大幅偏离 CNY |

### 2.2 CNY-CNH 价差（Spread）

```
CNY - CNH = 在岸与离岸汇率之差

正常情况：CNY ≈ CNH，价差在 ±100 pips 内（正常波动）
贬值压力：CNH 更弱（CNY - CNH > 0，即 CNH 贬值更多）
升值压力：CNH 更强（CNH - CNY > 0，即 CNH 升值更多）
```

| 价差状态 | 市场含义 |
|----------|----------|
| CNH 比 CNY 弱 200-500 pips | 正常贬值预期 |
| CNH 比 CNY 弱 1000+ pips | 强烈贬值预期，市场情绪紧张 |
| CNH 比 CNY 强 | 外资流入中国，市场对中国信心强 |
| 价差快速扩大 | 贬值预期升温，资本外流压力 |

**历史极值**：
- 2015 年 8.11 汇改后：CNH 一度比 CNY 弱 ~2000 pips（~2%）
- 2016 年初：CNH 比 CNY 弱约 1000-1300 pips
- 2022 年：CNH 比 CNY 弱约 500-800 pips（美元强势周期）

### 2.3 价差的驱动因素

| 因素 | 对 CNY-CNH 价差的影响 |
|------|----------------------|
| **资本外流预期** | CNH 弱于 CNY（做空 CNH / 换美元） |
| **外资流入中国** | CNH 强于 CNY（买入人民币资产） |
| **境内流动性收紧** | CNY 利率高，套利资金流入 CNY，CNH 弱 |
| **境内流动性宽松** | CNY 利率低，套利资金流向 CNH，CNH 强 |
| **地缘政治风险** | CNH 抛压加大，CNH 相对更弱 |
| **央行干预** | 通过离岸央票收紧 CNH → CNH 走强 |

## 三、离岸人民币利率

### 3.1 CNH Hibor（离岸人民币香港银行间利率）

CNH Hibor 是离岸人民币市场的基准利率，由香港银行间市场报价产生：

| 期限 | 期限 | 市场含义 |
|------|------|----------|
| **隔夜** | CNH Hibor O/N | 极短期流动性，波动最大 |
| **1 周** | CNH Hibor 1W | 短期利率 |
| **1 个月** | CNH Hibor 1M | 中期利率基准 |
| **3 个月** | CNH Hibor 3M | 最重要的离岸利率指标 |
| **1 年** | CNH Hibor 1Y | 长期利率参考 |

### 3.2 CNH Hibor vs CNY Shibor

| 维度 | CNH Hibor（离岸） | CNY Shibor（在岸） |
|------|------------------|-------------------|
| 报价方式 | 香港银行报价（非成交利率） | 成交利率（加权平均） |
| 波动性 | 较大，受资本流动影响 | 较稳定，受 PBOC 政策影响 |
| 敏感因素 | 外资流向、地缘风险 | 央行公开市场操作 |
| 参考性 | 反映离岸市场供需 | 反映境内流动性 |

**CNH Hibor 飙高的含义**：
- 离岸人民币流动性紧张
- 央行通过抽走 CNH 流动性提高做空成本（2016 年初多次出现）
- 做空人民币者被迫平仓，人民币升值压力

## 四、离岸央票机制

2018 年 11 月，中国人民银行在香港发行**离岸央票**（CNH Bills），这是央行调控离岸人民币流动性的核心工具：

### 4.1 机制原理

```
央行发行离岸央票（回收 CNH 流动性）
  → CNH 供给减少
  → CNH Hibor 利率上升
  → 做空人民币成本上升（借 CNH 成本更高）
  → 做空力量减弱
  → CNH 汇率企稳
```

### 4.2 离岸央票的作用

| 作用 | 机制 |
|------|------|
| **收紧 CNH 流动性** | 直接减少离岸人民币供给 |
| **提高做空成本** | CNH Hibor 上升 → 借 CNH 做空成本增加 |
| **提供利率基准** | 为离岸人民币产品提供定价参考 |
| **传递政策信号** | 发行节奏反映央行对离岸市场的态度 |
| **管理汇率预期** | 逆周期调节 CNH-CNY 价差 |

### 4.3 历史上离岸央票的使用

| 时间 | 背景 | 发行节奏 | 效果 |
|------|------|----------|------|
| 2018-11 | 人民币贬值压力加大 | 常态化发行 | 有效遏制 CNH 快速贬值 |
| 2019-08 | 贸易战加剧贬值 | 加速发行 | 稳定预期 |
| 2022 | 美元强势 + 资本外流 | 发行频率降低 | 市场化为主 |
| 2024 | 人民币国际化加速 | 适度发行 | 稳定 + 国际化并重 |

## 五、CNH 对市场的影响

### 5.1 对 A 股的影响

| 传导路径 | 影响 |
|----------|------|
| CNH 贬值 → 资本外流预期 | A 股流动性承压（外资撤离） |
| CNH 贬值 → 进口成本上升 | 通胀预期，输入性通胀 |
| CNH Hibor 飙升 → 流动性紧张 | A 股情绪受压 |
| CNH 稳定/升值 | 外资净流入 A 股概率提高 |

### 5.2 对 CNY 汇率的影响

CNH 通过以下渠道影响 CNY：
- **套利机制**：CNY-CNH 价差触发套利 → 价差自动收窄
- **预期传导**：CNH 反映市场真实预期，影响 CNY 中间价定价
- **央行参考**：央行制定 CNY 中间价时参考 CNH 水平

> **核心关系**：CNH 是 CNY 的"市场预期镜像"——央行可以通过 CNY 中间价调控，但 CNH 更真实地反映国际资本对中国资产的信心。

### 5.3 对跨境贸易的影响

| 影响 | 具体表现 |
|------|----------|
| CNH 稳定利于贸易定价 | 进出口企业愿意用人民币结算 |
| CNH 波动加大贸易成本 | 汇率风险增加，企业套保需求上升 |
| CNH 国际化推进 | 越来越多贸易用 CNH 计价结算 |

## 六、CNH 市场的产品体系

| 产品类型 | 举例 | 用途 |
|---------|------|------|
| **存款** | CNH 存款 | 最基础产品 |
| **债券** | 点心债（在香港发行的人民币债券） | 融资 / 投资 |
| **外汇** | CNH/CNY, CNH/USD 等 | 汇率交易 |
| **衍生品** | CNH 远期、NDF（不可交割远期）、期权 | 对冲 |
| **存款证** | CNH 存款证 | 银行存款产品 |
| **基金** | 离岸人民币基金 | 资产管理 |
