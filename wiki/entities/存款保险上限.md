# 存款保险上限

> 存款保险制度对单个存款人在同一银行的最高保障金额

## 核心定义

存款保险上限是指存款保险制度对单个存款人在同一银行账户中的最高保障金额。超过此金额的存款在银行破产时不受保护。

## 全球主要存款保险制度

| 国家/地区 | 保险机构 | 保障上限 | 货币 |
|----------|---------|---------|------|
| **美国** | FDIC | $250,000 | 美元 |
| **欧盟** | 各成员国 | €100,000 | 欧元 |
| **英国** | FSCS | £85,000 | 英镑 |
| **日本** | DPDC | ¥1,000万 | 日元 |
| **加拿大** | CDIC | CAD$100,000 | 加元 |
| **中国** | PIS | ¥500,000 | 人民币 |

## 存款保险覆盖范围

| 类型 | 是否覆盖 |
|------|---------|
| 个人储蓄账户 | ✅ 是 |
| 个人支票账户 | ✅ 是 |
| 共同账户 | ✅ 是（每人上限） |
| 企业账户 | ✅ 是 |
| 退休账户（IRA） | ✅ 是 |
| 股票/债券/基金 | ❌ 否 |
| 银行理财/结构性存款 | ❌ 否 |

## 争议观点

### 支持维持现状
- 道德风险：全额担保会增加冒险行为
- 财政负担：提高上限增加政府或有负债
- 市场纪律：未保存款人约束银行风险

### 支持提高上限
- 现实脱节：$25万上限自2008年未变
- 企业需求：现代企业运营需要更高余额
- 系统性风险：未保存款人挤兑会引发系统危机
