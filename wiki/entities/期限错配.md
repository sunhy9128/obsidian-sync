# 期限错配

> 负债短期化与资产长期化之间的风险 — 金融危机的核心诱因之一

**期限错配**（Maturity Mismatch / Asset-Liability Mismatch）是指金融机构或企业的资产（资金运用）与负债（资金来源）的期限结构不匹配。最典型的形式是"**短债长投**"——用短期资金支撑长期资产。

---

## 一、核心机制

### 1.1 银行资产负债表的两大错配

```
银行资产负债表：

资产端（资金运用）          负债端（资金来源）
┌──────────────────┐    ┌──────────────────┐
│ 长期贷款（5-30年）│    │ 活期存款（随时取）│ ← 短期
│ 住房抵押（20-30年）│    │ 定期存款（1-5年）│
│ 企业贷款（3-10年）│    │ 同业拆借（隔夜）  │ ← 最短
│ 长久期债券        │    │ 短期借款          │
│                   │    │ 长期债券（5年+）  │ ← 长期
└──────────────────┘    └──────────────────┘

错配：活期存款（1天）→ 长期贷款（10年）
→ "借短放长"是银行商业模式的核心，也是风险的根本来源
```

### 1.2 期限错配的放大效应

```
正常时期：
- 存款稳定流出 → 可预期、可管理
- 到期存款续存 → 净流出可控
- 银行盈利 → 利差覆盖成本

压力时期（任一触发）：
- 负面舆情 → 存款集中提取
- 同业市场收紧 → 无法续借
- 资产价格下跌 → 追加保证金
        ↓
资金净流出加速 → 必须抛售资产或紧急融资
        ↓
甩卖资产 → 价格下跌 → 更多机构受损（Fire Sale）
        ↓
市场wide流动性枯竭 → 系统性危机
```

## 二、期限错配的三个层次

### 2.1 银行层的期限错配

| 错配类型 | 说明 | 风险 |
|----------|------|------|
| **流动性错配** | 短期负债支撑长期非流动性资产 | 挤兑风险 |
| **再融资错配** | 资产期限 > 负债期限 | 再融资风险 |
| **利率错配** | 资产利率固定、负债利率浮动 | 利率风险 |

**典型案例：硅谷银行（SVB，2023）**

```
负债端（不稳定）：
- 科技创企存款，无息活期
- 2020-2022年存款爆炸式增长（科技融资热潮）
- 2023年科技行业降温 → 存款被大量提取

资产端（长久期 + 低流动性）：
- 持有大量 MBS（抵押贷款支持证券）
- 分类为 HTM（持有至到期）→ 账面价值不反映市价
- 2022年美联储激进加息 → MBS 市场价值暴跌 ~25%
- HTM 浮亏无法在账面上体现，但实际已无法变现

2023.3.10：SVB 宣布融资失败 → 股价暴跌 ~60% → 监管关闭
→ 48小时内成为美国历史上第二大银行倒闭案
```

### 2.2 影子银行层的期限错配

| 影子银行形态 | 错配表现 | 危机触发 |
|-------------|----------|---------|
| **货币市场基金** | 投资者随时赎回 → 投向隔夜/短期CP | 流动性枯竭时大规模赎回 |
| **银行理财** | 3-6个月理财 → 投向3-5年非标/信托 | 净值下跌 → 赎回潮 → 甩卖底层资产 |
| **ABCP 管道** | 发行3-6个月ABCP → 持有1-3年应收账款 | 滚动融资失败 |
| **对冲基金** | 低成本隔夜回购融资 → 持有长期资产 | 去杠杆时被迫清盘 |

**回购市场的期限错配（最危险）**

```
回购（Repo）= 用证券抵押借隔夜资金

正常运作：
Day 1：质押国债，借隔夜资金 1000万
Day 2：用借来的钱买国债，再质押，再借隔夜
→ 每日重复，杠杆可以做到很高

危机时：
- 隔夜资金出借方拒绝续借（对手方风险担忧）
- 雷曼依赖隔夜回购融资 ~$500 亿/天
- 雷曼破产后 → 回购市场利率从 ~2% 飙升至 10%+（2008.9.16）
- 整个市场冻结，所有依赖隔夜融资的机构同时陷入困境
```

### 2.3 企业层的期限错配

| 类型 | 说明 | 风险 |
|------|------|------|
| **短贷长投** | 借短期流动资金贷款投长期项目 | 再融资失败导致资金链断裂 |
| **美元债务错配** | 新兴市场企业借入美元外债投资本币资产 | 本币贬值时偿债成本飙升（1997亚洲危机） |
| **房地产开发商** | 高杠杆 + 预售款（负债）+ 土地/在建工程（资产）| 销售回款放缓 → 资金链断裂 |

## 三、期限错配的量化指标

### 3.1 期限缺口（Gap）分析

```
期限缺口 = 到期资产 - 到期负债

分时间段：
- 1个月缺口
- 3个月缺口
- 6个月缺口
- 1年缺口

示例：
1个月资产到期 = 500亿
1个月负债到期 = 800亿
1个月期限缺口 = -300亿（净流出300亿）

→ 需要通过新增融资或变现资产填补缺口
→ 缺口越大，流动性越脆弱
```

### 3.2 核心比率

| 指标 | 计算 | 监管要求 |
|------|------|---------|
| **存贷比（LDR）** | 贷款/存款 | 中国 ≤ 75%（已废止） |
| **LCR** | HQLA/30天净流出 ≥ 100% | Basel III ≥ 100% |
| **NSFR** | 可用稳定资金/所需稳定资金 ≥ 100% | Basel III ≥ 100% |
| **流动性比率** | 流动性资产/总资产 | 通常 ≥ 25% |

## 四、中国的期限错配特殊性

### 4.1 银行理财的期限错配

```
银行理财产品（2018年前）：
- 募集期：3-6个月
- 投向：非标资产（信托贷款、委托贷款）、长久期债券
- 资产期限：1-5年

结果：资金池运作
- 用新资金偿付到期理财（借新还旧）
- 本质上是"影子银行"模式
- 只要资金持续流入，就不会暴露错配问题
→ 一旦资金流入放缓或逆转，立即爆发流动性危机
```

### 4.2 信托计划的期限错配

```
信托公司：
- 募集资金（1-2年）
- 发放贷款（3-5年）
- 依赖滚动发行维持资金链

→ 与银行类似的期限错配
→ 信托产品违约 → 引发连锁反应
→ 2018年后大量信托产品到期无法兑付
```

## 五、历史案例中的期限错配

| 案例 | 错配核心 | 爆发机制 |
|------|---------|---------|
| **英国养老金危机（2022）** | LDI：短期回购融资投长期国债 | 英国国债收益率飙升 → 保证金追缴 → 抵押品危机 |
| **亚洲金融危机（1997）** | 银行借入短期外币（美元/日元）→ 贷给本国企业（本地货币长期项目） | 本币贬值 → 外币债务飙升 → 银行资产负债表崩溃 |
| **中国 P2P 危机（2018-2020）** | P2P平台短期募资 → 贷给长期借款人 | 备案延期 → 出借人恐慌 → 集中赎回 → 平台清盘 |
| **美国区域银行（2023）** | 科技存款（不稳定）→ 长久期 HTM 债券 | 加息 → 利率风险暴露 → 信心危机 → 挤兑 |

## 六、期限错配与系统性风险

### 6.1 期限错配的传染机制

```
单一机构期限错配：
  流动性压力
        ↓
被迫甩卖资产（Fire Sale）
        ↓
资产价格下跌
        ↓
其他持有同类资产的机构浮亏
        ↓
其他机构也被迫甩卖（连锁反应）
        ↓
系统性流动性枯竭
```

### 6.2 为什么期限错配难以消除

| 原因 | 说明 |
|------|------|
| **商业模式驱动** | 银行利润来自利差，借短放长是必然 |
| **监管套利** | 绕过存贷比、资本充足率等限制 |
| **竞争压力** | 不做错配的银行利差更薄、竞争力更弱 |
| **金融创新** | 影子银行将银行错配问题转移到监管之外 |
| **资本管制** | 发展中国家难以获得长期资金，只能依赖短期外债 |

## 七、应对策略

| 策略 | 方法 | 局限性 |
|------|------|--------|
| **流动性缓冲** | 持有 HQLA（国债等高流动性资产） | 降低收益 |
| **信用额度** | 与其他银行预先签订信贷协议 | 压力时期对手方也可能收紧 |
| **资产证券化** | 将长期资产转出表外 | 降低息差收入 |
| **久期管理** | 缩短资产久期或延长负债久期 | 市场竞争限制 |
| **流动性覆盖率** | Basel III 强制要求银行持有 30 天流动性缓冲 | 监管套利绕过 |
