# 世界银行（World Bank）

> 全球最大的发展融资机构——与 IMF 并称"布雷顿森林双胞胎"

## 一、核心定义

**世界银行**（World Bank）正式名称是 **"国际复兴开发银行"**（International Bank for Reconstruction and Development，IBRD），成立于 **1944 年 7 月**，与 IMF 同时诞生于**布雷顿森林会议**。总部位于美国华盛顿特区。

**核心职责**：
- 向发展中国家提供长期发展贷款
- 减贫与促进共享繁荣
- 提供技术援助与政策建议
- 协调全球发展议程

**世界银行集团**（World Bank Group，WBG）包括五个机构：

| 机构 | 简称 | 成立 | 主要职能 |
|------|------|------|---------|
| **国际复兴开发银行** | IBRD | 1944 | 中等收入国家贷款 |
| **国际开发协会** | IDA | 1960 | 最贫困国家优惠贷款 |
| **国际金融公司** | IFC | 1956 | 私营部门投资 |
| **多边投资担保机构** | MIGA | 1988 | 投资政治风险担保 |
| **国际投资争端解决中心** | ICSID | 1966 | 投资争端仲裁 |

**当前总裁（2026）**：**Ajay Banga**（印度裔美国人，2023 年上任）

## 二、历史演进

### 2.1 创立背景（1944）

```
1944 年布雷顿森林会议：

背景：
  - 二战即将结束
  - 全球重建成为首要任务
  - 国际金融体系需要重组

两个机构诞生：
  - IMF：处理短期国际收支失衡
  - IBRD（世行）：处理长期发展融资

设计差异：
  - IMF：稳定、短期、应急
  - IBRD：发展、长期、基础设施
```

### 2.2 战后重建（1944-1968）

```
1944-1948：创立与启动
  - 1944.7 布雷顿森林会议
  - 1946.6 正式开始运营
  - 首批贷款：法国、丹麦、荷兰、卢森堡

1948-1968：战后重建
  - 重点支持欧洲重建
  - 1968 年达到 100 亿美元贷款规模
  - 主要客户：欧洲、日本、韩国
       ↓
转折点（1948）：
  - 马歇尔计划启动（美国直接援助欧洲）
  - 世行角色转向发展中国家
```

### 2.3 转向发展中国家（1968-1980）

```
1968 Robert McNamara 任行长（1968-1981）：
  - 世行转向"减贫"导向
  - 大幅增加对发展中国家贷款
  - 农村发展、基础设施建设
  - 1972 年 IDA 显著扩张
       ↓
1980-1990：结构调整时代
  - 撒哈拉以南非洲大量贷款
  - "华盛顿共识"影响
  - 结构调整贷款（Structural Adjustment Loans, SAL）
  - 条件性强化（与 IMF 类似）
```

### 2.4 争议与改革（1980-2000）

```
1980s 拉美债务危机：
  - 世行大量贷款给拉美
  - 危机后大量损失
  - "债务可持续性"成为问题

1990s 后苏联转型：
  - 世行贷款支持俄罗斯、东欧转型
  - 结果参差不齐

结构调整的批评（Stiglitz，前世行首席经济学家）：
  - "华盛顿共识"过于激进
  - 忽视社会成本
  - "市场原教旨主义"
```

### 2.5 21 世纪的世行（2000-至今）

```
2000-2012 Wolfowitz/Zoellick 时代：
  - 重点：千年发展目标（MDGs）
  - 增加对非洲贷款
  - 2008 危机后大规模扩张

2012-2019 Jim Yong Kim 时代：
  - 2013 年首次增资
  - 强调"终结极端贫困"
  - 气候变化议程

2019-2023 David Malpass 时代：
  - 疫情应对
  - 暂停最贫困国家债务偿还
  - 气候变化投资

2023- Ajay Banga 时代：
  - 改革议程
  - 增加私营部门合作
  - "更好更大更高效"愿景
```

## 三、组织架构与治理

### 3.1 治理结构

```
世界银行集团治理：

Board of Governors（理事会）
  - 各成员国派一名理事（通常是财长）
  - 每年开年会
  - 重大事项决策机构

Board of Executive Directors（执行董事会）
  - 25 名执行董事
  - 5 大股东（美、日、德、法、英）各 1 名
  - 其他成员国按地区选举产生
  - 每周开例会

行长（President）
  - 惯例由美国总统提名（事实上）
  - 任期 5 年，可连任 1 次
  - 既是主席也是 CEO
       ↓
工作人员：~12,000 人（总部 + 国别办事处）
```

### 3.2 投票权与出资

```
主要成员国投票权（2026）：

| 排名 | 国家 | IBRD 投票权 | 占总股本 |
|------|------|------------|---------|
| 1 | **美国** | 16.7% | 16.7% |
| 2 | **日本** | 8.5% | 8.5% |
| 3 | **中国** | 6.8% | 6.0% |
| 4 | **德国** | 5.3% | 5.3% |
| 5 | **英国** | 4.2% | 4.2% |
| 6 | **法国** | 4.2% | 4.2% |
| 7 | **印度** | 3.1% | 3.1% |
| 8 | **俄罗斯** | 3.0% | 3.0% |
       ↓
关键差别（与 IMF）：
  - IMF 是配额（quota）
  - 世行是认缴股本（subscription）
  - 美国在两个机构都有一票否决权
```

### 3.3 资金来源

```
世行的资金来源：

1. 成员国认缴股本
   - 仅 5% 实缴
   - 95% 是"可调用股本"（股东担保）

2. 国际资本市场借款（最重要）
   - 世行在市场上发行 AAA 级债券
   - 凭借股东担保获得最优利率
   - 2024 年借款规模：~700 亿美元/年

3. IDA 的补充资金
   - IDA 每 3 年补充一次
   - 由捐助国提供
   - 2021 年第 20 次增资：930 亿美元

4. IBRD 净收入
   - 保留利润作为资本
   - 支持贷款能力
```

## 四、贷款机制

### 4.1 IBRD 贷款（中等收入国家）

```
IBRD 特点：

1. 客户
   - 中等收入国家
   - 信用良好（可在国际市场借款）
   - 2024 财年 IBRD 贷款承诺额：~1170 亿美元

2. 贷款条件
   - 期限：15-20 年（含 5 年宽限期）
   - 利率：基于世行自身借款成本
   - 通常低于市场利率（~50-100bp 利差）

3. 贷款类型
   - 发展政策贷款（DPL）：支持政策改革
   - 投资项目贷款（IPF）：支持具体项目
   - 应急贷款：应对危机

4. 关键约束
   - 债务可持续性
   - 借款人信用评级
   - 借款能力（"国家限额"）
```

### 4.2 IDA 贷款（最贫困国家）

```
IDA 特点（"世界银行的良心"）：

1. 客户
   - 75 个最贫困国家
   - 人均 GNI < $1,335（2024 标准）
   - 包括大量非洲国家

2. 优惠条件
   - 期限：38-40 年
   - 宽限期：10 年
   - 利率：0%（赠款成分 > 50%）
   - 部分为赠款（grants）

3. 资金来源
   - 每 3 年增资一次（Replenishment）
   - 由 50+ 捐助国提供
   - 2021 年第 20 次 IDA20：930 亿美元
   - IDA21（2024-2025）：1000 亿美元

4. 2024 财年 IDA 承诺额：~340 亿美元

5. 核心功能
   - 减贫
   - 极端贫困
   - "最后一根稻草"
```

### 4.3 IFC 投资（私营部门）

```
IFC 特点：

1. 客户
   - 发展中国家的私营企业
   - 金融机构
   - 基础设施项目

2. 业务
   - 直接股权投资
   - 贷款
   - 风险管理
   - 咨询

3. 规模
   - 2024 财年承诺：~560 亿美元
   - 投资组合总额：~700 亿美元

4. 关键理念
   - "催化"私营资本
   - 每 1 美元 IFC 投资 → 4-5 美元私营资本
```

### 4.4 MIGA 担保

```
MIGA 特点：

1. 客户
   - 跨境投资者
   - 对发展中国家投资

2. 业务
   - 政治风险担保（征收、战争、汇兑限制）
   - 改善投资环境

3. 规模
   - 2024 财年新担保：~50 亿美元
   - 总担保组合：~250 亿美元
```

## 五、世行 vs IMF：关键对比

### 5.1 核心差异

| 维度 | 世界银行 | IMF |
|------|---------|-----|
| **成立** | 1944 | 1944 |
| **总部** | 华盛顿 | 华盛顿 |
| **主要功能** | 长期发展融资 | 短期国际收支援助 |
| **贷款对象** | 中等收入 + 最贫困国家 | 所有成员国 |
| **贷款期限** | 长期（15-40 年） | 短期（1-5 年） |
| **利率** | 优惠利率 | 市场利率 + 利差 |
| **赠款** | IDA 部分赠款 | 无赠款 |
| **行长惯例** | 美国人 | 欧洲人 |
| **核心使命** | 减贫 | 汇率稳定 |
| **资金来源** | 资本市场借款 | 成员国配额 |

### 5.2 协同与分工

```
IMF 与世行的分工：

| 议题 | IMF | 世行 |
|------|-----|------|
| 短期国际收支危机 | ✓ 主要 | 协调 |
| 长期发展 | 协调 | ✓ 主要 |
| 汇率政策 | ✓ 主要 | 不涉及 |
| 减贫 | 不主要 | ✓ 主要 |
| 基础设施 | 不主要 | ✓ 主要 |
| 金融危机救援 | ✓ 主要 | 协调 |
| 公共债务 | 协调 | ✓ 主要 |
| 教育、健康 | 不主要 | ✓ 主要 |
       ↓
协同机制：
  - 春季会议（联合）
  - 秋季会议（联合）
  - "减贫战略文件"（联合）
  - 危机救援联合行动
```

### 5.3 布雷顿森林双胞胎

```
布雷顿森林体系（1944）的两大支柱：

IMF：
  - 维护汇率稳定
  - 短期国际收支援助
  - "国际货币的守门人"
  
世行：
  - 长期发展融资
  - 减贫
  - "国际发展的引擎"
       ↓
设计逻辑：
  - 短期问题 → IMF
  - 长期问题 → 世行
  - 紧急问题 → IMF
  - 根本问题 → 世行
       ↓
历史演变：
  - 初期：清晰分工
  - 现在：边界模糊
  - 2008 后：合作增加
  - 2020 后：高度协调（COVID 应对）
```

## 六、世行的关键议题

### 6.1 减贫

```
世行的"双目标"（2013 年起）：

1. 终结极端贫困
   - 标准：每日生活费 < $2.15（2022 PPP）
   - 2030 年降至 3% 以下
   - 2024 年现状：~7%（~700 万人）
   
2. 促进共享繁荣
   - 底层 40% 收入增长
   - 关注收入不平等
```

### 6.2 气候变化

```
气候变化投资：

规模：
  - 2021-2025 承诺：~350 亿美元/年
  - 2024 财年气候相关投资：~430 亿美元
  - 占世行总融资的 50%+

重点领域：
  - 可再生能源
  - 气候适应（防洪、抗旱）
  - 减少化石燃料投资（2023 后）
  - 损失与损害基金（2023 COP28）

争议：
  - 仍投资天然气（部分发展中国家需求）
  - 转型路径不清晰
  - 与私营部门合作挑战
```

### 6.3 债务可持续性

```
低收入国家债务危机（2020 后）：

背景：
  - 2020 COVID 后低收入国家债务激增
  - 54 国陷入债务困境或高风险
  - 部分国家违约（赞比亚、乍得、加纳）

世行的 DSSI 框架（2020.5 启动）：
  - 双边债务暂停偿还
  - 但中国未完全参与
  - 2021.11 改为"共同框架"
  - 进展缓慢

世行的新工具：
  - 暂停偿还请求（SDR 通道）
  - 长期债务重组
  - "债务互换"（debt-for-nature, debt-for-climate）
```

### 6.4 中国与世行的关系

```
中国与世行的演变：

1980：恢复世行席位
1990s：成为主要借款国
2010：成为第三大股东
2016：成为 IDA 重要捐助国（首次）
2024：仍是 IBRD 借款国（虽然是中高收入）
       ↓
中国角色变化：
  - 从"借款国"到"混合"角色
  - 既借款也捐助 IDA
  - 推动世行治理改革
       ↓
中国的"竞争者"：
  - AIIB（亚投行）：基础设施
  - NDB（新开发银行）：金砖国家
  - 但中国仍是世行最大借款国之一
```

## 八、世行的批评与争议

### 8.1 结构调整的争议

```
结构调整贷款（SAL，1980-1990s）：

IMF/世行联合推动的"华盛顿共识"：
  - 财政紧缩
  - 私有化
  - 自由化
  - 取消补贴
       ↓
批评（Stiglitz，2002《全球化及其不满》）：
  - 忽视社会成本
  - 损害穷人利益
  - "市场原教旨主义"
  - 拉美失去的十年
  
辩护：
  - 危机国家本就过度借贷
  - 结构性问题不解决，危机必重演
  - 长期看，发展援助确实减贫
```

### 8.2 治理失衡

```
核心问题：
  - 美国 16.7% 投票权（一票否决权）
  - 重大决策需 85% 多数
  - 美国国会经常阻挠增资
  
中国/新兴市场诉求：
  - 增加投票权
  - 改革行长任命惯例
  - 推动更公平治理
  
美国立场：
  - 拒绝削弱否决权
  - 2017 年特朗普否决增资改革
  - 但总体支持世行存在
```

### 8.3 项目质量争议

```
世行项目的常见批评：

1. 大坝项目
   - 1990s 前：大量支持大坝
   - 批评：环境破坏、移民安置
   - 现在：环境标准提升

2. 私有化推动
   - 水、电力私有化（2000s）
   - 批评：水价上涨、穷人负担不起
   - 现在：更谨慎

3. 采购腐败
   - 历史上部分项目存在腐败
   - 现在：加强合规与透明度

4. 债务可持续性
   - 危机前忽略债务可持续性
   - 2018 后引入 DSF/DSA 框架
```

## 九、世行的未来挑战

### 9.1 全球发展议程的变化

```
新议程（2030 SDGs）：

17 个可持续发展目标（SDGs）：
  1. 无贫困
  2. 零饥饿
  3. 健康
  4. 教育
  5. 性别平等
  ...
  17. 全球伙伴关系
  
挑战：
  - 比 MDGs 更宏大
  - 资金需求巨大（数万亿美元）
  - 协调困难
  - 与世行"双目标"的协调
```

### 9.2 气候与发展

```
核心矛盾：

  - 发展中国家需要能源发展（化石燃料）
  - 全球需要减排（停止化石燃料）
  - 世行处于两难

世行的 2023 立场：
  - 停止上游油气项目融资
  - 但保留"特殊情况"
  - 保留天然气过渡角色
  - 增加可再生能源投资
       ↓
争议：
  - 环保组织：做得不够
  - 发展中国家：损害发展权
  - 世行：寻找平衡
```

### 9.3 私营部门角色

```
世行的"催化"战略：

Banga 愿景（2023）：
```

## 十、世行的核心命题

### 10.1 世行成功的三大要素

```
1. 多边性质
   - 190+ 成员国共同所有
   - 中立性
   - 国际公共物品提供者

2. 财政能力
   - AAA 信用评级
   - 国际资本市场融资
   - 每年 700+ 亿美元贷款

3. 专业能力
   - 国别专业知识
   - 项目设计能力
   - 政策建议权威
```

### 10.2 世行的根本局限

```
1. 资源相对有限
   - 年贷款 1000+ 亿美元
   - 相比发展中国家需求（数万亿）微不足道
   - 仅能"催化"，不能"主导"

2. 治理失衡
   - 美国否决权
   - 政策反映美国利益
   - 发展中国家代表性不足

3. 政治化
   - 项目受政治影响
   - 援助分配不公
   - 美国对部分国家施压

4. 时代错位
   - 设计用于战后重建
   - 应对 21 世纪挑战能力有限
   - 改革进展缓慢
```

### 10.3 世行的历史定位

```
世行是 20 世纪最重要的多边发展机构之一：

贡献：
  - 战后欧洲重建
  - 东亚奇迹（韩、日、台）
  - 减贫数百万人
  - 推广发展实践
  
局限：
  - 结构调整争议
  - 治理失衡
  - 改革缓慢
       ↓
理解世行 = 理解多边发展治理的本质
```

## 十二、一句话总结

> **世界银行 = 全球最大的多边发展融资机构 + 减贫的核心平台 + 布雷顿森林双胞胎的"发展支柱"**。
>
> 它与 **IMF** 并称 **"布雷顿森林双胞胎"**——IMF 负责**短期汇率稳定**与**国际收支平衡**，世行负责**长期发展融资**与**减贫**。
>
> **世行的核心优势**：AAA 信用评级 + 国际资本市场融资 + 70+ 年专业经验 + 190+ 成员国支持
>
> **世行的根本局限**：**资源相对有限**（相比发展中国家需求）、**治理失衡**（美国否决权）、**政治化**（反映美国利益）、**改革缓慢**
>
> **理解世行 = 理解多边发展治理的本质**：从战后欧洲重建到东亚奇迹，从拉美结构调整到非洲减贫，从气候变化应对到 SDGs 实现——世行一直是**全球发展议程的核心协调者**和**多边主义的象征**。
>
> **未来 10 年，世行面临的最大挑战是**：**气候变化与发展权的平衡**、**私营资本的调动**、**与 AIIB 等新机构的竞争**、**治理改革的推进**。**世行不是完美的，但它仍是全球发展融资的"最重要机构"——"better than the alternatives"**。
