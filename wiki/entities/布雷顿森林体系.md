# 布雷顿森林体系

> 二战后国际货币秩序

## 核心定义

布雷顿森林体系（Bretton Woods System）是1944-1971年运行的国际货币制度，以美元为核心、美元与黄金挂钩、各国货币与美元挂钩。

## 核心机制

| 要素 | 内容 |
|------|------|
| **汇率制度** | 固定汇率（美元与黄金挂钩，35美元=1盎司黄金） |
| **储备货币** | 美元为核心储备 |
| **调节机制** | 经常项目平衡，资本流动管制 |
| **机构** | IMF + 世界银行 |

## 体系建立

| 时间 | 事件 |
|------|------|
| 1944年7月 | 布林顿森林会议（44国参加） |
| 1945年 | IMF和世界银行成立 |
| 1946年 | 体系正式运行 |

## 主要会议

- **布林顿森林会议**（1944年7月，美国新罕布什尔州）
- 参会国：44个国家
- 主导国：美国（拥有全球2/3黄金储备）
- 英国代表：约翰·梅纳德·凯恩斯
- 美国代表：哈里·怀特

## 核心规则

### 1. 双挂钩

```
美元 ← → 黄金（35美元=1盎司）
其他货币 ← → 美元（固定汇率）
```

### 2. 汇率波动幅度

- 平价 ±1%
- 超过则央行干预

### 3. 资本管制

- 经常项目自由
- 资本项目管制

## 瓦解过程

| 年份 | 事件 |
|------|------|
| 1950s | 美国国际收支恶化 |
| 1960s | 越战+社会福利膨胀 |
| 1961年 | 伦敦黄金池 |
| 1965年 | 黄金危机 |
| 1968年 | 黄金二次危机 |
| 1971年8月 | 尼克松"关闭黄金窗口" |
| 1971年12月 | 史密森协议 |
| 1973年 | 体系正式瓦解 |

## 与特里芬难题

特里芬在1960年预警：
- 美国无法同时满足全球美元储备需求 + 黄金可兑换承诺
- 体系内在矛盾 → 必然崩溃

## 后续影响

- 1974年：石油美元体系建立
- 浮动汇率取代固定汇率
- 黄金非货币化进程
