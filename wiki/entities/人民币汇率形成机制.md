# 人民币汇率形成机制

## 核心定义

人民币汇率形成机制是指中国人民银行管理人民币兑美元及一篮子货币汇率的制度安排和操作框架。自2005年"7·21汇改"以来，人民币汇率制度从"事实上盯住美元"逐步演变为"以市场供求为基础、参考一篮子货币进行调节、有管理的浮动汇率制度"。

## 历史演变

```
1981-1994：双重汇率时期
  ├── 官方牌价 + 外汇调剂市场汇率
  └── 1994年并轨为单一管理浮动汇率（8.7→8.3）
  ↓
1994-2005：事实上盯住美元
  ├── 1997亚洲金融危机后收窄波幅
  └── CNY长期稳定在 8.28 附近
  ↓
2005.7.21：第一次汇改
  ├── 一次性升值 2.1%（8.28→8.11）
  ├── 改为参考一篮子货币的有管理浮动
  └── 每日波幅 ±0.3%（逐步放宽至±2%）
  ↓
2005-2014：单边升值通道
  ├── CNY从 8.11 升至 6.04（累计升值 ~35%）
  └── 央行持续买入外汇，外汇储备从 $8000亿 增至 $4万亿
  ↓
2015.8.11：第二次汇改（"8·11汇改"）
  ├── 调整中间价形成机制：参考前一日收盘价
  ├── 一次性贬值 ~2%
  └── 引发贬值恐慌，资本外流约 $5000亿
  ↓
2016-2017：加强管控
  ├── 引入"逆周期因子"（2017.05）
  ├── 加强资本管制
  └── 消耗外汇储备稳汇率（$4万亿→$3万亿）
  ↓
2018-至今：成熟双向波动
  ├── 2018贸易战 → 贬值破7（2008年以来首次）
  ├── 2020-2021疫情 → 出口推动升值至 6.3
  ├── 2022美联储加息 → 贬至 7.3
  └── 2024-2025中美利差倒挂 → 持续贬值压力
```

## 现行机制（三层架构）

### 第1层：中间价设定（每日开盘锚）

```
中间价 = 前一日收盘价 + 一篮子货币隔夜变动 + 逆周期因子
```

- **定价时间**：每个交易日 9:15（中国外汇交易中心）
- **一篮子货币**：CFETS（中国外汇交易中心）篮子指数，含24种货币，美元权重约18-22%
- **逆周期因子**：2017年引入的**不透明工具**，央行可主动影响中间价，逆市场方向调节
- **目的**：防止顺周期行为（即贬值时中间价走弱加剧贬值预期）

> [!key-insight] 逆周期因子的操作逻辑
> 当市场出现单边贬值预期时，央行通过逆周期因子将中间价设定在强于市场预期的水平，信号效应→抑制贬值预期→引导市场回归理性。

### 第2层：日内波幅限制

- 当前每日浮动限制：CNY/USD **±2%**（2005年 ±0.3% → 2007年 ±0.5% → 2012年 ±1% → 2014年 ±2%）
- CNH（离岸人民币）**无波幅限制**，价格由市场完全决定

### 第3层：直接干预

- **在岸干预**：央行通过国有大行在外汇市场直接买卖美元
- **离岸干预**：央行通过国有大行在离岸市场发行央票回笼CNH流动性，提高做空成本
- **窗口指导**：要求企业"逢高结汇"，银行控制自营交易头寸

## 在岸 vs 离岸人民币（CNY vs CNH）

详见：[[离岸人民币]]（CNH 深度：CNH Hibor、离岸央票、价差驱动、CNH 市场产品）

| 维度 | CNY（在岸） | CNH（离岸） |
|------|------------|------------|
| 交易市场 | 中国银行间外汇市场 | 香港、新加坡、伦敦 |
| 波幅限制 | ±2% | 无限制 |
| 参与者 | 央行授权银行 | 全球各类投资者 |
| 定价机制 | 中间价 + 波幅限制 | 完全市场供求 |
| 管制 | 实需原则（贸易/投资需有真实交易背景） | 自由兑换 |
| 流动时间 | 9:30-23:30 | 24小时 |
| 价差 | 通常 CNH 比 CNY 更弱（贬值预期下） | 反映市场真实预期 |

**CNY-CNH 价差的意义**：
- 价差扩大 → 市场贬值预期强烈
- 价差收窄 → 预期稳定
- 历史极值：2015-2016年 CNH 比 CNY 弱 1000-2000 pips（1-2%）

## 央行的"汇率工具箱"全谱

| 工具 | 作用机制 | 使用频率 |
|------|----------|----------|
| 中间价设定 | 设定每日汇率锚 | 每天 |
| 逆周期因子 | 微调中间价，信号效应 | 单边市场时 |
| 直接外汇买卖 | 通过国有大行交易影响价格 | 必要时 |
| 外汇风险准备金 | 提高远期购汇成本（20%准备金无利息） | 遏制贬值压力 |
| 窗口指导 | 要求银行控制头寸 | 持续 |
| 资本管制加强 | 限制购汇额度、延长审批 | 危机时 |
| 离岸央票 | 收紧CNH流动性，提高做空成本 | 必要时 |
| 外汇储备消耗 | 直接使用储备稳汇率 | 极端情况 |

## 中美利差与汇率传导

```
中美利差 （中国10年期 - 美国10年期）
     ↓
利差为正 → 资本流入 → 人民币升值压力
利差为负 → 资本流出 → 人民币贬值压力
     ↓
2022年以前：中美利差长期为正（中国利率更高）
2022-2025：美联储加息 → 中美利差倒挂 → 贬值压力持续
     ↓
央行应对：宽货币（降息支撑经济）vs 稳汇率（不能降太多）
→ 两难：这是三元悖论在中国的现实体现
```

> [!contradiction] 三元悖论下的中国两难
> 中国选择"独立货币政策 + 有管理汇率 + 有限资本管制"。当美联储激进加息、中美利差深度倒挂时，资本外流压力巨大。央行若降息刺激经济 → 利差进一步扩大 → 贬值压力加大；若加息稳汇率 → 经济下行风险加剧。

## 关键结论

1. **人民币不是完全的市场汇率** — 央行深度参与价格形成，理解人民币汇率必须先理解央行意图
2. **中间价是核心信号** — 中间价的变化方向比即期汇率本身更具指示意义
3. **CNY-CNH 价差是市场情绪的"温度计"** — 价差扩大 = 市场不信任官方定价
4. **中美利差是中期驱动力** — 利差方向决定人民币的升值/贬值大趋势
5. **资本管制是防火墙** — 使中国在三元悖论中维持了政策空间，但同时也积累着套利压力
6. **人民币国际化与汇率稳定的内在矛盾** — 开放资本账户会削弱汇率管控能力
