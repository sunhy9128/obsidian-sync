# 瑞典央行（Sveriges Riksbank）

> **全球最古老的中央银行（1668）+ 全球第一个实施负利率的央行（2009）——央行政策的"双重开创者"**

  - ⭐ "负利率的先驱"

## 二、瑞典央行的核心特点

### 2.1 全球最古老的央行

```
【1668 成立背景】
  - 瑞典帝国衰落时期
  - 为筹集战争资金
  - 由瑞典议会（Riksdag）成立
  - ⭐ 比英格兰银行早 26 年

【1988 央行法改革】
  - 1988.10.1 新《央行法》生效
  - 确立央行独立性
  - 通胀目标制
  - ⭐ 现代化改革

【1999 总部搬迁】
  - 从老城区搬到新址
  - 标志性建筑"夹层楼"
```

### 2.2 通胀目标制（Inflation Targeting）的先驱

```
⭐ Riksbank 是现代通胀目标制的先驱：

【1993 实施通胀目标】
  - 目标：CPI 同比 2%
  - ±1% 容忍区间
  - ⭐ 全球最早实施通胀目标的央行之一
  - 比英格兰银行（1992）晚 1 年，但更系统化

【2017 改为 CPIF 目标】
  - CPIF = 固定利率消费者价格指数
  - 排除抵押贷款利率影响
  - ⭐ 更精确的通胀衡量
```

### 2.3 全球第一个负利率央行

```
⭐ Riksbank 的"全球第一"负利率：

【2009.7.1：DFR 降至 -0.25%】
  - 背景：2008 全球金融危机
  - 瑞典经济受到冲击
  - 通胀预期持续低迷
  - ⭐ 全球首个负利率央行
       ↓
【2010-2014：负利率退出与回归】
  - 2010.7 ：DFR 升至 0.25%（退出负利率）
  - 2014.7 ：DFR 再次降至 -0.10%（回归负利率）
       ↓
【2014-2019：负利率加深】
  - 2015.2 ：DFR -0.10% → -0.35%
  - 2015.7 ：DFR -0.35% → -0.50%
  - 2016.2 ：DFR -0.50% → -0.65%（历史最低）
       ↓
【2019.12：DFR 升至 0%（退出负利率）】
  - 11 年负利率实验结束
       ↓
⭐ 评价：
  - Riksbank 负利率持续 10 年（2009-2010 + 2014-2019）
  - 是全球主要央行中最长的负利率实验
  - 提供了宝贵的"自然实验"数据
```

## 三、瑞典央行的政策框架

### 3.1 政策框架（2026）

```
瑞典央行政策框架：

【1】政策利率：Policy Rate（Repo Rate）
  - 2026.6 Repo Rate：1.75%（从 4% 降息）
  - 调整频率：每 6 周

【2】通胀目标：CPIF 2%
  - ±1% 容忍区间
  - 历史上长期低于 2%

【3】利率走廊：
  - Repo Rate（基准）：1.75%
  - Lending Rate（上限）：2.00%
  - Deposit Rate（下限）：1.50%
  - 走廊宽度：50bp
```

## 四、瑞典央行的负利率实验评价

### 4.1 负利率的"自然实验"

```
⭐ Riksbank 的负利率提供了独特数据：

【实验 1：2009.7-2010.7（短期）**
  - 1 年负利率
  - 效果有限
  - 通胀未达目标

【实验 2：2014.7-2019.12（长期）**
  - 5 年半负利率
  - 持续深度负利率（-0.50%）
  - 房价上涨（瑞典大城市房价 +50%）
  - 通胀仍低于 2%

⭐ 关键发现：
  - 负利率压低长期利率
  - 推升资产价格
  - 通胀传导有限
  - ⭐ 负利率的"边际效应"递减
```

### 4.2 负利率的副作用

```
【副作用 1：房价飙升】
  - 瑞典大城市房价 +50%（2014-2019）
  - 家庭债务 / GDP 升至 190%
  - ⭐ 负利率催生资产泡沫

【副作用 2：银行利润受损】
  - 瑞典主要银行（SEB、Swedbank、Handelsbanken、Nordea）
  - 净息差大幅压缩
  - ⭐ 银行业 ROE 下降

【副作用 3：储蓄者受害】
  - 退休基金、保险基金收益下降
  - 实质上"被惩罚储蓄"
  - ⭐ 不平等加剧
```

## 五、瑞典央行的国际地位

### 5.1 瑞典克朗（SEK）

```
瑞典克朗的国际地位：

【全球外汇交易】
  - SEK 占比：~1.1%
  - 排名第 11 位

【避险货币属性】
  - 弱避险货币（远弱于瑞郎）
  - 但与瑞郎有相似性
  - ⭐ "小瑞郎"

【汇率政策】
  - 1992 退出 ERM 后自由浮动
  - 当前：完全自由浮动
  - ⭐ Riksbank 不设汇率目标
```

### 5.2 瑞典金融体系

```
⭐ 瑞典金融体系的特点：

【1】北欧元区】
  - 瑞典是 EU 成员（1995）
  - 但**不是欧元区**
  - 通过公投保留克朗
  - ⭐ "经济上依赖欧洲，货币上独立"

【2】北欧银行业**
  - 瑞典四大银行：SEB、Swedbank、Handelsbanken、Nordea
  - 都是 G-SIB（全球系统重要性银行）
  - 业务遍及整个北欧
  - ⭐ "北欧银行业巨头"

【3】养老基金**
  - 瑞典养老基金规模庞大
  - AP1-AP7 七大养老基金
  - 资产配置全球化
  - ⭐ "长期资金"
```

## 六、瑞典央行的历史事件

### 6.1 关键事件

```
【1668 成立】
  - 全球第一个央行
  - 比英格兰银行早 26 年

【1992.11 ERM 危机】
  - 瑞典被迫退出 ERM
  - 跟随英国（黑色星期三）
  - 克朗贬值 30%
  - ⭐ 独立货币政策

【1993 通胀目标制】
  - 全球最早实施
  - ⭐ 通胀目标制的先驱

【2009.7 全球首个负利率】
  - DFR -0.25%
  - ⭐ "负利率第一国"

【2019.12 退出负利率】
  - 11 年负利率实验结束
  - ⭐ "Riksbank 退出，ECB 跟随"

【2022-2023 加息周期】
  - 跟随全球加息
  - 2023 Repo Rate 4%（峰值）

【2024 降息周期】
  - 跟随全球降息
  - 当前 1.75%（2026.6）
```

## 十、一句话总结

> **瑞典央行（Riksbank）= 全球最古老的央行（1668）+ 全球第一个负利率央行（2009）——央行政策的"双重开创者"**。
>
> 它比英格兰银行早 26 年成立，比任何其他央行早 5 年实施负利率。**Riksbank 的负利率实验提供了央行政策的"自然实验"数据**——10 年负利率告诉我们：负利率对通胀传导有限，但会推升资产价格、伤害银行利润、惩罚储蓄者。
>
> **理解 Riksbank = 理解四件事**：
> 1. **央行历史**：从 1668 到现代，央行的演化
> 2. **通胀目标制**：Riksbank 是先驱
> 3. **负利率的先驱实验**：2009-2019 年 10 年实验
> 4. **北欧元区的角色**：瑞典不是欧元区但深度参与欧洲经济
>
> Riksbank 代表着央行政策的"开创精神"——**永远走在其他央行前面**。
