# 密特朗（François Mitterrand / 1916-1996）

> 1981-1988 年法国前总统——**"Tournant de la rigueur"（1983 法国"马克投降"）的核心人物**，欧元区诞生的真正起点

## 二、职业生涯

### 2.1 早期政治生涯（1940s-1980）

```
【二战期间（1940-1944）】
  - 1940 被德国战俘
  - 1941 逃脱
  - 加入抵抗运动
  - ⭐ "战时经历"影响政治观

【第四共和时期（1946-1958）】
  - 1946 当选国民议会议员
  - 多个部长职位
  - 反对阿尔及利亚战争
  - ⭐ "左翼反对派"

【戴高乐时代（1958-1981）】
  - 1958-1981 反对戴高乐/蓬皮杜
  - 1971 社会党第一书记
  - 1981 总统候选人
  - ⭐ "漫长的反对派"
```

### 2.2 1981 总统任期：实验与转向

```
⭐ 密特朗的"传奇 14 年"（1981-1995）：

【1981.5 选举获胜】
  - 击败德斯坦（Giscard）
  - 法国第 21 任总统
  - 第一次左翼总统
  - ⭐ "左翼的胜利"

【1981-1983 早期政策：失败实验】
  - 政策：刺激经济 + 提高最低工资 + 35 小时工作制
  - 货币政策：法郎贬值 3 次（1981.10, 1982.6, 1983.3）
  - 财政政策：扩大政府支出
  - ⭐ 1982-1983 通胀飙升到 13%
  - ⭐ 资本外流加剧
       ↓
【1983.3 ："Tournant de la rigueur"（转向）⭐ 历史转折点】
  - 1983.3 第三次法郎贬值
  - 1983.3 资本外流加剧
  - 密特朗任命雅克·德洛尔为财长
  - 实施"严苛紧缩"（Rigueur）
  - 法郎加入 ERM（欧洲汇率机制）
  - ⭐ "法国向德国马克投降"
       ↓
【1983-1995 ：欧洲化政策】
  - 法郎在 ERM 内
  - 跟随德国货币政策
  - 支持欧洲一体化
  - ⭐ "从'戴高乐主义'到'欧洲主义者'的转变"
```

### 2.3 1983 年"马克投降"：关键转折

```
⭐ 1983.3 法国"马克投降"——欧元区诞生的真正起点：

【背景】
  - 1981.5 密特朗当选
  - 实施"反紧缩"政策
  - 法郎贬值 3 次
  - 通胀飙升到 13%
  - 资本外流
  - ⭐ "法国发现'自主货币道路'行不通"

【1983.3 转折点】
  - 1983.3 法郎第三次贬值
  - 政府内部分裂
  - 密特朗被迫任命德洛尔
  - 实施"严苛紧缩"
  - ⭐ "左翼的右转"

【"Tournant de la rigueur"（1983.3）**
  - 取消部分福利承诺
  - 财政紧缩
  - 货币政策转向反通胀
  - 法郎加入 ERM
  - ⭐ "法国接受'德国模式'"

【德洛尔评价】
  - 雅克·德洛尔（财政部长）：
    - "我们向德国'借'了稳定"
  - ⭐ "法国向马克投降"
```

## 三、密特朗的"双重身份"

### 3.1 左翼 vs 欧洲主义者

```
⭐ 密特朗是"矛盾的统一体"：

【身份 1：左翼政治家】
  - 社会党成员
  - 福利国家支持者
  - 关注工人阶级
  - ⭐ "左翼的良心"

【身份 2：欧洲主义者】
  - 推动欧洲一体化
  - 1986 单一欧洲法案
  - 1989 欧洲经济货币联盟
  - ⭐ "戴高乐之后的戴高乐"

【双重身份的张力】
  - 1981：先做左翼实验
  - 1983：失败后转向欧洲化
  - ⭐ "左翼 vs 欧洲化"的张力
       ↓
⭐ 历史评价：
  - 密特朗通过"双重身份"实现了法国转型
  - ⭐ "从'戴高乐主义'到'欧洲一体化'"
  - ⭐ "为欧元铺平了道路"
```

### 3.2 戴高乐的继承者 vs 背叛者

```
【戴高乐时代（1958-1969）】
  - 戴高乐坚持"主权货币"（法郎）
  - 反对美元霸权
  - 1969 戴高乐辞职
  - ⭐ "戴高乐主义"

【蓬皮杜时代（1969-1974）】
  - 继承戴高乐主义
  - 1974 蓬皮杜去世

【德斯坦时代（1974-1981）】
  - 1979 戴高乐主义开始转变
  - 加入 EMS（欧洲货币体系）
  - ⭐ "欧洲化的开始"

【密特朗时代（1981-1995）】
  - 1981 反戴高乐主义实验
  - 1983 失败后"反戴高乐"
  - 推动欧洲一体化
  - ⭐ "戴高乐主义" 终结
       ↓
⭐ 关键判断：
  - 密特朗是"戴高乐主义的终结者"
  - 不是"反戴高乐"，而是"超越戴高乐"
  - ⭐ "欧洲一体化"是 1983 年转向的真正产物
```

## 四、密特朗与德国

### 4.1 1981-1983 的"反德国"实验

```
⭐ 密特朗与德国的关系演变：

【1981-1983 早期："反德国"】
  - 1981 密特朗试图摆脱德国货币政策
  - 实施"独立货币政策"
  - 资本外流到德国
  - ⭐ 法国发现"无法摆脱德国"

【1983 转折："向德国靠拢"】
  - 1983.3 法国"马克投降"
  - 加入 ERM（与马克绑定）
  - 实施德国模式
  - ⭐ "承认德国的领导地位"

【1983-1995 ：法德合作】
  - 密特朗与科尔（Helmut Kohl）合作
  - 推动欧洲一体化
  - 推动欧元
  - ⭐ "法德轴心"建立
```

### 4.2 法德"双引擎"

```
⭐ 密特朗与科尔的"法德轴心"：

【1981-1982 ：冷淡期】
  - 密特朗刚刚当选
  - 实施"左翼"政策
  - 科尔（1982-1998 任总理）谨慎

【1983 ：蜜月期开始】
  - 1983.3 法国"马克投降"
  - 密特朗主动向德国靠拢
  - 科尔回应

【1984 ：第一次峰会】
  - 1984 两国峰会
  - 共同推动欧洲一体化
  - "法德轴心"建立

【1986 ：单一欧洲法案】
  - 1986 单一欧洲法案
  - 推动欧洲单一市场
  - 法德合作的里程碑

【1989 ：马约的"政治意愿"】
  - 1989 密特朗推动马约
  - 1990 科尔配合
  - ⭐ "马斯特里赫特条约"诞生

【1992 ：马约签署】
  - 1992.2 签署马约
  - 法德共同领导
  - ⭐ "欧元区"的法律基础
       ↓
⭐ 关键判断：
  - 1983 是"法德合作"的分水岭
  - 1983 前：法国"反德国"
  - 1983 后：法国"跟随德国"
  - ⭐ "马克投降"创造了"法德轴心"
```

## 八、一句话总结

> **密特朗 = 1983 年法国"马克投降"的关键人物——欧元区诞生的真正起点**。
>
> 他是 1981-1995 年法国总统，1981 年实施"反戴高乐主义"实验失败后，**1983 年转向"严苛紧缩"**，**法郎加入 ERM（与马克绑定）**，**"法国向德国马克投降"**。这次"投降"为 1999 年欧元铺平了道路。
>
> **理解密特朗 = 理解三件事**：
> 1. **"马克投降"创造欧元区**：法国"接受德国模式"是欧洲货币联盟的前提
> 2. **"法德轴心"建立**：密特朗与科尔的合作是欧元区"政治意愿"的基础
> 3. **"核心-边缘"结构**：法国"跟随德国"导致欧元区"南北分歧"
>
> 在中国寻求"中国特色现代金融体系"的今天，**密特朗的故事是"国家选择'跟随强者'还是'自主道路'"的经典案例**——**法国通过"马克投降"获得了'稳定'，但也失去了'主权'**。这是任何"货币合作"都必须面对的根本张力。
