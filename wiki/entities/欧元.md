# 欧元（Euro / EUR / €）

> 人类历史上第二次大规模跨国货币实验——一个"没有国家"的货币，挑战了"货币 = 主权"的传统信条

## 一、核心定义

**欧元**（Euro，符号：€，ISO 4217 代码：**EUR**）是**欧元区**（Eurozone）的**官方货币**，由**欧洲中央银行**（ECB）和**欧元体系**（Eurosystem，含 ECB 与各国 NCB）共同管理。截至 2026 年，欧元是**全球第二大储备货币**、**第二大贸易融资货币**、**第二大债券发行货币**。

**核心数据（2026 年）**：

| 指标 | 数据 |
|------|------|
| 诞生 | 1999 年 1 月 1 日（电子/记账货币），2002 年 1 月 1 日（纸币硬币流通） |
| 欧元区成员国 | 20 个 |
| 欧元区人口 | 约 3.5 亿 |
| 欧元区 GDP | 约 16 万亿美元（占全球 ~14%） |
| 全球外汇储备占比 | ~20%（美元 ~58%） |
| 全球支付占比 | ~23%（美元 ~47%） |
| SDR 篮子权重 | 30%（2016 年人民币加入后调整） |
| ECB 总裁 | Christine Lagarde（2019 至今） |

**欧元的"二元结构"**：

```
1999 年 1 月 1 日：
  - 欧元作为电子/记账货币诞生
  - 11 个创始成员国锁定双边汇率
  - 1 欧元 = 1.1667 德国马克（首次定值）
  - 1 欧元 = 6.55957 法国法郎
  - 1 欧元 = 1936.27 意大利里拉
       ↓
2002 年 1 月 1 日：
  - 纸币和硬币正式流通
  - 双货币流通期（最长 2 个月）
  - 2002 年 2 月 28 日：旧货币退出流通
```

## 二、历史背景：欧洲一体化的"货币维度"

### 2.1 货币统一的"思想根源"

```
1950 年代：
  - 欧洲煤钢共同体（ECSC，1951）
  - 欧洲经济共同体（EEC，1957 罗马条约）
  - 一体化从关税同盟开始
       ↓
1970 年 Werner Plan（沃尔纳计划）：
  - 卢森堡首相 Pierre Werner 主持
  - 计划 1980 年实现经济和货币联盟
  - 因 1971 布雷顿森林崩溃 + 1973 石油危机失败
       ↓
1979 年欧洲货币体系（EMS）：
  - 法国总统德斯坦 + 德国总理施密特推动
  - 创立 ECU（欧洲货币单位）
  - 创立 ERM（欧洲汇率机制）
  - 1992 年被索罗斯击溃（详见 [[wiki/concepts/1992欧洲货币危机]]）
       ↓
1989 年 Delors Report（德洛尔报告）：
  - 欧委会主席 Jacques Delors 主持
  - 三阶段实现货币联盟
  - 奠定马约基础
```

### 2.2 货币统一的"经济动因"

```
为什么欧洲要统一货币？

1. 消除汇率波动的交易成本
   - 1990s 欧洲内部贸易占 60%+
   - 汇率波动损失巨大
   - 货币统一 = 消除汇率风险

2. 增强与美元、日元的抗衡
   - 单一货币 = 单一声音
   - ECB = 美联储级别的对手
   - 欧元挑战美元霸权

3. 锁定低通胀（以德国央行为锚）
   - 德国马克是低通胀的"硬通货"
   - 法国、意大利历史上有高通胀倾向
   - 通过货币统一"绑定"低通胀

4. 推动更深层次一体化
   - 货币是"最敏感"的主权象征
   - 货币统一 = 政治一体化加速
   - 终极目标：欧洲合众国
```

### 2.3 货币统一的"政治动因"

```
德国统一推动货币统一（1989-1990）：

柏林墙倒塌（1989.11.9）
       ↓
德国统一谈判（1990）
       ↓
法国总统密特朗的"交换"：
  - 德国统一 = 法国同意放弃法郎
  - "我们用一个货币换你们的统一"
  - 密特朗担忧德国统一后欧洲被德国主导
  - 用货币联盟"绑定"德国
       ↓
这一"政治交易"是欧元诞生的关键
- 没有德国统一，可能没有欧元
- 没有密特朗，可能没有欧元
```

## 三、各国谈判：《马斯特里赫特条约》博弈

### 3.1 条约谈判（1990-1992）

```
马斯特里赫特谈判的核心争议：

德国（Bundesbank 主导）：
  - 严守"独立央行 + 价格稳定"
  - 反对任何形式的货币联盟"政治控制"
  - 要求"趋同标准"严格
  - 拒绝为单一货币融资（无国债货币化）
       ↓
法国（密特朗主导）：
  - 主张经济政策"协调"
  - 担忧德国主导
  - 想要"经济政府"与央行并列
  - 最终妥协：接受央行独立，换取政治一体化
       ↓
英国（撒切尔 → 梅杰）：
  - 反对放弃英镑
  - 拒绝加入单一货币
  - 获得"退出条款"（Opt-out）
  - "英镑是英国身份的一部分"
       ↓
意大利、西班牙、葡萄牙、希腊、爱尔兰：
  - 担心"被德国模式锁定"
  - 想要更多财政灵活性
  - 最终全部签署条约
       ↓
丹麦：
  - 1992.6 全民公投否决条约
  - 后通过"爱丁堡协议"获得例外
  - 至今保留克朗
```

### 3.2 趋同标准（Convergence Criteria）

**马约规定加入欧元区必须满足 5 项标准**：

| 标准 | 要求 | 内涵 |
|------|------|------|
| **通胀率** | 不超过 3 个最低成员国平均通胀率的 1.5% | 价格稳定承诺 |
| **财政赤字** | 不超过 GDP 的 3% | 财政纪律 |
| **政府债务** | 不超过 GDP 的 60% | 债务可持续 |
| **汇率** | 加入 ERM II 至少 2 年未贬值 | 汇率稳定 |
| **长期利率** | 不超过 3 个最低通胀国平均利率的 2% | 金融市场认可 |

> **核心矛盾**：这些标准最初设计为"硬约束"，但实际执行中**多次被违反**（法国、德国 2003-2004 超过 3% 赤字；意大利、希腊从未达到 60% 债务标准；2004 加入的东欧国家普遍不达标）。

### 3.3 各国公投与"民主赤字"

```
马约批准过程（1992）：

1992.6.2 丹麦公投：50.7% 否决
  - 引发"马约危机"
  - 丹麦获得"例外条款"
       ↓
1992.9.20 法国公投：51.05% 勉强通过
  - 仅领先 100 万票
  - 反映民众高度疑虑
       ↓
1993.5.18 克罗地亚...（无关）
       ↓
英国议会批准（无需公投，但执政党分裂）
       ↓
1993.11.1 条约正式生效
       ↓
后续：
  - 1994 奥地利、芬兰、瑞典加入（瑞典公投否决，保留克朗）
  - 1998 ECB 成立
  - 1999.1.1 欧元诞生（11 国）
```

## 四、欧元的诞生与扩展

### 4.1 创世时刻（1999.1.1）

```
1999 年 1 月 1 日午夜：

11 个创始成员国锁定汇率
  奥地利、比利时、芬兰、法国、德国、爱尔兰、意大利、卢森堡、
  荷兰、葡萄牙、西班牙
       ↓
1 ECU = 1 欧元
  1 欧元 = 1.1667 德国马克
  1 欧元 = 6.55957 法国法郎
  1 欧元 = 1936.27 意大利里拉
  1 欧元 = 166.386 西班牙比塞塔
  ...
       ↓
欧洲中央银行（ECB）启动
  - 总部：德国法兰克福
  - 第一任总裁：Wim Duisenberg（荷兰人）
  - 政策利率：3%
       ↓
欧元在国际市场立即成为第二大货币
```

### 4.2 纸币硬币流通（2002.1.1）

```
"Big Bang" - 2002.1.1 午夜：

12 国同时启动纸币硬币流通
  （希腊晚 1 年加入，2001）
       ↓
双货币流通期：
  - 最长 2 个月（部分国家 1 个月）
  - 2002.2.28 旧货币退出流通
       ↓
物流奇迹：
  - 150 亿张纸币
  - 500 亿枚硬币
  - 价值 ~700 亿欧元
  - 卡车运输距离可绕地球 4 圈
       ↓
心理冲击：
  - 欧洲人第一次用"外国"纸币
  - 跨国旅行无需兑换
  - 货币认同的"实体化"
```

### 4.3 扩展与退出

```
欧元区扩展时间线：

1999：11 国创始
2001：希腊（2001.1.1，2002 纸币流通）
2007：斯洛文尼亚
2008：塞浦路斯、马耳他
2009：斯洛伐克
2011：爱沙尼亚
2014：拉脱维亚
2015：立陶宛
2023：克罗地亚
       ↓
至今（2026）：20 个成员国
       ↓
未加入欧盟国家：
  - 英国：永久 Opt-out（2016 公投脱欧）
  - 丹麦：永久 Opt-out
  - 瑞典：技术性未加入（公投否决）
       ↓
未加入欧元区的 EU 国家：
  - 保加利亚、捷克、匈牙利、波兰、罗马尼亚、瑞典
```

### 4.4 退出风险：希腊与"Grexit"

```
2012 年希腊危机：
  - 希腊债务占 GDP 175%+
  - 失业率 25%+
  - 讨论希腊退出欧元区（Grexit）
       ↓
退出成本的"恐怖计算"：
  - 银行系统崩溃
  - 新货币大幅贬值
  - 恶性通胀
  - 资本管制
  - 政治动乱
  - 传染到葡萄牙、西班牙、意大利
       ↓
最终选择：救助 + 留在欧元区
  - 三轮救助共 ~2890 亿欧元
  - 实施严厉紧缩
  - ECB 承诺"不惜一切代价"保欧元
  - 2018 年希腊退出救助计划
```

## 五、ECB 与货币政策

### 5.1 ECB 的"独立性"原则

```
欧洲中央银行（ECB）的独立性：

1. 制度独立
   - 不受欧盟理事会、欧委会指令
   - 章程规定禁止"政治干预"
   - 类比：德国 Bundesbank 模式

2. 人员独立
   - 执行委员会任期 8 年（不可连任）
   - 各国 NCB 行长任期至少 5 年
   - 解职需欧洲法院判决

3. 财务独立
   - ECB 自有资本（各 NCB 认购）
   - 不依赖成员国拨款
   - 禁止为政府债务货币化

4. 工具独立
   - 自主决定利率
   - 自主决定非传统工具（OMT、APP、PEPP、TLTRO）
```

### 5.2 货币政策工具

```
ECB 工具箱：

1. 传统工具
   - 主要再融资操作（MRO）：基准利率
   - 边际贷款便利：隔夜利率上限
   - 存款便利：隔夜利率下限

2. 非传统工具（危机后）
   - 资产购买计划（APP，2015-2018）：QE
   - 疫情紧急购买计划（PEPP，2020-2022）：~1.85 万亿欧元
   - 定向长期再融资操作（TLTRO）：向银行提供低成本贷款
   - OMT（2012）：无条件购买主权债务（仅"宣告"未实际使用）

3. 危机工具
   - 流动性覆盖（LCR）：2014
   - 单一监管机制（SSM）：2014
   - 单一决议机制（SRM）：2015
```

### 5.3 通胀目标与争议

```
ECB 通胀目标：

定义：HICP（调和消费者物价指数）"低于但接近 2%"
  - 2021 年 7 月明确为 2% 对称目标
  - 容忍偏离 ±1%
       ↓
历史表现：
  - 1999-2007：1.9%-2.4%（基本达标）
  - 2008-2019：0%-1.5%（长期低于目标）
  - 2020-2022：COVID 后通胀飙升 8-10%
  - 2023-2025：紧缩后回落到 2% 附近
       ↓
争议：
  - 2% 目标是否合适？
  - 是否需要"平均通胀目标"（类似美联储）
  - 房价不应纳入通胀？
  - 数字欧元是否影响传导
```

## 六、财政维度：欧洲财政一体化的"未完成拼图"

### 6.1 《稳定与增长公约》（SGP）

```
SGP 核心规则（1997 制定）：

1. 财政赤字 ≤ 3% GDP
2. 政府债务 ≤ 60% GDP（每年减债 1/20）
3. 过度赤字程序（EDP）
4. 制裁：最高 0.5% GDP 罚款
       ↓
执行问题：
  - 2003-2004 法德违反规则
  - 欧盟理事会拒绝制裁
  - "政治妥协"压倒规则
  - 规则被视为"软约束"
       ↓
改革（2005、2011、2024）：
  - 2011 "六条规则"（Six Pack）
  - 2013 "财政契约"（TSCG）
  - 2024 全面改革：中期路径 + 国别净支出路径
```

### 6.2 "无救助"条款（No Bailout Clause）

```
马约第 125 条：

"成员国不对欧盟其他成员国政府或机构的债务承担连带责任，
 也不为这些债务提供资金支持"
       ↓
立法目的：
  - 防止"道德风险"
  - 强迫各国财政自律
  - 模仿美国"州政府不能印钞"原则
       ↓
现实违反：
  - 2010 希腊救助：1100 亿欧元
  - 2012 ESM 永久机制：5000 亿欧元
  - 2012 银行联盟（直接资本重组 ESM）
  - 2020 SURE（疫情失业援助）：1000 亿欧元
  - 2020 NextGenerationEU（复苏基金）：~8000 亿欧元
       ↓
"无救助"条款事实上已被"系统性违反"
但欧盟坚持"这不是救助，是贷款"
```

### 6.3 欧洲稳定机制（ESM）

```
ESM 关键事实：

成立：2012 年 10 月（取代临时 EFSF）
总部：卢森堡
资本：7048 亿欧元（实缴 ~800 亿）
最大贷款能力：5000 亿欧元
成员国：19 国（除德国外所有欧元区国家）
       ↓
ESM 工具：
  1. 贷款
  2. 一级市场购买（政府债）
  3. 二级市场购买（政府债）
  4. 预防性信贷额度（PCCL、ECCL）
  5. 银行直接资本重组（2014 起）
  6. ESM 债券（评级 AAA）
       ↓
条件性：
  - 严格紧缩 + 结构性改革
  - "私人部门参与"（PSI，2012 希腊减记 53%）
  - 财政"监督"权（与欧委会联合）
  - 触发"宏观调整计划"
```

### 6.4 NextGenerationEU（NGEU）—— 财政联盟的"突破"

```
NGEU 关键事实：

启动：2020 年 5 月
规模：~8069 亿欧元（按 2018 价格 7500 亿）
  - 复苏与韧性机制（RRF）：~7240 亿（贷款 3850 + 拨款 3390）
  - 其他：~830 亿
       ↓
革命性创新：
  - 欧盟首次大规模"共同借款"
  - 欧盟委员会代表 EU 发行债券
  - 拨款（不需要偿还）形式资助南欧
  - "无救助条款"事实上被改写
       ↓
资金分配：
  - 意大利：~1916 亿（最大受益者）
  - 西班牙：~1400 亿
  - 法国：~395 亿
  - 德国：~258 亿
  - 波兰、匈牙利：~1580 亿（争议）
       ↓
争议：
  - 德国宪法法院判决合法（2020）
  - 北欧"净付款人"反对
  - 匈牙利否决：欧委会冻结资金
  - 2026 实施评估：成果参差不齐
```

## 七、欧元的国际地位

### 7.1 储备货币

```
全球外汇储备构成（2024-2026）：

美元：~58%
欧元：~20%
日元：~5.5%
英镑：~5%
人民币：~2.5%
其他：~9%
       ↓
欧元的历史地位变化：
  - 1999：18%（诞生即超越马克）
  - 2009：27.7%（金融危机高峰）
  - 2014：24%（ECB 紧缩预期）
  - 2020：24%
  - 2025：20%
       ↓
为何未挑战美元？
  - 欧元区无统一财政部
  - 主权债务分散
  - "安全资产"不足（safe asset shortage）
  - 美国资本市场更深
```

### 7.2 贸易与支付

```
欧元国际支付（SWIFT 数据，2025-2026）：

美元：~47%
欧元：~23%
英镑：~7%
人民币：~3-4%
日元：~3%
       ↓
欧元贸易融资：
  - 欧元区对外贸易 ~50% 用欧元
  - 能源贸易欧元化趋势下降（俄乌后）
  - 大宗商品仍以美元为主
```

### 7.3 资本市场的"欧元短板"

```
"安全资产短缺"（Safe Asset Shortage）：

问题：
  - 美国国债 = 单一 AAA 级安全资产
  - 欧元区无对等资产
  - 22 个分散的成员国国债
  - "碎片化"阻碍货币传导
       ↓
解决方案争议：
  1. 欧元区共同债券（Eurobonds）
  2. "蓝红"债券模型（BTP-Bund 混合）
  3. ESM 升级为"欧洲货币基金"
  4. NGEU 模式常态化
  5. 数字欧元作为新"无风险资产"
       ↓
2025 现状：
  - NGEU 模式"软"常态化
  - 但没有"硬"共同债券
  - "财政联盟"是未完成的拼图
```

## 八、欧元区的根本矛盾

### 8.1 "三元悖论"在欧元区的体现

```
蒙代尔-弗莱明三元悖论在欧元区：

欧元区集体：
  - ✅ 共同汇率（欧元对外部）
  - ✅ 资本自由流动
  - ❌ 独立货币政策（已放弃）
       ↓
但欧元区内部：
  - ❌ 共同汇率（成员国之间）
  - ❌ 独立货币政策（各 NCB 无政策主权）
  - ✅ 资本自由流动
  - 这违反了"稳定"原则
       ↓
根本问题：
  - 没有财政联盟 + 银行联盟的货币联盟
  - 必然引发"南北矛盾"
  - "最优货币区"标准未满足
```

### 8.2 "南北矛盾"

```
欧元区内部经济分化：

核心国（德国、荷兰、奥地利、芬兰）：
  - 制造业出口
  - 经常账户顺差
  - 低通胀
  - 紧缩财政
       ↓
边缘国（意大利、西班牙、葡萄牙、希腊）：
  - 服务业 + 旅游业
  - 经常账户逆差
  - 高通胀（曾）
  - 财政扩张倾向
       ↓
危机传导机制：
  核心国低利率 → 边缘国资本流入
       → 边缘国房地产泡沫 + 债务
       → 危机爆发 → 核心国"切断"资金
       → 边缘国被迫紧缩 + 衰退
       ↓
德国 2010s 经常账户顺差 8-9% GDP
= 边缘国对应逆差 8-9% GDP
= "出口德国的紧缩"（Bernd Lucke 批评）
```

### 8.3 民主赤字

```
欧元的"民主赤字"问题：

1. ECB 不对任何民选机构负责
   - 执行委员会由欧洲理事会任命
   - 议会只有"质询"权
   - 货币政策无问责

2. 财政规则由"专家"制定
   - 3% 赤字、60% 债务 = 任意数字
   - 缺乏民主合法性

3. 危机决策绕过成员国议会
   - 2012 ESM 条约由德国宪法法院"通过"
   - 2015 希腊救助"违背公投"（虽然希腊接受）
   - NGEU 8000 亿欧元共同债务

4. 知识精英主导
   - 货币主义 + 财政正统派
   - 与南欧民众价值观脱节
   - 民族主义反弹的温床
```

## 十、欧元的历史意义

### 10.1 货币理论突破

```
欧元对货币理论的贡献：

1. "没有国家的货币"
   - 挑战"货币 = 主权"传统
   - 货币联盟 = 主权让渡
   - "最优货币区"理论的实践

2. 央行独立性原则
   - 强化了央行政治独立的理念
   - 影响多国央行立法
   - 通胀目标制扩散

3. 货币联盟与财政联盟的张力
   - 揭示"货币联盟必须伴随财政联盟"
   - 否则必然不平稳
   - 欧元区是"半成品"货币联盟

4. 国际货币体系多极化
   - 美元 - 欧元 - 人民币三角
   - 货币储备多元化
   - 支付体系多元化
```

### 10.2 政治影响

```
欧元的政治遗产：

1. 加速了欧盟政治一体化
   - 货币联盟 = 不可逆
   - 退出成本巨大
   - 推动共同外交 + 防务

2. 改变了欧洲政治版图
   - "南北矛盾"取代"东西矛盾"
   - 民族主义政党兴起
   - 疑欧主义扩散

3. 与美国的关系重塑
   - 欧元区成为美国"对手 + 盟友"
   - 贸易争端 + 安全合作
   - 数字监管 + 国防协同

4. 中欧关系新维度
   - 人民币 - 欧元直接交易
   - 投资协定谈判
   - 应对美国制裁的"灰色地带"
```

### 10.3 一句话总结

> **欧元 = 1999 年诞生的超主权货币 = 20 个国家 3.5 亿人共同使用 = 全球第二大储备货币 = 但仍是"未完成的货币联盟"**。
>
> 它是人类历史上**第二次大规模跨国货币实验**（第一次是 19 世纪拉丁货币联盟），**挑战了"货币 = 主权"的传统信条**，**催生了 ECB 这一新型央行模式**。但它也是**矛盾的**：
>
> - **没有财政联盟的货币联盟**：2009-2012 年欧债危机充分暴露
> - **没有政治联盟的货币联盟**：民主赤字、民族反弹
> - **没有"安全资产"的货币联盟**：无法挑战美元霸权
> - **没有最优货币区的货币联盟**：南北分化、增长分化
>
> **理解欧元 = 理解 21 世纪多极货币体系的核心节点**。它的命运取决于欧洲能否在"深度一体化"与"民族反弹"之间找到平衡。**未来 10 年，欧元面临的最大挑战是竞争力下降、政治极化、数字货币兴起与地缘分裂。**
