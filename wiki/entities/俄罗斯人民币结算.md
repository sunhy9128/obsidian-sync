# 俄罗斯人民币结算

> 2022年俄乌冲突后，在西方全面金融制裁下，中俄能源贸易以人民币为核心结算货币的实战进程——从 <5% 到 >50% 的历史性突破

## 基本信息

| 项目 | 内容 |
|------|------|
| 全称 | Russian Energy RMB Settlement |
| 本质 | 中俄能源贸易（石油、天然气、煤炭）以人民币计价和清算的制度安排 |
| 关键转折点 | 2022年俄乌冲突 |
| 核心基础设施 | CIPS + SPFS 协同 |
| 当前状态 | 中俄贸易 90%+ 用人民币结算 |

---

## 一、制裁前的起点：美元惯性强大

### 1.1 2022年之前的中俄能源结算结构

```
【战前结构】
├── 中俄能源贸易美元结算占比：90%+
├── 俄罗斯央行人民币储备：<0.1%
├── 人民币在俄罗斯几乎无流通
└── CIPS 已有但不活跃

【为什么美元惯性强大？】
├── 美元是全球贸易标准语言
├── 俄罗斯金融体系深度融入美元体系
├── 中国企业同样习惯美元结算
└── 双方有本币结算意愿，但缺乏倒逼压力
```

---

## 二、制裁冲击：生存压力下的被迫转型

### 2.1 2022年制裁三阶段压力测试

> 来源：[[entities/SPFS]] §七

```
【Phase 1：金融核弹（2022.3-6）】
├── 7家俄主要银行被切 SWIFT
├── $3000亿外汇储备被冻结（共$6400亿）
├── 卢布暴跌 150+
└── 俄罗斯被迫寻找美元替代通道

【Phase 2：能源反制（2022.4-12）】
├── 卢布结算令（4月）：不友好国家须用卢布买俄气
├── SPFS 从"备胎"转正
├── 欧洲拒绝 → 份额转向中国
└── 实质演变为卢布 + 人民币双轨

【Phase 3：长期脱钩（2023-至今）】
└── 中俄贸易 90%+ 用人民币结算
```

### 2.2 卢布结算令：普京的天然气武器化

```
【卢布结算令设计】
├── "不友好国家"须在 Gazprombank 开设卢布账户
├── 买方将欧元/美元换成卢布，通过 SPFS 支付
└── 俄气以卢布计价和结算

【实际执行结果】
├── 匈牙利、奥地利等部分欧盟成员接受
├── 大多数欧盟国家拒绝，转向其他渠道
└── 中国接过大部分份额

【战略意涵】
强制货币兑换 = 买方必须持有卢布资产
= 向俄罗斯央行提供无息贷款
```

## 三、实战路径：中俄能源人民币结算机制

### 3.1 典型清算路径

```
【中俄天然气结算路径】

中国进口商（人民币）
  → 中国某银行（CIPS 直接参与者）
    → CIPS 系统（人民币跨境清算 + 报文）
      → 俄罗斯某银行（CIPS 间接参与者 + SPFS 接入）
        → SPFS（卢布境内清算）
          → Gazprom（俄气）

【中俄石油结算路径】（类似但更直接）

中国进口商（人民币）
  → 中国某银行
    → CIPS 系统
      → 俄罗斯石油公司账户
        → 俄罗斯央行卢布清算
```

### 3.2 关键节点：Gazprombank

```
Gazprombank（俄罗斯天然气工业银行）：
├── 唯一被允许处理卢布结算令资金流的主要银行
├── 2022年后成为 CIPS 间接参与者
└── 同时接入 SPFS + CIPS，实现"双轨清算"
```

### 3.3 关键数据对比

| 维度 | 2021年 | 2024-2025年 | 变化 |
|------|--------|-------------|------|
| 中俄能源贸易人民币结算占比 | <5% | **>50%** | +45pp |
| 俄罗斯央行人民币储备占比 | 0.1% | **30%+** | +30pp |
| 莫斯科交易所人民币交易量 | 很小 | **全球前三** | 质变 |
| CIPS 俄罗斯参与银行数 | 零星 | **大幅增加** | 成体系 |

## 四、为什么成功：不可复制的结构性优势

### 4.1 生存压力 vs 意识形态

```
【俄罗斯的决策逻辑：生存压力】
├── 2022年被切 SWIFT → 必须找到替代通道
├── $3000亿储备被冻结 → 美元资产不再安全
├── 欧洲市场丧失 → 需要新的能源买家
└── 普京必须成功（没有退路）

【伊朗的决策逻辑：意识形态】
├── 内贾德"反美"政治立场
├── 主动宣布欧元计价
└── 缺乏生存压力逼迫欧洲配合
```

### 4.2 中国的独特优势：不受二级制裁约束

| 条件 | 伊朗遇到的情况 | 俄罗斯（中国）遇到的情况 |
|------|--------------|----------------------|
| 二级制裁风险 | 欧盟企业面临失去美国市场 | 中国企业不受美国市场威胁 |
| 替代市场 | 欧洲可从沙特等替代 | **中国是唯一大规模替代买家** |
| 基础设施 | 无 | **CIPS + SPFS 已具备对接条件** |
| 政治意志 | 欧盟内部分裂 | **中俄高度战略协同** |

### 4.3 中俄能源互补的"锁死效应"

```
【为什么中俄能源人民币结算不可替代？】

俄罗斯需要：
├── 能源出口市场（欧洲已失）
└── 不被冻结的结算货币

中国需要：
├── 能源安全（多元化进口）
└── 人民币国际化的实物支撑

【锁死结构】
俄罗斯能源 + 人民币结算
  → 俄罗斯获人民币 → 购买中国商品
  → 中国获稳定能源 → 用人民币支付
  → 两国经济高度互补，无需美元中介
```

### 4.4 与伊朗的根本差异

| 对比维度 | 伊朗欧元计价（失败）| 俄罗斯人民币计价（成功）|
|---------|-------------------|---------------------|
| 替代货币 | 欧元（体系内）| 人民币（体系外）|
| 触发原因 | 主动挑战（意识形态）| **被动应对（生存压力）** |
| 政治意志 | 有限 | **极强** |
| 经济替代 | 缺乏内循环市场 | **中国14亿人口市场** |
| 基础设施 | 无 | **CIPS + SPFS 已就绪** |
| 结算货币可接受性 | 欧元受美国影响 | **人民币不受美国约束** |
| 二级制裁有效性 | 高（欧洲企业被迫退出）| **低（中国不吃这套）** |
| 结果 | 石油出口暴跌 | **能源收入基本维持** |

## 五、对石油美元体系的冲击

### 5.1 打破"石油→美元→美债"闭环

```
石油美元环流的四环节（来源：[[sources/石油美元体系]]）：
[石油出口] → [美元储备] → [投资美债] → [美元回流]

俄罗斯转向人民币：
├── 打破"石油→美元→美债"的闭环
├── 美元失去一个主要能源贸易对手方
└── 人民币向"能源计价货币"迈出关键一步
```

### 5.2 象征意义

> 石油美元体系1974年由沙特协议奠定。俄罗斯作为第二大石油出口国（2022年前），开始用人民币结算——这是该体系成立以来**最实质性的冲击之一**。

## 六、局限性与结构性约束

### 6.1 当前局限

| 约束 | 说明 |
|------|------|
| 美元清算仍依赖 CHIPS | 多数国家仍以美元为锚，真正的"独立性"有限 |
| 俄罗斯内循环不够 | 人民币在俄流通仍依赖中俄双边，非全球性 |
| 规模仍小 | CIPS 日均 $800亿，仅为 CHIPS 的 ~1/20 |
| 政治风险溢价 | 第三方国家担忧接入中国的政治代价 |

### 6.2 中期判断

**从 0 到 1 的突破已实现，但从 1 到 100（真正替代石油美元）仍需时日。**

## 七、核心洞察

> [!key-insight] "体系外替代"的可行性
>
> 俄罗斯人民币结算的成功，证明了**用独立于美元体系的经济体的货币进行能源计价，是可行的**。关键前提：该经济体必须不受美国二级制裁约束，且与能源出口国形成高度互补的贸易结构。

> [!key-insight] 制裁催生替代系统
>
> 美国对俄罗斯的 SWIFT 制裁 → SPFS 加速建设 → 人民币大规模替代 → 美元在俄罗斯彻底边缘化 → 美元武器化能力下降。**这是"金融核武器"无法频繁使用的根本原因。**

> [!key-insight] 互补性是结算货币的核心支撑
>
> 伊朗用欧元结算失败，因为伊朗持有欧元无法购买欧洲商品（制裁）。俄罗斯用人民币结算成功，因为俄罗斯持有人民币可以购买中国商品。**结算货币的可接受性，取决于背后实体经济的互补深度。**

> [!contradiction] 能源出口国的"双重依赖"
>
> 能源出口国（伊朗/俄罗斯）天然依赖全球贸易体系，因此极易受到制裁影响。唯一出路是找到"不依赖美国主导体系"的替代市场——而中国的规模恰好提供了这个出口。
