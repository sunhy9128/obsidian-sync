# 日元套利交易

> 全球规模最大、影响最广的套利交易 — 借入日元买全球

**日元套利交易**（Yen Carry Trade / Carry Trade）是全球外汇市场最重要的结构性交易之一。投资者借入极低利率的日元，换成高利率货币（如美元、澳元、巴西雷亚尔），投资于高收益资产赚取利差。

---

## 一、基本机制

### 1.1 操作流程

```
Step 1：借入日元
        → 成本：接近 0%（2016-2024 年日本负利率期间甚至为负）

Step 2：兑换为高利率货币（USD、AUD、BRL 等）
        → 汇率对冲（可选择）或承担汇率风险

Step 3：投资于高收益资产
        → 美股（标普 500）、新兴市场股票、
        → 澳元/巴西雷亚尔国债、
        → 黄金、
        → 其他高息货币计价的资产

Step 4：持有期间赚取利差
        年利差收益 = 高息资产收益率 - 日元融资成本

Step 5：平仓（解仓）
        → 卖出高息资产 → 换回日元 → 归还借款
        → 或通过外汇对冲锁定利差
```

### 1.2 利差规模的计算

| 高息货币 | 典型利率 | 日元融资成本 | 利差（估算） |
|----------|----------|-------------|-------------|
| 美元 | 4.5-5.5%（2024 年） | 0-0.1% | ~4-5% |
| 澳元 | 4.0-5.0% | 0% | ~4-5% |
| 巴西雷亚尔 | 10-13% | 0% | ~10-13% |
| 南非兰特 | 8-11% | 0% | ~8-11% |

> **实际利差 vs 名义利差**：上述是名义利差。扣除外汇对冲成本（借入日元换美元通常需用 FX swap 对冲，对冲成本取决于美元利率 - 日元利率的利差）后，**净利差约为名义利差的 30-60%**。

### 1.3 全球规模估算

| 时期 | 估算规模 | 说明 |
|------|---------|------|
| 2000 年代初 | ~1 万亿美元 | 克林顿时代日元国际化背景 |
| 2007 年（危机前） | ~1-1.5 万亿美元 | 高峰期 |
| 2013-2019 | ~0.5-1 万亿美元 | 安倍经济学导致日元贬值预期减少 |
| 2024 年（加息前） | ~0.5-1 万亿美元 | 日本负利率政策延续 |
| 2024 年 7-8 月日元升值后 | 部分解仓，规模下降 | 规模难以精确量化（统计口径不同） |

## 二、日元为何成为套利货币

### 2.1 日本利率环境演变

| 时期 | 日本政策利率 | 日元利率 | 套利吸引力 |
|------|------------|---------|-----------|
| 1991-1998 | 高→逐步降息 | 逐步降至 0.5% | 中等 |
| 1999-2000 | 零利率政策 | ~0% | 开启套利时代 |
| 2001-2006 | 量化宽松（QE） | ~0% | 套利大扩张 |
| 2006-2008 | 加息尝试 | 0.5% | 套利收敛 |
| 2010-2016 | 再宽松 | ~0% | 套利恢复 |
| 2016-2024.3 | 负利率政策（-0.1%） | 极低 | **套利最盛期** |
| 2024.3 起 | 逐步加息 | 0.1%→0.25%→0.5%→0.75%（2026 初） | 套利成本上升 |

### 2.2 日本成为套利货币的三重原因

| 原因 | 说明 |
|------|------|
| **长期低利率** | 日本银行间拆借利率长期接近零，融资成本极低 |
| **日元避险属性** | 日元是传统避险货币——危机时日元升值→套利者被迫平仓→日元进一步升值 |
| **日元持续贬值趋势（2000-2012）** | 借入日元做套利，不仅赚利差，还能额外获得日元贬值带来的汇率收益 |

> **日元贬值的套利双重收益**：2012-2021 年期间，日元对美元从 ~75 贬值至 ~150。做套利交易的投资者既赚取了 ~4% 的利差，又额外获得了 ~6%/年 的日元贬值收益——这就是为什么日元套利在 2013 年后仍然盛行。

## 三、套利资金的流向目的地

### 3.1 主要流向

| 流向 | 原因 | 规模 |
|------|------|------|
| **美国股市（标普 500）** | 美元高利率 + 美股上涨预期 | 最大流向之一 |
| **美国国债** | 高利率 + 安全资产 | 稳定流向 |
| **黄金** | 美元高利率下黄金不收利息，但日元融资成本极低使黄金配置可行 | 中等规模 |
| **澳大利亚** | 澳元高息（4-5%），资源国货币相对稳定 | 传统流向 |
| **巴西/南非等新兴市场** | 高利差，但风险大 | 规模较小，投机性强 |

### 3.2 套利交易对全球资产价格的影响

| 影响 | 机制 |
|------|------|
| **压低美国长端利率** | 套利资金买入美债，降低美国国债收益率 |
| **推高新兴市场资产** | 资金流入推升股市和房地产价格 |
| **推高黄金价格** | 低成本资金配置黄金 |
| **支持美元汇率** | 套利需买入美元，支持美元 |

## 四、解仓（去杠杆化）机制

### 4.1 解仓触发因素

| 触发因素 | 解仓逻辑 |
|----------|----------|
| **日元快速升值** | 汇率损失 > 利差收益，套利者开始平仓 |
| **日本国债收益率上升** | 投资者抛售海外资产买日本国债（日元资金成本上升） |
| **全球风险偏好下降** | 避险情绪上升→日元升值→套利成本上升 |
| **日本央行收紧政策** | 加息→日元融资成本上升→套利收益收窄 |
| **美元降息** | 美元利率下降→利差收窄→套利失去吸引力 |

### 4.2 解仓的连锁反应

```
日元升值
  ↓
套利者平仓（卖出高息资产，换回日元）
  ↓
高息资产被抛售（美股/新兴市场股市/黄金下跌）
  ↓
日元进一步升值（更多套利者平仓）
  ↓
全球风险偏好下降
  ↓
其他风险资产也遭抛售（股债汇三杀）
```

> 这就是为什么日元套利解仓往往引发**全球性风险事件**——它不是单一市场的现象，而是跨资产、跨市场的系统性去杠杆。

## 五、2024 年 8 月套利交易解仓：最新案例

### 5.1 事件经过

| 时间 | 事件 |
|------|------|
| 2024.7 | 日本央行释放鹰派信号，市场开始定价日本加息 |
| 2024.8.5 | 日元单日大幅升值（USD/JPY 从 160 跌至 142，一日跌幅 ~11%） |
| 2024.8 上旬 | 全球股市暴跌：日经 225 两日暴跌 ~20%，标普 500 暴跌 |
| 2024.8.7 后 | 日本央行表示不急于继续加息，市场企稳，套利交易部分重建 |

### 5.2 对全球市场的影响

| 市场 | 影响 | 说明 |
|------|------|------|
| **日经 225** | 两日跌近 20% | 1987 年黑色星期一以来的最大跌幅之一 |
| **标普 500** | 单周跌约 5% | 科技股领跌（高估值对利率敏感） |
| **纳斯达克** | 更大跌幅 | 高估值成长股对套利解仓最敏感 |
| **VIX** | 从 15 飙升至 40+ | 恐慌指数飙升反映全球风险偏好骤降 |
| **黄金** | 短暂下跌 | 去杠杆引发的流动性需求导致黄金被抛售（流动性危机初期） |
| **原油** | 下跌 | 需求预期下降 + 避险 |

### 5.3 关键教训

- **套利解仓是"自我强化"的**：日元升值 → 套利者被迫平仓 → 日元进一步升值 → 更多套利者平仓
- **日元是全球最重要的"风险触发器"**：日元低利率的逆转不只是一个国家的事情，而是全球风险资产的共同威胁
- **2024 年的教训**：即使是短暂的套利解仓（1-2 周），也可能造成全球股市 5-20% 的跌幅

## 六、套利交易与全球金融周期

### 6.1 日元套利作为全球金融周期指标

| 套利规模 | 市场含义 |
|----------|----------|
| **套利规模扩大** | 日元极度宽松 + 全球风险偏好高 = 全球流动性充裕 |
| **套利规模缩减** | 日元紧缩 / 全球风险偏好低 = 全球流动性收紧 |
| **套利极端高位** | 极端风险积累 → 未来解仓破坏力大（2007、2024） |

### 6.2 与美元流动性的关系

日元套利与[[wiki/concepts/美元流动性|美元流动性]]高度关联：

- **套利扩张期** = 美元流动性充裕 = 全球资产价格支撑
- **套利收缩期** = 美元流动性收紧 = 全球资产承压
- **日元套利解仓** = 类似"美元流动性危机"效应，即使美联储本身没有紧缩
