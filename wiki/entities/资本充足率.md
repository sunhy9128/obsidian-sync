# 资本充足率

> 金融监管指标

**资本充足率**是衡量银行资本充足程度的核心指标，计算公式为：

```
资本充足率 = 资本 / 风险加权资产
```

## 巴塞尔协议要求

| 指标 | 最低要求 |
|------|----------|
| 总资本充足率 | 8% |
| 核心一级资本充足率 | 4.5% |
| 一级资本充足率 | 6% |
| 资本缓冲 | 2.5% |

## 中国实践

| 银行类型 | 核心一级资本充足率 | 资本充足率 |
|---------|-------------------|------------|
| 国有大行 | ≥8.5% | ≥11.5% |
| 股份制银行 | ≥7.5% | ≥10.5% |
| 城商行 | ≥7.5% | ≥10.5% |
