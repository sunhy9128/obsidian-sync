# 大众汽车（Volkswagen AG）

## 基本信息

| 项目 | 内容 |
|------|------|
| 全称 | 大众汽车股份公司（Volkswagen AG） |
| 成立 | 1937年5月（纳粹政府支持） |
| 总部 | 德国沃尔夫斯堡 |
| 德语含义 | "人民的汽车" |
| 定位 | 世界顶级汽车制造商 |

## 大众logo演变

- 1937年成立时包含纳粹元素
- 战后逐步去政治化演变为现用logo

## 在收购战中的角色

- **2004年**：财务危机，由于历史遗留问题和人力成本过高，资金链濒临断绝
- **2005-2008年**：被保时捷持续收购，成为收购目标
- **皮耶希的反击**：
  - 成立"反保时捷委员会"
  - 游说政府保留《大众法》
  - 利用保时捷安插人员的法律条款
- **反向收购**：2009年保时捷流动性危机后，大众以84.6亿欧元收购保时捷汽车
- **最终结果**：大众100%控股保时捷汽车，但保时捷控股仍持有大众50.7%

## 与保时捷的关系

```
保时捷控股（持股50.7%）← 大众集团（母公司）
      ↓
保时捷汽车（100%归大众所有）
```

"我中有你，你中有你"的复杂股权结构。
