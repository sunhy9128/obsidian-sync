# 大商所（大连商品交易所, DCE）

大连商品交易所（**Dalian Commodity Exchange, DCE**）成立于1993年，是中国四大期货交易所之一，也是全球最大的**农产品、能源化工、黑色金属**期货市场之一。在猪周期管理中，大商所是**生猪期货（2019年上市）**和**豆粕期货（2000年上市）**的上市地，是产业链套保工具的核心提供者。

> **核心命题**：大商所是中国猪周期的"价格发现中心"——豆粕期货和生猪期货为产业链提供前瞻价格信号，从源头引导蛛网模型的产能决策，平滑波动。

## 三、生猪期货（2019年1月8日上市）

详见 [[猪周期]] 与 [[套期保值]]。

### 基本合约

| 维度 | 详情 |
|------|------|
| 合约代码 | LH |
| 合约单位 | 16吨/手（约120头生猪） |
| 报价单位 | 元/吨 |
| 最小变动 | 5元/吨（80元/手） |
| 涨跌停板 | ±4%（首日±8%） |
| 最低保证金 | 合约价值5%（约6000元/手） |
| 交割方式 | 实物交割（活体生猪） |
| 交割月份 | 1/3/5/7/9/11月 |

### 交割特点

中国生猪期货是**全球首个活体交割**的生猪期货：

- **交割品要求**：体重100-120kg的瘦肉型猪（外三元）
- **交割地点**：四川、湖北、河南等生猪主产区的指定交割库
- **质量争议**：活体猪个体差异大，交割有挑战性

### 价格发现功能

生猪期货上市后形成**远期价格曲线**，对蛛网模型产生重要影响：

| 维度 | 现货市场 | 期货市场 |
|------|----------|----------|
| 价格发现 | 即时，反映当下 | 远期，反映预期 |
| 套保 | 无法套保 | 可套保10-12个月后价格 |
| 产能引导 | 滞后 | 前瞻 |
| 政策反应 | 被动 | 主动 |

> **关键洞察**：生猪期货的核心价值不是投机，而是**让养殖户获得前瞻价格信号**——远期价格告诉养殖户"未来某月猪肉应该值多少钱"，从而平滑蛛网模型的滞后性。

### 套保案例（典型饲料+养殖企业）

```
2024年5月：养殖户预计10月出栏1000头猪
  - 担心猪价下跌（饲料价格高）
  - 在大商所卖出100手LH2501合约（每手16吨×1000/16=10手）
    实际操作：每手16吨 × 10手 = 160吨（≈1000头×120kg/1000=120吨）
  - 锁定价格：16500元/吨

2024年10月：实际出栏
  - 现货售价：15800元/吨（比5月跌700元/吨）
  - 期货平仓：平仓价15850元/吨
  - 期货盈利：16500-15850=650元/吨 × 160吨 = 104000元

  综合收益：现货亏700×160=112000元 + 期货赚104000元
          = -8000元（接近盈亏平衡，避免了更大损失）
```

详见 [[套期保值]]、[[基差风险]]。

## 四、豆粕期货（2000年7月上市）

详见 [[豆粕]]。

### 行业地位
- **全球最大的豆粕期货市场**
- 成交量长期占据大商所农产品板块头名
- 国际豆粕定价中心（与CBOT联动）

### 套保应用

- **压榨企业**：锁定加工利润（"压榨套利" = 买入大豆期货 + 卖出豆粕+豆油期货）
- **饲料企业**：锁定饲料成本（买入豆粕期货）
- **养殖企业**：锁定饲料采购成本

详见 [[套期保值]]。

## 五、大商所与CBOT的关系

详见 [[CBOT]] 与 [[大豆战争]]。

### 国际联动

| 维度 | CBOT | DCE |
|------|------|-----|
| 主导品种 | 大豆、豆粕、豆油、玉米、小麦 | 大豆、豆粕、豆油、玉米、生猪 |
| 定价基准 | **全球基准**（南美+北美） | **中国基准**（本土+进口） |
| 时区 | 美国中部 | 北京时间 |
| 价格传导 | CBOT → DCE（强） | DCE → CBOT（弱） |

**核心事实**：
- 中国豆粕价格几乎100%跟随CBOT大豆价格波动
- 中国大豆进口80%定价参考CBOT（升贴水模式）
- 大商所的国际化（铁矿石期货国际化、豆粕期权等）正在加强DCE的全球定价权

### 中国大豆定价权的困境

详见 [[大豆战争]] 与 [[五神]]：

- 中国是全球最大大豆进口国（年进口约1亿吨）
- 但**中国没有大豆定价权**——定价权在CBOT
- 大商所大豆期货虽是国内基准，但只能反映"国内+进口"综合价，无法影响国际定价

> **战略意义**：建立人民币计价的大宗商品定价权，是中国粮食安全的核心命题之一。大商所是承载这一使命的关键基础设施。

## 六、监管体系

### 1. 中国证监会
- 大商所的直接监管方
- 期货新品种上市的审批

### 2. 期货保证金监控中心
- 监控全市场风险
- 防范操纵市场

### 3. 中国期货业协会
- 行业自律
- 期货公司监管

### 4. 交易制度

| 制度 | 内容 |
|------|------|
| 涨跌停板 | 各品种不同（一般4%-5%） |
| 保证金 | 5%-12%（视品种和波动） |
| 持仓限额 | 防止大户操纵 |
| 大户报告 | 持仓超限需报告 |
| 强行平仓 | 风险度过高时强制平仓 |
| 异常交易监控 | 防范操纵 |

## 七、市场国际化（2018-）

### 1. 铁矿石期货国际化（2018年5月）
- 首个国际化品种
- 境外投资者可直接交易
- 推动人民币计价大宗商品定价权

### 2. 豆粕期权（2017年）
- 中国首个农产品期权
- 提供更精细的风险管理工具

### 3. 境外参与者
- 港交所、新交所等海外交易所接入
- 境外机构投资者参与
- 推动大商所的全球影响力

### 4. QFII参与
- 合格境外机构投资者可参与特定品种
- 引入长期资金

## 九、核心洞察

> 1. **大商所是中国猪周期的"价格发现中心"**——豆粕+生猪期货为产业链提供前瞻信号
> 2. **生猪期货（2019）改变了蛛网模型**——让养殖户获得远期价格，平滑产能决策
> 3. **中国豆粕是全球最大豆粕期货市场**——但定价权仍在CBOT
> 4. **大商所国际化正在推进**——人民币计价大宗商品定价权的关键基础设施
