# 沙特协议

## Basic Information
- Type: event
- Source: [[石油美元体系]]

## Description
沙特协议是1974年美国与沙特阿拉伯达成的具有里程碑意义的双边协议，是[[石油美元体系]]诞生的标志性事件。协议核心内容规定全球石油只能用美元计价和结算，作为交换美国向沙特等产油国提供军事保护与投资渠道。该协议的实施使全球石油贸易以美元为核心货币，奠定了石油美元体系的法律和制度基础。沙特协议的签订直接促成了[[美元循环]]机制的形成，使产油国获得的石油美元通过美债投资回流至美国资本市场。

协议签订的历史背景与1973年[[第一次石油危机]]后[[OPEC]]大幅提高油价密切相关，也回应了[[布雷顿森林体系瓦解]]后美元寻找新锚定支撑的迫切需求。在这一背景下，沙特协议确立了美元与石油的绑定关系，开启了[[美元霸权]]的新阶段，并形成了完整的[[石油美元环流]]闭环。
