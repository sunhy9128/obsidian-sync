# Hans Luther（汉斯·路德 / 1879-1962）

> 1923 年用"地租票据"在 24 小时内终结魏玛恶性通胀的"天才央行行长"——从 Reichsbank 行长到德国总理，再到中国国民政府顾问

## 二、职业生涯轨迹

### 2.1 早期政治生涯（1910-1923）

```
【柏林市政厅时期（1910-1918）】
  - 1910-1918 柏林市政府议员
  - 1921-1922 柏林市财政部长
  - 1922-1923 食品与农业部长（中央政府）
       ↓
⭐ 早期生涯：
  - 法律背景，但转向经济政策
  - 一战期间管理柏林粮食供应
  - ⭐ "行政经验"积累
```

### 2.2 Reichsbank 行长（1923-1930）：Rentenmark 之父

```
⭐ Luther 的"传奇八年"（1923-1930）：

【1923.10 临危受命】
  - 1923.10 担任 Reichsbank 行长
  - 此时魏玛通胀已失控
  - 1 美元 = 240 亿马克（10 月）
  - ⭐ 接手"烂摊子"

【1923.11.15 推出 Rentenmark 改革】
  - Luther 的"天才之作"
  - 24 小时内终结通胀
  - 1 美元 = 4.2 万亿马克 → 1 美元 = 1 旧马克
  - ⭐ 载入史册

【1924 Reichsbank 改革】
  - 1924.8.30 Rentenmark 退出流通
  - 新 Reichsbank 成立（继承 Rentenmark 精神）
  - Luther 继续担任行长

【1924-1930 稳定时期】
  - 马克汇率稳定
  - 通胀受控
  - "Rentenmark 改革"的"延续期"
  - ⭐ "稳定文化"的奠基人

【1930 卸任】
  - 1930.3.31 卸任 Reichsbank 行长
  - 共任 6.5 年
  - ⭐ "任职时间跨越 1923-1930 大萧条前夕"
```

### 2.3 德国总理（1930-1933）："魏玛最后守门人"

```
⭐ Luther 的"政治舞台"（1930-1933）：

【1930.3.30 担任总理】
  - Heinrich Brüning 之后
  - 担任"魏玛共和国最后一任温和总理"
  - 在大萧条背景下
  - ⭐ "政治责任"沉重

【任期内的挑战】
  - 1931 银行业危机（Danat Bank 倒闭）
  - 1932 年失业率 30%+
  - 1932-1933 政治极端化（纳粹崛起）
  - ⭐ 魏玛民主的"末期"

【1933.1.30 卸任】
  - 希特勒被任命为总理
  - Luther 卸任
  - 魏玛共和国终结
  - ⭐ "Luther 是魏玛的'最后一个温和声音'"

【1933 后退出政坛】
  - 1933.1 希特勒上台后，Luther 退出政坛
  - 进入学术界和咨询业
```

### 2.4 中国国民政府顾问（1933-1935）

```
⭐ Luther 的"中国阶段"（1933-1935）：

【背景】
  - 1933.1 希特勒上台
  - Luther 退出政坛
  - 应国民政府邀请担任"经济顾问"
       ↓
【工作内容】

  - ⭐ "魏玛教训通过 Luther 传到中国"
```

  - 享年 83 岁
```

## 三、Rentenmark 改革的设计思想

### 3.1 改革的"三大设计原则"

```
⭐ Luther 设计的 Rentenmark 改革遵循三大原则：

【原则 1：资产背书】
  - Rentenmark 以德国"地租"（土地收益）为抵押
  - 不是"凭空印钞"
  - ⭐ "实物资产"作支撑

【原则 2：数量限制】
  - Rentenbank 每日发行上限 1200 万 Rentenmark
  - 实际远未达到
  - ⭐ "避免再通胀"

【原则 3：法律保护】
  - 1923.10.15 《货币法》保障
  - Rentenbank 独立于 Reichsbank
  - ⭐ "独立央行"的早期实践
```

### 3.2 改革的"双层设计"

```
⭐ Luther 的"天才设计"——双层结构：

【上层：货币层】
  - Rentenmark（地租马克）发行
  - 与旧马克 1:1 等值
  - ⭐ "心理锚定"

【下层：资产层】
  - 德国全部土地（Grundbesitz）
  - 工业资产（Industrieanlagen）
  - 5% 年化地租（Hypothenrente）
  - ⭐ "硬资产抵押"

⭐ 关键洞察：
  - Luther 知道"资产只是名义"
  - 真正起作用的是"信用"
  - ⭐ "信用是货币的锚"
```

### 3.3 改革成功的"政治经济学"

```
⭐ Luther 对改革成功的"政治经济学"解读：

【市场信心的作用】
  - "Rentenmark 的价值"不是来自资产
  - 来自"市场对改革的信心"
  - ⭐ "信心" > "资产"

【政府承诺的作用】
  - 财政部接受 Rentenmark 缴税
  - 政府用 Rentenmark 发工资
  - ⭐ "政府背书" = 货币信用

【政治意愿的作用】
  - Luther 说服 Stresemann 总理支持
  - 获得政府授权
  - ⭐ "政治意愿"是改革前提

【改革的速度】
  - 1923.11.15 上午 9:00 Rentenbank 开业
  - 下午政府宣布支持
  - 24 小时内见效
  - ⭐ "改革需要'政治窗口'"
```

```

## 五、Luther 的"中国经历"

### 5.1 1933-1935 中国顾问时期

```
⭐ Luther 与中国的不解之缘：

【邀请背景】
  - 1933.1 希特勒上台
  - 国民政府寻求"国际经济专家"
  - Luther 离开德国政坛
  - ⭐ "偶然的中国经历"

【工作内容】
  - 1933-1935 在中国
  - 担任国民政府"经济顾问"
  - 推动币制改革
  - ⭐ 影响"法币改革"（1935.11）

【法币改革（1935.11.3）】
  - 国民政府废除银本位
  - 发行"法币"（法定货币）
  - 由中央银行、中国银行、交通银行发行
  - ⭐ 现代化中国货币体系

⭐ Luther 与法币改革：
  - 法币改革后，中国成为"信用货币"国家
  - 部分参考 Rentenmark 改革经验
  - ⭐ "魏玛教训 → Luther → 中国法币"
```

### 5.2 历史意义

```
⭐ Luther 中国经历的意义：

【1】魏玛教训的国际传播】
  - 魏玛恶性通胀 → Luther 改革 → 中国法币改革
  - ⭐ "100 年前的金融知识传播"

【2】中国"信用货币"的早期实践】
  - 1935 法币改革 = 中国第一次"信用货币"
  - ⭐ "中国版 Rentenmark"

【3】失败但有教训】
  - 法币改革后（1937-1945）法币大幅贬值
  - 1948-1949 金圆券崩溃
  - ⭐ 但"信用货币"思想已扎根
  - ⭐ 1948 人民币继承"信用货币"传统
```

## 八、一句话总结

> **Hans Luther = 1923 年用 Rentenmark 在 24 小时内终结百年恶性通胀的"天才央行行长"**。
>
> 他是 1923.11.15 货币改革的设计者，是从 Reichsbank 行长到德国总理（1930-1933）、再到中国国民政府顾问的跨政治经济外交传奇人物。
>
> **理解 Hans Luther = 理解三件事**：
> 1. **"信用是货币的锚"**：Rentenmark 的真正力量是"市场信心"，不是"资产抵押"
> 2. **央行行长可以"改变历史"**：Luther 的 24 小时改革与 Draghi 的"Whatever it takes"是央行行长的"个人英雄主义"
> 3. **"危"是"机"的窗口**：恶性通胀是改革的前提
>
> 在中国寻求"中国特色现代金融体系"的今天，**Luther 的故事是"央行行长 + 危机 + 改革"三位一体的经典案例**——也是中国 1935 法币改革的精神起点。
