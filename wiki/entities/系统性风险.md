# 系统性风险

> 单个事件可能引发整个金融体系连锁反应的风险

## 核心定义

**系统性风险**是指单个金融机构或市场的困境可能引发连锁反应，导致整个金融体系功能失调的风险。与非系统性风险不同，系统性风险无法通过分散投资完全消除。

## 主要特征

| 特征 | 说明 |
|------|------|
| 传染性 | 问题从一个机构传导至整个系统 |
| 负外部性 | 单个实体的损失外溢至其他实体 |
| 难以预测 | 通常在危机爆发时才被充分认识 |
| 后果严重 | 可能导致信贷紧缩、经济衰退 |

## 典型案例

### 2008年全球金融危机

- **触发点**：美国次级抵押贷款违约
- **传导机制**：MBS/CDO 减值 → 两房危机 → 雷曼破产 → 信贷市场冻结 → 全球衰退
- **系统性风险来源**：大型金融机构间的相互敞口 + 影子银行杠杆
- [[wiki/entities/次贷危机]] — 全景复盘：次级贷款机制、MBS/CDO 结构、全球传染链 ⭐

### 2023年美国银行危机

- **触发点**：SVB资产负债管理失误
- **传导机制**：存款挤兑 → 信心危机 → 区域银行连锁反应
- **系统性风险来源**：无息存款依赖 + 利率风险未对冲

## 监管应对

| 监管工具 | 目的 |
|---------|------|
| 资本缓冲 | 吸收损失，维持运营 |
| 流动性要求 | 防止挤兑引发的流动性枯竭 |
| "大而不能倒"监管 | 加强对系统重要性机构的监管 |
| 压力测试 | 评估极端情景下的韧性 |
