# 点阵图

> 美联储FOMC委员利率预测的可视化工具

## 核心定义

点阵图（Dot Plot）是美联储FOMC会议后发布的利率预测可视化图表，展示每位委员对未来各年联邦基金利率目标的预测分布。

## 解读方法

- 横轴 = 时间(年份)
- 竖轴 = 利率(%)
- 每个点 = 委员投票区间
- 中位数反映委员会共识预期
- 散点分布反映**委员内部分歧**

## 2026-06 FOMC 点阵图(moomoocat 解读)

| 项目 | 数值 |
|---|---|
| 投票委员总数 | 18 |
| 当前美元利率 | 3.50% ~ 3.75% |
| 认为年内应加息 | **9 票**(50%) |
| — 其中认为加息 2 次以上 | **6 票** |
| 认为应降息 | 9 票(50%) |

**关键观察**:本次会议**未加息**,符合市场预期;但 50:50 的内部投票分裂表明 FOMC 内部对利率路径存在严重分歧。

## 2026 年路径推演

```
当前:3.50-3.75%
    ↓
伊朗局势持续 → 油价高 → 通胀粘性 → 维持高利率
    ↓
停战协议落实 + 霍尔木兹开放
    ↓
油价回落至 70-80 美元
    ↓
通胀数据回落
    ↓
中长期:美元慢慢降息到 ~3%
```

## 市场意义

- 反映FOMC内部意见分布(分歧程度)
- 预测市场未来利率路径
- 影响债券市场(美债收益率)和汇率(美元指数)
- **2026 年的核心矛盾**:伊朗局势 → 油价 → 通胀 → 利率路径
