# 次贷危机

> 2007-2009 年全球金融危机的核心导火索 — 从次级贷款到全球金融海啸

**次贷危机**（Subprime Mortgage Crisis）是 2007 年爆发、2008 年深化、2009 年蔓延至实体经济的全球性金融危机。危机起源于美国次级抵押贷款市场的信用泡沫，经由资产证券化、杠杆放大、信心传染，最终演变为自大萧条以来最严重的全球金融危机。

---

## 一、危机的根源：次级贷款的诞生

### 1.1 什么是次级贷款

| 分类 | 定义 | 典型利率差 |
|------|------|-----------|
| **优贷（Prime）** | 信用记录良好、收入稳定的借款人 | 基准利率 + 0-1% |
| **Alt-A 贷款** | 信用记录不全但收入可核实 | 基准利率 + 1-3% |
| **次级贷款（Subprime）** | 信用记录差（ FICO < 620 ）、低首付 | 基准利率 + 3-6% |

### 1.2 次贷的规模扩张

```
1994 年：次级贷款市场规模 ~350 亿美元
2001 年：次级贷款市场规模 ~1200 亿美元（互联网泡沫后房产成为替代投资）
2006 年：次级贷款市场规模 ~6000 亿美元（峰值，占新增房贷的 20%+）

次级贷款占抵押贷款市场的比例：
2001: ~5% → 2006: ~20%+（约 1/5 的新增房贷是次级贷款）
```

### 1.3 次贷发放机构的动机

| 主体 | 动机 |
|------|------|
| **抵押贷款经纪商** | 收取手续费，发放越多收入越多 |
| **商业银行** | 快速将贷款卖给投行，转移风险 |
| **投行** | 打包成 MBS 销售，获取手续费 |
| **借款人** | "掠食性贷款"：诱导低收入群体申请超出偿还能力的贷款 |

```
典型次贷产品：2/28 可调利率抵押贷款（ARM）
- 前 2 年：固定利率（极低，如 1-2%）
- 后 28 年：浮动利率（市场利率 + 利差，通常升至 8-12%）
- 结果：2 年后还款额翻倍，大量借款人无力偿还

其他产品：interest-only、NINJA（无收入无工作无资产）
```

## 二、资产证券化：MBS 与 CDO

### 2.1 抵押贷款支持证券（MBS）

```
MBS 创造链条：

抵押贷款（次级房贷）打包
        ↓
抵押池（$10亿，1000笔贷款）
        ↓
特殊目的机构（SPV）隔离风险
        ↓
MBS 分档（Tranche）：

优先级（Senior）：AAA 评级，占 70%，收益率 5-6%
    ↓ 承担最早损失
夹层（Mezzanine）：A/BBB 评级，占 20%，收益率 7-9%
    ↓ 承担中等损失
股权档（Equity）：无评级，占 10%，收益率 15%+（或负收益）
    ↓ 承担最后损失

→ 风险从银行资产负债表转出
→ 各类投资者获得各自想要的收益/风险组合
```

### 2.2 担保债务凭证（CDO）

```
CDO：从 MBS 到 CDO 的二次分层

第一层：次级 MBS（BBB 评级为主）
        ↓
CDO 打包：CDO²（二次CDO）
        ↓
再次分档 → 再次评级
→ BBB 级次贷 MBS 重新打包 → 变成 AAA 级 CDO

结果：
- 大量次级贷款风险被"洗白"成 AAA 级证券
- 大量保险公司、共同基金、养老基金持有这些"AAA"资产
- 实际风险高度集中但被隐藏在复杂的结构性产品中
```

### 2.3 合成 CDO（Synthetic CDO）

```
合成 CDO：不转移贷款本身，转移违约风险

结构：
- CDO 发行方：不持有真实贷款
- 投资者：购买 CDO 份额，承担参考贷款池的违约损失
- 收取：保费收益（CDS 形式的保护费）

参考贷款池：可以是其他 CDO、MBS、企业贷款等

合成 CDO 的危害：
- 杠杆极高（名义本金 >> 实际资金投入）
- 风险敞口不透明（银行持有的 CDP 可能不知道具体参考了什么贷款池）
- 2007-2008 年危机中大量 CDO 与次贷 MBS 参考池相关
```

## 三、评级机构的角色

### 3.1 评级失灵的机制

| 问题 | 说明 |
|------|------|
| **评级购买模式** | 发行方付费购买评级 → 评级机构有动机给出高评级以获得业务 |
| **模型依赖** | 假设房价永远上涨 → 历史数据太短无法验证极端情景 |
| **静态评级** | 评级后不随基本面变化调整（直到危机爆发才大幅下调） |
| **结构复杂性** | CDO 包含数百种底层资产 → 评级模型无法完全捕捉 |

### 3.2 评级下调的连锁反应

```
危机前：AAA 级 CDO/MBS 被广泛持有
2007 年中：评级机构开始下调 → 触发止损条款
        ↓
持有机构被迫抛售 → 资产价格下跌
        ↓
更多持有机构出现浮亏 → 信心下降
        ↓
CDO 市场流动性枯竭 → 价格发现机制失灵
        ↓
次贷 → BBB → BB → B → 违约 → 实际损失显现
```

### 3.3 规模对比

| 数据 | 数值 |
|------|------|
| 2002-2007 年发行的 AAA 级 MBS/CDO | ~3 万亿美元 |
| 其中在 2007-2008 年被下调至低于 AAA | ~75% |
| 穆迪对次贷支持的 CDO 评级下调比例 | ~90% |

## 五、关键机构详解

### 5.1 两房危机（2008.7）

**房利美（Fannie Mae）** + **房地美（Freddie Mac）**：美国政府支持企业（GSE），持有或担保美国约一半的抵押贷款（$5.3 万亿）。

| 危机原因 | 说明 |
|----------|------|
| **持有大量次级 MBS** | 两房持有或担保的 MBS 中有大量次级贷款 |
| **双重杠杆** | 持有 MBS 的同时发行债券融资（杠杆 ~ 75:1） |
| **隐含政府担保** | 市场认为两房有政府隐性背书 → 低风险溢价 → 过度扩张 |

**接管措施**：
- 2008.9.7：美联储和财政部联合接管两房
- 接管时两房分别持有 ~$8300 亿和 ~$5100 亿次贷相关资产
- 美国纳税人最终承担了约 $3100 亿救助成本

### 5.2 雷曼兄弟（2008.9.15）

| 项目 | 内容 |
|------|------|
| 资产规模 | $6000 亿（破产时美国第四大投行） |
| 破产原因 | 持有大量 CDO/MBS → 评级下调 → 融资渠道枯竭 → 无法找到买家 |
| 失败关键 | 过度依赖隔夜回购市场融资 → 市场冻结时资金链瞬间断裂 |
| 并购谈判 | 曾与韩国产业银行、巴克莱银行、美国银行谈判并购，全部失败 |

> **雷曼破产是系统性危机的转折点**——此前救助贝尔斯登和两房，市场仍相信"大而不能倒"；雷曼的破产打破了这一信念，引发全球信贷市场完全冻结。

### 5.3 AIG 与 CDS 危机

**信用违约互换（CDS）**：AIG 出售了数千亿美元的 CDS 保护（相当于保险），收取保费。

```
CDS 的问题：
- 出售保护时几乎不需要资本
- 收取的保费（1-2%）看起来是"免费的钱"
- 实际上：参考资产（次贷 CDO）大规模违约 → CDS 需要赔付

2008.9.16：AIG 信用评级下调
→ CDS 合约触发追加保证金条款
→ AIG 无法支付 → 面临破产
→ 联邦储备银行紧急注资 $850 亿（后增至 $1820 亿）
→ AIG 成为美国历史上最大的企业救助

CDS 市场规模：
2008 年 CDS 名义本金：约 $62 万亿（超过全球 GDP）
CDS 持有者不是为了对冲，而是作为投机工具
```

## 六、全球传染路径

### 6.1 传染渠道

| 渠道 | 机制 |
|------|------|
| **信贷渠道** | 美国银行缩减放贷 → 跨境信贷收紧 → 全球美元融资困难 |
| **资产价格渠道** | 美国 MBS/CDO 暴跌 → 持有这些资产的欧洲银行浮亏 → 去杠杆 |
| **流动性渠道** | 回购市场冻结 → 欧洲银行无法获得美元融资 → 抛售资产 |
| **信心渠道** | 雷曼破产 → 全球金融机构互不信任 → 银行间拆借利率飙升 |
| **汇率渠道** | 避险需求 → 美元飙升 → 新兴市场货币贬值 + 美元债务负担加重 |
| **贸易渠道** | 美国消费下降 → 全球出口需求下降 → 新兴市场出口国受损 |

### 6.2 全球影响

| 国家/地区 | 影响 | 损失 |
|-----------|------|------|
| **欧洲** | 多个国家陷入主权债务危机（希腊、爱尔兰、葡萄牙、西班牙，即"欧猪五国"） | 银行业危机 |
| **英国** | 北岩银行挤兑、劳埃德银行被迫国有化 | 银行体系危机 |
| **日本** | 银行持有大量美元资产，汇率和流动性双重冲击 | 间接冲击 |
| **新兴市场** | 美元融资困难、出口下降、货币贬值 | 经济放缓 |
| **中国** | 出口下降、A 股暴跌 70%+（2007-2008）、4 万亿刺激计划 | 经济放缓 |
| **全球 GDP** | 2009 年全球 GDP 下降 0.6%（发达国家下降 3.4%） | 大衰退 |

## 七、政策应对

### 7.1 货币政策应对

| 工具 | 措施 |
|------|------|
| **降息** | 美联储从 5.25% 降至 0-0.25%（2007.9-2008.12） |
| **量化宽松（QE1）** | 2008.11-2010.3，购买 MBS/国债 ~$1.7 万亿 |
| **央行互换** | 美联储与欧央行、英格兰银行等建立美元互换机制 |
| **零利率政策** | 维持零利率至 2015 年底 |

### 7.2 财政应对

| 措施 | 规模 |
|------|------|
| **TARP** | $7000 亿不良资产救助计划 |
| **AIG 救助** | $1820 亿政府注资 + 接管 |
| **两房接管** | 承担约 $3100 亿损失 |
| **巴菲特注资** | 巴菲特向高盛投资 $50 亿（优先股） |
| **美国刺激计划** | 2009 年 $7870 亿经济刺激法案 |

### 7.3 监管应对

| 改革 | 内容 |
|------|------|
| **Dodd-Frank 法案（2010）** | 全面金融监管改革 |
| **沃尔克法则** | 限制银行自营交易（后被削弱） |
| **有序清算机制** | 允许监管机构有序关闭系统重要性机构（不再"太大而不能倒"） |
| **消费者保护局** | CFPB 监管消费金融产品 |
| **Basel III** | 提高资本充足率要求、新增流动性要求 |
| **衍生品改革** | OTC 衍生品强制清算 |

## 八、危机的根本原因

### 8.1 结构性问题

```
1. 激励机制扭曲
   - 抵押贷款经纪商：不承担贷款违约风险，只收手续费
   - 评级机构：收费模式（发行方付费）→ 给出过高评级
   - 投行：奖金基于短期收益 → 忽视长期风险

2. 监管漏洞
   - 影子银行（对冲基金、SPV）不受银行监管
   - 监管竞争（各州不同规则）
   - 美联储缺乏对非银行抵押贷款机构的监管权

3. 风险计量错误
   - 历史数据太短（房价从未全国性持续下跌）
   - 相关性假设错误（正常市场下相关性低，压力下趋于1）
   - 模型未考虑"所有人同时犯错"的情景

4. 道德风险
   - 隐含政府担保（两房）→ 过度冒险
   - "大而不能倒" → 金融机构认为不会破产
```

## 九、核心教训

### 9.1 对金融体系

| 教训 | 当前状态 |
|------|---------|
| 影子银行需要监管 | FSB 持续监测，Basel III 覆盖银行资产负债表外 |
| 衍生品需要透明化 | OTC 衍生品强制清算（部分执行） |
| 评级机构需要改革 | SEC 调查，付费模式争议持续 |
| "大而不能倒"需要解决 | 有序清算机制建立，但道德风险仍然存在 |

### 9.2 对投资者

| 教训 | 含义 |
|------|------|
| 复杂不等于安全 | 复杂结构性产品风险可能被隐藏 |
| 评级不等于事实 | 外部评级不可盲目信任 |
| 杠杆放大一切 | 去杠杆过程可能远超预期 |
| 流动性可能在最需要时消失 | 投资需要考虑流动性风险 |

### 9.3 对政策制定者

| 教训 | 含义 |
|------|------|
| 宏观审慎监管缺失 | Basel III 引入宏观审慎监管框架 |
| 危机跨市场传导 | 美联储全球央行协调机制建立 |
| 宽松监管与金融危机高度相关 | 危机后的监管改革（但 2018 年部分被废除）|
