# 美元霸权

## 核心定义

美元霸权是指美元作为全球主要储备货币、贸易结算货币和金融市场基准货币所享有的特殊地位，使美国能够享受"过度特权"，同时将通胀和金融风险转嫁给全世界。

## 美元霸权如何转移通胀

### 核心机制：美元循环

```
1. 美国输出美元 → 贸易逆差/资本输出 → 美元流入全球
2. 全球持有美元 → 储备货币 + 结算货币 → 被动接受美元
3. 美国收紧 → 美元回流美国 → 他国被动紧缩
4. 汇率贬值 → 进口成本上升 → 通胀传导至新兴市场
```

### 通胀转移四条通道

| 通道 | 机制 | 案例 |
|------|------|------|
| **汇率转嫁** | 美元升值 → 本币贬值 → 进口通胀 | 2022年新兴市场货币贬值30-50% |
| **资本流动** | 加息引资本回流 → 资产抛售 → 股债汇三杀 | 2022年Taper Tantrum |
| **债务陷阱** | 外债美元计价 → 货币贬值 → 债务重组 | 80年代拉美危机 |
| **大宗商品** | 原油以美元计价 → 美元涨=商品跌 | 石油美元定价权 |

### "美国放水 = 全球买单"

| 政策 | 美国国内 | 全球影响 |
|------|----------|----------|
| QE/降息 | 美元流出 | 全球通胀 |
| QT/加息 | 美元回流 | 他国危机 |
| 汇率武器 | 金融制裁 | SWIFT剔除=经济死刑 |

### 历史案例

| 年份 | 事件 | 通胀转嫁机制 |

## 关键要点

- **三大支柱**：[[石油美元体系]]（1974年沙特协议，石油只能用美元结算）、贸易结算（全球60%贸易用美元结算）、金融基础设施（SWIFT、CHIPS、美债市场）
- **七大特权**：印自己的货币还债、赤字可以无限、金融制裁权力（踢出SWIFT=经济死刑）、铸币税、低廉的融资成本、危机转嫁（美国放水=全球买单）、道德风险
- **特里芬难题（致命矛盾）**：美元作为世界货币要求保持贸易逆差（美元流出），但长期逆差会损害美元信誉
- **历史演变**：布雷顿森林体系（1944-1971，美元=黄金，固定汇率）→信用货币时期（1971至今，与黄金脱钩）→美元霸权确立（1974至今，石油美元+军事霸权支撑）
