# 瑞士央行（Swiss National Bank / SNB）

> **全球金融"避风港"的守护者**——负利率的早期试验者，瑞郎与欧元的"汇率战争"主导者

---

## 二、SNB 的核心特点

### 2.1 全球负利率的早期试验者

```
⭐ SNB 的"全球第一"：

【2014.12.18：SNB 实施负利率】
  - DFR：-0.25%
  - ⭐ 全球第二个负利率央行（瑞典 2009 后）
  - ⭐ 主要央行中第一个（ECB 2014.6 略早，但 SNB 是首个完整负利率体系）
       ↓
【2015.1.15：EUR/CHF 汇率下限取消】
  - SNB 放弃 1.20 下限
  - 瑞郎瞬间升值 30%+
  - ⭐ "黑色星期四"
  - 外汇市场剧烈波动
```

### 2.2 瑞士的特殊性

```
【特殊性 1：永久中立的"避风港"】
  - 瑞士 1815 年起永久中立
  - 不参与任何战争
  - ⭐ 政治、经济"避风港"

【特殊性 2：金融业占 GDP 10%+】
  - 银行业、保险业、私人银行高度发达
  - 瑞士是全球最大的离岸财富管理中心
  - ⭐ 银行业是瑞士经济的命脉

【特殊性 3：多元货币体系】
  - 瑞士不是 EU 成员
  - 瑞士不是欧元区
  - 但瑞郎与欧元高度相关
  - ⭐ "独立但紧密"

【特殊性 4：低通胀传统**
  - 瑞士长期通胀 0%-1%
  - 通缩风险长期存在
  - ⭐ 负利率成为长期工具
```

## 三、SNB 的货币政策

### 3.1 政策框架（2026）

```
瑞士央行政策框架：

【1】政策利率：SNB Policy Rate
  - 2026 SNB Policy Rate：0.5%（从 1.75% 降息）
  - 调整频率：每季度评估

【2】通胀目标：CPI < 2%
  - 历史上长期低于 2%
  - 多次通缩风险

【3】利率走廊：
  - SNB Policy Rate（基准）：0.5%
  - Sight Deposit Interest Rate（下限）：0.5%
  - ⭐ "走廊"非常窄（有时负利率曾达 -0.75%）

【4】外汇干预：
  - 主动买卖外汇
  - 干预瑞郎汇率
  - ⭐ SNB 的"外汇战争"
```

### 3.2 SNB 负利率实验（2014-2022）

```
SNB 负利率历史：

2014.12.18：DFR -0.25%（首次）
  - 当时其他主要央行未实施负利率
  - ⭐ SNB 是主要央行中的"首个完整负利率"

2015.1.15：EUR/CHF 下限取消
  - 瑞郎剧烈升值
  - 但负利率继续

2015.12.3 ：DFR -0.75%（最低）
  - 全球最低的政策利率之一
  - 持续至 2022

2022.6.16：DFR -0.25%（开始退出）

2022.9.22：DFR 0.50%（退出负利率）
  - 跟随全球加息周期

2024 起：降息周期
  - 当前 DFR：0.5%（2026.6）
       ↓
⭐ 评价：
  - SNB 负利率持续 7.5 年（2014.12-2022.9）
  - 比 ECB 早 8 个月
  - 是主要央行中负利率最深的（-0.75%）
```

### 3.3 SNB 的外汇干预

```
⭐ SNB 的"外汇战争"：

【EUR/CHF 下限（2011.9 - 2015.1.15）】

2011.9 ：SNB 设定 EUR/CHF 下限为 1.20
  - 背景：欧债危机，瑞郎持续升值
  - 目的：阻止瑞郎过度升值
  - ⭐ 全球首次"汇率下限"

2012-2014：SNB 持续干预
  - 大量买入欧元资产
  - 资产负债表膨胀至 ~7000 亿瑞郎
  - SNB 持有大量欧元资产

2015.1.15：取消下限（"黑色星期四"）
  - 瑞郎瞬间升值 30%+
  - 外汇市场剧烈波动
  - 部分券商破产（如 FXCM）
  - ⭐ SNB 的"重大政策失误"

【2022 后：再次干预】
  - 2022 瑞郎贬值压力
  - SNB 减持欧元资产
  - ⭐ SNB 资产从 9000 亿瑞郎降至 7000 亿
```

## 四、SNB 的资产负债表

### 4.1 SNB 的"特殊资产负债表"

```
⭐ SNB 资产负债表的"异常"：

【规模】
  - 2020 SNB 总资产：9000 亿瑞郎（约 1 万亿美元）
  - 占瑞士 GDP 比例：~130%
  - ⭐ 全球央行最高（远超 BOJ 的 130%）

【资产构成】
  - 外国债券：~50%（主要美元、欧元、日元、英镑）
  - 黄金：~10%
  - 股票：~20%
  - 其他：~20%
  - ⭐ "全球最大的对外投资者"

【外汇储备】
  - SNB 持有全球最大外汇储备之一
  - 用于外汇干预
  - ⭐ "央行中的对冲基金"
```

### 4.2 SNB 与黄金

```
⭐ SNB 的"黄金传统"：

【历史传统】
  - 瑞士是全球黄金交易中心
  - SNB 持有大量黄金
  - 占总资产 ~10%
  - ⭐ 全球央行中黄金比例最高之一

【黄金价值】
  - 2025 黄金价格：~3000 美元/盎司（创历史新高）
  - SNB 黄金价值 ~800 亿瑞郎
  - ⭐ 黄金成为 SNB 利润的重要来源
```

## 六、SNB 的历史事件

### 6.1 关键事件

```
【2015.1.15 EUR/CHF 下限取消（"黑色星期四"）】
  - 瑞郎对欧元瞬间升值 30%+
  - 外汇市场剧烈波动
  - 多个券商破产
  - 零售外汇交易者重大损失
  - ⭐ SNB 政策"重大失误"

【2014.12 SNB 实施负利率】
  - 比 ECB 略晚，但更激进
  - ⭐ SNB 是主要央行中负利率最早的

【2020 COVID 应对】
  - 维持负利率 -0.75%
  - 大量外汇干预
  - ⭐ SNB 相对保守

【2022 加息周期】
  - 比 ECB 早几个月
  - 2022.9 退出负利率
  - ⭐ SNB 政策反应快

【2024 降息周期】
  - 2024 起持续降息
  - 当前 0.5%（2026.6）
```

## 七、SNB 的争议

### 7.1 主要争议

```
【争议 1：汇率干预的有效性】
  - 2011-2014 的 EUR/CHF 下限
  - 2015.1.15 突然取消
  - 市场剧烈波动
  - ⭐ "汇率干预的极限"

【争议 2：负利率的有效性】
  - SNB 负利率 -0.75%
  - 持续 7 年
  - 通胀仍未达 2%
  - ⭐ "负利率工具的局限"

【争议 3：资产负债表的"异常"】
  - SNB 总资产 / GDP = 130%
  - 持有大量外汇、股票、黄金
  - ⭐ "央行的角色边界"

【争议 4：与黄金挂钩的讨论】
  - 瑞士曾有"金本位"传统
  - 2014 公投否决"黄金公投"
  - ⭐ "黄金货币化"的政治阻力
```

### 8.1 瑞士法郎的国际地位

```
瑞郎（CHF）的国际地位：

【全球外汇储备】
  - 美元：~58%
  - 欧元：~20%
  - 日元：~5.5%
  - 英镑：~5%
  - 人民币：~2.5%
  - **瑞郎：~0.2%**（较小）

  - 瑞郎是传统避险货币

  - 危机时瑞郎需求激增

【全球外汇交易】
  - 瑞郎在外汇交易中占比：~5%
  - 排名第 7 位
```

## 十、一句话总结

> **瑞士央行（SNB）= 全球金融"避风港"的守护者——负利率的早期试验者，瑞郎与欧元的"汇率战争"主导者**。
>
> 它成立于 1907 年，是瑞士永久中立、金融避风港的核心守护者。**SNB 拥有全球央行中最大的资产负债表（占 GDP 130%），最早实施负利率（2014.12），最有激进的外汇干预传统**。
>
> **理解 SNB = 理解四件事**：
> 1. **避风港货币的悖论**：避险地位导致瑞郎升值压力，需要负利率 + 外汇干预
> 2. **汇率下限的极限**：2011-2014 EUR/CHF 下限 + 2015.1.15 突然取消
> 3. **负利率的"瑞士实验"**：比 ECB 略晚但更激进（-0.75%）
> 4. **央行的"对冲基金"角色**：SNB 持有大量股票、外汇、黄金
>
> SNB 代表着央行政策的"极端形态"——**央行不仅是货币政策制定者，还是外汇市场最大的"玩家"**。
