# 外汇储备

## 核心定义

外汇储备（Foreign Exchange Reserves / FX Reserves）是中央银行持有的以外币计价的资产，主要用于干预外汇市场、维护本币汇率稳定、应对国际收支危机。它是国家的"金融弹药库"。

---

## 一、储备的功能

| 功能 | 说明 | 关键指标 |
|------|------|----------|
| **交易性需求** | 维持日常国际支付、偿还外债 | 进口覆盖月数（安全线：≥3个月） |
| **预防性需求** | 应对资本外流、金融危机 | 短期外债覆盖比（安全线：≥100%） |
| **信号功能** | 向市场展示干预能力 | 即期威慑效应 |
| **国家信用** | 主权评级、对外融资的基础 | 储备/GDP > 20% |

> [!key-insight] 亚洲金融危机后的"自我保险"动机
> 1997年亚洲金融危机后，亚洲国家集体意识到：IMF救援不仅条件苛刻，而且来得太慢。从此，它们开始大量积累外汇储备作为自我保险。中国 $3万亿、日本 $1.2万亿、韩国 $0.4万亿——这是全球储备分布的"亚洲特征"。

---

## 二、适度规模理论

### 传统指标

| 指标 | 公式 | 安全线 | 临界线 |
|------|------|--------|--------|
| 进口覆盖 | 储备 / 月均进口额 | ≥3个月 | <1.5个月 |
| 短期外债比 | 储备 / 短期外债 | ≥100% | <80% |
| 广义货币比 | 储备 / M2 | ≥20% | <10% |
| 总外债比 | 储备 / 总外债 | ≥50% | <30% |

### Guidotti-Greenspan 规则（最常用）

> **外汇储备 ≥ 一年内到期的全部外债**

逻辑：即使所有短期外债同时到期（极端情况），央行也能不违约地偿还。

### 中国的实际储备

| 年份 | 外汇储备 | 进口覆盖（月） | 短期外债比 |
|------|---------|---------------|-----------|
| 2006 | $1.0万亿 | 12 | 350% |
| 2014 | $4.0万亿（峰值） | 24 | 500% |
| 2017 | $3.1万亿 | 18 | 250% |
| 2024 | $3.2万亿 | 16 | 200% |

> [!contradiction] 储备过多也有成本
> - **财务成本**：持有美债收益率（~4-5%）vs 国内回报率（8-10%）→ 机会成本巨大
> - **汇率风险**：美元贬值时储备实际购买力缩水
> - **通胀输出**：外汇占款释放基础货币 → 国内通胀压力
> - **政治风险**：被冻结或制裁风险（俄罗斯被冻结 $3000亿外汇储备的前车之鉴）

## 三、储备构成与配置

### 全球外汇储备构成（IMF COFER 数据）

| 货币 | 全球占比（2024） | 趋势 |
|------|----------------|------|
| 美元 | ~58% | 缓慢下降（2000年 ~71%） |
| 欧元 | ~20% | 稳定 |
| 日元 | ~6% | 略有上升 |
| 英镑 | ~5% | 稳定 |
| 人民币 | ~2.5% | 缓慢上升 |
| 其他 | ~8.5% | 上升 |

### 储备资产类型

```
外汇储备资产谱系
├── 流动性最高/收益最低
│   ├── 美国国债（最核心，占美元储备60%+）
│   ├── 机构债券（GSE，房利美/房地美）
│   └── 央行存款/回购协议
├── 中等流动性/中等收益
│   ├── 其他发达经济体国债（德/法/英/日）
│   └── 准主权机构债券
└── 流动性较低/收益较高
    ├── 黄金（特殊类别，无信用风险）
    ├── SDR（IMF特别提款权）
    └── 国际金融机构存款
```

**金边原则**（Golden Rule of Reserve Management）：
> 外汇储备管理的第一目标是**安全性和流动性**，而非收益性。

### 中国储备配置（估计）

| 资产 | 估计占比 | 说明 |
|------|---------|------|
| 美国国债 | ~35% | ~$1.1万亿（美债最大外国持有者之一） |
| 机构债券 | ~15% | 房利美/房地美等 |
| 其他发达经济体国债 | ~20% | 欧洲、日本 |
| 黄金 | ~5% | ~2,300吨（持续增持中） |
| SDR + IMF 头寸 | ~3% | 国际机构配置 |
| 其他（非标等） | ~22% | 多元化配置 |

## 四、储备管理的"三难困境"

```
    安全性
      /\
     /  \
    /    \
   /      \
  /  平衡  \
 /__________\
流动性        收益性
```

三个目标相互冲突：
- **安全** → 持有美债 → 收益低
- **收益** → 投资股票/另类资产 → 流动性差
- **流动性** → 持有现金/短债 → 收益更低

**平衡策略**：按"分层"管理（Tiered Approach）

```
流动性层（~30%）：现金、短债、央行存款 → 满足日常需求
投资层（~50%）：中长期国债、机构债 → 获取稳定收益
增强层（~20%）：股票、黄金、非标资产 → 追求超额收益
```

## 五、储备的积累与消耗机制

### 积累渠道

| 渠道 | 机制 | 典型国家 |
|------|------|----------|
| 经常账户顺差 | 出口收入 > 进口支出，央行购汇 | 中国、德国、日本 |
| 资本账户流入 | FDI 流入，央行干预 | 瑞士、新加坡 |
| 外债发行 | 发行外币债券，央行持有 | 印度尼西亚 |
| 黄金购买 | 央行直接购买黄金 | 中国、俄罗斯、土耳其 |

### 消耗渠道

| 渠道 | 机制 | 典型情景 |
|------|------|----------|
| 汇率干预（阻贬） | 卖出美元买入本币 | 贬值压力时期（2015中国、2022日本） |
| 资本流出 | 资本外逃，央行被动耗储 | 危机时期 |
| 主权财富基金转移 | 将超额储备转移至 SWF | 挪威、新加坡、中国（中投） |
| 外债偿还 | 政府或国企偿还外币债务 | 到期高峰时 |

## 六、超级储备国专题

### 中国 | $3.2万亿

- 全球最大外汇储备持有国
- 2014年峰值达 $4万亿，后因干预消耗至 $3万亿
- 管理主体：国家外汇管理局（SAFE）
- 其中约 $1万亿通过**中投公司**（CIC）和**中金公司**进行海外投资
- 当前面临：中美利差倒挂下的储备保值压力

### 日本 | $1.2万亿

- 第二大外汇储备国
- 由**财务省**（MOF）决定干预策略，**日本央行**（BOJ）执行
- 2022-2024年因日元干预动用约 $1200亿
- 大部分储备以美元资产形式持有

### 瑞士 | $0.9万亿

- 储备规模 vs GDP 全球最高（~110%）
- 瑞士央行（SNB）通过大规模干预积累
- 管理策略：高度多元化（股债平衡，全球配置）
- 2023年因瑞郎升值损失约 $1300亿

## 七、2022年俄罗斯储备冻结：规则改变者

| 时间 | 事件 | 影响 |
|------|------|------|
| 2022.02 | 俄乌冲突爆发 | 西方冻结俄罗斯央行 ~$3000亿外汇储备 |
| 2022.03 | 卢布暴跌 | 市场信心冲击 |
| 2022.04 | 俄罗斯出台资本管制+卢布结算令 | 部分恢复稳定 |

**深远影响**：
1. **储备安全重新定义**：以前大家只担心市场风险，现在要考虑政治风险
2. **加速去美元化**：多国央行开始分散储备，增持黄金和人民币
3. **黄金需求飙升**：2022-2024年全球央行每年购金超1000吨（历史第四、五高）
4. **储备管理范式转换**：从"最大化收益"转向"安全性+多元化"

> [!key-insight] 俄罗斯冻结事件的启示
> 外汇储备不仅是经济工具，也是地缘政治工具。持有大量美元储备的国家面临一个"两难"：储备越多越安全（对市场），但也越脆弱（对制裁）。这一认知正在重塑全球央行的储备管理策略。
