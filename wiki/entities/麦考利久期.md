# 麦考利久期

> 债券各期现金流收回时间的加权平均

## 核心定义

麦考利久期（Macaulay Duration）是债券各期现金流收回时间的加权平均，由Frederick Macaulay于1938年提出，是所有久期概念的基础。

## 核心公式

$$D_{mac} = \frac{\sum_{t=1}^{n} \frac{t \cdot C}{(1+r)^t} + \frac{n \cdot F}{(1+r)^n}}{P}$$

其中：
- $t$ = 现金流发生的时间（期数）
- $C$ = 每期票息
- $F$ = 面值
- $r$ = 每期折现率
- $P$ = 债券价格

## 性质

| 性质 | 说明 |
|------|------|
| 时间单位 | 以"期数"或"年"为单位 |
| 范围 | 0 < $D_{mac}$ < 到期年限 |
| 零息债券 | $D_{mac}$ = 到期年限 |
| 永续债券 | $D_{mac}$ = (1+r)/r |

## 影响因素

| 因素 | 对麦考利久期的影响 |
|------|-------------------|
| 到期期限 | 期限越长，久期越长 |
| 票息率 | 票息越高，久期越短 |
| 市场利率 | 利率上升，久期缩短 |

## 与修正久期的关系

$$D_{mod} = \frac{D_{mac}}{1 + r}$$

修正久期 = 麦考利久期 / (1 + 每期利率)
