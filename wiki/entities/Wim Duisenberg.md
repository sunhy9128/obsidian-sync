> **ECB 的奠基者**——"Mr. Euro"，从荷兰走向欧洲金融史的核心

## 二、职业生涯轨迹

### 2.1 学术与早期（1935-1973）

```
1935.7.9 ：荷兰 Heerenveen 出生
1961 ：阿姆斯特丹大学经济学讲师
1965-1973：荷兰财政部高级经济学家
```

### 2.2 进入政界（1973-1997）

```
【政治生涯】
  1973-1977：荷兰经济事务部部长
  - 1973 石油危机期间
  - 处理能源价格冲击
       ↓
【女王任命】
  1977-1981：荷兰 PvdA（工党）议员
  1981 ：PvdA 议会党团领袖
       ↓
【学术回归】
  1982-1997：阿姆斯特丹大学教授
       ↓
【荷兰央行行长】
  1997.1.1 - 1998.12：荷兰银行（De Nederlandsche Bank）行长
  - 任期 2 年
  - 同时担任 ECB 筹备委员会主席
  - ⭐ 主导 ECB 的制度设计
```

### 2.3 ECB 首任行长（1998-2003）

```
1998.6.1 ：ECB 正式成立，Duisenberg 出任首任行长
  - 任期原本为 8 年
  - 但 2003 年提前 5 年辞职
       ↓
⭐ 提前辞职的原因：
  - 与法国达成"轮换协议"
  - 法国坚持 Trichet 应接替 Duisenberg
  - 实际是法德政治妥协
  - 2003.10.31：Duisenberg 卸任
  - Trichet 接任

⭐ 历史意义：
  - 他完成了"欧元制度"的创建
  - 1999.1.1 欧元作为电子货币诞生
  - 2002.1.1 欧元纸币硬币流通
  - 欧元区从 11 国扩张到 12 国（2001 希腊加入）
```

## 三、Duisenberg 的"欧元遗产"

### 3.1 核心贡献

```
【贡献 1：ECB 制度设计】
  - 1997-1998 担任 ECB 筹备委员会主席
  - 设计 ECB 治理结构、货币政策框架
  - 制定"价格稳定"单一目标
  - ⭐ ECB 的"建筑师"

【贡献 2：欧元创立】
  - 1999.1.1 欧元作为电子货币诞生
  - 2002.1.1 欧元纸币硬币流通
  - 11 国共同货币
  - ⭐ "Mr. Euro"

【贡献 3：早期货币政策】
  - 1999-2001 货币政策框架稳定
  - 设定欧元基准利率
  - 建立 ECB 与各国 NCB 的协调机制

【贡献 4：国际合作】
  - 与美联储、日本银行的协调
  - G7 央行行长俱乐部的常客
  - ⭐ 全球央行合作基础
```

### 3.2 任期的挑战

```
【挑战 1：欧元诞生初期的不确定性】
  - 1999-2002 欧元对美元持续贬值
  - 从 1.18 跌至 0.85（2000）
  - ⭐ "欧元贬值危机"

【挑战 2：经济增长放缓】
  - 2001-2003 美国互联网泡沫破裂
  - 欧元区经济增速放缓
  - ECB 多次降息

【挑战 3：政治妥协提前辞职】
  - 任期 8 年，但只做 5 年多
  - 2003.10.31 提前辞职
  - ⭐ 反映 ECB 行长职位的政治性
```

## 四、Duisenberg 的个人风格

### 4.1 领导特征

```
【1】"Mr. Euro"的亲和力
  - 经常出席公开活动
  - 用"欧元先生"形象推广欧元
  - 获得欧洲民众广泛认可

【2】荷兰务实主义
  - 强调"价格稳定"
  - 不追求"扩张性货币政策"
  - 严谨、稳定、可预测

【3】政治妥协
  - 在 ECB 筹备期与法德协商
  - 在行长任期上让步
  - ⭐ "政治家的妥协艺术"
```

### 4.2 历史评价

```
⭐ Duisenberg 的历史定位：

【积极评价】
  - "ECB 的建筑师"
  - "Mr. Euro"
  - 完成了不可能的任务（让 11 国同意放弃本国货币）
  - ⭐ 欧元区历史第一功臣

【争议】
  - 任期较短（5 年 vs 原定 8 年）
  - 欧元早期持续贬值（但这是外部因素）
  - ⭐ 评价褒大于贬
```

## 五、一句话总结

> **Wim Duisenberg = ECB 的奠基者 + "Mr. Euro"——欧元区历史的开创者**。
>
> 他在 1998 年创建 ECB，1999 年让欧元诞生，2002 年让欧元纸币硬币流通。他是 ECB 历史第一位行长，也是欧元区历史最重要的奠基者之一。
>
> **理解 Duisenberg = 理解三件事**：
> 1. **ECB 的诞生**：从无到有的制度设计
> 2. **欧元的诞生**：人类历史上第二次大规模跨国货币实验
> 3. **央行行长的政治性**：法德妥协、提前辞职
>
> Duisenberg 没有经历欧债危机，但他设计的 ECB 制度框架被 Draghi、 Lagarde 继承至今。
