# 布雷顿森林体系瓦解

## Basic Information
- Type: event
- Source: [[sources/石油美元体系|石油美元体系]]

## Description
布雷顿森林体系瓦解发生于1971年，是[[sources/石油美元体系|石油美元体系]]诞生的直接历史前奏。该体系崩溃前，美元与黄金以固定汇率挂钩（每盎司35美元），其他主要货币则与美元挂钩，形成了战后国际货币秩序的核心架构。1971年8月，尼克松政府单方面宣布终止美元兑换黄金的承诺（即"尼克松冲击"），这一举动直接切断了美元与黄金之间的最后信用纽带。此举的深层原因之一是[[entities/特里芬难题|特里芬难题]]——美国为向全球提供流动性而必须保持贸易逆差，但持续的逆差又使美元兑换黄金的承诺变得不可持续。美元失去黄金锚定物后，美国亟需为美元寻找新的价值支撑基础，这为后来通过[[entities/沙特协议|沙特协议]]将美元与全球最重要的大宗商品——石油——进行绑定创造了历史条件和战略需求。该事件也与随后发生的[[entities/第一次石油危机|第一次石油危机]]共同构成了20世纪70年代国际货币体系重构的关键节点。
