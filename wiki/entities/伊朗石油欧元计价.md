# 伊朗石油欧元计价

## Basic Information
- Type: event
- Source: [[sources/石油美元体系|石油美元体系]]

## Description
伊朗石油欧元计价是2000年代伊朗为挑战美元霸权而尝试的石油贸易结算方式改革，意图以欧元替代美元作为石油出口的计价和结算货币。这是[[sources/石油美元体系|石油美元体系]]历史上最具代表性的挑战尝试之一，直接触动了美元在石油贸易中的核心地位。该尝试招致了美国的严厉制裁，最终未能成功扩大影响。伊朗的尝试表明，任何试图绕开美元进行石油定价的行为都会面临巨大的政治和经济压力，也是[[sources/石油美元体系|石油美元体系]]自我维护机制的具体体现。
