# 伊朗石油欧元计价

> 2000年代伊朗挑战美元石油定价体系的首次系统性尝试——失败揭示了"体系内货币替代"的根本困境

## 基本信息

| 项目 | 内容 |
|------|------|
| 全称 | Iran Euro-denominated Oil Pricing |
| 时间 | 2000年代（内贾德时期，约2000-2012年）|
| 本质 | 用欧元替代美元作为石油出口计价和结算货币 |
| 结果 | **失败**：被全面制裁，石油出口从250万桶/日锐减至100万桶/日 |

---

## 一、背景：伊朗为何挑战石油美元

### 1.1 意识形态驱动

```
【内贾德时期的政治逻辑】
├── "反美"是伊朗外交核心立场
├── 用石油收入资助黎巴嫩真主党、哈马斯等"抵抗轴心"
├── 主动宣布石油以欧元计价：政治宣示 > 经济理性
└── 动机：挑战美元霸权，输出革命意识形态
```

### 1.2 与俄罗斯人民币结算的本质差异

> **核心区分**：伊朗试图用**体系内货币（欧元）**替代美元——欧元背后同样受美国影响。俄罗斯用**体系外货币（人民币）**替代美元——中国不受美国二级制裁约束。这是两者命运截然不同的根本原因。

| 对比维度 | 伊朗欧元计价 | 俄罗斯人民币结算 |
|---------|-------------|----------------|
| **替代货币** | 欧元（美元体系内）| 人民币（美元体系外）|
| **结果** | 失败 | 成功 |
| **制裁响应** | 全面崩溃 | 基本维持 |

---

## 二、失败机制解剖

### 2.1 美国的反制路径

```
【美国对伊朗的反制三阶段】

Phase 1：金融隔离（2012）
├── 切断伊朗银行 SWIFT 接入
└── 伊朗被完全踢出美元清算体系

Phase 2：次级制裁
├── 任何与伊朗贸易的第三方国家面临制裁风险
├── 欧盟企业被迫退出（道达尔、壳牌等）
└── 无人敢接手伊朗石油

Phase 3：军事威胁
├── 以色列暗示军事打击
├── 美国航母战斗群威慑
└── 伊朗谈判筹码消失
```

### 2.2 二级制裁的有效性

**核心问题**：欧洲企业面临"**失去美国市场 OR 失去伊朗石油**"的选择题——毫无疑问选择前者。

| 二级制裁有效性条件 | 伊朗遇到的情况 |
|-----------------|----------------|
| 对方高度依赖美国市场 | ✓ 欧洲企业美国市场利益巨大 |
| 对方是美国盟友或半盟友 | ✓ 欧盟是北约成员 |
| 对方无法找到替代市场 | ✗ 沙特、俄罗斯可填补缺口 |

### 2.3 欧盟为何不救伊朗

| 欧盟立场 | 具体原因 |
|---------|---------|
| **安全依赖** | 北约保护依赖美国，不敢对抗 |
| **经济利益** | 欧洲企业在美利益远大于伊朗 |
| **替代来源** | 沙特、俄罗斯可填补伊朗石油缺口 |
| **政治分裂** | 英国（伦敦金融城）率先退出 INSTEX |
| **意愿不足** | 欧盟并无"去美元"的真正政治意志 |

## 三、INSTEX 的失败：伊朗教训的欧盟版本

### 3.1 INSTEX 的设计缺陷

> 来源：[[INSTEX]] §五

```
【INSTEX 的五大失败原因】

① 人道主义商品的"双向贸易"难匹配
├── 伊朗需要进口：药品、医疗器械
├── 但伊朗能出口的：石油（被制裁）、石化
└── 进出口对等 → 难以找到匹配的"易货"组合

② 欧盟企业的合规恐惧
├── 美国长臂管辖 = 美元清算账户冻结
├── 任何欧盟企业参与都面临次级制裁风险
└── 风险收益比极差 → 企业不愿参与

③ 欧盟内部对美国的依赖差异
├── 安全依赖美国：北约、情报、军售
├── 经济依赖美国：金融市场、技术
└── 政治依赖美国：俄乌、中东议题需要美方支持

④ 基础设施缺陷
├── 缺乏与伊朗的有效银行通道
├── 没有对应的伊朗版 INSTEX 实体支持
└── 与 CIPS/SPFS 等成熟系统差距巨大

⑤ 政治意愿不足
├── 欧盟内部对 INSTEX 的政治支持有限
└── 沦为"政治表态工具"
```

### 3.2 INSTEX 揭示的规律

> **核心洞察**（来源：[[INSTEX]] §六）：非美元清算基础设施的建设，需要的不只是金融技术，更是**政治意志**。欧盟缺乏这种意志，所以失败；中俄有，所以成功。

## 四、与俄罗斯人民币结算的结构性对比

### 4.1 经济规模与内循环能力

| 维度 | 伊朗 | 俄罗斯 |
|------|------|--------|
| **GDP（2021）** | ~$3600亿 | ~$1.8万亿 |
| **外汇储备（2022前）** | ~$1000亿（被冻结）| ~$6400亿（$3000亿被冻结）|
| **能源出口市场** | 欧洲为主（可替代性高）| 欧洲+中国（可转向）|
| **替代市场** | 欧洲转向沙特/俄罗斯 | 中国接手欧洲份额 |
| **内循环能力** | 弱 | 较强 |

### 4.2 替代基础设施就绪度

| 维度 | 伊朗 | 俄罗斯 |
|------|------|--------|
| **独立清算系统** | 无（SEPAM规模极小）| **SPFS已运营** |
| **CIPS接入** | 无 | **已接入** |
| **本币互换协议** | 无 | **中俄本币互换已建立** |
| **银行通道** | 被完全切断 | 部分被切但核心通道保留 |

### 4.3 互补性差异

```
【为什么中俄能源人民币结算具有不可替代性？】

俄罗斯需要：
├── 能源出口市场（欧洲市场已失）
└── 不被美国冻结的结算货币

中国需要：
├── 能源安全（多元化进口来源）
└── 人民币国际化的实物支撑

【锁死结构】
俄罗斯能源 + 人民币结算
  → 俄罗斯获得人民币，可购买中国商品
  → 中国获得稳定能源，用人民币支付
  → 两国经济高度互补，无需美元中介

伊朗能源 + 欧元结算（对比）
  → 伊朗获得欧元，但无法购买欧洲商品（制裁）
  → 欧洲可从沙特等替代来源获得能源
  → 互补性低，易被替代
```

## 五、核心洞察

> [!key-insight] "体系内替代"的困境
>
> 伊朗欧元计价的失败揭示了一个根本规律：**用一种主权货币替代另一种主权货币（且两者背后是同一地缘政治阵营），注定失败。** 欧元受美国影响的深度，使它无法成为美元的真正替代者。

> [!key-insight] 二级制裁的有效性边界
>
> 二级制裁能有效威慑"盟友"，但对"战略竞争对手"效果有限。**中国不受美国二级制裁约束**——这是俄罗斯人民币结算成功的关键前提。

> [!key-insight] 能源武器的非对称性
>
> 伊朗的霍尔木兹海峡威胁本身会招来军事打击；俄罗斯的能源卢布结算令则与 CIPS 配合，形成了完整的经济反制体系。

> [!contradiction] 意识形态驱动的脆弱性
>
> 伊朗的欧元计价是"政治宣示 > 经济理性"的产物；俄罗斯的人民币结算是"生存压力 > 政治偏好"的产物。**后者具有更强的韧性和可持续性。**
