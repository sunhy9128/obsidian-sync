# 市场风险

> 金融市场价格波动导致损失的风险

## 核心定义

市场风险（Market Risk）是指因金融市场价格（利率、汇率、股票价格、商品价格）波动而导致资产价值变化或负债成本增加的风险。

## 主要类型

- 利率风险
- 汇率风险
- 股票价格风险
- 商品价格风险

## 监管框架

巴塞尔协议对市场风险有专门要求：

| 框架 | 方法 |
|------|------|
| **旧版 Basel II** | 标准法 + 内部模型法（VaR） |
| **Basel II.5（2009）** | VaR + Stress VaR 双资本附加 |
| **FRTB / Basel III.1（2019）** | Expected Shortfall（ES）替代 VaR |
