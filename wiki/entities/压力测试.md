# 压力测试

> 评估银行在极端经济情景下稳健性的分析方法

## 核心定义

压力测试（Stress Testing）是一种风险管理工具，通过模拟宏观经济危机、市场大幅下跌或特定风险事件等极端情景，评估金融机构的资本充足性和流动性。2008 年金融危机暴露了传统风险模型的局限性，压力测试从"可选工具"升级为**监管核心要求**。

## 主要类型

- **敏感性分析**：单一因素变化的影响——固定其他变量，只改变单一风险因子（如利率+100bps、股票指数-20%）。优点是直观快速，缺点是忽略风险因子之间的相关性
- **情景分析**：多因素组合情景，模拟真实极端情景（利率+汇率+股票+信用+流动性同时变化）
- **逆向压力测试**：从"失败结果"反向寻找可能导致崩溃的情景，打破正向思维、发现传统模型无法捕捉的尾部风险
- **宏观压力测试**：宏观经济模型驱动（如 Credit Portfolio View 利用 PD=f(GDP, 失业率, 利率, 房价)），主要用于监管要求和资本规划

## 情景构建方法论

### 历史情景

基于真实发生过的极端市场事件：

| 危机事件 | 核心冲击 | 传染路径 |
|----------|----------|----------|
| 2008 全球金融危机 | 信用利差飙升、流动性枯竭、房价暴跌 | MBS → 银行 → 实体经济 |
| 2010-2012 欧债危机 | 主权利差急剧扩大、欧元存续疑虑 | 主权 → 银行 → 实体 |
| 1997 亚洲金融危机 | 汇率崩溃、资本外流、资产价格暴跌 | 汇率 → 银行 → 企业 |
| 2015 中国股灾 | A股暴跌 + 人民币贬值 | 股市 → 流动性 → 汇率 |
| 2020 COVID-19 | 全球经济骤停、油价暴跌 | 供给冲击 → 需求冲击 |

### 假设情景

基于专家判断构建"历史上未发生但可能发生"的场景，例如全球性银行危机情景（3 家 G-SIB 同时陷入困境、汇率波动±30%、股指下跌 40-60%、主权利差扩大 300-500bps）。

### 混合情景

历史情景 × 当前市场条件修正。

## 监管要求

- 美联储 CCAR 年度压力测试（资产 ≥ $100 亿，假设 9 季度压力情景，含资本分配计划审查和定性评估）
- 美联储 DFAST 年度压力测试（假设 3 年压力情景，27 个宏观经济变量，公开披露 9 大模块结果）
- 巴塞尔协议要求：基准压力测试（监管提供统一情景）+ 机构内压力测试（各机构自行构建）+ 跨境一致性
- 中国监管：银保监会季度/年度压力测试（信用、市场、流动性）、央行金融稳定压力测试（宏观审慎）、房地产集中度专项压力测试

## 七步实施流程

1. **确定范围**：业务线、风险类型、时间范围
2. **构建情景**：历史/假设/逆向，设定严重程度（轻、中、重）
3. **风险因子映射**：宏观变量 → 市场风险因子、PD/LGD、流动性冲击
4. **风险计量**：VaR/ES 重算、PD/LGD 上调重算 RWA、现金流缺口分析
5. **汇总结果**：各业务线汇总，展示资本充足率/流动性覆盖率在压力下的表现
6. **管理层报告**：识别风险集中点和脆弱性
7. **制定应对计划**：资本缓冲规划、应急流动性计划（CFP）

### 常见错误

| 错误 | 说明 |
|------|------|
| 情景过于乐观 | 压力程度不够，无法真正测试抗压能力 |
| 忽视相关性崩塌 | 正常市场下相关性低的资产，压力下相关性急剧上升 |
| 静态资产负债表 | 假设银行不调整资产组合（现实中会减仓/对冲）|
| 单一情景依赖 | 结果对情景选择过于敏感 |
| 忽视二阶效应 | 第一轮冲击 → 传染效应 → 第二轮更大冲击 |

## 资本规划与压力测试

压力测试结果驱动资本需求与资本规划决策：

- 压力后 CET1 远高于要求 → 可增加分红/回购
- 压力后 CET1 接近要求 → 限制资本分配
- 压力后 CET1 低于要求 → 必须补充资本/减少风险加权资产

Basel III 资本层级：CET1 ≥ 4.5% + 资本保全缓冲 2.5% + 逆周期资本缓冲 0-2.5% + G-SIB 附加 1-3.5%。

## 流动性压力测试

现金流压力测试模拟压力情景下的资金流入/流出，覆盖 30 天（短期）、90 天（中期）、1 年（长期结构性）三个时间维度，关键指标为净现金流缺口和流动性覆盖率（LCR = HQLA / 30 天净现金流出）。典型情景包括机构自身危机（评级下调、存款流失）、系统性危机（融资困难、对手方传染）以及两者混合。

## 系统性风险与传染模型

- **直接敞口传染**：贷款、衍生品净额
- **间接传染**：资产价格联动、信息传染
- **流动性传染**：集体抛售同类资产

信贷损失的二阶循环效应：压力下 PD↑ → EL↑ → 拨备↑ → 利润↓ → 资本↓ → 信贷收缩 → 实体经济恶化 → PD 进一步上升。
