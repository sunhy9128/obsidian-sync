# 万科宝能股权之争

## 核心命题

2015-2017年万科宝能股权之争是中国资本市场最典型的控制权争夺战。核心矛盾：王石作为无实际控制权的「经理人」，通过管理层盈安合伙人暗中打压股价、增持控股，被宝能系姚振华识破并发动恶意收购。

**三方博弈格局**：
- 防守方：王石管理层 + 安邦
- 进攻方：宝能系（姚振华）
- 原大股东：华润（傅育宁时期袖手旁观）

**关键数据**：
- 万科账面资金600多亿，股价长期低迷
- 仅需230亿即可拿下第一大股东之位
- 宝能系从5%举牌到24.26%持股，仅用不到半年

## 战争时间线

| 时间 | 事件 |
|------|------|
| 2014年底 | 宝能系开始研究万科，筹备资金 |
| 2015年7月11日 | 宝能首次举牌，持股5% |
| 2015年8月27日 | 宝能持股超华润，成为新第一大股东 |
| 2015年12月7日 | 宝能持股达20.008% |
| 2015年12月8日 | 安邦举牌，加入防守方 |
| 2015年12月18日 | 万科紧急停牌，理由：筹划重大资产重组 |

## 关键人物

- **王石**：万科创始人，持股不足0.1%，但实际控制人
- **姚振华**：宝能系老板，农贸生意起家，打法激进凶猛
- **傅育宁**：华润董事长，关键时刻袖手旁观

## 资金来源

宝能系收购资金来源：
- 资管计划
- 发债
- 信用贷款
- 股权质押
- 万能险保险资金（备受争议）

层层嵌套，股权层层抵押。

## 核心机制

1. **中国特色企业治理**：人治穿插于规则之中，无股份也可实际控制
2. **管理层增持**：盈安合伙人4.48% + 打压股价暗中买进
3. **杠杆收购**：用少量自有资金撬动大额收购
4. **停牌武器**：利用杠杆资金时间成本，拖长战局

## 事件意义

这场争斗揭示了中国资本市场的本质：控制权可以脱离股权存在，管理层通过制度设计和人治优势可以实际控制企业，而当外部资本挑战这种控制权时，游戏规则往往让位于政治博弈。
