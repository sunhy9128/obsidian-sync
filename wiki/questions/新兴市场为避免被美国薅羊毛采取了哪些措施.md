# 新兴市场为避免被美国薅羊毛采取了哪些措施

## 问题

新兴市场为了避免被美国"薅羊毛"（美元收割），采取了哪些具体措施？

## 核心答案

新兴市场的应对措施可分为四类：**去美元化、资本管制、区域联盟、汇率弹性管理**。没有单一措施能完全抵御美元周期，但多管齐下可以降低脆弱性。

---

## 第一类：去美元化（减少对美元依赖）

### 1. 储备资产多元化

```
减持美元资产 → 增持黄金/欧元/人民币/本币
```

**典型操作**：
- 俄罗斯：2022年将外汇储备从美元全部转出，增持黄金 + 人民币
- 中国：美债持有量从2013年$1.3万亿降至2024年$7700亿；黄金储备从1058吨增至2262吨
- 全球央行：2022-2024年黄金购买量创历史新高

### 2. 贸易结算本币化

```
不用美元结算 → 用本币或人民币结算
```

**典型操作**：
- 中俄贸易：人民币结算从2014年<5%升至2024年约90%
- 中国-东盟：推进本币结算框架（LCS）
- 中东：部分石油贸易开始用人民币结算

### 3. 建立替代清算基础设施

```
绕开SWIFT/CHIPS → 建自己的系统
```

| 系统 | 主体 | 功能 |
|------|------|------|
| **CIPS** | 中国 | 人民币跨境支付 |
| **SPFS** | 俄罗斯 | 卢布跨境清算 |
| **mBridge** | 多边 | 央行数字货币（CBDC）桥接 |
| **INSTEX** | 欧洲 | 欧元结算（绕开SWIFT） |
| **BRICS Pay** | 金砖 | 金砖国家本币结算（推进中） |

---

## 第二类：外汇管制（资本防火墙）

### 4. 加强资本管制

```
限制资本自由流动 → 防止资金快进快出
```

**典型操作**：
- 设置外资持仓比例上限
- 要求出口企业强制结汇
- 限制居民购汇额度
- 对短期热钱征托宾税

### 5. 建立外储缓冲池

```
积累充足外汇储备 → 应对资本外流冲击
```

**逻辑**：外汇储备够多 = 有能力在美元升值时干预汇市 + 偿还到期美元债 = 不容易被一次性击溃

---

## 第三类：区域联盟（抱团取暖）

### 6. 地缘政治联盟

```
与美国不是一路 → 找别的伙伴
```

- **金砖国家**：讨论共同货币 + 建立金砖支付系统
- **上合组织**：推进本币结算合作
- **RCEP/东盟**：扩大人民币使用场景
- **中阿合作**：石油人民币结算试点

---

## 第四类：主动管理汇率弹性

### 7. 增加汇率弹性

```
不让汇率死盯美元 → 让汇率市场化调节
```

**逻辑**：汇率有弹性 = 贬值是缓冲垫，不需要一次性崩到底 = 比硬盯美元的汇率制度更能扛

---

## 这些措施管用吗？

| 措施 | 有效性 | 局限 |
|------|--------|------|
| 去美元化（储备） | 中长期有效 | 美元资产仍是最佳流动性资产，短期难完全替代 |
| 本币结算 | 双向贸易有效 | 单边依赖美元时无效 |
| 替代清算系统 | 绕过制裁有效 | 规模不够，SWIFT仍主导 |
| 资本管制 | 短期防火墙 | 长期扭曲市场，抑制增长 |
| 外汇储备缓冲 | 直接有效 | 消耗后难补充 |
| 汇率弹性 | 缓冲有效 | 增加不确定性 |
