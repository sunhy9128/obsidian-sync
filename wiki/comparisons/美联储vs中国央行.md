# 美联储 vs 中国央行：制度范式全面对比

> 两个央行分属不同制度范式，理解其差异需要跳出"工具罗列"层面，从制度基因、传导逻辑、危机哲学三个层面把握。

> **核心命题**：美联储代表"形式独立 + 总量调节 + 印钱换时间"的盎格鲁-撒克逊央行范式；中国央行代表"协调型自主 + 结构性定向 + 借新还旧换时间"的党管金融范式。两种范式在工具名称上多有相似，背后的制度基因、传导逻辑、危机哲学截然不同。

---

## 一、制度定位与独立性

| 维度 | [[wiki/entities/美联储\|美联储]] | [[wiki/entities/中国央行\|中国央行（PBOC）]] |
|---|---|---|
| **法律地位** | 联邦储备法下的独立机构，1930 年代大萧条后确立独立性 | 国务院下属正部级单位，受国务院领导 |
| **独立性程度** | 高（虽然近年政治压力增加） | 中等（党管金融 + 国务院领导） |
| **国际比较** | 介于 ECB（极高）和 BOJ 之间 | 接近政府主导端（但有效得多） |
| **核心约束** | 联邦储备法 §14(b)：原则上不能直接买新发国债，例外上限 $50 亿，需国会监督 | 《人民银行法》不允许央行直接认购国债 |

> **关键洞察**：独立性不是"非此即彼"，而是连续谱。美联储在"形式独立、实质协调"上经历了 2020 年压力测试；中国央行则在"高度协调"框架内通过 PSL、再贷款等结构性工具保持操作自主性（详见 [[wiki/concepts/美联储独立性\|美联储独立性]]）。

---

## 二、决策架构与人事

| 维度 | 美联储 | 中国央行 |
|---|---|---|
| **决策机构** | FOMC（联邦公开市场委员会，12 名成员） | 央行党委 + 行长办公会 |
| **关键人事** | 主席（4 年）+ 7 名理事（**14 年**任期，远超总统 4 年任期） | 行长（5 年）+ 副行长 |
| **历史关键人物** | [[wiki/entities/鲍威尔\|鲍威尔]]（2018-2026）→ 沃什（2026-，鸽派）| 周小川（2002-2018，钱荒决策者）→ 易纲 → 潘功胜 |
| **当前政治格局** | 特朗普施压降息 vs 鲍威尔鹰派制衡 | 与国务院、财政部政策协同 |

人事对比的反差：
- 美联储通过超长任期实现"政治隔离"
- 中国央行行长任期与政府保持同步，体现"协调型独立"

## 三、政策工具箱

| 工具类型 | 美联储 | 中国央行 |
|---|---|---|
| **价格型（基准利率）** | 联邦基金利率（FOMC 决定） | [[wiki/concepts/LPR\|LPR]]（18 家报价行每月 20 日） |
| **数量型（准备金）** | 准备金率（较少调整） | [[wiki/concepts/存款准备金率\|存款准备金率]]（**主要工具**，全球独特） |
| **中期流动性** | 公开市场操作（OMO） | [[wiki/concepts/MLF\|MLF]]（麻辣粉，2013 年创设） |
| **长期结构性** | QE（资产购买） | [[wiki/concepts/PSL\|PSL]]（抵押补充贷款，政策性银行通道） |
| **超短期** | 贴现窗口、回购 | SLF、SLO |
| **创新工具** | TAF、PDCF、CPFF、BTFP、MMLF | 买断式逆回购、离岸央票 |
| **窗口指导** | 不采用 | 历史上有（详见 [[wiki/concepts/窗口指导\|窗口指导]]） |

> **关键差异**：中国央行的工具箱是"**多档期限 + 结构性定向**"组合，而美联储偏向"**总量调节 + 市场传导**"。中国央行的存款准备金率作为常规冲销工具的用法在全球范围内较为少见。

## 四、传导机制

### 4.1 美联储的传导链

```
FOMC 设定联邦基金利率目标
    ↓
银行间隔夜市场成交价 → 联邦基金利率
    ↓
影响短端利率（国库券、商业票据）
    ↓
影响长端利率（房贷、企业债）
    ↓
影响投资、消费、汇率
```

工具：[[wiki/concepts/扩表与缩表\|扩表（QE）]] 直接影响银行准备金和资产价格。

### 4.2 中国央行的传导链

```
央行 7 天逆回购利率（政策利率）
    ↓
    ↓ 2019 年改革后
[[wiki/concepts/LPR\|LPR]]（18 家报价行，MLF/7 天逆回购 + 加点）
    ↓
银行贷款利率
    ↓
影响实体融资成本
```

工具：MLF、PSL、再贷款定向投放，**绕过"宽货币→宽信用"陷阱**。

> **关键洞察**：美联储主要靠"价格信号"传导，中国央行主要靠"数量 + 结构"双轮驱动（详见 [[wiki/concepts/央行政策\|央行政策]]）。

## 五、危机应对哲学（最深刻差异）

| 维度 | 美联储 QE | 中国化债 |
|---|---|---|
| **本质** | 印钱买债（创造货币） | 借新还旧（结构转换） |
| **资金流向** | 直接进入市场流通 | 冻结在政府账户 |
| **M2 影响** | 大幅扩张 | 短期收缩 |
| **流动性** | 注入 | 冻结 |
| **通胀代价** | 显著（2021-2022 美 9.1%）| 温和 |
| **银行代价** | 短期受益 | 长期 ROE 下降 |
| **本质标签** | Monetary Creation（货币创造） | Monetary Restructuring（货币结构重塑） |
| **中国版命名** | — | QBS（Quantity with Bank Shrinkage，量化但银行收缩） |

> **核心命题**（详见 [[wiki/concepts/化债核心命题\|化债核心命题]] 和 [[wiki/concepts/QE与化债对比\|QE与化债对比]]）：
>
> **QE 让货币"变多"，化债让货币"搬家"。**
>
> **QE 注入流动性，化债冻结流动性。**
>
> **QE 是"被动救市"，化债是"主动出清"。**

这是两种截然不同的危机哲学：美联储用"印钱换时间"，中国用"金融体系让利换时间"。

## 六、当前周期定位（2026 年）

| 维度 | 美联储 | 中国央行 |
|---|---|---|
| **政策利率** | 2024.9 降息 50bp，进入降息周期 | LPR 维持低位（1Y 3.0%, 5Y+ 3.5%） |
| **资产负债表** | ~$6.7 万亿，QT 接近尾声 | 扩表受限，结构性工具有限使用 |
| **关键讨论** | 沃什主张"降息 + 缩表"双重紧缩组合 | 化债推进 + 房地产软着陆 + 财政体制改革 |
| **核心矛盾** | 政治压力降息 vs 抗通胀遗留 | 银行 ROE 下降 vs 化债资金需求 |
| **未来路径** | 是否再次扩表应对经济放缓 | 第三类债务处置 + 化债第三阶段 |

## 七、全球角色与汇率

| 维度 | 美联储 | 中国央行 |
|---|---|---|
| **国际地位** | 全球储备货币发行国 | 第二大经济体，人民币国际化推进中 |
| **跨境工具** | 美元互换额度（多国央行） | [[wiki/concepts/mBridge\|mBridge]]、双边本币互换 |
| **冲销实践** | 较少使用（详见 [[wiki/concepts/冲销式干预\|冲销式干预]]） | **全球主要实践者**（2002 年以来大量使用） |
| **关键约束** | 美元储备货币地位的双刃剑 | 资本管制环境使冲销更有效 |

中国央行的冲销实践具有全球独特性：一方面将存款准备金率作为常规冲销工具灵活使用，另一方面通过发行离岸央票延伸了冲销的地理边界。

---

## 八、五句话总结

1. **独立性**：美联储追求"形式独立"，中国央行追求"协调型自主"，路径不同但都在与政府博弈中保持核心原则。

2. **工具箱**：美联储的工具偏"总量 + 价格信号"，中国央行的工具偏"数量 + 结构性定向"，后者更精细但传导链条更长。

3. **危机哲学**：美联储信奉"印钱换时间"（QE），中国信奉"借新还旧换时间"（化债），前者付通胀代价，后者付银行 ROE 代价。

4. **杠杆态度**：美联储允许宏观杠杆大幅扩张（2008: $0.8 万亿 → 2022: $9 万亿，10 倍），中国坚持"不大水漫灌"，通过结构性工具对冲化债的流动性冻结。

5. **未来路径**：美联储关注政治化压力下独立性边界，中国央行关注化债完成后的第三类债务处置与财政体制改革协同。
