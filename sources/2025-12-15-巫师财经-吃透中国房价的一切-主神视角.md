# 吃透中国房价的一切，但是主神视角

## 源信息

- **平台**：微信公众号
- **作者**：巫师财经
- **发布日期**：2025-12-15
- **字数**：约20,000字

## 核心命题

### 主神的战略：稳房价就是稳人心

稳房价 → 稳市场 → 稳银行 → 稳金融 → 稳秩序 → 稳社会 → **稳人心**

- **硬着陆**：价格下行夹角逼近90度，一刀跌透（跳楼）
- **软着陆**：拉长时间轴，让夹角变小；房价下行要持续**一代人甚至两代人**

### 锚的理论

巫师的「锚」概念：
- 人类的唯一锚 = **劳动时间**
- 钱、公司、股票、债券都是「金融幻象」，是抽取/转移锚的工具
- **债务**是把「抽取锚」的力量发挥到极致的机制
- 人类群体博弈本质：「我享受，而别人工作」
- Aha moment = 交流 → 交换 → 交易链条的建立

### 传导链：崩盘后果推演

小A 1000万买房 → 房价暴跌 → 断贷 → 银行系统崩溃 → 经济通缩 → 失业 → 社会动荡

**「共识」是最危险的**：一旦全民形成房价必跌共识 → 死亡冲锋 → 社会不安定因素大量形成

### 力挽狂澜：分摊痛苦

四招：
1. **暗抬购买力**：第四代住宅，加量不加价
2. **减少供给**：房地产开发投资-30%，新开工面积接近腰斩
3. **价格保护机制**：窗口指导 + 业主自发「保价联盟」
4. **分摊痛苦**：A→B→C→D阶梯式接盘化解社会不安定因素

### 稳字当头

- **保交楼**：2024年白名单贷款5.6万亿，交付率河南95%，黑/宁/青100%
- **救房企**：债务重组 = 短债拉长 + 利息打折 + 债转股
- **12万亿化债方案**：高息短债→低息长债
- **财政历史性转折**：2025赤字率4%（改革开放以来新高）

### 小市场分析框架

**「巧克力块」理论**：每个小区是一个独立小市场，旱的旱死涝的涝死

唯一决定因素：「有个大聪明乐意接，愿意多少钱接」

**不可替代价值** = 学区（举例北京）

### 卖方 vs 买方

**卖方**：
- 「绝对亏」vs「相对亏」：你跌200万，别人跌400万 = 你相对赚200万
- 六个大字：**早出手就是赚**

**买方**：
- 俩字：**别急**
- 时间站在买方这边
- **宁当凤尾，不当鸡头**
- **选好物业**：把物业费当投资，不是消费
- **不选期房**：所见即所得
- **房龄5-10年但维护如新**的小区最优

### 房产税

- 不会突然大规模推出，而是**循序渐进+突然文件**
- **真正低点**：房产税通知到执行之间的抛盘窗口期
- 房产税**不是利空出尽**，未来还会有更多持有成本

## 数据亮点

- 房地产开发投资完成额：逐年-30%
- 房屋新开工面积：2025Q3较2023同期接近腰斩
- 房地产对GDP拉动率：已变为负数（开始拖累GDP）
- 白名单贷款：5.6万亿，支持1400万套住房建设交付
- 2025赤字率：4%（改革开放以来新高）
- 北京上海房价收入比：从约50倍降至37倍（下降约30%）
- 租金回报率：从1.3%回升至2%，已追上5年/10年期国债收益率
